# 10-7-6 我的解答

对应题目：[[01_题目/10_圆锥曲线/10-7_圆锥曲线中的焦点三角形问题/10-7-6.md|10-7-6]]

## 完整题目

6.(2024 广东深圳二调)已知在 \( \bigtriangleup  {ABC} \) 中， \( \tan \frac{B}{2} \; = 3\tan \frac{C}{2} \) ，双曲线 \( E \) 以 \( B \) ， \( C \) 为焦点，且经过点 \( A \) ，则 \( E \) 的两条渐近线的夹角为___； \( \tan \frac{A}{2} \; + \tan \frac{C}{2} \) 的取值范围是___.

## 题目勘误
原文中的 `\bigtriangleup {ABC}` 排版有误，应更正为 `\triangle ABC`，属于 OCR 识别导致的符号错误。

## 思路分析
本题将三角形中的半角关系与双曲线的定义相结合。先利用正弦定理将双曲线的离心率用三角形的角表示，再通过已知的半角正切关系及万能公式求出离心率，得到渐近线夹角；第二空则将目标式用参数表示，利用函数单调性求出取值范围。

## 详细解答
设 \(\triangle ABC\) 的内角 \(A,B,C\) 所对的边分别为 \(a=BC\)，\(b=AC\)，\(c=AB\)。  
以 \(B,C\) 为焦点的双曲线经过点 \(A\)，则其焦距 \(2c_0 = BC = a\)，实轴长 \(2a_0 = |AB - AC| = |c - b|\)。  
离心率  
\[
e = \frac{c_0}{a_0} = \frac{a}{|c - b|}.
\]

由正弦定理 \(a = 2R\sin A,\; b = 2R\sin B,\; c = 2R\sin C\)，得  
\[
e = \frac{\sin A}{|\sin B - \sin C|}.
\]

已知 \(\displaystyle \tan\frac{B}{2} = 3\tan\frac{C}{2}\)。令 \(t = \tan\frac{C}{2}\)，由 \(C \in (0,\pi)\) 知 \(t > 0\)，则 \(\tan\frac{B}{2} = 3t\)。
由 \(A+B+C = \pi\) 得 \(\frac{B+C}{2} = \frac{\pi}{2} - \frac{A}{2}\)，故  
\[
\cot\frac{A}{2} = \tan\frac{B+C}{2} = \frac{\tan\frac{B}{2} + \tan\frac{C}{2}}{1 - \tan\frac{B}{2}\tan\frac{C}{2}} = \frac{4t}{1 - 3t^2},
\]
所以  
\[
\tan\frac{A}{2} = \frac{1 - 3t^2}{4t}.
\]
因为 \(\frac{A}{2} \in (0,\frac{\pi}{2})\)，所以 \(\tan\frac{A}{2} > 0\)，得 \(1 - 3t^2 > 0\)，即 \(0 < t < \frac{1}{\sqrt{3}}\).

利用万能公式将 \(\sin A,\sin B,\sin C\) 用 \(t\) 表示：
\[
\sin B = \frac{2\tan\frac{B}{2}}{1+\tan^2\frac{B}{2}} = \frac{6t}{1+9t^2},
\quad
\sin C = \frac{2\tan\frac{C}{2}}{1+\tan^2\frac{C}{2}} = \frac{2t}{1+t^2}.
\]
\[
\tan\frac{A}{2} = \frac{1-3t^2}{4t},\;
1 + \tan^2\frac{A}{2} = \frac{(1+9t^2)(1+t^2)}{16t^2},
\]
\[
\sin A = \frac{2\tan\frac{A}{2}}{1+\tan^2\frac{A}{2}} = \frac{8t(1-3t^2)}{(1+9t^2)(1+t^2)}.
\]
从而  
\[
\sin B - \sin C = \frac{6t}{1+9t^2} - \frac{2t}{1+t^2}
= \frac{4t(1-3t^2)}{(1+9t^2)(1+t^2)}.
\]
因为 \(t > 0\) 且 \(1-3t^2 > 0\)，所以 \(\sin B > \sin C\)，绝对值可去掉：
\[
|\sin B - \sin C| = \sin B - \sin C = \frac{4t(1-3t^2)}{(1+9t^2)(1+t^2)}.
\]
于是离心率  
\[
e = \frac{\sin A}{\sin B - \sin C}
= \frac{\frac{8t(1-3t^2)}{(1+9t^2)(1+t^2)}}{\frac{4t(1-3t^2)}{(1+9t^2)(1+t^2)}} = 2.
\]

**第一空**：由双曲线离心率 \(e = 2\)，渐近线斜率绝对值为 \(\frac{b_0}{a_0} = \sqrt{e^2-1} = \sqrt{3}\)。  
一条渐近线的倾斜角为 \(60^\circ\)，另一条为 \(120^\circ\)，两渐近线所成角中锐角为 \(180^\circ - 120^\circ = 60^\circ\)，即 \(\frac{\pi}{3}\)。

**第二空**：计算  
\[
\tan\frac{A}{2} + \tan\frac{C}{2}
= \frac{1-3t^2}{4t} + t
= \frac{1+t^2}{4t}
= \frac{1}{4}\Bigl(t + \frac{1}{t}\Bigr).
\]
其中 \(t = \tan\frac{C}{2} \in \bigl(0, \frac{1}{\sqrt{3}}\bigr)\)。  
函数 \(f(t) = \frac{1}{4}(t + \frac{1}{t})\) 在 \((0,1]\) 上单调递减，因此在区间 \((0, \frac{1}{\sqrt{3}})\) 上有
\[
f(t) > f\Bigl(\frac{1}{\sqrt{3}}\Bigr) = \frac{1}{4}\Bigl(\frac{1}{\sqrt{3}} + \sqrt{3}\Bigr) = \frac{\sqrt{3}}{3}.
\]
当 \(t \to 0^+\) 时 \(f(t) \to +\infty\)，故所求取值范围为 \(\bigl(\frac{\sqrt{3}}{3}, +\infty\bigr)\)。

## 答案
第一空：\(\displaystyle \frac{\pi}{3}\)（或 \(60^\circ\)）  
第二空：\(\displaystyle \Bigl(\frac{\sqrt{3}}{3}, +\infty\Bigr)\)

## 总结
解决此类焦点三角形问题，关键是将几何条件转化为边角关系，进而用三角函数或半角参数表示离心率。通过整体代换、消元求值，再结合函数性质求范围，可有效避免复杂的解析几何计算。
