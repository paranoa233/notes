# 10-3-3 我的解答

对应题目：[[01_题目/10_圆锥曲线/10-3_抛物线的标准方程及简单的几何性质/10-3-3.md|10-3-3]]

## 完整题目

3.(2024 山东济南一模)与抛物线 \( {x}^{2} = {2y} \) 和圆 \( {x}^{2} + {\left( y + 1\right) }^{2} = 1 \) 都相切的直线的条数为( )

A. 0 B. 1

C. 2 D. 3

## 题目勘误
无明显勘误。

## 思路分析
抛物线开口向上，顶点在原点，圆圆心为 \((0, -1)\)，半径为 \(1\)，且圆恰好经过抛物线的顶点。可先判断出一条显然的公切线，即它们在原点处的公切线 \(y = 0\)。然后通过联立直线与圆锥曲线的方程，利用判别式为零（对抛物线）和距离等于半径（对圆）建立含参数的方程组，解出公切线的斜率和截距，从而确定公切线的条数。

## 详细解答

设公切线方程为 \(y = kx + m\)。

**与抛物线相切：**  
将 \(y = kx + m\) 代入 \(x^2 = 2y\)，得  
\[
x^2 = 2(kx + m) \quad\Longrightarrow\quad x^2 - 2kx - 2m = 0.
\]  
由相切可知判别式 \(\Delta = 0\)，即  
\[
4k^2 + 8m = 0 \quad\Longrightarrow\quad m = -\frac{k^2}{2}. \tag{1}
\]

**与圆相切：**  
圆的标准方程为 \(x^2 + (y+1)^2 = 1\)，圆心为 \((0, -1)\)，半径 \(r = 1\)。  
圆心到直线 \(kx - y + m = 0\) 的距离应等于半径：  
\[
\frac{|k \cdot 0 - (-1) + m|}{\sqrt{k^2 + 1}} = 1 \quad\Longrightarrow\quad |m + 1| = \sqrt{k^2 + 1}.
\]  
两边平方得  
\[
(m + 1)^2 = k^2 + 1. \tag{2}
\]

将 (1) 式代入 (2) 式：  
\[
\left(-\frac{k^2}{2} + 1\right)^2 = k^2 + 1.
\]  
整理：  
\[
1 - k^2 + \frac{k^4}{4} = k^2 + 1,
\]  
\[
\frac{k^4}{4} - 2k^2 = 0,
\]  
\[
k^2\left(\frac{k^2}{4} - 2\right) = 0.
\]  
解得 \(k^2 = 0\) 或 \(k^2 = 8\).

- 当 \(k = 0\) 时，代入 (1) 得 \(m = 0\)，公切线为 \(y = 0\)。  
- 当 \(k^2 = 8\) 时，\(m = -\dfrac{8}{2} = -4\)，对应两条直线：  
  \(y = 2\sqrt{2}\,x - 4\) 和 \(y = -2\sqrt{2}\,x - 4\).

经验证，这三条直线均满足与圆相切的条件（\(|m+1| = \sqrt{k^2+1}\) 成立），且与抛物线相切。

因此，满足条件的直线共有 \(3\) 条。

## 答案
D. 3

## 总结
本题综合了直线与抛物线相切（代数上用判别式为零）以及直线与圆相切（几何上用距离等于半径）的条件，通过联立方程求出所有可能的公切线。由于抛物线顶点恰好在圆上，且两者在该处有同一条切线，再结合对称性可解出另外两条公切线。遇到此类问题时，建议先结合图形直观判断大致位置，再通过方程组严格求解。
