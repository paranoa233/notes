# 10-3-8 我的解答

对应题目：[[01_题目/10_圆锥曲线/10-3_抛物线的标准方程及简单的几何性质/10-3-8.md|10-3-8]]

## 完整题目

8.(2022 福建福州质检)已知抛物线 \( C : {y}^{2} = {4x} \) 的焦点为 \( F \) ,过点 \( F \) 的动直线 \( l \) 交 \( C \) 于点 \( A, B \) 两点,过点 \( A, B \) 分别作 \( C \) 的切线 \( {l}_{1},{l}_{2},{l}_{1} \) 与 \( {l}_{2} \) 交于点 \( P \) . 经探究可知点 \( P \) 必在一条定直线上,其方程为___；记 \( {l}_{1},{l}_{2} \) 与 \( y \) 轴的交点分别为

\( M, N \) ,若 \( l \) 的倾斜角为 \( {30}^{ \circ  } \) ,则四边形 \( {PMFN} \) 的面积为___.

## 题目勘误
无明显勘误。

## 思路分析
本题主要考查抛物线的标准方程与几何性质，具体涉及焦点弦、切线方程、动直线过定点以及四边形面积的计算。  
第一空：设出 \(A,B\) 坐标，写出切线 \(l_1,l_2\) 方程，联立求出交点 \(P\) 的坐标，再利用焦点弦的韦达定理消去参数，发现 \(P\) 的横坐标为定值，从而得到定直线方程。  
第二空：由倾斜角确定直线 \(l\) 的方程，结合韦达定理求出 \(y_1+y_2,\;y_1y_2\) 以及 \(|y_1-y_2|\)，然后求出 \(M,N\) 的坐标，最后用坐标法（或分割法）计算四边形 \(PMFN\) 的面积。

## 详细解答

### 第一空
抛物线 \(C:y^2=4x\) 中，\(2p=4\)，故 \(p=2\)，焦点 \(F(1,0)\)。

设动直线 \(l\) 与 \(C\) 交于 \(A(x_1,y_1),\;B(x_2,y_2)\)。
对 \(y^2=4x\) 求导得 \(2y\cdot y'=4\)，故 \(y'=\dfrac{2}{y}\)。
根据抛物线的切线方程公式（或求导结果），在 \(A\) 处的切线 \(l_1\) 为
\[
y_1 y = 2(x + x_1),
\]
同理 \(l_2\) 为
\[
y_2 y = 2(x + x_2).
\]
联立求交点 \(P\)：
\[
\begin{cases}
y_1 y = 2x + 2x_1,\\[2pt]
y_2 y = 2x + 2x_2.
\end{cases}
\]
两式相减得
\[
(y_1 - y_2) y = 2(x_1 - x_2).
\]
又 \(A,B\) 在抛物线上，故 \(x_1=\dfrac{y_1^2}{4},\;x_2=\dfrac{y_2^2}{4}\)，从而
\[
2(x_1 - x_2) = \frac{y_1^2 - y_2^2}{2} = \frac{(y_1 - y_2)(y_1 + y_2)}{2}.
\]
当 \(y_1\ne y_2\) 时，代入得
\[
y = \frac{y_1 + y_2}{2}.
\]
将其代回 \(l_1\) 方程：
\[
y_1\cdot\frac{y_1+y_2}{2} = 2x + \frac{y_1^2}{2} \;\Longrightarrow\; 2x = \frac{y_1 y_2}{2} \;\Longrightarrow\; x = \frac{y_1 y_2}{4}.
\]
设过焦点 \(F(1,0)\) 的直线 \(l\) 的方程为 \(x = my + 1\)（\(m\) 为倾斜角余切，可避免斜率不存在情形），代入抛物线得
\[
y^2 = 4(my + 1) \;\Longrightarrow\; y^2 - 4my - 4 = 0.
\]
由韦达定理：
\[
y_1 + y_2 = 4m,\quad y_1 y_2 = -4.
\]
于是交点 \(P\) 的坐标满足
\[
x_P = \frac{-4}{4} = -1,\quad y_P = \frac{4m}{2} = 2m.
\]
可见 \(P\) 的横坐标恒为 \(-1\)，与 \(m\) 无关，故点 \(P\) 必在定直线 \(x = -1\) 上。

**第一空答案：** \(x = -1\)（或 \(x + 1 = 0\)）。

### 第二空
当直线 \(l\) 的倾斜角为 \(30^\circ\) 时，斜率 \(k = \tan 30^\circ = \dfrac{1}{\sqrt{3}}\)。
此时直线方程可写为 \(y = \dfrac{1}{\sqrt{3}}(x-1)\)，即 \(x = \sqrt{3}\,y + 1\)，对应参数 \(m = \sqrt{3}\)。
由韦达定理得
\[
y_1 + y_2 = 4\sqrt{3},\quad y_1 y_2 = -4.
\]
计算
\[
(y_1 - y_2)^2 = (y_1 + y_2)^2 - 4y_1y_2 = (4\sqrt{3})^2 - 4 \times (-4) = 48 + 16 = 64,
\]
所以 \(|y_1 - y_2| = 8\)。

切线 \(l_1\) 与 \(y\) 轴的交点 \(M\)：令 \(x=0\)，由 \(y_1 y = 2x + \dfrac{y_1^2}{2}\) 得 \(y_M = \dfrac{y_1}{2}\)，即 \(M\!\left(0,\dfrac{y_1}{2}\right)\)。
同理 \(N\!\left(0,\dfrac{y_2}{2}\right)\)。故 \(|MN| = \left|\dfrac{y_1}{2} - \dfrac{y_2}{2}\right| = \dfrac{|y_1 - y_2|}{2} = 4\)。

由第一空，\(P\) 点坐标为 \((-1,\,2m) = (-1,\,2\sqrt{3})\)，又 \(F(1,0)\)。
四边形 \(PMFN\) 的顶点按顺序为 \(P(-1,2\sqrt{3}),\;M\!\left(0,\dfrac{y_1}{2}\right),\;F(1,0),\;N\!\left(0,\dfrac{y_2}{2}\right)\)。
用坐标法求面积：
\[
\begin{aligned}
S_{PMFN} &= \frac12 \Big| x_P y_M + x_M y_F + x_F y_N + x_N y_P - \big(y_P x_M + y_M x_F + y_F x_N + y_N x_P\big) \Big| \\[4pt]
&= \frac12 \Big| (-1)\cdot\frac{y_1}{2} + 0\cdot 0 + 1\cdot\frac{y_2}{2} + 0\cdot 2\sqrt{3} \\
&\qquad - \big( 2\sqrt{3}\cdot 0 + \frac{y_1}{2}\cdot 1 + 0\cdot 0 + \frac{y_2}{2}\cdot (-1) \big) \Big| \\[4pt]
&= \frac12 \Big| -\frac{y_1}{2} + \frac{y_2}{2} - \Big(\frac{y_1}{2} - \frac{y_2}{2}\Big) \Big| \\[4pt]
&= \frac12 \Big| -y_1 + y_2 \Big| = \frac12 |y_1 - y_2| = \frac12 \times 8 = 4.
\end{aligned}
\]

**第二空答案：** \(4\)。

## 答案
第一空：\(x = -1\)；第二空：\(4\)。

## 总结
1. 抛物线 \(y^2=2px\) 上一点 \((x_0,y_0)\) 处的切线方程为 \(y_0y = p(x+x_0)\)，两条切线的交点横坐标可由韦达定理迅速确定。  
2. 焦点弦问题常常设直线方程为 \(x = my + \dfrac{p}{2}\)，联立后使用韦达定理可简化运算。  
3. 四边形面积可用“坐标法”直接代入顶点顺序计算，也可分割为两个三角形处理，关键在于准确表达各点坐标。
