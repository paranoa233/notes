# 10-1-14 我的解答

对应题目：[[01_题目/10_圆锥曲线/10-1_椭圆的标准方程及简单的几何性质/10-1-14.md|10-1-14]]

## 完整题目

4.(2021全国乙卷理，11)设 \( B \) 是椭圆 \( C : \frac{{x}^{2}}{{a}^{2}} + \; \frac{{y}^{2}}{{b}^{2}} = 1\left( {a > b > 0}\right) \) 的上顶点,若 \( C \) 上的任意一点 \( P \) 都满足 \( \left| {PB}\right|  \leq  {2b} \) ,则 \( C \) 的离心率的取值范围是 ( )

A. \( \left\lbrack  {\frac{\sqrt{2}}{2},1}\right) \) B. \( \left\lbrack  {\frac{1}{2},1}\right) \)

C. \( \left( {0,\frac{\sqrt{2}}{2}}\right\rbrack \) D. \( \left( {0,\frac{1}{2}}\right\rbrack \)

## 题目勘误
无明显勘误。

## 思路分析
上顶点 \(B(0,b)\)，设 \(P(x,y)\) 为椭圆上任意一点，由椭圆方程消去 \(x\) 后可将 \(|PB|^2\) 表示成关于 \(y\) 的二次函数。再利用二次函数在闭区间 \([-b,b]\) 上的最大值满足 \(|PB|^2 \leq 4b^2\)，分类讨论对称轴的位置，从而解出离心率的取值范围。

## 详细解答
椭圆 \(C:\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1\;(a>b>0)\) 的上顶点 \(B(0,b)\)。  
设 \(P(x,y)\) 为 \(C\) 上任意一点，则  
\[
|PB|^2 = x^2 + (y-b)^2.
\]
由椭圆方程得 \(x^2 = a^2\!\left(1-\dfrac{y^2}{b^2}\right)\)，代入上式：
\[
|PB|^2 = a^2\!\left(1-\frac{y^2}{b^2}\right) + y^2 - 2by + b^2 
= \left(1-\frac{a^2}{b^2}\right)y^2 - 2by + a^2 + b^2.
\]
记 \(c^2 = a^2 - b^2\)，则 \(1-\dfrac{a^2}{b^2} = -\dfrac{c^2}{b^2}\)，于是
\[
f(y) = |PB|^2 = -\frac{c^2}{b^2}y^2 - 2by + c^2 + 2b^2, \quad y\in[-b,b].
\]
因为 \(-\dfrac{c^2}{b^2} < 0\)，\(f(y)\) 是开口向下的二次函数，其对称轴为
\[
y_0 = -\frac{-2b}{2\!\left(-\dfrac{c^2}{b^2}\right)} = -\frac{b^3}{c^2} < 0.
\]
题目要求 \(|PB| \leq 2b\) 恒成立，即 \(f(y) \leq 4b^2\) 恒成立。由于 \(f(y)\) 在 \([-b,b]\) 上的最大值必在对称轴或端点处取得，以下按对称轴与区间的位置分类：

**情况 1：** \(y_0 \geq -b\)，即 \(-\dfrac{b^3}{c^2} \geq -b \;\Rightarrow\; b^2 \leq c^2 \;\Rightarrow\; b \leq c\).  
此时最大值在 \(y_0\) 处取得：
\[
f_{\max} = f(y_0) = -\frac{c^2}{b^2}\!\left(-\frac{b^3}{c^2}\right)^2 - 2b\!\left(-\frac{b^3}{c^2}\right) + c^2 + 2b^2 = \frac{b^4}{c^2} + c^2 + 2b^2.
\]
要求 \(f_{\max} \leq 4b^2\)，即
\[
\frac{b^4}{c^2} + c^2 - 2b^2 \leq 0 \;\Rightarrow\; \left(\frac{b^2}{c} - c\right)^2 \leq 0.
\]
因为平方非负，只能 \(\dfrac{b^2}{c} - c = 0\)，得 \(b^2 = c^2\)，此时 \(b=c\)，离心率 \(e = \dfrac{c}{a} = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}\)。若 \(b<c\) 则不满足条件。

**情况 2：** \(y_0 < -b\)，即 \(-\dfrac{b^3}{c^2} < -b \;\Rightarrow\; b^2 > c^2 \;\Rightarrow\; b > c\).  
此时 \(y_0 \notin [-b,b]\)，且抛物线开口向下，在区间 \([-b,b]\) 上 \(f(y)\) 单调递减（因区间在对称轴右侧），最大值在左端点 \(y=-b\) 处取得：
\[
f_{\max} = f(-b) = -\frac{c^2}{b^2}\cdot b^2 - 2b(-b) + c^2 + 2b^2 = 4b^2.
\]
此时 \(f(y) \leq 4b^2\) 恒成立（当 \(P\) 为下顶点 \((0,-b)\) 时取等）。条件 \(b>c\) 等价于 \(a^2-c^2 > c^2 \;\Rightarrow\; a^2 > 2c^2 \;\Rightarrow\; e^2 = \dfrac{c^2}{a^2} < \dfrac{1}{2}\)，又 \(e>0\)，故 \(0 < e < \dfrac{\sqrt{2}}{2}\).

综合两种情况，椭圆离心率的取值范围是 \((0, \frac{\sqrt{2}}{2}]\)。

## 答案
C

## 总结
本题以椭圆上顶点为背景，将距离最值问题转化为二次函数在闭区间上的最大值问题，借助对称轴与区间的位置关系进行分类讨论，得到离心率的临界值，最终确定取值范围。解题时需注意二次项系数为负时最值点的判断，以及等号成立的条件。
