# 10-1-8 我的解答

对应题目：[[01_题目/10_圆锥曲线/10-1_椭圆的标准方程及简单的几何性质/10-1-8.md|10-1-8]]

## 完整题目

8.(2022 江苏苏锡常镇调研(一))已知椭圆 \( C \) : \( \frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} = 1\left( {a > b > 0}\right) \) 的离心率为 \( \frac{\sqrt{2}}{2} \) ,且椭圆 \( C \) 的右焦点 \( F \) 到右准线的距离为 \( \sqrt{3}.A \) 是第一象限内的定点, \( M, N \) 是椭圆 \( C \) 上两个不同的动点 (均异于点 \( A \) )，且直线 \( {AM} \) ， \( {AN} \) 的倾斜角互补.

(1)求椭圆 \( C \) 的标准方程；

(2)若直线 \( {MN} \) 的斜率 \( k = 1 \) ，求点 \( A \) 的坐标.

## 题目勘误
无明显勘误。但需注意：第(2)问中“\(A\) 是第一象限内的定点”未明确 \(A\) 是否在椭圆 \(C\) 上。结合常规题型及“\(M,N\) 是椭圆 \(C\) 上两个不同的动点（均异于点 \(A\)）”的表述，通常隐含 \(A\) 亦在椭圆上，否则条件不足以确定点 \(A\) 的坐标。以下解答默认 \(A\) 在椭圆 \(C\) 上。若原题确无此条件，建议补充“\(A\) 在椭圆 \(C\) 上”。

## 思路分析
(1) 由离心率得到 \(a\) 与 \(c\) 的关系；利用右焦点到右准线的距离及椭圆中 \(a,b,c\) 的关系求出 \(a,b\)，即得标准方程。
(2) 设 \(A(x_0,y_0)\) 在椭圆上。由直线 \(AM,AN\) 的倾斜角互补，得斜率互为相反数。分别设出直线方程并与椭圆联立，利用平移坐标系及韦达定理，推导直线 \(MN\) 的斜率（发现为定值且与所设斜率无关）。令该斜率等于 \(1\)，结合 \(A\) 在椭圆上解出坐标。

## 详细解答
**(1) 求椭圆 \(C\) 的标准方程**

设椭圆半焦距为 \(c\)，离心率 \(e = \frac{c}{a} = \frac{\sqrt{2}}{2}\)，则 \(c = \frac{\sqrt{2}}{2}a\)。  
右焦点 \(F(c,0)\)，右准线方程为 \(x = \frac{a^2}{c}\)。由题意，两点间距离为
\[
\frac{a^2}{c} - c = \sqrt{3}.
\]
又椭圆中 \(b^2 = a^2 - c^2\)，故
\[
\frac{a^2}{c} - c = \frac{a^2 - c^2}{c} = \frac{b^2}{c} = \sqrt{3}.
\]
由 \(c = \frac{\sqrt{2}}{2}a\)，得 \(b^2 = a^2 - \left(\frac{\sqrt{2}}{2}a\right)^2 = a^2 - \frac{1}{2}a^2 = \frac{1}{2}a^2\)。代入上式：
\[
\frac{\frac{1}{2}a^2}{\frac{\sqrt{2}}{2}a} = \frac{a}{\sqrt{2}} = \sqrt{3} \implies a = \sqrt{6}.
\]
于是
\[
c = \frac{\sqrt{2}}{2} \times \sqrt{6} = \sqrt{3}, \quad b^2 = a^2 - c^2 = 6 - 3 = 3.
\]
因此椭圆 \(C\) 的标准方程为
\[
\frac{x^2}{6} + \frac{y^2}{3} = 1.
\]

**(2) 若直线 \(MN\) 的斜率 \(k=1\)，求点 \(A\) 的坐标**

依题意，点 \(A\) 在第一象限且在椭圆上，设 \(A(x_0,y_0)\)，则
\[
\frac{x_0^2}{6} + \frac{y_0^2}{3} = 1, \quad x_0 > 0,\; y_0 > 0. \tag{1}
\]

由直线 \(AM,AN\) 的倾斜角互补，可知两直线的斜率互为相反数。  
设直线 \(AM\) 的斜率为 \(t\)（\(t \neq 0\)），则直线 \(AN\) 的斜率为 \(-t\)。

平移坐标系，令新的原点为 \(A\)：设 \(X = x - x_0,\; Y = y - y_0\)。  
椭圆方程在新坐标系下变为
\[
\frac{(X + x_0)^2}{6} + \frac{(Y + y_0)^2}{3} = 1.
\]
展开并利用 \(A\) 满足 \(\frac{x_0^2}{6} + \frac{y_0^2}{3} = 1\)，化简得
\[
\frac{X^2}{6} + \frac{x_0 X}{3} + \frac{Y^2}{3} + \frac{2y_0 Y}{3} = 0.
\]
两边乘 \(6\)：
\[
X^2 + 2x_0 X + 2Y^2 + 4y_0 Y = 0. \tag{2}
\]
（以上步骤等价于将椭圆方程改造为过 \(A\) 的二次齐次式，便于处理斜率问题。）

直线 \(AM\) 在新系下为 \(Y = tX\)，代入 (2) 式求交点（除 \(A\) 外）：
\[
X^2 + 2x_0 X + 2t^2 X^2 + 4y_0 t X = 0.
\]
整理得
\[
(1 + 2t^2)X^2 + 2(x_0 + 2y_0 t)X = 0.
\]
由于 \(X = 0\) 对应点 \(A\)，另一交点 \(M\) 对应的横坐标 \(X_M\) 为
\[
X_M = -\frac{2(x_0 + 2y_0 t)}{1 + 2t^2}.
\]
同理，直线 \(AN\) 为 \(Y = -tX\)，代入 (2) 得
\[
(1 + 2t^2)X^2 + 2(x_0 - 2y_0 t)X = 0,
\]
故另一交点 \(N\) 的横坐标
\[
X_N = -\frac{2(x_0 - 2y_0 t)}{1 + 2t^2}.
\]

于是点 \(M,N\) 的坐标可写为（在新系下）：
\[
M(X_M,\; tX_M), \quad N(X_N,\; -tX_N).
\]
由于平移不改变直线的斜率，直线 \(MN\) 的斜率为
\[
k_{MN} = \frac{tX_M - (-tX_N)}{X_M - X_N} = t \cdot \frac{X_M + X_N}{X_M - X_N}.
\]
计算：
\[
X_M + X_N = -\frac{2(x_0 + 2y_0 t) + 2(x_0 - 2y_0 t)}{1 + 2t^2} = -\frac{4x_0}{1 + 2t^2},
\]
\[
X_M - X_N = -\frac{2(x_0 + 2y_0 t) - 2(x_0 - 2y_0 t)}{1 + 2t^2} = -\frac{8y_0 t}{1 + 2t^2}.
\]
代入斜率表达式：
\[
k_{MN} = t \cdot \frac{-\frac{4x_0}{1 + 2t^2}}{-\frac{8y_0 t}{1 + 2t^2}} = t \cdot \frac{4x_0}{8y_0 t} = \frac{x_0}{2y_0}.
\]
可见直线 \(MN\) 的斜率为定值 \(\frac{x_0}{2y_0}\)，与 \(t\) 无关。

由题设 \(k_{MN} = 1\)，故
\[
\frac{x_0}{2y_0} = 1 \implies x_0 = 2y_0.
\]
代入 (1) 式：
\[
\frac{(2y_0)^2}{6} + \frac{y_0^2}{3} = 1 \implies \frac{4y_0^2}{6} + \frac{y_0^2}{3} = \frac{2}{3}y_0^2 + \frac{1}{3}y_0^2 = y_0^2 = 1.
\]
解得 \(y_0 = 1\)（负值舍去），则 \(x_0 = 2\)。

因此点 \(A\) 的坐标为 \((2, 1)\)。

## 答案
(1) 椭圆 \(C\) 的标准方程为 \(\displaystyle \frac{x^2}{6} + \frac{y^2}{3} = 1\)；  
(2) 点 \(A\) 的坐标为 \(\displaystyle (2, 1)\)。

## 总结
本题综合考查椭圆的几何性质与直线与椭圆的位置关系。第(1)问是基础计算；第(2)问核心在于利用倾斜角互补等价于斜率互为相反数，借助平移坐标系与韦达定理将 \(MN\) 的斜率表达为仅与 \(A\) 坐标有关的定值，从而建立方程求解。此类“定点 \(A\) 出发的两条弦斜率之和为定值（本题为 \(0\)）”的问题，所得弦的斜率往往为定值，可直接使用结论简化运算。
