# 10-9-14 我的解答

对应题目：[[01_题目/10_圆锥曲线/10-9_圆锥曲线中的定点和定值问题/10-9-14.md|10-9-14]]

## 完整题目

3. (2022 全国乙卷理，20)已知椭圆 \( E \) 的中心为坐标原点,对称轴为 \( x \) 轴、 \( y \) 轴,且过 \( A\left( {0, - 2}\right) \) , \( B\left( {\frac{3}{2}, - 1}\right) \) 两点.

(1)求 \( E \) 的方程；

(2)设过点 \( P\left( {1, - 2}\right) \) 的直线交 \( E \) 于 \( M \) ， \( N \) 两点，过 \( M \) 且平行于 \( x \) 轴的直线与线段 \( {AB} \) 交于点 \( T \) ， 点 \( H \) 满足 \( \overrightarrow{MT} = \overrightarrow{TH} \) ,求证:直线 \( {HN} \) 过定点.

## 题目勘误
无明显勘误。

## 思路分析
第(1)问：已知椭圆中心在原点，对称轴为坐标轴，且过两点 \(A(0,-2)\) 和 \(B(\frac{3}{2},-1)\)，可设椭圆方程为 \(mx^2+ny^2=1\) (\(m>0, n>0, m\neq n\))，代入两点坐标求出参数即可。

第(2)问：要证明直线 \(HN\) 过定点，这是典型的圆锥曲线中的定点问题。设出直线 \(MN\) 的方程，与椭圆联立得到韦达定理关系。由题意，\(T\) 是过 \(M\) 且平行于 \(x\) 轴的直线与线段 \(AB\) 的交点，可写出点 \(T\) 的坐标；由 \(\overrightarrow{MT}=\overrightarrow{TH}\) 可得 \(T\) 是 \(MH\) 的中点，从而表示出 \(H\) 的坐标。最后写出直线 \(HN\) 的方程，利用韦达定理化简，找到直线所过的定点。注意讨论直线 \(MN\) 斜率不存在的情况。

## 详细解答

**(1) 求 \(E\) 的方程**

设椭圆 \(E\) 的方程为 \(mx^2 + ny^2 = 1\) (\(m>0, n>0, m\neq n\))。代入 \(A(0,-2)\) 和 \(B(\frac{3}{2},-1)\)：

点 \(A\)：\(m\cdot 0^2 + n\cdot (-2)^2 = 1 \implies 4n = 1 \implies n = \frac14\)。
点 \(B\)：\(m\cdot \left(\frac{3}{2}\right)^2 + \frac14\cdot (-1)^2 = 1 \implies \frac94 m + \frac14 = 1 \implies \frac94 m = \frac34 \implies m = \frac13\)。

故椭圆 \(E\) 的方程为 \(\frac13 x^2 + \frac14 y^2 = 1\)，即 \(\frac{x^2}{3} + \frac{y^2}{4} = 1\)。
注意：\(A(0,-2)\) 在 \(y\) 轴上，\(4>3\)，焦点在 \(y\) 轴上，方程为标准形式 \(\frac{y^2}{a^2} + \frac{x^2}{b^2} = 1\) 即 \(\frac{y^2}{4}+\frac{x^2}{3}=1\)。与上述结果一致。

所以 \(E\) 的方程为 \(\frac{x^2}{3} + \frac{y^2}{4} = 1\)。

**(2) 证明直线 \(HN\) 过定点**

由 \(A(0,-2)\)，\(B(\frac{3}{2},-1)\)，得直线 \(AB\) 的方程为：
斜率 \(k_{AB} = \frac{-1 - (-2)}{\frac{3}{2} - 0} = \frac{1}{\frac{3}{2}} = \frac23\)，
方程为 \(y + 2 = \frac23 (x - 0) \implies y = \frac23 x - 2\)，即 \(3y = 2x - 6\)。

① 当直线 \(MN\) 的斜率不存在时，其方程为 \(x = 1\)。
代入椭圆 \(\frac{1}{3} + \frac{y^2}{4} = 1 \implies \frac{y^2}{4} = \frac23 \implies y^2 = \frac83\)，即 \(y = \pm \frac{2\sqrt{6}}{3}\)。
不妨设 \(M(1, \frac{2\sqrt{6}}{3})\)，\(N(1, -\frac{2\sqrt{6}}{3})\)。
过 \(M\) 且平行于 \(x\) 轴的直线为 \(y = \frac{2\sqrt{6}}{3}\)，与线段 \(AB: y = \frac23 x - 2\) 联立，得：
\(\frac{2\sqrt{6}}{3} = \frac23 x - 2 \implies \frac23 x = \frac{2\sqrt{6}}{3} + 2 = \frac{2\sqrt{6}+6}{3} \implies x = \sqrt{6} + 3\)。
因为 \(A(0,-2), B(\frac32, -1)\)，所以线段 \(AB\) 上点的横坐标范围为 \([0, \frac32]\)，而 \(\sqrt{6}+3 > 3 > \frac32\)，因此点 \(T\) 不在线段 \(AB\) 上，这种情况不符合题意。
故直线 \(MN\) 斜率必然存在。

② 当直线 \(MN\) 斜率存在时，设直线 \(MN\) 的方程为 \(y + 2 = k(x - 1)\)，即 \(y = kx - k - 2\)。
设 \(M(x_1, y_1)\)，\(N(x_2, y_2)\)。

联立：
\[
\begin{cases}
y = kx - k - 2 \\
\frac{x^2}{3} + \frac{y^2}{4} = 1
\end{cases}
\]
消去 \(y\)：
\[
4x^2 + 3(kx - k - 2)^2 = 12
\]
展开：
\[
4x^2 + 3(k^2 x^2 - 2k(k+2)x + (k+2)^2) = 12
\]
\[
(4 + 3k^2)x^2 - 6k(k+2)x + 3(k+2)^2 - 12 = 0
\]
注意：\(3(k+2)^2 - 12 = 3[(k+2)^2 - 4] = 3(k+2-2)(k+2+2) = 3k(k+4)\)。
所以方程化为：
\[
(3k^2 + 4)x^2 - 6k(k+2)x + 3k(k+4) = 0
\]
由韦达定理：
\[
x_1 + x_2 = \frac{6k(k+2)}{3k^2 + 4}, \quad x_1 x_2 = \frac{3k(k+4)}{3k^2 + 4}
\]

过 \(M\) 且平行于 \(x\) 轴的直线为 \(y = y_1\)。与线段 \(AB: y = \frac23 x - 2\) 的交点 \(T(x_T, y_1)\) 满足：
\[
y_1 = \frac23 x_T - 2 \implies x_T = \frac{3}{2}(y_1 + 2).
\]
注意 \(y_1 = kx_1 - k - 2\)，所以 \(y_1 + 2 = k(x_1 - 1)\)，
于是 \(x_T = \frac{3}{2} k (x_1 - 1)\)，

点 \(T\) 坐标为 \(\left( \frac{3}{2} k (x_1 - 1),\; y_1 \right)\)。

点 \(H\) 满足 \(\overrightarrow{MT} = \overrightarrow{TH}\)，即 \(T\) 是线段 \(MH\) 的中点。设 \(H(x_H, y_H)\)，则：
\[
x_T = \frac{x_1 + x_H}{2} \implies x_H = 2x_T - x_1,
\]
\[
y_T = \frac{y_1 + y_H}{2} = y_1 \implies y_H = y_1.
\]
因此 \(H\) 坐标为 \((3k(x_1 - 1) - x_1,\; y_1)\)，即 \(((3k-1)x_1 - 3k,\; y_1)\)。

现在求直线 \(HN\) 的方程。\(H((3k-1)x_1 - 3k, y_1)\)，\(N(x_2, y_2)\)，
斜率 \(k_{HN} = \frac{y_2 - y_1}{x_2 - [(3k-1)x_1 - 3k]} = \frac{y_2 - y_1}{x_2 - (3k-1)x_1 + 3k}\)。

直线 \(HN\) 的方程为 \(y - y_2 = \frac{y_2 - y_1}{x_2 - (3k-1)x_1 + 3k} (x - x_2)\)。

为了找到定点，我们猜测定点在 \(y\) 轴上（由对称性或常见题型），不妨尝试求 \(x=0\) 时的纵坐标是否为定值。令 \(x=0\)，找到一个可能的定点 \((0, -2)\) 或其他。以下进行严格推导：

由于 \(y_1 = kx_1 - k - 2\)，\(y_2 = kx_2 - k - 2\)，
所以 \(y_2 - y_1 = k(x_2 - x_1)\)。

直线 \(HN\) 方程可写为：
\[
y - y_2 = \frac{k(x_2 - x_1)}{x_2 - 3k x_1 + x_1 + 3k} (x - x_2)
\]
即 
\[
y - y_2 = \frac{k(x_2 - x_1)}{(x_2 + x_1) + 3k - 3k x_1} (x - x_2)? \quad \text{注意分母：} x_2 - (3k-1)x_1 + 3k = x_2 + x_1 - 3k x_1 + 3k.
\]

我们想证明该直线恒过定点 \((a, b)\)，即对任意满足条件的 \(x_1, x_2, k\) 均成立。通常利用韦达定理将表达式转化为与 \(x_1, x_2\) 的关系，并消去。或者直接设定点为 \((0, -2)\) 进行验证。

设直线 \(HN\) 过定点 \((u, v)\)，代入：
\[
v - y_2 = \frac{k(x_2 - x_1)}{x_2 - (3k-1)x_1 + 3k} (u - x_2)
\]
整理为：
\[
(v - y_2) [x_2 - (3k-1)x_1 + 3k] = k(x_2 - x_1)(u - x_2)
\]
将 \(y_2 = kx_2 - k - 2\) 代入 \(v - y_2 = v + k + 2 - kx_2\)，
所以左 = \((v + k + 2 - kx_2)(x_2 - (3k-1)x_1 + 3k)\)，
右 = \(k(x_2 - x_1)(u - x_2)\)。

展开左边：
\[
(v + k + 2 - kx_2)(x_2 - 3k x_1 + x_1 + 3k)
\]
直接全部展开可能较繁，采用特殊值法求定点，再证明。

我们可以取两组特殊的直线 \(MN\)，求出 \(HN\) 的交点，即为定点，然后证明一般情况。

取 \(k=0\)，直线 \(MN: y = -2\)。代入椭圆：\(\frac{x^2}{3} + \frac{(-2)^2}{4} = 1 \implies \frac{x^2}{3} + 1 = 1 \implies x = 0\)。
此时直线与椭圆切于点 \((0, -2)\)，\(M=N\) 无法构成弦。不符合题意。

再取 \(k=1\)，直线 \(MN: y = x - 3\)。联立椭圆：
\[
4x^2 + 3(x-3)^2 = 12 \implies 4x^2 + 3(x^2 -6x+9) = 12 \implies 7x^2 -18x +15 = 0,
\]
判别式 \(\Delta = 324 - 420 < 0\)，无交点。不可取。

取 \(k=-1\)，直线 \(MN: y = -x - 1\)。联立：
\[
4x^2 + 3(-x-1)^2 = 12 \implies 4x^2 + 3(x^2+2x+1) = 12 \implies 7x^2 + 6x - 9 = 0,
\]
\(\Delta = 36 + 252 = 288 > 0\)，有交点。
\(x_1 + x_2 = -6/7\)，\(x_1 x_2 = -9/7\)。
不妨解出 \(x_1, x_2 = \frac{-6 \pm \sqrt{288}}{14} = \frac{-6 \pm 12\sqrt{2}}{14} = \frac{-3 \pm 6\sqrt{2}}{7}\)。
取 \(M\) 对应 \(x_1 = \frac{-3 + 6\sqrt{2}}{7}\)，\(y_1 = -x_1 - 1 = \frac{3 - 6\sqrt{2}}{7} - 1 = \frac{-4 - 6\sqrt{2}}{7}\)。
则 \(T\): \(x_T = \frac{3}{2} k (x_1 - 1) = \frac{3}{2}(-1)(x_1 -1) = -\frac{3}{2}(x_1 -1)\)。
\(= -\frac{3}{2}\left( \frac{-3+6\sqrt{2}}{7} - 1 \right) = -\frac{3}{2}\cdot \frac{-10+6\sqrt{2}}{7} = \frac{15 - 9\sqrt{2}}{7}\)。
\(H\): \(x_H = 2x_T - x_1 = \frac{30 - 18\sqrt{2}}{7} - \frac{-3+6\sqrt{2}}{7} = \frac{33 - 24\sqrt{2}}{7}\)，\(y_H = y_1 = \frac{-4-6\sqrt{2}}{7}\)。
\(N(x_2, y_2)\)：\(x_2 = \frac{-3 - 6\sqrt{2}}{7}\)，\(y_2 = -x_2 -1 = \frac{3+6\sqrt{2}}{7} - 1 = \frac{-4+6\sqrt{2}}{7}\)。
求直线 \(HN\) 方程，找与另一条线交点。
再取 \(k=2\)，直线 \(MN: y = 2x - 4\)。联立：
\(4x^2 + 3(2x-4)^2 = 12 \implies 4x^2 + 3(4x^2 -16x+16)=12 \implies 16x^2 -48x + 48 = 12 \implies 16x^2 -48x + 36 =0 \implies 4x^2 -12x + 9=0\)，\(\Delta=144-144=0\)，切点 \((3/2, -1)\) 就是 \(B\) 点，不符合弦的要求。

其实可以选取另外的 \(k\)，但计算复杂。我们直接通过代数方法证明过定点 \((0,-2)\)。

回到一般情况的方程，我们试着将直线 \(HN\) 方程化为点斜式并寻找与 \(x_1, x_2\) 无关的点。观察到 \(A(0,-2)\) 是椭圆上顶点。联想到许多结论：过 \(P(1,-2)\) 的直线交椭圆于 \(M,N\)，过 \(M\) 作平行于 \(x\) 轴的直线交 \(AB\) 于 \(T\)，\(T\) 为 \(MH\) 中点，证明 \(HN\) 过定点。有可能定点就是 \(A(0,-2)\)。我们来验证 \(A(0,-2)\) 是否在 \(HN\) 上，即证 \(H,N,A\) 三点共线，也即 \(k_{HA} = k_{HN}\) 或向量共线。

\(H(((3k-1)x_1 - 3k), y_1)\)，\(A(0,-2)\)，\(N(x_2, y_2)\)。
向量 \(\overrightarrow{HA} = ( - (3k-1)x_1 + 3k , -2 - y_1 )\)，
\(\overrightarrow{HN} = (x_2 - (3k-1)x_1 + 3k , y_2 - y_1)\)。

要证三点共线，即 \(\overrightarrow{HA} \times \overrightarrow{HN} = 0\) (叉积为零)：
\[
(- (3k-1)x_1 + 3k)(y_2 - y_1) - (-2 - y_1)(x_2 - (3k-1)x_1 + 3k) = 0.
\]
代入 \(y_1 = kx_1 - k - 2\)，\(y_2 = kx_2 - k - 2\)。
那么 \(-2 - y_1 = -2 - (kx_1 - k - 2) = -kx_1 + k = k(1 - x_1)\)，
\(y_2 - y_1 = k(x_2 - x_1)\)。

设 \(A_1 = - (3k-1)x_1 + 3k = -3k x_1 + x_1 + 3k = (1-3k)x_1 + 3k\)。
左边：
\[
A_1 \cdot k(x_2 - x_1) - k(1-x_1) \cdot [x_2 + A_1] \quad (\text{注意 } x_2 - (3k-1)x_1 + 3k = x_2 + A_1).
\]
\[
= k \left[ A_1 (x_2 - x_1) - (1-x_1)(x_2 + A_1) \right]
\]
\[
= k \left[ A_1 x_2 - A_1 x_1 - x_2 - A_1 + x_1 x_2 + x_1 A_1 \right]
\]
\[
= k \left[ A_1 x_2 - A_1 x_1 - x_2 - A_1 + x_1 x_2 + x_1 A_1 \right]
\]
消去 \(-A_1 x_1 + x_1 A_1 = 0\)，得：
\[
= k \left[ A_1 x_2 - x_2 - A_1 + x_1 x_2 \right]
= k \left[ (A_1 - 1)x_2 + x_1 x_2 - A_1 \right].
\]
计算 \(A_1 - 1 = (1-3k)x_1 + 3k - 1 = (1-3k)(x_1 - 1)\)。
又 \(A_1 = (1-3k)x_1 + 3k = (1-3k)x_1 - (1-3k) + 1 = (1-3k)(x_1 - 1) + 1\)。
则
\[
(A_1 - 1)x_2 + x_1 x_2 - A_1 = (1-3k)(x_1 - 1)x_2 + x_1 x_2 - [(1-3k)(x_1 - 1) + 1]
\]
\[
= (1-3k)(x_1 - 1)(x_2 - 1) + x_1 x_2 - 1.
\]

因此左边等于 \(k [ (1-3k)(x_1 - 1)(x_2 - 1) + x_1 x_2 - 1 ]\)。

我们要证该式为0，即需证：
\[
(1-3k)(x_1 - 1)(x_2 - 1) + x_1 x_2 - 1 = 0.
\]
即
\[
(1-3k)(x_1 x_2 - (x_1 + x_2) + 1) + x_1 x_2 - 1 = 0.
\]
利用韦达定理：
\(x_1 + x_2 = \frac{6k(k+2)}{3k^2 + 4}\)，
\(x_1 x_2 = \frac{3k(k+4)}{3k^2 + 4}\)。

计算 \(x_1 x_2 - (x_1 + x_2) + 1 = \frac{3k(k+4) - 6k(k+2) + (3k^2 + 4)}{3k^2 + 4}\)
\[
= \frac{3k^2 + 12k - 6k^2 - 12k + 3k^2 + 4}{3k^2 + 4} = \frac{4}{3k^2 + 4}.
\]

所以 \((1-3k)(x_1 - 1)(x_2 - 1) = \frac{4(1-3k)}{3k^2 + 4}\)。

又 \(x_1 x_2 - 1 = \frac{3k(k+4)}{3k^2 + 4} - 1 = \frac{3k^2 + 12k - 3k^2 - 4}{3k^2 + 4} = \frac{12k - 4}{3k^2 + 4} = \frac{4(3k - 1)}{3k^2 + 4}\)。

两者相加：
\[
\frac{4(1-3k)}{3k^2 + 4} + \frac{4(3k - 1)}{3k^2 + 4} = 0.
\]
完美成立！因此左边整个为0，说明 \(\overrightarrow{HA}\) 与 \(\overrightarrow{HN}\) 共线，即 \(A, H, N\) 三点共线。

故直线 \(HN\) 恒过定点 \(A(0, -2)\)。

注意：以上推导中直线 \(MN\) 斜率存在且保证交点 \(T\) 在线段 \(AB\) 上。事实上通过几何条件，定点即为 \(A\)。检验斜率不存在的情况，确实无符合条件的 \(T\) 点，但一般情况仍可说明当满足题意时（即 \(T\) 在线段 \(AB\) 上），\(HN\) 必过定点 \((0,-2)\)。

## 答案
(1) 椭圆 \(E\) 的方程为 \(\frac{x^2}{3} + \frac{y^2}{4} = 1\)。

(2) 直线 \(HN\) 过定点 \((0, -2)\)，证明见解析。

## 总结
本题第(1)问属于基础待定系数法，注意椭圆标准方程的设法。第(2)问是典型的圆锥曲线中直线过定点问题，核心思路是设出直线方程，利用韦达定理表示相关点坐标，再通过三点共线或直线系的方法证明直线经过某定点。在化简过程中，需要巧妙利用因式分解和整体代换，体现了数学运算的核心素养。同时注意对直线斜率不存在的特殊情形进行讨论，确保证明的完整性。
