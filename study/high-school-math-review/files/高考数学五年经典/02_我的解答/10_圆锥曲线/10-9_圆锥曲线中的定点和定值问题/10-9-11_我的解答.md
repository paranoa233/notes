# 10-9-11 我的解答

对应题目：[[01_题目/10_圆锥曲线/10-9_圆锥曲线中的定点和定值问题/10-9-11.md|10-9-11]]

## 完整题目

2.(2020 新高考卷Ⅰ，22)已知椭圆 \( C : \frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} = \; 1\left( {a > b > 0}\right) \) 的离心率为 \( \frac{\sqrt{2}}{2} \) ,且过点 \( A\left( {2,1}\right) \) .

(1)求 \( C \) 的方程；

(2)点 \( M, N \) 在 \( C \) 上，且 \( {AM}\bot {AN},{AD}\bot {MN} \) ， 垂足为 \( D \) . 求证:存在定点 \( Q \) ，使得 \( \left| {DQ}\right| \) 为定值.

## 题目勘误
无明显勘误。
## 思路分析
第(1)问由离心率和椭圆上一点，直接列方程组解出 \(a^2\) 与 \(b^2\)。
第(2)问是典型的圆锥曲线定点定值问题。可从直线 \(MN\) 入手，设出直线方程并与椭圆联立，利用 \(AM \perp AN\) 得到斜率或坐标的约束条件，证明直线 \(MN\) 恒过定点 \(P\)。再由 \(AD \perp MN\) 知点 \(D\) 在以线段 \(AP\) 为直径的圆上，从而该圆的圆心 \(Q\) 即为所求定点，半径 \( \frac{1}{2}|AP| \) 为定值。
## 详细解答
**(1) 求 \(C\) 的方程：**
由离心率 \( e = \frac{c}{a} = \frac{\sqrt{2}}{2} \)，得 \( c^2 = \frac{1}{2}a^2 \)，又 \( a^2 = b^2 + c^2 \)，所以 \( b^2 = a^2 - \frac{1}{2}a^2 = \frac{1}{2}a^2 \)。
椭圆过点 \( A(2,1) \)，代入方程：
\[
\frac{4}{a^2} + \frac{1}{b^2} = 1
\]
将 \( b^2 = \frac{1}{2}a^2 \) 代入：
\[
\frac{4}{a^2} + \frac{2}{a^2} = 1 \implies \frac{6}{a^2} = 1 \implies a^2 = 6
\]
于是 \( b^2 = 3 \)。
故椭圆 \( C \) 的方程为：
\[
\frac{x^2}{6} + \frac{y^2}{3} = 1
\]

**(2) 证明存在定点 \(Q\) 使 \(|DQ|\) 为定值：**
① 当直线 \(MN\) 斜率不存在时，设 \(M(x_0, y_0), N(x_0, -y_0)\)。
由 \(AM \perp AN\)，得 \((x_0-2, y_0-1) \cdot (x_0-2, -y_0-1) = 0\)，即 \((x_0-2)^2 - (y_0^2 - 1) = 0\)。
又点 \(M\) 在椭圆上：\( \frac{x_0^2}{6} + \frac{y_0^2}{3} = 1 \implies y_0^2 = 3 - \frac{x_0^2}{2} \)。
联立解得 \(x_0 = 2\)（舍去，此时为点 \(A\)）或 \(x_0 = \frac{2}{3}\)。此时直线 \(MN\) 方程为 \(x = \frac{2}{3}\)，过定点 \(P_1(\frac{2}{3}, 0)\)？（代入检验：斜率不存在时 \(MN\) 为 \(x = \frac{2}{3}\)）。
需验证：当 \(x_0=\frac{2}{3}\)，得 \(y_0^2 = 3 - \frac{2}{9} = \frac{25}{9}\)，\(MN\) 过定点的横坐标为 \(\frac{2}{3}\)，但纵坐标不确定？实际上这只是一条竖直线的特殊情况，可作为直线系过定点的特例。

② 当直线 \(MN\) 斜率存在时，设 \(M(x_1, y_1), N(x_2, y_2)\)，直线 \(MN: y = kx + m\)。
与椭圆联立：
\[
\begin{cases}
y = kx + m \\
x^2 + 2y^2 = 6
\end{cases}
\]
消去 \(y\)：
\[
x^2 + 2(kx+m)^2 = 6 \implies (1+2k^2)x^2 + 4kmx + 2m^2 - 6 = 0
\]
\(\Delta > 0\)（后略），由韦达定理：
\[
x_1 + x_2 = -\frac{4km}{1+2k^2}, \quad x_1 x_2 = \frac{2m^2 - 6}{1+2k^2}
\]
由 \(AM \perp AN\)，即 \(\overrightarrow{AM} \cdot \overrightarrow{AN} = 0\)：
\[
(x_1-2)(x_2-2) + (y_1-1)(y_2-1) = 0
\]
将 \(y_i = kx_i + m\) 代入：
\[
(x_1-2)(x_2-2) + (kx_1 + m - 1)(kx_2 + m - 1) = 0
\]
展开得：
\[
(1+k^2)x_1x_2 + [k(m-1)-2](x_1+x_2) + (m-1)^2 + 4 = 0
\]
韦达代入并化简，得：
\[
(1+k^2)\frac{2m^2-6}{1+2k^2} + [k(m-1)-2]\left(-\frac{4km}{1+2k^2}\right) + (m-1)^2 + 4 = 0
\]
两边同乘 \(1+2k^2\)：
\[
(1+k^2)(2m^2-6) - 4km[k(m-1)-2] + [(m-1)^2+4](1+2k^2) = 0
\]
展开并整理：
\[
2m^2 - 6 + 2k^2m^2 - 6k^2 - 4k^2m(m-1) + 8km + (m^2 - 2m + 5)(1+2k^2) = 0
\]
继续合并同类项：
\[
2m^2 - 6 + 2k^2m^2 - 6k^2 - 4k^2m^2 + 4k^2m + 8km + m^2 - 2m + 5 + 2k^2m^2 - 4k^2m + 10k^2 = 0
\]
化简后所有含 \(k^2 m^2\) 的项：\(2-4+2=0\)，含 \(k^2 m\) 的项：\(4 - 4 = 0\)。剩余：
\[
(2m^2 + m^2) - 6 + 5 - 2m + (-6k^2 + 10k^2) + 8km = 0
\]
即：
\[
3m^2 - 2m - 1 + 4k^2 + 8km = 0
\]
化为：
\[
(3m^2 - 2m - 1) + 4k(2m + k) = 0
\]
因式分解 \(3m^2-2m-1 = (3m+1)(m-1)\)，检验是否为完全平方式？
尝试配方：写成关于 \(k\) 的二次式 \(4k^2 + 8mk + 3m^2 - 2m - 1 = 0\)。
判别式需为完全平方？这里不是求解，而是寻找定点，通常应能分解出 \(m\) 与 \(k\) 的线性关系。
将上述方程因式分解：
\(4k^2 + 8mk + 3m^2 - 2m - 1 = (2k + m + 1)(2k + 3m - 1) = 0\)？验证：
\((2k + m + 1)(2k + 3m - 1) = 4k^2 + 6km - 2k + 2km + 3m^2 - m + 2k + 3m - 1 = 4k^2 + 8km + 3m^2 + 2m - 1\)，不对，一次项系数为 \(2m\)，并非 \(-2m\)。
再试：\((2k + m - 1)(2k + 3m + 1) = 4k^2 + 6km + 2k + 2km + 3m^2 + m - 2k - 3m - 1 = 4k^2 + 8km + 3m^2 - 2m - 1\)。正确！
所以：
\[
(2k + m - 1)(2k + 3m + 1) = 0
\]
若 \(2k + m - 1 = 0\)，则 \(m = 1 - 2k\)，直线为 \(y = kx + 1 - 2k = k(x-2) + 1\)，恒过点 \(A(2,1)\)，不合题意（\(M,N\) 异于 \(A\)）。
故 \(2k + 3m + 1 = 0\)，即 \(m = -\frac{2}{3}k - \frac{1}{3}\)。
此时直线 \(MN\) 方程为：
\[
y = kx - \frac{2}{3}k - \frac{1}{3} = k\left(x - \frac{2}{3}\right) - \frac{1}{3}
\]
因此直线 \(MN\) 恒过定点 \(P\left(\frac{2}{3}, -\frac{1}{3}\right)\)。

③ 结合斜率不存在情况，当 \(k\) 不存在时 \(MN: x = \frac{2}{3}\)，也过点 \(P\left(\frac{2}{3}, -\frac{1}{3}\right)\)。综上，直线 \(MN\) 恒过定点 \(P\left(\frac{2}{3}, -\frac{1}{3}\right)\)。

④ 由 \(AD \perp MN\) 于 \(D\)，可知点 \(D\) 在以 \(AP\) 为直径的圆上。
设定点 \(Q\) 为线段 \(AP\) 的中点，则 \(Q\) 的坐标为 \(\left( \frac{2 + \frac{2}{3}}{2}, \frac{1 - \frac{1}{3}}{2} \right) = \left( \frac{4}{3}, \frac{1}{3} \right)\)。
\(|DQ|\) 为此圆的半径，即为 \(\frac{1}{2}|AP|\)，是定值。
计算 \(|AP| = \sqrt{\left(2 - \frac{2}{3}\right)^2 + \left(1 + \frac{1}{3}\right)^2} = \sqrt{\left(\frac{4}{3}\right)^2 + \left(\frac{4}{3}\right)^2} = \frac{4\sqrt{2}}{3}\)。
故 \(|DQ| = \frac{2\sqrt{2}}{3}\) 为定值。证毕。
## 答案
(1) \(C: \frac{x^2}{6} + \frac{y^2}{3} = 1\)；
(2) 存在定点 \(Q\left(\frac{4}{3}, \frac{1}{3}\right)\)，使得 \(|DQ| = \frac{2\sqrt{2}}{3}\) 为定值。
## 总结
本题第(2)问核心在于证明直线 \(MN\) 过定点。通过设直线方程、联立韦达、利用垂直条件构造关于参数 \(k,m\) 的二次方程，并因式分解得到定点坐标。进而利用垂直关系将动点轨迹转化为圆，定点即为圆心，半径即为定值。注意排除过 \(A\) 点的情形，同时对斜率不存在的情况分类讨论以确保完备性。
