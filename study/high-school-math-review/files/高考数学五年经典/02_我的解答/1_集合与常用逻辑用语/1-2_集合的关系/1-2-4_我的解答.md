# 1-2-4 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-2_集合的关系/1-2-4.md|1-2-4]]

## 完整题目

4.(2023 安徽、云南、吉林、黑龙江联考)设集合 \( A = \{ 2,3,{a}^{2} - {2a} - 3\} , B = \{ 0,3\} , C = \{ 2, a\} . \) 若 \( B \subseteq  A, A \cap  C = \{ 2\} \) ,则实数 \( a = \) ( )

A. -3 B. -1 C. 1 D. 3

## 题目勘误
无明显勘误。
## 思路分析
由 \( B \subseteq A \) 可推出 \( 0 \in A \)，从而得到关于 \( a \) 的方程。解出 \( a \) 后，再利用 \( A \cap C = \{2\} \) 及集合元素的互异性进行检验，即可确定 \( a \) 的值。
## 详细解答
因为 \( B = \{0, 3\} \subseteq A = \{2, 3, a^{2} - 2a - 3\} \)，  
所以 \( 0 \in A \)，即集合 \( A \) 中元素 \( a^{2} - 2a - 3 = 0 \)。

解方程：  
\[
a^{2} - 2a - 3 = 0 \;\Longrightarrow\; (a - 3)(a + 1) = 0,
\]  
解得 \( a = 3 \) 或 \( a = -1 \)。

又已知 \( A \cap C = \{2\} \)，其中 \( C = \{2, a\} \)。对上述两个解分别检验：

- 当 \( a = 3 \) 时，  
  \( C = \{2, 3\} \)，此时 \( A = \{2, 3, 0\} \)，  
  \( A \cap C = \{2, 3\} \)，与已知 \( A \cap C = \{2\} \) 矛盾，舍去。

- 当 \( a = -1 \) 时，  
  \( C = \{2, -1\} \)，此时 \( A = \{2, 3, 0\} \)，  
  \( A \cap C = \{2\} \)，符合题意。同时 \( B = \{0, 3\} \subseteq A \) 也成立，且集合元素满足互异性。

综上，实数 \( a = -1 \)。
## 答案
B
## 总结
处理集合包含关系时，关键是要将符号条件转化为元素与集合的从属关系。本题中先利用 \( B \subseteq A \) 求值，再利用交集条件并结合集合元素的互异性进行取舍，避免漏掉检验步骤。
