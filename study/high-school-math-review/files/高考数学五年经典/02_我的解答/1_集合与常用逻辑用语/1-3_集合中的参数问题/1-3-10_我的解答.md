# 1-3-10 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-3_集合中的参数问题/1-3-10.md|1-3-10]]

## 完整题目

1.(2020 全国卷Ⅰ理，2)设集合 \( A = \left\{  {x \mid  {x}^{2} - 4 \leq  }\right. \; 0\} , B = \{ x \mid  {2x} + a \leq  0\} \) ,且 \( A \cap  B = \{ x \mid   - 2 \leq  x \leq \) 1\},则 \( a = \) ( )

A. -4 B. -2 C. 2 D. 4

## 题目勘误
无明显勘误。原题中集合 A 的书写格式 `\left\{ {x \mid {x}^{2} - 4 \leq }\right. \; 0\}` 存在视觉分割，实际意为 \(A=\{x\mid x^2-4\le 0\}\)，不影响题意理解。
## 思路分析
先解出集合 \(A\) 的具体区间，再解出含参数 \(a\) 的集合 \(B\) 的区间。根据交集的结果，确定 \(B\) 的区间端点，从而求出 \(a\) 的值。
## 详细解答
1. 解集合 \(A\)：
   \[
   x^2 - 4 \le 0 \implies (x-2)(x+2)\le 0 \implies -2 \le x \le 2.
   \]
   所以
   \[
   A = \{x \mid -2 \le x \le 2\}.
   \]

2. 解集合 \(B\)：
   \[
   2x + a \le 0 \implies 2x \le -a \implies x \le -\frac{a}{2}.
   \]
   所以
   \[
   B = \{x \mid x \le -\frac{a}{2}\}.
   \]

3. 求交集 \(A \cap B\)：
   \[
   A \cap B = \{x \mid -2 \le x \le 2\} \cap \{x \mid x \le -\frac{a}{2}\}
            = \{x \mid -2 \le x \le \min(2, -\frac{a}{2})\}.
   \]
   题目已知
   \[
   A \cap B = \{x \mid -2 \le x \le 1\}.
   \]
   对比区间右端点，可得
   \[
   \min(2, -\frac{a}{2}) = 1.
   \]
   由于 \(1 < 2\)，必有 \(-\frac{a}{2} = 1\)，解得
   \[
   a = -2.
   \]

4. 检验：当 \(a = -2\) 时，\(-\frac{a}{2} = 1\)，\(B = \{x \mid x \le 1\}\)，\(A \cap B = [-2,2] \cap (-\infty, 1] = [-2, 1]\)，符合题意。
## 答案
B
## 总结
处理含参集合的交集问题时，通常先将固定集合化简为明确区间，再将含参集合表示为关于参数的形式，最后利用已知的交集（或并集）结果“对端点”建立方程，从而求出参数。本题的核心是理解“交集的右端点由两个集合右端点中较小者决定”，并据此列式。
