# 1-3-9 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-3_集合中的参数问题/1-3-9.md|1-3-9]]

## 完整题目

9.(2023 江苏南京、盐城期末)已知集合 \( A = \{ x \mid \; \left. {\frac{x - 1}{x - a} < 0}\right\} \) . 若 \( A \cap  {\mathbf{N}}^{ * } = \varnothing \) ,则实数 \( a \) 的取值范

围是 ( )

A. \( \{ 1\} \) B. \( \left( {-\infty ,1}\right) \)

C. \( \left\lbrack  {1,2}\right\rbrack \) D. \( ( - \infty ,2\rbrack \)

## 题目勘误
无明显勘误。

## 思路分析
先将分式不等式等价转化为二次不等式。因为 \(A \cap \mathbf{N}^* = \varnothing\)，即区间解集中不能含有任何正整数。根据参数 \(a\) 与 \(1\) 的大小关系进行分类讨论，分别求出满足条件的 \(a\) 的范围，最后取并集即可。

## 详细解答
集合 \(A\) 的定义为：
\[
A = \left\{ x \;\middle|\; \frac{x-1}{x-a} < 0 \right\}.
\]
分母 \(x-a \neq 0\)，且 \(\frac{x-1}{x-a}<0\) 等价于 \((x-1)(x-a)<0\)（因为 \(x=a\) 会使分子分母不同时为零且不满足不等式，解集一致）。

因此 \(A = \{x \mid (x-1)(x-a) < 0\}\)。

条件 \(A \cap \mathbf{N}^* = \varnothing\) 要求该二次不等式的解集中不包含任何正整数。

下面按 \(a\) 与 \(1\) 的大小关系讨论：

1. **当 \(a < 1\) 时**，不等式 \((x-1)(x-a) < 0\) 的解集为 \(a < x < 1\)，即 \(A = (a,\,1)\)。  
   此时 \(A\) 中的元素均小于 \(1\)，不含任何正整数，满足 \(A \cap \mathbf{N}^* = \varnothing\)。

2. **当 \(a = 1\) 时**，原不等式变为 \(\frac{x-1}{x-1} < 0\)，即 \(1 < 0\)，解集为空集 \(A = \varnothing\)。  
   空集与任何集合的交集为空，满足条件。

3. **当 \(a > 1\) 时**，不等式解集为 \(1 < x < a\)，即 \(A = (1,\,a)\)。  
   要使 \(A \cap \mathbf{N}^* = \varnothing\)，则区间 \((1,\,a)\) 内不能有正整数。  
   因为正整数从 \(1\) 开始，\(1\) 不在区间内，下一个正整数是 \(2\)。因此只需 \(2 \notin (1,\,a)\)，即 \(a \le 2\)。  
   结合前提 \(a > 1\)，得 \(1 < a \le 2\)。

综合以上三种情况，合并可得实数 \(a\) 的取值范围是 \(a \le 2\)，即 \((-\infty,\,2]\)。

对照选项，该范围对应选项 D。

## 答案
D

## 总结
处理含参分式不等式与整数集交集的空集问题时，通常先转化为标准不等式，再抓住参数对区间端点的影响进行分类讨论。重点关注正整数集从 \(1\) 开始且无上限的特点，逐一排查区间是否能“避开”所有正整数。本题中 \(a=1\) 对应空集、\(a=2\) 对应区间 \((1,2)\) 不含整数，均符合题意，最终范围是 \((-\infty,\,2]\)。
