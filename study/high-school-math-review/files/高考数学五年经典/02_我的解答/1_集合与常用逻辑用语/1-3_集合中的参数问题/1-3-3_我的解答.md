# 1-3-3 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-3_集合中的参数问题/1-3-3.md|1-3-3]]

## 完整题目

3. (多选题, 2022 上海三林中学月考) 设集合 \( A = \{ x \mid  {x}^{2} - {2x} - 3 > 0\} , B = \left\{  {x \mid  {x}^{2} - {ax} + b \leq  }\right. \; 0\} \) ,若 \( A \cup  B = \mathbf{R}, A \cap  B = (3,4\rbrack \) ,则有

A. \( a = 3, b = 4 \) B. \( a = 3, b =  - 4 \)

C. \( \frac{a}{{c}^{2} + 1} > \frac{b}{{c}^{2} + 1} \) D. \( a\left| c\right|  > b\left| c\right| \)

## 题目勘误
1. 集合 \(B\) 的原文 “\(B = \left\{  {x \mid  {x}^{2} - {ax} + b \leq  }\right. \; 0\}\)” 疑似 OCR 识别错误，建议更正为：
   \[
   B = \{ x \mid x^2 - ax + b \le 0 \}.
   \]
2. 选项 C 和 D 中出现了未在题干中定义的字母 \(c\)，根据常规命题习惯推测 \(c\) 为任意实数；若原题对 \(c\) 有额外限定（如 \(c \neq 0\)），需人工核对。

## 思路分析
先求出集合 \(A\)，它是一个二次不等式的解集。再根据 \(A \cup B = \mathbf{R}\) 与 \(A \cap B = (3,4]\) 这两个条件，结合二次不等式解集的形式，反推出集合 \(B\) 对应的区间，进而利用韦达定理求出参数 \(a, b\) 的值。最后将求得的 \(a, b\) 代入各选项逐一判断正误。

## 详细解答
**第一步：求集合 \(A\)。**
解不等式 \(x^2 - 2x - 3 > 0\)：
\[
x^2 - 2x - 3 = (x-3)(x+1) > 0,
\]
解得 \(x < -1\) 或 \(x > 3\)。
故
\[
A = (-\infty, -1) \cup (3, +\infty).
\]

**第二步：由条件推断集合 \(B\)。**
\(B\) 是二次不等式 \(x^2 - ax + b \le 0\) 的解集。由于二次项系数为正，其解集必为闭区间（若判别式非负）或空集。
由 \(A \cup B = \mathbf{R}\) 知，\(B\) 必须覆盖 \(A\) 未覆盖的部分，即区间 \([-1, 3]\)。
由 \(A \cap B = (3,4]\) 知，在 \(x > 3\) 的部分，\(B\) 的上界为 \(4\)，且区间端点为 \(4\)（包含），下界不影响交集（只要不超过 \(3\)），同时 \(B\) 需包含 \([-1, 3]\)。
因此 \(B\) 只能是区间 \([-1, 4]\)。此时检验：
\[
A \cup B = \big[(-\infty, -1) \cup (3, +\infty)\big] \cup [-1,4] = \mathbf{R},
\]
\[
A \cap B = \big[(-\infty, -1) \cup (3, +\infty)\big] \cap [-1,4] = (3,4],
\]
符合题意。

**第三步：求 \(a, b\) 的值。**
\(B = [-1, 4]\) 意味着方程 \(x^2 - ax + b = 0\) 的两根为 \(-1\) 和 \(4\)。
由韦达定理：
\[
a = (-1) + 4 = 3, \qquad b = (-1) \times 4 = -4.
\]

**第四步：判断选项。**
- 选项 A：\(a=3,\; b=4\)（错误）。
- 选项 B：\(a=3,\; b=-4\)（正确）。
- 选项 C：\(\displaystyle \frac{a}{c^2+1} > \frac{b}{c^2+1}\)。因 \(c^2+1 \ge 1 > 0\)，不等式等价于 \(a > b\)。将 \(a=3,\; b=-4\) 代入，\(3 > -4\) 恒成立，故 C 正确。
- 选项 D：\(a|c| > b|c|\)。代入得 \(3|c| > -4|c|\)，即 \(7|c| > 0\)。当 \(c = 0\) 时，\(0 > 0\) 不成立；当 \(c \neq 0\) 时成立。若 \(c\) 为任意实数，该不等式并非恒成立，故 D 错误。

因此正确选项为 B 和 C。

## 答案
B、C

## 总结
处理集合中的参数问题时，要充分利用集合间的关系（交、并）反推出不等式的解集边界，再借助韦达定理或区间端点条件建立方程。对于包含未定义参数的选项，需结合参数的可能取值进行严谨的恒成立判断。
