# 1-3-11 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-3_集合中的参数问题/1-3-11.md|1-3-11]]

## 完整题目

2.(2023全国甲卷理，1)设集合 \( A = \{ x \mid  x = {3k} + \)

\( 1, k \in  \mathbf{Z}\} , B = \{ x \mid  x = {{3k} + 2}, k \in  \mathbf{Z}\} , U \) 为整数集, \( {\complement }_{U}\left( {A \cup  B}\right)  = \) ( )

A. \( \{ x \mid  x = {3k}, k \in  \mathbf{Z}\} \)

B. \( \{ x \mid  x = {3k} - 1, k \in  \mathbf{Z}\} \)

C. \( \{ x \mid  x = {3k} - 2, k \in  \mathbf{Z}\} \)

D. \( \varnothing \)

## 题目勘误
无明显勘误。

## 思路分析
全集 \(U\) 为整数集 \(\mathbf{Z}\)。集合 \(A\) 与 \(B\) 中的元素分别是除以 \(3\) 余 \(1\) 和余 \(2\) 的整数。将整数按除以 \(3\) 的余数分类，只有余 \(0\)、余 \(1\)、余 \(2\) 三种情况。\(A \cup B\) 包含了余 \(1\) 和余 \(2\) 的全部整数，其补集自然就是余 \(0\) 的整数，即 \(3\) 的倍数。对应选项验证即可。

## 详细解答
集合 \(A\) 表示所有除以 \(3\) 余 \(1\) 的整数：
\[ A = \{ x \mid x = 3k + 1, \, k \in \mathbf{Z} \} \]
集合 \(B\) 表示所有除以 \(3\) 余 \(2\) 的整数：
\[ B = \{ x \mid x = 3k + 2, \, k \in \mathbf{Z} \} \]
于是 \(A \cup B\) 包含了所有除以 \(3\) 余 \(1\) 或余 \(2\) 的整数。  
全集 \(U = \mathbf{Z}\) 中的整数除以 \(3\) 的余数只可能为 \(0, 1, 2\)。  
所以 \(A \cup B\) 在 \(U\) 中的补集就是余数为 \(0\) 的整数所构成的集合：
\[ \complement_U (A \cup B) = \{ x \mid x = 3k, \, k \in \mathbf{Z} \} \]

对照选项：
A. \(\{ x \mid x = 3k, \, k \in \mathbf{Z} \}\) ——符合；
B. \(\{ x \mid x = 3k - 1, \, k \in \mathbf{Z} \}\) ——即除以 \(3\) 余 \(2\)，属于 \(B\)；
C. \(\{ x \mid x = 3k - 2, \, k \in \mathbf{Z} \}\) ——即除以 \(3\) 余 \(1\)，属于 \(A\)；
D. \(\varnothing\) ——非空。

故正确选项为 A。

## 答案
A

## 总结
本题考查整数的带余除法分类以及集合的交、并、补运算。关键点在于将描述性集合转化为余数特征，再利用全集分类直接得出补集。此类问题常以“模 \(n\) 同余类”的形式出现，理解集合元素的规律是快速解题的核心。
