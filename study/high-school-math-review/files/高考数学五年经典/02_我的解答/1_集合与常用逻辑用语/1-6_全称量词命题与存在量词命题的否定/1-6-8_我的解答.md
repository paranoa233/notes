# 1-6-8 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-6_全称量词命题与存在量词命题的否定/1-6-8.md|1-6-8]]

## 完整题目

9. (多选题, 2022 广东广大附中高三 10 月月考) \( \forall x \in  \mathbf{R},\left\lbrack  x\right\rbrack \) 表示不超过 \( x \) 的最大整数,十八世纪, \( y = \left\lbrack  x\right\rbrack \) 被“数学王子”高斯采用,因此得名为高斯函数，人们更习惯称为“取整函数”，则下列命题中的真命题是 ( )

A. \( \exists x \in  \mathbf{R}, x = \left\lbrack  x\right\rbrack   - 1 \)

B. \( \exists x \in  \mathbf{R}, x = \left\lbrack  x\right\rbrack   + 1 \)

C. \( \forall x, y \in  \mathbf{R},\left\lbrack  x\right\rbrack   + \left\lbrack  y\right\rbrack   \leq  \left\lbrack  {x + y}\right\rbrack \)

D. 函数 \( y = x - \left\lbrack  x\right\rbrack  \left( {x \in  \mathbf{R}}\right) \) 的值域为 \( \lbrack 0,1) \)

## 题目勘误
无明显勘误。
## 思路分析
本题以高斯函数（取整函数）为载体，考查存在量词命题与全称量词命题的真假判断。需要结合取整函数的定义：\([x]\) 表示不超过 \(x\) 的最大整数，即对任意实数 \(x\)，存在唯一整数 \(n\) 使得 \(n \leq x < n+1\)，此时 \([x]=n\)。对于每个选项，可通过设出整数部分与小数部分，或举反例，逐项推理判断。
## 详细解答
**A.** 命题：\(\exists x \in \mathbf{R}, x = [x] - 1\)。  
假设存在这样的 \(x\)，设 \([x]=n \in \mathbf{Z}\)，则 \(x = n-1\)。由定义 \(n \leq x < n+1\)，代入 \(x=n-1\) 得 \(n \leq n-1\)，矛盾。故不存在这样的 \(x\)，A 为假命题。

**B.** 命题：\(\exists x \in \mathbf{R}, x = [x] + 1\)。  
假设存在 \(x\)，设 \([x]=n\)，则 \(x = n+1\)。由定义 \(n \leq x < n+1\)，应有 \(x < n+1\)，但 \(x=n+1\) 不满足严格小于，且此时 \([x]=n+1 \neq n\)，矛盾。故不存在，B 为假命题。

**C.** 命题：\(\forall x, y \in \mathbf{R}, [x] + [y] \leq [x+y]\)。  
任取 \(x,y \in \mathbf{R}\)，记 \(x = n + \alpha\)，\(y = m + \beta\)，其中 \(n,m \in \mathbf{Z}\)，\(\alpha,\beta \in [0,1)\)。则 \([x]=n\)，\([y]=m\)。于是  
\[
x+y = (n+m) + (\alpha+\beta),\quad \alpha+\beta \in [0,2).
\]  
因此 \([x+y] = n+m\) 或 \(n+m+1\)，从而  
\[
[x] + [y] = n+m \leq [x+y].
\]  
该不等式恒成立，C 为真命题。

**D.** 命题：函数 \(y = x - [x] \; (x \in \mathbf{R})\) 的值域为 \([0,1)\)。  
令 \(x = n + \alpha\)，\(n \in \mathbf{Z}\)，\(\alpha \in [0,1)\)，则 \(y = \alpha\)。当 \(x\) 遍历 \(\mathbf{R}\) 时，\(\alpha\) 可取遍 \([0,1)\) 内的所有值，且能取到 \(0\)（当 \(x\) 为整数），不能取到 \(1\)。故值域为 \([0,1)\)，D 为真命题。

## 答案
CD
## 总结
处理取整函数问题，常将实数写为整数部分与小数部分（小数部分 ∈ [0,1)），并据此分析不等式或等式是否恒成立。本题中，存在型命题常可通过赋值反证，全称型命题需借助定义严格推导。易错点：混淆 [x] 的定义，忽略小数部分为非负且小于 1。
