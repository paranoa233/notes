# 1-1-7 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-7.md|1-1-7]]

## 完整题目

7. (多选题。2021 湖北武汉 3 月质检)图中矩形表示集合 \( U \) , \( A, B \) 是 \( U \) 的两个子集,则阴影部分可以表示为 )

![[05_附件/images/5_587_322_226_158_0.jpg]]

A. \( \left( {{\complement }_{U}A}\right)  \cap  B \) B. \( {\complement }_{B}\left( {A \cap  B}\right) \)

C. \( {\complement }_{U}\left( {A \cap  \left( {{\complement }_{U}B}\right) }\right) \) D. \( {\complement }_{A \cup  B}A \)

## 题目勘误
题目中包含图片，图片信息需人工核对。根据常见题型及选项推断，阴影部分应为集合 $B$ 中去掉 $A \cap B$ 的部分，即 $\{x \mid x \in B \text{ 且 } x \notin A\}$。若无特殊标注，文本信息无明显勘误。

## 思路分析
本题是多选题，考查集合的交、并、补运算以及 Venn 图的理解。需要将阴影部分用集合语言描述，并与各选项表示的集合进行等价比较。根据 Venn 图阴影部分的位置，判断出阴影部分属于 $B$ 但不属于 $A$，即 $B - A$。然后逐个选项化简或变形，看是否与 $B - A$ 等价。

## 详细解答
设全集为 $U$，阴影部分为 $S$。由图可知，$S$ 由属于 $B$ 且不属于 $A$ 的元素组成，因此：
\[
S = \{ x \mid x \in B \text{ 且 } x \notin A \} = B \cap (\complement_U A).
\]

**判断各选项：**

**A.** $(\complement_U A) \cap B$  
由定义，$\complement_U A = \{ x \in U \mid x \notin A \}$，则 $(\complement_U A) \cap B = \{ x \mid x \in B \text{ 且 } x \notin A \} = B - A$，与阴影部分一致，**正确**。

**B.** $\complement_B (A \cap B)$  
$\complement_B (A \cap B)$ 表示 $A \cap B$ 在 $B$ 中的补集，即从 $B$ 中去掉 $A \cap B$ 的部分：
\[
\complement_B (A \cap B) = B - (A \cap B) = \{ x \mid x \in B \text{ 且 } x \notin A \} = B - A,
\]
与阴影部分一致，**正确**。

**C.** $\complement_U (A \cap (\complement_U B))$  
先计算 $A \cap (\complement_U B)$，它表示属于 $A$ 但不属于 $B$ 的元素，即 $A - B$。再取其在全集 $U$ 中的补集：
\[
\complement_U (A - B) = U - (A - B) = (\complement_U A) \cup (A \cap B).
\]
一般情况下 $(\complement_U A) \cup (A \cap B)$ 不等于 $B - A$，除非 $A$ 与 $B$ 满足特殊关系。因此该选项不一定表示阴影部分，**错误**。

**D.** $\complement_{A \cup B} A$  
$\complement_{A \cup B} A$ 表示 $A$ 在 $A \cup B$ 中的补集，即从 $A \cup B$ 中去掉 $A$ 的部分：
\[
\complement_{A \cup B} A = (A \cup B) - A = B - A,
\]
与阴影部分一致，**正确**。

综上所述，正确选项为 A、B、D。

## 答案
A、B、D

## 总结
处理 Venn 图阴影部分的关键在于准确将图形区域转化为集合运算表达式，常用的方法有：直接观察法（属于谁且不属于谁）、分块标注法。得到阴影部分的集合表示后，再利用集合运算律（如补集、差集定义）对各选项进行等价变形或化简，逐一比对即可。注意差集 $B - A$ 可等价写为 $B \cap (\complement_U A)$ 或 $\complement_B (A \cap B)$ 等形式。
