# 1-1-16 我的解答

## 完整题目

7.(2024 新高考卷 I，1)已知集合 \( A = \{ x \mid   - 5 < \; \left. {{x}^{3} < 5}\right\}  , B = \{  - 3, - 1,0,2,3\} \) ,则 \( A \cap  B = \) ( )

A. \( \{  - 1,0\} \) B. \( \{ 2,3\} \)

C. \( \{  - 3, - 1,0\} \) D. \( \{  - 1,0,2\} \)

## 题目勘误
无明显勘误。

## 思路分析
先解出集合 \(A\) 中元素 \(x\) 的取值范围，再与有限集 \(B\) 取交集。由于 \(B\) 中元素个数少，可逐一判断其是否属于 \(A\)。

## 详细解答
集合 \(A = \{x \mid -5 < x^3 < 5\}\)。  
解不等式：  
\[
-5 < x^3 < 5
\]  
因为 \(y = x^3\) 是 \(\mathbf{R}\) 上的增函数，开立方得  
\[
\sqrt[3]{-5} < x < \sqrt[3]{5}
\]  
即  
\[
-\sqrt[3]{5} < x < \sqrt[3]{5}.
\]  
估算：\(\sqrt[3]{5} \approx 1.71\)，所以 \(A \approx (-1.71,\;1.71)\)。

集合 \(B = \{-3,\,-1,\,0,\,2,\,3\}\)。在 \(B\) 中找出属于 \(A\) 的元素：
- \(-3 < -1.71\)，不属于 \(A\)；
- \(-1 \in (-1.71,\,1.71)\)，属于 \(A\)；
- \(0 \in (-1.71,\,1.71)\)，属于 \(A\)；
- \(2 > 1.71\)，不属于 \(A\)；
- \(3 > 1.71\)，不属于 \(A\)。
故 \(A \cap B = \{-1,\,0\}\)。

## 答案
A

## 总结
本题主要考查集合的交运算与无理不等式的解法。抓住立方函数的单调性，利用立方根进行估算即可筛选出有限集中的交集元素。
