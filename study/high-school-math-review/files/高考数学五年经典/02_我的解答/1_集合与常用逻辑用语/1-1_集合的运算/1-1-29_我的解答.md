# 1-1-29 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-29.md|1-1-29]]

## 完整题目

13.(2020 全国卷 II 理，1)已知集合 \( U = \{  - 2 \) ， \( - 1,0,1,2,3\} , A = \{  - 1,0,1\} , B = \{ 1,2\} \) ,则 \( {\complement }_{U}\left( {A \cup  B}\right)  = \) ( )

A. \( \{  - 2,3\} \) B. \( \{  - 2,2,3\} \)

C. \( \{  - 2, - 1,0,3\} \) D. \( \{  - 2, - 1,0,2,3\} \)

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查集合的基本运算：并集与补集。解题思路是先求出集合 \(A\) 与 \(B\) 的并集 \(A \cup B\)，然后在全集 \(U\) 中求 \(A \cup B\) 的补集 \(\complement_U(A \cup B)\)，最后与选项比对得出答案。
## 详细解答
已知全集 \(U = \{-2, -1, 0, 1, 2, 3\}\)，集合 \(A = \{-1, 0, 1\}\)，\(B = \{1, 2\}\)。

首先计算 \(A\) 与 \(B\) 的并集：
\[
A \cup B = \{-1, 0, 1\} \cup \{1, 2\} = \{-1, 0, 1, 2\}.
\]

然后求 \(A \cup B\) 在全集 \(U\) 中的补集，即从 \(U\) 中去掉所有属于 \(A \cup B\) 的元素：
\[
\complement_U(A \cup B) = U \setminus (A \cup B) = \{-2, -1, 0, 1, 2, 3\} \setminus \{-1, 0, 1, 2\} = \{-2, 3\}.
\]

对照选项，\(\{-2, 3\}\) 对应选项 A。
## 答案
A
## 总结
本题考查集合的并集与补集运算，属于基础题。解题的关键是准确理解并集（取所有元素）与补集（在全集中去掉给定集合的元素）的定义，并细心计算，避免遗漏或重复元素。
