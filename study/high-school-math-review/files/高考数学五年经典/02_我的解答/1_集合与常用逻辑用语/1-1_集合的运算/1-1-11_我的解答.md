# 1-1-11 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-11.md|1-1-11]]

## 完整题目

2.(2022 新高考卷 I , 1)若集合 \( M = \{ x \mid  \sqrt{x} < \) 4) \( N = \{ x \mid  {3x} \geq  1\} \) ,则 \( M \cap  N = \) ( )

A. \( \{ x \mid  0 \leq  x < 2\} \) B. \( \left\{  {x\left| {\;\frac{1}{3} \leq  x < 2}\right. }\right\} \)

C. \( \{ x \mid  3 \leq  x < {16}\} \) D. \( \left\{  {x\left| {\;\frac{1}{3} \leq  x < {16}}\right. }\right\} \)

## 题目勘误
原题 OCR 识别存在轻微格式错误：  
- 集合 \( M \) 的表示：`\( M = \{ x \mid  \sqrt{x} < \) 4)` 疑似漏掉右侧花括号，且 `4)` 应为 `4\}`，建议更正为 `\( M = \{ x \mid \sqrt{x} < 4 \} \)`。  
- 集合 \( N \) 的表示：`\( N = \{ x \mid  {3x} \geq  1\} \)` 正确，但花括号和分隔符可统一写成 `\{ x \mid 3x \ge 1 \}`。  
- 选项 B、D 中的复杂格式 `\left\{  {x\left| {\;\frac{1}{3} \leq  x < 2}\right. }\right\}` 可简化为 `\( \{ x \mid \frac{1}{3} \le x < 2 \} \)` 等。  
以上均不影响题意，原意明确，图片信息无需额外核对。

## 思路分析
本题主要考查集合的表示法、简单不等式的解法以及集合的交集运算。  
1. 先解集合 \( M \) 中的不等式 \(\sqrt{x}<4\)，注意偶次根式的定义域。  
2. 解集合 \( N \) 中的不等式 \(3x \ge 1\)。  
3. 将两集合的解集在数轴上表示，取公共部分即为 \( M \cap N \)。

## 详细解答
**第一步：求集合 \( M \)**  
不等式 \(\sqrt{x} < 4\) 中，根号下需要 \(x \ge 0\)。  
两边平方（均为非负数，平方不等号方向不变）：  
\[ x < 16 \]  
结合定义域得：  
\[ M = \{ x \mid 0 \le x < 16 \} \]

**第二步：求集合 \( N \)**  
解不等式 \(3x \ge 1\)：  
\[ x \ge \frac{1}{3} \]  
所以：  
\[ N = \left\{ x \mid x \ge \frac{1}{3} \right\} \]

**第三步：求交集 \( M \cap N \)**  
取 \( 0 \le x < 16 \) 与 \( x \ge \dfrac{1}{3} \) 的公共部分：  
\[ \frac{1}{3} \le x < 16 \]  
用集合表示即为：  
\[ M \cap N = \left\{ x \mid \frac{1}{3} \le x < 16 \right\} \]  
该结果与选项 D 一致。

## 答案
D

## 总结
处理此类集合运算题，关键是准确解出每个集合对应的不等式（组），特别注意偶次根式的定义域限制。得到两个集合后，利用数轴或直接比较端点大小求出交集。本题难度较低，属于基础题，注意细节即可避免失误。
