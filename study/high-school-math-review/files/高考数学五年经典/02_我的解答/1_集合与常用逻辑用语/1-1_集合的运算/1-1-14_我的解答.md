# 1-1-14 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-14.md|1-1-14]]

## 完整题目

5.(2020 新高考卷 II ,1)设集合 \( A = \{ 2,3,5,7\} \) ， \( B = \{ 1,2,3,5,8\} \) ,则 \( A \cap  B = \) ( )

A. \( \{ 1,3,5,7\} \) B. \( \{ 2,3\} \)

C. \( \{ 2,3,5\} \) D. \( \{ 1,2,3,5,7,8\} \)

## 题目勘误
无明显勘误。
## 思路分析
本题考查集合的交集运算。交集的定义是由既属于集合 \(A\) 又属于集合 \(B\) 的所有元素组成的集合。只需找出两个集合中共同的元素即可。
## 详细解答
集合 \(A = \{ 2, 3, 5, 7 \}\)，集合 \(B = \{ 1, 2, 3, 5, 8 \}\)。  
观察可知，同时出现在两个集合中的元素有 \(2, 3, 5\)。  
根据交集的定义：
\[
A \cap B = \{ x \mid x \in A \text{ 且 } x \in B \} = \{ 2, 3, 5 \}.
\]
对比选项，C 选项为 \(\{ 2, 3, 5 \}\)，符合计算结果。
## 答案
C
## 总结
解决集合交、并、补运算的关键是准确理解定义，交集关注“公共部分”，并集关注“所有合并”，补集关注“全集下的剩余部分”。对于有限集，直接列举观察是最可靠的方法，注意不要漏掉或多加元素。
