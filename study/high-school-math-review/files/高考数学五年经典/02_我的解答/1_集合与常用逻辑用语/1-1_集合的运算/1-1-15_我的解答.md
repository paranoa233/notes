# 1-1-15 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-15.md|1-1-15]]

## 完整题目

6.(2020 新高考卷 I ，1)设集合 \( A = \{ x \mid  1 \leq  x \leq \; 3\} , B = \{ x \mid  2 < x < 4\} \) ,则 \( A \cup  B = \)

A. \( \{ x \mid  2 < x \leq  3\} \) B. \( \{ x \mid  2 \leq  x \leq  3\} \)

C. \( \{ x \mid  1 \leq  x < 4\} \) D. \( \{ x \mid  1 < x < 4\} \)

## 题目勘误
无明显勘误。
## 思路分析
求两个集合的并集 \( A \cup B \). 集合 \(A\) 是闭区间 \([1,3]\)，集合 \(B\) 是开区间 \((2,4)\). 并集即为两个区间覆盖的所有实数，需要在数轴上找出从最小下界到最大上界的连续范围，并准确判断端点是否包含。
## 详细解答
集合 \(A\)：\(1 \leq x \leq 3\)，区间表示为 \([1,3]\)。  
集合 \(B\)：\(2 < x < 4\)，区间表示为 \((2,4)\)。  
求 \(A \cup B\)，即求两个区间的并集。  
在数轴上，\(A\) 覆盖了从 \(1\) 到 \(3\)（包含端点），\(B\) 覆盖了从 \(2\) 到 \(4\)（不包含端点）。  
两个区间在 \(2\) 到 \(3\) 处重叠，合并后的整体覆盖范围从 \(1\) 开始，到 \(4\) 结束。  
因为 \(1 \in A\)，所以 \(1 \in A \cup B\)，左端点为闭；因为 \(4 \notin A\) 且 \(4 \notin B\)，所以 \(4 \notin A \cup B\)，右端点为开。  
因此，\(A \cup B = \{ x \mid 1 \leq x < 4 \}\)。

选项分析：  
A. \(\{ x \mid 2 < x \leq 3\}\) —— 缺少 \([1,2]\) 和 \((3,4)\) 部分，错误。  
B. \(\{ x \mid 2 \leq x \leq 3\}\) —— 缺少 \(1\) 和 \((3,4)\) 部分，且 \(2\) 不在 \(B\) 中，但 \(2 \in A\)，故并集中应包含 \(2\)，不过此选项仍不完整，错误。  
C. \(\{ x \mid 1 \leq x < 4\}\) —— 正确。  
D. \(\{ x \mid 1 < x < 4\}\) —— 不包含 \(1\)，错误。
## 答案
C
## 总结
本题考察集合的并集运算，实质是区间在数轴上的合并。需注意端点的开闭情况：由参与并集的集合中端点所属情况决定，只要至少有一个集合包含该端点，并集中就包含。建议画数轴辅助分析，避免遗漏。
