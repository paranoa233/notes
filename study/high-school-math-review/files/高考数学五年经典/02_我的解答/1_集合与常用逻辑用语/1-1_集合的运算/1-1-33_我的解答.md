# 1-1-33 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-33.md|1-1-33]]

## 完整题目

17.(2024全国甲卷文,2)集合 \( A = \{ 1,2,3,4,5 \) , 9) \( B = \{ x \mid  x + 1 \in  A\} \) ,则 \( A \cap  B = \) ( )

A. \( \{ 1,3,4\} \) B. \( \{ 2,3,4\} \)

C. \( \{ 1,2,3,4\} \) D. \( \{ 0,1,2,3,4,9\} \)

## 题目勘误
无明显勘误。

## 思路分析
集合 \(A\) 的元素已明确给出，集合 \(B\) 由描述法定义，条件为 \(x+1 \in A\)。通过解方程 \(x+1 = a\)（\(a\) 遍历 \(A\) 中所有元素），可求得 \(B\) 中的所有元素，再与 \(A\) 取交集即可。

## 详细解答
由题意，\(A = \{1, 2, 3, 4, 5, 9\}\)。  
集合 \(B\) 中的元素 \(x\) 满足 \(x + 1 \in A\)，即 \(x + 1\) 依次等于 \(A\) 中的每一个元素：
\[
\begin{aligned}
x + 1 &= 1 \implies x = 0, \\
x + 1 &= 2 \implies x = 1, \\
x + 1 &= 3 \implies x = 2, \\
x + 1 &= 4 \implies x = 3, \\
x + 1 &= 5 \implies x = 4, \\
x + 1 &= 9 \implies x = 8.
\end{aligned}
\]
因此，\(B = \{0, 1, 2, 3, 4, 8\}\)。  
再求交集：
\[
A \cap B = \{1, 2, 3, 4, 5, 9\} \cap \{0, 1, 2, 3, 4, 8\} = \{1, 2, 3, 4\}.
\]

## 答案
C. \(\{1, 2, 3, 4\}\)

## 总结
本题是集合基本运算题，关键在于正确理解描述法定义的集合，通过解简单等式还原出集合 \(B\) 的元素，再求交集。注意求交集时不要遗漏或多余元素，并检查所得集合的互异性。
