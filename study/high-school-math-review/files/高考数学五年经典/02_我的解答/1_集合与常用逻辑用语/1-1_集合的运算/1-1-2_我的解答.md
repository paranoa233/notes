# 1-1-2 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-2.md|1-1-2]]

## 完整题目

2.(2022 江苏南京、盐城二模)已知集合 \( A = \{ x \mid \; y = \ln \left( {x - 2}\right) \} , B = \left\{  {x \mid  {x}^{2} - {4x} + 3 \leq  0}\right\} \) ,则 \( A \cup  B = \) ( )

A. \( \left\lbrack  {1,3}\right\rbrack \) B. \( \left( {2,3}\right\rbrack \)

C. \( \left\lbrack  {1, + \infty }\right) \) D. \( \left( {2, + \infty }\right) \)

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查对数函数的定义域、一元二次不等式的解法以及集合的并集运算。首先需分别求出集合 \(A\) 和 \(B\) 中 \(x\) 的取值范围，再将它们在数轴上表示出来，取所有覆盖的部分即可得到 \(A \cup B\)。
## 详细解答
1. **求集合 \(A\)**：
   已知 \(A = \{x \mid y = \ln(x-2)\}\)，由对数函数的真数大于 \(0\)，得：
   \[
   x - 2 > 0 \implies x > 2
   \]
   所以 \(A = (2, +\infty)\)。

2. **求集合 \(B\)**：
   已知 \(B = \{x \mid x^2 - 4x + 3 \leq 0\}\)，解一元二次不等式：
   \[
   x^2 - 4x + 3 \leq 0 \implies (x-1)(x-3) \leq 0
   \]
   由二次函数图像或口诀“大于取两边，小于取中间”，解得：
   \[
   1 \leq x \leq 3
   \]
   所以 \(B = [1, 3]\)。

3. **求并集 \(A \cup B\)**：
   在数轴上画出 \(A = (2, +\infty)\) 与 \(B = [1, 3]\)：
   - \(B\) 覆盖了从 \(1\) 到 \(3\) 的所有点（含端点）；
   - \(A\) 覆盖了从 \(2\) 向右的所有点（不含 \(2\)）。
   
   合并后，从 \(1\)（含）往右一直到正无穷的所有实数都被覆盖，故：
   \[
   A \cup B = [1, +\infty)
   \]

4. **对照选项**：
   - A. \([1,3]\) 范围不全，缺 \((3,+\infty)\)；
   - B. \((2,3]\) 范围不全，且未包含 \([1,2]\)；
   - C. \([1, +\infty)\) 与所求结果一致；
   - D. \((2, +\infty)\) 缺少 \([1,2]\) 部分。
## 答案
C
## 总结
本题为基础运算题，解题关键在于准确求出定义域和解不等式，并理解并集是两集合所有元素的合并，借助数轴能更直观地得出正确区间。注意端点的开闭情况：\(A\) 中 \(x=2\) 取不到，但 \(B\) 中包含 \(x=1\) 和 \(x=3\)，故并集的左端点取闭区间。
