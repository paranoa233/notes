# 1-1-21 我的解答

## 完整题目

5.(2022全国甲卷文，1)设集合 \( A = \{  - 2, - 1 \) ， \( 0,1,2\} , B = \left\{  {x\left| {\;0 \leq  x < \frac{5}{2}}\right. }\right\} \) ,则 \( A \cap  B = \) ( )

A, \( \{ 0,1,2\} \) B. \( \{  - 2, - 1,0\} \)

C. \( \{ 0,1\} \) D. \( \{ 1,2\} \)

## 题目勘误
无明显勘误。
## 思路分析
本题考查集合的交集运算。集合 \(A\) 是有限集，集合 \(B\) 是用描述法表示的无限集。求 \(A \cap B\)，就是找出集合 \(A\) 中所有满足集合 \(B\) 中条件 \(0 \leq x < \frac{5}{2}\) 的元素。将 \(A\) 中的元素逐一验证即可。
## 详细解答
集合 \(A = \{-2, -1, 0, 1, 2\}\)。
集合 \(B = \left\{ x \;\middle|\; 0 \leq x < \frac{5}{2} \right\}\)，即区间 \([0, 2.5)\)。
求交集 \(A \cap B\)，需在 \(A\) 中找出满足 \(0 \leq x < 2.5\) 的元素：
- \(-2 < 0\)，不满足；
- \(-1 < 0\)，不满足；
- \(0 \geq 0\) 且 \(0 < 2.5\)，满足；
- \(1 \geq 0\) 且 \(1 < 2.5\)，满足；
- \(2 \geq 0\) 且 \(2 < 2.5\)，满足。

因此 \(A \cap B = \{0, 1, 2\}\)。
对应选项 A。
## 答案
A
## 总结
解决有限集与无限集的交集问题时，通常以有限集中的元素为基准，逐一验证其是否属于无限集对应的不等式范围。注意区间端点是否可取，本题中 \(0\) 可以取到，所以包含 \(0\)。
