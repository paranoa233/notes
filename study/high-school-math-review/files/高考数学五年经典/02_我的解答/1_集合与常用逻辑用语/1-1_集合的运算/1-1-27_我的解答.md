# 1-1-27 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-27.md|1-1-27]]

## 完整题目

11.(2021全国乙卷文，1)已知全集 \( U = \{ 1,2,3 \) , \( 4,5\} \) ,集合 \( M = \{ 1,2\} , N = \{ 3,4\} \) ,则 \( {\complement }_{U}\left( {M \cup  N}\right) \; = \) ( )

A. \{5\} B. \( \{ 1,2\} \)

C. \( \{ 3,4\} \) D. \( \{ 1,2,3,4\} \)

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查集合的基本运算。需要先求出集合 \(M\) 与 \(N\) 的并集，再根据全集 \(U\) 求该并集的补集。运算顺序清晰，属于基础题。
## 详细解答
已知全集 \(U = \{1, 2, 3, 4, 5\}\)，集合 \(M = \{1, 2\}\)，\(N = \{3, 4\}\)。

第一步：求并集 \(M \cup N\)。
根据并集的定义，\(M \cup N = \{x \mid x \in M \text{ 或 } x \in N\}\)。
因此，\(M \cup N = \{1, 2\} \cup \{3, 4\} = \{1, 2, 3, 4\}\)。

第二步：求补集 \(\complement_{U}(M \cup N)\)。
根据补集的定义，\(\complement_{U}(M \cup N) = \{x \mid x \in U \text{ 且 } x \notin M \cup N\}\)。
全集 \(U\) 中的元素为 \(1, 2, 3, 4, 5\)，其中 \(1, 2, 3, 4\) 均在并集 \(\{1, 2, 3, 4\}\) 中，只有元素 \(5\) 不在并集中。
故 \(\complement_{U}(M \cup N) = \{5\}\)。
## 答案
A
## 总结
求解集合混合运算时，应严格按照先算并集、交集等基本运算，再算补集（或根据题设括号顺序运算）。本题为全国卷文科数学第1题，难度较低，重点在于对运算定义的理解和细心程度。
