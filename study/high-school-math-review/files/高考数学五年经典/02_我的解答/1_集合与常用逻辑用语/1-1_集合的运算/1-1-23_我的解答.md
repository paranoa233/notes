# 1-1-23 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-23.md|1-1-23]]

## 完整题目

7.(2021 全国甲卷理，1)设集合 \( M = \{ x \mid  0 < x < \) 4) \( N = \left\{  {x\left| {\;\frac{1}{3} \leq  x \leq  5}\right. }\right\} \) ,则 \( M \cap  N = \) ( )

A. \( \left\{  {x\left| {\;0 < x \leq  \frac{1}{3}}\right. }\right. \} \) B. \( \left\{  {x\left| {\;\frac{1}{3} \leq  x < 4}\right. }\right\} \)

C. \( \{ x \mid  4 \leq  x < 5\} \) D. \( \{ x \mid  0 < x \leq  5\} \)

## 题目勘误
无明显影响题意的核心勘误，但原题录入存在多余括号与不规范的集合表示，建议更正为：
- 原题：\( M = \{ x \mid  0 < x < \) 4)
- 更正：\( M = \{ x \mid 0 < x < 4 \} \)
- 理由：多余的右括号“)”为录入错误，标准集合应使用花括号闭合。

## 思路分析
本题考查集合的交集运算。两个集合均为用描述法表示的数集，可先将集合的符号语言“翻译”成区间，再借助数轴找出公共部分。交集 \( M \cap N \) 即同时满足 \( x \in M \) 与 \( x \in N \) 的 \( x \) 的取值范围。解题时需重点检查端点值是否可取（开区间还是闭区间）。

## 详细解答
由题可知：
\[
M = \{ x \mid 0 < x < 4 \} = (0, 4)
\]
\[
N = \left\{ x \mid \frac{1}{3} \leq x \leq 5 \right\} = \left[ \frac{1}{3}, 5 \right]
\]
求 \( M \cap N \)，即求两个区间公共部分。
在数轴上分别表示 \( M \) 和 \( N \)：
- \( M \) 覆盖从 \( 0 \)（不包含）到 \( 4 \)（不包含）的部分。
- \( N \) 覆盖从 \( \frac{1}{3} \)（包含）到 \( 5 \)（包含）的部分。

重叠部分从 \( \frac{1}{3} \) 开始（\( \frac{1}{3} \) 在 \(0\) 与 \(4\) 之间，且 \( N \) 包含该点，\( M \) 也包含该点，故公共部分包含 \( \frac{1}{3} \)），到 \( 4 \) 结束（\( M \) 不包含 \(4\)，\( N \) 包含 \(4\)，故公共部分不包含 \(4\)）。
因此：
\[
M \cap N = \left\{ x \mid \frac{1}{3} \leq x < 4 \right\} = \left[ \frac{1}{3}, 4 \right)
\]
对照选项，即为 B。

## 答案
B

## 总结
处理描述法表示的数集交集时，转化为区间是最直观的方法。要注意“开区间”端点用“\( < \)”或“\( > \)”，对应空心点，不属于集合；“闭区间”端点用“\( \leq \)”或“\( \geq \)”，对应实心点，属于集合。交集取两端点中公共且严格的限制。
