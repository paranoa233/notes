# 1-1-20 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-20.md|1-1-20]]

## 完整题目

4.(2020全国卷Ⅲ 文，1)已知集合 \( A = \{ 1,2,3 \) ， \( 5,7,{11}\} , B = \{ x \mid  3 < x < {15}\} \) ,则 \( A \cap  B \) 中元素的个数为 ( )

A. 2 B. 3 C. 4 D. 5

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查集合的交集运算及集合中元素个数的计算。集合 \(A\) 是有限集，直接给出了具体的元素；集合 \(B\) 是用描述法表示的无限集。求 \(A \cap B\) 中元素的个数，只需从集合 \(A\) 的元素中筛选出满足集合 \(B\) 中条件 \(3 < x < 15\) 的元素，再统计个数即可。
## 详细解答
集合 \(A = \{1, 2, 3, 5, 7, 11\}\)，
集合 \(B = \{x \mid 3 < x < 15\}\)。
\(A \cap B\) 表示既属于 \(A\) 又属于 \(B\) 的元素组成的集合，即集合 \(A\) 中满足 \(3 < x < 15\) 的元素。
检查集合 \(A\) 中的每一个元素：
- \(1\)：不满足 \(3 < x < 15\)，舍去；
- \(2\)：不满足 \(3 < x < 15\)，舍去；
- \(3\)：不满足 \(x > 3\)（注意 \(3 < x\) 是严格大于），舍去；
- \(5\)：满足 \(3 < 5 < 15\)，保留；
- \(7\)：满足 \(3 < 7 < 15\)，保留；
- \(11\)：满足 \(3 < 11 < 15\)，保留。
因此 \(A \cap B = \{5, 7, 11\}\)，共有 \(3\) 个元素。
## 答案
B
## 总结
处理有限集与无限集的交集问题时，通常以有限集中的元素为基础，逐一验证是否满足无限集的条件，从而确定交集中的元素并统计个数。验证时需注意边界值是否可取（本题为严格不等号，故端点 \(3\) 和 \(15\) 不满足条件）。
