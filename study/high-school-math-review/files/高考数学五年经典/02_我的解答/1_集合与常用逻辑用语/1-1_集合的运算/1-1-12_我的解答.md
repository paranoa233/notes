# 1-1-12 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-12.md|1-1-12]]

## 完整题目

3.(2022 新高考卷 II ,1)已知集合 \( A = \{  - 1,1 \) ， \( 2,4\} , B = \{ x\left| \right| x - 1 \mid   \leq  1\} \) ，则 \( A \cap  B = \) ( )

A. \( \{  - 1,2\} \) B. \( \{ 1,2\} \)

C. \( \{ 1,4\} \) D. \( \{  - 1,4\} \)

## 题目勘误
原文 `\{  - 1,1 \) ， \( 2,4\} ` 建议更正为 `\{-1, 1, 2, 4\}`；原文 `B = \{ x\left| \right| x - 1 \mid   \leq  1\}` 建议更正为 `B = \{ x \mid |x-1| \leq 1 \}`。理由：LaTeX 代码缺失分隔符，可能导致渲染异常，但不影响题意理解。

## 思路分析
先解绝对值不等式求出集合 \(B\) 的具体区间，再求 \(A\) 与 \(B\) 的交集，即找出 \(A\) 中所有满足 \(B\) 中条件的元素。

## 详细解答
集合 \(A = \{-1, 1, 2, 4\}\)。  
集合 \(B = \{x \mid |x-1| \leq 1\}\)。  
解绝对值不等式：  
\[
|x-1| \leq 1 \implies -1 \leq x-1 \leq 1 \implies 0 \leq x \leq 2,
\]  
所以 \(B = [0, 2]\)（或写成 \(\{x \mid 0 \leq x \leq 2\}\)）。  
求交集 \(A \cap B\)，即找出 \(A\) 中属于区间 \([0, 2]\) 的元素：  
\(-1 \notin [0,2]\)；\(1 \in [0,2]\)；\(2 \in [0,2]\)；\(4 \notin [0,2]\)。  
因此 \(A \cap B = \{1, 2\}\)。

## 答案
B

## 总结
本题考察集合的交集运算与绝对值不等式的解法，属于基础题。先化简集合 \(B\)，再找公共元素即可。注意不等式的边界 \(0\) 和 \(2\) 能否取到，以及集合 \(A\) 中的元素是否满足条件。
