# 1-1-5 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-5.md|1-1-5]]

## 完整题目

5. (2020 浙江镇海中学期中)已知集合 \( A = \{ x \in \; \mathbf{Z}\left| {\;{x}^{2} - {4x} - 5 \leq  0}\right. \} , B = \{ x \mid  0 < \ln x < 2\} \) ,则 \( A \cap \; B \) 的元素的个数为 ( )

A. 2 B. 3 C. 4 D. 7

## 题目勘误
无明显勘误。
## 思路分析
集合 \(A\) 是通过一元二次不等式限制，且元素必须为整数；可先解不等式求出整数解集。集合 \(B\) 是对数不等式，利用对数函数的单调性转化为关于 \(x\) 的不等式。然后取两集合的交集，并统计交集中元素的个数。
## 详细解答
先求集合 \(A\)：
由 \(x^2 - 4x - 5 \leq 0\) 得 \((x-5)(x+1) \leq 0\)，解得 \(-1 \leq x \leq 5\)。
又 \(x \in \mathbf{Z}\)，所以
\[ A = \{-1, 0, 1, 2, 3, 4, 5\}. \]

再求集合 \(B\)：
由 \(0 < \ln x < 2\) 得 \(\ln 1 < \ln x < \ln e^2\)，由于 \(y = \ln x\) 是增函数，所以
\[ 1 < x < e^2. \]
而 \(e \approx 2.718\)，\(e^2 \approx 7.389\)，故
\[ B = \{ x \mid 1 < x < e^2 \}. \]

求交集 \(A \cap B\)：
\(A \cap B\) 中的元素需同时满足：是整数且属于 \((-1 \leq x \leq 5)\)，又在 \((1, e^2)\) 内。
因此元素为 \(2, 3, 4, 5\)（注意 \(e^2 \approx 7.389 > 5\)，故 \(A\) 中小于等于 \(5\) 且大于 \(1\) 的整数全部符合）。
所以
\[ A \cap B = \{2, 3, 4, 5\}. \]
元素的个数为 \(4\)。
## 答案
C
## 总结
处理集合运算时，需准确求出每个集合的具体元素，特别注意定义域的限制（如本题中 \(A\) 限制为整数）。对含有对数、指数的不等式，要化为同底并利用单调性求解，必要时估算常数（如 \(e^2\)）的范围以确定整数边界。
