# 1-1-10 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-10.md|1-1-10]]

## 完整题目

1.(2023 新高考卷I，1)已知集合 \( M = \{  - 2, - 1,0 \) ， \( 1,2\} , N = \left\{  {x \mid  {x}^{2} - x - 6 \geq  0}\right\} \) ,则 \( M \cap  N = \) ( )

A. \( \{  - 2, - 1,0,1\} \) B. \( \{ 0,1,2\} \)

C. \( \{  - 2\} \) D. \( \{ 2\} \)

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查集合的交集运算以及一元二次不等式的解法。首先解集合 \(N\) 中的不等式 \(x^2 - x - 6 \ge 0\)，确定 \(N\) 的元素范围，然后与给定的有限集 \(M\) 求交集。
## 详细解答
对于集合 \(N\)，解不等式：
\[
x^2 - x - 6 \ge 0
\]
因式分解得 \((x-3)(x+2) \ge 0\)，
解得 \(x \le -2\) 或 \(x \ge 3\)。
即 \(N = \{ x \mid x \le -2 \text{ 或 } x \ge 3 \}\)。

已知 \(M = \{-2, -1, 0, 1, 2\}\)，
求 \(M \cap N\)，即从 \(M\) 中找出属于 \(N\) 的元素：
- \(-2\)：满足 \(-2 \le -2\)，属于 \(N\)；
- \(-1, 0, 1, 2\)：均不满足 \(x \le -2\)，也不满足 \(x \ge 3\)（因为 \(2 < 3\)），故均不属于 \(N\)。
因此，\(M \cap N = \{ -2 \}\)。
## 答案
C
## 总结
解一元二次不等式时，注意“大于取两边，小于取中间”的口诀（针对二次项系数为正的情况）。求交集时，将具体元素逐一验证是否符合不等式条件，避免漏解。本题属基础题，需保证运算准确。
