# 1-1-13 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-13.md|1-1-13]]

## 完整题目

4.(2021 新高考卷Ⅰ，1)设集合 \( A = \{ x \mid   - 2 < \; x < 4\} , B = \{ 2,3,4,5\} \) ,则 \( A \cap  B = \) ( )

A. \( \{ 2\} \) B. \( \{ 2,3\} \)

C. \( \{ 3,4\} \) D. \( \{ 2,3,4\} \)

## 题目勘误
无明显勘误。
## 思路分析
本题考察集合的交集运算。集合 \(A\) 是用描述法给出的不等式解集，代表一个连续区间内的实数；集合 \(B\) 是列举法给出的有限整数集。求 \(A \cap B\)，即找出 \(B\) 中所有满足 \(-2 < x < 4\) 的元素。
## 详细解答
1. **明确集合 \(A\) 的元素**：
   \[
   A = \{x \mid -2 < x < 4\}
   \]
   表示所有大于 \(-2\) 且小于 \(4\) 的实数，开区间。

2. **明确集合 \(B\) 的元素**：
   \[
   B = \{2, 3, 4, 5\}
   \]
   包含 \(4\) 个整数。

3. **逐项判断 \(B\) 中元素是否属于 \(A\)**：
   - \(2\): 满足 \(-2 < 2 < 4\)，属于 \(A\)。
   - \(3\): 满足 \(-2 < 3 < 4\)，属于 \(A\)。
   - \(4\): 不满足 \(x < 4\)（严格小于），不属于 \(A\)。
   - \(5\): 不满足 \(x < 4\)，不属于 \(A\)。

4. **写出交集**：
   \[
   A \cap B = \{2, 3\}
   \]
   对照选项，B 项正确。
## 答案
B
## 总结
处理集合运算题的关键是“看准元素”——连续型集合与离散型集合取交集时，通常只需逐一检验离散集合的每个元素是否满足连续集合的条件。注意区间端点不等号是否带等号，避免误判边界值。
