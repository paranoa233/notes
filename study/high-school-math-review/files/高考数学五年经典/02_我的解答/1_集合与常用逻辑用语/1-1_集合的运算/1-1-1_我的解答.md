# 1-1-1 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-1.md|1-1-1]]

## 完整题目

1. (2024 湖北八市 3 月联考)设集合 \( A = \{ 1,2\} \) ， \( B = \{ 2,3\} , C = \{ 1,2,3,4\} \) ,则 ( )

A. \( A \cap  B = \varnothing \) B. \( A \cup  B = C \)

C. \( A \cup  C = C \) D. \( A \cap  C = B \)

## 题目勘误

无明显勘误。

## 思路分析

本题是典型的集合运算选择题，考查交集、并集的运算以及集合相等关系的判断。

- **已知条件**：三个有限集 \( A = \{1, 2\} \)，\( B = \{2, 3\} \)，\( C = \{1, 2, 3, 4\} \)。
- **考查点**：\( \cap \)（交集）和 \( \cup \)（并集）的定义，以及子集关系的判定。
- **关键转化**：直接对每个选项进行运算验证，逐个判断即可。本题直接计算量小，关键是准确理解交集和并集的定义。
- **突破口**：注意到 \( A \subseteq C \)，因此 \( A \cup C = C \) 总是成立，选项 C 极有可能是正确答案。

## 详细解答

已知：

\[
A = \{1, 2\},\quad B = \{2, 3\},\quad C = \{1, 2, 3, 4\}
\]

逐项判断四个选项：

**选项 A**：\( A \cap B = \varnothing \)

\[
A \cap B = \{1, 2\} \cap \{2, 3\} = \{2\} \neq \varnothing
\]

交集取两集合的公共元素，\( 2 \) 同时在 \( A \) 和 \( B \) 中，故 \( A \cap B = \{2\} \)，选项 A 错误。

**选项 B**：\( A \cup B = C \)

\[
A \cup B = \{1, 2\} \cup \{2, 3\} = \{1, 2, 3\} \neq C = \{1, 2, 3, 4\}
\]

并集取两集合所有元素的合并，\( 4 \) 不在 \( A \cup B \) 中而在 \( C \) 中，故不相等，选项 B 错误。

**选项 C**：\( A \cup C = C \)

\[
A \cup C = \{1, 2\} \cup \{1, 2, 3, 4\} = \{1, 2, 3, 4\} = C
\]

因为 \( A = \{1, 2\} \subseteq C = \{1, 2, 3, 4\} \)，所以 \( A \cup C = C \) 恒成立。选项 C 正确。

**选项 D**：\( A \cap C = B \)

\[
A \cap C = \{1, 2\} \cap \{1, 2, 3, 4\} = \{1, 2\} \neq B = \{2, 3\}
\]

\( A \cap C = \{1, 2\} \)，但 \( B = \{2, 3\} \)，两者不等，选项 D 错误。

## 答案

**C**

## 总结

- **方法**：逐项代入验证法。对于集合运算选择题，直接计算四个选项的运算结果，与题目给出的等式对比即可。
- **易错点**：
  - 交并集符号混淆：\( \cap \) 取公共部分，\( \cup \) 取合并部分。
  - 遗漏元素：计算并集时要确保不重复不遗漏。
  - \( A \cup C = C \) 等价于 \( A \subseteq C \)，这是常用的等价转化。
- **可迁移套路**：当遇到"下列哪个正确"类型的集合运算选择题时，逐项验证是最稳妥的方法。注意利用子集关系可以快速判断并集等式。
