# 1-1-25 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-1_集合的运算/1-1-25.md|1-1-25]]

## 完整题目

9.(2020 全国卷Ⅱ文，1)已知集合 \( A = \{ x\left| \right| x \mid   < 3 \) ， \( x \in  \mathbf{Z}\} , B = \{ x\left| \right| x \mid   > 1, x \in  \mathbf{Z}\} \) ,则 \( A \cap  B =  \) (   )

A. \( \varnothing \) B. \( \{  - 3, - 2,2,3\} \)

C. \( \{  - 2,0,2\} \) D. \( \{  - 2,2\} \)

## 题目勘误
原题中集合表示存在排版问题：  
`A = \{ x\left| \right| x \mid < 3 ， x \in \mathbf{Z}\}` 和 `B = \{ x\left| \right| x \mid > 1, x \in \mathbf{Z}\}`  
中的 `\left| \right|` 与 `\mid` 系绝对值符号识别错误，本意应为：  
\(A = \{ x \mid |x| < 3, x \in \mathbf{Z} \}\)，\(B = \{ x \mid |x| > 1, x \in \mathbf{Z} \}\)。  
解答按标准含义进行。

## 思路分析
本题考查集合的表示法与交集运算，涉及绝对值不等式与整数的双重条件。  
集合 \(A\) 表示绝对值小于 \(3\) 的整数，集合 \(B\) 表示绝对值大于 \(1\) 的整数。  
\(A \cap B\) 表示既属于 \(A\) 又属于 \(B\) 的整数，即绝对值在 \(1\) 和 \(3\) 之间（不含端点）的整数。

## 详细解答
1. **确定集合 \(A\) 的元素**  
   \( |x| < 3 \) 且 \( x \in \mathbf{Z} \)，解得整数 \(x\) 满足 \(-3 < x < 3\)。  
   整数解为：\(-2, -1, 0, 1, 2\)。  
   即 \( A = \{-2, -1, 0, 1, 2\} \)。

2. **确定集合 \(B\) 的元素**  
   \( |x| > 1 \) 且 \( x \in \mathbf{Z} \)，解得整数 \(x\) 满足 \(x < -1\) 或 \(x > 1\)。  
   整数解为：\(\dots, -4, -3, -2, 2, 3, 4, \dots\)（所有绝对值大于1的整数）。  
   即 \( B = \{\dots, -3, -2, 2, 3, \dots\} \)。

3. **求交集 \(A \cap B\)**  
   \(A \cap B = \{ x \mid x \in A \text{ 且 } x \in B \}\)。  
   从 \(A\) 中筛选出绝对值大于 \(1\) 的整数：  
   \(-2\) 的绝对值为 \(2 > 1\)，符合；  
   \(-1\) 的绝对值为 \(1\)，不符合；  
   \(0\) 的绝对值为 \(0\)，不符合；  
   \(1\) 的绝对值为 \(1\)，不符合；  
   \(2\) 的绝对值为 \(2 > 1\)，符合。  
   因此 \(A \cap B = \{-2, 2\}\)。

## 答案
D. \(\{ -2, 2\}\)

## 总结
- 解集合运算题时，需先将抽象条件转化为具体元素（尤其是有限集或可列举的整数集），再执行交、并、补等运算。
- 涉及绝对值与整数结合的题目，可直接列举符合条件的整数以避免符号逻辑错误。
- 注意端点值的取舍：本题中 \(|x|<3\) 不包含 \(\pm 3\)，\(|x|>1\) 不包含 \(\pm 1\)。
