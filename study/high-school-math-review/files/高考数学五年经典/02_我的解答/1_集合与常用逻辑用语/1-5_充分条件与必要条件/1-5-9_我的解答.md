# 1-5-9 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-5_充分条件与必要条件/1-5-9.md|1-5-9]]

## 完整题目

9.(2024 浙江台州第二次教学质量评估)已知 \( x \) ， \( y \) 为正实数，则能成为“ \( x < y \) ”的充要条件的是

A. \( \frac{1}{x} < \frac{1}{y} \)

B. \( x + \ln y < y + \ln x \)

C. \( \sin x < \sin y \)

D. \( x - \cos y < y - \cos x \)

## 题目勘误
无明显勘误。

## 思路分析
本题要求从四个选项中找出“在 \(x,y>0\) 时，与 \(x<y\) 等价”的充要条件。  
基本策略是：  
1. 将每个选项等价变形，转化为比较两个关于 \(x\) 和 \(y\) 的表达式的大小；  
2. 构造辅助函数，利用导数分析其单调性（或举反例）判断是否与 \(x<y\) 等价。

## 详细解答
对选项逐一分析：

**A.**  \(\frac{1}{x} < \frac{1}{y}\)  
∵ \(x,y>0\)，∴ 两边同乘 \(xy\) 得 \(y < x\)。  
该不等式等价于 \(x > y\)，是 \(x < y\) 的充要条件的反面，因此 **不是** 充要条件。

**B.**  \(x + \ln y < y + \ln x\)  
移项得 \(x - \ln x < y - \ln y\)。  
令 \(f(t) = t - \ln t\ (t>0)\)，则 \(f'(t) = 1 - \frac{1}{t}\)。  
当 \(0<t<1\) 时 \(f'(t)<0\)，\(f\) 递减；当 \(t>1\) 时 \(f'(t)>0\)，\(f\) 递增。  
\(f(t)\) 不单调，故 \(x<y\) 与 \(f(x)<f(y)\) 不等价。  
例如取 \(x=0.2,\ y=0.1\)，此时 \(x>y\) 但 \(f(0.2)\approx 1.81 < f(0.1)\approx 2.40\)，不等式成立而 \(x \nless y\)。  
因此 B **不是** 充要条件。

**C.**  \(\sin x < \sin y\)  
正弦函数在 \((0,+\infty)\) 上不单调。  
例如 \(x=\frac{\pi}{2},\ y=\pi\)，满足 \(x<y\)，但 \(\sin\frac{\pi}{2}=1 > 0=\sin\pi\)，原不等式不成立。  
因此 C **不是** 充要条件。

**D.**  \(x - \cos y < y - \cos x\)  
移项得 \(x + \cos x < y + \cos y\)。  
令 \(g(t) = t + \cos t\ (t>0)\)，则 \(g'(t) = 1 - \sin t \ge 0\)，且只在 \(t = \frac{\pi}{2}+2k\pi\) 处为 \(0\)，故 \(g(t)\) 在 \((0,+\infty)\) 上严格单调递增。  
于是 \(x < y \iff g(x) < g(y) \iff x + \cos x < y + \cos y \iff x - \cos y < y - \cos x\)。  
该选项同时满足充分性与必要性，是 \(x<y\) 的充要条件。

## 答案
D

## 总结
判断充要条件的关键是验证“\(p \Rightarrow q\)”与“\(q \Rightarrow p\)”是否同时成立。处理含变量大小比较的选项时，常通过移项构造辅助函数，利用导数讨论其单调性，这是确定充要条件的有效方法。注意区分类似 A 那样表面上类似实则不等价的表达式。
