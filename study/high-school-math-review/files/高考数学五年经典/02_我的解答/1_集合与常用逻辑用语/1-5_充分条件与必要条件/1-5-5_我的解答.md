# 1-5-5 我的解答

对应题目：[[01_题目/1_集合与常用逻辑用语/1-5_充分条件与必要条件/1-5-5.md|1-5-5]]

## 完整题目

5.(2023 广东广州一模)已知 \( p : \left( {x + 2}\right) \left( {x - 3}\right) \; < 0, q : \left| {x - 1}\right|  < 2 \) ,则 \( p \) 是 \( q \) 的 ( )

A. 充分不必要条件

B. 必要不充分条件

C. 充要条件

D. 既不充分也不必要条件

## 题目勘误
无明显勘误。
## 思路分析
分别解出条件 \(p\) 与条件 \(q\) 中 \(x\) 的取值范围，得到对应的集合 \(P\) 与 \(Q\)。根据集合间的包含关系判断条件之间的推出关系：若 \(P\subseteq Q\)，则 \(p\Rightarrow q\)（充分性成立）；若 \(Q\subseteq P\)，则 \(q\Rightarrow p\)（必要性成立）。最后确定 \(p\) 是 \(q\) 的何种条件。
## 详细解答
先解条件 \(p\)：
\[
p: (x+2)(x-3)<0,
\]
由一元二次不等式解法，得
\[
-2<x<3.
\]
记集合 \(P=(-2,3)\)。

再解条件 \(q\)：
\[
q: |x-1|<2,
\]
由绝对值不等式性质，得
\[
-2<x-1<2,
\]
即
\[
-1<x<3.
\]
记集合 \(Q=(-1,3)\)。

比较集合 \(P\) 与 \(Q\)：
- 因为 \(P=(-2,3)\)，\(Q=(-1,3)\)，显然 \(Q\subsetneqq P\)，\(P\) 并不是 \(Q\) 的子集。
- 由 \(Q\subsetneqq P\) 可知：\(q\Rightarrow p\)，但 \(p\nRightarrow q\)（例如取 \(x=-1.5\in P\)，但 \(x=-1.5\notin Q\)，满足 \(p\) 却不满足 \(q\)）。

因此 \(p\) 是 \(q\) 的必要不充分条件。
## 答案
B
## 总结
求解充分条件与必要条件问题的常用方法是“集合法”：将条件转化为集合，利用集合的包含关系直接判断。注意区分“\(p\) 是 \(q\) 的什么条件”与“\(q\) 是 \(p\) 的什么条件”，避免反向。本题中 \(Q\subsetneqq P\) 对应 \(q\Rightarrow p\)，故 \(p\) 是 \(q\) 的必要条件，且不充分。
