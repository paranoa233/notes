# 8-10-2 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-10_立体几何中的动态问题/8-10-2.md|8-10-2]]

## 完整题目

2. (多选题, 2023 山西 1 月调研) 在棱长为 1 的正方体 \( {ABCD} - {A}_{1}{B}_{1}{C}_{1}{D}_{1} \) 中,点 \( P \) 在侧面 \( C{C}_{1}{D}_{1}D \) (含边界) 内运动,点 \( Q \) 在底面 \( {ABCD} \) (含边界)内运动，则下列说法正确的是 ( )

A. 若直线 \( {BP} \) 与直线 \( {AD} \) 所成角为 \( {30}^{ \circ  } \) ，则点 \( P \) 的轨迹为圆弧

B. 若直线 \( {BP} \) 与平面 \( {ABCD} \) 所成角为 \( {30}^{ \circ  } \) ,则点 \( P \) 的轨迹为双曲线的一部分

C. 若 \( {D}_{1}Q = \frac{\sqrt{5}}{2} \) ,则点 \( Q \) 的轨迹为线段

D. 若点 \( Q \) 到直线 \( D{D}_{1} \) 的距离等于点 \( Q \) 到平面 \( {AB}{B}_{1}{A}_{1} \) 的距离，则点 \( Q \) 的轨迹为抛物线的一部分

## 题目勘误
无明显勘误。

## 思路分析
本题是正方体中的动态轨迹问题，涉及线线角、线面角、距离关系和点到平面距离等条件的转化。通过建立空间直角坐标系，将几何条件转化为代数方程，结合动点的区域限制（侧面或底面含边界），判断轨迹的具体形状（圆、双曲线、抛物线或线段）。对四个选项逐一分析验证即可。

## 详细解答

如图，以 \(D\) 为原点，建立空间直角坐标系，使 \(x\) 轴、\(y\) 轴、\(z\) 轴的正方向分别为 \(\overrightarrow{DA}\)、\(\overrightarrow{DC}\)、\(\overrightarrow{DD_1}\) 的方向。

（注：此处取 \(A(0,0,0)\) 亦可，不影响最终定性结论，为方便计算，采用常见建系方式：\(D(0,0,0), C(1,0,0), B(1,1,0), A(0,1,0)\)，\(D_1(0,0,1), C_1(1,0,1), B_1(1,1,1), A_1(0,1,1)\)。）

**正方体棱长为 1.**
- 点 \(P\) 在侧面 \(CC_1D_1D\)（含边界）内，该平面为 \(y = 0\)，且 \(0 \le x \le 1,\ 0 \le z \le 1\). 设 \(P(x, 0, z)\).
- 点 \(Q\) 在底面 \(ABCD\)（含边界）内，该平面为 \(z = 0\)，且 \(0 \le x \le 1,\ 0 \le y \le 1\). 设 \(Q(x, y, 0)\).

**选项 A**  
\(B(1,1,0)\)，\(\overrightarrow{BP} = (x-1,\ -1,\ z)\).  
直线 \(AD\) 过 \(A(0,1,0)\) 和 \(D(0,0,0)\)，方向向量取 \(\vec{v} = (0,1,0)\).  
设直线 \(BP\) 与 \(AD\) 所成角为 \(\theta = 30^\circ\)，
\[
\cos 30^\circ = \frac{|\overrightarrow{BP} \cdot \vec{v}|}{|\overrightarrow{BP}|\,|\vec{v}|} = \frac{|-1|}{\sqrt{(x-1)^2 + (-1)^2 + z^2}} = \frac{1}{\sqrt{(x-1)^2 + z^2 + 1}}.
\]
由 \(\cos 30^\circ = \frac{\sqrt{3}}{2}\)，平方得
\[
\frac{1}{(x-1)^2 + z^2 + 1} = \frac{3}{4} \implies (x-1)^2 + z^2 = \frac{1}{3}.
\]
该方程表示以 \((1,0)\) 为圆心、\(\frac{\sqrt{3}}{3}\) 为半径的圆，位于侧面区域 \(0 \le x \le 1,\ 0 \le z \le 1\) 内的部分是一段圆弧。故 **A 正确**.

**选项 B**  
底面 \(ABCD\) 方程为 \(z = 0\)，其法向量可取 \(\vec{n} = (0,0,1)\). 直线 \(BP\) 与平面所成角为 \(30^\circ\)，有
\[
\sin 30^\circ = \frac{|\overrightarrow{BP} \cdot \vec{n}|}{|\overrightarrow{BP}|\,|\vec{n}|} = \frac{|z|}{\sqrt{(x-1)^2 + 1 + z^2}}.
\]
因为 \(P\) 在侧面，\(z \ge 0\)，且 \(\sin 30^\circ = \frac{1}{2}\)，平方整理得
\[
\frac{1}{4} = \frac{z^2}{(x-1)^2 + 1 + z^2} \implies (x-1)^2 + 1 + z^2 = 4z^2 \implies 3z^2 - (x-1)^2 = 1.
\]
该方程为双曲线形式，考虑到 \(x \in [0,1], z \in [0,1]\)，轨迹为双曲线的一部分。故 **B 正确**.

**选项 C**  
\(D_1(0,0,1)\)，\(Q(x,y,0)\)，由 \(D_1Q = \frac{\sqrt{5}}{2}\) 得
\[
\sqrt{(x-0)^2 + (y-0)^2 + (0-1)^2} = \frac{\sqrt{5}}{2} \implies x^2 + y^2 + 1 = \frac{5}{4} \implies x^2 + y^2 = \frac{1}{4}.
\]
这是以原点为圆心、\(\frac{1}{2}\) 为半径的圆。\(Q\) 限制在正方形底面 \(0 \le x \le 1, 0 \le y \le 1\) 内，轨迹应为四分之一圆弧，不是线段。故 **C 错误**.

**选项 D**  
直线 \(DD_1\) 为 \(z\) 轴，过 \(D(0,0,0)\) 和 \(D_1(0,0,1)\). 点 \(Q(x,y,0)\) 到直线 \(DD_1\) 的距离为 \(\sqrt{x^2 + y^2}\).  
平面 \(ABB_1A_1\) 过 \(A(0,1,0), B(1,1,0)\) 等，方程为 \(y = 1\). 点 \(Q\) 到该平面的距离为 \(|y - 1|\)，因 \(y \in [0,1]\)，故距离为 \(1 - y\).  
依题意，
\[
\sqrt{x^2 + y^2} = 1 - y \implies x^2 + y^2 = (1 - y)^2 = 1 - 2y + y^2 \implies x^2 = 1 - 2y,
\]
整理得 \(y = -\frac{1}{2}x^2 + \frac{1}{2}\)，这是开口向下的抛物线。结合 \(x, y \in [0,1]\)，轨迹为抛物线的一部分。故 **D 正确**.

综上所述，正确选项为 A、B、D.

## 答案
ABD

## 总结
1. 立体几何动态问题中，建立适当的空间直角坐标系，将几何约束转化为代数方程是核心方法。
2. 必须注意动点的区域范围（含边界），轨迹是整体曲线落在区域内的那一部分。
3. 区分各种角的计算公式：线线角用余弦绝对值，线面角用正弦；点到平面距离和点到直线距离公式要正确使用。
4. 对于圆、椭圆、双曲线、抛物线的方程形式要熟悉，能通过方程判断轨迹类型。
