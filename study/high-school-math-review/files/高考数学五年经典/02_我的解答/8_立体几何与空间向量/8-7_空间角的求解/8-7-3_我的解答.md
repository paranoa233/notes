# 8-7-3 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-7_空间角的求解/8-7-3.md|8-7-3]]

## 完整题目

3. (多选题，2022 新高考卷 I ，9)已知正方体 \( {ABCD} - {A}_{1}{B}_{1}{C}_{1}{D}_{1} \) ，则 ( )

A. 直线 \( B{C}_{1} \) 与 \( D{A}_{1} \) 所成的角为 \( {90}^{ \circ  } \)

B. 直线 \( B{C}_{1} \) 与 \( C{A}_{1} \) 所成的角为 \( {90}^{ \circ  } \)

C. 直线 \( B{C}_{1} \) 与平面 \( B{B}_{1}{D}_{1}D \) 所成的角为 \( {45}^{ \circ  } \)

D. 直线 \( B{C}_{1} \) 与平面 \( {ABCD} \) 所成的角为 \( {45}^{ \circ  } \)

## 题目勘误
无明显勘误。

## 思路分析
本题是正方体背景下的空间角多选题。通过建立空间直角坐标系，将几何问题转化为向量坐标运算，可统一求解线线角与线面角；也可利用正方体中的垂直与平行关系进行几何推理。重点是对每个选项逐一验证角的大小。

## 详细解答

设正方体棱长为 1，以 \(D\) 为原点，分别以 \(\overrightarrow{DA},\overrightarrow{DC},\overrightarrow{DD_1}\) 为 \(x,y,z\) 轴正方向建立空间直角坐标系，则各点坐标如下：
\[
\begin{aligned}
&D(0,0,0),\; A(1,0,0),\; B(1,1,0),\; C(0,1,0),\\
&A_1(1,0,1),\; B_1(1,1,1),\; C_1(0,1,1),\; D_1(0,0,1).
\end{aligned}
\]

**A. 直线 \(BC_1\) 与 \(DA_1\) 所成的角**
\[
\overrightarrow{BC_1} = (-1,\,0,\,1),\quad \overrightarrow{DA_1} = (1,\,0,\,1).
\]
计算点积：
\[
\overrightarrow{BC_1}\cdot\overrightarrow{DA_1} = -1\times1 + 0 + 1\times1 = 0,
\]
∴ \(\overrightarrow{BC_1}\perp\overrightarrow{DA_1}\)，所成角为 \(90^\circ\)，**A 正确**。

**B. 直线 \(BC_1\) 与 \(CA_1\) 所成的角**
\[
\overrightarrow{CA_1} = (1-0,\,0-1,\,1-0) = (1,\,-1,\,1).
\]
点积：
\[
\overrightarrow{BC_1}\cdot\overrightarrow{CA_1} = -1\times1 + 0\times(-1) + 1\times1 = 0,
\]
∴ \(\overrightarrow{BC_1}\perp\overrightarrow{CA_1}\)，所成角为 \(90^\circ\)，**B 正确**。

（几何解释：\(BC_1\) 垂直于平面 \(A_1B_1CD\)，而 \(CA_1\) 在该平面内）

**C. 直线 \(BC_1\) 与平面 \(BB_1D_1D\) 所成的角**

平面 \(BB_1D_1D\) 的对角线 \(BD_1\) 方向，其法向量可取与 \(BD\) 垂直且与 \(BB_1\) 垂直的向量。显然 \(\overrightarrow{AC} = (-1,\,1,\,0)\) 为该平面的一个法向量。

设线面角为 \(\theta\)，则
\[
\sin\theta = \frac{|\overrightarrow{BC_1}\cdot\overrightarrow{AC}|}{|\overrightarrow{BC_1}||\overrightarrow{AC}|}.
\]
计算：
\[
\overrightarrow{BC_1}\cdot\overrightarrow{AC} = (-1)\times(-1) + 0\times1 + 1\times0 = 1,
\]
\[
|\overrightarrow{BC_1}| = \sqrt{2},\quad |\overrightarrow{AC}| = \sqrt{2},
\]
\[
\sin\theta = \frac{1}{\sqrt{2}\cdot\sqrt{2}} = \frac12 \quad\Rightarrow\quad \theta = 30^\circ \neq 45^\circ,
\]
**C 错误**。

**D. 直线 \(BC_1\) 与平面 \(ABCD\) 所成的角**

平面 \(ABCD\) 即底面 \(xOy\)，其法向量为 \(\vec{n}=(0,0,1)\)。
\[
\sin\varphi = \frac{|\overrightarrow{BC_1}\cdot\vec{n}|}{|\overrightarrow{BC_1}||\vec{n}|}
= \frac{|(-1,0,1)\cdot(0,0,1)|}{\sqrt{2}\times1} = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2},
\]
\(\varphi = 45^\circ\)。或由几何直观：\(BC_1\) 在底面的投影为 \(BC\)，\(\triangle BCC_1\) 为等腰直角三角形，\(\angle CBC_1=45^\circ\)，**D 正确**。

## 答案
ABD

## 总结
本题全面考查正方体中异面直线夹角与线面角的计算。建系统一求解可减少作图难度，向量坐标法对此类问题具有一般性与高效性；熟悉正方体中的垂直结构（如 \(BC_1\perp\) 平面 \(A_1B_1CD\)、\(BC_1\) 在底面的投影等）亦可快速推理，提高解题速度。注意线面角公式 \(\sin\theta = |\cos\langle\vec{v},\vec{n}\rangle|\)，避免记错。
