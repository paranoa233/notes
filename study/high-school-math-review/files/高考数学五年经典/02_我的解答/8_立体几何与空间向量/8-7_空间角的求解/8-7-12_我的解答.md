# 8-7-12 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-7_空间角的求解/8-7-12.md|8-7-12]]

## 完整题目

3.(2022全国甲卷理，7)在长方体 \( {ABCD} \) - \( {A}_{1}{B}_{1}{C}_{1}{D}_{1} \) 中，已知 \( {B}_{1}D \) 与平面 \( {ABCD} \) 和平面 \( A{A}_{1}{B}_{1}B \) 所成的角均为 \( {30}^{ \circ  } \) ,则 ( )

A. \( {AB} = {2AD} \)

B. \( {AB} \) 与平面 \( A{B}_{1}{C}_{1}D \) 所成的角为 \( {30}^{ \circ  } \)

C. \( {AC} = C{B}_{1} \)

D. \( {B}_{1}D \) 与平面 \( B{B}_{1}{C}_{1}C \) 所成的角为 \( {45}^{ \circ  } \)

## 题目勘误
无明显勘误。

## 思路分析
根据已知条件，利用线面角的定义转化为直角三角形的边角关系，求出长方体棱长之间的关系。然后针对各选项逐一计算或论证，判断正误。涉及长方体的几何性质、线面角的定义与计算、空间几何中的距离与角度。

## 详细解答
设长方体的棱长 \(AD = a\)，\(AB = b\)，\(AA_1 = c\)。

已知 \(B_1D\) 与平面 \(ABCD\) 所成的角为 \(30^\circ\)，  
连接 \(BD\)，则 \(BD\) 是 \(B_1D\) 在平面 \(ABCD\) 内的射影，  
所以 \(\angle B_1DB = 30^\circ\)。  
在 \(\text{Rt}\triangle B_1DB\) 中，\(\tan 30^\circ = \dfrac{BB_1}{BD} = \dfrac{c}{\sqrt{a^2 + b^2}}\)，  
得 \(c = \dfrac{\sqrt{3}}{3} \sqrt{a^2 + b^2}\)。   
又因为 \(\sin 30^\circ = \dfrac{BB_1}{B_1D} = \dfrac{1}{2}\)，所以 \(B_1D = 2c\)。

又已知 \(B_1D\) 与平面 \(AA_1B_1B\) 所成的角为 \(30^\circ\)，  
过点 \(D\) 作平面 \(AA_1B_1B\) 的垂线，垂足为 \(A\)（因为 \(AD \perp AB\)，\(AD \perp AA_1\)），  
连接 \(AB_1\)，则 \(AB_1\) 是 \(B_1D\) 在该平面内的射影，  
所以 \(\angle AB_1D = 30^\circ\)（或利用 \(\sin\) 值）。  
在 \(\text{Rt}\triangle AB_1D\) 中，\(\sin 30^\circ = \dfrac{AD}{B_1D} = \dfrac{a}{2c} = \dfrac{1}{2}\)，  
所以 \(a = c\)。  
代入 \(c = \dfrac{\sqrt{3}}{3} \sqrt{a^2 + b^2}\)，得 \(a = \dfrac{\sqrt{3}}{3} \sqrt{a^2 + b^2}\)，  
两边平方得 \(a^2 = \dfrac{1}{3}(a^2 + b^2)\)，  
整理得 \(3a^2 = a^2 + b^2\)，即 \(b^2 = 2a^2\)，所以 \(b = \sqrt{2}a\)。

即长方体的棱长分别为：\(AD = a\)，\(AB = \sqrt{2}a\)，\(AA_1 = a\)。

下面逐项判断：

**A.** \(AB = \sqrt{2}a\)，\(2AD = 2a\)，显然 \(\sqrt{2}a \neq 2a\)，选项 A 错误。

**B.** 求 \(AB\) 与平面 \(AB_1C_1D\) 所成的角。  
平面 \(AB_1C_1D\) 过点 \(A(0,0,0)\)、\(B_1(\sqrt{2}a,0,a)\)、\(C_1(\sqrt{2}a,a,a)\)、\(D(0,a,0)\)。  
向量 \(\overrightarrow{AB_1} = (\sqrt{2}a, 0, a)\)，\(\overrightarrow{AD} = (0, a, 0)\)，  
设平面法向量为 \(\boldsymbol{n}\)，则  
\(\boldsymbol{n} = \overrightarrow{AB_1} \times \overrightarrow{AD} = (-a^2, 0, \sqrt{2}a^2)\)，可取 \(\boldsymbol{n} = (-1, 0, \sqrt{2})\)。  
直线 \(AB\) 的方向向量为 \((1,0,0)\)，  
设 \(AB\) 与平面所成角为 \(\theta\)，则  
\(\sin \theta = |\cos \langle (1,0,0), \boldsymbol{n} \rangle| = \dfrac{|-1|}{\sqrt{1^2 + (\sqrt{2})^2}} = \dfrac{1}{\sqrt{3}}\)。  
\(\sin 30^\circ = \dfrac{1}{2} \neq \dfrac{1}{\sqrt{3}}\)，所以 \(\theta \neq 30^\circ\)，选项 B 错误。

**C.** \(AC\) 是底面矩形的对角线，  
\(AC = \sqrt{AB^2 + AD^2} = \sqrt{(\sqrt{2}a)^2 + a^2} = \sqrt{3}a\)。  
\(CB_1\) 中，\(C(\sqrt{2}a, a, 0)\)，\(B_1(\sqrt{2}a, 0, a)\)，  
\(CB_1 = \sqrt{(\sqrt{2}a - \sqrt{2}a)^2 + (a-0)^2 + (0-a)^2} = \sqrt{2}a\)。  
\(\sqrt{3}a \neq \sqrt{2}a\)，选项 C 错误。

**D.** 求 \(B_1D\) 与平面 \(BB_1C_1C\) 所成的角。  
平面 \(BB_1C_1C\) 即平面 \(x = \sqrt{2}a\)（以 \(A\) 为原点建系）。  
点 \(D(0,a,0)\) 到平面的垂足为 \(C(\sqrt{2}a, a, 0)\)，  
\(B_1D\) 在平面内的射影为 \(B_1C\)，  
所以线面角即为 \(\angle DB_1C\)。  
计算边长：\(B_1C = \sqrt{(\sqrt{2}a - \sqrt{2}a)^2 + (a-0)^2 + (0-a)^2} = \sqrt{2}a\)，  
\(CD = |\sqrt{2}a - 0| = \sqrt{2}a\)，  
\(B_1D = \sqrt{(\sqrt{2}a)^2 + a^2 + a^2} = \sqrt{4a^2} = 2a\)。  
满足 \(B_1C^2 + CD^2 = 2a^2 + 2a^2 = 4a^2 = B_1D^2\)，  
所以 \(\triangle B_1CD\) 是等腰直角三角形，\(\angle DB_1C = 45^\circ\)。  
因此 \(B_1D\) 与平面 \(BB_1C_1C\) 所成的角为 \(45^\circ\)，选项 D 正确。

## 答案
D

## 总结
本题以长方体为载体考查线面角的概念与计算，关键是把文字条件转化为棱长关系，确定长方体的具体形状（棱长比 \(AD:AA_1:AB = 1:1:\sqrt{2}\)），然后通过建系或几何推理逐一验证选项。计算时要注意射影的准确寻找及空间直角坐标系中向量法求线面角的正弦公式。
