# 8-3-11 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-3_简单几何体的表面积问题/8-3-11.md|8-3-11]]

## 完整题目

2.(2022 新高考卷 II , 7)已知正三棱台的高为 1 ， 上、下底面边长分别为 \( 3\sqrt{3} \) 和 \( 4\sqrt{3} \) ，其顶点都在同一球面上，则该球的表面积为 ( )

A. \( {100\pi } \) B. \( {128\pi } \)

C. \( {144\pi } \) D. \( {192\pi } \)

## 题目勘误
无明显勘误。

## 思路分析
正三棱台的顶点都在同一球面上，则该球为其外接球。由正三棱台的对称性，外接球的球心必在上下底面中心的连线上。设出球心到上（或下）底面中心的距离，利用球心到各顶点距离等于半径 \(R\)，结合上下底面的外接圆半径和台体的高建立方程，求出 \(R^2\)，进而得到球的表面积。

## 详细解答
设正三棱台上、下底面均为正三角形，其边长分别为 \(a_1 = 3\sqrt{3}\)，\(a_2 = 4\sqrt{3}\)，高 \(h = 1\)。
正三角形的外接圆半径公式为 \(r = \frac{a}{\sqrt{3}}\)，所以：
上底面外接圆半径 \(r_1 = \frac{3\sqrt{3}}{\sqrt{3}} = 3\)，
下底面外接圆半径 \(r_2 = \frac{4\sqrt{3}}{\sqrt{3}} = 4\)。

由于正三棱台的所有顶点都在同一球面上，由对称性可知球心 \(O\) 在上下底面中心 \(O_1, O_2\) 的连线上。
以 \(O_2\) 为原点，直线 \(O_2O_1\) 为轴（向上为正方向），设球心 \(O\) 的坐标为 \(x\)，则球心 \(O\) 到下底面中心 \(O_2\) 的距离为 \(|x|\)，到上底面中心 \(O_1\) 的距离为 \(|x - h|\)。
球半径 \(R\) 满足：
\[ R^2 = r_2^2 + x^2 = r_1^2 + (x - h)^2 \]
代入已知数据：
\[ 4^2 + x^2 = 3^2 + (x - 1)^2 \]
\[ 16 + x^2 = 9 + x^2 - 2x + 1 \]
化简得：
\[ 16 = 10 - 2x \]
\[ 2x = -6 \quad \Rightarrow \quad x = -3 \]
因此球心 \(O\) 在 \(O_2\) 下方距 \(O_2\) 为 \(3\) 处。
\[ R^2 = r_2^2 + x^2 = 16 + (-3)^2 = 25 \]
外接球的表面积 \(S = 4\pi R^2 = 4\pi \times 25 = 100\pi\)。

## 答案
A. \( {100\pi } \)

## 总结
求解正棱台（或正棱柱、正棱锥）的外接球问题，关键是利用对称性确定球心在上下底面中心的连线上，再通过建立球心到各顶点距离相等的方程求解半径。本题中需注意球心可能在台体外部，通过代数坐标可避免对位置关系的分类讨论。
