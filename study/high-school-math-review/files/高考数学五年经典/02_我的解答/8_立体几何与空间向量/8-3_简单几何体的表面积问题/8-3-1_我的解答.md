# 8-3-1 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-3_简单几何体的表面积问题/8-3-1.md|8-3-1]]

## 完整题目

1. (2024 重庆主城二测)民间娱乐健身工具陀螺起源于我国，最早出土的石制陀螺在山西夏县的新石器时代遗址中发现. 如图, 这是一个陀螺的立体结构图 (上端是圆柱,下端是圆锥). 已知底面圆的直径 \( {AB} = \) 8,圆柱体的高 \( {BC} = 5 \) ,圆锥体的高 \( {CD} = 3 \) ,则这个陀螺的表面积为 ( )

![[05_附件/images/121_629_683_205_248_0.jpg]]

A. \( {60\pi } \) B. \( {76\pi } \)

C. \( {92\pi } \) D. \( {96\pi } \)

## 题目勘误
无明显勘误。
## 思路分析
陀螺由上部圆柱和下部圆锥拼接而成，两者共用底面。陀螺的表面积为外露表面之和，包括圆柱的上底面、圆柱的侧面以及圆锥的侧面（拼合处的圆面重合不算表面积）。已知底面圆的直径，可求半径；已知圆柱高和圆锥高，可分别求圆柱侧面积、底面积以及圆锥母线长和侧面积，最后加总即可。
## 详细解答
由题意，陀螺结构是圆柱在上、圆锥在下，两部分共用一个底面圆，该圆面不在表面内。
已知底面圆的直径 \( AB = 8 \)，故底面半径  
\[
r = \frac{8}{2} = 4.
\]
圆柱高 \( BC = 5 \)，圆锥高 \( CD = 3 \)。

**1. 求圆柱部分表面积（外露部分）**  
圆柱侧面积：  
\[
S_{\text{柱侧}} = 2\pi r \cdot BC = 2\pi \times 4 \times 5 = 40\pi.
\]
圆柱上底面积（顶部外露）：  
\[
S_{\text{底}} = \pi r^2 = \pi \times 4^2 = 16\pi.
\]

**2. 求圆锥部分侧面积**  
先求圆锥母线长 \( l \)：  
\[
l = \sqrt{r^2 + CD^2} = \sqrt{4^2 + 3^2} = \sqrt{16 + 9} = \sqrt{25} = 5.
\]
圆锥侧面积：  
\[
S_{\text{锥侧}} = \pi r l = \pi \times 4 \times 5 = 20\pi.
\]

**3. 陀螺总表面积**  
\[
S_{\text{总}} = S_{\text{柱侧}} + S_{\text{底}} + S_{\text{锥侧}} = 40\pi + 16\pi + 20\pi = 76\pi.
\]

## 答案
\( \boxed{\text{B}} \)

## 总结
求解组合体表面积，关键在于分清哪些面是外表面。本题中圆柱与圆锥拼合处的圆面不计入表面积，其余三个面直接相加。通过圆锥的母线长公式 \( l = \sqrt{r^2 + h^2} \)，再结合柱体与锥体侧面积公式即可快速求解。
