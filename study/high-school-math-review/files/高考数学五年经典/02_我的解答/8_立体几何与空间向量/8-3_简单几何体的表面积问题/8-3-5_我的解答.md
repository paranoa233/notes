# 8-3-5 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-3_简单几何体的表面积问题/8-3-5.md|8-3-5]]

## 完整题目

5.(2024 江苏南京二模)在圆台 \( {O}_{1}{O}_{2} \) 中，圆 \( {O}_{2} \) 的半径是圆 \( {O}_{1} \) 半径的 2 倍,且 \( {O}_{2} \) 恰为该圆台外接球的球心,则圆台的侧面积与球的表面积之比为 ( )

A. \( 3 : 4 \) B. \( 1 : 2 \) C. \( 3 : 8 \) D. \( 3 : {10} \)

## 题目勘误
无明显勘误。
## 思路分析
设圆 \(O_{1}\) 的半径为 \(r\)，则圆 \(O_{2}\) 的半径为 \(2r\)。由 \(O_{2}\) 为圆台外接球的球心，可得球半径 \(R\) 等于圆 \(O_{2}\) 的半径。利用球心到圆台底面和顶面圆周的距离均为 \(R\)，可求出圆台的高和母线长，进而求得圆台侧面积和球表面积，得出比值。
## 详细解答
设圆 \(O_{1}\) 的半径为 \(r\)，因为圆 \(O_{2}\) 的半径是圆 \(O_{1}\) 半径的 2 倍，所以圆 \(O_{2}\) 的半径 \(r_{2}=2r\)。

由 \(O_{2}\) 恰为圆台外接球的球心，可知外接球半径 \(R = r_{2}=2r\)。

连接 \(O_{2}\) 与圆 \(O_{1}\) 圆周上任意一点 \(A\)，则 \(O_{2}A = R = 2r\)。
又 \(O_{1}A = r\)，且 \(O_{1}O_{2}\) 为圆台的高 \(h\)，在 \(\text{Rt}\triangle O_{1}AO_{2}\) 中，由勾股定理：
\[
h = O_{1}O_{2} = \sqrt{O_{2}A^{2} - O_{1}A^{2}} = \sqrt{(2r)^{2} - r^{2}} = \sqrt{3}r
\]

圆台的母线长 \(l\) 满足：
\[
l = \sqrt{h^{2} + (r_{2} - r)^{2}} = \sqrt{(\sqrt{3}r)^{2} + (2r - r)^{2}} = \sqrt{3r^{2} + r^{2}} = 2r
\]

圆台的侧面积：
\[
S_{\text{侧}} = \pi (r + r_{2}) \, l = \pi (r + 2r) \cdot 2r = 6\pi r^{2}
\]

外接球的表面积：
\[
S_{\text{球}} = 4\pi R^{2} = 4\pi (2r)^{2} = 16\pi r^{2}
\]

所求比值为：
\[
S_{\text{侧}} : S_{\text{球}} = 6\pi r^{2} : 16\pi r^{2} = 3 : 8
\]
## 答案
C
## 总结
本题主要考查圆台与外接球的几何关系，关键是由球心位置确定球的半径与圆台下底半径相同，并利用轴截面中的直角三角形求出高和母线长，进而计算侧面积和表面积之比。
