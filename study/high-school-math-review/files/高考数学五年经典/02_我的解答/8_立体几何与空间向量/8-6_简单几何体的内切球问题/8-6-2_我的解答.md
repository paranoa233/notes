# 8-6-2 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-6_简单几何体的内切球问题/8-6-2.md|8-6-2]]

## 完整题目

2.(2023 河北石家庄二检)已知圆台的上、下底面圆的半径之比为 \( \frac{1}{2} \) ,侧面积为 \( {9\pi } \) ,在圆台的内部有一球 \( O \) ,该球与圆台的上、下底面及母线均相切,则球 \( O \) 的表面积为 0

A. \( {3\pi } \) B. \( {5\pi } \)

C. \( {8\pi } \) D. \( {9\pi } \)

## 题目勘误
题目末尾“则球 O 的表面积为 0”中，“0”疑似为选项填空标记或标点识别错误，应为“（ ）”，不影响实际解题。
## 思路分析
圆台的内切球问题通常转化为轴截面等腰梯形的内切圆来研究。设上、下底面半径，利用半径比和侧面积，结合等腰梯形存在内切圆的充要条件（两腰之和等于两底之和）求出几何量，再根据内切球直径等于圆台的高求出球的半径，进而求表面积。
## 详细解答
设圆台上底面半径为 \(r\)，下底面半径为 \(2r\)，母线长为 \(l\)。  
由侧面积公式：  
\[
S_{\text{侧}} = \pi (r + 2r) l = 3\pi r l = 9\pi
\]  
解得  
\[
r l = 3 \tag{1}
\]  

因为球 \(O\) 与圆台的上、下底面及母线均相切，该球是圆台的内切球。考虑圆台的轴截面，它是一个等腰梯形，上底长为 \(2r\)，下底长为 \(4r\)，两腰长为 \(l\)。等腰梯形存在内切圆（球的大圆）的充要条件是两腰之和等于两底之和，即  
\[
2r + 4r = 2l \implies 6r = 2l \implies l = 3r \tag{2}
\]  

联立 (1)(2)：  
\[
r \cdot 3r = 3 \implies 3r^2 = 3 \implies r = 1
\]  
则下底面半径为 \(2\)，母线长 \(l = 3\)。  

轴截面等腰梯形的高 \(h\) 即为圆台的高，也是内切球的直径：  
\[
h = \sqrt{l^2 - (2r - r)^2} = \sqrt{3^2 - 1^2} = \sqrt{8} = 2\sqrt{2}
\]  
内切球半径  
\[
R = \frac{h}{2} = \sqrt{2}
\]  
球的表面积  
\[
S = 4\pi R^2 = 4\pi (\sqrt{2})^2 = 8\pi
\]  

选项 C 符合。
## 答案
C
## 总结
处理圆台内切球问题的核心是抓住轴截面等腰梯形，利用内切圆条件建立母线长与上、下底半径的关系，再结合已知的侧面积等条件联立求解。本题属于经典模型，计算量适中。
