# 8-5-8 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-5_简单几何体的外接球问题/8-5-8.md|8-5-8]]

## 完整题目

8.(2023 福建福州二检)已知三棱锥 P-ABC 的四个顶点都在球 \( O \) 的球面上, \( {PA} = {PB} = {PC} = \; {AB} = \sqrt{6},\angle {ACB} = \frac{2\pi }{3} \) ，则球 \( O \) 的体积为( )

A. \( {3\pi } \) B. \( \frac{27\pi }{8} \) C. \( \frac{9\pi }{2} \) D. \( {9\pi } \)

## 题目勘误
无明显勘误。

## 思路分析
1. 由 \(PA = PB = PC\) 可知，点 \(P\) 在底面 \(\triangle ABC\) 所在平面上的射影是 \(\triangle ABC\) 的外心 \(O'\)。
2. 在 \(\triangle ABC\) 中，已知 \(AB = \sqrt{6}\)，\(\angle ACB = \frac{2\pi}{3}\)，利用正弦定理可求得外接圆半径 \(r = O'A\)。
3. 在 \( {\rm Rt}\triangle PO'A\) 中，利用 \(PA = \sqrt{6}\) 和 \(r\) 求出棱锥的高 \(h = PO'\)。
4. 三棱锥的外接球球心 \(O\) 在直线 \(PO'\) 上，设球心到底面的距离为 \(d\)，利用球心到顶点 \(P\) 和到底面顶点 \(A\) 的距离相等建立方程，求出球半径 \(R\)。
5. 代入球的体积公式 \(V = \frac{4}{3}\pi R^3\) 得到答案。

## 详细解答
**第一步：求底面 \(\triangle ABC\) 的外接圆半径 \(r\)**
在 \(\triangle ABC\) 中，由正弦定理：
\[
\frac{AB}{\sin \angle ACB} = 2r
\]
代入 \(AB = \sqrt{6}\)，\(\angle ACB = \frac{2\pi}{3}\)（\(\sin\frac{2\pi}{3} = \frac{\sqrt{3}}{2}\)）：
\[
2r = \frac{\sqrt{6}}{\frac{\sqrt{3}}{2}} = \frac{2\sqrt{6}}{\sqrt{3}} = 2\sqrt{2}
\]
所以
\[
r = \sqrt{2}.
\]

**第二步：求棱锥的高 \(h\)**
设 \(O'\) 为 \(\triangle ABC\) 的外心，则 \(O'A = r = \sqrt{2}\)。
因为 \(PA = PB = PC\)，所以 \(PO' \perp\) 平面 \(ABC\)，即 \(PO'\) 为棱锥的高 \(h\)。
在 \({\rm Rt}\triangle PO'A\) 中：
\[
h = PO' = \sqrt{PA^2 - O'A^2} = \sqrt{(\sqrt{6})^2 - (\sqrt{2})^2} = \sqrt{6 - 2} = 2.
\]

**第三步：确定外接球球心 \(O\) 的位置并求半径 \(R\)**
三棱锥的外接球球心 \(O\) 必在直线 \(PO'\) 上。设 \(O\) 到底面 \(ABC\) 的距离为 \(d\)（即 \(O\) 在底面所在平面的垂足为 \(O'\)，\(OO' = d\)），则：
- 球心 \(O\) 到点 \(A\) 的距离：\(R = \sqrt{d^2 + r^2}\)；
- 球心 \(O\) 到点 \(P\) 的距离：因为 \(P\) 与底面距离为 \(h\)，且 \(O\) 在 \(PO'\) 上，所以 \(R = |h - d|\)。

由 \(R\) 相等得：
\[
(h - d)^2 = d^2 + r^2
\]
展开：
\[
h^2 - 2hd + d^2 = d^2 + r^2 \implies h^2 - 2hd = r^2
\]
解得：
\[
d = \frac{h^2 - r^2}{2h} = \frac{2^2 - (\sqrt{2})^2}{2 \times 2} = \frac{4 - 2}{4} = \frac{1}{2}.
\]
代入求半径：
\[
R = h - d = 2 - \frac{1}{2} = \frac{3}{2} \quad \text{或} \quad R = \sqrt{d^2 + r^2} = \sqrt{\left(\frac{1}{2}\right)^2 + 2} = \sqrt{\frac{9}{4}} = \frac{3}{2}.
\]

**第四步：求球的体积**
\[
V = \frac{4}{3}\pi R^3 = \frac{4}{3}\pi \left(\frac{3}{2}\right)^3 = \frac{4}{3}\pi \cdot \frac{27}{8} = \frac{9}{2}\pi.
\]

## 答案
C. \(\frac{9\pi}{2}\)

## 总结
本题关键有两点：一是利用侧棱相等得出顶点在底面的射影为底面三角形的外心；二是通过正弦定理将已知角的条件转化为外接圆半径，再结合“垂线型”外接球模型求半径。熟练掌握 \(h,d,r\) 三者间的关系式 \(h^2 - 2hd = r^2\)（或直接解二次方程）是快速求解此类问题的核心。
