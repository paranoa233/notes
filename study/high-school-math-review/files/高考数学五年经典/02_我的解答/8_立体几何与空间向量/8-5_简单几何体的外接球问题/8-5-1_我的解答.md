# 8-5-1 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-5_简单几何体的外接球问题/8-5-1.md|8-5-1]]

## 完整题目

1.(2023 福建厦门等七市一检)在正四棱台 \( {ABCD} - {A}_{1}{B}_{1}{C}_{1}{D}_{1} \) 中, \( {AB} = {2A}{A}_{1} = 2{A}_{1}{B}_{1} = \; 2\sqrt{2} \) ，且各顶点都在同一球面上，则该球体的表面积为 ( )

A. \( {20\pi } \) B. \( 5\sqrt{5}\pi \) C. \( {10\pi } \) D. \( {5\pi } \)

## 题目勘误
无明显勘误。

## 思路分析
正四棱台的外接球球心必在上下底面中心的连线上。由已知的棱长关系可求出上下底面的边长、侧棱长，进而求出棱台的高。建立空间直角坐标系或利用几何关系，在中心连线上设球心坐标，根据球心到上、下底面顶点的距离相等列方程，解出球心位置和半径，最后计算球的表面积。

## 详细解答

设正四棱台 \(ABCD-A_1B_1C_1D_1\)，其中 \(ABCD\) 为下底面，\(A_1B_1C_1D_1\) 为上底面。  
由已知 \(AB = 2AA_1 = 2A_1B_1 = 2\sqrt{2}\)，得
\[
AB = 2\sqrt{2},\quad AA_1 = \sqrt{2},\quad A_1B_1 = \sqrt{2}.
\]
即下底面正方形边长为 \(2\sqrt{2}\)，上底面正方形边长为 \(\sqrt{2}\)，侧棱长为 \(\sqrt{2}\)。

**1. 求棱台的高 \(h\)**  
下底面对角线一半（外接圆半径）：
\[
R_d = \frac{\sqrt{(2\sqrt{2})^2 + (2\sqrt{2})^2}}{2} = \frac{\sqrt{16}}{2} = 2.
\]
上底面对角线一半：
\[
R_u = \frac{\sqrt{(\sqrt{2})^2 + (\sqrt{2})^2}}{2} = \frac{\sqrt{4}}{2} = 1.
\]
侧面为等腰梯形，上底 \(\sqrt{2}\)，下底 \(2\sqrt{2}\)，腰 \(\sqrt{2}\)。斜高 \(h'\)：
\[
h' = \sqrt{AA_1^2 - \left(\frac{AB - A_1B_1}{2}\right)^2}
= \sqrt{(\sqrt{2})^2 - \left(\frac{\sqrt{2}}{2}\right)^2}
= \sqrt{2 - \frac{1}{2}} = \frac{\sqrt{6}}{2}.
\]
下底面边心距 \(d = \frac{AB}{2} = \sqrt{2}\)，上底面边心距 \(d_1 = \frac{A_1B_1}{2} = \frac{\sqrt{2}}{2}\)。  
棱台的高 \(h\) 满足：
\[
h^2 + (d - d_1)^2 = h'^2
\]
\[
h^2 + \left(\sqrt{2} - \frac{\sqrt{2}}{2}\right)^2 = \left(\frac{\sqrt{6}}{2}\right)^2
\]
\[
h^2 + \left(\frac{\sqrt{2}}{2}\right)^2 = \frac{6}{4}
\]
\[
h^2 + \frac{1}{2} = \frac{3}{2} \implies h^2 = 1 \implies h = 1.
\]

**2. 确定球心位置与半径**  
设下底面中心为 \(O\)，上底面中心为 \(O_1\)，则 \(OO_1 \perp\) 底面，且 \(OO_1 = h = 1\)。  
球心 \(P\) 在直线 \(OO_1\) 上，设 \(P\) 与 \(O\) 的距离为 \(z\)（若 \(P\) 在 \(O\) 下方，\(z<0\)）。以 \(O\) 为原点建立坐标系。  
球的半径 \(R\) 满足：
\[
R^2 = z^2 + R_d^2 = z^2 + 4,
\]
上底面顶点到球心距离：
\[
R^2 = (1 - z)^2 + R_u^2 = (1 - z)^2 + 1.
\]
联立两式：
\[
z^2 + 4 = (1 - z)^2 + 1
\]
\[
z^2 + 4 = 1 - 2z + z^2 + 1
\]
\[
4 = -2z + 2 \implies 2z = -2 \implies z = -1.
\]
代回得：
\[
R^2 = (-1)^2 + 4 = 5 \implies R = \sqrt{5}.
\]

**3. 求球的表面积**  
\[
S = 4\pi R^2 = 4\pi \times 5 = 20\pi.
\]

## 答案
A. \(20\pi\)

## 总结
解决正四棱台外接球问题的关键是确定球心在上下底面中心连线上。通过侧面梯形关系求棱台的高，利用球心到多边形顶点的距离相等建立方程，即可解得球半径。本题中计算得的棱台高恰好与下底面对角线一半形成直角三角形，使球心落在底面下方，最终半径为 \(\sqrt{5}\)，表面积为 \(20\pi\)。
