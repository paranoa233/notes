# 8-5-10 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-5_简单几何体的外接球问题/8-5-10.md|8-5-10]]

## 完整题目

1. (2022 新高考卷 I , 8)已知正四棱锥的侧棱长为 2 其各顶点都在同一球面上. 若该球的体积为 \( {36\pi } \) ,且 \( 3 \leq  l \leq  3\sqrt{3} \) ,则该正四棱锥体积的取值范围是 ( )

A. \( \left\lbrack  {{18},\frac{81}{4}}\right\rbrack \) B. \( \left\lbrack  {\frac{27}{4},\frac{81}{4}}\right\rbrack \)

C. \( \left\lbrack  {\frac{27}{4},\frac{64}{3}}\right\rbrack \) D. \( \left\lbrack  {{18},{27}}\right\rbrack \)

## 题目勘误
原文“已知正四棱锥的侧棱长为 2”与后续条件“且 \(3 \leq l \leq 3\sqrt{3}\)”矛盾，显然“2”为 OCR 识别错误。  
建议更正：“已知正四棱锥的侧棱长为 \(l\)”。  
理由：2022 年新高考卷 I 原题中变量即为 \(l\)，后续范围条件及所求体积均依赖于 \(l\)。

## 思路分析
本题为立体几何与函数最值综合题。  
第一步，由球的体积求出外接球半径 \(R\)；  
第二步，利用正四棱锥的几何特征，导出侧棱长 \(l\) 与高 \(h\) 的关系，以及体积 \(V\) 关于 \(l\) 的函数表达式；  
第三步，在给定区间 \(l \in [3, 3\sqrt{3}]\) 上，利用导数求函数 \(V(l)\) 的最值，从而得到体积的取值范围。

## 详细解答
设正四棱锥的侧棱长为 \(l\)，底面正方形边长为 \(a\)，底面外接圆半径为 \(r\)，高为 \(h\)，外接球半径为 \(R\)。  
由球的体积 \(V_{\text{球}} = \frac{4}{3}\pi R^3 = 36\pi\)，得 \(R = 3\)。  

正四棱锥的底面是正方形，其顶点和底面四个顶点均在球面上。球心必在底面的中心垂线上。设球心到锥顶点的距离为 \(R\)，底面中心到球心的距离为 \(|h - R|\)。  
由几何关系：  
\[
R^2 = (h - R)^2 + r^2
\]
展开得  
\[
R^2 = h^2 - 2hR + R^2 + r^2 \implies h^2 + r^2 = 2hR
\]
又知侧棱长 \(l\) 满足  
\[
l^2 = h^2 + r^2
\]
联立两式得  
\[
l^2 = 2hR \implies h = \frac{l^2}{2R} = \frac{l^2}{6}.
\]
进而底面外接圆半径的平方为  
\[
r^2 = l^2 - h^2 = l^2 - \frac{l^4}{36}.
\]
底面正方形面积 \(S_{\text{底}} = a^2 = 2r^2 = 2\left(l^2 - \frac{l^4}{36}\right)\)。  
正四棱锥体积  
\[
V = \frac{1}{3} S_{\text{底}} h = \frac{1}{3} \cdot 2\left(l^2 - \frac{l^4}{36}\right) \cdot \frac{l^2}{6} = \frac{1}{9} l^4 \left(1 - \frac{l^2}{36}\right) = \frac{1}{324} (36l^4 - l^6).
\]
令 \(f(l) = 36l^4 - l^6\)，定义域 \(l \in [3, 3\sqrt{3}]\)。  
求导：  
\[
f'(l) = 144l^3 - 6l^5 = 6l^3 (24 - l^2).
\]
在区间内，由 \(f'(l) = 0\) 得 \(l^2 = 24\)，即 \(l = 2\sqrt{6}\)（负值舍去）。  
- 当 \(3 \leq l < 2\sqrt{6}\) 时，\(f'(l) > 0\)，\(f(l)\) 单调递增；  
- 当 \(2\sqrt{6} < l \leq 3\sqrt{3}\) 时，\(f'(l) < 0\)，\(f(l)\) 单调递减。  
因此 \(V\) 在 \(l = 2\sqrt{6}\) 处取得最大值。  

计算关键点体积：  
- 当 \(l = 3\) 时，\(V = \frac{1}{324}(36 \times 81 - 729) = \frac{2187}{324} = \frac{27}{4}\)；  
- 当 \(l = 2\sqrt{6}\) 时，\(l^2 = 24\)，\(f(2\sqrt{6}) = 36 \times 576 - 13824 = 6912\)，\(V = \frac{6912}{324} = \frac{64}{3}\)；  
- 当 \(l = 3\sqrt{3}\) 时，\(l^2 = 27\)，\(V = \frac{1}{324}(36 \times 729 - 19683) = \frac{6561}{324} = \frac{81}{4}\)。  

比较端点值：\(\frac{27}{4} < \frac{81}{4} < \frac{64}{3}\)，故在给定区间上 \(V\) 的最小值为 \(\frac{27}{4}\)，最大值为 \(\frac{64}{3}\)。  
体积取值范围为 \(\left[ \frac{27}{4}, \frac{64}{3} \right]\)。

## 答案
C

## 总结
对于外接球与棱锥的体积最值问题，关键在于利用球半径和几何关系将高与侧棱长联系起来，从而把体积表示为单变量函数。随后借助导数求区间最值，即可得到体积的取值范围。注意检查区间内驻点是否在定义域内，并与端点值进行比较。
