# 8-5-3 我的解答

对应题目：[[01_题目/8_立体几何与空间向量/8-5_简单几何体的外接球问题/8-5-3.md|8-5-3]]

## 完整题目

3.(2020 天津市耀华中学月考)在《九章算术》 中，将底面为长方形且有一条侧棱与底面垂直的四棱锥称为 “阳马”. 现有一 “阳马” \( P - {ABCD} \) , \( {PA} \bot \) 底面 \( {ABCD},{PA} = {AB} = 2,{AD} = 1 \) ,则该 “阳马”的最长棱长为___，外接球表面积为 ___.

## 题目勘误
无明显勘误。

## 思路分析
题目给出了一个“阳马”的几何体，即底面为长方形且有一条侧棱垂直于底面的四棱锥。已知 \(PA \perp \) 底面 \(ABCD\)，\(PA=AB=2\)，\(AD=1\)。要求最长棱长和外接球表面积。

对于最长棱长，可以通过建立空间直角坐标系或利用勾股定理依次计算各条棱的长度，比较得出最大值。

对于外接球，因为该阳马有一条侧棱垂直于底面，且底面为长方形，可以考虑将其补形为长方体。该长方体的长、宽、高分别为 \(AB\)、\(AD\)、\(PA\)，此时阳马的外接球即为该长方体的外接球，其直径等于长方体的体对角线长，从而求得半径和表面积。

## 详细解答

**1. 求最长棱长**

由题意知，底面 \(ABCD\) 为长方形，\(PA \perp\) 底面 \(ABCD\)，且 \(PA = AB = 2\)，\(AD = 1\)。

不妨设长方形中 \(AB = 2\)，\(AD = 1\)，则 \(BC = AD = 1\)，\(CD = AB = 2\)。

连接各棱，需要计算所有棱的长度：
- 底面棱：\(AB = 2\)，\(BC = AD = 1\)，\(CD = 2\)，\(DA = 1\)。
- 侧棱：
  - \(PA = 2\)（已知）
  - 连接 \(PB\)：在 \(\text{Rt}\triangle PAB\) 中，\(PA \perp AB\)，
    \[
    PB = \sqrt{PA^2 + AB^2} = \sqrt{2^2 + 2^2} = 2\sqrt{2}.
    \]
  - 连接 \(PD\)：在 \(\text{Rt}\triangle PAD\) 中，\(PA \perp AD\)，
    \[
    PD = \sqrt{PA^2 + AD^2} = \sqrt{2^2 + 1^2} = \sqrt{5}.
    \]
  - 连接 \(PC\)：由线面垂直性质可知 \(PA \perp AC\)，先求底面长方形对角线 \(AC\)：
    \[
    AC = \sqrt{AB^2 + BC^2} = \sqrt{2^2 + 1^2} = \sqrt{5},
    \]
    则在 \(\text{Rt}\triangle PAC\) 中，
    \[
    PC = \sqrt{PA^2 + AC^2} = \sqrt{2^2 + (\sqrt{5})^2} = \sqrt{4 + 5} = 3.
    \]

比较所有棱长：\(2, 1, 2, 1, 2, 2\sqrt{2} \approx 2.828, \sqrt{5} \approx 2.236, 3\)。
最长的棱为 \(PC = 3\)。

**2. 求外接球表面积**

因为该阳马满足 \(PA \perp\) 底面 \(ABCD\)，且底面为长方形，可将四棱锥补形成以 \(AB\)、\(AD\)、\(AP\) 为长、宽、高的长方体。该阳马的外接球与所补成的长方体的外接球是同一个球。

长方体的体对角线即为外接球的直径 \(2R\)：
\[
2R = \sqrt{AB^2 + AD^2 + AP^2} = \sqrt{2^2 + 1^2 + 2^2} = \sqrt{9} = 3,
\]
所以外接球半径 \(R = \dfrac{3}{2}\)。

外接球的表面积：
\[
S = 4\pi R^2 = 4\pi \left( \frac{3}{2} \right)^2 = 4\pi \cdot \frac{9}{4} = 9\pi.
\]

## 答案
最长棱长为 \(3\)，外接球表面积为 \(9\pi\)。

## 总结
本题主要考查在特殊四棱锥（阳马）中棱长的计算以及外接球表面积的求解。关键点有两个：一是利用线面垂直关系，通过多次勾股定理计算所有棱长，从而找出最长棱；二是抓住“一条侧棱垂直于底面，且底面为长方形”这一特征，将阳马补形成长方体，利用长方体的体对角线即为外接球直径的性质快速求解球的表面积。补形法在处理含有垂直关系的棱锥外接球问题时是非常高效的手段。
