# 5-7-6 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-6.md|5-7-6]]

## 完整题目

6. (多选题，2021 湖北八市 3 月联考)若 \( \bigtriangleup  {ABC} \) 的内角 \( A, B, C \) 所对的边分别为 \( a, b, c \) ，且满足 \( b - {2a} + \; {4a}{\sin }^{2}\frac{A + B}{2} = 0 \) ，则下列结论正确的是 ( )

A. \( C \) 一定为锐角

B. \( {a}^{2} + 2{b}^{2} - {c}^{2} = 0 \)

C. \( 3\tan A + \tan C = 0 \)

D. \( \tan B \) 的最小值为 \( \frac{\sqrt{3}}{3} \)

## 题目勘误
无明显勘误。
## 思路分析
条件等式含有半角平方，先利用三角形内角和关系将 \(\sin^2\frac{A+B}{2}\) 化为 \(\cos^2\frac{C}{2}\)，再用降幂公式统一为 \(\cos C\)，从而得到关于边的简单关系或关于角的正弦、余弦关系。结合正弦定理与余弦定理，可以将边角混合条件转化为纯角或纯边条件，进而逐项判定各选项的正确性。选项 D 涉及最值，需要将 \(\tan B\) 用 \(\tan A\) 表示，利用基本不等式求最值并注意等号成立条件。
## 详细解答
已知 \(A+B+C=\pi\)，所以
\[
\frac{A+B}{2} = \frac{\pi}{2} - \frac{C}{2},\quad \sin\frac{A+B}{2} = \cos\frac{C}{2}.
\]
于是
\[
\sin^2\frac{A+B}{2} = \cos^2\frac{C}{2} = \frac{1+\cos C}{2}.
\]
代入条件式 \(b - 2a + 4a\sin^2\frac{A+B}{2} = 0\)，得
\[
b - 2a + 4a\cdot\frac{1+\cos C}{2} = 0 \implies b - 2a + 2a(1+\cos C) = 0,
\]
\[
b + 2a\cos C = 0. \tag{1}
\]
由 (1) 式得 \(\cos C = -\dfrac{b}{2a}\)。因为 \(a>0,\;b>0\)，所以 \(\cos C < 0\)，又 \(C\in(0,\pi)\)，故 \(C\) 为钝角，选项 A 错误。

由余弦定理 \(c^2 = a^2 + b^2 - 2ab\cos C\)，结合 (1) 式 \(2a\cos C = -b\)，代入得
\[
c^2 = a^2 + b^2 - b(-b) = a^2 + 2b^2,
\]
即 \(a^2 + 2b^2 - c^2 = 0\)，选项 B 正确。

将 (1) 式用正弦定理 \(a=2R\sin A,\;b=2R\sin B\) 转化：
\[
\sin B + 2\sin A\cos C = 0.
\]
又 \(\sin B = \sin(A+C) = \sin A\cos C + \cos A\sin C\)，代入上式：
\[
\sin A\cos C + \cos A\sin C + 2\sin A\cos C = 0 \implies 3\sin A\cos C + \cos A\sin C = 0.
\]
因为 \(C\) 为钝角，\(\cos C \neq 0\)，两边同除以 \(\cos A\cos C\) 得
\[
3\tan A + \tan C = 0. \tag{2}
\]
选项 C 正确。

由 (2) 式得 \(\tan C = -3\tan A\)。又 \(B = \pi - (A+C)\)，所以
\[
\tan B = -\tan(A+C) = -\frac{\tan A + \tan C}{1 - \tan A\tan C} = -\frac{\tan A - 3\tan A}{1 - \tan A\cdot(-3\tan A)} = \frac{2\tan A}{1 + 3\tan^2 A}.
\]
因 \(C\) 为钝角，\(A+B\) 为锐角，故 \(A\) 为锐角，\(\tan A > 0\)。设 \(t = \tan A > 0\)，则
\[
\tan B = \frac{2t}{1+3t^2} = \frac{2}{\frac{1}{t} + 3t}.
\]
由基本不等式，\(\frac{1}{t} + 3t \ge 2\sqrt{\frac{1}{t}\cdot 3t} = 2\sqrt{3}\)，当且仅当 \(\frac{1}{t}=3t\)，即 \(t=\frac{\sqrt{3}}{3}\) 时等号成立。
此时分母取得最小值，从而 \(\tan B\) 取得最大值 \(\frac{2}{2\sqrt{3}} = \frac{\sqrt{3}}{3}\)，而非最小值。当 \(t\to 0^+\) 或 \(t\to +\infty\) 时，\(\tan B \to 0\)，无最小正值（下确界为 0）。选项 D 错误。

综上，正确选项为 B 和 C。
## 答案
BC
## 总结
本题通过将条件中的半角平方化为单角余弦，是化简此类条件的关键一步。后续既可以选择“边”的视角（用余弦定理），也可以选择“角”的视角（用正弦定理和和角公式），灵活转化才能顺利判断各个选项。对于三角形中的最值问题，注意角度范围的约束以及基本不等式取等条件与“最大值/最小值”表述的匹配。
