# 5-7-19 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-19.md|5-7-19]]

## 完整题目

6.(2020 全国卷Ⅰ理，16)如图， 在三棱锥 \( P - {ABC} \) 的平面展开图中, \( {AC} = 1,{AB} = {AD} = \sqrt{3} \) , \( {AB} \bot  {AC},{AB} \bot  {AD},\angle {CAE} = \; {30}^{ \circ  } \) ,则 \( \cos \angle {FCB} = \) ___.

## 题目勘误
无明显勘误。（图片信息需人工核对）

## 思路分析
本题是三棱锥的平面展开图问题，核心是折叠前后线段长度不变、对应点重合。由条件可推断出点 \(D\)、\(E\)、\(F\) 分别为三棱锥 \(P-ABC\) 的侧棱剪开后顶点 \(P\) 所分裂出的三个对应点。根据已知的垂直关系与角度，可依次求出底面边 \(BC\) 及侧棱 \(PB\)、\(PC\) 的长度，最后在 \(\triangle FCB\) 中利用余弦定理求出 \(\cos\angle FCB\)。

## 详细解答

**第一步：确定对应关系与底面边长**
在三棱锥 \(P-ABC\) 的平面展开图中，通常沿侧棱 \(PA\)、\(PB\)、\(PC\) 剪开。顶点 \(P\) 分裂为三个点，结合条件：
- \(AB = AD = \sqrt{3}\)，且 \(AB\perp AD\)，故 \(D\) 为侧面 \(\triangle PAB\) 中 \(P\) 的对应点，\(AD = AP = \sqrt{3}\)，\(\angle PAB = 90^\circ\)。
- \(\angle CAE = 30^\circ\)，且 \(A\)、\(C\)、\(E\) 相连，故 \(E\) 为侧面 \(\triangle PAC\) 中 \(P\) 的对应点，\(AE = AP = \sqrt{3}\)，\(\angle CAE = \angle PAC = 30^\circ\)。
- 剩下的点 \(F\) 则为侧面 \(\triangle PBC\) 中 \(P\) 的对应点。
底面上，已知 \(AB\perp AC\)，\(AB = \sqrt{3}\)，\(AC = 1\)，则在 \({\rm Rt}\triangle ABC\) 中：
\[
BC = \sqrt{AB^2 + AC^2} = \sqrt{3 + 1} = 2.
\]

**第二步：求侧棱 \(PB\) 与 \(PC\) 的长度**
在侧面展开的 \({\rm Rt}\triangle ABD\) 中，\(AB = AD = \sqrt{3}\)，\(\angle BAD = 90^\circ\)，由折叠不变性得：
\[
PB = BD = \sqrt{AB^2 + AD^2} = \sqrt{3 + 3} = \sqrt{6}.
\]
在侧面展开的 \(\triangle ACE\) 中，\(AC = 1\)，\(AE = \sqrt{3}\)，\(\angle CAE = 30^\circ\)，由余弦定理：
\[
PC = CE = \sqrt{AC^2 + AE^2 - 2\cdot AC\cdot AE\cdot\cos30^\circ} 
= \sqrt{1 + 3 - 2\times 1\times\sqrt{3}\times\frac{\sqrt{3}}{2}} 
= \sqrt{4 - 3} = 1.
\]

**第三步：在 \(\triangle FCB\) 中求 \(\cos\angle FCB\)**
点 \(F\) 与点 \(P\) 重合，故 \(FC = PC = 1\)，\(FB = PB = \sqrt{6}\)，又已知 \(BC = 2\)。在 \(\triangle FCB\) 中，由余弦定理：
\[
\cos\angle FCB = \frac{FC^2 + BC^2 - FB^2}{2\cdot FC\cdot BC}
= \frac{1^2 + 2^2 - (\sqrt{6})^2}{2\times 1\times 2}
= \frac{1 + 4 - 6}{4}
= -\frac{1}{4}.
\]

## 答案
\[
\boxed{-\dfrac{1}{4}}
\]

## 总结
1. 三棱锥平面展开图问题的关键是“折叠前后线段长度不变，对应点重合”，需根据条件将展开图中的点与立体图形中的顶点对应起来。
2. 本题利用垂直关系和已知角度，在展开图的平面三角形中先求出所有棱长，再回到目标三角形中用余弦定理求解角度。
3. 注意余弦定理计算时的正负号，钝角的余弦值为负是正常结果。
