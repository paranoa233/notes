# 5-7-13 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-13.md|5-7-13]]

## 完整题目

4.(2024 新高考卷 I，15)记 \( \bigtriangleup  {ABC} \) 的内角 \( A \) ， \( B, C \) 的对边分别为 \( a, b, c \) ，已知 \( \sin C = \; \sqrt{2}\cos B,{a}^{2} + {b}^{2} - {c}^{2} = \sqrt{2}{ab}. \)

(1)求 \( B \) ；

(2)若 \( \bigtriangleup  {ABC} \) 的面积为 \( 3 + \sqrt{3} \) ，求 \( c \) .

## 题目勘误
无明显勘误。
## 思路分析
本题第（1）问给出两个条件：一个是边的关系式 \(a^2+b^2-c^2=\sqrt{2}ab\)，这直接与余弦定理关联；另一个是角的关系式 \(\sin C=\sqrt{2}\cos B\)。可先用余弦定理求出角 \(C\)，再代入角的关系式求出角 \(B\)。
第（2）问在已知三角形面积和已求各角的前提下，可利用正弦定理将边 \(a,b\) 用边 \(c\) 表示，代入面积公式 \(S=\frac{1}{2}ab\sin C\)，从而解出 \(c\)。
## 详细解答
**（1）求角 \(B\)**
已知 \(a^2+b^2-c^2=\sqrt{2}ab\)，
由余弦定理得：
\[
\cos C = \frac{a^2+b^2-c^2}{2ab} = \frac{\sqrt{2}ab}{2ab} = \frac{\sqrt{2}}{2},
\]
因为 \(C\in(0,\pi)\)，所以
\[
C = \frac{\pi}{4}.
\]
又已知 \(\sin C = \sqrt{2}\cos B\)，将 \(C=\frac{\pi}{4}\) 代入：
\[
\sin\frac{\pi}{4} = \sqrt{2}\cos B \quad\Longrightarrow\quad \frac{\sqrt{2}}{2} = \sqrt{2}\cos B,
\]
解得
\[
\cos B = \frac{1}{2},
\]
由 \(B\in(0,\pi)\)，得
\[
B = \frac{\pi}{3}.
\]
此时 \(A = \pi - B - C = \pi - \frac{\pi}{3} - \frac{\pi}{4} = \frac{5\pi}{12}\)。

**（2）求边 \(c\)**
三角形面积 \(S = 3 + \sqrt{3}\)。
由正弦定理，设 \(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = k\)，则
\[
a = k\sin A,\quad b = k\sin B,\quad c = k\sin C.
\]
面积公式为：
\[
S = \frac{1}{2}ab\sin C = \frac{1}{2}\cdot k\sin A \cdot k\sin B \cdot \sin C = \frac{1}{2}k^2 \sin A\sin B\sin C.
\]
又由 \(c = k\sin C\) 得 \(k = \frac{c}{\sin C}\)，代入面积表达式中：
\[
S = \frac{1}{2}\cdot \frac{c^2}{\sin^2 C}\cdot \sin A\sin B\sin C = \frac{1}{2}c^2 \cdot \frac{\sin A\sin B}{\sin C}.
\]
下面计算各三角函数值：
\[
\begin{aligned}
\sin A &= \sin\frac{5\pi}{12} = \sin\left(\frac{\pi}{4}+\frac{\pi}{6}\right) = \sin\frac{\pi}{4}\cos\frac{\pi}{6} + \cos\frac{\pi}{4}\sin\frac{\pi}{6} \\
&= \frac{\sqrt{2}}{2}\cdot\frac{\sqrt{3}}{2} + \frac{\sqrt{2}}{2}\cdot\frac{1}{2} = \frac{\sqrt{2}(\sqrt{3}+1)}{4}, \\[4pt]
\sin B &= \sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}, \\[4pt]
\sin C &= \sin\frac{\pi}{4} = \frac{\sqrt{2}}{2}.
\end{aligned}
\]
计算 \(\frac{\sin A\sin B}{\sin C}\)：
\[
\sin A\sin B = \frac{\sqrt{2}(\sqrt{3}+1)}{4} \cdot \frac{\sqrt{3}}{2} = \frac{\sqrt{2}(3+\sqrt{3})}{8},
\]
\[
\frac{\sin A\sin B}{\sin C} = \frac{\frac{\sqrt{2}(3+\sqrt{3})}{8}}{\frac{\sqrt{2}}{2}} = \frac{3+\sqrt{3}}{8} \cdot 2 = \frac{3+\sqrt{3}}{4}.
\]
于是面积可表示为：
\[
S = \frac{1}{2}c^2 \cdot \frac{3+\sqrt{3}}{4} = \frac{c^2(3+\sqrt{3})}{8}.
\]
将已知面积 \(S = 3+\sqrt{3}\) 代入：
\[
\frac{c^2(3+\sqrt{3})}{8} = 3+\sqrt{3},
\]
两边同除 \(3+\sqrt{3}\)（显然不为零）：
\[
\frac{c^2}{8} = 1 \quad\Longrightarrow\quad c^2 = 8.
\]
因为边长为正，故
\[
c = 2\sqrt{2}.
\]
## 答案
（1）\(B = \frac{\pi}{3}\)（或 \(60^\circ\)）；  
（2）\(c = 2\sqrt{2}\).
## 总结
本题的核心是正、余弦定理与面积公式的联合应用。先由边的关系式借助余弦定理求出角 \(C\)，再结合正弦与余弦的等量关系求得角 \(B\)。在已知三个角与面积时，利用正弦定理将边的关系转化为只含所求边的等式，是解三角形的常见思路。注意特殊角的三角函数值要牢记，计算时保持代数式的整体化简。
