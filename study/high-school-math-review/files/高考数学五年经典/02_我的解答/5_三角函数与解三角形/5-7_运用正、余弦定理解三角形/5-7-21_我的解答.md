# 5-7-21 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-21.md|5-7-21]]

## 完整题目

8.(2022全国乙卷文，17)记 \( \bigtriangleup  {ABC} \) 的内角 \( A \) ， \( B, C \) 的对边分别为 \( a, b, c \) ，已知 \( \sin C\sin \left( {A - B}\right) \; = \sin B\sin \left( {C - A}\right) \) .

(1)若 \( A = {2B} \) ，求 \( C \) ；

(2)求证: \( 2{a}^{2} = {b}^{2} + {c}^{2} \) .

## 题目勘误
无明显勘误。
## 思路分析
(1) 将 \(A = 2B\) 代入已知的三角等式，利用三角形内角和消去 \(C\)，得到关于 \(B\) 的三角方程 \(\sin 3B = \sin 5B\)，结合 \(B\) 的取值范围解出 \(B\)，进而求得 \(C\)。

(2) 将已知等式两边展开，利用和角公式、三角形内角和及正弦定理，将角的关系转化为边的关系 \(a^2 = 2bc\cos A\)，再代入余弦定理即可证明 \(2a^2 = b^2 + c^2\)。
## 详细解答
**(1) 求 \(C\)**

已知 \(\sin C\sin(A - B) = \sin B\sin(C - A)\)，且 \(A = 2B\)。

由 \(A+B+C=\pi\) 及 \(A=2B\) 得  
\[
C = \pi - (A+B) = \pi - 3B.
\]

将上式及 \(A=2B\) 代入已知等式：
\[
\sin(\pi - 3B)\sin(2B - B) = \sin B\sin\bigl((\pi - 3B) - 2B\bigr),
\]
\[
\sin 3B \cdot \sin B = \sin B \cdot \sin(\pi - 5B) = \sin B \cdot \sin 5B.
\]

因为 \(0<B<\pi\) 且 \(A=2B<\pi\)，所以 \(0<B<\dfrac{\pi}{2}\)，\(\sin B \neq 0\)，两边约去 \(\sin B\) 得  
\[
\sin 3B = \sin 5B.
\]

于是  
\[
5B = 3B + 2k\pi \quad\text{或}\quad 5B = \pi - 3B + 2k\pi \quad (k\in\mathbb{Z}).
\]

由 \(5B = 3B + 2k\pi\) 得 \(B = k\pi\)，在 \(\bigl(0, \frac{\pi}{2}\bigr)\) 内无解；  
由 \(5B = \pi - 3B + 2k\pi\) 得 \(8B = \pi + 2k\pi\)，即 \(B = \dfrac{\pi}{8} + \dfrac{k\pi}{4}\)。  
在 \(\bigl(0, \frac{\pi}{2}\bigr)\) 内，\(k=0\) 时 \(B = \dfrac{\pi}{8}\)；\(k=1\) 时 \(B = \dfrac{3\pi}{8}\)，此时 \(A = 2B = \dfrac{3\pi}{4}\)，\(A+B = \dfrac{9\pi}{8} > \pi\)，不符合三角形内角和，舍去。

因此 \(B = \dfrac{\pi}{8}\)，进而  
\[
C = \pi - 3B = \pi - \frac{3\pi}{8} = \frac{5\pi}{8}.
\]

**(2) 求证：\(2a^2 = b^2 + c^2\)**

已知 \(\sin C\sin(A - B) = \sin B\sin(C - A)\)。

左边展开：\(\sin C(\sin A\cos B - \cos A\sin B)\)，  
右边展开：\(\sin B(\sin C\cos A - \cos C\sin A)\)。

移项并整理：
\[
\sin C\sin A\cos B - \sin C\cos A\sin B - \sin B\sin C\cos A + \sin B\cos C\sin A = 0,
\]
\[
\sin A(\sin C\cos B + \sin B\cos C) - 2\sin B\sin C\cos A = 0.
\]

由和角公式，\(\sin C\cos B + \sin B\cos C = \sin(B+C) = \sin(\pi - A) = \sin A\)，代入即得：
\[
\sin A \cdot \sin A - 2\sin B\sin C\cos A = 0,
\]
\[
\sin^2 A = 2\sin B\sin C\cos A.
\]

利用正弦定理 \(a = 2R\sin A,\; b = 2R\sin B,\; c = 2R\sin C\)，将上式两边同乘以 \((2R)^2\)：
\[
a^2 = 2bc\cos A.
\]

再由余弦定理 \(\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}\)，代入上式：
\[
a^2 = 2bc \cdot \frac{b^2 + c^2 - a^2}{2bc} = b^2 + c^2 - a^2,
\]
移项得
\[
2a^2 = b^2 + c^2.
\]
证毕。
## 答案
(1) \(C = \dfrac{5\pi}{8}\)；  
(2) 证明过程见上。
## 总结
(1) 利用三角形内角和消元，结合三角方程求解是常见思路，需注意内角的取值范围以取舍解。  
(2) 证明边的关系时，通常利用正弦定理将角化为边，或结合余弦定理进行转化，本题通过展开已知等式并借助和角公式得到 \(\sin^2 A = 2\sin B\sin C\cos A\)，是顺利推出目标式的关键。
