# 5-7-18 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-18.md|5-7-18]]

## 完整题目

5.(2024全国甲卷理，11)在 \( \bigtriangleup  {ABC} \) 中，内角 \( A \) ， \( B, C \) 所对的边分别为 \( a, b, c \) ,若 \( B = \frac{\pi }{3},{b}^{2} = \; \frac{9}{4}{ac} \) ,则 \( \sin A + \sin C = \) (   )

A. \( \frac{2\sqrt{39}}{13} \) B. \( \frac{\sqrt{39}}{13} \)

C. \( \frac{\sqrt{7}}{2} \) D. \( \frac{3\sqrt{13}}{13} \)

![[05_附件/images/73_1332_739_207_295_0.jpg]]

\( F\left( P\right) \)

## 题目勘误
无明显勘误。
## 思路分析
已知 \(B\) 和 \(b^2\) 与 \(ac\) 的关系，可先利用余弦定理求出 \(a\) 与 \(c\) 的关系，进而得到 \(a+c\) 与 \(b\) 的关系，最后利用正弦定理将 \(\sin A+\sin C\) 转化为边的关系求解。
## 详细解答
由余弦定理得：
\[
b^2 = a^2 + c^2 - 2ac\cos B = a^2 + c^2 - 2ac\cos\frac{\pi}{3} = a^2 + c^2 - ac.
\]
又已知 \(b^2 = \frac{9}{4}ac\)，代入上式得：
\[
a^2 + c^2 - ac = \frac{9}{4}ac \implies a^2 + c^2 = \frac{13}{4}ac.
\]
配方得：
\[
(a+c)^2 = a^2 + c^2 + 2ac = \frac{13}{4}ac + 2ac = \frac{21}{4}ac.
\]
将 \(ac = \frac{4}{9}b^2\) 代入：
\[
(a+c)^2 = \frac{21}{4} \cdot \frac{4}{9}b^2 = \frac{7}{3}b^2 \implies \frac{a+c}{b} = \sqrt{\frac{7}{3}}.
\]
由正弦定理 \(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R\) 得：
\[
\sin A + \sin C = \frac{a+c}{2R} = \frac{a+c}{b} \cdot \sin B.
\]
将 \(\frac{a+c}{b} = \sqrt{\frac{7}{3}}\) 和 \(\sin B = \sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}\) 代入：
\[
\sin A + \sin C = \sqrt{\frac{7}{3}} \cdot \frac{\sqrt{3}}{2} = \frac{\sqrt{7}}{2}.
\]
## 答案
C
## 总结
本题综合考查了余弦定理和正弦定理的边角互化。关键在于利用已知条件先求边的关系，再利用正弦定理将角的正弦和转化为边的和，避免了直接求角。
