# 5-7-1 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-1.md|5-7-1]]

## 完整题目

1.(2023 浙江杭州期末)在 \( \bigtriangleup  {ABC} \) 中， \( a\cos B = \; \frac{9}{\sqrt{15}}, b\cos A = \frac{6}{\sqrt{15}}, c = \sqrt{15} \) ,则 \( \frac{\sin \left( {B - A}\right) }{\sin C} \) 的值为 ( )

A. \( \frac{1}{5} \) B. \( - \frac{1}{5} \)

C. \( \frac{\sqrt{15}}{15} \) D. \( - \frac{\sqrt{15}}{15} \)

## 题目勘误
无明显勘误。
## 思路分析
题设直接给出 \(a\cos B\) 与 \(b\cos A\) 的数值及边 \(c\)，可利用射影定理验证一致性。目标是求 \(\frac{\sin(B-A)}{\sin C}\)，利用正弦定理将 \(a\cos B\) 和 \(b\cos A\) 转化为含外接圆半径 \(R\) 的三角函数式，并借助 \(\sin C = \sin(A+B)\) 展开，直接将已知值代入即可消去未知参数得解。
## 详细解答
由正弦定理 \(a = 2R\sin A,\; b = 2R\sin B\)（\(R\) 为外接圆半径），则
\[
a\cos B = 2R\sin A\cos B = \frac{9}{\sqrt{15}}, \quad
b\cos A = 2R\sin B\cos A = \frac{6}{\sqrt{15}}.
\]
于是有
\[
\sin A\cos B = \frac{9}{2R\sqrt{15}}, \quad \sin B\cos A = \frac{6}{2R\sqrt{15}}.
\]
在 \(\triangle ABC\) 中，\(C = \pi - (A+B)\)，故 \(\sin C = \sin(A+B)\)。所求式子可化为
\[
\frac{\sin(B-A)}{\sin C}
= \frac{\sin B\cos A - \cos B\sin A}{\sin A\cos B + \sin B\cos A}
= \frac{ \frac{6}{2R\sqrt{15}} - \frac{9}{2R\sqrt{15}} }{ \frac{9}{2R\sqrt{15}} + \frac{6}{2R\sqrt{15}} }
= \frac{-3}{15}
= -\frac{1}{5}.
\]
## 答案
\(B\)
## 总结
本题直接利用正弦定理将边角混合条件转化为角的三角函数关系，并将目标比值的分子分母同时用已知量表示，消去了外接圆半径 \(R\)，无需解出具体角度即得结果。关键在于识别出 \(\frac{\sin(B-A)}{\sin C}\) 可展开为含 \(\sin A\cos B\) 与 \(\sin B\cos A\) 的分式，从而与条件直接挂钩。
