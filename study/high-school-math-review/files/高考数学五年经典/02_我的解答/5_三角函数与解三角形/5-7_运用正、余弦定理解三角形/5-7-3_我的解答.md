# 5-7-3 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-3.md|5-7-3]]

## 完整题目

3.(2021 江苏南京学情调研)在 \( \bigtriangleup  {ABC} \) 中，角 \( A \) ， \( B, C \) 的对边分别为 \( a, b, c \) . 若 \( {2b}\cos C \leq  {2a} - c \) , 则 \( B \) 的取值范围是 ( )

A. \( \left( {0,\frac{\pi }{3}}\right\rbrack \) B. \( \left( {0,\frac{2\pi }{3}}\right\rbrack \)

C. \( \left\lbrack  {\frac{\pi }{3},\pi }\right) \) D. \( \left\lbrack  {\frac{2\pi }{3},\pi }\right) \)

## 题目勘误
无明显勘误。

## 思路分析
已知条件为边角混合不等式 \(2b\cos C \leq 2a - c\)，目标是求角 \(B\) 的取值范围。有两种常见思路：
1. **正弦定理化边为角**：利用 \(a = 2R\sin A\) 等，结合内角和 \(A = \pi - (B + C)\) 及两角和公式，消去角 \(A\)，转化为只含 \(B, C\) 的不等式，再利用 \(\sin C > 0\) 化简，得到关于 \(\cos B\) 的不等式，从而解出 \(B\) 的范围。
2. **余弦定理化角为边**：利用 \(\cos C = \frac{a^2 + b^2 - c^2}{2ab}\) 代入，化简得到边的不等式，再通过边的关系或余弦定理转化为角 \(B\) 的范围。  
这里采用思路1，过程更简洁。

## 详细解答
由正弦定理，\(a = 2R\sin A\)，\(b = 2R\sin B\)，\(c = 2R\sin C\)（\(R\) 为 \(\triangle ABC\) 外接圆半径）。代入已知不等式：
\[
2b\cos C \leq 2a - c \;\Longrightarrow\; 2\sin B\cos C \leq 2\sin A - \sin C.
\]
在 \(\triangle ABC\) 中，\(A = \pi - (B + C)\)，所以 \(\sin A = \sin(B + C) = \sin B\cos C + \cos B\sin C\)。代入上式得：
\[
2\sin B\cos C \leq 2(\sin B\cos C + \cos B\sin C) - \sin C.
\]
展开并移项：
\[
2\sin B\cos C \leq 2\sin B\cos C + 2\cos B\sin C - \sin C,
\]
\[
0 \leq 2\cos B\sin C - \sin C = \sin C\,(2\cos B - 1).
\]
因为 \(C\) 是三角形内角，\(\sin C > 0\)，两边同除以 \(\sin C\) 不等号方向不变：
\[
2\cos B - 1 \geq 0 \;\Longrightarrow\; \cos B \geq \frac{1}{2}.
\]
又 \(B\) 为三角形内角，\(B \in (0, \pi)\)。在 \((0, \pi)\) 上，余弦函数递减，由 \(\cos B \geq \frac{1}{2}\) 解得：
\[
0 < B \leq \frac{\pi}{3}.
\]
即 \(B\) 的取值范围是 \(\left(0, \frac{\pi}{3}\right]\)。

## 答案
A

## 总结
本题属于三角形中的边角互化与不等式综合题。关键步骤是利用正弦定理将边化为角，通过三角形内角和公式将 \(\sin A\) 改写成 \(\sin(B+C)\)，然后化简消元，最终得到关于 \(\cos B\) 的一元不等式。解题中需注意三角形内角的正弦恒正，可放心约分。本题易错点为忽视等号能否取到，由推导可知 \(B = \frac{\pi}{3}\) 时不等式取等，此时三角形满足条件（比如正三角形等情形），故区间左端点可取闭。
