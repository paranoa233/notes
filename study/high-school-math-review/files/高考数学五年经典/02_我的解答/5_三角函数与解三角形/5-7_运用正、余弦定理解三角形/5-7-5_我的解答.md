# 5-7-5 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-5.md|5-7-5]]

## 完整题目

5.(2024 福建厦门四检)记锐角三角形 \( {ABC} \) 的内角 \( A, B, C \) 的对边分别为 \( a, b, c \) . 若 \( 2\cos C = \; \frac{3b}{a} - \frac{a}{b} \) ，则角 \( B \) 的取值范围是___.

## 题目勘误
无明显勘误。
## 思路分析
条件式涉及边和角的混合，可通过余弦定理将其转化为边的关系，得出 \(c^2\) 关于 \(a,b\) 的表达式。进而用边表示 \(\cos B\)，结合锐角三角形的限制（各角余弦大于零）确定边之比的范围，最后求出角 \(B\) 的取值范围。
## 详细解答
由余弦定理知  
\[
\cos C = \frac{a^2 + b^2 - c^2}{2ab}.
\]
代入已知条件 \( \displaystyle 2\cos C = \frac{3b}{a} - \frac{a}{b} \) 得  
\[
2\cdot \frac{a^2 + b^2 - c^2}{2ab} = \frac{3b}{a} - \frac{a}{b},
\]
化简  
\[
\frac{a^2 + b^2 - c^2}{ab} = \frac{3b^2 - a^2}{ab},
\]
去分母得  
\[
a^2 + b^2 - c^2 = 3b^2 - a^2,
\]
整理得  
\[
c^2 = 2a^2 - 2b^2. \tag{1}
\]
由于边长为正，得 \(a^2 > b^2\)，即 \(a > b\)，从而 \(\displaystyle x = \frac{a}{b} > 1\)。

在 \(\triangle ABC\) 中，由余弦定理求 \(\cos B\)：
\[
\cos B = \frac{a^2 + c^2 - b^2}{2ac}.
\]
将 (1) 式代入  
\[
\cos B = \frac{a^2 + (2a^2 - 2b^2) - b^2}{2a\sqrt{2a^2 - 2b^2}} = \frac{3a^2 - 3b^2}{2a\sqrt{2(a^2 - b^2)}}.
\]
分子分母同除以 \(b^2\) 并令 \(x = a/b\)，得  
\[
\cos B = \frac{3(x^2 - 1)}{2x\cdot\sqrt{2(x^2 - 1)}} = \frac{3\sqrt{x^2 - 1}}{2\sqrt{2}\,x} = \frac{3}{2\sqrt{2}}\cdot\frac{\sqrt{x^2 - 1}}{x}.
\]

因为 \(\triangle ABC\) 是锐角三角形，所以三个角均为锐角：\(\cos A > 0,\ \cos B > 0,\ \cos C > 0\)。

- 由 (1) 及 \(a > b\) 易得 \(\cos A = \frac{b^2 + c^2 - a^2}{2bc} = \frac{a^2 - b^2}{2bc} > 0\) 自动满足。
- \(\cos B > 0\) 自然成立，下面的讨论可知结论也符合。
- \(\cos C > 0\) 利用已知条件：\(\displaystyle 2\cos C = \frac{3b}{a} - \frac{a}{b} > 0 \Rightarrow \frac{3b}{a} > \frac{a}{b} \Rightarrow a^2 < 3b^2\)，即 \(x^2 < 3\)。

结合 \(x > 1\)，得 \(1 < x < \sqrt{3}\)。

考察 \(\cos B\) 关于 \(x\) 的单调性：\(\displaystyle \frac{\sqrt{x^2 - 1}}{x} = \sqrt{1 - \frac{1}{x^2}}\) 在 \(x > 1\) 时是增函数，所以 \(\cos B\) 随 \(x\) 增大而增大（\(\cos B\) 本身在 \((0,\frac{\pi}{2})\) 内是减函数）。计算端点：

- 当 \(x \to 1^+\) 时，\(\cos B \to 0^+\)，则 \(B \to \frac{\pi}{2}^-\)；
- 当 \(x \to \sqrt{3}^-\) 时，\(\cos B \to \frac{3}{2\sqrt{2}}\cdot\frac{\sqrt{3 - 1}}{\sqrt{3}} = \frac{3}{2\sqrt{2}}\cdot\frac{\sqrt{2}}{\sqrt{3}} = \frac{\sqrt{3}}{2}\)，对应 \(B = \frac{\pi}{6}\)。

由于 \(x\) 取不到端点值，\(\cos B \in (0,\ \frac{\sqrt{3}}{2})\)，从而角 \(B\) 的取值范围为 \(\left( \frac{\pi}{6},\ \frac{\pi}{2} \right)\)。
## 答案
\[
\boxed{\left( \frac{\pi}{6},\ \frac{\pi}{2} \right)}
\]
## 总结
本题关键在于将边角混合条件用余弦定理统一为边的关系，进而用边之比表示所求角的余弦。最后利用锐角三角形的余弦符号限制比例范围，结合函数单调性求出角的取值范围。注意锐角三角形不仅内角小于 \(\frac{\pi}{2}\)，还需保证所有角的余弦值为正，本题中 \(\cos C > 0\) 给出了关键的边界。
