# 5-7-8 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-8.md|5-7-8]]

## 完整题目

8.(2022 辽宁沈阳质检)从① \( b\sin C = \sqrt{3}c\cos B \) ， ② \( {b}^{2} + {ac} = {a}^{2} + {c}^{2} \) 这两个条件中任选一个，补充到下面已知条件中进行解答.

已知在 \( \bigtriangleup  {ABC} \) 中，角 \( A, B, C \) 的对边分别为 \( a, b, c \) ， 且___.

(1)求角 \( B \) ；

(2)若 \( b = 2 \) ， \( \bigtriangleup  {ABC} \) 的面积为 \( \sqrt{3} \) ，求 \( a \) .

## 题目勘误
无明显勘误。
## 思路分析
本题为结构不良问题，需从给出的两个条件中任选一个补充完整，然后求解。  
（1）求角 \(B\)：  
- 若选择条件①，可利用正弦定理将边化为角，得到关于 \(B\) 的三角方程求解。  
- 若选择条件②，可整理等式并结合余弦定理求出 \(\cos B\)。  
（2）已知 \(b=2\) 和面积，结合（1）中求出的 \(B\)，可利用三角形面积公式求出 \(ac\)，再利用余弦定理建立 \(a\) 与 \(c\) 的关系，进而解出 \(a\)。
## 详细解答
**（1）求角 \(B\)**

**若选条件①：** \(b\sin C = \sqrt{3}c\cos B\)

由正弦定理 \(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}\)，得 \(b = 2R\sin B\)，\(c = 2R\sin C\)，代入得
\[
\sin B \cdot \sin C = \sqrt{3} \sin C \cos B
\]
因为 \(\triangle ABC\) 中 \(\sin C \neq 0\)，两边约去 \(\sin C\) 得
\[
\sin B = \sqrt{3} \cos B
\]
即 \(\tan B = \sqrt{3}\)，又 \(0 < B < \pi\)，所以
\[
B = \frac{\pi}{3}
\]

**若选条件②：** \(b^{2} + ac = a^{2} + c^{2}\)

移项得
\[
a^{2} + c^{2} - b^{2} = ac
\]
由余弦定理 \(\cos B = \frac{a^{2} + c^{2} - b^{2}}{2ac}\)，代入上式得
\[
\cos B = \frac{ac}{2ac} = \frac{1}{2}
\]
又 \(0 < B < \pi\)，所以
\[
B = \frac{\pi}{3}
\]

综上，无论选哪个条件，均有 \(B = \frac{\pi}{3}\)。

**（2）求 \(a\)**

已知 \(b = 2\)，\(\triangle ABC\) 的面积 \(S = \sqrt{3}\)，且 \(B = \frac{\pi}{3}\)。  
三角形面积公式 \(S = \frac{1}{2}ac\sin B\)，代入得
\[
\sqrt{3} = \frac{1}{2}ac \cdot \sin\frac{\pi}{3} = \frac{1}{2}ac \cdot \frac{\sqrt{3}}{2} = \frac{\sqrt{3}}{4}ac
\]
解得
\[
ac = 4
\]
由余弦定理 \(b^{2} = a^{2} + c^{2} - 2ac\cos B\)，代入 \(b=2\)，\(\cos B = \frac{1}{2}\)，\(ac=4\) 得
\[
4 = a^{2} + c^{2} - 2 \times 4 \times \frac{1}{2} = a^{2} + c^{2} - 4
\]
所以
\[
a^{2} + c^{2} = 8
\]
结合 \(ac = 4\)，可得
\[
(a + c)^{2} = a^{2} + c^{2} + 2ac = 8 + 8 = 16 \Rightarrow a + c = 4 \,(取正)
\]
\[
(a - c)^{2} = a^{2} + c^{2} - 2ac = 8 - 8 = 0 \Rightarrow a = c
\]
因此 \(a = c = 2\)。

## 答案
（1）\(B = \frac{\pi}{3}\)；（2）\(a = 2\)。

## 总结
本题属于结构不良的选条件解三角形问题，主要考查正弦定理、余弦定理与三角形面积公式的综合应用。  
- 第（1）问中，两个条件本质是统一的，一个给出边角关系，一个给出三边平方关系，均可快速得到 \(B = 60^\circ\)。  
- 第（2）问利用面积公式建立 \(ac\) 的乘积关系，再借助余弦定理建立平方和关系，联立求出 \(a=c=2\)。注意不要遗漏三角形边长的实际意义，如两边之和为正等。
