# 5-7-10 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-7_运用正、余弦定理解三角形/5-7-10.md|5-7-10]]

## 完整题目

1.(2020 新高考卷Ⅰ，17)在① \( {ac} = \sqrt{3} \) ，② \( c\sin A = 3 \) ， ③ \( c = \sqrt{3}b \) 这三个条件中任选一个，补充在下面问题中，若问题中的三角形存在，求 \( c \) 的值；若问题中的三角形不存在，请说明理由.

问题: 是否存在 \( \bigtriangleup {ABC} \) ,它的内角 \( A, B, C \) 的对

边分别为 \( a, b, c \) ,且 \( \sin A = \sqrt{3}\sin B, C = \; \frac{\pi }{6} \) ,___?

## 题目勘误
无明显勘误。

## 思路分析
已知 \( \sin A = \sqrt{3}\sin B \) 和 \( C = \frac{\pi}{6} \)。先用正弦定理将角的关系化为边的关系：\( a = \sqrt{3}b \)。再结合已知角 \( C \) 用余弦定理找出 \( c \) 与 \( b \) 的关系，发现无论选哪个条件，三角形若存在，形状均唯一确定（\( c=b \)，\( B=C=\frac{\pi}{6} \)，\( A=\frac{2\pi}{3} \)）。最后将所选条件代入已得的关系，求出 \( c \) 的具体值或判断三角形不存在。

## 详细解答

**已知**：在 \( \triangle ABC \) 中，\( \sin A = \sqrt{3}\sin B \)，\( C = \dfrac{\pi}{6} \)。

**第一步：由正弦定理转化角的关系**
由正弦定理 \( \dfrac{a}{\sin A} = \dfrac{b}{\sin B} \) 得
\[
\frac{a}{b} = \frac{\sin A}{\sin B} = \sqrt{3} \quad \Rightarrow \quad a = \sqrt{3}b.
\]

**第二步：由余弦定理找 \( c \) 与 \( b \) 的关系**
已知 \( C = \dfrac{\pi}{6} \)，由余弦定理：
\[
c^2 = a^2 + b^2 - 2ab\cos C = (\sqrt{3}b)^2 + b^2 - 2\cdot\sqrt{3}b \cdot b \cdot \cos\frac{\pi}{6}
\]
\[
= 3b^2 + b^2 - 2\sqrt{3}b^2 \cdot \frac{\sqrt{3}}{2} = 4b^2 - 3b^2 = b^2,
\]
所以 \( c = b \)（边长为正）。

**第三步：判断三角形的形状**
由 \( c=b \) 及正弦定理得 \( \sin C = \sin B \)，又 \( C=\frac{\pi}{6} \)，所以
\[
B = \frac{\pi}{6} \quad \text{或} \quad B = \frac{5\pi}{6}\ (\text{舍去，否则 } A=0).
\]
从而 \( B = \frac{\pi}{6} \)，\( A = \pi - B - C = \frac{2\pi}{3} \)。三角形存在且边满足 \( a = \sqrt{3}b \)，\( c = b \)。

**第四步：分别代入三个条件求解**

- **若选条件①** \( ac = \sqrt{3} \)  
  将 \( a = \sqrt{3}b \)，\( c = b \) 代入：
  \[
  ac = \sqrt{3}b \cdot b = \sqrt{3}b^2 = \sqrt{3} \quad \Rightarrow \quad b^2 = 1 \quad \Rightarrow \quad b = 1.
  \]
  所以 \( c = b = 1 \)，三角形存在。

- **若选条件②** \( c\sin A = 3 \)  
  \( c = b \)，\( \sin A = \sin\dfrac{2\pi}{3} = \dfrac{\sqrt{3}}{2} \)，代入：
  \[
  b \cdot \frac{\sqrt{3}}{2} = 3 \quad \Rightarrow \quad b = \frac{6}{\sqrt{3}} = 2\sqrt{3}.
  \]
  所以 \( c = 2\sqrt{3} \)，三角形存在。

- **若选条件③** \( c = \sqrt{3}b \)  
  由前面推导知 \( c = b \)，代入得 \( b = \sqrt{3}b \)。因为 \( b>0 \)，等式不成立。故此条件下三角形不存在。

## 答案
若选①，三角形存在，\( c = 1 \)；  
若选②，三角形存在，\( c = 2\sqrt{3} \)；  
若选③，三角形不存在。

## 总结
本题为“结构不良”型解三角形问题，核心是先利用正弦定理、余弦定理挖掘题设中隐含的边角关系（得出 \( a=\sqrt{3}b \)，\( c=b \) 以及各角大小），再结合所选附加条件计算或判断存在性。解题时注意三角形中两边相等所对应的角的关系，以及三角形内角范围的限制。
