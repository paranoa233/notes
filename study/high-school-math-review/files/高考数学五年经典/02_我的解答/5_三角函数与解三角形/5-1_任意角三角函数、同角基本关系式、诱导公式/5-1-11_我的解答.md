# 5-1-11 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-1_任意角三角函数、同角基本关系式、诱导公式/5-1-11.md|5-1-11]]

## 完整题目

2.(2023全国甲卷理，7) “sin \( {}^{2}\alpha  + \sin {}^{2}\beta  = 1 \) ”是 “ \( \sin \alpha  + \cos \beta  = 0 \) ”的 )

A. 充分条件但不是必要条件

B. 必要条件但不是充分条件

C. 充要条件

D. 既不充分条件也不是必要条件

## 题目勘误
无明显勘误。
## 思路分析
本题考查三角函数基本关系与充要条件的判定。先把条件“\(\sin^2\alpha + \sin^2\beta = 1\)”转化为更简洁的等式，再与“\(\sin\alpha + \cos\beta = 0\)”对比，分析谁推出谁，从而判断充分性与必要性。
## 详细解答
**第一步：化简条件 \(p: \sin^2\alpha + \sin^2\beta = 1\)。**  
由同角基本关系 \(\sin^2\beta + \cos^2\beta = 1\)，得  
\[
\sin^2\alpha + \sin^2\beta = 1 \iff \sin^2\alpha = 1 - \sin^2\beta = \cos^2\beta.
\]  
即 \(|\sin\alpha| = |\cos\beta|\)。  
于是条件 \(p\) 等价于 \(\sin\alpha = \cos\beta\) 或 \(\sin\alpha = -\cos\beta\)。

**第二步：分析条件 \(q: \sin\alpha + \cos\beta = 0\)。**  
条件 \(q\) 等价于 \(\sin\alpha = -\cos\beta\)。

**第三步：判断充分性与必要性。**  
- 充分性：\(p \Rightarrow q\)？  
  当 \(\sin\alpha = \cos\beta\) 且 \(\sin\alpha \neq 0\) 时，满足 \(p\)，但 \(\sin\alpha + \cos\beta = 2\sin\alpha \neq 0\)，不满足 \(q\)。  
  因此 \(p\) 不能推出 \(q\)，充分性不成立。

- 必要性：\(q \Rightarrow p\)？  
  若 \(q\) 成立，即 \(\sin\alpha = -\cos\beta\)，两边平方得  
  \[
  \sin^2\alpha = \cos^2\beta = 1 - \sin^2\beta,
  \]  
  整理得 \(\sin^2\alpha + \sin^2\beta = 1\)，即 \(p\) 成立。  
  因此 \(q\) 能推出 \(p\)，必要性成立。

综上，\(p\) 是 \(q\) 的必要不充分条件。

## 答案
B

## 总结
处理此类“三角等式+充要条件”问题时，关键是把条件化到最简，再寻找反例或严格推导。本题中把 \(\sin^2\alpha + \sin^2\beta = 1\) 转化为 \(|\sin\alpha| = |\cos\beta|\) 是突破口，它包含了符号相同与相反两种情况，而目标条件只含符号相反的情况，故必要不充分。
