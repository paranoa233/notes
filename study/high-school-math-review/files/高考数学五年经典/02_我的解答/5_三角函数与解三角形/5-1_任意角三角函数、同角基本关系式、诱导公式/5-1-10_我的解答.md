# 5-1-10 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-1_任意角三角函数、同角基本关系式、诱导公式/5-1-10.md|5-1-10]]

## 完整题目

1.(2020全国卷Ⅱ理，2)若 \( \alpha \) 为第四象限角，则

( )

A. \( \cos {2\alpha } > 0 \) B. \( \cos {2\alpha } < 0 \)

C. \( \sin {2\alpha } > 0 \) D. \( \sin {2\alpha } < 0 \)

## 题目勘误
无明显勘误。
## 思路分析
由 \(\alpha\) 为第四象限角, 可得 \(\alpha\) 的取值范围, 进而推出 \(2\alpha\) 的取值范围, 再结合各象限中正弦、余弦的符号规律即可判断 \(\sin 2\alpha\) 与 \(\cos 2\alpha\) 的符号. 也可用特值法辅助排除.
## 详细解答
因为 \(\alpha\) 为第四象限角, 所以
\[
2k\pi - \frac{\pi}{2} < \alpha < 2k\pi, \quad k \in \mathbb{Z}.
\]
两边同乘 \(2\) 得
\[
4k\pi - \pi < 2\alpha < 4k\pi, \quad k \in \mathbb{Z}.
\]
即 \(2\alpha\) 的终边位于第三或第四象限, 或终边落在 \(x\) 轴负半轴等位置. 因此 \(\sin 2\alpha < 0\) 恒成立, 而 \(\cos 2\alpha\) 可能为正也可能为负.

也可取特殊值验证:
- 取 \(\alpha = -\dfrac{\pi}{6}\) (第四象限角), 则 \(2\alpha = -\dfrac{\pi}{3}\), \(\cos 2\alpha = \dfrac{1}{2} > 0\), \(\sin 2\alpha = -\dfrac{\sqrt{3}}{2} < 0\).
- 取 \(\alpha = -\dfrac{\pi}{3}\) (第四象限角), 则 \(2\alpha = -\dfrac{2\pi}{3}\), \(\cos 2\alpha = -\dfrac{1}{2} < 0\), \(\sin 2\alpha = -\dfrac{\sqrt{3}}{2} < 0\).

可见 \(\cos 2\alpha\) 符号不确定, \(\sin 2\alpha\) 一定为负.
## 答案
D
## 总结
本题主要考查象限角的表示与三角函数值符号的判断. 由 \(\alpha\) 的范围推出 \(2\alpha\) 的范围再判断符号是通法, 也可结合特殊角快速排除错误选项, 但需注意特殊值不能证明恒成立, 正式解答建议用范围推导.
