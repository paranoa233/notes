# 5-11-5 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-11_三角函数的综合运用/5-11-5.md|5-11-5]]

## 完整题目

5.(2024 河南郑州二模)声音是由物体振动产生的声波，其中包含着正弦函数. 纯音的数学模型是函数 \( y = A\sin {\omega t} \) ，但我们平时听到的乐音不只是一个音在响，而是许多个音的结合，称为复合音. 若一个复合音的数学模型是函数 \( f\left( x\right)  = \sin x + \; \frac{1}{2}\sin {2x}\left( {x \in  \mathbf{R}}\right) \) ,则下列说法正确的是 ( )

A. \( f\left( x\right) \) 的一个周期为 \( \pi \)

B. \( f\left( x\right) \) 的最大值为 \( \frac{3}{2} \)

C. \( f\left( x\right) \) 的图象关于点 \( \left( {\frac{\pi }{2},0}\right) \) 对称

D. \( f\left( x\right) \) 在区间 \( \left\lbrack  {0,\pi }\right\rbrack \) 上有 2 个零点

## 题目勘误
无明显勘误。
## 思路分析
将函数化简为 \(f(x)=\sin x + \frac12\sin 2x = \sin x(1+\cos x)\)，然后分别对四个选项进行分析：利用周期定义判断 A；利用导数或三角变形求最值判断 B；利用对称中心的定义判断 C；通过解方程 \(f(x)=0\) 判断 D。
## 详细解答

函数 \(f(x)=\sin x + \frac{1}{2}\sin 2x\)，运用二倍角公式 \(\sin 2x = 2\sin x\cos x\)，得
\[
f(x)=\sin x + \frac12 \cdot 2\sin x\cos x = \sin x + \sin x\cos x = \sin x(1+\cos x).
\]

**选项 A：** 判断 \(\pi\) 是否为 \(f(x)\) 的周期。

若 \(\pi\) 为周期，则对任意 \(x\) 有 \(f(x+\pi)=f(x)\)。计算：
\[
f(x+\pi)=\sin(x+\pi)\bigl(1+\cos(x+\pi)\bigr) = (-\sin x)\bigl(1-\cos x\bigr) = -\sin x(1-\cos x).
\]
而 \(f(x)=\sin x(1+\cos x)\)。取 \(x=\frac{\pi}{2}\)，\(f(\frac{\pi}{2})=1\cdot(1+0)=1\)，\(f(\frac{3\pi}{2}) = -1\cdot(1-0)=-1\)，两者不相等。故 \(\pi\) 不是周期。A 错误。

**选项 B：** 求 \(f(x)\) 的最大值。

\(f(x)=\sin x + \frac12\sin 2x\)，求导：
\[
f'(x)=\cos x + \cos 2x = \cos x + 2\cos^2 x -1 = (2\cos x -1)(\cos x +1).
\]
令 \(f'(x)=0\)，得 \(\cos x = \frac12\) 或 \(\cos x = -1\)。
当 \(\cos x = \frac12\) 时，\(\sin x = \pm\frac{\sqrt3}{2}\)，此时
\[
f(x)=\sin x\Bigl(1+\frac12\Bigr) = \frac32\sin x = \pm\frac{3\sqrt3}{4}.
\]
最大值为 \(\frac{3\sqrt3}{4}\)。而 \(\frac{3\sqrt3}{4} \approx 1.299 < 1.5 = \frac32\)，故 \(\frac32\) 不是最大值。B 错误。

**选项 C：** 判断图象是否关于点 \(\left(\frac{\pi}{2},0\right)\) 对称。

若关于点 \((a,0)\) 对称，则 \(f(a+x)+f(a-x)=0\) 恒成立，且必有 \(f(a)=0\)。
计算 \(f\left(\frac{\pi}{2}\right)=\sin\frac{\pi}{2} + \frac12\sin\pi = 1 \neq 0\)，图象不过该点，不可能是对称中心。C 错误。

**选项 D：** 求 \(f(x)\) 在 \([0,\pi]\) 上的零点个数。

由 \(f(x)=\sin x(1+\cos x)=0\)，得
\[
\sin x = 0 \quad \text{或} \quad 1+\cos x = 0.
\]
在 \([0,\pi]\) 上，
- \(\sin x = 0 \Rightarrow x=0,\ \pi\).
- \(1+\cos x = 0 \Rightarrow \cos x = -1 \Rightarrow x=\pi\).
综上，零点为 \(x=0\) 和 \(x=\pi\)，共 \(2\) 个。D 正确。

## 答案
D

## 总结
本题以复合音为背景，考查三角函数的恒等变形、周期性、最值、对称性与零点。关键是将函数转化为 \(f(x)=\sin x(1+\cos x)\)，再利用基本性质和导数工具逐项判断，注意对称中心必须位于函数图像上且满足对称条件。
