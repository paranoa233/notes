# 5-9-3 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-9_三角形中的爪型图问题/5-9-3.md|5-9-3]]

## 完整题目

3.(2023 浙江绍兴期中)在 \( \bigtriangleup  {ABC} \) 中，角 \( A \) ， \( B \) ， \( C \) 所对的边分别为 \( a, b, c,\angle {ABC} = \frac{\pi }{3},\angle {ABC} \) 的平分线交 \( {AC} \) 于点 \( D \) ，且 \( {BD} = \sqrt{3} \) ，则 \( {4a} + c \) 的最小值为___.

## 题目勘误
无明显勘误。

## 思路分析
已知三角形一边所对的角、该角平分线的长度，求另两边线性表达式的最小值。关键步骤：
1. 利用角平分线长度公式（用两边及夹角表示），结合已知条件建立边 \(a\) 与 \(c\) 的方程。
2. 化简得到 \(a\) 与 \(c\) 的倒数关系 \(\frac{1}{a}+\frac{1}{c}=1\)。
3. 将目标式 \(4a+c\) 乘以 \(\left(\frac{1}{a}+\frac{1}{c}\right)\) 或消元后利用基本不等式求最值（或直接使用柯西/基本不等式）。

## 详细解答
在 \(\triangle ABC\) 中，已知 \(\angle ABC = \frac{\pi}{3}\)，角 \(B\) 的平分线 \(BD\) 交 \(AC\) 于点 \(D\)，且 \(BD = \sqrt{3}\)。
记 \(a = BC\)，\(c = AB\)，\(B = \frac{\pi}{3}\)。
由**角平分线长度公式**（用两边及夹角表示）：
\[
BD = \frac{2ac \cos\frac{B}{2}}{a + c}
\]
代入 \(B = \frac{\pi}{3}\)，则 \(\cos\frac{B}{2} = \cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}\)，且 \(BD = \sqrt{3}\)，得
\[
\sqrt{3} = \frac{2ac \cdot \frac{\sqrt{3}}{2}}{a + c} = \frac{\sqrt{3}\,ac}{a + c}
\]
两边约去 \(\sqrt{3}\)：
\[
1 = \frac{ac}{a + c} \quad\Longrightarrow\quad a + c = ac
\]
两边同除以 \(ac\)（显然 \(a>0, c>0\)）：
\[
\frac{1}{a} + \frac{1}{c} = 1
\]
目标：求 \(4a + c\) 的最小值。利用基本不等式：
\[
4a + c = (4a + c)\left(\frac{1}{a} + \frac{1}{c}\right) = 4 + \frac{4a}{c} + \frac{c}{a} + 1 = 5 + \frac{4a}{c} + \frac{c}{a}
\]
由均值不等式：
\[
\frac{4a}{c} + \frac{c}{a} \ge 2\sqrt{\frac{4a}{c} \cdot \frac{c}{a}} = 4
\]
当且仅当 \(\frac{4a}{c} = \frac{c}{a}\)，即 \(c = 2a\) 时取等。
等号成立时，结合 \(\frac{1}{a} + \frac{1}{c} = 1\)，得 \(\frac{1}{a} + \frac{1}{2a} = \frac{3}{2a} = 1\)，解得 \(a = \frac{3}{2}\)，\(c = 3\)，满足三角形边角条件。
因此
\[
4a + c \ge 5 + 4 = 9
\]
故最小值为 \(9\)。

## 答案
\(9\)

## 总结
本题核心是**角平分线长度的向量或边长公式**，将已知的角平分线长转化为边的关系 \(\frac{1}{a}+\frac{1}{c}=1\)，再通过“乘‘1’法”或消元结合基本不等式求线性表达式的最值，这是处理“\(ka + mc\) 在 \(\frac{1}{a}+\frac{1}{c}=1\) 条件下的最值”的典型思路。
