# 5-9-5 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-9_三角形中的爪型图问题/5-9-5.md|5-9-5]]

## 完整题目

5.(2024 安徽黄山一检)记 \( \bigtriangleup  {ABC} \) 的内角 \( A \) ， \( B \) ， \( C \) 的对边分别为 \( a \) ， \( b \) ， \( c \) ，其外接圆半径为 \( 2\sqrt{3} \) ，且 \( \cos {2B} = 2 - 5\cos \left( {A + C}\right) \) ,则角 \( B \) 的大小为___. 若点 \( D \) 在边 \( {AC} \) 上， \( {DC} = {2AD} \) ， \( {BD} \) =2，则 \( \bigtriangleup  {ABC} \) 的面积为___.(本题第一空 2 分, 第二空 3 分)

## 题目勘误
无明显勘误。

## 思路分析
第一空：利用三角形内角和关系将 \(\cos(A+C)\) 化为 \(-\cos B\)，代入已知等式得到关于 \(\cos B\) 的一元二次方程，解方程并结合 \(B\in(0,\pi)\) 求角 \(B\)。  
第二空：先用正弦定理及外接圆半径求出边 \(b=AC=6\)，由 \(DC=2AD\) 得 \(AD=2,\,DC=4\)；再利用向量关系或两次余弦定理建立关于边 \(a,c\) 的方程组，结合第一问的角 \(B\) 求出 \(a,c\)，最后用面积公式 \(S=\frac12 ac\sin B\) 计算。

## 详细解答

**第一空：求角 \(B\)**

在 \(\triangle ABC\) 中，\(A+C=\pi-B\)，所以  
\[
\cos(A+C)=\cos(\pi-B)=-\cos B.
\]  
已知  
\[
\cos 2B = 2 - 5\cos(A+C),
\]  
代入得  
\[
\cos 2B = 2 - 5(-\cos B) = 2 + 5\cos B.
\]  
又由二倍角公式 \(\cos 2B = 2\cos^2 B - 1\)，得  
\[
2\cos^2 B - 1 = 2 + 5\cos B,
\]  
整理得  
\[
2\cos^2 B - 5\cos B - 3 = 0.
\]  
因式分解：  
\[
(2\cos B + 1)(\cos B - 3) = 0,
\]  
解得 \(\cos B = -\dfrac12\) 或 \(\cos B = 3\)（舍去）．  
因为 \(B\in(0,\pi)\)，所以  
\[
B = \frac{2\pi}{3}\ \text{（或 \(120^\circ\)）}.
\]  
故第一空填：\(\dfrac{2\pi}{3}\)（或 \(120^\circ\)）．

---

**第二空：求 \(\triangle ABC\) 的面积**

（1）**求边 \(AC\)（即 \(b\)）**  
由正弦定理 \(b = 2R\sin B\)，已知外接圆半径 \(R = 2\sqrt{3}\)，\(B=\dfrac{2\pi}{3}\)，  
\[
b = 2 \times 2\sqrt{3} \times \sin\frac{2\pi}{3} = 4\sqrt{3} \times \frac{\sqrt{3}}{2} = 6.
\]  
所以 \(AC = b = 6\)．  

（2）**标出线段长度**  
点 \(D\) 在边 \(AC\) 上，且 \(DC = 2AD\)，故  
\[
AD = \frac13 AC = 2,\qquad DC = \frac23 AC = 4.
\]  

（3）**建立关于 \(a=BC\)，\(c=AB\) 的方程组**  
设 \(\overrightarrow{BA},\ \overrightarrow{BC}\) 为基底．由 \(AD:DC=1:2\) 得  
\[
\overrightarrow{BD} = \overrightarrow{BA} + \overrightarrow{AD}
= \overrightarrow{BA} + \frac13\overrightarrow{AC}
= \overrightarrow{BA} + \frac13(\overrightarrow{BC} - \overrightarrow{BA})
= \frac23\overrightarrow{BA} + \frac13\overrightarrow{BC}.
\]  
两边平方：  
\[
BD^2 = \Bigl(\frac23 c\Bigr)^2 + \Bigl(\frac13 a\Bigr)^2 + 2\cdot\frac23\cdot\frac13\cdot \overrightarrow{BA}\cdot\overrightarrow{BC}.
\]  
其中 \(\overrightarrow{BA}\cdot\overrightarrow{BC} = a c \cos B = a c \cos\frac{2\pi}{3} = -\frac12 ac\)．  
已知 \(BD = 2\)，代入：  
\[
4 = \frac49 c^2 + \frac19 a^2 + \frac49\Bigl(-\frac12 ac\Bigr)
= \frac49 c^2 + \frac19 a^2 - \frac29 ac.
\]  
两边乘 \(9\)：  
\[
36 = 4c^2 + a^2 - 2ac. \tag{1}
\]  
在 \(\triangle ABC\) 中，由余弦定理，  
\[
b^2 = a^2 + c^2 - 2ac\cos B,
\]  
代入 \(b=6\)，\(\cos\frac{2\pi}{3}=-\frac12\)，得  
\[
36 = a^2 + c^2 + ac. \tag{2}
\]  

（4）**解方程组**  
\((1)-(2)\)：  
\[
(4c^2 + a^2 - 2ac) - (a^2 + c^2 + ac) = 0 \implies 3c^2 - 3ac = 0,
\]  
即 \(3c(c-a)=0\)，由 \(c>0\) 得 \(c = a\)．  
代入 \((2)\)：  
\[
a^2 + a^2 + a^2 = 3a^2 = 36 \implies a^2 = 12 \implies a = 2\sqrt{3}.
\]  
故 \(a = c = 2\sqrt{3}\)．  

（5）**求面积**  
\[
S_{\triangle ABC} = \frac12 ac\sin B
= \frac12 \times 2\sqrt{3} \times 2\sqrt{3} \times \sin\frac{2\pi}{3}
= \frac12 \times 12 \times \frac{\sqrt{3}}{2}
= 3\sqrt{3}.
\]

## 答案
第一空：\(\displaystyle \frac{2\pi}{3}\)（或 \(120^\circ\)）；  
第二空：\(3\sqrt{3}\)．

## 总结
本题是三角形中“爪型图”（中线或分角线）与边角关系的综合题．核心方法：第一空利用三角形内角和将多角三角函数转化为单角方程求解；第二空结合正弦定理、向量分点关系（或两次余弦定理）建立方程组，求出边长，进而得面积．注意向量形式的活用可简化运算．
