# 5-9-6 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-9_三角形中的爪型图问题/5-9-6.md|5-9-6]]

## 完整题目

6.(2024 河南郑州二模)在 \( \bigtriangleup  {ABC} \) 中， \( A \) ， \( B \) ， \( C \) 的对边分别为 \( a, b, c \) ，已知 \( a = \sqrt{2}, b = 4, c\cos B \; + a = 0 \) ,则边 \( c = \) ___，点 \( D \) 在线段 \( {AB} \) 上， 且 \( \angle {CDA} = \frac{3\pi }{4} \) ，则 \( {CD} = \) ___.

## 题目勘误
无明显勘误。

## 思路分析
本题考查解三角形中的求边长及“爪子型”线段长度问题。第一空利用已知条件 \(c\cos B + a = 0\)，结合余弦定理求出边 \(c\)。第二空已知点 \(D\) 在 \(AB\) 上，且 \(\angle CDA = \frac{3\pi}{4}\)，求 \(CD\) 的长度。可选择在 \(\triangle ACD\) 或 \(\triangle BCD\) 中使用正弦定理，也可借助面积关系。由于三边已知，可先求出一组内角的正弦、余弦值，再结合已知角和公共边 \(CD\) 在两个三角形中分别建立关系求解。

## 详细解答
**第一空：求边 \(c\)**
由余弦定理，\(\cos B = \dfrac{a^2 + c^2 - b^2}{2ac}\)，代入已知条件 \(c\cos B + a = 0\) 得：
\[
c \cdot \frac{a^2 + c^2 - b^2}{2ac} + a = 0
\]
化简为 \(\dfrac{a^2 + c^2 - b^2}{2a} + a = 0\)，两边乘 \(2a\)：
\[
a^2 + c^2 - b^2 + 2a^2 = 0 \quad \Rightarrow \quad 3a^2 + c^2 - b^2 = 0.
\]
代入 \(a = \sqrt{2},\; b = 4\)：
\[
3 \times 2 + c^2 - 16 = 0 \quad \Rightarrow \quad c^2 = 10.
\]
因边长取正值，故 \(c = \sqrt{10}\)。

**第二空：求 \(CD\) 的长**
由三边 \(a=\sqrt{2},\; b=4,\; c=\sqrt{10}\)，利用余弦定理求角 \(A\) 与角 \(B\) 的正弦值。
\[
\cos A = \frac{b^2 + c^2 - a^2}{2bc} = \frac{16 + 10 - 2}{2 \times 4 \times \sqrt{10}} = \frac{24}{8\sqrt{10}} = \frac{3}{\sqrt{10}} = \frac{3\sqrt{10}}{10},
\]
\[
\sin A = \sqrt{1 - \cos^2 A} = \frac{1}{\sqrt{10}} = \frac{\sqrt{10}}{10}.
\]
\[
\cos B = \frac{a^2 + c^2 - b^2}{2ac} = \frac{2 + 10 - 16}{2 \times \sqrt{2} \times \sqrt{10}} = \frac{-4}{2\sqrt{20}} = -\frac{1}{\sqrt{5}} = -\frac{\sqrt{5}}{5},
\]
\[
\sin B = \sqrt{1 - \cos^2 B} = \frac{2}{\sqrt{5}} = \frac{2\sqrt{5}}{5}.
\]

点 \(D\) 在 \(AB\) 上，故 \(\angle CDB = \pi - \angle CDA = \frac{\pi}{4}\)。

在 \(\triangle ACD\) 中，\(\angle CDA = \frac{3\pi}{4}\)，由正弦定理：
\[
\frac{CD}{\sin A} = \frac{AC}{\sin \angle CDA} = \frac{4}{\sin \frac{3\pi}{4}} = \frac{4}{\frac{\sqrt{2}}{2}} = 4\sqrt{2}.
\]
所以
\[
CD = 4\sqrt{2} \cdot \sin A = 4\sqrt{2} \cdot \frac{\sqrt{10}}{10} = \frac{4\sqrt{20}}{10} = \frac{8\sqrt{5}}{10} = \frac{4\sqrt{5}}{5}.
\]

（验证：在 \(\triangle BCD\) 中，\(\angle CDB = \frac{\pi}{4}\)，由正弦定理：
\[
\frac{CD}{\sin B} = \frac{BC}{\sin \angle CDB} = \frac{\sqrt{2}}{\frac{\sqrt{2}}{2}} = 2,
\]
得 \(CD = 2 \sin B = 2 \cdot \frac{2\sqrt{5}}{5} = \frac{4\sqrt{5}}{5}\)，结果一致，答案可靠。）

## 答案
\(c = \sqrt{10}\)，\(CD = \dfrac{4\sqrt{5}}{5}\).

## 总结
本题属于三角形“爪型图”问题，核心方法为：
1. 灵活运用余弦定理将边角混合条件转化为纯边的关系，从而求出未知边；
2. 在已知三边的基础上，准确求出一组内角的正弦与余弦值；
3. 对含公共边的两个小三角形，分别使用正弦定理，并利用补角关系（\(\angle CDA + \angle CDB = \pi\)）建立等量关系，可相互验证，提高正确率。
注意在求解过程中关注三角形内角的范围，避免出现不合题意的解。
