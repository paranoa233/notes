# 5-9-8 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-9_三角形中的爪型图问题/5-9-8.md|5-9-8]]

## 完整题目

8. (2024 安徽六校教育研究会二测) 在 \( \bigtriangleup {ABC} \) 中,内角 \( A, B, C \) 的对边分别为 \( a, b, c,\sin (B + \; \left. \frac{\pi }{6}\right)  = \frac{a + b}{2c} \) .

(1)求角 \( C \) 的大小；

(2)若 \( a + b = 2\sqrt{3}, c = \sqrt{6} \) ，求角 \( C \) 的平分线 \( {CD} \) 的长度.

## 题目勘误
题目公式中 `\sin (B + \; \left. \frac{\pi }{6}\right)` 存在多余排版符号，应为 `\(\sin\left(B+\frac{\pi}{6}\right)\)`。其余文字清晰，无明显勘误。

## 思路分析
(1) 已知条件给出边角混合等式，先用正弦定理将边化为角，得到只含角的方程。再通过展开、整理并利用 \( \sin B \neq 0 \) ，可消去变量 \( B \)，得到关于 \( C \) 的三角方程，从而解得角 \( C \)。

(2) 已知两边和与第三边及角 \( C \)，先用余弦定理求出 \( ab \) 的值。求角平分线长可选用面积分割法或角平分线长公式 \( CD = \frac{2ab}{a+b}\cos\frac{C}{2} \)，代入即可。

## 详细解答
### (1) 求角 \( C \) 的大小
由正弦定理 \( \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \)，得
\[
a = 2R\sin A,\quad b = 2R\sin B,\quad c = 2R\sin C.
\]
代入已知等式 \( \sin\!\left(B + \frac{\pi}{6}\right) = \frac{a+b}{2c} \)，得
\[
\sin\!\left(B + \frac{\pi}{6}\right) = \frac{\sin A + \sin B}{2\sin C}.
\]
在 \( \triangle ABC \) 中，\( A = \pi - (B+C) \)，所以 \( \sin A = \sin(B+C) \)。于是
\[
\sin\!\left(B + \frac{\pi}{6}\right) = \frac{\sin(B+C) + \sin B}{2\sin C}.
\]
将左边展开，右边分子用和差化积：
\[
\sin(B+C) + \sin B = 2\sin\!\left(B + \frac{C}{2}\right)\cos\frac{C}{2}.
\]
分母 \( 2\sin C = 4\sin\frac{C}{2}\cos\frac{C}{2} \)，因此
\[
\frac{\sin(B+C) + \sin B}{2\sin C} = \frac{2\sin\!\left(B + \frac{C}{2}\right)\cos\frac{C}{2}}{4\sin\frac{C}{2}\cos\frac{C}{2}} = \frac{\sin\!\left(B + \frac{C}{2}\right)}{2\sin\frac{C}{2}}.
\]
所以原方程化为
\[
\sin\!\left(B + \frac{\pi}{6}\right) = \frac{\sin\!\left(B + \frac{C}{2}\right)}{2\sin\frac{C}{2}}.
\]
将左边用和角公式展开：
\[
\sin B\cos\frac{\pi}{6} + \cos B\sin\frac{\pi}{6} = \frac{\sqrt{3}}{2}\sin B + \frac{1}{2}\cos B.
\]
将右边也展开：
\[
\frac{1}{2\sin\frac{C}{2}}\left( \sin B\cos\frac{C}{2} + \cos B\sin\frac{C}{2} \right) = \frac{\cos\frac{C}{2}}{2\sin\frac{C}{2}}\sin B + \frac{1}{2}\cos B.
\]
两边对比 \( \sin B \) 和 \( \cos B \) 的系数，由 \( \cos B \) 的系数得 \( \frac{1}{2} = \frac{1}{2} \) 恒成立；由 \( \sin B \) 的系数得
\[
\frac{\sqrt{3}}{2} = \frac{\cos\frac{C}{2}}{2\sin\frac{C}{2}} \quad\Longrightarrow\quad \cot\frac{C}{2} = \sqrt{3}.
\]
由 \( 0 < C < \pi \) 知 \( 0 < \frac{C}{2} < \frac{\pi}{2} \)，故
\[
\frac{C}{2} = \frac{\pi}{6} \quad\Longrightarrow\quad C = \frac{\pi}{3}.
\]
（亦可直接由 \( \sqrt{3}\sin C - \cos C = 1 \) 得到 \( \sin(C - \frac{\pi}{6}) = \frac{1}{2} \)，结果一致。）

### (2) 求角平分线 \( CD \) 的长度
已知 \( a+b = 2\sqrt{3} \)，\( c = \sqrt{6} \)，\( C = \frac{\pi}{3} \)。
由余弦定理：
\[
c^2 = a^2 + b^2 - 2ab\cos C = (a+b)^2 - 2ab(1+\cos C).
\]
代入已知数据：
\[
(\sqrt{6})^2 = (2\sqrt{3})^2 - 2ab\left(1 + \cos\frac{\pi}{3}\right),
\]
\[
6 = 12 - 2ab\left(1 + \frac{1}{2}\right) = 12 - 3ab,
\]
解得 \( ab = 2 \)。

由角平分线长公式（可由面积分割推导）：
\[
CD = \frac{2ab}{a+b}\cos\frac{C}{2}.
\]
代入 \( a+b = 2\sqrt{3} \)，\( ab = 2 \)，\( \frac{C}{2} = \frac{\pi}{6} \)：
\[
CD = \frac{2 \times 2}{2\sqrt{3}} \cos\frac{\pi}{6} = \frac{4}{2\sqrt{3}} \times \frac{\sqrt{3}}{2} = \frac{2}{\sqrt{3}} \times \frac{\sqrt{3}}{2} = 1.
\]
所以角平分线 \( CD \) 的长度为 \( 1 \)。

## 答案
(1) \( C = \dfrac{\pi}{3} \)（或 \( 60^\circ \)）；
(2) \( CD = 1 \).

## 总结
本题属于三角形中的“爪型图”问题。第一问通过正弦定理实现边角互化，抓住 \( \sin B \neq 0 \) 消去变量 \( B \) 得到角 \( C \) 的方程。第二问求角平分线长，可结合余弦定理先求出两边之积 \( ab \)，再套用角平分线长公式或面积分割法快速求解。此类问题注意三角恒等变形的方向（统一角度、消元）以及基本几何量的计算公式即可。
