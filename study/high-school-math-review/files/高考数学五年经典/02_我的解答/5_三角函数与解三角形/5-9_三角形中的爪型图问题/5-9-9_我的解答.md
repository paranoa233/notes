# 5-9-9 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-9_三角形中的爪型图问题/5-9-9.md|5-9-9]]

## 完整题目

9.(2024 江苏南京、盐城一模)在 \( \bigtriangleup  {ABC} \) 中， \( \sin \left( {B - A}\right)  + \sqrt{2}\sin A = \sin C. \)

(1)求角 \( B \) 的大小；

(2)延长 \( {BC} \) 至点 \( M \) ，使得 \( 2\overrightarrow{BC} = \overrightarrow{CM} \) ，若 \( \angle {CAM} \; = \frac{\pi }{4} \) ,求 \( \angle {BAC} \) 的大小.

## 题目勘误
原文中 \(\bigtriangleup {ABC}\) 符号不规范，建议更正为 \(\triangle ABC\)，理由：三角形符号的标准 LaTeX 写法为 `\triangle`。
## 思路分析
(1) 利用三角形内角和 \(C = \pi - A - B\) 及三角恒等式将已知等式化为仅含 \(A, B\) 的形式，化简后求出 \(\cos B\)，进而得 \(B\) 的大小。  
(2) 由向量条件确定 \(C\) 在 \(B, M\) 之间且 \(CM = 2BC\)，利用邻补角关系得到 \(\angle ACM = \pi - \angle ACB\)，从而表达出 \(\triangle ACM\) 的内角。在 \(\triangle ACM\) 和 \(\triangle ABC\) 中分别对公共边 \(AC\) 使用正弦定理，建立方程求出 \(\angle BAC\)，需注意多解情况的合理性。
## 详细解答
### (1) 求角 \(B\)
在 \(\triangle ABC\) 中，由内角和 \(A + B + C = \pi\)，得 \(C = \pi - (A + B)\)，则 \(\sin C = \sin(A + B) = \sin A\cos B + \cos A\sin B\)。  
已知 \(\sin(B - A) + \sqrt{2}\sin A = \sin C\)，代入得：
\[
\sin B\cos A - \cos B\sin A + \sqrt{2}\sin A = \sin A\cos B + \cos A\sin B.
\]
移项合并：
\[
\sin B\cos A - \cos A\sin B - \cos B\sin A + \sqrt{2}\sin A - \sin A\cos B = 0,
\]
即
\[
0 - 2\cos B\sin A + \sqrt{2}\sin A = 0,
\]
整理得
\[
\sin A(\sqrt{2} - 2\cos B) = 0.
\]
在 \(\triangle ABC\) 中，\(0 < A < \pi\)，\(\sin A > 0\)，故
\[
\sqrt{2} - 2\cos B = 0 \implies \cos B = \frac{\sqrt{2}}{2}.
\]
又 \(0 < B < \pi\)，所以 \(B = \dfrac{\pi}{4}\)。

### (2) 求 \(\angle BAC\)
由延长条件 \(2\overrightarrow{BC} = \overrightarrow{CM}\)，知 \(C\) 在线段 \(BM\) 上，且 \(BC : CM = 1 : 2\)。设 \(BC = x\)，则 \(CM = 2x\)，\(BM = 3x\)。  
在 \(\triangle ABC\) 中，\(B = \dfrac{\pi}{4}\)，令 \(\angle BAC = A\)，则 \(\angle ACB = C = \pi - A - \dfrac{\pi}{4} = \dfrac{3\pi}{4} - A\)。  
因为 \(B, C, M\) 共线且 \(C\) 在中间，\(\angle ACM\) 与 \(\angle ACB\) 互为邻补角，所以
\[
\angle ACM = \pi - \angle ACB = \pi - \left(\frac{3\pi}{4} - A\right) = \frac{\pi}{4} + A.
\]
在 \(\triangle ACM\) 中，已知 \(\angle CAM = \dfrac{\pi}{4}\)，故
\[
\angle AMC = \pi - \angle CAM - \angle ACM = \pi - \frac{\pi}{4} - \left(\frac{\pi}{4} + A\right) = \frac{\pi}{2} - A.
\]

在 \(\triangle ACM\) 中，由正弦定理：
\[
\frac{AC}{\sin\angle AMC} = \frac{CM}{\sin\angle CAM},
\]
即
\[
\frac{AC}{\sin\left(\frac{\pi}{2} - A\right)} = \frac{2x}{\sin\frac{\pi}{4}}.
\]
由 \(\sin\left(\frac{\pi}{2} - A\right) = \cos A\)，\(\sin\dfrac{\pi}{4} = \dfrac{\sqrt{2}}{2}\)，得
\[
\frac{AC}{\cos A} = \frac{2x}{\frac{\sqrt{2}}{2}} = 2\sqrt{2}\,x,
\]
所以
\[
AC = 2\sqrt{2}\,x \cos A. \tag{1}
\]

在 \(\triangle ABC\) 中，由正弦定理：
\[
\frac{AC}{\sin B} = \frac{BC}{\sin A},
\]
即
\[
\frac{AC}{\sin\frac{\pi}{4}} = \frac{x}{\sin A}.
\]
由 \(\sin\dfrac{\pi}{4} = \dfrac{\sqrt{2}}{2}\)，得
\[
AC = \frac{x \cdot \frac{\sqrt{2}}{2}}{\sin A} = \frac{\sqrt{2}\,x}{2\sin A}. \tag{2}
\]

联立 (1)(2) 得
\[
2\sqrt{2}\,x \cos A = \frac{\sqrt{2}\,x}{2\sin A}.
\]
由于 \(x > 0\)，两边同除以 \(\sqrt{2}\,x\)：
\[
2\cos A = \frac{1}{2\sin A} \implies 4\sin A\cos A = 1 \implies 2\sin 2A = 1,
\]
所以
\[
\sin 2A = \frac{1}{2}.
\]
在 \(\triangle ABC\) 中，\(0 < A < \pi - B = \dfrac{3\pi}{4}\)，于是 \(0 < 2A < \dfrac{3\pi}{2}\)。  
满足 \(\sin 2A = \dfrac{1}{2}\) 的 \(2A\) 为 \(\dfrac{\pi}{6}\) 或 \(\dfrac{5\pi}{6}\)，故
\[
A = \frac{\pi}{12} \quad \text{或} \quad A = \frac{5\pi}{12}.
\]
经检验，两个值对应的三角形内角均符合条件（\(A = 15^\circ\) 时 \(C = 120^\circ\)，\(A = 75^\circ\) 时 \(C = 60^\circ\)），且均满足 \(\triangle ACM\) 的边角关系，故均成立。
## 答案
(1) \(B = \dfrac{\pi}{4}\)  
(2) \(\angle BAC = \dfrac{\pi}{12}\) 或 \(\dfrac{5\pi}{12}\)
## 总结
本题是三角形中的爪型图问题，关键步骤：  
- 第(1)问灵活运用内角和与三角恒等式化简求角；  
- 第(2)问需正确解读向量条件得出边的比例，结合邻补角转换三角形内角，再在两个三角形中借助公共边与正弦定理建立方程；  
- 处理三角方程时要注意内角取值范围，全面检验多解情况。
