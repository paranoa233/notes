# 5-12-10 我的解答

## 完整题目

1.(2021全国甲卷理，8)2020 年 12 月 8 日，中国和尼泊尔联合公布珠穆朗玛峰最新高程为 8848.86 (单位: \( \mathrm{m} \) ),三角高程测量法是珠峰高程测量方法之一. 如图是三角高程测量法的一个示意图, 现有 \( A, B, C \) 三点,且 \( A, B, C \) 在同一水平面上的投影 \( {A}^{\prime },{B}^{\prime },{C}^{\prime } \) 满足 \( \angle {A}^{\prime }{C}^{\prime }{B}^{\prime } = {45}^{ \circ  } \) , \( \angle {A}^{\prime }{B}^{\prime }{C}^{\prime } = {60}^{ \circ  } \) . 由点 \( C \) 测得点 \( B \) 的仰角为 \( {15}^{ \circ  } \) , \( B{B}^{\prime } \) 与 \( C{C}^{\prime } \) 的差为 100,由点 \( B \) 测得点 \( A \) 的仰角为 \( {45}^{ \circ  } \) ,则 \( A, C \) 两点到水平面 \( {A}^{\prime }{B}^{\prime }{C}^{\prime } \) 的高度差 \( A{A}^{\prime } - C{C}^{\prime } \) 约为 \( \left( {\sqrt{3} \approx  {1.732}}\right) \) ( )

## 题目勘误
无明显勘误。

## 思路分析
本题是一个立体几何与三角函数的实际应用题，主要涉及仰角、投影关系以及解三角形。关键是将实际高度差和水平距离转化为三角形中的边角关系：
1. 根据仰角和高度差求出水平距离 \(C'B'\)。
2. 在水平投影三角形 \(A'B'C'\) 中，利用已知角度和正弦定理求出 \(A'B'\) 的长度。
3. 再利用另一仰角关系求出 \(A, C\) 两点的高度差 \(AA' - CC'\)。
通过代入近似值即可得到答案。

## 详细解答
设 \(A, B, C\) 在水平面上的投影分别为 \(A', B', C'\)，记 \(AA' = h_A\)，\(BB' = h_B\)，\(CC' = h_C\)。

在点 \(C\) 测得点 \(B\) 的仰角为 \(15^\circ\)，则水平距离为 \(C'B'\)，高度差为 \(h_B - h_C\)，满足：
\[
\tan 15^\circ = \frac{h_B - h_C}{C'B'}
\]
已知 \(BB'\) 与 \(CC'\) 的差为 \(100\)（由题意可知 \(h_B - h_C = 100\)），故：
\[
C'B' = \frac{100}{\tan 15^\circ}
\]
计算 \(\tan 15^\circ = \tan(45^\circ - 30^\circ) = \frac{1 - \frac{\sqrt{3}}{3}}{1 + \frac{\sqrt{3}}{3}} = 2 - \sqrt{3}\)，所以：
\[
C'B' = \frac{100}{2 - \sqrt{3}} = 100(2 + \sqrt{3})
\]

在水平面三角形 \(A'B'C'\) 中，已知 \(\angle A'C'B' = 45^\circ\)，\(\angle A'B'C' = 60^\circ\)，则：
\[
\angle C'A'B' = 180^\circ - 45^\circ - 60^\circ = 75^\circ
\]
由正弦定理：
\[
\frac{A'B'}{\sin 45^\circ} = \frac{C'B'}{\sin 75^\circ}
\]
因此：
\[
A'B' = C'B' \cdot \frac{\sin 45^\circ}{\sin 75^\circ}
\]
代入 \(\sin 45^\circ = \frac{\sqrt{2}}{2}\)，\(\sin 75^\circ = \sin(45^\circ + 30^\circ) = \frac{\sqrt{6} + \sqrt{2}}{4}\)：
\[
\frac{\sin 45^\circ}{\sin 75^\circ} = \frac{\frac{\sqrt{2}}{2}}{\frac{\sqrt{6} + \sqrt{2}}{4}} = \frac{2}{\sqrt{3} + 1} = \sqrt{3} - 1
\]
所以：
\[
A'B' = 100(2 + \sqrt{3})(\sqrt{3} - 1) = 100(\sqrt{3} + 1)
\]

在点 \(B\) 测得点 \(A\) 的仰角为 \(45^\circ\)，水平距离为 \(A'B'\)，高度差为 \(h_A - h_B\)，满足：
\[
\tan 45^\circ = \frac{h_A - h_B}{A'B'} = 1 \implies h_A - h_B = A'B'
\]
于是，\(A, C\) 两点到水平面的高度差为：
\[
h_A - h_C = (h_A - h_B) + (h_B - h_C) = A'B' + 100 = 100(\sqrt{3} + 1) + 100 = 100\sqrt{3} + 200
\]
取 \(\sqrt{3} \approx 1.732\)，得：
\[
h_A - h_C \approx 100 \times 1.732 + 200 = 173.2 + 200 = 373.2
\]
与选项比对，最接近的是 \(373\)。

## 答案
B

## 总结
本题将仰角与投影三角形结合起来，考查了立体几何与解三角形的综合应用。解题的关键在于将空间高度差转化为水平距离，再通过水平投影三角形的边角关系（正弦定理）求出所需长度。涉及 \(\tan 15^\circ\) 和 \(\sin 75^\circ\) 的计算时，灵活运用两角和差公式能有效避免计算错误。近似计算时注意题目给定的 \(\sqrt{3} \approx 1.732\) 以提高精度。
