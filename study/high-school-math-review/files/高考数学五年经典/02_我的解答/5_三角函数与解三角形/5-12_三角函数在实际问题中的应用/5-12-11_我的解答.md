## 题目勘误
图片信息需人工核对。题目文本中的选项部分出现严重 OCR 识别乱码，如：  
原文“表高×表距一表高”“表高×表距+一表距”不可读。  
建议更正：参照原题图片，四个选项应形如：  
A. \(\dfrac{\text{表高}\times\text{表距}}{\text{表目距的差}}+\text{表高}\)  
B. \(\dfrac{\text{表高}\times\text{表距}}{\text{表目距的差}}-\text{表高}\)  
C. \(\dfrac{\text{表高}\times\text{表距}}{\text{表目距的差}}+\text{表距}\)  
D. \(\dfrac{\text{表高}\times\text{表距}}{\text{表目距的差}}-\text{表距}\)  
理由：原题选自《海岛算经》重差法，结论公式为上述形式之一。

## 思路分析
本题源于魏晋《海岛算经》的重差法测高，图形中 \(DE\) 和 \(FG\) 为等高的“表高”（设为 \(h\)），\(EG\) 为“表距”（设为 \(d\)），\(EH\) 和 \(GC\) 分别为两表的“表目距”，它们的差 \(GC-EH\) 称为“表目距的差”（设为 \(\delta\)），目标是求海岛的高 \(AB\)。  
利用相似三角形建立比例关系，消去中间变量（如 \(AH\) 等），推出 \(AB\) 关于 \(h,d,\delta\) 的表达式。

## 详细解答
如图，因为 \(DE\perp AC\)，\(FG\perp AC\)，\(AB\perp AC\)，所以 \(DE\parallel FG\parallel AB\)。

由 \(\triangle DEH \sim \triangle ABH\)，得：
\[
\frac{DE}{AB}=\frac{EH}{AH}
\]
即
\[
\frac{h}{AB}=\frac{EH}{AH} \tag{1}
\]

由 \(\triangle FGC \sim \triangle ABC\)，得：
\[
\frac{FG}{AB}=\frac{GC}{AC}
\]
即
\[
\frac{h}{AB}=\frac{GC}{AC} \tag{2}
\]

由 (1) 得 \(AH = \frac{AB \cdot EH}{h}\)；由 (2) 得 \(AC = \frac{AB \cdot GC}{h}\)。

又 \(AC - AH = HC\)，但从图可知 \(HC = EG + GC - EH\)？这里我们直接用另一组比例更方便：  
过 \(G\) 作 \(AC\) 的平行线或直接利用重差原理：  
注意到 \(EG = AC - AH - (GC - EH)\)？简单推导如下：  
\[
AC = AH + HC
\]
而 \(HC = HG + GC\)，\(HG = EG - EH\)？实际上 \(AH + HG = AG\)，\(AG + GC = AC\)。  
更直接利用相似消去 \(AH\) 和 \(AC\)：

由 (1)、(2) 分别得：
\[
AH = \frac{h}{AB}\cdot EH,\qquad AC = \frac{h}{AB}\cdot GC
\]
两式相减：
\[
AC - AH = \frac{h}{AB}(GC - EH) = \frac{h}{AB}\cdot\delta
\]
而由图形几何关系，\(AC - AH = HC\)，且 \(HC = EG\)？仔细看图：点顺序为 \(A, E, H, G, C\) 或类似。根据题意，\(E,H,G\) 在 \(AC\) 上。结合“表距 \(EG\)”，可知 \(E\) 和 \(G\) 之间距离为 \(d\)。  
观察有：\(AH + HE = AE\)，等等。其实重差法经典结论中 \(AC - AH\) 并不直接等于 \(d\)，而是 \(AC - AH = (AG + GC) - (AE + EH)\) …  
我们用更标准的重差推导：考察 \(DF\) 与 \(AB\) 延长线。这里直接给出代数处理：  
由 (1)：\(\frac{AB}{h} = \frac{AH}{EH}\) → \(AB = h\cdot\frac{AH}{EH}\)  
由 (2)：\(\frac{AB}{h} = \frac{AC}{GC}\) → \(AB = h\cdot\frac{AC}{GC}\)  
不妨设 \(AH = x\)，\(AC = y\)，则 \(y - x = AC - AH = HC = (HE + EG + GC) - EH = EG + GC\)？这样比例式会变得复杂。  
经典重差公式推导如下：因为 \(\frac{h}{AB} = \frac{EH}{AH} \Rightarrow \frac{1}{AB} = \frac{EH}{h\cdot AH}\)，同理 \(\frac{1}{AB} = \frac{GC}{h\cdot AC}\)。  
两式相减不能直接得公式。常用方法：  
令 \(AB = H\)，由 (1) \(AH = \frac{EH}{h}H\)，由 (2) \(AC = \frac{GC}{h}H\)。  
又因为 \(AC - AH = (AG + GC) - (AE + EH)\)，而 \(AG - AE = EG = d\)，所以  
\[
AC - AH = d + GC - EH = d + \delta
\]
于是：
\[
\frac{GC}{h}H - \frac{EH}{h}H = d + \delta
\]
\[
\frac{\delta}{h}H = d + \delta
\]
\[
H = \frac{h(d + \delta)}{\delta} = \frac{hd}{\delta} + h
\]
即海岛的高 \(AB = \frac{\text{表高} \times \text{表距}}{\text{表目距的差}} + \text{表高}\)。

## 答案
\[
AB = \frac{\text{表高} \times \text{表距}}{\text{表目距的差}} + \text{表高}
\]
对应原题选项应为上述形式（具体字母选项需对照原图，按 OCR 常规为 A 项）。

## 总结
本题以古代数学名著《海岛算经》为背景，实际考查利用相似三角形建立比例方程以及代数变形求解高度。关键点在于正确识图、准确用差量消去不可直接测量的中间线段长度，最终得到重差公式。这类“高远”测量问题的通法是构造两组相似比，通过作差引入已知的“表距”和“表目距的差”。
