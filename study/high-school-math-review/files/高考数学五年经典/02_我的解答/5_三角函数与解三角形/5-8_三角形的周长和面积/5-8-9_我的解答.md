# 5-8-9 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-8_三角形的周长和面积/5-8-9.md|5-8-9]]

## 完整题目

9.(2024 广东广州一模)记 \( \bigtriangleup  {ABC} \) 的内角 \( A, B \) ， \( C \) 的对边分别为 \( a, b, c,\bigtriangleup {ABC} \) 的面积为 \( S \) . 已知 \( S =  - \frac{\sqrt{3}}{4}\left( {{a}^{2} + {c}^{2} - {b}^{2}}\right) . \)

(1)求角 \( B \) 的大小；

(2)若点 \( D \) 在边 \( {AC} \) 上，且 \( \angle {ABD} = \frac{\pi }{2},{AD} = \; {2DC} = 2 \) ,求 \( \bigtriangleup {ABC} \) 的周长.

## 题目勘误
无明显勘误。

## 思路分析
(1) 已知条件等式同时涉及三角形的面积 \(S\) 和边的平方和差 \(a^{2}+c^{2}-b^{2}\)，自然联想到余弦定理与面积公式，将两者代入化简即可求出角 \(B\)。
(2) 由条件 \(AD=2DC=2\) 得 \(AD=2, DC=1, AC=3\)。已知 \(B=\dfrac{2\pi}{3}\)，\(\angle ABD = \dfrac{\pi}{2}\)，因此 \(\triangle ABD\) 为直角三角形，\(\angle DBC = 30^{\circ}\)。利用直角三角形的边角关系与 \(\triangle ABC\) 的正弦定理建立关于 \(c\) 和 \(A\) 的方程，解出 \(A\) 后可得 \(a,c\)，最后求出周长。

## 详细解答
**(1) 求角 \(B\)**  
由余弦定理得  
\[
a^{2}+c^{2}-b^{2}=2ac\cos B.
\]
三角形面积公式为  
\[
S=\frac{1}{2}ac\sin B.
\]
已知  
\[
S=-\frac{\sqrt{3}}{4}(a^{2}+c^{2}-b^{2}),
\]
代入得  
\[
\frac{1}{2}ac\sin B = -\frac{\sqrt{3}}{4}\cdot 2ac\cos B = -\frac{\sqrt{3}}{2}ac\cos B.
\]
因为 \(ac\neq 0\)，两边约去 \(ac\) 并整理：  
\[
\frac{1}{2}\sin B = -\frac{\sqrt{3}}{2}\cos B \quad\Rightarrow\quad \tan B = -\sqrt{3}.
\]
又 \(0<B<\pi\)，故  
\[
B = \frac{2\pi}{3}.
\]

**(2) 求 \(\triangle ABC\) 的周长**  
由题意，点 \(D\) 在边 \(AC\) 上，且 \(AD=2DC=2\)，所以  
\[
AD=2,\quad DC=1,\quad AC = b = 3.
\]
又 \(B=\dfrac{2\pi}{3}\)，\(\angle ABD=\dfrac{\pi}{2}\)，则  
\[
\angle DBC = \angle ABC - \angle ABD = \frac{2\pi}{3}-\frac{\pi}{2}=\frac{\pi}{6}.
\]
在直角 \(\triangle ABD\) 中，\(\angle ABD=90^{\circ}\)，\(AD=2\) 为斜边，\(AB=c\) 为直角边，故  
\[
\cos A = \frac{AB}{AD}= \frac{c}{2}. \tag{1}
\]
在 \(\triangle ABC\) 中，由正弦定理：  
\[
\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = \frac{3}{\sin\frac{2\pi}{3}} = \frac{3}{\frac{\sqrt{3}}{2}} = 2\sqrt{3}.
\]
所以  
\[
a = 2\sqrt{3}\sin A,\quad c = 2\sqrt{3}\sin C.
\]
又 \(C = \pi - A - B = \pi - A - \frac{2\pi}{3} = \frac{\pi}{3} - A\)，于是  
\[
c = 2\sqrt{3}\sin\!\left(\frac{\pi}{3}-A\right) = 2\sqrt{3}\left(\sin\frac{\pi}{3}\cos A - \cos\frac{\pi}{3}\sin A\right)
= 2\sqrt{3}\left(\frac{\sqrt{3}}{2}\cos A - \frac{1}{2}\sin A\right) = 3\cos A - \sqrt{3}\sin A. \tag{2}
\]
将 (1) 式 \(c = 2\cos A\) 代入 (2) 式：  
\[
2\cos A = 3\cos A - \sqrt{3}\sin A \quad\Rightarrow\quad \sqrt{3}\sin A = \cos A \quad\Rightarrow\quad \tan A = \frac{\sqrt{3}}{3}.
\]
由 \(A\) 为三角形内角且 \(B\) 为钝角，可知 \(A\) 为锐角，故  
\[
A = \frac{\pi}{6}.
\]
从而  
\[
c = 2\cos\frac{\pi}{6} = \sqrt{3},\qquad
a = 2\sqrt{3}\sin\frac{\pi}{6} = \sqrt{3}.
\]
因此 \(\triangle ABC\) 的周长为  
\[
a+b+c = \sqrt{3}+3+\sqrt{3} = 3+2\sqrt{3}.
\]

## 答案
(1) \(B = \dfrac{2\pi}{3}\)；  
(2) \(\triangle ABC\) 的周长为 \(3+2\sqrt{3}\).

## 总结
本题综合考查余弦定理、面积公式、正弦定理以及直角三角形的边角关系。第 (1) 问直接代入公式即可；第 (2) 问关键在于利用 \(AD=2,\,DC=1\) 及 \(\angle ABD=90^{\circ}\) 将线段关系转化为角的关系，再结合正弦定理求出待定边。注意解题时合理利用直角三角形带来的简化条件，避免复杂运算。
