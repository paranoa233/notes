# 5-8-11 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-8_三角形的周长和面积/5-8-11.md|5-8-11]]

## 完整题目

2.(2022 新高考卷 II，18)记 \( \bigtriangleup  {ABC} \) 的内角 \( A \) ， \( B, C \) 的对边分别为 \( a, b, c \) ,分别以 \( a, b, c \) 为边长的三个正三角形的面积依次为 \( {S}_{1},{S}_{2},{S}_{3} \) . 已知 \( {S}_{1} - {S}_{2} + {S}_{3} = \frac{\sqrt{3}}{2},\sin B = \frac{1}{3}. \)

(1)求 \( \bigtriangleup  {ABC} \) 的面积；

(2)若 \( \sin A\sin C = \frac{\sqrt{2}}{3} \) ，求 \( b \) .

## 题目勘误
无明显勘误。
## 思路分析
（1）利用等边三角形面积公式将 \(S_1, S_2, S_3\) 用 \(a,b,c\) 表示，代入已知等式化简可得 \(a^2 + c^2 - b^2 = 2\)。由余弦定理知该表达式等于 \(2ac\cos B\)，从而得到 \(ac\cos B = 1\)。结合 \(\sin B = \frac{1}{3}\) 可算出 \(\cos B\)，并由此确定 \(\cos B > 0\)，继而求出 \(ac\) 及 \(\triangle ABC\) 的面积 \(S = \frac{1}{2}ac\sin B\)。（2）利用正弦定理将 \(ac\) 用 \(b\) 及 \(\sin A, \sin C\) 表示，代入已知的 \(\sin A\sin C\) 及（1）中求出的 \(ac\)，解关于 \(b\) 的方程即得。
## 详细解答
(1) 边长为 \(x\) 的正三角形的面积为 \(\frac{\sqrt3}{4}x^2\)，所以
\[
S_1 = \frac{\sqrt3}{4}a^2,\quad S_2 = \frac{\sqrt3}{4}b^2,\quad S_3 = \frac{\sqrt3}{4}c^2.
\]
代入已知等式 \(S_1 - S_2 + S_3 = \frac{\sqrt3}{2}\) 得
\[
\frac{\sqrt3}{4}(a^2 - b^2 + c^2) = \frac{\sqrt3}{2}
\implies a^2 + c^2 - b^2 = 2.
\]
由余弦定理 \(b^2 = a^2 + c^2 - 2ac\cos B\) 可得
\[
a^2 + c^2 - b^2 = 2ac\cos B.
\]
因此
\[
2ac\cos B = 2 \implies ac\cos B = 1. \tag{1}
\]
因为 \(a^2 + c^2 - b^2 = 2 > 0\)，由(1)知 \(\cos B > 0\)，又已知 \(\sin B = \frac{1}{3}\)，所以
\[
\cos B = \sqrt{1 - \sin^2 B} = \sqrt{1 - \frac{1}{9}} = \frac{2\sqrt2}{3}.
\]
代入(1)得
\[
ac = \frac{1}{\cos B} = \frac{3}{2\sqrt2} = \frac{3\sqrt2}{4}.
\]
于是 \(\triangle ABC\) 的面积为
\[
S_{\triangle ABC} = \frac{1}{2}ac\sin B = \frac{1}{2} \cdot \frac{3\sqrt2}{4} \cdot \frac{1}{3} = \frac{\sqrt2}{8}.
\]

(2) 由正弦定理 \(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R\)，可得
\[
a = 2R\sin A,\quad b = 2R\sin B,\quad c = 2R\sin C.
\]
从而
\[
ac = 4R^2 \sin A \sin C,\qquad b^2 = 4R^2 \sin^2 B.
\]
两式相除消去 \(4R^2\) 得
\[
\frac{ac}{b^2} = \frac{\sin A \sin C}{\sin^2 B}.
\]
已知 \(\sin A\sin C = \frac{\sqrt2}{3}\)，\(\sin B = \frac{1}{3}\)，且由(1)知 \(ac = \frac{3\sqrt2}{4}\)，代入上式：
\[
\frac{\frac{3\sqrt2}{4}}{b^2} = \frac{\frac{\sqrt2}{3}}{\left(\frac{1}{3}\right)^2} = \frac{\frac{\sqrt2}{3}}{\frac{1}{9}} = 3\sqrt2.
\]
解得
\[
b^2 = \frac{3\sqrt2}{4} \cdot \frac{1}{3\sqrt2} = \frac{1}{4},
\]
故 \(b = \frac{1}{2}\)（边长取正值）。
## 答案
(1) \(\triangle ABC\) 的面积为 \(\displaystyle\frac{\sqrt2}{8}\)；
(2) \(b = \displaystyle\frac{1}{2}\)。
## 总结
本题主要考查正三角形的面积公式、余弦定理、三角形面积公式以及正弦定理的综合运用。关键在于从条件中提炼出 \(a^2 + c^2 - b^2 = 2\)，并利用余弦定理将其与 \(ac\cos B\) 建立联系，进而求得 \(ac\)。同时需要注意通过中间量的符号（如 \(a^2 + c^2 - b^2 > 0\)）判断角 \(B\) 的范围，避免错漏。
