# 5-8-13 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-8_三角形的周长和面积/5-8-13.md|5-8-13]]

## 完整题目

4.(2024 新高考卷 II，15)记 \( \bigtriangleup  {ABC} \) 的内角 \( A, B, C \) 的对边分别为 \( a, b, c \) ，已知 \( \sin A + \sqrt{3}\cos A = 2 \) .

(1)求 \( A \) ；

(2)若 \( a = 2,\sqrt{2}b\sin C = c\sin {2B} \) ，求 \( \bigtriangleup  {ABC} \) 的周长.

## 题目勘误
无明显勘误。

## 思路分析
(1) 条件给出关于角 \(A\) 的三角方程，可利用辅助角公式将其化为正弦型函数，结合三角形内角的取值范围求出 \(A\)。
(2) 条件给出边角混合等式，先用正弦定理统一为角的等式，化简求出角 \(B\)，进而得到角 \(C\)。已知边 \(a\) 及三个角，再次利用正弦定理求出边 \(b\) 和 \(c\)，最后求和得周长。

## 详细解答
**(1) 求 \(A\)**
由已知 \(\sin A + \sqrt{3}\cos A = 2\)，
提取 2 得
\[
2\left( \frac{1}{2}\sin A + \frac{\sqrt{3}}{2}\cos A \right) = 2
\]
即
\[
2\sin\left(A + \frac{\pi}{3}\right) = 2
\]
所以
\[
\sin\left(A + \frac{\pi}{3}\right) = 1
\]
因为 \(A \in (0, \pi)\)，故 \(A + \frac{\pi}{3} \in \left( \frac{\pi}{3}, \frac{4\pi}{3} \right)\)。
在此区间内正弦值为 1 的角只有 \(\frac{\pi}{2}\)，所以
\[
A + \frac{\pi}{3} = \frac{\pi}{2} \implies A = \frac{\pi}{6}
\]

**(2) 求 \(\triangle ABC\) 的周长**
已知 \(a=2\)，\(\sqrt{2} b \sin C = c \sin 2B\)。
由正弦定理 \(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R\)，代入边 \(b, c\) 得
\[
\sqrt{2} \cdot 2R\sin B \cdot \sin C = 2R\sin C \cdot \sin 2B
\]
因为 \(\sin C > 0\)，两边约去 \(2R\sin C\)：
\[
\sqrt{2} \sin B = \sin 2B = 2\sin B \cos B
\]
又 \(\sin B > 0\)，两边约去 \(\sin B\)：
\[
\sqrt{2} = 2\cos B \implies \cos B = \frac{\sqrt{2}}{2}
\]
在 \((0,\pi)\) 内可得 \(B = \frac{\pi}{4}\)。

于是
\[
C = \pi - A - B = \pi - \frac{\pi}{6} - \frac{\pi}{4} = \frac{7\pi}{12}
\]

计算 \(a\) 对应的外接圆相关量：
\[
\frac{a}{\sin A} = \frac{2}{\sin\frac{\pi}{6}} = \frac{2}{\frac{1}{2}} = 4
\]
由正弦定理得
\[
b = 4\sin B = 4\sin\frac{\pi}{4} = 4 \cdot \frac{\sqrt{2}}{2} = 2\sqrt{2}
\]
\[
c = 4\sin C = 4\sin\frac{7\pi}{12}
\]
而 \(\sin\frac{7\pi}{12} = \sin\left(\frac{\pi}{3} + \frac{\pi}{4}\right) = \sin\frac{\pi}{3}\cos\frac{\pi}{4} + \cos\frac{\pi}{3}\sin\frac{\pi}{4}\)
\[
= \frac{\sqrt{3}}{2}\cdot\frac{\sqrt{2}}{2} + \frac{1}{2}\cdot\frac{\sqrt{2}}{2} = \frac{\sqrt{6} + \sqrt{2}}{4}
\]
所以
\[
c = 4 \cdot \frac{\sqrt{6} + \sqrt{2}}{4} = \sqrt{6} + \sqrt{2}
\]

故 \(\triangle ABC\) 的周长
\[
L = a + b + c = 2 + 2\sqrt{2} + \sqrt{6} + \sqrt{2} = 2 + 3\sqrt{2} + \sqrt{6}
\]

## 答案
(1) \( A = \dfrac{\pi}{6} \)  
(2) 周长为 \( 2 + 3\sqrt{2} + \sqrt{6} \)

## 总结
本题主要考查辅助角公式、正弦定理以及三角形内角的范围限制。解决此类边角混合问题时，先用正弦定理统一为角的关系往往能够简化计算。注意特殊角的三角函数值及和角公式的灵活应用。
