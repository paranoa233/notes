# 5-8-5 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-8_三角形的周长和面积/5-8-5.md|5-8-5]]

## 完整题目

5. (多选题, 2021 广东深圳外国语学校月考) 一个等腰直角三角形 \( {ABC} \) 内有一个内接等腰直角三角形 \( {PQR} \) (即 \( P, Q, R \) 三点分别在 \( \bigtriangleup {ABC} \) 三边或顶点上),则两三角形面积比 \( \frac{{S}_{\bigtriangleup {PRQ}}}{{S}_{\bigtriangleup {ABC}}} \) 的值可能为 ( )

A. \( \frac{1}{4} \) B. \( \frac{1}{5} \) C. \( \frac{1}{6} \) D. \( \frac{1}{7} \)

## 题目勘误
无明显勘误。

## 思路分析
本题为多选题，考查等腰直角三角形内接等腰直角三角形的面积比问题。大三角形为等腰直角三角形，内接三角形也是等腰直角三角形，且三个顶点分别落在大三角形的三条边上（可在顶点处）。由于图形的对称性与连续性，面积比通常可取一个区间内的所有值，需要在不同直角顶点位置下分类讨论，求出面积比的取值范围，再与选项比对。

设大三角形 \(ABC\)，\( \angle C = 90^\circ\)，\(AC=BC=1\)，面积 \(S_{\triangle ABC}=\frac12\)。以 \(C\) 为原点，\(CA\) 为 \(x\) 轴，\(CB\) 为 \(y\) 轴建系，则 \(A(1,0),\;B(0,1)\)，斜边 \(AB\) 所在直线为 \(x+y=1\)。

内接等腰直角三角形 \(PQR\) 的三个顶点应分别落在 \(AB,BC,CA\) 三条边上（或端点）。依据直角顶点所在的边分类：

1. 直角顶点在斜边 \(AB\) 上；
2. 直角顶点在直角边 \(CA\) 上（由对称性，直角顶点在 \(CB\) 上结果相同）。

在每一类中，设出坐标，利用等腰直角条件（垂直且两直角边等长）建立方程，解出参数关系，进而得到小三角形面积 \(S_{\triangle PQR} = \frac12 |PQ|^2\)，面积比 \(\lambda = |PQ|^2\)（因为大三角形面积为 \(\frac12\)），并确定 \(\lambda\) 的取值范围。

## 详细解答

**大三角形设定：**
设等腰直角三角形 \(ABC\)，\(\angle C=90^\circ\)，\(AC=BC=1\)，则面积 \(S_{\triangle ABC}= \frac12\)。
建系：\(C(0,0),\; A(1,0),\; B(0,1)\)，斜边 \(AB: x+y=1\;(0\le x\le 1)\)。

内接等腰直角三角形 \(PQR\)，三个顶点分别在 \(\triangle ABC\) 的三边上（可在端点）。记其面积 \(S_{\triangle PQR}\)，所求比值为 \(\lambda = \dfrac{S_{\triangle PQR}}{S_{\triangle ABC}} = 2S_{\triangle PQR}\)。

---

**情形一：直角顶点在斜边 \(AB\) 上**

设直角顶点为 \(P\)（在 \(AB\) 上），另外两个顶点 \(Q\in CA,\, R\in CB\)。
令 \(P(t,1-t),\; Q(s,0),\; R(0,u)\)，其中 \(t,s,u\in[0,1]\)。

\[
\overrightarrow{PQ} = (s-t,\, t-1),\quad \overrightarrow{PR} = (-t,\, u+t-1)
\]

等腰直角条件：
\[
\overrightarrow{PQ} \cdot \overrightarrow{PR} = 0 \quad\Rightarrow\quad -t(s-t) + (t-1)(u+t-1)=0 \tag{1}
\]
\[
|\overrightarrow{PQ}|^2 = |\overrightarrow{PR}|^2 \quad\Rightarrow\quad (s-t)^2 + (t-1)^2 = t^2 + (u+t-1)^2 \tag{2}
\]

引入斜率：设直线 \(PQ\) 斜率为 \(k\)（\(k\neq 0\)），则 \(PR\) 斜率为 \(-\frac1k\)。
由点斜式求交点：
\[
Q: y=0 \;\Rightarrow\; 0 = k(s-t) + (1-t) \;\Rightarrow\; s = t - \frac{1-t}{k}
\]
\[
R: x=0 \;\Rightarrow\; u = -\frac{1}{k}(0-t) + 1-t = \frac{t}{k} + 1-t
\]
计算两直角边长平方：
\[
|PQ|^2 = (s-t)^2 + (1-t)^2 = \frac{(1-t)^2}{k^2} + (1-t)^2 = (1-t)^2\left(1+\frac1{k^2}\right)
\]
\[
|PR|^2 = t^2 + \left(\frac{t}{k}\right)^2 = t^2\left(1+\frac1{k^2}\right)
\]
由 \(|PQ|=|PR|\) 得 \( (1-t)^2 = t^2 \Rightarrow 1-t = t\)，解得 \(t=\frac12\)。
此时 \(P\) 为 \(AB\) 中点 \(\left(\frac12,\frac12\right)\)。

代回交点坐标：
\[
s = \frac12 - \frac{1}{2k},\qquad u = \frac12 + \frac{1}{2k}
\]
要求 \(s\in[0,1],\, u\in[0,1]\)，得 \(\frac1k \in [-1,1]\)（\(k\to\infty\) 即 \(\frac1k=0\) 也允许，此时 \(PQ\perp x\) 轴）。
面积比（大三角形面积为 \(\frac12\)）：
\[
\lambda = |PQ|^2 = \frac14\left(1+\frac1{k^2}\right) \in \left[\frac14,\;\frac12\right]
\]
此情形下 \(\lambda\) 可取 \([\frac14,\frac12]\) 内的一切值。

---

**情形二：直角顶点在直角边 \(CA\) 上**

设直角顶点为 \(P(p,0)\) 在 \(CA\) 上，另外两点 \(Q(0,q)\) 在 \(CB\) 上、\(R(r,1-r)\) 在 \(AB\) 上，其中 \(p,q,r\in[0,1]\)。

\[
\overrightarrow{PQ} = (-p,\;q),\quad \overrightarrow{PR} = (r-p,\;1-r)
\]

等腰直角条件：
\[
\overrightarrow{PQ} \cdot \overrightarrow{PR} = -p(r-p) + q(1-r) = 0 \tag{3}
\]
\[
|\overrightarrow{PQ}|^2 = |\overrightarrow{PR}|^2 \;\Rightarrow\; p^2+q^2 = (r-p)^2 + (1-r)^2 \tag{4}
\]

几何上，\(\triangle PQR\) 为等腰直角三角形且直角在 \(P\)，等价于向量 \(\overrightarrow{PR}\) 可由 \(\overrightarrow{PQ}\) 旋转 \(90^\circ\) 得到。由 \(\overrightarrow{PQ}=(-p,q)\)，逆时针或顺时针旋转：
- 逆时针 \(90^\circ\)：\((-q,-p)\)，对应 \(\overrightarrow{PR}=(-q,-p)\)，但与 \(R\) 在 \(x+y=1\) 上且坐标非负验证无解；
- 顺时针 \(90^\circ\)：\((q,p)\)，对应 \(\overrightarrow{PR}=(q,p)\)。

因此有：
\[
r-p = q,\qquad 1-r = p
\]
解得：
\[
r = p+q,\qquad p = 1-r = 1-p-q \;\Rightarrow\; 2p = 1-q \;\Rightarrow\; p = \frac{1-q}{2}
\]
进而
\[
r = \frac{1+q}{2}
\]
由 \(p,q,r\in[0,1]\) 得 \(q\in[0,1]\)。

此时面积比：
\[
\lambda = |PQ|^2 = p^2 + q^2 = \left(\frac{1-q}{2}\right)^2 + q^2 = \frac{1 -2q +5q^2}{4}
\]
关于 \(q\) 的二次函数在 \([0,1]\) 上的最小值为 \(q=\frac15=0.2\) 时：
\[
\lambda_{\min} = \frac{1 -2\cdot\frac15 +5\cdot\frac{1}{25}}{4} = \frac{1 -0.4 +0.2}{4} = \frac{0.8}{4} = \frac15
\]
最大值为端点 \(q=1\) 时 \(\lambda=1\)，\(q=0\) 时 \(\lambda=\frac14\)。
故此情形 \(\lambda \in \left[\frac15,\;1\right]\)，取遍区间内所有值。

由对称性，直角顶点在 \(CB\) 边上时，结论完全相同。

---

**综合取值范围：**
情形一提供区间 \(\left[\frac14,\;\frac12\right]\)
情形二提供区间 \(\left[\frac15,\;1\right]\)
整体可能取值 \(\lambda \in \left[\frac15,\;1\right]\)。

对比选项：
A. \(\frac14=0.25\) 在区间内；  
B. \(\frac15=0.2\) 在区间端点；  
C. \(\frac16\approx0.1667 < 0.2\)，不在区间内；  
D. \(\frac17\approx0.1429 < 0.2\)，不在区间内。

故可能取值为 A、B。

## 答案
A、B

## 总结
本题通过建立坐标系并分类讨论内接等腰直角三角形直角顶点所在边，将几何条件转化为代数方程，求出面积比的连续取值范围。关键点在于：
1. 根据三点分别落在三边上合理分配顶点位置；
2. 利用等腰直角对应的向量垂直且模长相等建立方程；
3. 挖掘参数的几何或代数范围，得到比值的区间；
4. 合并各类情形的区间后，判断选项是否包含在内。
最终得到面积比可能取 \( \left[\frac15,\;1\right] \) 内的任意值，因此 \( \frac14 \) 和 \( \frac15 \) 是可能的值。
