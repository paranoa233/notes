# 5-8-8 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-8_三角形的周长和面积/5-8-8.md|5-8-8]]

## 完整题目

8.(2021 湖北武汉 3 月质检)在 \( \bigtriangleup  {ABC} \) 中，已知它的内角 \( A, B, C \) 的对边分别为 \( a, b, c \) ,且 \( B = \; \frac{2\pi }{3}, b = \sqrt{6} \) .

(1)若 \( \cos A\cos C = \frac{2}{3} \) ，求 \( \bigtriangleup  {ABC} \) 的面积.

(2)试问 \( \frac{1}{a} + \frac{1}{c} = 1 \) 能否成立？若能成立，求此时 \( \bigtriangleup  {ABC} \) 的周长；若不能成立，请说明理由.

## 题目勘误
无明显勘误。
## 思路分析
第(1)问已知一边及其对角，以及另外两角余弦之积，可通过三角形内角和关系求出两角正弦之积，再结合正弦定理得到两边之积，最后代入面积公式。
第(2)问为存在性探究，可将条件转化为边的关系，再利用余弦定理或正弦定理判断对应的二次方程是否有符合三角形边角限制的解。
## 详细解答
(1) 在 \(\triangle ABC\) 中，\(B = \dfrac{2\pi}{3}\)，所以 \(A + C = \pi - B = \dfrac{\pi}{3}\)。
由正弦定理：
\[
\frac{a}{\sin A} = \frac{c}{\sin C} = \frac{b}{\sin B} = \frac{\sqrt{6}}{\sin\frac{2\pi}{3}} = \frac{\sqrt{6}}{\frac{\sqrt{3}}{2}} = 2\sqrt{2},
\]
故 \(a = 2\sqrt{2}\sin A,\; c = 2\sqrt{2}\sin C\)，两式相乘得
\[
ac = 8\sin A\sin C.
\]
已知 \(\cos A\cos C = \dfrac{2}{3}\)，又
\[
\cos(A + C) = \cos A\cos C - \sin A\sin C,
\]
而 \(A + C = \dfrac{\pi}{3}\)，\(\cos\dfrac{\pi}{3} = \dfrac{1}{2}\)，所以
\[
\frac{1}{2} = \frac{2}{3} - \sin A\sin C,
\]
解得 \(\sin A\sin C = \dfrac{2}{3} - \dfrac{1}{2} = \dfrac{1}{6}\)，从而
\[
ac = 8 \times \frac{1}{6} = \frac{4}{3}.
\]
因此 \(\triangle ABC\) 的面积
\[
S = \frac{1}{2}ac\sin B = \frac{1}{2} \times \frac{4}{3} \times \sin\frac{2\pi}{3} = \frac{2}{3} \times \frac{\sqrt{3}}{2} = \frac{\sqrt{3}}{3}.
\]

(2) 不能成立。理由如下：
由余弦定理得
\[
b^2 = a^2 + c^2 - 2ac\cos B,
\]
代入 \(b = \sqrt{6},\; B = \dfrac{2\pi}{3}\) 得
\[
6 = a^2 + c^2 - 2ac\left(-\frac{1}{2}\right) = a^2 + c^2 + ac. \quad \text{(i)}
\]
假设 \(\dfrac{1}{a} + \dfrac{1}{c} = 1\) 能成立，即 \(\dfrac{a + c}{ac} = 1\)，故
\[
a + c = ac. \quad \text{(ii)}
\]
将 (ii) 两边平方得 \(a^2 + c^2 + 2ac = (ac)^2\)，结合 (i) 有
\[
6 = (ac)^2 - 2ac + ac = (ac)^2 - ac,
\]
整理得 \((ac)^2 - ac - 6 = 0\)，解得 \(ac = 3\) 或 \(ac = -2\)（舍去）。
从而 \(a + c = 3\)，此时 \(a, c\) 为方程 \(x^2 - 3x + 3 = 0\) 的两根，
但其判别式 \(\Delta = (-3)^2 - 4 \times 3 = -3 < 0\)，不存在实数解，即不存在满足条件的正数 \(a, c\)。
因此 \(\dfrac{1}{a} + \dfrac{1}{c} = 1\) 不能成立。

## 答案
(1) \(\dfrac{\sqrt{3}}{3}\)；(2) 不能成立，理由见详细解答。
## 总结
本题围绕已知一边对角的三角形，考查了正弦定理、余弦定理与三角恒等变换的综合应用。第(1)问关键在于利用三角恒等变换求出 \(\sin A\sin C\)，进而得到两边之积；第(2)问把条件代数化后，结合基本定理由判别式或余弦值范围判定不存在符合条件的三角形。处理此类“能否成立”问题，几何推理与代数验算都可行，选择最简便的转化路径即可。
