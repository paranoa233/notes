# 5-8-6 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-8_三角形的周长和面积/5-8-6.md|5-8-6]]

## 完整题目

6.(2024 江苏南通如皋、连云港调研)在 \( \bigtriangleup  {ABC} \) 中,角 \( A, B, C \) 所对的边分别为 \( a, b, c \) ,若 \( a = 2, c \; = 3,\cos B = b\cos C \) ,点 \( P, Q \) 分别在边 \( {AB} \) 和 \( {CB} \) 上,且 \( {PQ} \) 把 \( \bigtriangleup {ABC} \) 的面积分成相等的两部分, 则 \( {PQ} \) 长的最小值为___.

## 题目勘误
原题目中等式“\(\cos B = b\cos C\)”存在量纲不一致问题（左边为无量纲数值，右边为长度量纲），怀疑为 OCR 识别遗漏字符。常见题型中等式应为“\(c\cos B = b\cos C\)”或“\(a\cos B = b\cos C\)”。结合已知边 \(a=2, c=3\)，若为“\(c\cos B = b\cos C\)”，可推出 \(B=C\)，三角形为等腰三角形，后续可解。若无图片核对，本解答按“\(c\cos B = b\cos C\)”进行。若原题为其他等式，需相应调整。（图片信息需人工核对）

## 思路分析
首先根据修正后的等式解三角形，求出边长和面积。接着设出点 \(P, Q\) 的位置参数，利用面积平分条件得出两边长乘积关系，然后用余弦定理表示 \(PQ\) 的长度，转化为函数求最值问题。

## 详细解答
**第一步：解三角形**
由正弦定理及条件 \(c\cos B = b\cos C\)，得
\[
\sin C\cos B = \sin B\cos C \implies \sin(C - B) = 0.
\]
因为 \(B, C \in (0, \pi)\)，所以 \(B = C\)，即 \(\triangle ABC\) 为等腰三角形，\(b = c = 3\)。
已知 \(a = 2\)，\(\triangle ABC\) 中，\(AB = AC = 3\)，底边 \(BC = 2\)。
由余弦定理，
\[
\cos B = \frac{a^2 + c^2 - b^2}{2ac} = \frac{2^2 + 3^2 - 3^2}{2 \times 2 \times 3} = \frac{4}{12} = \frac{1}{3}.
\]
\[
\sin B = \sqrt{1 - \cos^2 B} = \sqrt{1 - \frac{1}{9}} = \frac{2\sqrt{2}}{3}.
\]
\(\triangle ABC\) 的面积
\[
S_{\triangle ABC} = \frac{1}{2} ac \sin B = \frac{1}{2} \times 2 \times 3 \times \frac{2\sqrt{2}}{3} = 2\sqrt{2}.
\]

**第二步：建立变量关系**
点 \(P\) 在边 \(AB\) 上，点 \(Q\) 在边 \(CB\) 上（即 \(BC\) 上）。设 \(BP = x\)，\(BQ = y\)，则 \(x \in [0, 3]\)，\(y \in [0, 2]\)。
\(PQ\) 将 \(\triangle ABC\) 面积平分，即
\[
S_{\triangle PBQ} = \frac{1}{2} S_{\triangle ABC} = \sqrt{2}.
\]
又 \(S_{\triangle PBQ} = \frac{1}{2} xy \sin B = \frac{1}{2} xy \cdot \frac{2\sqrt{2}}{3} = \frac{\sqrt{2}}{3}xy\)，
所以
\[
\frac{\sqrt{2}}{3}xy = \sqrt{2} \implies xy = 3.
\]
由 \(y = \frac{3}{x} \le 2\) 得 \(x \ge \frac{3}{2}\)，又 \(x \le 3\)，故 \(x \in \left[\frac{3}{2}, 3\right]\)。

**第三步：求 \(PQ\) 的最小值**
在 \(\triangle PBQ\) 中，由余弦定理，
\[
PQ^2 = x^2 + y^2 - 2xy\cos B = x^2 + y^2 - 2 \times 3 \times \frac{1}{3} = x^2 + y^2 - 2.
\]
将 \(y = \frac{3}{x}\) 代入，得
\[
x^2 + y^2 = x^2 + \frac{9}{x^2}.
\]
令 \(t = x^2\)，由 \(x \in \left[\frac{3}{2}, 3\right]\) 得 \(t \in \left[\frac{9}{4}, 9\right]\)。函数 \(f(t) = t + \frac{9}{t}\) 在 \(t = 3\) 时取最小值 \(6\)（均值不等式 \(t + \frac{9}{t} \ge 2\sqrt{9} = 6\)，当且仅当 \(t = 3\) 时取等，此时 \(x = \sqrt{3} \in \left[\frac{3}{2}, 3\right]\) 满足）。
因此 \(x^2 + y^2\) 的最小值为 \(6\)，从而
\[
PQ_{\min}^2 = 6 - 2 = 4 \implies PQ_{\min} = 2.
\]

## 答案
\(2\)

## 总结
本题需通过边角关系判断三角形形状（等腰），利用面积比建立动线段端点的约束，最后转化为对勾函数求最值。注意等号取得的条件须在定义域内验证。原题条件“\(\cos B = b\cos C\)”疑有笔误，常见正确表述为“\(c\cos B = b\cos C\)”，审题时应结合量纲和常规命题思路灵活推断。
