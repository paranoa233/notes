# 5-8-15 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-8_三角形的周长和面积/5-8-15.md|5-8-15]]

## 完整题目

2.(2020 全国卷Ⅰ文，18) \( \bigtriangleup  {ABC} \) 的内角 \( A, B, C \) 的对边分别为 \( a, b, c \) . 已知 \( B = {150}^{ \circ  } \) .

(1)若 \( a = \sqrt{3}c, b = 2\sqrt{7} \) ，求 \( \bigtriangleup  {ABC} \) 的面积；

(2)若 \( \sin A + \sqrt{3}\sin C = \frac{\sqrt{2}}{2} \) ，求 \( C \) .

## 题目勘误
无明显勘误。

## 思路分析
（1）已知角 \(B\) 及其对边 \(b\)，以及 \(a\) 与 \(c\) 的倍数关系，可直接利用余弦定理建立关于 \(c\) 的方程，解出各边长，再用面积公式 \(S = \frac{1}{2}ac\sin B\) 求面积。  
（2）已知 \(B=150^\circ\)，可得 \(A+C=30^\circ\)，将 \(A\) 用 \(C\) 表示，代入所给等式，利用两角差的正弦公式化简，得到关于 \(C\) 的三角方程，结合角的范围求解。

## 详细解答
### (1) 求 \(\triangle ABC\) 的面积
由余弦定理  
\[
b^2 = a^2 + c^2 - 2ac\cos B
\]
将 \(B = 150^\circ\)（\(\cos150^\circ = -\frac{\sqrt{3}}{2}\)），\(a = \sqrt{3}c\)，\(b = 2\sqrt{7}\) 代入：
\[
(2\sqrt{7})^2 = (\sqrt{3}c)^2 + c^2 - 2(\sqrt{3}c)(c) \cdot \left(-\frac{\sqrt{3}}{2}\right)
\]
\[
28 = 3c^2 + c^2 + 3c^2
\]
\[
28 = 7c^2 \quad \Rightarrow \quad c^2 = 4
\]
解得 \(c = 2\)（负值舍去），则 \(a = \sqrt{3} \times 2 = 2\sqrt{3}\)。  
三角形的面积  
\[
S = \frac{1}{2}ac\sin B = \frac{1}{2} \times 2\sqrt{3} \times 2 \times \sin150^\circ = 2\sqrt{3} \times \frac{1}{2} = \sqrt{3}.
\]

### (2) 求角 \(C\)
在 \(\triangle ABC\) 中，\(B = 150^\circ\)，\(A + B + C = 180^\circ\)，所以  
\[
A + C = 30^\circ \quad \Rightarrow \quad A = 30^\circ - C.
\]
于是  
\[
\sin A = \sin(30^\circ - C) = \sin30^\circ\cos C - \cos30^\circ\sin C = \frac{1}{2}\cos C - \frac{\sqrt{3}}{2}\sin C.
\]
代入已知条件 \(\sin A + \sqrt{3}\sin C = \frac{\sqrt{2}}{2}\)：
\[
\left( \frac{1}{2}\cos C - \frac{\sqrt{3}}{2}\sin C \right) + \sqrt{3}\sin C = \frac{\sqrt{2}}{2}
\]
\[
\frac{1}{2}\cos C + \frac{\sqrt{3}}{2}\sin C = \frac{\sqrt{2}}{2}
\]
两边同乘 \(2\)：
\[
\cos C + \sqrt{3}\sin C = \sqrt{2}
\]
利用辅助角公式，左边可化为 \(2\sin(C + 30^\circ)\)（或 \(2\cos(C - 60^\circ)\)）：
\[
2\left( \frac{1}{2}\cos C + \frac{\sqrt{3}}{2}\sin C \right) = 2\sin(C + 30^\circ) = \sqrt{2}
\]
\[
\sin(C + 30^\circ) = \frac{\sqrt{2}}{2}.
\]
由 \(A = 30^\circ - C > 0\) 得 \(0^\circ < C < 30^\circ\)，故 \(30^\circ < C + 30^\circ < 60^\circ\)。  
在此范围内满足 \(\sin(C + 30^\circ) = \frac{\sqrt{2}}{2}\) 的角只有 \(45^\circ\)：
\[
C + 30^\circ = 45^\circ \quad \Rightarrow \quad C = 15^\circ.
\]

## 答案
（1）\(\sqrt{3}\)  
（2）\(15^\circ\)

## 总结
本题是解三角形的常规题型：第一问利用余弦定理与面积公式，注意特殊角的三角函数值；第二问利用三角形内角和消元，结合三角恒等变换与辅助角公式，最后必须根据已知角的范围合理取舍三角函数方程的解。
