# 5-2-10 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-2_三角函数的化简、求值/5-2-10.md|5-2-10]]

## 完整题目

1.(2021 新高考卷 I,6)若 \( \tan \theta  =  - 2 \) ，则 \( \frac{\sin \theta \left( {1 + \sin {2\theta }}\right) }{\sin \theta  + \cos \theta } = \) ( )

A. \( - \frac{6}{5} \) B. \( - \frac{2}{5} \)

C. \( \frac{2}{5} \) D. \( \frac{6}{5} \)

## 题目勘误
无明显勘误。
## 思路分析
本题已知 \(\tan \theta = -2\)，求一个含 \(\sin \theta\) 和 \(\cos \theta\) 的分式值。直接代入显然不现实，需要将目标式化简。观察到分子中含有 \(1 + \sin 2\theta\)，联想到二倍角公式和完全平方公式 \((\sin \theta + \cos \theta)^2 = 1 + \sin 2\theta\)，从而将分子与分母联系起来。化简后得到 \(\sin^2 \theta + \sin \theta \cos \theta\)，这是一个关于 \(\sin \theta\)、\(\cos \theta\) 的二次齐次式。利用平方关系 \(\sin^2 \theta + \cos^2 \theta = 1\) 构造分母，分子分母同除以 \(\cos^2 \theta\)，即可转化为只含 \(\tan \theta\) 的表达式，代入求值即可。
## 详细解答
因为 \(\tan \theta = -2 \neq -1\)，所以 \(\sin \theta + \cos \theta \neq 0\)，分母有意义。

利用二倍角公式及完全平方公式：
\[
1 + \sin 2\theta = \sin^2 \theta + \cos^2 \theta + 2\sin \theta \cos \theta = (\sin \theta + \cos \theta)^2
\]

代入原式：
\[
\frac{\sin \theta (1 + \sin 2\theta)}{\sin \theta + \cos \theta} = \frac{\sin \theta (\sin \theta + \cos \theta)^2}{\sin \theta + \cos \theta} = \sin \theta (\sin \theta + \cos \theta)
\]
展开得：
\[
\sin^2 \theta + \sin \theta \cos \theta
\]

此式为二次齐次式，由 \(\sin^2 \theta + \cos^2 \theta = 1\) 可得：
\[
\sin^2 \theta + \sin \theta \cos \theta = \frac{\sin^2 \theta + \sin \theta \cos \theta}{\sin^2 \theta + \cos^2 \theta}
\]

分子分母同除以 \(\cos^2 \theta\) (\(\cos \theta \neq 0\))：
\[
\frac{\tan^2 \theta + \tan \theta}{\tan^2 \theta + 1}
\]

将 \(\tan \theta = -2\) 代入：
\[
\frac{(-2)^2 + (-2)}{(-2)^2 + 1} = \frac{4 - 2}{4 + 1} = \frac{2}{5}
\]

原式值为 \(\dfrac{2}{5}\)。
## 答案
C
## 总结
本题的核心在于利用 \(1 + \sin 2\theta = (\sin \theta + \cos \theta)^2\) 进行化简，将分式转化为二次齐次式，再通过“弦化切”技巧（分子分母同除以 \(\cos^2 \theta\)）整体转化为 \(\tan \theta\) 的函数求值。此类题目要注意分母是否为零，以及三角函数值的符号（本题因化为二次齐次式求值，代入即可，无需单独讨论象限）。
