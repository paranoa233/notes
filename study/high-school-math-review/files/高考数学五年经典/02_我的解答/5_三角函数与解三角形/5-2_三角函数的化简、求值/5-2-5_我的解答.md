# 5-2-5 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-2_三角函数的化简、求值/5-2-5.md|5-2-5]]

## 完整题目

5.(2022 山东济宁质检)已知 \( \sin \alpha  - \cos \beta  = \; 3\cos \alpha  - 3\sin \beta \) ,且 \( \sin \left( {\alpha  + \beta }\right)  \neq  1 \) ,则 \( \sin (\alpha  - \; \beta ) = \) ___.

## 题目勘误
无明显勘误。
## 思路分析
已知等式为 \(\sin \alpha - \cos \beta = 3\cos \alpha - 3\sin \beta\)，可通过移项将其化为 \(\sin \alpha - 3\cos \alpha = \cos \beta - 3\sin \beta\)。两边结构相似，均可利用辅助角公式合并成一个三角函数：左边化为 \(\sqrt{10}\sin(\alpha - \varphi)\)，右边化为 \(\sqrt{10}\cos(\beta + \varphi)\)（其中辅助角 \(\varphi\) 满足 \(\cos\varphi = \frac{1}{\sqrt{10}},\ \sin\varphi = \frac{3}{\sqrt{10}}\)）。通过诱导公式将余弦转为正弦，得到 \(\sin(\alpha - \varphi) = \sin\bigl(\frac{\pi}{2} - \beta - \varphi\bigr)\)。由此得出两组角的关系，其中一组会导致 \(\sin(\alpha + \beta) = 1\)，与已知矛盾，故舍去；取另一组解出 \(\alpha - \beta\) 与 \(\varphi\) 的关系，最后用二倍角公式求出 \(\sin(\alpha - \beta)\) 的值。
## 详细解答
由已知等式
\[
\sin \alpha - \cos \beta = 3\cos \alpha - 3\sin \beta,
\]
移项整理得
\[
\sin \alpha - 3\cos \alpha = \cos \beta - 3\sin \beta. \tag{1}
\]
将 \((1)\) 式两边同时除以 \(\sqrt{1^2 + (-3)^2} = \sqrt{10}\)，并引入辅助角 \(\varphi\)，使得
\[
\cos\varphi = \frac{1}{\sqrt{10}},\quad \sin\varphi = \frac{3}{\sqrt{10}}.
\]
则
\[
\frac{1}{\sqrt{10}}\sin \alpha - \frac{3}{\sqrt{10}}\cos \alpha = \sin\alpha\cos\varphi - \cos\alpha\sin\varphi = \sin(\alpha - \varphi),
\]
\[
\frac{1}{\sqrt{10}}\cos \beta - \frac{3}{\sqrt{10}}\sin \beta = \cos\beta\cos\varphi - \sin\beta\sin\varphi = \cos(\beta + \varphi).
\]
于是 \((1)\) 式化为
\[
\sqrt{10}\sin(\alpha - \varphi) = \sqrt{10}\cos(\beta + \varphi),
\]
即
\[
\sin(\alpha - \varphi) = \cos(\beta + \varphi).
\]
利用诱导公式 \(\cos\theta = \sin\bigl(\frac{\pi}{2} - \theta\bigr)\)，得
\[
\sin(\alpha - \varphi) = \sin\Bigl(\frac{\pi}{2} - \beta - \varphi\Bigr). \tag{2}
\]
由 \((2)\) 式可得
\[
\alpha - \varphi = \frac{\pi}{2} - \beta - \varphi + 2k\pi \quad \text{或} \quad \alpha - \varphi = \pi - \Bigl(\frac{\pi}{2} - \beta - \varphi\Bigr) + 2k\pi,\quad k \in \mathbb{Z}.
\]
若取第一组解：
\[
\alpha - \varphi = \frac{\pi}{2} - \beta - \varphi + 2k\pi \;\Longrightarrow\; \alpha + \beta = \frac{\pi}{2} + 2k\pi,
\]
则
\[
\sin(\alpha + \beta) = \sin\Bigl(\frac{\pi}{2} + 2k\pi\Bigr) = 1,
\]
与已知条件 \(\sin(\alpha + \beta) \neq 1\) 矛盾，故舍去。
因此只能取第二组解：
\[
\alpha - \varphi = \pi - \frac{\pi}{2} + \beta + \varphi + 2k\pi = \frac{\pi}{2} + \beta + \varphi + 2k\pi.
\]
整理得
\[
\alpha - \beta = \frac{\pi}{2} + 2\varphi + 2k\pi.
\]
所以
\[
\sin(\alpha - \beta) = \sin\Bigl(\frac{\pi}{2} + 2\varphi + 2k\pi\Bigr) = \sin\Bigl(\frac{\pi}{2} + 2\varphi\Bigr) = \cos 2\varphi.
\]
由 \(\cos\varphi = \frac{1}{\sqrt{10}},\ \sin\varphi = \frac{3}{\sqrt{10}}\)，利用二倍角公式：
\[
\cos 2\varphi = \cos^2\varphi - \sin^2\varphi = \Bigl(\frac{1}{\sqrt{10}}\Bigr)^2 - \Bigl(\frac{3}{\sqrt{10}}\Bigr)^2 = \frac{1}{10} - \frac{9}{10} = -\frac{8}{10} = -\frac{4}{5}.
\]
故
\[
\sin(\alpha - \beta) = -\frac{4}{5}.
\]
## 答案
\[
\boxed{-\dfrac{4}{5}}
\]
## 总结
本题主要考查三角恒等变换，包括辅助角公式、诱导公式以及二倍角公式的应用。通过将已知等式变形并利用辅助角统一函数名称，得到角的关系后需注意根据题设限制舍去增根，体现了分类讨论的思想。求 \(\sin(\alpha - \beta)\) 的关键在于正确建立 \(\alpha - \beta\) 与辅助角 \(\varphi\) 之间的联系。
