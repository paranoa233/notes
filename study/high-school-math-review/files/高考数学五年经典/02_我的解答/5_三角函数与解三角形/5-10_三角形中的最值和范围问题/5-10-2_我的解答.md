# 5-10-2 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-10_三角形中的最值和范围问题/5-10-2.md|5-10-2]]

## 完整题目

2.(2024 江苏南京二模)在斜三角形 \( {ABC} \) 中，若 \( \sin A = \cos B \) ,则 \( 3\tan B + \tan C \) 的最小值为 ( )

A. \( \sqrt{2} \) B. \( \sqrt{5} \) C. \( \sqrt{6} \) D. \( 4\sqrt{3} \)

## 题目勘误
无明显勘误。
## 思路分析
在斜三角形 \(ABC\) 中，由条件 \(\sin A = \cos B\) 结合三角形内角和 \(A+B+C=\pi\)，推导出角的关系。由于斜三角形排除直角情况，分析得到唯一可能的关系 \(A = \frac{\pi}{2} + B\)，进而用 \(B\) 表示 \(C\)。将目标式 \(3\tan B + \tan C\) 化为关于 \(\tan B\) 的函数，利用基本不等式或对勾函数性质求最小值，注意定义域限制和取等条件验证。
## 详细解答
由 \(\sin A = \cos B = \sin\left(\frac{\pi}{2} - B\right)\)，在 \((0,\pi)\) 内正弦相等有两种情形：  
\[
A = \frac{\pi}{2} - B \quad \text{或} \quad A = \pi - \left(\frac{\pi}{2} - B\right) = \frac{\pi}{2} + B.
\]  
若 \(A = \frac{\pi}{2} - B\)，则 \(A + B = \frac{\pi}{2}\)，由内角和得 \(C = \pi - (A+B) = \frac{\pi}{2}\)，与“斜三角形”（非直角三角形）矛盾，故舍去。  
因此 \(A = \frac{\pi}{2} + B\)，又由 \(A + B + C = \pi\) 得  
\[
C = \pi - \left(\frac{\pi}{2} + B\right) - B = \frac{\pi}{2} - 2B.
\]  
由 \(B>0\)、\(C>0\) 得 \(0 < B < \frac{\pi}{4}\)。  
目标式为  
\[
y = 3\tan B + \tan C = 3\tan B + \tan\left(\frac{\pi}{2} - 2B\right) = 3\tan B + \cot 2B.
\]  
令 \(t = \tan B\)，由 \(B \in \left(0, \frac{\pi}{4}\right)\) 知 \(t \in (0, 1)\)。利用二倍角公式 \(\tan 2B = \frac{2t}{1-t^2}\)，有 \(\cot 2B = \frac{1-t^2}{2t}\)，代入得  
\[
y = 3t + \frac{1-t^2}{2t} = 3t + \frac{1}{2t} - \frac{t}{2} = \frac{5}{2}t + \frac{1}{2t} = \frac{1}{2}\left(5t + \frac{1}{t}\right).
\]  
对正数 \(t\)，由基本不等式  
\[
5t + \frac{1}{t} \geq 2\sqrt{5t \cdot \frac{1}{t}} = 2\sqrt{5},
\]  
当且仅当 \(5t = \frac{1}{t}\) 即 \(t = \frac{1}{\sqrt{5}}\) 时取等号，该值在 \((0,1)\) 内，满足定义域。  
所以 \(y\) 的最小值为  
\[
y_{\min} = \frac{1}{2} \times 2\sqrt{5} = \sqrt{5}.
\]
## 答案
B
## 总结
处理三角形中的最值问题，关键是将已知条件转化为内角关系，消元后将目标式化为单变量函数，结合定义域和不等式工具求解。本题通过排除直角三角形情形锁定角关系，利用正切二倍角公式和基本不等式得到结果，注意验证取等条件是否在三角形允许范围内。
