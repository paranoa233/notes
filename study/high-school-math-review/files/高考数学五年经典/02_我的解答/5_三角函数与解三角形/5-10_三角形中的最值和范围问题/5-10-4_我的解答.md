# 5-10-4 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-10_三角形中的最值和范围问题/5-10-4.md|5-10-4]]

## 完整题目

4. (2021 福建漳州平和第一中学期中) 已知在锐角三角形 \( {ABC} \) 中,角 \( A, B, C \) 的对边分别为 \( a \) , \( b, c \) ,若 \( {\sin }^{2}A - \sin B\sin C = 0 \) ,则 \( \frac{\sin B - \sin C}{2\sin A} \) 的取值范围是___.

## 题目勘误
无明显勘误。

## 思路分析
已知条件为 \(\sin^2 A = \sin B\sin C\)，由正弦定理可得边的关系 \(a^2 = bc\)。目标是求 \(\frac{\sin B-\sin C}{2\sin A} = \frac{b-c}{2a}\) 的取值范围。  
方法一（边的关系）：令 \(t=\frac{b}{c}>0\)，将目标化为 \(\frac{t-1}{2\sqrt{t}}\)，利用锐角三角形条件（余弦大于零）解出 \(t\) 的范围，再用函数单调性求值域。  
方法二（三角恒等变换）：利用和差化积及积化和差，将条件和目标均用角 \(A\) 表达，目标可化简为 \(\frac{1}{2}\sqrt{2\cos A-1}\) 的形式，结合锐角三角形对 \(A\) 的限制求出范围。方法二更简洁，本题采用方法二。

## 详细解答
在 \(\triangle ABC\) 中，由正弦定理 \(\frac{a}{\sin A}=\frac{b}{\sin B}=\frac{c}{\sin C}=2R\)，已知 \(\sin^2 A - \sin B\sin C = 0\)，即
\[
\sin^2 A = \sin B\sin C. \tag{1}
\]

目标式
\[
\frac{\sin B - \sin C}{2\sin A}.
\]
利用和差化积：
\[
\sin B - \sin C = 2\cos\frac{B+C}{2}\sin\frac{B-C}{2} = 2\cos\frac{\pi - A}{2}\sin\frac{B-C}{2} = 2\sin\frac{A}{2}\sin\frac{B-C}{2}.
\]
分母 \(2\sin A = 4\sin\frac{A}{2}\cos\frac{A}{2}\)，因此
\[
\frac{\sin B - \sin C}{2\sin A} = \frac{2\sin\frac{A}{2}\sin\frac{B-C}{2}}{4\sin\frac{A}{2}\cos\frac{A}{2}} = \frac{\sin\frac{B-C}{2}}{2\cos\frac{A}{2}}. \tag{2}
\]

再将条件 (1) 用积化和差变形：
\[
\sin B\sin C = \frac{1}{2}\bigl[\cos(B-C) - \cos(B+C)\bigr] = \frac{1}{2}\bigl[\cos(B-C) + \cos A\bigr].
\]
代入 (1) 得
\[
\sin^2 A = \frac{1}{2}\cos(B-C) + \frac{1}{2}\cos A.
\]
又 \(\sin^2 A = 1 - \cos^2 A\)，所以
\[
1 - \cos^2 A = \frac{1}{2}\cos(B-C) + \frac{1}{2}\cos A,
\]
整理得
\[
\cos(B-C) = -2\cos^2 A - \cos A + 2. \tag{3}
\]

由于三角形为锐角三角形，\(A, B, C \in \left(0, \frac{\pi}{2}\right)\)，且 \(B + C = \pi - A\)。
由 \(B, C < \frac{\pi}{2}\) 及 \(B + C = \pi - A\) 可得
\[
|B-C| < A < \frac{\pi}{2},
\]
余弦函数在 \((0, \frac{\pi}{2})\) 内单调递减，因此 \(\cos(B-C) > \cos A\)。
代入 (3)：
\[
-2\cos^2 A - \cos A + 2 > \cos A \implies -2\cos^2 A - 2\cos A + 2 > 0 \implies \cos^2 A + \cos A - 1 < 0.
\]
令 \(x = \cos A\)，结合 \(x \in (0,1)\)，解得 \(0 < x < \frac{\sqrt{5}-1}{2}\)。
同时，由 \(\cos(B-C) \le 1\)（当 \(B=C\) 时取等）得
\[
-2x^2 - x + 2 \le 1 \implies 2x^2 + x - 1 \ge 0 \implies (2x-1)(x+1) \ge 0.
\]
因 \(x>0\)，故 \(x \ge \frac{1}{2}\)。
综上，\(\cos A\) 的取值范围是
\[
x \in \left[\frac{1}{2}, \frac{\sqrt{5}-1}{2}\right).
\]
当 \(x = \frac{1}{2}\) 时，\(A = \frac{\pi}{3}\)，此时 \(\cos(B-C)=1\)，即 \(B=C\)，三角形为等边三角形；当 \(x \to \frac{\sqrt{5}-1}{2}\) 时，\(|B-C| \to A\)，有一角趋向直角，不能取等。

现在用 \(x\) 表示 (2) 中的目标式。由 (3)，\(\cos(B-C) = -2x^2 - x + 2\)，于是
\[
\sin\frac{B-C}{2} = \pm \sqrt{\frac{1 - \cos(B-C)}{2}} = \pm \sqrt{\frac{2x^2 + x - 1}{2}}.
\]
又 \(\cos\frac{A}{2} = \sqrt{\frac{1+x}{2}}\)，代入 (2) 得
\[
\frac{\sin B - \sin C}{2\sin A} = \pm \frac{1}{2} \sqrt{\frac{2x^2 + x - 1}{1+x}}.
\]
注意到 \(2x^2 + x - 1 = (2x-1)(x+1)\)，于是
\[
\sqrt{\frac{2x^2 + x - 1}{1+x}} = \sqrt{2x-1}.
\]
因此
\[
\frac{\sin B - \sin C}{2\sin A} = \pm \frac{1}{2}\sqrt{2x-1}, \quad x \in \left[\frac{1}{2}, \frac{\sqrt{5}-1}{2}\right).
\]
当 \(B>C\) 时取正号，\(B<C\) 时取负号，符号对称，故整体取值范围关于原点对称。

令 \(g(x) = \frac{1}{2}\sqrt{2x-1}\)，在 \(x \in \left[\frac{1}{2}, \frac{\sqrt{5}-1}{2}\right)\) 上单调递增。
- 当 \(x = \frac{1}{2}\) 时，\(g(x) = 0\)；
- 当 \(x \to \left(\frac{\sqrt{5}-1}{2}\right)^{-}\) 时，\(2x-1 \to \sqrt{5}-2\)，故 \(g(x) \to \frac{1}{2}\sqrt{\sqrt{5}-2}\)（取不到）。

因此原式的取值范围是
\[
\left( -\frac{1}{2}\sqrt{\sqrt{5}-2}, \frac{1}{2}\sqrt{\sqrt{5}-2} \right).
\]

## 答案
\[
\boxed{\left( -\frac{\sqrt{\sqrt{5}-2}}{2}, \frac{\sqrt{\sqrt{5}-2}}{2} \right)}
\]

## 总结
本题关键在于将正弦条件转化为边的关系或角的余弦关系，并紧紧抓住锐角三角形给出角的严格不等式来确定参数范围。利用三角恒等变换可将目标式化归为关于 \(\cos A\) 的简单根式函数，避免了复杂的边比讨论。注意验证等号是否可取，锐角三角形排除了直角边界。
