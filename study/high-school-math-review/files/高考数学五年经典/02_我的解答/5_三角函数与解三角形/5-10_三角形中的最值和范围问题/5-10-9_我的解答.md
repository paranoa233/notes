# 5-10-9 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-10_三角形中的最值和范围问题/5-10-9.md|5-10-9]]

## 完整题目

9.(2021 哈师大附中、东北师大附中、辽宁省实验中学联考) 在锐角三角形 \( {ABC} \) 中,角 \( A, B, C \) 的对边分别为 \( a, b, c \) ，已知 \( a\sin {2B} = b\sin A \) .

(1)若 \( a = 3, b = \sqrt{7} \) ，求 \( c \) ；

(2)求 \( \frac{a\cos C - c\cos A}{b} \) 的取值范围.

## 题目勘误
无明显勘误。
## 思路分析
第(1)问：由已知等式 `\( a\sin{2B}=b\sin A \)`，利用正弦定理化边为角，消去公共因式，结合锐角三角形的条件可求出角 `\( B \)`。已知 `\( a, b \)` 及 `\( B \)`，代入余弦定理求出 `\( c \)` 的值，并根据锐角三角形条件检验取舍。
第(2)问：先用正弦定理将待求式 `\( \frac{a\cos C - c\cos A}{b} \)` 转化为角的关系式，化简为 `\( \frac{\sin(A-C)}{\sin B} \)`。利用三角形内角和及第(1)问得到的 `\( B \)` 值，将表达式统一为关于 `\( A \)` 的三角函数。由锐角三角形条件确定 `\( A \)` 的取值范围，进而求得函数值域，即为所求取值范围。
## 详细解答
**第一步：由条件求角 `\( B \)`**
已知 `\( a\sin{2B} = b\sin A \)`，由正弦定理 `\( \frac{a}{\sin A} = \frac{b}{\sin B} = 2R \)`，得 `\( a = 2R\sin A \)`，`\( b = 2R\sin B \)`。
代入条件：
\[
2R\sin A \cdot \sin{2B} = 2R\sin B \cdot \sin A
\]
因为 `\( \sin A > 0 \)`（`\( A \)` 为三角形内角），两边同除以 `\( 2R\sin A \)`：
\[
\sin{2B} = \sin B \quad\Rightarrow\quad 2\sin B\cos B = \sin B
\]
又 `\( \sin B > 0 \)`，故 `\( 2\cos B = 1 \)`，得 `\( \cos B = \frac{1}{2} \)`。在锐角三角形中，`\( B = \frac{\pi}{3} \)`。

**第二步：解答第(1)问**
已知 `\( B = \frac{\pi}{3} \)，`\( a = 3 \)`，`\( b = \sqrt{7} \)`。
由余弦定理 `\( b^2 = a^2 + c^2 - 2ac\cos B \)`：
\[
(\sqrt{7})^2 = 3^2 + c^2 - 2 \times 3 \times c \times \frac{1}{2}
\]
\[
7 = 9 + c^2 - 3c \quad\Rightarrow\quad c^2 - 3c + 2 = 0
\]
解得 `\( c = 1 \)` 或 `\( c = 2 \)`。

检验锐角条件：
- 若 `\( c = 1 \)`，计算最大角（大边对大角，`\( a=3 \)` 最大）的余弦值：
  \[
  \cos A = \frac{b^2 + c^2 - a^2}{2bc} = \frac{7 + 1 - 9}{2\sqrt{7}} = -\frac{1}{2\sqrt{7}} < 0
  \]
  `\( A \)` 为钝角，与锐角三角形矛盾，舍去。
- 若 `\( c = 2 \)`，三边为 `\( a=3, b=\sqrt{7}, c=2 \)`，各角余弦均大于 `\( 0 \)`（如 `\( \cos C = \frac{9+4-7}{2\times3\times2} = \frac{1}{2}>0 \)`），满足条件。
综上，`\( c = 2 \)`。

**第三步：解答第(2)问**
求 `\( \frac{a\cos C - c\cos A}{b} \)` 的取值范围。
利用正弦定理 `\( a = 2R\sin A, b = 2R\sin B, c = 2R\sin C \)`，得：
\[
\frac{a\cos C - c\cos A}{b} = \frac{\sin A\cos C - \sin C\cos A}{\sin B} = \frac{\sin(A-C)}{\sin B}
\]
由第(1)问知 `\( B = \frac{\pi}{3} \)`，所以 `\( \sin B = \frac{\sqrt{3}}{2} \)，`\( A + C = \pi - \frac{\pi}{3} = \frac{2\pi}{3} \)`。将 `\( C = \frac{2\pi}{3} - A \)` 代入：
\[
\sin(A-C) = \sin\left[ A - \left(\frac{2\pi}{3} - A\right) \right] = \sin\left(2A - \frac{2\pi}{3}\right)
\]
于是原式化为：
\[
\frac{\sin(A-C)}{\sin B} = \frac{\sin\left(2A - \frac{2\pi}{3}\right)}{\frac{\sqrt{3}}{2}} = \frac{2}{\sqrt{3}}\sin\left(2A - \frac{2\pi}{3}\right)
\]

由锐角三角形条件：
`\( A \in \left(0, \frac{\pi}{2}\right) \)`，`\( C = \frac{2\pi}{3} - A \in \left(0, \frac{\pi}{2}\right) \Rightarrow A \in \left(\frac{\pi}{6}, \frac{\pi}{2}\right) \)`。
因此 `\( 2A - \frac{2\pi}{3} \)` 的取值范围为：
\[
2 \times \frac{\pi}{6} - \frac{2\pi}{3} = -\frac{\pi}{3}, \quad 2 \times \frac{\pi}{2} - \frac{2\pi}{3} = \frac{\pi}{3}
\]
由于 `\( A \)` 取不到端点，`\( 2A - \frac{2\pi}{3} \in \left(-\frac{\pi}{3}, \frac{\pi}{3}\right) \)`。
函数 `\( \sin x \)` 在 `\( \left(-\frac{\pi}{3}, \frac{\pi}{3}\right) \)` 上单调递增，其值域为 `\( \left(-\frac{\sqrt{3}}{2}, \frac{\sqrt{3}}{2}\right) \)`。
乘上系数 `\( \frac{2}{\sqrt{3}} \)`，即得原式的取值范围：
\[
\left( \frac{2}{\sqrt{3}} \times \left(-\frac{\sqrt{3}}{2}\right),\; \frac{2}{\sqrt{3}} \times \frac{\sqrt{3}}{2} \right) = (-1, 1)
\]

## 答案
(1) `\( c = 2 \)`；
(2) 取值范围为 `\( (-1, 1) \)`。

## 总结
本题的关键在于利用正弦定理实现边角互化，这是处理三角形中边角混合关系的主要思路。第(1)问通过解三角方程明确 `\( B \)` 角，再用余弦定理解出边长并检验锐角条件；第(2)问将代数式统一为角 `\( A \)` 的三角函数，结合锐角三角形内角的限制确定角度范围，最终转化为三角函数的值域问题。注意锐角三角形意味着每个内角均在 `\( (0,\frac{\pi}{2}) \)` 内，这是确定自变量范围的核心依据。
