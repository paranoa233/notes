# 5-6-10 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-6_三角函数的值域和最值/5-6-10.md|5-6-10]]

## 完整题目

1.(2022全国乙卷文，11)函数 \( f\left( x\right)  = \cos x + \; \left( {x + 1}\right) \sin x + 1 \) 在区间 \( \left\lbrack  {0,{2\pi }}\right\rbrack \) 上的最小值、最大值分别为 ( )

A. \( - \frac{\pi }{2},\frac{\pi }{2} \) B. \( - \frac{3\pi }{2},\frac{\pi }{2} \)

C. \( - \frac{\pi }{2},\frac{\pi }{2} + 2 \) D. \( - \frac{3\pi }{2},\frac{\pi }{2} + 2 \)

## 题目勘误
无明显勘误。
## 思路分析
函数 \(f(x)\) 为三角函数与多项式乘积的组合，定义在闭区间上。求最值可直接求导，利用 \(f'(x)=0\) 找到区间内的可能极值点，再与端点值比较大小。求导后表达式可以简化，驻点容易求出。
## 详细解答
函数为
\[
f(x)=\cos x+(x+1)\sin x+1,\quad x\in[0,2\pi].
\]
对 \(x\) 求导：
\[
\begin{aligned}
f'(x) &= -\sin x + \sin x + (x+1)\cos x \\
&= (x+1)\cos x.
\end{aligned}
\]
令 \(f'(x)=0\)，得
\[
(x+1)\cos x = 0.
\]
在区间 \([0,2\pi]\) 上：
- \(x+1=0 \Rightarrow x=-1\) 不在区间内；
- \(\cos x = 0 \Rightarrow x = \frac{\pi}{2}\) 或 \(x = \frac{3\pi}{2}\)，均在区间内。

计算区间端点及驻点处的函数值：
\[
\begin{aligned}
f(0) &= \cos 0 + (0+1)\sin 0 + 1 = 1 + 0 + 1 = 2, \\
f(2\pi) &= \cos 2\pi + (2\pi+1)\sin 2\pi + 1 = 1 + 0 + 1 = 2, \\
f\!\left(\frac{\pi}{2}\right) &= \cos\frac{\pi}{2} + \left(\frac{\pi}{2}+1\right)\sin\frac{\pi}{2} + 1 = 0 + \left(\frac{\pi}{2}+1\right)\cdot 1 + 1 = \frac{\pi}{2} + 2, \\
f\!\left(\frac{3\pi}{2}\right) &= \cos\frac{3\pi}{2} + \left(\frac{3\pi}{2}+1\right)\sin\frac{3\pi}{2} + 1 = 0 + \left(\frac{3\pi}{2}+1\right)\cdot(-1) + 1 = -\frac{3\pi}{2}.
\end{aligned}
\]
比较四个值：
\[
-\frac{3\pi}{2} < 2 < \frac{\pi}{2}+2,
\]
因此最小值为 \(-\frac{3\pi}{2}\)，最大值为 \(\frac{\pi}{2}+2\)。
## 答案
D
## 总结
对于三角与多项式混合型函数在闭区间上的最值，优先考虑导数法。本题求导后形式简洁，驻点由 \(\cos x=0\) 直接得到。计算端点与驻点函数值并比较即可。需注意特殊角的三角函数值以及代数运算的准确性。
