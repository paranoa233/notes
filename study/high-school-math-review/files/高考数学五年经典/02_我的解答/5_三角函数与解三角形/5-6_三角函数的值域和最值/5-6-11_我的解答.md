# 5-6-11 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-6_三角函数的值域和最值/5-6-11.md|5-6-11]]

## 完整题目

2.(2024全国甲卷文，13)函数 \( f\left( x\right)  = \sin x - \; \sqrt{3}\cos x \) 在 \( \left\lbrack  {0,\pi }\right\rbrack \) 上的最大值是___。

## 题目勘误
无明显勘误。

## 思路分析
将函数化为 \(f(x)=A\sin(x+\varphi)\) 的标准形式，利用辅助角公式合并 \(\sin x\) 与 \(\cos x\)。然后根据给定区间求出相位角的范围，结合正弦函数的图像与单调性，找到最大值点并计算最大值。

## 详细解答
函数为
\[
f(x) = \sin x - \sqrt{3}\cos x.
\]
提取 \(\sqrt{1^2 + (\sqrt{3})^2} = 2\)，利用辅助角公式：
\[
f(x) = 2\left( \frac{1}{2}\sin x - \frac{\sqrt{3}}{2}\cos x \right).
\]
由于 \(\cos\frac{\pi}{3} = \frac{1}{2},\; \sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}\)，所以
\[
f(x) = 2\left( \sin x \cos\frac{\pi}{3} - \cos x \sin\frac{\pi}{3} \right) = 2\sin\!\left(x - \frac{\pi}{3}\right).
\]
已知 \(x \in [0, \pi]\)，则
\[
x - \frac{\pi}{3} \in \left[-\frac{\pi}{3}, \frac{2\pi}{3}\right].
\]
令 \(t = x - \frac{\pi}{3}\)，则 \(t \in \left[-\frac{\pi}{3}, \frac{2\pi}{3}\right]\)，函数变为 \(y = 2\sin t\)。

正弦函数 \(y = \sin t\) 在区间 \(\left[-\frac{\pi}{3}, \frac{\pi}{2}\right]\) 上单调递增，在区间 \(\left[\frac{\pi}{2}, \frac{2\pi}{3}\right]\) 上单调递减。因此 \(y = 2\sin t\) 的最大值在 \(t = \frac{\pi}{2}\) 处取得，此时
\[
x = \frac{\pi}{2} + \frac{\pi}{3} = \frac{5\pi}{6},
\]
最大值为 \(2\sin\frac{\pi}{2} = 2\).

验证端点值：
当 \(x = 0\) 时，\(f(0) = \sin 0 - \sqrt{3}\cos 0 = -\sqrt{3}\)；
当 \(x = \pi\) 时，\(f(\pi) = \sin\pi - \sqrt{3}\cos\pi = \sqrt{3}\)。
均小于 \(2\)，故最大值确为 \(2\).

## 答案
\(2\)

## 总结
本题主要考查辅助角公式在三角函数最值问题中的应用。通过将 \(a\sin x + b\cos x\) 化为 \(\sqrt{a^2+b^2}\sin(x+\varphi)\)，再利用有界区间内正弦函数的单调性求最值。注意辅助角的准确选取和相位范围的正确计算。
