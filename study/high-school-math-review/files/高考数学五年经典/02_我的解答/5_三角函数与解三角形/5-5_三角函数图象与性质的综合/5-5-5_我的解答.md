# 5-5-5 我的解答

对应题目：[[01_题目/5_三角函数与解三角形/5-5_三角函数图象与性质的综合/5-5-5.md|5-5-5]]

## 完整题目

5. (多选题, 2023 江苏苏锡常镇调研(一)) 已知函数 \( f\left( x\right)  = \sin \left( {{\omega x} + \frac{\pi }{6}}\right)  + \sin \left( {{\omega x} - \frac{\pi }{6}}\right)  + \; \cos {\omega x}\left( {\omega  > 0}\right) \) ,则下列结论正确的有

A. 将函数 \( y = 2\sin {\omega x} \) 的图象向左平移 \( \frac{\pi }{6} \) 个单位长度,总能得到 \( y = f\left( x\right) \) 的图象

B. 若 \( \omega  = 3 \) ,则当 \( x \in  \left\lbrack  {0,\frac{2\pi }{9}}\right\rbrack \) 时, \( f\left( x\right) \) 的取值范围是 \( \left\lbrack  {1,2}\right\rbrack \)

C. 若 \( f\left( x\right) \) 在区间 \( \left( {0,{2\pi }}\right) \) 上恰有 3 个极大值点， 则 \( \frac{13}{6} < \omega  \leq  \frac{19}{6} \)

D. 若 \( f\left( x\right) \) 在区间 \( \left( {\frac{\pi }{3},\frac{5\pi }{12}}\right) \) 上单调递减，则 \( 1 \leq \; \omega  \leq  \frac{16}{5} \)

## 题目勘误
无明显勘误。

## 思路分析
首先利用三角恒等变换将 \(f(x)\) 化简为一个正弦型函数，然后针对每个选项依次分析：
- A 选项：写出平移后的解析式，与 \(f(x)\) 比较，判断是否对任意 \(\omega>0\) 都成立。
- B 选项：代入 \(\omega=3\)，由 \(x\) 的范围求出相位范围，结合正弦函数单调性求值域。
- C 选项：令相位等于 \(\frac{\pi}{2}+2k\pi\) 求出极大值点的横坐标通式，根据恰有 3 个落在 \((0,2\pi)\) 内列出关于 \(k\) 的不等式，解出 \(\omega\) 范围。
- D 选项：单调递减即导数非正，或整体落在正弦的一个单调递减区间内，列出相位区间包含关系，求 \(\omega\) 并检验命题的充分必要性。

## 详细解答
首先化简 \(f(x)\)：
\[
\begin{aligned}
f(x) &= \sin\left(\omega x + \frac{\pi}{6}\right) + \sin\left(\omega x - \frac{\pi}{6}\right) + \cos\omega x \\
&= 2\sin(\omega x)\cos\frac{\pi}{6} + \cos\omega x \\
&= \sqrt{3}\sin(\omega x) + \cos(\omega x) \\
&= 2\sin\left(\omega x + \frac{\pi}{6}\right).
\end{aligned}
\]

**A 选项：**
将 \(y=2\sin(\omega x)\) 向左平移 \(\frac{\pi}{6}\) 个单位长度，所得函数为
\[
y = 2\sin\left[\omega\left(x + \frac{\pi}{6}\right)\right] = 2\sin\left(\omega x + \frac{\omega\pi}{6}\right).
\]
要与 \(f(x)=2\sin\left(\omega x + \frac{\pi}{6}\right)\) 的图象完全重合，必须对任意 \(x\) 有 \(\sin\left(\omega x + \frac{\omega\pi}{6}\right) = \sin\left(\omega x + \frac{\pi}{6}\right)\)。
因为频率相同，只能是相位相差 \(2k\pi\)（\(k\in\mathbb{Z}\)），即 \(\frac{\omega\pi}{6} = \frac{\pi}{6} + 2k\pi\)，解得 \(\omega = 1 + 12k\)。这并非对任意 \(\omega>0\) 都能成立，故 A 错误。

**B 选项：**
当 \(\omega=3\) 时，\(f(x) = 2\sin\left(3x + \frac{\pi}{6}\right)\)。
若 \(x\in\left[0,\frac{2\pi}{9}\right]\)，则 \(3x+\frac{\pi}{6}\in\left[\frac{\pi}{6}, \frac{5\pi}{6}\right]\)。
在 \(\left[\frac{\pi}{6}, \frac{5\pi}{6}\right]\) 上，\(\sin t \in \left[\frac{1}{2}, 1\right]\)，所以 \(f(x)\in[1,2]\)。故 B 正确。

**C 选项：**
\(f(x)\) 取得极大值当且仅当 \(\sin\left(\omega x + \frac{\pi}{6}\right)=1\)，即
\[
\omega x + \frac{\pi}{6} = \frac{\pi}{2} + 2k\pi \quad (k\in\mathbb{Z}),
\]
解得极大值点横坐标为
\[
x_k = \frac{\frac{\pi}{3} + 2k\pi}{\omega} = \frac{(6k+1)\pi}{3\omega}.
\]
要求在开区间 \((0, 2\pi)\) 内恰有 3 个极大值点，即存在且只存在 3 个整数 \(k\) 使得
\[
0 < \frac{(6k+1)\pi}{3\omega} < 2\pi \;\Longleftrightarrow\; 0 < 6k+1 < 6\omega.
\]
由 \(0<6k+1\) 得 \(k>- \frac{1}{6}\)，故最小整数为 \(k=0\)。要使恰有 3 个整数 \(k\)，必须是 \(k=0,1,2\) 满足不等式，而 \(k=3\) 不满足，即
\[
\begin{cases}
6\times 2 + 1 < 6\omega \;\Rightarrow\; \omega > \dfrac{13}{6}, \\[6pt]
6\times 3 + 1 \ge 6\omega \;\Rightarrow\; \omega \le \dfrac{19}{6}.
\end{cases}
\]
因此 \(\frac{13}{6} < \omega \le \frac{19}{6}\)。故 C 正确。

**D 选项：**
\(f(x) = 2\sin\left(\omega x + \frac{\pi}{6}\right)\) 在区间 \(\left(\frac{\pi}{3}, \frac{5\pi}{12}\right)\) 上单调递减，当且仅当该区间整体落在正弦函数的某个单调递减区间内，即存在整数 \(k\) 使
\[
\frac{\pi}{2} + 2k\pi \le \omega x + \frac{\pi}{6} \le \frac{3\pi}{2} + 2k\pi
\]
对任意 \(x\in\left(\frac{\pi}{3}, \frac{5\pi}{12}\right)\) 成立。这等价于
\[
\begin{cases}
\omega\cdot\frac{\pi}{3} + \frac{\pi}{6} \ge \frac{\pi}{2} + 2k\pi, \\[6pt]
\omega\cdot\frac{5\pi}{12} + \frac{\pi}{6} \le \frac{3\pi}{2} + 2k\pi.
\end{cases}
\]
化简得
\[
\begin{cases}
\omega \ge 1 + 6k, \\[6pt]
\omega \le \frac{16}{5} + \frac{24}{5}k.
\end{cases}
\]
当 \(k=0\) 时，得 \(1 \le \omega \le \frac{16}{5}\)；
当 \(k=1\) 时，得 \(7 \le \omega \le 8\)；
（其余整数 \(k\) 无满足 \(\omega>0\) 的解）。
可见，除了 \(\omega\in\left[1,\frac{16}{5}\right]\) 外，\(\omega\in[7,8]\) 等也满足题意。选项 D 的结论只包含其中一部分范围，并非由条件必然推出的全部结果，因此作为“若……则……”的命题是假的。故 D 错误。

## 答案
BC

## 总结
本题以正弦型函数的化简为起点，综合考查了图象变换、给定区间求值域、利用极值点个数求参数范围以及单调性求参数范围等常见题型。  
关键点：  
- 熟练运用和差化积与辅助角公式将函数化为标准形式 \(A\sin(\omega x+\varphi)\)。  
- 对于“恰有 \(k\) 个极值点”类问题，应写出极值点通式，利用区间长度列出整数 \(k\) 的不等式。  
- 判断单调性时，注意正弦函数的周期性和多个递减区间的存在，避免遗漏参数的其它可能范围。  
- 多选题务必逐项严格推理，谨防“以偏概全”的陷阱。
