# 4-7-6 我的解答

对应题目：[[01_题目/4_导数/4-7_利用导数研究函数中的任意与存在问题/4-7-6.md|4-7-6]]

## 完整题目

6.(2021 湖南名校教育联合体入学考试)已知函数 \( f\left( x\right)  = \cos x\left( {x \geq  0}\right) \) .

(1)求证: \( f\left( x\right)  \geq  1 - \frac{1}{2}{x}^{2} \) ；

(2)若 \( f\left( x\right)  + a{\mathrm{e}}^{x} - x \geq  2 \) ，求实数 \( a \) 的取值范围.

## 题目勘误
第二问“若 \( f(x) + a\mathrm{e}^{x} - x \geq 2 \)”缺少不等式成立的范围或量词，疑似遗漏“对任意 \(x \geq 0\) 恒成立”等条件。根据题目所属专题（任意与存在问题）及第一问的铺垫作用，此处通常理解为“若对任意 \(x \geq 0\)， \(f(x)+a\mathrm{e}^x - x \geq 2\) 恒成立”。若原题为存在性问题，则解答思路完全不同，请核对原题图片或原文确认。

## 思路分析
第一问是经典不等式证明，通过构造函数 \(g(x)=\cos x - 1 + \frac{1}{2}x^2\)，利用导数研究单调性，并结合 \(x \geq \sin x\) 这一基本事实完成。
第二问可根据第一问的放缩方向，将 \(\cos x\) 缩小为 \(1-\frac12x^2\)，同时利用 \(\mathrm{e}^x \geq 1+x+\frac12x^2\ (x \geq 0)\)，把原不等式转化为关于 \(a\) 的简单不等式，再通过取特殊点得到必要条件，最终确定参数取值范围。

## 详细解答
**(1) 证明**  
令 \(g(x) = \cos x - 1 + \dfrac{1}{2}x^{2},\ x \geq 0\)，  
则 \(g'(x) = -\sin x + x = x - \sin x\)。  
再令 \(h(x) = x - \sin x,\ x \geq 0\)，  
则 \(h'(x) = 1 - \cos x \geq 0\)，所以 \(h(x)\) 在 \([0,+\infty)\) 上单调递增，  
故当 \(x \geq 0\) 时，\(h(x) \geq h(0) = 0\)，即 \(x - \sin x \geq 0\)。  
从而 \(g'(x) \geq 0\)，\(g(x)\) 在 \([0,+\infty)\) 上单调递增，  
所以 \(g(x) \geq g(0) = \cos 0 - 1 + 0 = 0\)，  
即 \(\cos x \geq 1 - \dfrac{1}{2}x^{2}\)。证毕。

**(2) 解**  
（补充说明：此处按“对任意 \(x \geq 0\) 恒成立”理解并解答）  
由题意，不等式 \(\cos x + a\mathrm{e}^x - x \geq 2\) 对任意 \(x \geq 0\) 恒成立，  
整理得 \(a\mathrm{e}^x \geq 2 + x - \cos x\)。

由(1)知 \(\cos x \geq 1 - \dfrac{1}{2}x^{2}\)，  
故 \(2 + x - \cos x \leq 2 + x - \left(1 - \dfrac{1}{2}x^{2}\right) = 1 + x + \dfrac{1}{2}x^{2}\)。

又由常用不等式 \(\mathrm{e}^x \geq 1 + x + \dfrac{1}{2}x^{2}\ (x \geq 0)\)（构造函数 \(\varphi(x)=\mathrm{e}^x - 1 - x - \dfrac{x^2}{2}\) 求导可证），  
得 \(1 + x + \dfrac{1}{2}x^{2} \leq \mathrm{e}^x\)。

因此 \(2 + x - \cos x \leq \mathrm{e}^x\)。  
要使 \(a\mathrm{e}^x \geq 2 + x - \cos x\) 恒成立，只需  
\(a\mathrm{e}^x \geq \mathrm{e}^x\)，即 \(a \geq 1\)。

**充分性验证**：当 \(a \geq 1\) 时，  
\[
\cos x + a\mathrm{e}^x - x \geq \cos x + \mathrm{e}^x - x \geq \left(1 - \frac12x^2\right) + \left(1 + x + \frac12x^2\right) - x = 2,
\]  
等号在 \(x=0\) 时成立，满足条件。

**必要性验证**：若 \(a < 1\)，取 \(x=0\)，  
则 \(\cos 0 + a\mathrm{e}^0 - 0 = 1 + a < 2\)，与不等式恒成立矛盾。

综上，实数 \(a\) 的取值范围是 \([1, +\infty)\)。

## 答案
\(a \geq 1\)（即 \([1, +\infty)\)）

## 总结
1. 第一问证明 \(\cos x \geq 1 - \frac12x^2\) 的关键是转化为函数单调性，并用导数结合“\(x \geq \sin x\)”完成。
2. 第二问是典型的“铺垫—应用”结构，利用第一问的放缩结论和 \(\mathrm{e}^x\) 的泰勒展开式前几项所对应的不等式 \(\mathrm{e}^x \geq 1+x+\frac12x^2\) 实现了整体代换，避免了对复杂函数直接求导的繁琐。
3. 处理恒成立问题时，先通过特殊点（如 \(x=0\)）得到参数的必要条件，再证明充分性，是简化讨论的有效策略。
