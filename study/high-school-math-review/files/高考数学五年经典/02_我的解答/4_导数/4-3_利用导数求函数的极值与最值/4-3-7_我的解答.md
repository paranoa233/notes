# 4-3-7 我的解答

对应题目：[[01_题目/4_导数/4-3_利用导数求函数的极值与最值/4-3-7.md|4-3-7]]

## 完整题目

7.(2024 安徽黄山 一检)已知函数 \( f\left( x\right)  = \frac{3}{2}{x}^{2} \; - {4ax} + {a}^{2}\ln x \) 在 \( x = 1 \) 处取得极大值.

(1)求实数 \( a \) 的值；

(2)求函数 \( f\left( x\right) \) 在区间 \( \left\lbrack  {\frac{1}{\mathrm{e}},\mathrm{e}}\right\rbrack \) 上的最大值.

## 题目勘误
无明显勘误。

## 思路分析
(1) 由函数在 \( x = 1 \) 处取得极大值，可得 \( f'(1) = 0 \)，解出参数 \( a \) 的值。由于 \( f'(1) = 0 \) 是极值点的必要条件，还需根据极值点的充分条件（一阶导数在 \( x=1 \) 两侧的符号变化或二阶导数符号）检验，舍去不满足极大值的解。

(2) 确定 \( a \) 的值后，写出函数解析式，在给定区间 \( \left[ \frac{1}{\mathrm{e}}, \mathrm{e} \right] \) 上求最大值。需求出区间内的驻点，并与区间端点处的函数值进行比较，或利用单调性直接确定最大值。

## 详细解答
**函数的定义域**  
由 \( \ln x \) 得函数的定义域为 \( (0, +\infty) \)。

**(1) 求实数 \( a \) 的值**  
求导得  
\[
f'(x) = 3x - 4a + \frac{a^2}{x} \quad (x > 0).
\]  
因为 \( x = 1 \) 为极大值点，所以 \( f'(1) = 0 \)，即  
\[
3 \times 1 - 4a + a^2 = 0 \implies a^2 - 4a + 3 = 0 \implies (a-1)(a-3)=0,
\]  
解得 \( a = 1 \) 或 \( a = 3 \)。

**检验哪个解满足“极大值”条件：**

- 若 \( a = 1 \)，则  
  \[
  f'(x) = 3x - 4 + \frac{1}{x} = \frac{3x^2 - 4x + 1}{x} = \frac{(3x-1)(x-1)}{x}.
  \]  
  当 \( 0 < x < 1 \) 时，\( 3x-1 \) 符号不确定，但临界点 \( x=\frac{1}{3} \) 将区间分开。取 \( x = 0.5 \)，\( 3x-1 = 0.5 > 0 \)，\( x-1 < 0 \)，分子为负，分母为正，故 \( f'(x) < 0 \)；  
  当 \( x > 1 \) 时，\( 3x-1 > 0 \)，\( x-1 > 0 \)，故 \( f'(x) > 0 \)。  
  导数由负变正，因此 \( x=1 \) 为极小值点，不符合题意。

- 若 \( a = 3 \)，则  
  \[
  f'(x) = 3x - 12 + \frac{9}{x} = \frac{3x^2 - 12x + 9}{x} = \frac{3(x-1)(x-3)}{x}.
  \]  
  当 \( 0 < x < 1 \) 时，\( x-1 < 0 \)，\( x-3 < 0 \)，乘积为正，故 \( f'(x) > 0 \)；  
  当 \( 1 < x < 3 \) 时，\( x-1 > 0 \)，\( x-3 < 0 \)，乘积为负，故 \( f'(x) < 0 \)。  
  导数由正变负，因此 \( x=1 \) 为极大值点，符合题意。

综上，实数 \( a \) 的值为 \( 3 \)。

**(2) 求函数在区间 \( \left[ \frac{1}{\mathrm{e}}, \mathrm{e} \right] \) 上的最大值**  
由 (1) 知 \( a = 3 \)，函数解析式为  
\[
f(x) = \frac{3}{2}x^2 - 12x + 9\ln x.
\]  
导数为  
\[
f'(x) = \frac{3(x-1)(x-3)}{x}, \quad x > 0.
\]  
在区间 \( \left[ \frac{1}{\mathrm{e}}, \mathrm{e} \right] \) 上，因为 \( \mathrm{e} \approx 2.718 < 3 \)，所以驻点只有 \( x = 1 \)。  
由 (1) 中 \( f'(x) \) 符号分析可知：  
当 \( \frac{1}{\mathrm{e}} \le x < 1 \) 时，\( f'(x) > 0 \)，函数单调递增；  
当 \( 1 < x \le \mathrm{e} \) 时，\( f'(x) < 0 \)，函数单调递减。  
因此函数在 \( x = 1 \) 处取得区间上的最大值。  
最大值  
\[
f(1) = \frac{3}{2} \times 1^2 - 12 \times 1 + 9\ln 1 = \frac{3}{2} - 12 = -\frac{21}{2}.
\]

## 答案
(1) \( a = 3 \)；  
(2) 函数在区间 \( \left[ \frac{1}{\mathrm{e}}, \mathrm{e} \right] \) 上的最大值为 \( -\dfrac{21}{2} \)。

## 总结
本题考查利用导数求函数的极值与最值。需要注意：
1. \( f'(x_0) = 0 \) 只是可导函数在 \( x_0 \) 取得极值的必要条件，求出参数后必须检验极值类型（极大或极小）。
2. 求闭区间上连续函数的最值时，若区间内驻点唯一且单调性明确，可直接利用单调性确定最值，避免复杂的端点值比较。
