# 4-3-2 我的解答

对应题目：[[01_题目/4_导数/4-3_利用导数求函数的极值与最值/4-3-2.md|4-3-2]]

## 完整题目

2.(2024 江苏南通等七市二调)若函数 \( f\left( x\right)  = \; {\mathrm{e}}^{ax} + {2x} \) 有大于零的极值点，则实数 \( a \) 的取值范围是 ( )

A. \( \{ a \mid  a >  - 2\} \) B. \( \left\{  {a\left| {\;a >  - \frac{1}{2}}\right. }\right. \) )

C. \( \{ a \mid  a <  - 2\} \) D. \( \left\{  {a\left| {\;a <  - \frac{1}{2}}\right. }\right. \} \)

## 题目勘误
原题选项中 B、D 存在 OCR 产生的多余括号与格式混乱，如 B 显示为 `\left\{  {a\left| {\;a >  - \frac{1}{2}}\right. }\right. \) )`，实际应为 \( \{a \mid a > -\frac{1}{2}\} \)；D 显示为 `\left\{  {a\left| {\;a <  - \frac{1}{2}}\right. }\right. \}`，实际应为 \( \{a \mid a < -\frac{1}{2}\} \)。其余文字无明显歧义，函数式为 \( f(x) = e^{ax} + 2x \)。

## 思路分析
函数有大于零的极值点，首先要保证导函数有零点，且该零点对应原函数的极值点（可通过二阶导或一阶导符号变化判断），再令该极值点大于零解出 \(a\) 的范围。

- 求导 \( f'(x) \)。
- 令 \( f'(x) = 0 \)，分析方程有解的条件（需对 \(a\) 分类讨论，尤其注意 \(a=0\) 的情形）。
- 解出极值点表达式，利用极值点大于 0 建立不等式，结合参数符号求解。
- 验证该点确为极值点（二阶导符号）。

## 详细解答
函数 \( f(x) = e^{ax} + 2x \)，定义域为 \( \mathbb{R} \)。

对 \(x\) 求导：
\[ f'(x) = a e^{ax} + 2. \]
二阶导：
\[ f''(x) = a^2 e^{ax}. \]

**第一步：极值点的存在条件**
极值点需满足 \( f'(x)=0 \)，即
\[ a e^{ax} + 2 = 0 \quad \Rightarrow \quad a e^{ax} = -2. \]
若 \( a = 0 \)，则 \( f'(x) = 2 \neq 0 \)，函数单调递增，无极值点，不满足题意。

当 \( a \neq 0 \) 时，改写为
\[ e^{ax} = -\frac{2}{a}. \]
因为指数函数 \( e^{ax} > 0 \) 恒成立，故必须有
\[ -\frac{2}{a} > 0 \quad \Rightarrow \quad a < 0. \]
所以 \(a\) 的取值范围首先限制在 \( (-\infty, 0) \)。

**第二步：求出极值点**
由 \( e^{ax} = -\frac{2}{a} \)，取对数：
\[ ax = \ln\!\left(-\frac{2}{a}\right) \quad \Rightarrow \quad x = \frac{1}{a} \ln\!\left(-\frac{2}{a}\right). \]
由于 \(a<0\)，指数方程对任意正右端有意义，该 \(x\) 即为 \(f'(x)\) 的唯一零点。

**第三步：验证极值点类型**
因 \( a \neq 0 \)，\( a^2 > 0 \)，\( e^{ax} > 0 \)，故
\[ f''(x) = a^2 e^{ax} > 0 \]
对一切 \(x\) 成立。因此 \(f(x)\) 是严格凸函数（下凸），唯一的驻点必为极小值点。满足题意要求。

**第四步：极值点大于零的条件**
要求该极值点 \( x > 0 \)：
\[ \frac{1}{a} \ln\!\left(-\frac{2}{a}\right) > 0. \]
已知 \( a < 0 \)，故 \( \frac{1}{a} < 0 \)。两边同乘 \(a\)（负数，不等号方向改变）：
\[ \ln\!\left(-\frac{2}{a}\right) < 0. \]
由对数函数性质，\(\ln u < 0 \iff 0 < u < 1\)，得
\[ 0 < -\frac{2}{a} < 1. \]
因为 \(a<0\)，\(-\frac{2}{a} > 0\) 自动满足。解右半不等式 \(-\frac{2}{a} < 1\)：
两边同乘 \(a\)（负数，不等号方向改变）：
\[ -2 > a \quad \Rightarrow \quad a < -2. \]
结合 \(a<0\)，最终得
\[ a < -2. \]

**第五步：结论**
实数 \(a\) 的取值范围是 \( (-\infty, -2) \)，对应选项 C。

## 答案
C

## 总结
- 对于含参数的指数型函数极值点问题，先由导数为零列出方程，利用 \(e^{ax}>0\) 锁定参数符号。
- 极值点大于零的条件转化为对数不等式时，注意参数 \(a<0\) 带来的不等号变向。
- 可通过二阶导符号快速确认极值点类型，避免繁琐的一阶导符号讨论。
