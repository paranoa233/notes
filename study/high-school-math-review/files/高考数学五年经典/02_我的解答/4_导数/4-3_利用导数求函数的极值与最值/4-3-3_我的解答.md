# 4-3-3 我的解答

对应题目：[[01_题目/4_导数/4-3_利用导数求函数的极值与最值/4-3-3.md|4-3-3]]

## 完整题目

3.(2024 江苏苏锡常镇调研(一))已知 \( a, b \in  (0 \) ， 1) \( \cup  \left( {1, + \infty }\right) ,4{\log }_{a}b + {\log }_{b}a = 4 \) ,则 \( \frac{2}{b} + \ln \frac{a}{b} \) 的最小值为___.

## 题目勘误
原题中“已知 \( a, b \in  (0 \) ， 1) \( \cup  \left( {1, + \infty }\right)\)”存在格式断裂，疑似 OCR 识别错误，建议更正为：已知 \(a, b \in (0,1) \cup (1, +\infty)\)。其余部分无明显勘误。
## 思路分析
条件为关于 \(a\) 与 \(b\) 的对数方程，通过换元（设 \(t=\log_a b\)）可解出 \(t\)，进而得到 \(a\) 与 \(b\) 的幂关系 \(a=b^2\)。将目标式中的 \(a\) 替换，转化为只含变量 \(b\) 的函数，利用导数求出该函数在定义域上的最小值。
## 详细解答
由 \(a, b \in (0,1) \cup (1,+\infty)\) 知 \(\log_a b\) 与 \(\log_b a\) 均存在且非零。
设 \(t = \log_a b\)，则 \(\log_b a = \frac{1}{t}\)。
条件 \(4\log_a b + \log_b a = 4\) 化为：
\[
4t + \frac{1}{t} = 4
\]
整理得 \(4t^2 - 4t + 1 = 0\)，即 \((2t - 1)^2 = 0\)，故 \(t = \frac{1}{2}\)。
所以 \(\log_a b = \frac{1}{2}\)，即 \(a = b^{1/2}\) 或 \(a = b^2\)？（按对数定义：\(\log_a b = 1/2 \Rightarrow b = a^{1/2} \Rightarrow a = b^2\)）。注意，\(b = a^{1/2}\) 且 \(a = b^2\)。此处 \(b>0\) 且 \(b\neq 1\)。
目标表达式为：
\[
\frac{2}{b} + \ln\frac{a}{b} = \frac{2}{b} + \ln\frac{b^2}{b} = \frac{2}{b} + \ln b.
\]
令 \(f(b) = \frac{2}{b} + \ln b\)，其定义域为 \(b \in (0,1) \cup (1,+\infty)\)。
求导：
\[
f'(b) = -\frac{2}{b^2} + \frac{1}{b} = \frac{b - 2}{b^2}.
\]
令 \(f'(b) = 0\)，得 \(b = 2\)。
当 \(0 < b < 2\) 时 \(f'(b) < 0\)，\(b > 2\) 时 \(f'(b) > 0\)。
因此 \(f(b)\) 在 \(b=2\) 处取得极小值，也就是最小值。
计算最小值：
\[
f(2) = \frac{2}{2} + \ln 2 = 1 + \ln 2.
\]
此时 \(a = b^2 = 4\)，满足 \(a \in (0,1)\cup(1,+\infty)\)，符合题意。
## 答案
\(1 + \ln 2\)
## 总结
本题通过换元将隐含的对数关系转化为代数方程，从而得到变量间的确定性关系。后续利用导数求一元函数的最值是常规操作，需注意定义域对最值的影响。注意对数运算的准确性和换元后变量的范围。
