# 4-3-8 我的解答

对应题目：[[01_题目/4_导数/4-3_利用导数求函数的极值与最值/4-3-8.md|4-3-8]]

## 完整题目

8.(2022 山东济宁质检)已知函数 \( f\left( x\right)  = \; a\cos x + b{\mathrm{e}}^{x}\left( {a, b \in  \mathbf{R}}\right) \) ,曲线 \( y = f\left( x\right) \) 在点 \( (0 \) , \( f\left( 0\right) ) \) 处的切线方程为 \( y =  - x \) .

(1)求实数 \( a, b \) 的值；

(2)当 \( x \in  \left\lbrack  {-\frac{\pi }{2}, + \infty }\right) \) 时， \( f\left( x\right)  \leq  c\left( {c \in  \mathbf{Z}}\right) \) 恒成立,求 \( c \) 的最小值.

## 题目勘误
无明显勘误。

## 思路分析
第(1)问：已知函数和切线方程，通过切点在曲线上和切点处的导数值等于切线斜率两个条件建立方程组，解出参数 \(a, b\)。
第(2)问：将 \(a, b\) 的值代入 \(f(x)\)，问题转化为在给定区间上 \(f(x)\) 的最大值不超过整数 \(c\)，求 \(c\) 的最小值。通过求导分析函数单调性，证明最大值点满足一个超越方程（隐零点），估算最大值所在的范围，结合整数的要求确定 \(c\) 的最小值。

## 详细解答
**(1) 求实数 \(a, b\) 的值**

函数 \(f(x)=a\cos x+b e^{x}\)，其导数为 \(f'(x)=-a\sin x+b e^{x}\)。
由切线方程 \(y=-x\) 可知，切点为 \((0, f(0))\)，切线斜率为 \(-1\)。
切点在切线上，故 \(f(0)=0\)，即
\[ a\cos 0 + b e^{0} = a+b=0. \]
切线斜率等于函数在 \(x=0\) 处的导数，故 \(f'(0)=-1\)，即
\[ -a\sin 0 + b e^{0} = b = -1. \]
解得 \(b=-1\)，代入 \(a+b=0\) 得 \(a=1\)。

所以实数 \(a, b\) 的值为 \(a=1,\; b=-1\)。

**(2) 求常数 \(c\) 的最小值**

由(1)知 \(f(x)=\cos x - e^{x}\)，定义域为 \(\left[-\frac{\pi}{2}, +\infty\right)\)。
原题等价于存在整数 \(c\) 使 \(f(x)\le c\) 恒成立，求满足条件的最小整数 \(c\)。因此需要先求 \(f(x)\) 在 \(\left[-\frac{\pi}{2}, +\infty\right)\) 上的最大值。

求导得
\[ f'(x)=-\sin x - e^{x} = -(\sin x + e^{x}). \]

令 \(g(x)=\sin x + e^{x}\)，则 \(g'(x)=\cos x + e^{x}\)。
当 \(x\in\left[-\frac{\pi}{2}, +\infty\right)\) 时，\(e^{x}>0\)，\(\cos x \ge -1\)。
在 \(x=-\frac{\pi}{2}\) 处，\(g'\left(-\frac{\pi}{2}\right)=0+e^{-\frac{\pi}{2}}>0\)；在 \(\left[-\frac{\pi}{2}, \pi\right]\) 等区间上，即使 \(\cos x\) 可能为负，由于 \(e^{x}\) 恒正且逐渐增大，可判断 \(g'(x)>0\) 恒成立（也可分段说明）。事实上，\(e^{x}\) 的最小值 \(e^{-\frac{\pi}{2}}\approx0.2\)，当 \(|\cos x|\le1\) 时总有 \(\cos x+e^{x}>0\)，因此 \(g(x)\) 在区间上单调递增。

又
\[ g\left(-\frac{\pi}{2}\right) = \sin\left(-\frac{\pi}{2}\right) + e^{-\frac{\pi}{2}} = -1 + e^{-\frac{\pi}{2}} < 0, \]
\[ g(0) = \sin 0 + e^{0} = 1 > 0. \]
由零点定理，存在唯一的 \(x_{0}\in \left(-\frac{\pi}{2}, 0\right)\) 使得 \(g(x_{0})=0\)，即 \(\sin x_{0}+e^{x_{0}}=0\)，从而 \(f'(x_{0})=0\)。

当 \(x\in\left[-\frac{\pi}{2}, x_{0}\right)\) 时，\(g(x)<0\)，\(f'(x)>0\)，\(f(x)\) 单调递增；
当 \(x\in\left(x_{0}, +\infty\right)\) 时，\(g(x)>0\)，\(f'(x)<0\)，\(f(x)\) 单调递减。
所以 \(f(x)\) 在 \(x=x_{0}\) 处取得最大值，最大值为
\[ f(x_{0}) = \cos x_{0} - e^{x_{0}}. \]
由 \(e^{x_{0}} = -\sin x_{0}\) 代入得
\[ f(x_{0}) = \cos x_{0} + \sin x_{0} = \sqrt{2}\sin\left(x_{0}+\frac{\pi}{4}\right). \]

下面确定 \(x_{0}\) 的范围。检验 \(g\left(-\frac{\pi}{4}\right)\)：
\[ g\left(-\frac{\pi}{4}\right) = \sin\left(-\frac{\pi}{4}\right) + e^{-\frac{\pi}{4}} = -\frac{\sqrt{2}}{2} + e^{-\frac{\pi}{4}}. \]
因为 \(e^{-\frac{\pi}{4}} < e^{0}=1\)，且 \(\frac{\sqrt{2}}{2}\approx0.707\)，而 \(e^{-\frac{\pi}{4}}\approx0.456\)，故 \(g\left(-\frac{\pi}{4}\right) < 0\)。
又 \(g(0)>0\)，所以 \(x_{0} \in \left(-\frac{\pi}{4}, 0\right)\)。

此时 \(x_{0}+\frac{\pi}{4} \in \left(0, \frac{\pi}{4}\right)\)，因此
\[ 0 < \sin\left(x_{0}+\frac{\pi}{4}\right) < \frac{\sqrt{2}}{2}, \]
\[ 0 < f(x_{0}) = \sqrt{2}\sin\left(x_{0}+\frac{\pi}{4}\right) < 1. \]

所以 \(f(x)\) 在 \(\left[-\frac{\pi}{2}, +\infty\right)\) 上的最大值 \(M\) 满足 \(0 < M < 1\)。
要使得 \(f(x)\le c\) 对任意 \(x\in\left[-\frac{\pi}{2}, +\infty\right)\) 恒成立，只需 \(c \ge M\)。
结合 \(c \in \mathbf{Z}\)，满足条件的最小整数 \(c\) 为 \(1\)。

## 答案
(1) \(a=1,\; b=-1\)；
(2) \(c\) 的最小值为 \(1\)。

## 总结
本题考查了导数的几何意义、利用导数求函数最值以及隐零点问题的处理技巧。第(1)问属于常规的切线问题，抓住“切点既在曲线上又在切线上，且该点导数值等于切线斜率”即可。第(2)问的难点在于导数零点不可直接解出，需借助零点存在定理确定隐零点的范围，并将最大值转化为该点处解析式的形式，通过适度放缩（如利用 \(e^{-\frac{\pi}{4}}<\frac{\sqrt{2}}{2}\)）锁定最大值区间，最后结合整数要求得出答案。
