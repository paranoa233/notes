# 4-10-9 我的解答

对应题目：[[01_题目/4_导数/4-10_导数中的指对同构问题/4-10-9.md|4-10-9]]

## 完整题目

9. 证明: 当 \( x \in  \left( {0,1}\right) \) 时, \( {\mathrm{e}}^{x - 1} \cdot  \ln x < {x}^{2} - x \) .

## 题目勘误
无明显勘误。
## 思路分析
本题需证明含指数、对数与多项式的超越不等式，且 \(x \in (0,1)\)。观察可得左边为 \(e^{x-1} \ln x\)（负值），右边为 \(x(x-1)\)（负值）。可利用两个经典不等式进行放缩：其一，当 \(0<x<1\) 时，\(e^{x-1} > x\)；其二，\(\ln x < x-1\)。因 \(\ln x<0\)，将 \(e^{x-1}>x\) 两边同乘 \(\ln x\) 不等号反向，可得 \(e^{x-1}\ln x < x\ln x\)；再将 \(\ln x < x-1\) 两边同乘正数 \(x\) 得 \(x\ln x < x(x-1) = x^2-x\)。串联即得证。该方法体现了指对同构与经典不等式放缩的结合。
## 详细解答
**欲证：** 当 \(x \in (0,1)\) 时，\(e^{x-1} \cdot \ln x < x^2 - x\).

**证明：**

**第一步：证明 \(e^{x-1} > x\)（\(0<x<1\)）。**
令 \(g(x) = e^{x-1} - x\)，则
\[ g'(x) = e^{x-1} - 1. \]
因为 \(x \in (0,1)\)，所以 \(x-1 < 0\)，从而 \(e^{x-1} < 1\)，得 \(g'(x) < 0\)。
故 \(g(x)\) 在 \((0,1)\) 上严格单调递减，
\[ g(x) > g(1) = e^{0} - 1 = 0, \]
即 \(e^{x-1} > x\) 成立。

**第二步：初步放缩。**
当 \(x \in (0,1)\) 时，\(\ln x < 0\)。对 \(e^{x-1} > x\) 两边同乘负值 \(\ln x\)，不等号方向改变：
\[ e^{x-1} \ln x < x \ln x. \quad (1) \]

**第三步：证明 \(x\ln x < x^2 - x\)（\(0<x<1\)）。**
先证熟知的不等式 \(\ln x < x-1\)（\(0<x<1\)）。
令 \(h(x) = \ln x - x + 1\)，则
\[ h'(x) = \frac{1}{x} - 1. \]
当 \(0<x<1\) 时，\(\frac{1}{x} > 1\)，故 \(h'(x) > 0\)，\(h(x)\) 在 \((0,1)\) 上严格单调递增，
\[ h(x) < h(1) = 0, \]
即 \(\ln x < x - 1\)。
因为 \(x > 0\)，两边乘以 \(x\) 得
\[ x\ln x < x(x-1) = x^2 - x. \quad (2) \]

**第四步：串联不等式。**
综合 (1) 与 (2) 可得
\[ e^{x-1} \ln x < x\ln x < x^2 - x. \]
因此，当 \(x \in (0,1)\) 时，原不等式 \(e^{x-1} \ln x < x^2 - x\) 恒成立。证毕.
## 答案
证明见详细解答.
## 总结
本题通过联想常用不等式（\(e^{x-1} \ge x\) 及 \(\ln x \le x-1\)），并结合区间内对数的正负进行同向放缩串联，避免了复杂的参数讨论。关键在于注意到乘以负数时的变号规则，以及利用单调性严格确定不等号方向。
