# 4-8-1 我的解答

对应题目：[[01_题目/4_导数/4-8_利用导数研究不等式的问题/4-8-1.md|4-8-1]]

## 完整题目

1.(2023 江苏南京、盐城期末)若函数 \( f\left( x\right)  = {x}^{3} \; + b{x}^{2} + {cx} + d \) 满足 \( f\left( {1 - x}\right)  + f\left( {1 + x}\right)  = 0 \) 对一切实数 \( x \) 恒成立，则不等式 \( {f}^{\prime }\left( {{2x} + 3}\right)  < {f}^{\prime }(x \) -1)的解集为 ( )

A. \( \left( {0, + \infty }\right) \)

B. \( \left( {-\infty , - 4}\right) \)

C. \( \left( {-4,0}\right) \)

D. \( \left( {-\infty , - 4}\right)  \cup  \left( {0, + \infty }\right) \)

## 题目勘误
原题中不等式为“\( {f}^{\prime }\left( {{2x} + 3}\right)  < {f}^{\prime }(x \) -1)”，疑似公式识别或排版错误，括号位置不当。  
**建议更正**：\( f'(2x+3) < f'(x-1) \)。  
**理由**：根据整体结构和常规命题逻辑，应是比较 \( f' \) 在 \( 2x+3 \) 与 \( x-1 \) 处的函数值。

## 思路分析
1. 由恒等式 \( f(1-x)+f(1+x)=0 \) 对所有实数 \( x \) 成立，结合三次多项式的形式，可确定系数 \( b \) 以及 \( c, d \) 的关系。  
2. 求出 \( f'(x) \)，并发现它是关于 \( x=1 \) 对称的开口向上的二次函数。  
3. 将不等式 \( f'(2x+3) < f'(x-1) \) 转化为这两个自变量到对称轴 \( x=1 \) 的距离大小的比较，进而解出 \( x \) 的范围。

## 详细解答
已知 \( f(x) = x^3 + bx^2 + cx + d \)，且  
\[
f(1-x) + f(1+x) = 0 \quad \text{对一切实数 } x \text{ 恒成立}.
\]
代入并展开：
\[
\begin{aligned}
&(1-x)^3 + b(1-x)^2 + c(1-x) + d + (1+x)^3 + b(1+x)^2 + c(1+x) + d \\
&= (1-3x+3x^2-x^3) + b(1-2x+x^2) + c - cx + d \\
&\quad + (1+3x+3x^2+x^3) + b(1+2x+x^2) + c + cx + d \\
&= (2+6x^2) + b(2+2x^2) + 2c + 2d \\
&= (6+2b)x^2 + (2+2b+2c+2d) = 0.
\end{aligned}
\]
此式对所有 \( x \) 恒成立，故有
\[
\begin{cases}
6 + 2b = 0, \\
2 + 2b + 2c + 2d = 0.
\end{cases}
\]
解得 \( b = -3 \)，代入第二式得 \( 2 - 6 + 2c + 2d = 0 \)，即 \( c+d=2 \)。  
因此
\[
f(x) = x^3 - 3x^2 + cx + d,
\]
求导得
\[
f'(x) = 3x^2 - 6x + c = 3(x^2-2x) + c = 3[(x-1)^2-1] + c = 3(x-1)^2 + (c-3).
\]
可见 \( f'(x) \) 是开口向上的二次函数，其对称轴为 \( x=1 \)。

不等式 \( f'(2x+3) < f'(x-1) \) 中，令 \( u = 2x+3 \)，\( v = x-1 \)。  
因为 \( f'(x) \) 开口向上，函数值的大小由自变量到对称轴 \( x=1 \) 的距离决定：距离越近，函数值越小。  
故 \( f'(u) < f'(v) \) 等价于
\[
|u-1| < |v-1|.
\]
即
\[
|(2x+3)-1| < |(x-1)-1| \Rightarrow |2x+2| < |x-2|.
\]
两边平方（均为非负数）：
\[
(2x+2)^2 < (x-2)^2 \Rightarrow 4x^2+8x+4 < x^2-4x+4.
\]
化简得
\[
3x^2 + 12x < 0 \Rightarrow 3x(x+4) < 0 \Rightarrow -4 < x < 0.
\]
因此不等式的解集为 \( (-4, 0) \)。

## 答案
C

## 总结
本题的关键在于：利用恒成立条件 \( f(1-x)+f(1+x)=0 \) 推出函数图像关于点 \( (1,0) \) 对称，进而确定部分系数，并发现导函数 \( f'(x) \) 关于直线 \( x=1 \) 对称。将抽象的不等式转化为自变量到对称轴的距离比较，是简化二次型导数不等式求解的常用技巧。
