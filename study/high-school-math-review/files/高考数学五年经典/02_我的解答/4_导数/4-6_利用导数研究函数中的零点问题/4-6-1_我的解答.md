# 4-6-1 我的解答

对应题目：[[01_题目/4_导数/4-6_利用导数研究函数中的零点问题/4-6-1.md|4-6-1]]

## 完整题目

1. (多选题，2020 山东济南三中质检)已知函数 \( f\left( x\right) \) 及其导数 \( {f}^{\prime }\left( x\right) \) . 若存在 \( {x}_{0} \) ,使得 \( f\left( {x}_{0}\right)  = \; {f}^{\prime }\left( {x}_{0}\right) \) ,则称 \( {x}_{0} \) 是 \( f\left( x\right) \) 的一个“巧值点”. 下面给出 4 个函数，有“巧值点”的函数为 ( )

A. \( f\left( x\right)  = {x}^{2} \) B. \( f\left( x\right)  = \frac{1}{{\mathrm{e}}^{x}} \)

C. \( f\left( x\right)  = \ln x \) D. \( f\left( x\right)  = \frac{1}{x} \)

## 题目勘误
无明显勘误。

## 思路分析
本题的核心是理解“巧值点”的定义：若存在 \(x_0\) 使得 \(f(x_0) = f'(x_0)\)，则称 \(x_0\) 为函数 \(f(x)\) 的一个巧值点。  
问题转化为对每个选项中的函数，判断方程 \(f(x) = f'(x)\) 在各自的定义域内是否有实数解。  
分别求出四个函数的导数，列出方程，结合定义域和函数性质判断解的存在性即可。

## 详细解答
**A. \(f(x) = x^2\)**  
定义域：\(\mathbb{R}\)。  
导数：\(f'(x) = 2x\)。  
方程：\(x^2 = 2x\)。  
化简得 \(x^2 - 2x = 0\)，即 \(x(x - 2) = 0\)。  
解得 \(x = 0\) 或 \(x = 2\)，均在定义域内。  
故 A 有“巧值点”。

**B. \(f(x) = \dfrac{1}{\mathrm{e}^x} = \mathrm{e}^{-x}\)**  
定义域：\(\mathbb{R}\)。  
导数：\(f'(x) = -\mathrm{e}^{-x}\)。  
方程：\(\mathrm{e}^{-x} = -\mathrm{e}^{-x}\)。  
因为 \(\mathrm{e}^{-x} > 0\) 恒成立，左边正、右边负，方程不可能成立。  
故 B 无“巧值点”。

**C. \(f(x) = \ln x\)**  
定义域：\((0, +\infty)\)。  
导数：\(f'(x) = \dfrac{1}{x}\)。  
方程：\(\ln x = \dfrac{1}{x}\)，即 \(\ln x - \dfrac{1}{x} = 0\)。  
令 \(g(x) = \ln x - \dfrac{1}{x}\)，则 \(g(x)\) 在 \((0, +\infty)\) 上连续。  
计算 \(g(1) = \ln 1 - 1 = -1 < 0\)，  
\(g(\mathrm{e}) = 1 - \dfrac{1}{\mathrm{e}} > 0\)。  
由零点存在定理，存在 \(x_0 \in (1, \mathrm{e})\) 使得 \(g(x_0) = 0\)，即方程有解。  
故 C 有“巧值点”。

**D. \(f(x) = \dfrac{1}{x}\)**  
定义域：\((-\infty, 0) \cup (0, +\infty)\)。  
导数：\(f'(x) = -\dfrac{1}{x^2}\)。  
方程：\(\dfrac{1}{x} = -\dfrac{1}{x^2}\)。  
因为 \(x \neq 0\)，两边同乘 \(x^2\) 得 \(x = -1\)。  
\(x = -1\) 在定义域内，且满足方程。  
故 D 有“巧值点”。

综合以上，有“巧值点”的函数为 A、C、D。

## 答案
A、C、D

## 总结
判断“巧值点”的本质是求解方程 \(f(x) = f'(x)\)，需注意：  
1. 方程的解必须在原函数的定义域内；  
2. 对于超越方程（如 C 中的 \(\ln x = 1/x\)），可构造辅助函数利用零点存在定理判断；  
3. 若左右两边的值域无交集（如 B），可直接断定无解。本题考查了基本初等函数的求导及方程思想。
