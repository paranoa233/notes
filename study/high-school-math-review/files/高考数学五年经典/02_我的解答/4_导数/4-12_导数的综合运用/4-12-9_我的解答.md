# 4-12-9 我的解答

对应题目：[[01_题目/4_导数/4-12_导数的综合运用/4-12-9.md|4-12-9]]

## 完整题目

9.(2021 江苏南京学情调研)已知函数 \( f\left( x\right)  = \; {kx} - x\ln x, k \in  \mathbf{R} \) .

(1)当 \( k = 2 \) 时，求函数 \( f\left( x\right) \) 的单调区间；

(2)当 \( 0 < x \leq  1 \) 时， \( f\left( x\right)  \leq  k \) 恒成立，求 \( k \) 的取值范围;

(3)设 \( n \in  {\mathbf{N}}^{ * } \) ，求证: \( \frac{\ln 1}{2} + \frac{\ln 2}{3} + \cdots  + \frac{\ln n}{n + 1} \leq \; \frac{n\left( {n - 1}\right) }{4} \)

## 题目勘误
无明显勘误。

## 思路分析
本题是导数综合运用题，包含单调区间、恒成立求参和数列不等式证明三个典型设问。
- **(1)**：直接求导，解导函数不等式即可得单调区间。
- **(2)**：恒成立问题，优先考虑分离参数，将不等式转化为 \(k \ge g(x)\)，通过求 \(g(x)\) 在 \((0,1]\) 上的最大值（或上确界）得到参数范围。
- **(3)**：观察所证不等式的结构，将右边的 \(\frac{n(n-1)}{4}\) 视为 \(\sum_{i=1}^n \frac{i-1}{2}\)，只需证局部不等式 \(\frac{\ln i}{i+1} \le \frac{i-1}{2}\)，整理后等价于 \(\ln i \le \frac{i^2-1}{2}\)。这可以通过构造函数并借助导数证明其单调性来完成。

## 详细解答
### (1) 当 \(k=2\) 时，求单调区间
函数定义域为 \((0, +\infty)\)。当 \(k=2\) 时，
\[ f(x) = 2x - x\ln x \]
求导得：
\[ f'(x) = 2 - (\ln x + 1) = 1 - \ln x \]
令 \(f'(x) = 0\)，得 \(\ln x = 1\)，即 \(x = e\)。
- 当 \(0 < x < e\) 时，\(\ln x < 1\)，\(f'(x) > 0\)，\(f(x)\) 单调递增；
- 当 \(x > e\) 时，\(\ln x > 1\)，\(f'(x) < 0\)，\(f(x)\) 单调递减。

故 \(f(x)\) 的单调递增区间为 \((0, e)\)，单调递减区间为 \((e, +\infty)\)。

### (2) 当 \(0< x \le 1\) 时，\(f(x) \le k\) 恒成立，求 \(k\) 的取值范围
由 \(f(x) \le k\) 得 \(kx - x\ln x \le k\)，整理为：
\[ k(x-1) \le x\ln x \]
- 当 \(x = 1\) 时，左边 \(=0\)，右边 \(=0\)，不等式恒成立。
- 当 \(0 < x < 1\) 时，\(x-1 < 0\)，分离参数得：
\[ k \ge \frac{x\ln x}{x-1} \]
令 \(g(x) = \dfrac{x\ln x}{x-1}\)，\(x \in (0,1)\)。只需 \(k \ge g(x)_{\max}\)。
对 \(g(x)\) 求导：
\[ g'(x) = \frac{(\ln x + 1)(x-1) - x\ln x}{(x-1)^2} = \frac{x - 1 - \ln x}{(x-1)^2} \]
再令 \(\varphi(x) = x - 1 - \ln x\)，则 \(\varphi'(x) = 1 - \frac{1}{x}\)。
当 \(x \in (0,1)\) 时，\(\varphi'(x) < 0\)，\(\varphi(x)\) 在 \((0,1)\) 上单调递减。
所以 \(\varphi(x) > \varphi(1) = 0\)。又 \((x-1)^2 > 0\)，故 \(g'(x) > 0\)。
因此 \(g(x)\) 在 \((0,1)\) 上单调递增。
计算极限：
\[ \lim_{x \to 0^+} g(x) = \lim_{x \to 0^+} \frac{x\ln x}{x-1} = 0 \]
\[ \lim_{x \to 1^-} g(x) = \lim_{x \to 1^-} \frac{x\ln x}{x-1} \xlongequal{\text{洛必达}} \lim_{x \to 1^-} \frac{\ln x + 1}{1} = 1 \]
所以 \(g(x) \in (0,1)\)。
要使 \(k \ge g(x)\) 恒成立，必有 \(k \ge 1\)。
综上，\(k\) 的取值范围为 \([1, +\infty)\)。

### (3) 设 \(n \in \mathbf{N}^*\)，求证：\(\frac{\ln 1}{2} + \frac{\ln 2}{3} + \cdots + \frac{\ln n}{n+1} \le \frac{n(n-1)}{4}\)
先证一个辅助不等式：当 \(x \ge 1\) 时，
\[ \ln x \le \frac{x^2 - 1}{2} \]
**证明**：令 \(F(x) = \ln x - \dfrac{x^2 - 1}{2}\)，\(x \ge 1\)。
求导得 \(F'(x) = \dfrac{1}{x} - x = \dfrac{1 - x^2}{x} \le 0\)。
所以 \(F(x)\) 在 \([1, +\infty)\) 上单调递减，故 \(F(x) \le F(1) = 0\)。
即 \(\ln x \le \dfrac{x^2 - 1}{2}\)。

取 \(x = i \;(i \in \mathbf{N}^*)\)，得：
\[ \ln i \le \frac{i^2 - 1}{2} = \frac{(i-1)(i+1)}{2} \]
两边同除以正数 \(i+1\)，得：
\[ \frac{\ln i}{i+1} \le \frac{i-1}{2} \]
对 \(i\) 从 \(1\) 到 \(n\) 求和：
\[ \sum_{i=1}^n \frac{\ln i}{i+1} \le \sum_{i=1}^n \frac{i-1}{2} = \frac{1}{2} \cdot \frac{n(n-1)}{2} = \frac{n(n-1)}{4} \]
其中 \(i=1\) 时，\(\frac{\ln 1}{2} = 0 = \frac{1-1}{2}\)，等号成立。故原不等式得证。

## 答案
(1) 单调递增区间为 \((0, e)\)，单调递减区间为 \((e, +\infty)\)；
(2) \(k\) 的取值范围为 \([1, +\infty)\)；
(3) 证明见详细解答。

## 总结
1. 求单调区间时注意定义域，解导函数不等式即可。
2. 含参恒成立问题优先尝试分离参数，转化为求函数最值（或确界），注意区间端点处的极限处理。
3. 数列求和型不等式常需先借助导数证明一个局部函数不等式，再通过赋值、累加完成证明。本题通过构造 \(\ln x \le \dfrac{x^2-1}{2}\) 实现了对通项的放缩。
