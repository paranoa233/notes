# 4-4-1 我的解答

对应题目：[[01_题目/4_导数/4-4_导数中的参数问题/4-4-1.md|4-4-1]]

## 完整题目

1. (2024 广西 4 月模拟)已知 \( x = 0 \) 是函数 \( f\left( x\right) \; = {x}^{2}\left( {x - a}\right) \) 的极小值点，则实数 \( a \) 的取值范围是 ( )

A. \( \left( {-\infty ,0}\right) \) B. \( \left( {-\infty ,\frac{3}{2}}\right) \)

C. \( \left( {0, + \infty }\right) \) D. \( \left( {\frac{3}{2}, + \infty }\right) \)

## 题目勘误
无明显勘误。
## 思路分析
本题给出含参三次函数，并指定 \( x=0 \) 为极小值点，要求参数 \( a \) 的取值范围。  
可先求导得到一阶、二阶导函数。优先使用二阶导数判定极值的充分条件（\( f''(0) > 0 \)），若失效（即 \( f''(0) = 0 \)），则回归一阶导数符号变化进行判断。由此解出 \( a \) 的范围并对照选项。
## 详细解答
函数 \( f(x) = x^2(x - a) = x^3 - a x^2 \)，定义域为 \( \mathbf{R} \)。

求导得：
\[
f'(x) = 3x^2 - 2a x = x(3x - 2a),
\]
\[
f''(x) = 6x - 2a.
\]

由题意，\( x = 0 \) 是极小值点，则必有 \( f'(0) = 0 \)，代入成立。  
计算二阶导数在 \( x = 0 \) 处的值：
\[
f''(0) = -2a.
\]

**情形一：** 若 \( f''(0) > 0 \)，即 \( -2a > 0 \Rightarrow a < 0 \)，由极值的充分条件知 \( x = 0 \) 是极小值点。

**情形二：** 若 \( f''(0) = 0 \)，即 \( a = 0 \)。此时 \( f(x) = x^3 \)，\( f'(x) = 3x^2 \ge 0 \) 且不恒为零，函数在 \( \mathbf{R} \) 上单调递增，\( x = 0 \) 不是极值点（驻点但不变号），舍去。

**情形三：** 若 \( f''(0) < 0 \)，即 \( a > 0 \)，由充分条件知 \( x = 0 \) 是极大值点，不符合题意。

综上，实数 \( a \) 的取值范围是 \( (-\infty, 0) \)。
## 答案
A
## 总结
处理含参函数的极值点问题时，优先考虑二阶导数判定，当二阶导数为零时，应回到一阶导数的符号变化（或更高阶导数）进行判定，避免遗漏参数边界。本题需排除 \( a = 0 \) 的退化情况。
