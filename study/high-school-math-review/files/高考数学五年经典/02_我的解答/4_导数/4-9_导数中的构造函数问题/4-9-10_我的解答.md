# 4-9-10 我的解答

对应题目：[[01_题目/4_导数/4-9_导数中的构造函数问题/4-9-10.md|4-9-10]]

## 完整题目

1.(2021 全国乙卷理，12)设 \( a = {2\ln }{1.01}, b = \; \ln {1.02}, c = \sqrt{1.04} - 1 \) ,则 ( )

A. \( a < b < c \) B. \( b < c < a \)

C. \( b < a < c \) D. \( c < a < b \)

## 题目勘误
无明显勘误。

## 思路分析
本题为比较三个数 \(a=2\ln 1.01\)，\(b=\ln 1.02\)，\(c=\sqrt{1.04}-1\) 的大小。由于它们都与 \(0.01\) 的相关倍数有关，可令 \(x=0.01\)，则 \(a=2\ln(1+x)\)，\(b=\ln(1+2x)\)，\(c=\sqrt{1+4x}-1\)。先利用对数性质易得 \(a>b\)。接下来分别构造差函数 \(f(x)=2\ln(1+x)-\sqrt{1+4x}+1\)（比较 \(a\) 与 \(c\)）和 \(g(x)=\ln(1+2x)-\sqrt{1+4x}+1\)（比较 \(b\) 与 \(c\)），通过求导判断单调性并结合端点 \(x=0\) 处取值即可得到三者的大小关系。

## 详细解答
设 \(x=0.01\)，则
\[
a = 2\ln(1+x),\quad b = \ln(1+2x),\quad c = \sqrt{1+4x}-1.
\]

**第一步：比较 \(a\) 与 \(b\)**
由对数性质，
\[
a = 2\ln(1+x) = \ln(1+x)^2 = \ln(1+2x+x^2).
\]
因为 \(x=0.01>0\)，所以 \(1+2x+x^2 > 1+2x\)。又 \(y=\ln t\) 为增函数，得
\[
\ln(1+2x+x^2) > \ln(1+2x) = b,
\]
故 \(a > b\)。

**第二步：比较 \(a\) 与 \(c\)**
构造函数
\[
f(x) = a - c = 2\ln(1+x) - \sqrt{1+4x} + 1,\quad x \ge 0.
\]
求导：
\[
f'(x) = \frac{2}{1+x} - \frac{2}{\sqrt{1+4x}} = 2\left(\frac{1}{1+x} - \frac{1}{\sqrt{1+4x}}\right).
\]
当 \(x>0\) 时，比较 \(1+x\) 与 \(\sqrt{1+4x}\)：
\[
(1+x)^2 = 1+2x+x^2,\quad (\sqrt{1+4x})^2 = 1+4x.
\]
由 \(1+2x+x^2 - (1+4x) = x^2 - 2x = x(x-2)\)，当 \(0 < x < 2\) 时，\(x(x-2) < 0\)，即 \(1+x < \sqrt{1+4x}\)，从而
\[
\frac{1}{1+x} > \frac{1}{\sqrt{1+4x}} \implies f'(x) > 0.
\]
故 \(f(x)\) 在 \([0,2)\) 上单调递增。又 \(f(0)=2\ln 1 - 1 + 1 = 0\)，所以当 \(x=0.01\) 时，\(f(0.01) > 0\)，即
\[
a - c > 0 \implies a > c.
\]

**第三步：比较 \(b\) 与 \(c\)**
构造函数
\[
g(x) = b - c = \ln(1+2x) - \sqrt{1+4x} + 1,\quad x \ge 0.
\]
求导：
\[
g'(x) = \frac{2}{1+2x} - \frac{2}{\sqrt{1+4x}} = 2\left(\frac{1}{1+2x} - \frac{1}{\sqrt{1+4x}}\right).
\]
当 \(x>0\) 时，比较 \(1+2x\) 与 \(\sqrt{1+4x}\)：
\[
(1+2x)^2 = 1+4x+4x^2 > 1+4x,
\]
即 \(1+2x > \sqrt{1+4x}\)，从而
\[
\frac{1}{1+2x} < \frac{1}{\sqrt{1+4x}} \implies g'(x) < 0.
\]
故 \(g(x)\) 在 \([0,+\infty)\) 上单调递减。又 \(g(0)=\ln 1 - 1 + 1 = 0\)，所以当 \(x=0.01\) 时，\(g(0.01) < 0\)，即
\[
b - c < 0 \implies b < c.
\]

**结论：**
由以上比较得 \(b < c < a\)。

## 答案
B

## 总结
处理此类数值接近的“比较大小”问题，常见的思路是：
1. **作差构造函数**：将某个关键倍数设为变量 \(x\)，将待比较的式子表示成函数，借助导数确定单调性，结合端点值判断大小。
2. **灵活运用初等性质**：如本题中利用对数性质 \(\ln A - \ln B = \ln\frac{A}{B}\) 或 \(n\ln x = \ln x^n\) 先简化，如比较 \(a\) 与 \(b\) 时一步到位。
3. **注意精确度与不等式方向**：构造函数后要确认定义域和导数的符号，并检验端点值，避免因忽略“等号成立条件”而误判。
