# 4-9-1 我的解答

对应题目：[[01_题目/4_导数/4-9_导数中的构造函数问题/4-9-1.md|4-9-1]]

## 完整题目

1. 已知实数 \( a = \frac{3}{2}{\mathrm{e}}^{\frac{1}{2}}, b = \frac{4}{3}{\mathrm{e}}^{\frac{2}{3}}, c = \frac{8}{7}{\mathrm{e}}^{\frac{6}{7}} \) (e 为自然对数的底数),则 \( a, b, c \) 的大小关系为 ( )

A. \( a < b < c \) B. \( b < c < a \)

C. \( c < b < a \) D. \( b < a < c \)

## 题目勘误
无明显勘误。

## 思路分析
本题给出的三个数 \(a, b, c\) 均具有“系数为 \(2-x\)，指数为 \(x\)”的统一结构：
\[
a = \frac{3}{2}\mathrm{e}^{\frac{1}{2}},\quad b = \frac{4}{3}\mathrm{e}^{\frac{2}{3}},\quad c = \frac{8}{7}\mathrm{e}^{\frac{6}{7}} .
\]
可以将其看作同一个函数 \(g(x) = (2-x)\mathrm{e}^{x}\) 分别在 \(x=\frac{1}{2}, \frac{2}{3}, \frac{6}{7}\) 处的函数值。通过求导分析 \(g(x)\) 在区间 \(\left[\frac{1}{2}, \frac{6}{7}\right]\) 上的单调性，即可比较 \(a, b, c\) 的大小。

## 详细解答
构造函数
\[
g(x) = (2-x)\mathrm{e}^{x}, \quad x\in\left[\frac{1}{2}, \frac{6}{7}\right].
\]
则
\[
a = g\!\left(\frac{1}{2}\right),\quad b = g\!\left(\frac{2}{3}\right),\quad c = g\!\left(\frac{6}{7}\right).
\]
对 \(g(x)\) 求导：
\[
g'(x) = -\mathrm{e}^{x} + (2-x)\mathrm{e}^{x} = (1-x)\mathrm{e}^{x}.
\]
当 \(x \in \left[\frac{1}{2}, \frac{6}{7}\right]\) 时，\(x \le \frac{6}{7} < 1\)，所以 \(1-x > 0\)，又 \(\mathrm{e}^{x} > 0\)，因此
\[
g'(x) > 0,
\]
即 \(g(x)\) 在 \(\left[\frac{1}{2}, \frac{6}{7}\right]\) 上单调递增。
又因为
\[
\frac{1}{2} < \frac{2}{3} < \frac{6}{7},
\]
由单调递增性得
\[
g\!\left(\frac{1}{2}\right) < g\!\left(\frac{2}{3}\right) < g\!\left(\frac{6}{7}\right),
\]
即
\[
a < b < c.
\]

## 答案
A

## 总结
对于这类“系数与指数有内在关联”的实数比较问题，关键在于观察出统一的结构，将其转化为同一函数在不同自变量处的取值。通过求导判断函数在目标区间上的单调性，便可简洁地得到大小关系。本题中构造的函数 \(g(x)=(2-x)\mathrm{e}^{x}\) 在 \(x<1\) 时单调递增，因此自变量越大对应的值越大。
