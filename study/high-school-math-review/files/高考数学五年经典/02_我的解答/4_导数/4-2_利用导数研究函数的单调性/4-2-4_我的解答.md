# 4-2-4 我的解答

对应题目：[[01_题目/4_导数/4-2_利用导数研究函数的单调性/4-2-4.md|4-2-4]]

## 完整题目

4.(2023 江苏苏锡常镇调研(一))已知 \( f\left( x\right) \) 是定义在 \( \mathbf{R} \) 上的偶函数,当 \( x \geq  0 \) 时, \( f\left( x\right)  = {\mathrm{e}}^{x} + \; \sin x \) ，则不等式 \( f\left( {{2x} - 1}\right)  < {\mathrm{e}}^{\pi } \) 的解集是( )

A. \( \left( {\frac{1 + \pi }{2}, + \infty }\right) \) B. \( \left( {0,\frac{1 + \pi }{2}}\right) \)

C. \( \left( {0,\frac{1 + {\mathrm{e}}^{\pi }}{2}}\right) \) D. \( \left( {\frac{1 - \pi }{2},\frac{1 + \pi }{2}}\right) \)

## 题目勘误
无明显勘误。

## 思路分析
首先根据函数在 \(x \ge 0\) 时的解析式，利用导数判断其单调性；再由偶函数性质得到函数在整个定义域上的对称单调性。将常数 \(e^\pi\) 与函数值 \(f(\pi)\) 对应，进而将函数不等式 \(f(2x-1) < e^\pi\) 转化为关于 \(2x-1\) 的绝对值不等式，最后解出 \(x\) 的范围即可。

## 详细解答
1. **分析 \(f(x)\) 在 \([0,+\infty)\) 上的单调性**  
   当 \(x \ge 0\) 时，\(f(x) = e^x + \sin x\)，求导得  
   \[ f'(x) = e^x + \cos x \]  
   由于 \(x \ge 0\)，有 \(e^x \ge 1\)，且 \(\cos x \ge -1\)，故 \(f'(x) \ge 0\)，且等号不恒成立（例如 \(e^x=1\) 仅当 \(x=0\) 时成立，此时 \(\cos 0=1\)，\(f'(0)=2>0\)），所以 \(f(x)\) 在 \([0,+\infty)\) 上严格单调递增。

2. **确定关键函数值**  
   计算得 \(f(\pi) = e^\pi + \sin\pi = e^\pi\)。

3. **利用偶函数性质转化不等式**  
   因 \(f(x)\) 为偶函数且 \([0,+\infty)\) 上单调递增，可知 \(f(x)\) 在 \((-\infty,0]\) 上单调递减。不等式 \(f(y) < e^\pi = f(\pi)\) 的求解：
   - 当 \(y \ge 0\) 时，由单调递增得 \(0 \le y < \pi\)；
   - 当 \(y < 0\) 时，利用偶函数性质 \(f(y)=f(-y)\)，此时 \(-y > 0\)，\(f(-y) < f(\pi)\) 等价于 \(-y < \pi\)，即 \(y > -\pi\)。  
   综上，对于任意实数 \(y\)，\(f(y) < e^\pi\) 等价于 \(-\pi < y < \pi\)。

4. **代入并解出 \(x\)**  
   令 \(y = 2x - 1\)，不等式 \(f(2x-1) < e^\pi\) 化为  
   \[ -\pi < 2x - 1 < \pi \]  
   移项得 \(1 - \pi < 2x < 1 + \pi\)，两边同除以 2 得  
   \[ \frac{1-\pi}{2} < x < \frac{1+\pi}{2} \]  
   故解集为 \(\left( \dfrac{1-\pi}{2}, \dfrac{1+\pi}{2} \right)\)。

## 答案
D

## 总结
本题主要考查利用导数研究函数单调性以及奇偶性的综合应用。关键步骤是判断出 \(f(x)\) 在 \([0,+\infty)\) 上递增，并由偶函数得到对称区间单调性，最终将函数不等式转化为常规的一次绝对值不等式求解。注意准确找出与常数对应的自变量特殊值。
