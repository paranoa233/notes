# 4-2-13 我的解答

对应题目：[[01_题目/4_导数/4-2_利用导数研究函数的单调性/4-2-13.md|4-2-13]]

## 完整题目

4.(2020 全国卷 II 文,21)已知函数 \( f\left( x\right)  = \; 2\ln x + 1 \) .

(1)若 \( f\left( x\right)  \leq  {2x} + c \) ，求实数 \( c \) 的取值范围；

(2)设 \( a > 0 \) ，讨论函数 \( g\left( x\right)  = \frac{f\left( x\right)  - f\left( a\right) }{x - a} \) 的单调性.

## 题目勘误
无明显勘误。
## 思路分析
(1) 将不等式 \(f(x) \le 2x + c\) 分离参数，转化为 \(c \ge f(x) - 2x\)，构造函数 \(h(x) = 2\ln x + 1 - 2x\)，利用导数求其最大值，即得 \(c\) 的范围。
(2) 先将 \(g(x)\) 的表达式化简，注意到其为分式且与 \(a\) 有关，可通过换元 \(t = \dfrac{x}{a}\) 将问题转化为关于 \(t\) 的函数单调性讨论，通过求导判断导数的符号。
## 详细解答
**(1)** 由题设 \(f(x) = 2\ln x + 1\)，定义域为 \((0, +\infty)\)。  
不等式 \(f(x) \le 2x + c\) 可化为  
\[
c \ge 2\ln x + 1 - 2x.
\]  
令 \(h(x) = 2\ln x + 1 - 2x\)，\(x > 0\)，求导得  
\[
h'(x) = \frac{2}{x} - 2 = \frac{2(1 - x)}{x}.
\]  
令 \(h'(x) = 0\) 得 \(x = 1\)。  
当 \(0 < x < 1\) 时，\(h'(x) > 0\)，\(h(x)\) 单调递增；  
当 \(x > 1\) 时，\(h'(x) < 0\)，\(h(x)\) 单调递减。  
因此 \(h(x)\) 在 \(x = 1\) 处取得最大值  
\[
h(1) = 2\ln 1 + 1 - 2 = -1.
\]  
要使 \(c \ge h(x)\) 恒成立，只需 \(c \ge h_{\max} = -1\)。  
故实数 \(c\) 的取值范围是 \([-1, +\infty)\)。

**(2)** 由 \(a > 0\)，\(f(x) = 2\ln x + 1\)，则  
\[
g(x) = \frac{f(x) - f(a)}{x - a} = \frac{(2\ln x + 1) - (2\ln a + 1)}{x - a} = \frac{2(\ln x - \ln a)}{x - a},\quad x > 0,\; x \ne a.
\]  
将表达式变形为  
\[
g(x) = \frac{2}{x - a} \cdot \ln\frac{x}{a}.
\]  
令 \(t = \dfrac{x}{a}\) (\(t > 0\)，\(t \ne 1\))，则 \(x = at\)，  
\[
g(x) = \frac{2}{a(t - 1)} \cdot \ln t.
\]  
此时 \(g(x)\) 的单调性取决于函数 \(\psi(t) = \dfrac{\ln t}{t - 1}\) 的单调性（\(a > 0\) 为常数）。为直接研究 \(g(x)\)，对原式求导：  
\[
g'(x) = 2 \cdot \frac{\frac{1}{x}(x - a) - (\ln x - \ln a)}{(x - a)^2}
= \frac{2\left(1 - \dfrac{a}{x} - \ln\dfrac{x}{a}\right)}{(x - a)^2}.
\]  
令 \(t = \dfrac{x}{a}\)，则 \(g'(x)\) 的分子符号取决于  
\[
\varphi(t) = 1 - \frac{1}{t} - \ln t,\quad t > 0,\; t \ne 1.
\]  
对 \(\varphi(t)\) 求导：  
\[
\varphi'(t) = \frac{1}{t^2} - \frac{1}{t} = \frac{1 - t}{t^2}.
\]  
当 \(0 < t < 1\) 时，\(\varphi'(t) > 0\)，\(\varphi(t)\) 单调递增；  
当 \(t > 1\) 时，\(\varphi'(t) < 0\)，\(\varphi(t)\) 单调递减。  
又 \(\varphi(1) = 1 - 1 - 0 = 0\)，故对一切 \(t > 0\) 且 \(t \ne 1\)，有 \(\varphi(t) < 0\)。  
从而当 \(x > 0\)，\(x \ne a\) 时，\(g'(x) < 0\)。  
所以 \(g(x)\) 在 \((0, a)\) 和 \((a, +\infty)\) 上均单调递减。
## 答案
(1) \(c \in [-1, +\infty)\)；  
(2) \(g(x)\) 在 \((0, a)\) 和 \((a, +\infty)\) 上分别单调递减。
## 总结
(1) 不等式恒成立问题，常用分离参数法，转化为求函数的最值，导数工具是分析单调性与最值的核心。  
(2) 分式型函数的单调性讨论，可先化简表达式；求导后若出现比值形式，通过整体换元（如令 \(t = x/a\)）可简化导数的符号判定。注意区间的划分通常由定义域和间断点共同决定。
