# 3-4-8 我的解答

对应题目：[[01_题目/3_函数/3-4_函数的单调性/3-4-8.md|3-4-8]]

## 完整题目

8.(2022 江苏南通一调)写出一个同时具有下列性质①②③的三次函数 \( f\left( x\right)  = \) ___.

① \( f\left( x\right) \) 为奇函数; ② \( f\left( x\right) \) 存在 3 个不同的零点; ③ \( f\left( x\right) \) 在 \( \left( {1, + \infty }\right) \) 上单调递增.

## 题目勘误
无明显勘误。
## 思路分析
本题要求构造一个三次函数 \( f(x) \)，同时满足奇函数、有 3 个不同零点、在 \((1,+\infty)\) 上单调递增这三个条件。可从一般三次函数出发，利用奇函数性质消去偶次项，再利用零点条件确定系数异号，最后结合单调性得到导数的约束，并取一组满足条件的特值得出答案。
## 详细解答
设三次函数的一般形式为
\[ f(x) = ax^3 + bx^2 + cx + d \quad (a \neq 0). \]

**① \( f(x) \) 为奇函数**  
由奇函数的定义 \( f(-x) = -f(x) \) 得
\[
a(-x)^3 + b(-x)^2 + c(-x) + d = -\bigl(ax^3 + bx^2 + cx + d\bigr)
\]
\[
-ax^3 + bx^2 - cx + d = -ax^3 - bx^2 - cx - d
\]
比较系数可得 \( b = 0 \)，\( d = 0 \)。  
因此 \( f(x) = ax^3 + cx \)。

**② \( f(x) \) 存在 3 个不同的零点**  
令 \( f(x) = 0 \)，得
\[
x(ax^2 + c) = 0.
\]
一个零点为 \( x_1 = 0 \)。要存在另外两个不同的零点，一元二次方程 \( ax^2 + c = 0 \) 必须有两个不相等的非零实根，所以 \( a \) 与 \( c \) 异号且 \( c \neq 0 \)，即
\[
ac < 0, \quad c \neq 0.
\]

**③ \( f(x) \) 在 \((1, +\infty)\) 上单调递增**  
求导得
\[
f'(x) = 3ax^2 + c.
\]
为保证在 \((1,+\infty)\) 上 \( f(x) \) 单调递增，需在该区间上 \( f'(x) \geq 0 \) 恒成立。  
– 若 \( a < 0 \)，则 \( c > 0 \)，此时 \( f'(x) = 3ax^2 + c \) 当 \( x \to +\infty \) 时趋向 \( -\infty \)，不可能在 \((1,+\infty)\) 上恒非负，舍去。  
– 故必有 \( a > 0 \)，从而由 \( ac < 0 \) 得 \( c < 0 \)。  
此时 \( y = 3ax^2 + c \) 是开口向上的抛物线，且对称轴为 \( x = 0 \)，在 \((0,+\infty)\) 上单调递增，因此在 \((1,+\infty)\) 上的最小值为 \( f'(1) = 3a + c \)。  
要让 \( f'(x) \geq 0 \) 在 \( x > 1 \) 时恒成立，只需
\[
f'(1) = 3a + c \geq 0.
\]
综合得 \( a > 0 \)，\( c < 0 \)，且 \( 3a + c \geq 0 \)。

**构造特例**  
取 \( a = 1 \)，取 \( c = -2 \)（满足 \( c < 0 \) 且 \( 3\times 1 + (-2) = 1 \geq 0 \)），则
\[
f(x) = x^3 - 2x.
\]
验证：
- 奇函数：\( f(-x) = -x^3 + 2x = -f(x) \)；
- 三个零点：\( x(x^2 - 2) = 0 \)，得 \( x = 0, \sqrt{2}, -\sqrt{2} \)；
- 单调性：\( f'(x) = 3x^2 - 2 \)，当 \( x > 1 \) 时 \( f'(x) > 1 > 0 \)，严格单调递增。

满足所有条件，因此可以作为一个答案。
## 答案
\[
f(x) = x^3 - 2x \quad (\text{答案不唯一，满足条件的均可})
\]
## 总结
处理此类“写出满足若干性质的三次函数”的开放题，通常先设一般式，依次代入条件缩小参数范围：奇函数消去偶次项，零点个数决定方程根的情况，单调性转化为导数符号问题，最后取符合范围的一组简单数值即可。需特别注意三次函数零点的分布与导数极值点的关系，以及“单调递增”的区间端点导数可否为零。
