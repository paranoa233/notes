# 3-4-5 我的解答

对应题目：[[01_题目/3_函数/3-4_函数的单调性/3-4-5.md|3-4-5]]

## 完整题目

5.(多选题，2023 安徽、云南、吉林、黑龙江联考) 已知 \( f\left( x\right) \) 是定义在 \( \mathbf{R} \) 上的偶函数， \( g\left( x\right) \) 是定义在 \( \mathbf{R} \) 上的奇函数，且 \( f\left( x\right) \) ， \( g\left( x\right) \) 在 \( ( - \infty ,0\rbrack \) 上单调递减，则 ( )

A. \( f\left\lbrack  {f\left( 1\right) }\right\rbrack   < f\left\lbrack  {f\left( 2\right) }\right\rbrack \)

B. \( f\left\lbrack  {g\left( 1\right) }\right\rbrack   < f\left\lbrack  {g\left( 2\right) }\right\rbrack \)

C. \( g\left\lbrack  {f\left( 1\right) }\right\rbrack   < g\left\lbrack  {f\left( 2\right) }\right\rbrack \)

D. \( g\left\lbrack  {g\left( 1\right) }\right\rbrack   < g\left\lbrack  {g\left( 2\right) }\right\rbrack \)

## 题目勘误
无明显勘误。

## 思路分析
本题主要考查函数的奇偶性与单调性的综合应用。首先根据 \( f(x) \) 为偶函数、\( g(x) \) 为奇函数，结合它们在 \((-\infty,0]\) 上的单调性，推导出它们在 \([0,+\infty)\) 以及整个 \(\mathbf{R}\) 上的单调性。然后分析各选项中内外层函数自变量的大小关系，并注意自变量落在哪个单调区间，从而利用单调性比较函数值的大小。因为是多选题，需要逐项判断恒成立与否。

## 详细解答
**第一步：推导 \(f(x)\) 与 \(g(x)\) 的单调性**

\(f(x)\) 是偶函数，图像关于 \(y\) 轴对称。已知 \(f(x)\) 在 \((-\infty,0]\) 上单调递减，则在 \([0,+\infty)\) 上单调递增。

\(g(x)\) 是奇函数，图像关于原点对称，且 \(g(0)=0\)。已知 \(g(x)\) 在 \((-\infty,0]\) 上单调递减。
设 \(0 \le x_1 < x_2\)，则 \(-x_2 < -x_1 \le 0\)。由单调递减得 \(g(-x_2) > g(-x_1)\)。
由奇函数性质 \(g(-x) = -g(x)\)，得 \(-g(x_2) > -g(x_1)\)，即 \(g(x_2) < g(x_1)\)。
所以 \(g(x)\) 在 \([0,+\infty)\) 上也单调递减。又因为 \(g(0)=0\)，故 \(g(x)\) 在整个 \(\mathbf{R}\) 上单调递减。

由单调性可知：
- 对于 \(f(x)\)：在 \((-\infty,0]\) 上，\(x_1 < x_2 \le 0 \implies f(x_1) > f(x_2)\)；在 \([0,+\infty)\) 上，\(0 \le x_1 < x_2 \implies f(x_1) < f(x_2)\)。
- 对于 \(g(x)\)：在 \(\mathbf{R}\) 上，\(x_1 < x_2 \implies g(x_1) > g(x_2)\)。

并且，因为 \(1 < 2\)，有 \(f(1) < f(2)\)，\(g(1) > g(2)\)，且由 \(g(x)\) 在 \([0,+\infty)\) 递减且过原点知 \(g(2) < g(1) < 0\)。

**第二步：逐项分析**

**A.** \(f[f(1)] < f[f(2)]\)

已知 \(f(1) < f(2)\)，但这两个自变量的大小不能确定它们相对于 \(0\) 的位置。\(f(x)\) 的最小值为 \(f(0)\)，\(f(1),f(2) \ge f(0)\)，可能同时为正、同时为负或一正一负。
- 若 \(0 \le f(1) < f(2)\)，由于 \(f\) 在 \([0,+\infty)\) 递增，则 \(f(f(1)) < f(f(2))\)。
- 若 \(f(1) < f(2) \le 0\)，由于 \(f\) 在 \((-\infty,0]\) 递减，则 \(f(f(1)) > f(f(2))\)。
因此 A 不一定成立。

**B.** \(f[g(1)] < f[g(2)]\)

由 \(g\) 的单调性知 \(g(2) < g(1) < 0\)，即自变量均落在 \((-\infty,0]\) 内。\(f(x)\) 在该区间单调递减，所以当 \(g(2) < g(1)\) 时，\(f(g(2)) > f(g(1))\)，即 \(f(g(1)) < f(g(2))\)。B 恒成立。

**C.** \(g[f(1)] < g[f(2)]\)

\(g(x)\) 在 \(\mathbf{R}\) 上单调递减。已知 \(f(1) < f(2)\)，由单调递减得 \(g(f(1)) > g(f(2))\)，与选项不等式相反。C 不成立。

**D.** \(g[g(1)] < g[g(2)]\)

\(g(x)\) 在 \(\mathbf{R}\) 上单调递减。由前面知 \(g(1) > g(2)\)，所以 \(g(g(1)) < g(g(2))\)。D 恒成立。

## 总结
1. 对于抽象函数的大小比较，关键是利用奇偶性将函数的单调区间补充完整，并严格判断自变量所在的单调区间。
2. 偶函数在关于原点对称的区间上单调性相反；奇函数在对称区间上单调性相同（本题中从而推出在 \(\mathbf{R}\) 上单调递减）。
3. 需注意复合函数的内层函数值作为外层函数的自变量，判断其落在哪个单调区间，不能想当然地直接使用单调性。
4. 涉及不定情况时（如 A 选项中自变量正负不定），要能想到反例，判断该选项是否恒成立。
