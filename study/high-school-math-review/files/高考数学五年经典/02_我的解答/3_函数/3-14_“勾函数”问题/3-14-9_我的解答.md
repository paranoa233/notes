# 3-14-9 我的解答

对应题目：[[01_题目/3_函数/3-14_“勾函数”问题/3-14-9.md|3-14-9]]

## 完整题目

9.(2023 辽宁本溪期中)若 \( a\left( {\sin x + \cos x}\right)  \leq  2 \; + \sin x\cos x \) 对任意 \( x \in  \left( {0,\frac{\pi }{2}}\right) \) 恒成立，则 \( a \) 的最大值为___.

## 题目勘误
无明显勘误。
## 思路分析
不等式对任意 \(x \in \left(0,\frac{\pi}{2}\right)\) 恒成立，求参数 \(a\) 的最大值。因为当 \(x \in \left(0,\frac{\pi}{2}\right)\) 时，\(\sin x + \cos x > 0\)，可将参数 \(a\) 分离出来。问题转化为求右边表达式在给定区间上的最小值。通过换元 \(t = \sin x + \cos x\)，可将右边转化为关于 \(t\) 的函数，再利用对勾函数的单调性求出最小值，从而得到 \(a\) 的最大值。
## 详细解答
因为 \(x \in \left(0, \frac{\pi}{2}\right)\)，所以 \(\sin x + \cos x = \sqrt{2}\sin\left(x+\frac{\pi}{4}\right) > 0\)。
原不等式 \(a(\sin x + \cos x) \leq 2 + \sin x \cos x\) 可化为：
\[
a \leq \frac{2 + \sin x \cos x}{\sin x + \cos x}
\]
要使原不等式对任意 \(x \in \left(0, \frac{\pi}{2}\right)\) 恒成立，只需 \(a \leq \left(\frac{2 + \sin x \cos x}{\sin x + \cos x}\right)_{\min}\)。

令 \(t = \sin x + \cos x\)，则
\[
t = \sqrt{2}\sin\left(x+\frac{\pi}{4}\right) \in (1, \sqrt{2}]
\]
又因为 \((\sin x + \cos x)^2 = 1 + 2\sin x \cos x\)，所以
\[
\sin x \cos x = \frac{t^2 - 1}{2}
\]
代入右边表达式，得到关于 \(t\) 的函数：
\[
f(t) = \frac{2 + \frac{t^2 - 1}{2}}{t} = \frac{t^2 + 3}{2t} = \frac{t}{2} + \frac{3}{2t} \quad t \in (1, \sqrt{2}]
\]
求 \(f(t)\) 在 \((1, \sqrt{2}]\) 上的最小值。
\(f(t) = \frac{1}{2}\left(t + \frac{3}{t}\right)\)，由对勾函数的单调性可知，\(y = t + \frac{3}{t}\) 在 \((0, \sqrt{3})\) 上单调递减。
因为 \((1, \sqrt{2}] \subset (0, \sqrt{3})\)，所以 \(f(t)\) 在 \((1, \sqrt{2}]\) 上单调递减。
当 \(t = \sqrt{2}\) 时，\(f(t)\) 取得最小值：
\[
f(\sqrt{2}) = \frac{\sqrt{2}}{2} + \frac{3}{2\sqrt{2}} = \frac{2\sqrt{2}}{4} + \frac{3\sqrt{2}}{4} = \frac{5\sqrt{2}}{4}
\]
所以 \(a \leq \frac{5\sqrt{2}}{4}\)，即 \(a\) 的最大值为 \(\frac{5\sqrt{2}}{4}\)。

## 答案
\(\frac{5\sqrt{2}}{4}\)

## 总结
本题主要考查了含参不等式恒成立问题的求解方法，通过分离参数转化为求函数的最值。解题的关键是利用三角恒等式进行换元，将三角表达式转化为代数函数，并结合对勾函数的单调性求得最值。注意换元后新变量的取值范围必须准确。
