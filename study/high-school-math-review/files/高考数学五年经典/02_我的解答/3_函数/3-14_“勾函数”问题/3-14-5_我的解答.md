# 3-14-5 我的解答

对应题目：[[01_题目/3_函数/3-14_“勾函数”问题/3-14-5.md|3-14-5]]

## 完整题目

5. (2024 浙江温州二模) 若关于 \( x \) 的方程 \( {x}^{2} + \; {mx} + 1\left| +\right| {x}^{2} - {mx} + 1\left| { = 2}\right| {mx} \mid \) 的整数根有且仅有两个,则实数 \( m \) 的取值范围是 ( )

A. \( \left\lbrack  {2,\frac{5}{2}}\right) \)

B. \( \left( {2,\frac{5}{2}}\right) \)

C. \( \left( {-\frac{5}{2}, - 2}\right\rbrack   \cup  \left\lbrack  {2,\frac{5}{2}}\right) \)

D. \( \left( {-\frac{5}{2}, - 2}\right)  \cup  \left( {2,\frac{5}{2}}\right) \)

## 题目勘误
原题中方程排版存在 OCR 识别错误：`{x}^{2} + \; {mx} + 1\left| +\right| {x}^{2} - {mx} + 1\left| { = 2}\right| {mx} \mid`。建议更正为：
\[
|x^2+mx+1| + |x^2-mx+1| = 2|mx|
\]
理由：原式中的 `\left|`、`\right|` 及末尾 `\mid` 为绝对值符号的误识别，结合题意应为两个含绝对值的二次式之和等于 \(2|mx|\)。

## 思路分析
借助绝对值恒等式化简方程，将其等价转化为关于 \(x\) 的不等式，进而利用“对勾函数”的性质讨论整数解的个数。由整数根恰有两个，反推出参数 \(m\) 的取值范围。

## 详细解答
令 \(a = x^2+1\)，\(b = mx\)。由于 \(x^2+1 > 0\)，故 \(a > 0\)。
利用绝对值恒等式：
\[
|a+b| + |a-b| = 2\max(|a|, |b|)
\]
原方程左边 \(|x^2+mx+1| + |x^2-mx+1| = 2\max(x^2+1, |mx|)\)，右边为 \(2|mx|\)。于是方程等价于
\[
2\max(x^2+1,\, |mx|) = 2|mx| \quad\Longleftrightarrow\quad \max(x^2+1,\, |mx|) = |mx|
\]
即
\[
|mx| \ge x^2+1
\]
当 \(x=0\) 时，不等式左边为 \(0\)，右边为 \(1\)，不成立。对 \(x \ne 0\)，两边除以 \(|x|\)：
\[
|m| \ge |x| + \frac{1}{|x|}
\]
记 \(t = |x|\)，则 \(t\) 为正整数（\(1,2,3,\dots\)）。问题转化为：使得上述不等式成立的整数 \(t\) 恰有两个（即 \(\pm x\) 成对出现，共两个整数根）。

函数 \(f(t) = t + \frac{1}{t}\)（\(t\ge 1\)）是“对勾函数”的离散形式，最小值为 \(f(1)=2\)，随 \(t\) 增大单调递增：
\[
f(1)=2,\quad f(2)=2 + \frac{1}{2} = \frac{5}{2},\quad f(3)=3 + \frac{1}{3} = \frac{10}{3},\dots
\]
要使方程恰有两个整数根，必须且只须 \(t=1\) 满足不等式，而 \(t=2\) 不满足，即
\[
\begin{cases}
|m| \ge 2, \\
|m| < \dfrac{5}{2}.
\end{cases}
\]
因此 \(2 \le |m| < \dfrac{5}{2}\)。将绝对值展开得
\[
m \in \left(-\frac{5}{2}, -2\right] \cup \left[2, \frac{5}{2}\right)
\]
需注意端点：当 \(|m| = \frac{5}{2}\) 时，\(t=2\) 也会成为根，导致四个整数根，故不含 \(\pm\frac{5}{2}\)；当 \(|m| = 2\) 时，\(t=1\) 满足，\(t=2\) 不满足，符合要求，故包含 \(\pm 2\)。

## 答案
选项 C
\[
\left(-\frac{5}{2}, -2\right] \cup \left[2, \frac{5}{2}\right)
\]

## 总结
本题的关键是将复杂绝对值方程通过恒等式 \(|a+b|+|a-b| = 2\max(|a|,|b|)\) 化简为简洁的不等式 \(|mx| \ge x^2+1\)。然后分离参数，将对整数根个数的要求转化为对勾函数 \(t+\frac{1}{t}\) 的取值区间问题，最后依据边界条件确定参数的取值范围。这类题型体现了“多绝对值和”化归为“最大值”的典型思想。
