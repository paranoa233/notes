# 3-2-8 我的解答

对应题目：[[01_题目/3_函数/3-2_函数的定义域与值域/3-2-8.md|3-2-8]]

## 完整题目

8.(2021 江苏南通 3 月模拟)已知集合 \( A = \; \left\{  {x\left| {\;y = \frac{{\left( x + 1\right) }^{0}}{\sqrt{2\left| x\right|  - x}}}\right. }\right. , B = \left\{  {x\left| {\;\frac{x + 2}{x - 1} \leq  0}\right. , x \in  \mathbf{Z}}\right. \) ,则 \( A \cap  B = \) ___.

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查函数的定义域、分式不等式的解法以及集合的交集运算。  
- 集合 \(A\) 是通过函数解析式给出的自然定义域，需同时考虑分母不为零、偶次根号下非负且分母整体不为零、零指数幂底数不为零等限制条件。  
- 集合 \(B\) 是通过分式不等式并限制 \(x \in \mathbf{Z}\) 给出的整数解集，需将其转化为整式不等式组求解，再找出其中的整数。  
- 最后求 \(A \cap B\)，可先化简集合 \(B\)，再逐个检验是否满足 \(A\) 中的定义域限制。
## 详细解答
**第一步：求集合 \(A\)**  
集合 \(A\) 中的元素 \(x\) 必须使解析式 \(y = \dfrac{(x+1)^0}{\sqrt{2|x| - x}}\) 有意义。需满足：
1. 零指数幂的底数不为零：\(x+1 \neq 0 \Rightarrow x \neq -1\)。  
2. 分母不为零且偶次根号下非负，而分母整体不为零意味着根号下必须大于零：  
   \(2|x| - x > 0\)。
   - 当 \(x \geq 0\) 时，\(2x - x = x > 0 \Rightarrow x > 0\)；
   - 当 \(x < 0\) 时，\(-2x - x = -3x > 0\)，由于 \(x < 0\)，该不等式恒成立，但要结合 \(x \neq -1\)，即 \(x < 0\) 且 \(x \neq -1\)；  
   - 注意 \(x = 0\) 使分母为零，不满足 \(x > 0\)，故舍去。  
综上，集合 \(A = \{x \mid x < -1 \text{ 或 } -1 < x < 0 \text{ 或 } x > 0\}\)，即 \(A = \mathbf{R} \setminus \{-1, 0\}\)。

**第二步：求集合 \(B\)**  
集合 \(B\) 由不等式 \(\dfrac{x+2}{x-1} \leq 0\) 且 \(x \in \mathbf{Z}\) 确定。  
不等式等价于
\[
\begin{cases}
(x+2)(x-1) \leq 0, \\
x - 1 \neq 0,
\end{cases}
\]
解得 \(-2 \leq x < 1\)。  
又 \(x\) 是整数，所以 \(x\) 的可能取值为 \(-2, -1, 0\)。  
因此 \(B = \{-2, -1, 0\}\)。

**第三步：求 \(A \cap B\)**  
从集合 \(B\) 中剔除集合 \(A\) 没有的元素 \(-1\) 和 \(0\)，只保留 \(-2\)。  
故 \(A \cap B = \{-2\}\)。
## 答案
\(\{-2\}\)
## 总结
求解此类涉及定义域与不等式整数解的交集问题时，关键点在于：  
1. 定义域的约束条件要完整（分母不为零、偶次根号非负、零次幂底数非零等）。  
2. 解分式不等式时注意分母不为零，并优先转化为整式不等式组。  
3. 涉及整数解时，先求出连续的解区间，再取出整数元素参与后续集合运算。
