# 3-2-4 我的解答

对应题目：[[01_题目/3_函数/3-2_函数的定义域与值域/3-2-4.md|3-2-4]]

## 完整题目

4.(2020 河南郑州一中期中)已知函数 \( y = \; f\left( {x + 1}\right) \) 的定义域为 \( \left\lbrack  {-2,3}\right\rbrack \) ,则 \( y = f\left( {{2x} - 1}\right) \) 的定义域为 ( )

A. \( \left\lbrack  {0,\frac{5}{2}}\right\rbrack \) B. \( \left\lbrack  {-1,4}\right\rbrack \)

C. \( \left\lbrack  {-5,5}\right\rbrack \) D. \( \left\lbrack  {-3,7}\right\rbrack \)

## 题目勘误
无明显勘误。
## 思路分析
这是一道抽象函数定义域问题。关键要明确：定义域始终指自变量 \(x\) 的取值范围。对于 \(y = f(x+1)\)，已知定义域即可求得括号内整体 \(x+1\) 的取值范围，这个范围就是函数 \(f\) 的定义域。再根据 \(f\) 的定义域不变，列出关于 \(2x-1\) 的不等式，解出 \(x\) 的范围即为 \(y = f(2x-1)\) 的定义域。
## 详细解答
因为 \(y = f(x+1)\) 的定义域为 \([-2, 3]\)，所以自变量 \(x\) 满足：
\[
-2 \le x \le 3.
\]
由此可得 \(x+1\) 的取值范围：
\[
-2 + 1 \le x + 1 \le 3 + 1 \quad \Rightarrow \quad -1 \le x+1 \le 4.
\]
所以函数 \(f\) 的定义域为 \([-1, 4]\)。

对于函数 \(y = f(2x-1)\)，由 \(f\) 的定义域不变，可知：
\[
-1 \le 2x - 1 \le 4.
\]
解该不等式：
\[
\begin{aligned}
-1 + 1 &\le 2x \le 4 + 1, \\
0 &\le 2x \le 5, \\
0 &\le x \le \frac{5}{2}.
\end{aligned}
\]
因此，\(y = f(2x-1)\) 的定义域为 \(\left[0, \dfrac{5}{2}\right]\)。
## 答案
A
## 总结
求解抽象函数定义域问题的核心步骤：
1. 由已知复合函数的定义域（\(x\) 的范围），求出 \(f\) 作用对象整体的取值范围，即得 \(f\) 的定义域；
2. 利用 \(f\) 的定义域不变，令新复合函数中括号内的整体落在该范围内，解出 \(x\) 的范围即为所求定义域。
