# 3-2-1 我的解答

对应题目：[[01_题目/3_函数/3-2_函数的定义域与值域/3-2-1.md|3-2-1]]

## 完整题目

1. (2020 海南儋州一中一模)函数 \( f\left( x\right)  = \; \frac{\sqrt{4 - {x}^{2}}}{\ln \left( {x + 1}\right) } \) 的定义域为 ( )

A. \( \lbrack  - 2, - 1) \cup  ( - 1,2\rbrack \)

B. \( \left( {-1,0}\right)  \cup  (0,2\rbrack \)

C. \( \left\lbrack  {-2,2}\right\rbrack \)

D. \( ( - 1,2\rbrack \)

## 题目勘误
无明显勘误。
## 思路分析
求函数定义域，需使解析式各部分均有意义：偶次根式被开方数非负；对数真数大于零；分母不为零。将各部分条件转化为不等式（组），求交集并整理成区间形式，最后与选项比对。
## 详细解答
函数 \( f(x) = \frac{\sqrt{4 - x^2}}{\ln(x + 1)} \) 有意义的条件：
1. 被开方数非负：\( 4 - x^2 \ge 0 \Rightarrow x^2 \le 4 \Rightarrow -2 \le x \le 2 \)。
2. 对数真数大于零：\( x + 1 > 0 \Rightarrow x > -1 \)。
3. 分母不为零：\( \ln(x + 1) \ne 0 \Rightarrow x + 1 \ne 1 \Rightarrow x \ne 0 \)。

将上述条件取交集：
由条件1和2得 \( -1 < x \le 2 \)，再结合条件3去掉 \( x = 0 \)，得到定义域为：
\[
(-1, 0) \cup (0, 2]
\]
对照选项，B 选项为 \( \left( {-1,0} \right) \cup (0,2\rbrack \)，与所求一致。
## 答案
B
## 总结
求解此类定义域问题，需全面列出根式、分式、对数等结构的限制条件，求交集时注意端点取舍，并准确用区间或集合表示。
