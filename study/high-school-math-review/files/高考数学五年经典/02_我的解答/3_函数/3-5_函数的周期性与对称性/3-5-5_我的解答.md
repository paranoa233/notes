# 3-5-5 我的解答

对应题目：[[01_题目/3_函数/3-5_函数的周期性与对称性/3-5-5.md|3-5-5]]

## 完整题目

5.(2023 福建福州二检)已知函数 \( f\left( x\right) , g\left( x\right) \) 的定义域均为 \( \mathbf{R}, f\left( {x + 1}\right) \) 是奇函数，且 \( f\left( {1 - x}\right)  + \; g\left( x\right)  = 2, f\left( x\right)  + g\left( {x - 3}\right)  = 2 \) ,则 ( )

A. \( f\left( x\right) \) 为奇函数

B. \( g\left( x\right) \) 为奇函数

C. \( \mathop{\sum }\limits_{{k = 1}}^{{20}}f\left( k\right)  = {40} \)

D. \( \mathop{\sum }\limits_{{k = 1}}^{{20}}g\left( k\right)  = {40} \)

## 题目勘误
无明显勘误。

## 思路分析
由“\(f(x+1)\)是奇函数”得到对称关系 \(f(1-x)=-f(1+x)\)。将题设给出的两个函数方程联立，消去 \(g(x)\) 并利用对称关系导出 \(f(x+2)=-f(x)\)，从而知道 \(f(x)\) 的周期为 \(4\)。进而推出 \(g(x)\) 也以 \(4\) 为周期，并判断 \(f,g\) 的对称性（都不是奇函数）。最后利用周期性分别计算前 \(20\) 项和，验证选项 C 与 D。

## 详细解答
### 1. 转化奇函数条件
因为 \(f(x+1)\) 是奇函数，所以对其自变量取相反数有
\[
f\bigl((-x)+1\bigr) = -f(x+1),
\]
即
\[
f(1-x) = -f(1+x). \tag{1}
\]

### 2. 利用两个方程建立联系
已知
\[
f(1-x) + g(x) = 2, \tag{2}
\]
\[
f(x) + g(x-3) = 2. \tag{3}
\]
在 (3) 中令 \(x = t+3\)，得
\[
f(t+3) + g(t) = 2 \quad\Longrightarrow\quad g(x) = 2 - f(x+3). \tag{4}
\]
联立 (2) 与 (4) 得
\[
f(1-x) = f(x+3). \tag{5}
\]

### 3. 导出 \(f(x)\) 的周期
将 (5) 代入 (1)：
\[
f(x+3) = -f(1+x).
\]
令 \(u = x+1\)，则 \(x = u-1\)，代入得
\[
f(u+2) = -f(u),
\]
即
\[
f(x+2) = -f(x).
\]
于是
\[
f(x+4) = -f(x+2) = f(x),
\]
所以 \(f(x)\) 是周期为 \(4\) 的周期函数。

### 4. 求关键点处的函数值
在 (1) 中令 \(x=0\)：\(f(1) = -f(1) \;\Longrightarrow\; 2f(1)=0 \;\Longrightarrow\; f(1)=0\)。
结合 \(f(x+2) = -f(x)\)，令 \(x=1\) 得 \(f(3) = -f(1) = 0\)。
又因周期为 \(4\)，可得 \(f(5)=f(1)=0,\; f(7)=f(3)=0\) 等，且连续四个整数的函数值之和：
\[
f(1)+f(2)+f(3)+f(4) = 0 + f(2) + 0 + f(4).
\]
由 \(f(2) = -f(0)\)，\(f(4) = f(0)\)，故上式为 \(0\)。所以任何一个完整周期内四个整数点的函数值之和为 \(0\)。

### 5. 逐一判断选项

**A. \(f(x)\) 为奇函数**
奇函数要求 \(f(-x) = -f(x)\)，特别地，\(f(0)=0\)。但题设仅给出关于 \((1,0)\) 的对称性 \(f(1-x)=-f(1+x)\) 和周期关系，无法保证 \(f(0)=0\)。事实上，可取 \(f(0)=c \neq 0\) 构造出符合所有条件的函数，故 \(f(x)\) 不一定是奇函数。  
\(\Rightarrow\) **A 错误。**

**B. \(g(x)\) 为奇函数**
在 (2) 中把 \(x\) 换成 \(-x\)：
\[
f(1+x) + g(-x) = 2. \tag{6}
\]
由 (2) 解出 \(g(x)=2-f(1-x)\)，结合 (1) 得
\[
g(x) = 2 + f(1+x).
\]
代入 (6)：
\[
g(-x) = 2 - f(1+x) = 2 - \bigl(g(x)-2\bigr) = 4 - g(x).
\]
所以 \(g(-x) + g(x) = 4\)，若 \(g\) 为奇函数则应有 \(g(-x)+g(x)=0\)，矛盾。  
\(\Rightarrow\) **B 错误。**

**C. \(\displaystyle\sum_{k=1}^{20} f(k) = 40\)**
由于 \(f(x)\) 周期为 \(4\) 且一个周期内整数的函数值之和为 \(0\)，而 \(20\) 是 \(4\) 的倍数，因此
\[
\sum_{k=1}^{20} f(k) = 5 \times \bigl[f(1)+f(2)+f(3)+f(4)\bigr] = 0 \neq 40.
\]
\(\Rightarrow\) **C 错误。**

**D. \(\displaystyle\sum_{k=1}^{20} g(k) = 40\)**
由 (2) 得 \(g(x) = 2 - f(1-x)\)，所以
\[
\sum_{k=1}^{20} g(k) = \sum_{k=1}^{20} 2 - \sum_{k=1}^{20} f(1-k) = 40 - \sum_{k=1}^{20} f(1-k).
\]
当 \(k=1,2,\dots,20\) 时，\(1-k\) 依次取 \(0, -1, -2, \dots, -19\)，同样构成 \(5\) 个完整周期（每个周期内整数点函数值之和为 \(0\)，因为周期性），故 \(\sum_{k=1}^{20} f(1-k)=0\)。从而
\[
\sum_{k=1}^{20} g(k) = 40.
\]
\(\Rightarrow\) **D 正确。**

## 答案
D

## 总结
本题以抽象函数为背景，综合考查了奇偶性、对称性与周期性的推导。核心步骤是：
1. 将“\(f(x+1)\) 是奇函数”转化为恒等式 \(f(1-x) = -f(1+x)\)；
2. 联立两个函数方程消去 \(g\)，得到 \(f\) 的递推关系并确定周期 \(T=4\)；
3. 借助周期与赋值求特殊值，计算前 \(20\) 项和。  
注意区分“中心对称在何处”，避免将关于 \((1,0)\) 对称错误地等同于奇函数。
