# 3-15-8 我的解答

对应题目：[[01_题目/3_函数/3-15_函数的零点问题/3-15-8.md|3-15-8]]

## 完整题目

8. (多选题,2022 河北石家庄质检(一)已知 \( a, b \) 分别是方程 \( {2}^{x} + x = 0,{3}^{x} + x = 0 \) 的两个实数根, 则下列选项正确的是 ( )

A. \( - 1 < b < a < 0 \) B. \( - 1 < a < b < 0 \)

C. \( b \cdot  {3}^{a} < a \cdot  {3}^{b} \) D. \( a \cdot  {2}^{b} < b \cdot  {2}^{a} \)

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查指数函数的性质、函数的零点以及利用导数研究函数单调性证明不等式。首先需要分析方程 \(2^x+x=0\) 与 \(3^x+x=0\) 的根的范围及大小关系，可通过考察对应函数的单调性与零点存在性定理来确定。其次，对于含参不等式选项，通过等价变形构造辅助函数，利用导数判断其在所讨论区间内的单调性，进而比较大小，判断选项正误。
## 详细解答
**第一步：确定 \(a, b\) 的范围与大小关系**

设 \(f(x) = 2^x + x\)，\(g(x) = 3^x + x\)。
因为 \(f(x)\) 与 \(g(x)\) 均为定义在 \(\mathbb{R}\) 上的连续函数（且可导）。
计算端点值：
\(f(-1) = 2^{-1} - 1 = -\frac{1}{2} < 0\)，\(f(0) = 1 > 0\)。
由零点存在性定理，\(f(x)\) 在 \((-1, 0)\) 内至少有一个零点，又 \(f'(x) = 2^x \ln 2 + 1 > 0\) 恒成立，所以 \(f(x)\) 严格递增，零点唯一。从而 \(-1 < a < 0\)。
同理，\(g(-1) = 3^{-1} - 1 = -\frac{2}{3} < 0\)，\(g(0) = 1 > 0\)，且 \(g'(x) = 3^x \ln 3 + 1 > 0\)，故 \(g(x)\) 在 \((-1, 0)\) 内有唯一零点，即 \(-1 < b < 0\)。

比较 \(a, b\) 大小：
当 \(x \in (-1, 0)\) 时，由于底数越大函数值越小，有 \(3^x < 2^x\)，于是
\[
g(x) = 3^x + x < 2^x + x = f(x).
\]
将 \(a\) 代入 \(f\) 得 \(f(a)=0\)，从而 \(g(a) < f(a) = 0\)。
又由 \(b\) 是 \(g(x)\) 的零点知 \(g(b) = 0\)。
故 \(g(a) < g(b)\)。因为 \(g(x)\) 严格递增，所以 \(a < b\)。
结合范围得：\(-1 < a < b < 0\)。
由此 **A 错误，B 正确**。

**第二步：判断选项 C**
已知 \(a, b < 0\)，则 \(ab > 0\)。
不等式 \(b \cdot 3^a < a \cdot 3^b\) 两边同除以正数 \(ab\)，不等号方向不变，等价于
\[
\frac{3^a}{a} < \frac{3^b}{b}.
\]
构造辅助函数 \(h(x) = \frac{3^x}{x}\)，\(x \in (-1, 0)\)。
求导得：
\[
h'(x) = \frac{3^x \ln 3 \cdot x - 3^x}{x^2} = \frac{3^x (x \ln 3 - 1)}{x^2}.
\]
在区间 \((-1, 0)\) 内，\(x < 0\)，故 \(x \ln 3 - 1 < 0\)，且 \(3^x > 0, x^2 > 0\)，所以 \(h'(x) < 0\)，即 \(h(x)\) 在 \((-1, 0)\) 上严格单调递减。
由 \(a < b\)，得 \(h(a) > h(b)\)，即
\[
\frac{3^a}{a} > \frac{3^b}{b}.
\]
这与要证的 \(\frac{3^a}{a} < \frac{3^b}{b}\) 矛盾，因此 **C 错误**。

**第三步：判断选项 D**
同理，对不等式 \(a \cdot 2^b < b \cdot 2^a\)，两边同除以 \(ab > 0\)，得等价形式
\[
\frac{2^b}{b} < \frac{2^a}{a}.
\]
令 \(k(x) = \frac{2^x}{x}\)，\(x \in (-1, 0)\)。
求导得：
\[
k'(x) = \frac{2^x (x \ln 2 - 1)}{x^2}.
\]
由于在 \((-1, 0)\) 上 \(x < 0\)，有 \(x \ln 2 - 1 < 0\)，故 \(k'(x) < 0\)，即 \(k(x)\) 在 \((-1, 0)\) 上严格单调递减。
由 \(a < b\) 得 \(k(a) > k(b)\)，即
\[
\frac{2^a}{a} > \frac{2^b}{b} \implies \frac{2^b}{b} < \frac{2^a}{a}.
\]
该结论与选项 D 等价，因此 **D 正确**。

综上，正确选项为 B 和 D。
