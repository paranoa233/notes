# 3-11-11 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-11.md|3-11-11]]

## 完整题目

2.(2022 新高考卷 I ，7)设 \( a = {0.1}{\mathrm{e}}^{0.1} \) ， \( b = \frac{1}{9} \) ， \( c =  - \ln {0.9} \) ,则 ( )

A. \( a < b < c \) B. \( c < b < a \)

C. \( c < a < b \) D. \( a < c < b \)

## 题目勘误
无明显勘误。
## 思路分析
本题比较三个数 \(a=0.1\mathrm{e}^{0.1},\; b=\frac{1}{9},\; c=-\ln 0.9\) 的大小。发现三个数均与 \(0.1\) 有关：  
\( b = \frac{0.1}{0.9} = \frac{x}{1-x}\big|_{x=0.1} \)，  
\(c = -\ln(1-x)\big|_{x=0.1} \)。  
因此可令 \(x=0.1\in(0,1)\)，将问题转化为比较函数 \(a(x)=x\mathrm{e}^x,\; b(x)=\frac{x}{1-x},\; c(x)=-\ln(1-x)\) 在 \(x=0.1\) 处的值。可利用导数构造函数证明不等式，或利用常见不等式与泰勒展开进行比较。目标得到 \(c<a<b\)，对应选项 C。
## 详细解答
令 \(x = 0.1\)，则  
\[
a = x\mathrm{e}^x,\qquad b = \frac{1}{9} = \frac{x}{1-x},\qquad 
c = -\ln(1-x).
\]

**第一步：比较 \(a\) 与 \(b\)**  
考虑 \(a<b\)，即 \(x\mathrm{e}^x < \dfrac{x}{1-x}\)。因为 \(x>0\)，两边同除以 \(x\) 并移项，等价于
\[
\mathrm{e}^x < \frac{1}{1-x} \iff (1-x)\mathrm{e}^x < 1.
\]
设 \(f(x) = (1-x)\mathrm{e}^x - 1\)，\(x\in[0,1)\)，则
\[
f'(x) = -\mathrm{e}^x + (1-x)\mathrm{e}^x = -x\mathrm{e}^x \le 0,
\]
且 \(f'(x)=0\) 仅在 \(x=0\) 时成立。因此 \(f(x)\) 在 \([0,1)\) 上单调递减，结合 \(f(0)=0\)，得当 \(x>0\) 时 \(f(x)<0\)，即 \((1-x)\mathrm{e}^x < 1\)。  
故 \(a < b\)。

**第二步：比较 \(c\) 与 \(b\)**  
这里可直接用常见对数不等式。或者利用导数：  
构造函数 \(g(x) = \ln(1+x) - \dfrac{x}{1+x}\)，经求导可得当 \(x>0\) 时 \(\ln(1+x) > \dfrac{x}{1+x}\)。  
令 \(x = \dfrac{1}{9}\)，则 \(\ln\!\Bigl(1+\frac{1}{9}\Bigr) > \frac{1/9}{1+1/9} = \frac{1}{10}\)，即 \(\ln\frac{10}{9} > 0.1\)。  
于是 \(c = -\ln 0.9 = \ln\frac{10}{9} > 0.1\)。  
另一方面，由 \(\ln(1+x) < x\;(x>0)\) 得 \(\ln\frac{10}{9} < \frac{1}{9}\)，即 \(c < b\)。  
结合即得 \(0.1 < c < b\)。不过此处我们还需要精确比较 \(a\) 与 \(c\)。下面用导数直接比较 \(a\) 与 \(c\)。

**第三步：比较 \(a\) 与 \(c\)**  
设 \(h(x) = x\mathrm{e}^x + \ln(1-x)\)，\(x\in[0,0.1]\)，则  
\[
h(0) = 0.
\]  
求导：  
\[
h'(x) = \mathrm{e}^x + x\mathrm{e}^x - \frac{1}{1-x} 
= \mathrm{e}^x(1+x) - \frac{1}{1-x}.
\]  
又 \(h'(0)=1-1=0\)。再求二阶导数：  
\[
h''(x) = \mathrm{e}^x(1+x) + \mathrm{e}^x - \frac{1}{(1-x)^2} 
= \mathrm{e}^x(2+x) - \frac{1}{(1-x)^2}.
\]  
当 \(x\in[0,0.1]\) 时，\(\mathrm{e}^x \ge 1+x+\frac{x^2}{2} \ge 1\)，且 \((1-x)^{-2}\) 在区间内最大为 \((0.9)^{-2}=\frac{100}{81}\approx1.235\)，而 \(\mathrm{e}^x(2+x)\) 在 \(x=0.1\) 处约 \(1.105\times2.1\approx2.32\)，易证 \(h''(x)>0\) 恒成立（也可通过变形为 \(\mathrm{e}^x > \frac{1}{(1-x)^2(2+x)}\) 并利用单调性判断）。故 \(h'(x)\) 在 \([0,0.1]\) 上单调递增，结合 \(h'(0)=0\) 得 \(h'(x) > 0\) 对 \(x\in(0,0.1]\) 成立。  
从而 \(h(x)\) 在 \([0,0.1]\) 上单调递增，\(h(0.1) > h(0) = 0\)，即  
\[
0.1\mathrm{e}^{0.1} + \ln 0.9 > 0 \implies 0.1\mathrm{e}^{0.1} > -\ln 0.9,
\]  
所以 \(a > c\)。

综合上述两步得 \(c < a < b\)。

## 答案
C
## 总结
本题通过构造函数将三个数统一为同一变量的函数值，利用导数或已知不等式进行大小比较。关键是识别出 \(b=\frac{x}{1-x},\; c=-\ln(1-x)\) 的联系，并选用合适的辅助函数。构造函数时注意初值 \(f(0)=0\) 及导数的符号判断，体现了“同构”与导数工具在数式比较中的典型应用。
