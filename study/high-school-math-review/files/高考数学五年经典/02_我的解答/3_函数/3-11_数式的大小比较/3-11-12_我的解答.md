# 3-11-12 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-12.md|3-11-12]]

## 完整题目

1.(2020 全国卷Ⅲ文，10)设 \( a = {\log }_{3}2 \) ， \( b = {\log }_{5}3 \) ， \( c = \frac{2}{3} \) ,则 ( )

A. \( a < c < b \) B. \( a < b < c \)

C. \( b < c < a \) D. \( c < a < b \)

## 题目勘误
无明显勘误。

## 思路分析
本题比较对数式 \(a=\log_3 2\)、\(b=\log_5 3\) 与分数 \(c=\dfrac{2}{3}\) 的大小。对数与分数直接比较有困难，可考虑将对数与分数同乘以某个正数，将分数化为同底对数的真数，从而利用对数函数的单调性进行比较。具体地，将 \(a\) 与 \(c\) 的比较转化为 \(3a\) 与 \(2\) 的比较；将 \(b\) 与 \(c\) 的比较转化为 \(3b\) 与 \(2\) 的比较。

## 详细解答
1. **比较 \(a\) 与 \(c\)**：
   已知 \(a = \log_3 2\)，\(c = \dfrac{2}{3}\)。
   计算 \(3a = 3\log_3 2 = \log_3 2^3 = \log_3 8\)。
   而 \(3c = 3 \times \dfrac{2}{3} = 2 = \log_3 3^2 = \log_3 9\)。
   因为底数 \(3>1\)，函数 \(y=\log_3 x\) 在 \((0,+\infty)\) 上单调递增，且 \(8 < 9\)，
   所以 \(\log_3 8 < \log_3 9\)，即 \(3a < 3c\)，
   从而 \(a < c\)。

2. **比较 \(b\) 与 \(c\)**：
   已知 \(b = \log_5 3\)，\(c = \dfrac{2}{3}\)。
   计算 \(3b = 3\log_5 3 = \log_5 3^3 = \log_5 27\)。
   而 \(3c = 2 = \log_5 5^2 = \log_5 25\)。
   因为底数 \(5>1\)，函数 \(y=\log_5 x\) 在 \((0,+\infty)\) 上单调递增，且 \(27 > 25\)，
   所以 \(\log_5 27 > \log_5 25\)，即 \(3b > 3c\)，
   从而 \(b > c\)。

综合可得：\(a < c < b\)。

## 答案
A

## 总结
比较对数式与分数的大小时，可利用不等式的性质，将两者同时乘以某个适当的正数（通常取分数的分母），将对数前面的系数“拿进去”变成真数的指数，从而使分数化为整数或对数形式，再利用对数函数的单调性通过比较真数来判定大小。这种方法避免了复杂的换底运算，过程简洁直观。
