# 3-11-8 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-8.md|3-11-8]]

## 完整题目

8.(2023 福建省质检)已知 \( a = \ln 2, b = \mathrm{e} - \frac{1}{a}, c \; = {2}^{a} - a \) ,则 ( )

A. \( b > c > a \) B. \( b > a > c \)

C. \( c > a > b \) D. \( c > b > a \)

## 题目勘误
无明显勘误。

## 思路分析
本题为数值大小比较的选择题，三个数均可用常数 \(a=\ln 2\) 表示。  
直接比较困难，可先对 \(a\) 进行估值，再将 \(b\) 和 \(c\) 转化为关于 \(a\) 的表达式并估值。  
借助 \(e^a = 2\) 可将 \(c=2^a-a\) 化为 \(c=e^{a^2}-a\)，便于利用指数函数的单调性估算范围。  
最终通过范围比较得出三者的大小关系。

## 详细解答
已知 \(a = \ln 2\)，由常用对数估值可得  
\[
0.69 < a < 0.70 .
\]

**估算 \(b\)：**  
\[
b = \mathrm{e} - \frac{1}{a}.
\]  
由 \(a \in (0.69, 0.70)\) 得 \(\frac{1}{a} \in \left( \frac{1}{0.70}, \frac{1}{0.69} \right) \approx (1.4286, 1.4493)\)。  
又 \(\mathrm{e} \approx 2.7183\)，所以  
\[
b \in (2.7183 - 1.4493,\; 2.7183 - 1.4286) \approx (1.2690, 1.2897).
\]  
从而有 \(b > 1.25\)。

**估算 \(c\)：**  
由 \(a = \ln 2\) 知 \(\mathrm{e}^a = 2\)，因此  
\[
c = 2^a - a = (\mathrm{e}^a)^a - a = \mathrm{e}^{a^2} - a.
\]  
由 \(a \in (0.69, 0.70)\) 得 \(a^2 \in (0.4761, 0.4900)\)，利用指数函数单调性：  
\[
\mathrm{e}^{0.4761} \approx 1.6099,\quad \mathrm{e}^{0.49} \approx 1.6323.
\]  
于是  
\[
c = \mathrm{e}^{a^2} - a \in (1.6099 - 0.70,\; 1.6323 - 0.69) = (0.9099, 0.9423).
\]  
从而有 \(0.9 < c < 1\)。

**三者比较：**  
\[
a \approx 0.693 < 1,\quad 0.9 < c < 1,\quad b > 1.25,
\]  
所以  
\[
b > c > a.
\]

对应选项为 A。

## 答案
A

## 总结
比较“同源”表达式的大小时，可先对核心参数（本题为 \(a=\ln 2\)）进行合理估值，再借助恒等式（如 \(e^a=2\)）将各量统一为同一类函数，最后利用函数单调性确定范围并比较。估值时精度需保证能区分大小关系。
