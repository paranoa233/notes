# 3-11-10 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-10.md|3-11-10]]

## 完整题目

1.(2021 新高考卷 II,7)已知 \( a = {\log }_{52}b \) ， \( {\log }_{8}3, c = \frac{1}{2} \) ,则下列判断正确的是 ( )

A. \( a > b > c \) B. \( b > a > c \)

C. \( b > c > a \) D. \( c > b > a \)

## 题目勘误
原文：“已知 \( a = {\log }_{52}b \) ， \( {\log }_{8}3, c = \frac{1}{2} \)”  
建议更正：“已知 \( a = \log_5 2 \)，\( b = \log_8 3 \)，\( c = \frac{1}{2} \)”  
理由：原题OCR识别混乱，对数底数与真数分离错误，且缺少变量 \( b \) 的声明，参考2021年新高考II卷真题应为三个已知量 \(a,b,c\)。

## 思路分析
比较三个数的大小，其中 \(c=\frac12\) 是常数，优先考虑以 \(\frac12\) 为中间值分别与 \(a\)、\(b\) 比较。将对数与常数转化：\(\frac12 = \log_5 \sqrt{5} = \log_8 \sqrt{8}\)，借助底数与真数的大小关系即可判断对数与 \(\frac12\) 的大小。最后综合排序得出选项。

## 详细解答
由题意，更正后已知：
\[ a = \log_5 2,\quad b = \log_8 3,\quad c = \frac12. \]

1. **比较 \(a\) 与 \(c\)**：
\[ c = \frac12 = \log_5 5^{\frac12} = \log_5 \sqrt{5}. \]
因为 \(y = \log_5 x\) 是增函数，且 \(\sqrt{5} \approx 2.236\)，
\[ 2 < \sqrt{5} \;\Rightarrow\; \log_5 2 < \log_5 \sqrt{5} \;\Rightarrow\; a < c. \]

2. **比较 \(b\) 与 \(c\)**：
\[ c = \frac12 = \log_8 8^{\frac12} = \log_8 \sqrt{8}. \]
因为 \(y = \log_8 x\) 是增函数，且 \(\sqrt{8} = 2\sqrt{2} \approx 2.828\)，
\[ 3 > \sqrt{8} \;\Rightarrow\; \log_8 3 > \log_8 \sqrt{8} \;\Rightarrow\; b > c. \]

3. **综合排序**：
由 \(a < c\) 且 \(b > c\) 得：
\[ b > c > a. \]

对应选项 C。

## 答案
C

## 总结
比较对数式与常数的大小时，常使用“中间值法”，将常数化为同底对数，利用对数函数的单调性判断。本题选取 \(\frac12\) 为中间值，通过将对数与 \(\log_5\sqrt{5}\)、\(\log_8\sqrt{8}\) 挂钩，轻松得出大小关系。注意巩固特殊常数（如 \(0, \frac12, 1\) 等）与对数的互化技巧。
