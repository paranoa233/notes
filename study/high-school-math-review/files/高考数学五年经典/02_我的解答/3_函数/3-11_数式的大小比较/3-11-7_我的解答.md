# 3-11-7 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-7.md|3-11-7]]

## 完整题目

7.(2021 江苏苏州八校联盟检测)设 \( x = {\log }_{0.4}{0.5} \) ， \( y = {\log }_{1.5}{0.5} \) ,则 ( )

A. \( {xy} < x + y < 0 \) B. \( x + y < {xy} < 0 \)

C. \( x + y < 0 < {xy} \) D. \( {xy} < 0 < x + y \)

## 题目勘误
无明显勘误。
## 思路分析
本题为对数式的大小比较问题。首先需要判断 \(x\) 与 \(y\) 各自的符号及大致范围，然后比较 \(xy\)、\(x+y\) 与 \(0\) 的大小关系。由于直接估算真数、底数较为困难，可利用对数的换底公式或倒数关系，将 \(x+y\) 与 \(xy\) 转化为关于 \(\frac{1}{x}+\frac{1}{y}\) 的形式，通过不等式性质进行判断。
## 详细解答
首先分析 \(x\) 与 \(y\) 的符号及范围：
\[
x = \log_{0.4}0.5,\quad y = \log_{1.5}0.5
\]
- 对于 \(x\)，底数 \(0<0.4<1\) 且真数 \(0<0.5<1\)，由对数函数性质知 \(x>0\)。进一步，因为 \(\log_{0.4}0.4=1\) 且函数递减，又 \(0.5>0.4\)，故 \(0<x<1\)。
- 对于 \(y\)，底数 \(1.5>1\) 且真数 \(0<0.5<1\)，故 \(y<0\)。

于是有 \(xy<0\)，从而排除选项 C（\(xy>0\)）和选项 D（\(0<x+y\) 需进一步判断）。接下来比较 \(x+y\) 与 \(0\) 以及 \(xy\) 的大小。

利用换底公式（或取倒数）：
\[
\frac{1}{x} = \log_{0.5}0.4,\quad \frac{1}{y} = \log_{0.5}1.5
\]
二者相加：
\[
\frac{1}{x}+\frac{1}{y} = \log_{0.5}(0.4 \times 1.5) = \log_{0.5}0.6
\]
因为 \(y=\log_{0.5}t\) 是定义在 \((0,+\infty)\) 上的减函数，且 \(0.5 < 0.6 < 1\)，所以
\[
\log_{0.5}0.5 = 1 > \log_{0.5}0.6 > \log_{0.5}1 = 0
\]
即
\[
0 < \frac{1}{x}+\frac{1}{y} < 1
\]
又因为 \(\frac{1}{x}+\frac{1}{y} = \frac{x+y}{xy}\)，
\[
0 < \frac{x+y}{xy} < 1
\]
已知 \(xy<0\)，不等式两边同时乘以 \(xy\)（负数），不等号方向改变：
\[
0 > x+y > xy
\]
即
\[
xy < x+y < 0
\]
这恰好对应选项 A。

## 答案
A
## 总结
对于含不同底数的对数大小比较，常用的策略是统一底数或使用倒数关系，将复杂的和与积转化为同底对数的运算，再利用对数函数的单调性确定其范围。本题中，通过倒数相加得到 \(\log_{0.5}0.6\) 并利用函数单调性判断其位于 \((0,1)\) 是关键步骤。得出 \(\frac{x+y}{xy}\in(0,1)\) 后，结合 \(xy<0\) 即可确定 \(x+y\) 与 \(xy\) 的大小及符号。
