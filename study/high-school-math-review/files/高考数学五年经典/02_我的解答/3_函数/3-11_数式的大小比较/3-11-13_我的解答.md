# 3-11-13 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-13.md|3-11-13]]

## 完整题目

2.(2020全国卷Ⅲ理，12)已知 \( {5}^{5} < {8}^{4} \) ， \( {13}^{4} < {8}^{5} \) . 设 \( a = {\log }_{5}3, b = {\log }_{8}5, c = {\log }_{13}8 \) ,则 ( )

A. \( a < b < c \) B. \( b < a < c \)

C. \( b < c < a \) D. \( c < a < b \)

## 题目勘误
无明显勘误。

## 思路分析
本题是比较三个对数的大小，通常可借助中间值（如某些分数）来连接。已知条件给出了两个指数不等式，可以转化为对数与分数的关系。观察 \(a=\log_5 3\)、\(b=\log_8 5\)、\(c=\log_{13}8\)，猜测它们与 \(\frac34\)、\(\frac45\) 等分数有大小关系。通过将分数还原为指数形式，结合已知不等式和简单的指数估算，即可确定三个数的大小顺序。

## 详细解答

首先，将目标对数与一些特殊分数比较。

1. 比较 \(a\) 与 \(\frac34\)：
   \(a = \log_5 3 < \frac34 \iff 5^{\frac34} > 3\)。
   因为 \(5^{\frac34} = (5^3)^{\frac14} = 125^{\frac14}\)，而 \(3 = 81^{\frac14}\)，
   显然 \(125^{\frac14} > 81^{\frac14}\)，所以 \(5^{\frac34} > 3\) 成立，得
   \[ a < \frac34. \]

2. 比较 \(b\) 与 \(\frac34\)：
   \(b = \log_8 5 > \frac34 \iff 8^{\frac34} < 5\)。
   计算 \(8^{\frac34} = (8^3)^{\frac14} = 512^{\frac14}\)，而 \(5 = 625^{\frac14}\)，
   因为 \(512^{\frac14} < 625^{\frac14}\)，所以 \(8^{\frac34} < 5\) 成立，得
   \[ b > \frac34. \]
   结合步骤1知 \(a < \frac34 < b\)，即
   \[ a < b. \]

3. 利用已知不等式得出 \(b\) 与 \(c\) 的大小关系：
   已知 \(5^5 < 8^4\)，两边取以8为底的对数：
   \[ \log_8(5^5) < \log_8(8^4) \implies 5\log_8 5 < 4 \implies 5b < 4 \implies b < \frac45. \]
   已知 \(13^4 < 8^5\)，两边取以13为底的对数：
   \[ \log_{13}(13^4) < \log_{13}(8^5) \implies 4 < 5\log_{13}8 \implies 4 < 5c \implies c > \frac45. \]
   因此
   \[ b < \frac45 < c \implies b < c. \]

4. 综合可知 \(a < b\) 且 \(b < c\)，所以
   \[ a < b < c. \]

对照选项，正确选项为 A。

## 答案
A

## 总结
本题通过寻找中间分数 \(\frac34\) 和 \(\frac45\) 来分段比较三个对数的大小。关键操作是将对数与分数的不等式还原为指数形式，利用已知的指数不等式 \(5^5<8^4\) 和 \(13^4<8^5\) 直接得到 \(b<\frac45<c\)，再借助简单估算得到 \(a<\frac34<b\)。此类“搭桥”比较法在解决对数大小比较问题时非常高效。
