# 3-11-2 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-2.md|3-11-2]]

## 完整题目

2.(2023 广东广州二模)已知 \( a = {3}^{\frac{2}{3}}, b = {2}^{\frac{3}{4}}, c = \; {4}^{\frac{1}{3}} \) ,则 ( )

A. \( c < a < b \) B. \( b < c < a \)

C. \( b < a < c \) D. \( c < b < a \)

## 题目勘误
无明显勘误。
## 思路分析
本题是典型的指数式大小比较问题，核心是灵活运用指数函数的单调性，通过化为同底数、同指数或同时乘方等方式，将陌生比较转化为熟悉的形式。观察三个数：\(a=3^{\frac{2}{3}}\)，\(b=2^{\frac{3}{4}}\)，\(c=4^{\frac{1}{3}}\)。我们可以先将 \(c\) 改写成以 \(2\) 为底的形式，这样 \(b\) 与 \(c\) 底数相同，可直接比较指数；而 \(a\) 与 \(c\) 的指数形式接近，可化为同指数比较底数。最后 \(a\) 与 \(b\) 底数、指数均不同，则可考虑同时将指数通分为以 \(12\) 为分母，比较乘方后的大小。
## 详细解答
1. **化简 \(c\) 并比较 \(a\) 与 \(c\)**：
\[
c = 4^{\frac{1}{3}} = (2^2)^{\frac{1}{3}} = 2^{\frac{2}{3}}.
\]
此时 \(a = 3^{\frac{2}{3}}\)，\(c = 2^{\frac{2}{3}}\)，两者指数相同（均为 \(\frac{2}{3}\)）。
因为幂函数 \(y = x^{\frac{2}{3}}\) 在 \(x > 0\) 时单调递增，且底数 \(3 > 2\)，
所以 \(a > c\)。

2. **比较 \(b\) 与 \(c\)**：
\(b = 2^{\frac{3}{4}}\)，\(c = 2^{\frac{2}{3}}\)，两者底数相同（均为 \(2\)）。
因为指数函数 \(y = 2^x\) 在 \(\mathbb{R}\) 上单调递增，且指数 \(\frac{3}{4} > \frac{2}{3}\)，
所以 \(b > c\)。

3. **比较 \(a\) 与 \(b\)**：
将 \(a\) 与 \(b\) 的指数通分为分母为 \(12\) 的分数：
\[
a = 3^{\frac{2}{3}} = 3^{\frac{8}{12}} = (3^8)^{\frac{1}{12}}, \quad b = 2^{\frac{3}{4}} = 2^{\frac{9}{12}} = (2^9)^{\frac{1}{12}}.
\]
现在只需比较底数的乘方：
\[
3^8 = 6561, \quad 2^9 = 512.
\]
显然 \(6561 > 512\)，又因为 \(y = x^{\frac{1}{12}}\) 在 \(x > 0\) 时单调递增，
所以 \((3^8)^{\frac{1}{12}} > (2^9)^{\frac{1}{12}}\)，即 \(a > b\)。

4. **综合排序**：
由 \(a > b\)，\(b > c\)，\(a > c\) 可得：
\[
c < b < a.
\]
对应选项 D。
## 答案
D
## 总结
比较几个正数指数幂的大小，常用的通法有：
- **底数相同比指数**：利用指数函数的单调性。
- **指数相同比底数**：利用幂函数的单调性。
- **底数、指数均不同**：可以取对数转化为线性比较，或同时乘方（如本题将指数通分为同分母）化为同指数，再比较新的底数。
本题三种情况均有体现，依次突破即可清晰得到大小关系。
