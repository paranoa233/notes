# 3-11-17 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-17.md|3-11-17]]

## 完整题目

6.(2020全国卷Ⅱ理，11)若 \( {2}^{x} - {2}^{y} < {3}^{-x} - {3}^{-y} \) ，则 ( )

A. \( \ln \left( {y - x + 1}\right)  > 0 \)

B. \( \ln \left( {y - x + 1}\right)  < 0 \)

C. \( \ln \left| {x - y}\right|  > 0 \)

D. \( \ln \left| {x - y}\right|  < 0 \)

## 题目勘误
无明显勘误。
## 思路分析
已知不等式两边同时包含指数函数，考虑通过移项将变量分离到不等号两侧，构造函数并利用其单调性比较 \(x\) 与 \(y\) 的大小。得出 \(x<y\) 后，代入各个选项的对数表达式判断符号。
## 详细解答
由 \( 2^x - 2^y < 3^{-x} - 3^{-y} \) 移项，将含 \(x\) 的项移至一边，含 \(y\) 的项移至另一边：
\[
2^x - 3^{-x} < 2^y - 3^{-y}
\]
构造函数 \( F(t) = 2^t - 3^{-t} = 2^t - \left(\frac{1}{3}\right)^t \)，则上述不等式即
\[
F(x) < F(y).
\]
对 \(F(t)\) 求导判断单调性：
\[
F'(t) = 2^t \ln 2 + 3^{-t} \ln 3 > 0,
\]
故 \(F(t)\) 在 \(\mathbb{R}\) 上严格单调递增。由 \(F(x) < F(y)\) 可得
\[
x < y \quad \text{即} \quad y - x > 0.
\]
逐个分析选项：
- **A 选项**：\(\ln(y-x+1)\)。因 \(y-x>0\)，则 \(y-x+1 > 1\)，所以 \(\ln(y-x+1) > \ln 1 = 0\)。A 恒成立。
- **B 选项**：\(\ln(y-x+1) < 0\)。由 A 知该式大于 0，故不成立。
- **C 选项**：\(\ln|x-y|\)。由 \(x<y\) 知 \(|x-y| = y-x > 0\)，但 \(y-x\) 可能小于 1、等于 1 或大于 1，因此 \(\ln|x-y|\) 的符号可为负、零或正，无法确定恒大于 0。故 C 不一定成立。
- **D 选项**：\(\ln|x-y| < 0\)。由 C 分析同理，不一定成立。
因此正确选项为 A。
## 答案
A
## 总结
此类“数式大小比较”问题，常见套路是将不等式变形为 \(f(x) < f(y)\) 的形式，通过研究 \(f\) 的单调性得到 \(x\) 与 \(y\) 的大小关系，进而判断含对数的选项。注意变形时需正确移项，使两边函数结构一致且单调性易于判断。本题中构造 \(F(t)=2^t - 3^{-t}\) 并利用其严格递增是关键。
