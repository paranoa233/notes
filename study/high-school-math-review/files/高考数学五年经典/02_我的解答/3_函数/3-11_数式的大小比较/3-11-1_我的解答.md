# 3-11-1 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-1.md|3-11-1]]

## 完整题目

1.(2021 辽宁实验中学等五校期末)设 \( a = {5}^{0.4} \) ， \( b = {\log }_{0.4}{0.5}, c = {\log }_{5}{0.4} \) ,则 \( a, b, c \) 的大小关系是 ( )

A. \( a < b < c \) B. \( c < a < b \)

C. \( c < b < a \) D. \( b < c < a \)

## 题目勘误
无明显勘误。

## 思路分析
本题主要考查指数函数与对数函数的单调性，以及利用“0”和“1”作为中间桥梁进行数式大小比较的方法。分别判断 \(a, b, c\) 的大致范围，再排序即可。

## 详细解答
比较 \(a = 5^{0.4}, \quad b = \log_{0.4}0.5, \quad c = \log_5 0.4\) 的大小。

**第一步：判断 \(a\) 的范围**
由于指数函数 \(y = 5^x\) 在 \(\mathbb{R}\) 上单调递增，且 \(0.4 > 0\)，
\[
a = 5^{0.4} > 5^0 = 1
\]
又 \(0.4 < 1\)，所以 \(5^{0.4} < 5^1 = 5\)，故 \(a > 1\)。

**第二步：判断 \(b\) 的范围**
对数函数 \(y = \log_{0.4} x\) 的底数 \(0.4 \in (0,1)\)，故在 \((0, +\infty)\) 上单调递减。
因为 \(0.5 \in (0.4, 1)\)，由单调递减性可得：
\[
\log_{0.4} 1 < \log_{0.4} 0.5 < \log_{0.4} 0.4
\]
即
\[
0 < b < 1
\]
（另解：也可使用换底公式 \(b = \frac{\lg 0.5}{\lg 0.4}\)，由 \(\lg 0.5 < 0\)，\(\lg 0.4 < 0\)，且 \(|\lg 0.5| < |\lg 0.4|\)，得 \(0 < b < 1\)）

**第三步：判断 \(c\) 的范围**
对数函数 \(y = \log_5 x\) 的底数 \(5 > 1\)，在 \((0, +\infty)\) 上单调递增。
因为 \(0.4 < 1\)，
\[
c = \log_5 0.4 < \log_5 1 = 0
\]
故 \(c < 0\)。

**第四步：综合排序**
由以上分析可得：
\[
c < 0 < b < 1 < a
\]
因此 \(a, b, c\) 的大小关系为 \(c < b < a\)。

## 答案
C

## 总结
比较指数式与对数式大小的常用策略：
1. 底数相同用单调性，底数不同常借助中间值“0”和“1”。
2. 指数函数：底数大于1时，大于1的指数值大于1；底数在0与1之间时，大于0的指数小于1。
3. 对数函数：底数大于1时，真数小于1的对数值小于0；底数在0与1之间时，真数在底数与1之间的对数值在0到1之间。
本题的选项 C 正确。
