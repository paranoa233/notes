# 3-11-14 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-14.md|3-11-14]]

## 完整题目

3.(2023 全国甲卷文,11)已知函数 \( f\left( x\right)  = \; {\mathrm{e}}^{-{\left( x - 1\right) }^{2}} \) . 记 \( a = f\left( \frac{\sqrt{2}}{2}\right) , b = f\left( \frac{\sqrt{3}}{2}\right) , c = f\left( \frac{\sqrt{6}}{2}\right) \) ,则 ( )

A. \( b > c > a \) B. \( b > a > c \)

C. \( c > b > a \) D. \( c > a > b \)

## 题目勘误
无明显勘误。
## 思路分析
函数 \(f(x)=e^{-(x-1)^2}\) 是一个指数型复合函数，其单调性由指数部分的二次函数决定。因为指数函数的底数 \(e>1\) 且外层为递减关系（指数带负号），\(f(x)\) 的大小实质上取决于 \(|x-1|\) 的大小：\(|x-1|\) 越小，\((x-1)^2\) 越小，\(-(x-1)^2\) 越大，从而 \(f(x)\) 越大。因此，比较 \(a,b,c\) 的大小，只需比较各自变量到 \(1\) 的距离。
## 详细解答
函数 \(f(x)=e^{-(x-1)^2}\) 可看作 \(f(x)=e^{-u}\)，其中 \(u=(x-1)^2\ge 0\)。  
因为 \(e^{-u}\) 关于 \(u\) 严格单调递减，所以 \(f(x)\) 的大小由 \(u\) 的大小反向决定：\(u\) 越小，\(f(x)\) 越大。  
又 \(u=(x-1)^2\) 在对称轴 \(x=1\) 两侧随 \(|x-1|\) 的增大而增大。因此 **\(|x-1|\) 越小，\(f(x)\) 越大**。

分别计算三个自变量到 \(1\) 的距离的平方（方便比较）：
\[
\begin{aligned}
d_1 &= \left|\frac{\sqrt{2}}{2} - 1\right| = \frac{2-\sqrt{2}}{2}, \quad d_1^2 = \frac{(2-\sqrt{2})^2}{4} = \frac{6-4\sqrt{2}}{4} = \frac{3-2\sqrt{2}}{2}; \\[4pt]
d_2 &= \left|\frac{\sqrt{3}}{2} - 1\right| = \frac{2-\sqrt{3}}{2}, \quad d_2^2 = \frac{(2-\sqrt{3})^2}{4} = \frac{7-4\sqrt{3}}{4}; \\[4pt]
d_3 &= \left|\frac{\sqrt{6}}{2} - 1\right| = \frac{\sqrt{6}-2}{2}, \quad d_3^2 = \frac{(\sqrt{6}-2)^2}{4} = \frac{10-4\sqrt{6}}{4} = \frac{5-2\sqrt{6}}{2}.
\end{aligned}
\]

比较这三个距离的平方：
- 比较 \(d_2^2\) 与 \(d_3^2\)：  
\(d_2^2 = \frac{7-4\sqrt{3}}{4}\)，\(d_3^2 = \frac{10-4\sqrt{6}}{4}\)。  
只需比较分子：\(7-4\sqrt{3}\) 与 \(10-4\sqrt{6}\)。  
\(4\sqrt{3}\approx 6.928\)，\(4\sqrt{6}\approx 9.798\)，  
\(7-4\sqrt{3}\approx 0.072\)，\(10-4\sqrt{6}\approx 0.202\)，  
所以 \(d_2^2 < d_3^2\)，即 \(d_2 < d_3\)。

- 比较 \(d_3^2\) 与 \(d_1^2\)：  
\(d_3^2 = \frac{10-4\sqrt{6}}{4}\)，\(d_1^2 = \frac{6-4\sqrt{2}}{4}\)。  
比较分子：\(10-4\sqrt{6}\approx 0.202\)，\(6-4\sqrt{2}\approx 6-5.657=0.343\)，  
所以 \(d_3^2 < d_1^2\)，即 \(d_3 < d_1\)。

综上，\(d_2 < d_3 < d_1\)，也就是 \(|x_2-1| < |x_3-1| < |x_1-1|\)。  
因此 \(f(x_2) > f(x_3) > f(x_1)\)，即 \(b > c > a\)。
## 答案
A
## 总结
本题聚焦复合函数的单调性与对称性。将函数值大小转化为自变量到对称轴的距离比较，利用了指数函数保序以及二次型绝对距离的性质。此类“数式大小比较”问题，核心是抓准函数的增减区间与对称特征。
