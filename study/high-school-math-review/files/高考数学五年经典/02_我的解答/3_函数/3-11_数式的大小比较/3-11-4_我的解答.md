# 3-11-4 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-4.md|3-11-4]]

## 完整题目

4.(2024 江西重点中学协作体 3 月联考)已知 \( a = \sin \frac{\pi }{5}, b = \ln \frac{3}{2}, c = \frac{1}{2} \) ,则 ( )

A. \( a > b > c \) B. \( c > a > b \)

C. \( c > b > a \) D. \( a > c > b \)

## 题目勘误
无明显勘误。

## 思路分析
本题要求比较三个数的大小：\( a = \sin \frac{\pi}{5}, b = \ln \frac{3}{2}, c = \frac{1}{2} \)。可先找出它们与 \( \frac{1}{2} \) 的关系：
1. 利用正弦函数在 \( [0, \frac{\pi}{2}] \) 上的单调性，比较 \( \sin \frac{\pi}{5} \) 与 \( \sin \frac{\pi}{6} = \frac{1}{2} \)。
2. 利用对数函数不等式（例如 \( \ln(1+x) < x \)，\( x>0 \)），比较 \( \ln \frac{3}{2} = \ln(1+\frac{1}{2}) \) 与 \( \frac{1}{2} \)。
然后即可得出三者的顺序。

## 详细解答
**第一步：比较 \( a \) 与 \( c \)**
因为 \( \frac{\pi}{5} = 36^\circ > 30^\circ = \frac{\pi}{6} \)，且正弦函数 \( y = \sin x \) 在 \( [0, \frac{\pi}{2}] \) 上单调递增，
所以
\[
a = \sin \frac{\pi}{5} > \sin \frac{\pi}{6} = \frac{1}{2} = c.
\]
即 \( a > c \).

**第二步：比较 \( b \) 与 \( c \)**
由对数不等式：当 \( x > 0 \) 时，有
\[
\frac{x}{1+x} < \ln(1+x) < x.
\]
令 \( x = \frac{1}{2} \)，得
\[
\ln \left( 1 + \frac{1}{2} \right) = \ln \frac{3}{2} < \frac{1}{2}.
\]
即 \( b < c \).

**第三步：综合判断**
由 \( a > c \) 且 \( c > b \)，得到
\[
a > c > b.
\]
对照选项，选 D.

## 答案
D

## 总结
比较数的大小时，常借助中间量（如本题中的 \( \frac{1}{2} \)）作为桥梁，并利用基本初等函数的单调性或经典不等式（如 \( \ln(1+x) < x (x>0) \)、\( \sin x < x \) 等）进行推理，避免直接估算近似值。
