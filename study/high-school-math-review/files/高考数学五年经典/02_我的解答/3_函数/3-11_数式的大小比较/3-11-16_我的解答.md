# 3-11-16 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-16.md|3-11-16]]

## 完整题目

5.(2020全国卷Ⅰ理，12)若 \( {2}^{a} + {\log }_{2}a = {4}^{b} + \; 2{\log }_{4}b \) ,则 ( )

A. \( a > {2b} \) B. \( a < {2b} \)

C. \( a > {b}^{2} \) D. \( a < {b}^{2} \)

## 题目勘误
无明显勘误。
## 思路分析
已知等式涉及指数与对数，可先统一底数，然后构造函数利用单调性比较大小。注意到 \(4^b = 2^{2b}\)，\(2\log_4 b = \log_2 b\)，等式化为 \(2^a + \log_2 a = 2^{2b} + \log_2 b\)。构造函数 \(f(x) = 2^x + \log_2 x\)，分析 \(f(a)\) 与 \(f(2b)\) 的关系，即可得出 \(a\) 与 \(2b\) 的关系。对于 \(a\) 与 \(b^2\)，可通过取特殊值进行排除。
## 详细解答
由对数真数大于零知 \(a>0, b>0\)。  
右边化简：  
\[
4^b = (2^2)^b = 2^{2b},\qquad 2\log_4 b = 2\cdot\frac{\log_2 b}{\log_2 4} = 2\cdot\frac{\log_2 b}{2} = \log_2 b.
\]  
于是原等式化为  
\[
2^a + \log_2 a = 2^{2b} + \log_2 b. \tag{1}
\]  
构造函数 \(f(x) = 2^x + \log_2 x\ (x>0)\)。因为 \(y=2^x\) 和 \(y=\log_2 x\) 在 \((0,+\infty)\) 上均为增函数，所以 \(f(x)\) 是增函数。  

计算 \(f(2b)\)：  
\[
f(2b) = 2^{2b} + \log_2(2b) = 2^{2b} + \log_2 2 + \log_2 b = 2^{2b} + 1 + \log_2 b.
\]  
由 (1) 式，\(2^{2b} + \log_2 b = 2^a + \log_2 a = f(a)\)，代入得  
\[
f(2b) = f(a) + 1.
\]  
所以 \(f(2b) > f(a)\)。因 \(f(x)\) 单调递增，故 \(2b > a\)，即 \(a < 2b\)，选项 B 正确。  

对于选项 C、D：  
- 取 \(b=1\)，等式化为 \(2^a + \log_2 a = 4\)。通过试探（\(a=1.5\) 时和约 \(3.41\)，\(a=2\) 时和为 \(5\)）知 \(a\) 约为 \(1.7\)。此时 \(a > 1 = b^2\)，C 成立，D 不成立。  
- 取 \(b=2\)，等式化为 \(2^a + \log_2 a = 17\)。\(a=4\) 时和为 \(18\)，\(a=3.9\) 时和约 \(16.89\)，故 \(a\) 约为 \(3.9\)。此时 \(a < 4 = b^2\)，D 成立，C 不成立。  

因此 \(a\) 与 \(b^2\) 的大小关系不确定，C、D 均错误。
## 答案
B
## 总结
处理指数与对数混合的等式，常化为同底并构造单调函数，利用函数值的大小关系反推自变量的大小。对于不确定选项，可取特殊值进行排除。注意隐含条件 \(a,b>0\)。
