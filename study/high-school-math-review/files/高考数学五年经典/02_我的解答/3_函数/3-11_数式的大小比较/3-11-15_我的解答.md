# 3-11-15 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-15.md|3-11-15]]

## 完整题目

4.(2022全国甲卷理，12)已知 \( a = \frac{31}{32}, b = \cos \frac{1}{4} \) ， \( c = 4\sin \frac{1}{4} \) ,则 ( )

A. \( c > b > a \) B. \( b > a > c \)

C. \( a > b > c \) D. \( a > c > b \)

## 题目勘误
无明显勘误。

## 思路分析
本题要求比较三个数的大小：\( a = \frac{31}{32} \)，\( b = \cos \frac{1}{4} \)，\( c = 4\sin \frac{1}{4} \)。注意角度 \(\frac{1}{4}\) 为弧度制，且 \(0 < \frac{1}{4} < \frac{\pi}{2}\)。
比较 \( a \) 与 \( b \)：可借助不等式 \(\cos x > 1 - \frac{x^2}{2}\ (x \neq 0)\)，得 \( b > a \)。
比较 \( b \) 与 \( c \)：即比较 \(\cos\frac{1}{4}\) 与 \(4\sin\frac{1}{4}\)，利用 \(\tan x > x\ (0 < x < \frac{\pi}{2})\) 可推得 \( c > b \)。
综合得 \( c > b > a \)。

## 详细解答
首先明确 \( \frac{1}{4} \) 为弧度制，且 \( 0 < \frac{1}{4} < \frac{\pi}{2} \)。

**比较 \( b \) 与 \( a \)**：
已知 \( a = \frac{31}{32} = 1 - \frac{1}{32} \)。
由余弦函数的泰勒展开或构造函数可证：当 \( x \neq 0 \) 时，\(\cos x > 1 - \frac{x^2}{2}\)。
令 \( x = \frac{1}{4} \)，则
\[
b = \cos\frac{1}{4} > 1 - \frac{\left(\frac{1}{4}\right)^2}{2} = 1 - \frac{1}{32} = a.
\]
即 \( b > a \)。

**比较 \( c \) 与 \( b \)**：
\( c = 4\sin\frac{1}{4} \)，\( b = \cos\frac{1}{4} \)。
考虑比值
\[
\frac{c}{b} = \frac{4\sin\frac{1}{4}}{\cos\frac{1}{4}} = 4\tan\frac{1}{4}.
\]
由常见不等式：当 \( 0 < x < \frac{\pi}{2} \) 时，\(\tan x > x\)。
取 \( x = \frac{1}{4} \)，有 \(\tan\frac{1}{4} > \frac{1}{4}\)，因此
\[
4\tan\frac{1}{4} > 4 \times \frac{1}{4} = 1.
\]
故 \(\frac{c}{b} > 1\)，又 \( b = \cos\frac{1}{4} > 0 \)，所以 \( c > b \)。

综上，\( c > b > a \)。

## 答案
A

## 总结
本题通过两个常见不等式（\(\cos x > 1 - \frac{x^2}{2}\) 与 \(\tan x > x\)）完成大小比较。关键在于识别出小角度弧度值，并灵活运用三角函数的不等式，避免直接计算近似值，提高解题严谨性与速度。
