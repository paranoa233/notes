# 3-11-3 我的解答

对应题目：[[01_题目/3_函数/3-11_数式的大小比较/3-11-3.md|3-11-3]]

## 完整题目

3. (2024 广东广州一模)已知 \( a = \frac{3}{2} \) ， \( {3}^{b} = 5 \) ， \( {5}^{c} = \) 8, 则 ( )

A. \( a < b < c \) B. \( a < c < b \)

C. \( c < b < a \) D. \( b < c < a \)

## 题目勘误
无明显勘误。
## 思路分析
题目给出了三个与指数和对数相关的数：\(a=\frac{3}{2}\)，\(b\) 满足 \(3^b=5\)，\(c\) 满足 \(5^c=8\)。直接比较它们的大小，可以考虑引入中间值（如 \(\frac{3}{2}\) 和 \(\frac{4}{3}\)），利用指数函数的单调性，将数与中间值进行比较，从而确定它们的大小顺序。
## 详细解答
1. **分析 \(a\) 与 \(b\) 的大小**  
   \(a = \dfrac{3}{2}\)，\(b = \log_3 5\)。  
   计算 \(3^{3/2} = 3 \cdot \sqrt{3}\)。  
   因为 \(\sqrt{3} > 1.7\)，所以 \(3^{3/2} > 3 \times 1.7 = 5.1 > 5 = 3^b\)。  
   由于函数 \(y = 3^x\) 在 \(\mathbb{R}\) 上单调递增，故 \(a = \dfrac{3}{2} > b\)。

2. **比较 \(b\) 与 \(\frac{4}{3}\)**  
   计算 \(3^{4/3} = \sqrt[3]{3^4} = \sqrt[3]{81}\)。  
   由于 \(81 < 125\)，所以 \(\sqrt[3]{81} < \sqrt[3]{125} = 5 = 3^b\)。  
   由 \(y = 3^x\) 单调递增，得 \(\dfrac{4}{3} < b\)。

3. **分析 \(c\) 与 \(\frac{4}{3}\) 的大小**  
   \(c = \log_5 8\)，计算 \(5^{4/3} = \sqrt[3]{5^4} = \sqrt[3]{625}\)。  
   因为 \(625 > 512\)，所以 \(\sqrt[3]{625} > \sqrt[3]{512} = 8 = 5^c\)。  
   由 \(y = 5^x\) 单调递增，得 \(\dfrac{4}{3} > c\)。

4. **综合大小关系**  
   由上述推导可得：\(c < \dfrac{4}{3} < b < \dfrac{3}{2} = a\)，  
   即 \(c < b < a\)。
## 答案
C
## 总结
本题是典型的数式大小比较问题，关键在于找到合适的中间值（如 \(\frac{4}{3}\)），并通过将待比较的数转化为同底的指数幂，利用指数函数的单调性完成不等关系的传递。注意完全避免近似计算，全部使用精确的不等式放缩。
