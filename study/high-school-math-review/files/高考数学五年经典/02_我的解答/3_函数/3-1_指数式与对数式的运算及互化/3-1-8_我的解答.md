# 3-1-8 我的解答

对应题目：[[01_题目/3_函数/3-1_指数式与对数式的运算及互化/3-1-8.md|3-1-8]]

## 完整题目

8.(2023 辽宁期末联考)若 \( x{\log }_{2}3 = 2 \) ，则 \( {\left( \frac{1}{3}\right) }^{\prime } \; = \)

## 题目勘误
原文“\( {\left( \frac{1}{3}\right) }^{\prime } \)”中，上标 \( \prime \) 疑似 OCR 识别错误，根据已知条件推断，此处应为变量 \( x \)，即原题为 \( {\left( \frac{1}{3}\right) }^{x} \)。

## 思路分析
由已知等式 \( x \log_2 3 = 2 \) 解出 \( x \)，利用对数的换底公式或指对互化得到 \( x \) 的精确对数表达式。然后将 \( x \) 代入所求的 \( \left( \frac{1}{3} \right)^x \)，通过指数运算与对数恒等式 \( a^{\log_a N} = N \) 求出结果。

## 详细解答
由已知条件 \( x{\log }_{2}3 = 2 \)，可得  
\[ x = \frac{2}{\log_2 3}. \]  
利用换底公式 \( \frac{1}{\log_a b} = \log_b a \)，有  
\[ x = 2 \log_3 2 = \log_3 (2^2) = \log_3 4. \]  
于是所求式子  
\[ \left( \frac{1}{3} \right)^x = 3^{-x} = 3^{-\log_3 4} = \frac{1}{3^{\log_3 4}}. \]  
由对数恒等式 \( 3^{\log_3 4} = 4 \)，代入得  
\[ \left( \frac{1}{3} \right)^x = \frac{1}{4}. \]

## 答案
\( \dfrac{1}{4} \)

## 总结
本题主要考查对数的运算性质、换底公式及对数恒等式 \( a^{\log_a N} = N \)。解题时需将已知等式转化为 \( x \) 的对数形式，再进行指数与对数的互化，最终化简求值。
