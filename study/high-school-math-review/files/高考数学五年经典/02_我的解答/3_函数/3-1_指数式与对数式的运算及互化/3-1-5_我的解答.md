# 3-1-5 我的解答

对应题目：[[01_题目/3_函数/3-1_指数式与对数式的运算及互化/3-1-5.md|3-1-5]]

## 完整题目

5.(2024 安徽阜阳一模)设 \( a = {\log }_{2}3, b = {\log }_{8}{12} \) ， \( c = \lg {15} \) ,则 \( a, b, c \) 的大小关系为

A. \( a < b < c \) B. \( a < c < b \)

C. \( b < a < c \) D. \( c < b < a \)

## 题目勘误
无明显勘误。
## 思路分析
本题给出三个对数式，要求比较大小。观察三个数：\(a=\log_{2}3\)，\(b=\log_{8}12\)，\(c=\lg 15\)。由于底数不同，应先考虑化为同底或利用特殊值进行拆分。\(b\) 与 \(c\) 的真数可以拆成 \(10 \times 1.5\) 和 \(8 \times 1.5\)，可写成 \(1\) 加一个对数，利用底数与对数大小的关系直接比较。\(a\) 与 \(b\) 可通过换底公式统一成以 \(2\) 为底，作差比较。
## 详细解答
首先比较 \(b\) 和 \(c\)：
\[
b = \log_{8}12 = \log_{8}(8 \times 1.5) = 1 + \log_{8}1.5,
\]
\[
c = \lg 15 = \log_{10}(10 \times 1.5) = 1 + \log_{10}1.5.
\]
因为 \(1.5>1\)，且底数 \(8<10\)，所以 \(\log_{8}1.5 > \log_{10}1.5\)。于是
\[
b = 1 + \log_{8}1.5 > 1 + \log_{10}1.5 = c.
\]
即 \(b > c\)。

再比较 \(a\) 和 \(b\)：
将 \(b\) 换底成以 \(2\) 为底：
\[
b = \log_{8}12 = \frac{\log_{2}12}{\log_{2}8} = \frac{\log_{2}12}{3} = \frac{\log_{2}(3 \times 4)}{3} = \frac{\log_{2}3 + \log_{2}4}{3} = \frac{a + 2}{3}.
\]
作差比较：
\[
a - b = a - \frac{a+2}{3} = \frac{2a-2}{3} = \frac{2(a-1)}{3}.
\]
因为 \(a = \log_{2}3 > \log_{2}2 = 1\)，所以 \(a-1>0\)，故 \(a-b>0\)，即 \(a > b\)。

综上，\(a > b > c\)，即 \(c < b < a\)，对应选项 D。
## 答案
D
## 总结
比较不同底数的对数大小时，优先考虑统一底数或构造函数利用单调性。本题将 \(b\) 和 \(c\) 拆分成 \(1+\log_{\text{底}}1.5\) 的形式，直接根据底数大小判断对数大小，十分巧妙。比较 \(a\) 与 \(b\) 时利用换底公式建立两者之间的线性关系，再通过作差比较大小，步步紧扣对数基本性质。
