# 3-7-10 我的解答

对应题目：[[01_题目/3_函数/3-7_二次函数的图象与性质/3-7-10.md|3-7-10]]

## 完整题目

10. (2021 江苏淮阴中学测试)设二次函数 \( f\left( x\right) \; = a{x}^{2} + {bx} + c \) ,函数 \( F\left( x\right)  = f\left( x\right)  - x \) 的两个零点分别为 \( m, n\left( {m < n}\right) \) .

(1)若 \( m =  - 1, n = 2 \) ，求不等式 \( F\left( x\right)  > 0 \) 的解集;

(2)若 \( a > 0 \) ，且 \( 0 < x < m < n < \frac{1}{a} \) ，试比较 \( f\left( x\right) \) 与 \( m \) 的大小.

## 题目勘误
无明显勘误。

## 思路分析
（1）由零点写出 \( F(x) = a(x - m)(x - n) \)，代入 \( m=-1, n=2 \) 得到表达式，根据二次项系数 \( a \) 的正负分类讨论解二次不等式。
（2）比较 \( f(x) \) 与 \( m \) 的大小，作差变形为 \( f(x) - m = a(x-m)(x-n) + x - m = (x-m)[a(x-n) + 1] \)，利用已知不等式 \( 0 < x < m < n < 1/a \) 逐步判断各因式的符号，得出结论。

## 详细解答
由题知，二次函数 \( f(x) = ax^2 + bx + c \)，则
\[ F(x) = f(x) - x = ax^2 + (b-1)x + c. \]
因为 \( F(x) \) 的两个零点为 \( m, n\ (m < n) \)，可设
\[ F(x) = a(x - m)(x - n). \]

**（1）** 当 \( m = -1, n = 2 \) 时，
\[ F(x) = a(x + 1)(x - 2). \]
不等式 \( F(x) > 0 \) 即 \( a(x + 1)(x - 2) > 0 \).

- 若 \( a > 0 \)，则 \( (x + 1)(x - 2) > 0 \)，解得 \( x < -1 \) 或 \( x > 2 \)；
- 若 \( a < 0 \)，则 \( (x + 1)(x - 2) < 0 \)，解得 \( -1 < x < 2 \).

故不等式的解集为：
当 \( a > 0 \) 时，\( \{x \mid x < -1 \text{ 或 } x > 2\} \)；
当 \( a < 0 \) 时，\( \{x \mid -1 < x < 2\} \).

**（2）** 由 \( f(x) = F(x) + x = a(x - m)(x - n) + x \)，作差
\[
\begin{aligned}
f(x) - m &= a(x - m)(x - n) + x - m \\
&= (x - m)\bigl[a(x - n) + 1\bigr].
\end{aligned}
\]

已知 \( a > 0 \)，且 \( 0 < x < m < n < \dfrac{1}{a} \)，分别判断各因式的符号：

- 因 \( x < m \)，所以 \( x - m < 0 \).
- 因 \( x < n \)，故 \( x - n < 0 \)，又 \( a > 0 \)，得 \( a(x - n) < 0 \)，从而
\[ a(x - n) + 1 = 1 - a(n - x). \]
由 \( n < \dfrac{1}{a} \) 得 \( an < 1 \)，而 \( n - x < n \)，故
\[ a(n - x) < an < 1, \]
所以 \( 1 - a(n - x) > 0 \)，即 \( a(x - n) + 1 > 0 \).

综上，\( f(x) - m = (x - m)[a(x - n) + 1] < 0 \)，因此
\[ f(x) < m. \]

## 答案
（1）当 \( a > 0 \) 时，解集为 \( (-\infty, -1) \cup (2, +\infty) \)；当 \( a < 0 \) 时，解集为 \( (-1, 2) \)；
（2）\( f(x) < m \).

## 总结
本题以二次函数为基础，考查了二次不等式解法中的分类讨论思想，以及利用作差法与已知不等式范围判断函数值大小的典型方法。在比较 \( f(x) \) 与 \( m \) 时，关键是提取公因式并利用条件 \( n < \frac{1}{a} \) 推出关键因式的正负。
