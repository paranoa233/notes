# 3-3-16 我的解答

对应题目：[[01_题目/3_函数/3-3_函数的奇偶性/3-3-16.md|3-3-16]]

## 完整题目

3.(2023全国甲卷理，13)若 \( y = {\left( x - 1\right) }^{2} + {ax} + \; \sin \left( {x + \frac{\pi }{2}}\right) \) 为偶函数，则 \( a = \) ___.

## 题目勘误
无明显勘误。
## 思路分析
函数表达式中含有二次项、一次项和三角函数，可利用偶函数的定义 \( f(-x) = f(x) \) 建立恒等式，比较系数求出 \( a \)；也可先将三角函数化简，再根据偶函数的多项式部分奇次项系数为零的性质直接得到结果。
## 详细解答
首先化简所给函数：
\[
y = (x-1)^2 + ax + \sin\left(x + \frac{\pi}{2}\right)
\]
利用诱导公式 \(\sin\left(x + \frac{\pi}{2}\right) = \cos x\)，并将 \((x-1)^2\) 展开：
\[
y = x^2 - 2x + 1 + ax + \cos x = x^2 + (a-2)x + 1 + \cos x
\]
令 \( f(x) = y \)。由于 \( f(x) \) 是偶函数，其充要条件是对任意实数 \( x \) 都有 \( f(-x) = f(x) \)。
计算 \( f(-x) \)：
\[
f(-x) = (-x)^2 + (a-2)(-x) + 1 + \cos(-x) = x^2 - (a-2)x + 1 + \cos x
\]
令 \( f(-x) = f(x) \)：
\[
x^2 - (a-2)x + 1 + \cos x = x^2 + (a-2)x + 1 + \cos x
\]
两边消去相同的项 \( x^2 + 1 + \cos x \)，得：
\[
- (a-2)x = (a-2)x \quad \Rightarrow \quad 2(a-2)x = 0
\]
该等式对任意 \( x \) 恒成立，故一次项系数必须为零：
\[
a - 2 = 0
\]
解得 \( a = 2 \)。
## 答案
\( 2 \)
## 总结
本题考查函数奇偶性的判定与应用。通过化简表达式，将函数拆分为偶函数部分（ \( x^2 \)、常数、\( \cos x \) ）和奇函数部分（ \( (a-2)x \) ），利用偶函数不含奇次项的性质，可迅速求出参数值。注意三角诱导公式的准确使用。
