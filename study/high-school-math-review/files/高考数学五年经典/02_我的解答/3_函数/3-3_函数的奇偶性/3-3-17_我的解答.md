# 3-3-17 我的解答

对应题目：[[01_题目/3_函数/3-3_函数的奇偶性/3-3-17.md|3-3-17]]

## 完整题目

4.(2022全国乙卷文，16)若 \( f\left( x\right)  = \ln \left| {a + \frac{1}{1 - x}}\right|  + \; b \) 是奇函数，则 \( a = \) ___， \( b = \) ___.

## 题目勘误
无明显勘误。
## 思路分析
本题要求根据函数的奇偶性求参数值。已知 \( f(x) = \ln\left|a + \frac{1}{1-x}\right| + b \) 是奇函数。奇函数需满足两个条件：①定义域关于原点对称；②对定义域内任意 \( x \)，有 \( f(-x) = -f(x) \)。先利用恒等式 \( f(x) + f(-x) = 0 \) 建立关于 \( a, b \) 的方程，通过整理对数表达式使其恒为常数来确定参数，最后验证定义域是否关于原点对称。
## 详细解答
由 \( f(x) \) 是奇函数知，在定义域内恒有
\[
f(x) + f(-x) = 0.
\]
计算 \( f(-x) \)：
\[
f(-x) = \ln\left|a + \frac{1}{1-(-x)}\right| + b = \ln\left|a + \frac{1}{1+x}\right| + b.
\]
于是
\[
f(x) + f(-x) = \ln\left|\left(a + \frac{1}{1-x}\right)\left(a + \frac{1}{1+x}\right)\right| + 2b = 0.
\]
即
\[
\ln\left|\left(a + \frac{1}{1-x}\right)\left(a + \frac{1}{1+x}\right)\right| = -2b.
\]
上式右端为常数，因此左端绝对值内的式子必须为常数（与 \( x \) 无关）。化简乘积：
\[
\begin{aligned}
\left(a + \frac{1}{1-x}\right)\left(a + \frac{1}{1+x}\right)
&= a^2 + \frac{a}{1+x} + \frac{a}{1-x} + \frac{1}{(1-x)(1+x)} \\
&= a^2 + \frac{a(1-x) + a(1+x)}{(1-x)(1+x)} + \frac{1}{1-x^2} \\
&= a^2 + \frac{2a}{1-x^2} + \frac{1}{1-x^2} \\
&= a^2 + \frac{2a+1}{1-x^2}.
\end{aligned}
\]
令该式与 \( x \) 无关，必须
\[
2a + 1 = 0 \quad \Rightarrow \quad a = -\frac{1}{2}.
\]
此时乘积等于 \( a^2 = \frac{1}{4} \)，于是
\[
\ln\left|\frac{1}{4}\right| + 2b = 0 \quad \Rightarrow \quad \ln\frac{1}{4} + 2b = 0.
\]
因为 \( \ln\frac{1}{4} = -\ln 4 = -2\ln 2 \)，代入得
\[
-2\ln 2 + 2b = 0 \quad \Rightarrow \quad b = \ln 2.
\]
验证定义域：当 \( a = -\frac{1}{2} \) 时，
\[
f(x) = \ln\left|-\frac{1}{2} + \frac{1}{1-x}\right| + \ln 2 = \ln\left|\frac{1+x}{2(1-x)}\right| + \ln 2 = \ln\left|\frac{1+x}{1-x}\right|.
\]
定义域要求 \( \frac{1+x}{1-x} \neq 0 \) 且分母 \( 1-x \neq 0 \)，即 \( x \neq \pm 1 \)，关于原点对称。且 \( f(x) = \ln\left|\frac{1+x}{1-x}\right| \) 满足 \( f(-x) = \ln\left|\frac{1-x}{1+x}\right| = -f(x) \)，确为奇函数。
## 答案
\[
a = -\frac{1}{2},\quad b = \ln 2
\]
## 总结
处理含参数的奇函数问题，常用恒等式 \( f(x) + f(-x) = 0 \) 建立方程。关键步骤是将乘积化简为 \( \text{常数} + \frac{\text{系数}}{1-x^2} \) 的形式，令分母项系数为零即可消去变量依赖，进而解出参数。最终务必检验定义域是否关于原点对称，确保奇函数定义的完整性。
