# 3-3-8 我的解答

对应题目：[[01_题目/3_函数/3-3_函数的奇偶性/3-3-8.md|3-3-8]]

## 完整题目

8. (多选题,2024 江苏南京二模) 已知函数 \( f\left( x\right) \) 满足 \( f\left( x\right) f\left( y\right)  = f\left( {xy}\right)  + \left| x\right|  + \left| y\right| \) ,则(   )

A. \( f\left( 0\right)  = 1 \)

B. \( f\left( 1\right)  =  - 1 \)

C. \( f\left( x\right) \) 是偶函数

D. \( f\left( x\right) \) 是奇函数

## 题目勘误
无明显勘误。

## 思路分析
本题为抽象函数多选题，需通过赋值法求出 \(f(0), f(1)\)，进而确定函数的奇偶性。可先令 \(x=y=0\) 和 \(x=0,y=1\) 解出 \(f(0)\) 并排除矛盾；再令 \(y=1\) 得到 \(f(x)\) 的显式表达式，最后根据表达式判断奇偶性。

## 详细解答
**1. 求 \(f(0)\) 与 \(f(1)\)**  
令 \(x=y=0\)，代入条件：
\[
f(0)f(0) = f(0) + |0| + |0| \implies [f(0)]^2 = f(0)
\]
解得 \(f(0)=0\) 或 \(f(0)=1\)。

再令 \(x=0, y=1\)：
\[
f(0)f(1) = f(0) + |0| + |1| = f(0) + 1
\]
若 \(f(0)=0\)，则 \(0 = 0+1\)，矛盾。故必有 \(f(0)=1\)。  

代入上式得 \(1 \cdot f(1) = 1 + 1\)，即 \(f(1)=2\)。

选项 A：\(f(0)=1\)，正确。  
选项 B：\(f(1)=-1\)，错误。

**2. 求 \(f(x)\) 的表达式**  
在条件中取 \(y=1\)，得：
\[
f(x)f(1) = f(x \cdot 1) + |x| + |1| \implies 2f(x) = f(x) + |x| + 1
\]
整理得：
\[
f(x) = |x| + 1
\]

**3. 验证函数满足条件**  
将 \(f(x)=|x|+1\) 代入原方程：
左边：\((|x|+1)(|y|+1) = |xy| + |x| + |y| + 1\)  
右边：\(f(xy)+|x|+|y| = |xy|+1 + |x| + |y|\)  
两边相等，故该函数为满足条件的解，且由推导过程可知解唯一。

**4. 判断奇偶性**  
\(f(x)=|x|+1\) 的定义域为 \(\mathbb{R}\)，关于原点对称。  
计算 \(f(-x) = |-x|+1 = |x|+1 = f(x)\)，所以 \(f(x)\) 是偶函数。  
选项 C 正确；由于 \(f(0)=1\neq0\)，不可能是奇函数，选项 D 错误。

## 答案
AC

## 总结
解抽象函数问题的关键是合理赋值，从特殊值出发逐步缩小范围，最终确定函数表达式或目标性质。本题中通过赋值 \(x=y=0\) 及 \(x=0,y=1\) 求得 \(f(0),f(1)\)，再令 \(y=1\) 直接解出 \(f(x)=|x|+1\)，使奇偶性一目了然。赋值时注意检验所得值是否满足所有条件，避免产生矛盾解。
