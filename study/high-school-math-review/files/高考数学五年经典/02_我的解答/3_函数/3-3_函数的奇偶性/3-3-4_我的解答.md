# 3-3-4 我的解答

对应题目：[[01_题目/3_函数/3-3_函数的奇偶性/3-3-4.md|3-3-4]]

## 完整题目

4.(2022 江苏南通等七市二调)已知 \( f\left( x\right) \) 是定义域为 \( \mathbf{R} \) 的偶函数, \( f\left( {5.5}\right)  = 2, g\left( x\right)  = \left( {x - 1}\right)  \cdot  f\left( x\right) \) . 若 \( g\left( {x + 1}\right) \) 是偶函数，则 \( g\left( {-{0.5}}\right)  = \) ( )

A. -3 B. -2 C. 2 D. 3

## 题目勘误
无明显勘误。
## 思路分析
本题给出抽象函数 \(f(x)\) 的奇偶性与一个特殊值，以及 \(g(x)\) 的表达式和 \(g(x+1)\) 的奇偶性，要求 \(g(-0.5)\)。解题关键是利用 \(g(x+1)\) 为偶函数推出 \(g(x)\) 的对称性，再结合 \(f(x)\) 是偶函数，导出 \(f(x)\) 满足的递推关系，进而求出周期。利用周期将已知的 \(f(5.5)=2\) 转化，并结合对称性解出所需的 \(f(0.5)\)，最后代入 \(g(x)\) 求值。
## 详细解答
因为 \(g(x+1)\) 是偶函数，所以 \(g(1+x)=g(1-x)\) 对任意实数 \(x\) 成立。

由已知 \(g(x)=(x-1)f(x)\)，代入得：
\[
g(1+x) = (1+x-1)f(1+x) = x f(1+x)
\]
\[
g(1-x) = (1-x-1)f(1-x) = -x f(1-x)
\]
因此 \(x f(1+x) = -x f(1-x)\)。

当 \(x \neq 0\) 时，有 \(f(1+x) = -f(1-x)\)。
令 \(t=1+x\)，则 \(1-x=2-t\)，所以 \(f(t) = -f(2-t)\)，即
\[
f(x) = -f(2-x) \quad \text{①}
\]

又已知 \(f(x)\) 是偶函数，故 \(f(-x)=f(x)\)。
在①式中将 \(x\) 换成 \(-x\)：
\[
f(-x) = -f(2+x) \implies f(x) = -f(2+x) \quad \text{②}
\]
由②得 \(f(2+x) = -f(x)\)，从而
\[
f(x+4) = f\big(2+(x+2)\big) = -f(x+2) = -[-f(x)] = f(x)
\]
所以 \(f(x)\) 是以 \(4\) 为周期的周期函数。

已知 \(f(5.5)=2\)，利用周期性：
\[
f(5.5) = f(5.5-4) = f(1.5) = 2
\]

在①式中取 \(x=1.5\)：
\[
f(1.5) = -f(2-1.5) = -f(0.5)
\]
所以 \(f(0.5) = -f(1.5) = -2\)。

计算目标 \(g(-0.5)\)：
\[
g(-0.5) = (-0.5-1)f(-0.5) = -1.5 \cdot f(-0.5)
\]
由偶函数性质 \(f(-0.5)=f(0.5) = -2\)，得
\[
g(-0.5) = -1.5 \times (-2) = 3
\]

## 答案
D
## 总结
本题综合考查抽象函数的对称性、奇偶性与周期性。关键步骤是由 \(g(x+1)\) 的偶函数性质得到 \(f(x)\) 的中心对称关系 \(f(x)=-f(2-x)\)，联合 \(f(x)\) 为偶函数推出周期性，从而沟通已知与所求。
