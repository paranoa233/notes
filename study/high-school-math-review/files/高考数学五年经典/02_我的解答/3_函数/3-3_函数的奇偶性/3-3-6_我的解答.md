# 3-3-6 我的解答

对应题目：[[01_题目/3_函数/3-3_函数的奇偶性/3-3-6.md|3-3-6]]

## 完整题目

6.(2023 江苏南通等八市二调)已知函数 \( f\left( x\right) \) 的定义域为 \( \mathbf{R}, y = f\left( x\right)  + {\mathrm{e}}^{x} \) 是偶函数, \( y = \; f\left( x\right)  - 3{\mathrm{e}}^{x} \) 是奇函数,则 \( f\left( x\right) \) 的最小值为 ( )

A. e B. \( 2\sqrt{2} \) C. \( 2\sqrt{3} \) D. 2e

## 题目勘误
无明显勘误。
## 思路分析
本题已知两个分别具有偶函数与奇函数性质的组合函数，通过奇偶性的定义列出关于 \(f(x)\) 与 \(f(-x)\) 的方程组，解出 \(f(x)\) 的解析式。再结合基本不等式（或导数）求出 \(f(x)\) 在 \(\mathbf{R}\) 上的最小值。
## 详细解答
由题意：
1. \(y = f(x) + e^x\) 为偶函数，根据偶函数定义 \(g(-x) = g(x)\)，得
\[
f(-x) + e^{-x} = f(x) + e^x \quad \text{①}
\]
2. \(y = f(x) - 3e^x\) 为奇函数，根据奇函数定义 \(h(-x) = -h(x)\)，得
\[
f(-x) - 3e^{-x} = -\bigl[f(x) - 3e^x\bigr] = -f(x) + 3e^x \quad \text{②}
\]

联立 ①、② 解 \(f(x)\)。
① 可化为：\(f(x) - f(-x) = e^{-x} - e^x\)
② 可化为：\(f(x) + f(-x) = 3e^x + 3e^{-x}\)

两式相加得：
\[
2f(x) = (e^{-x} - e^x) + (3e^x + 3e^{-x}) = 2e^x + 4e^{-x}
\]
所以
\[
f(x) = e^x + 2e^{-x}
\]

由于 \(e^x > 0\)，\(e^{-x} > 0\)，对 \(f(x)\) 使用基本不等式：
\[
f(x) = e^x + 2e^{-x} \ge 2\sqrt{e^x \cdot 2e^{-x}} = 2\sqrt{2}
\]
当且仅当 \(e^x = 2e^{-x}\)，即 \(e^{2x} = 2\)，\(x = \frac{1}{2}\ln 2\) 时等号成立。
因此 \(f(x)\) 的最小值为 \(2\sqrt{2}\)。
## 答案
B
## 总结
本题核心是利用函数奇偶性“赋值”得到方程组解出 \(f(x)\)，再借助基本不等式求最值。注意奇偶性定义中对定义域对称的要求，本题定义域为 \(\mathbf{R}\) 自然满足。求最值时需验证等号成立的条件。
