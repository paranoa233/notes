# 3-3-14 我的解答

对应题目：[[01_题目/3_函数/3-3_函数的奇偶性/3-3-14.md|3-3-14]]

## 完整题目

1.(2023 全国乙卷理，4)已知 \( f\left( x\right)  = \frac{x{\mathrm{e}}^{x}}{{\mathrm{e}}^{ax} - 1} \) 为偶函数,则 \( a = \) ( )

A. -2 B. -1 C. 1 D. 2

## 题目勘误
无明显勘误。
## 思路分析
函数为偶函数的充要条件是对定义域内任意 \(x\) 都有 \(f(-x)=f(x)\)。本题中分母含 \(e^{ax}-1\)，定义域为 \(\{x\mid x\neq 0\}\)，关于原点对称。可先由偶函数定义列式，化简后得到关于 \(a\) 的恒等式,通过取特殊值（如 \(x=1\)）并利用函数 \(g(t)=e^t+e^{-t}\) 的单调性求出 \(a\) 的值，最后代入检验分母是否恒为零。
## 详细解答
函数 \(f(x)=\dfrac{x e^{x}}{e^{ax}-1}\) 的定义域需满足 \(e^{ax}-1\neq 0\)，即 \(ax\neq 0\)，因此 \(x\neq 0\)。定义域关于原点对称。

由 \(f(x)\) 为偶函数，对任意 \(x\neq 0\) 均有 \(f(-x)=f(x)\)，即
\[
\frac{-x e^{-x}}{e^{-ax}-1} = \frac{x e^{x}}{e^{ax}-1}.
\]
因为 \(x\neq 0\)，两边同除以 \(x\) 得
\[
-\frac{e^{-x}}{e^{-ax}-1} = \frac{e^{x}}{e^{ax}-1}.
\]
交叉相乘：
\[
-e^{-x}(e^{ax}-1) = e^{x}(e^{-ax}-1),
\]
\[
-e^{-x+ax} + e^{-x} = e^{x-ax} - e^{x}.
\]
整理得
\[
e^{(a-1)x} + e^{-(a-1)x} = e^{x} + e^{-x}.
\]
该等式对任意 \(x\neq 0\) 恒成立。

取 \(x=1\) 得
\[
e^{a-1} + e^{-(a-1)} = e + e^{-1}.
\]
令 \(g(t)=e^t+e^{-t}\)，易知 \(g(t)\) 是偶函数且在 \([0,+\infty)\) 上单调递增。由 \(g(a-1)=g(1)\) 得
\[
|a-1| = 1,
\]
解得 \(a=0\) 或 \(a=2\)。

若 \(a=0\)，则分母 \(e^{0}-1\equiv 0\)，函数无意义，舍去。
若 \(a=2\)，则 \(f(x)=\dfrac{x e^x}{e^{2x}-1}\)，定义域为 \(\{x\neq 0\}\)，且验证可知满足偶函数条件。

因此 \(a=2\)。

## 答案
D

## 总结
1. 利用函数奇偶性求参数时，先保证定义域关于原点对称，再用等式 \(f(-x)=f(x)\)（偶）或 \(f(-x)=-f(x)\)（奇）建立方程。
2. 若恒等式对所有 \(x\) 成立，可代入特殊值简化计算，但需检验所得参数是否使原函数有意义。
3. 本题利用 \(e^t+e^{-t}\) 的偶性与单调性，将方程转化为绝对值相等，是快速求解的典型技巧。
