# 3-12-9 我的解答

对应题目：[[01_题目/3_函数/3-12_函数的图象及应用/3-12-9.md|3-12-9]]

## 完整题目

9. (多选题,2023 重庆一诊) 函数 \( f\left( x\right)  = \; \left\{  \begin{array}{l} \left| {x - 1}\right| , x \leq  2, \\  5 - x,\;x > 2, \end{array}\right. \) 且 \( f\left( a\right)  = f\left( b\right)  = f\left( c\right) (a < b < \) c),则 ( )

A. \( f\left( x\right) \) 的值域为 \( \lbrack 0, + \infty ) \)

B. 不等式 \( f\left( x\right)  \geq  1 \) 的解集为 \( ( - \infty ,0\rbrack \)

C. \( a + b = 2 \)

D. \( a + b + c \in  \lbrack 6,7) \)

## 题目勘误
无明显勘误。

## 思路分析
本题为多选题，考查分段函数的图象与性质，以及函数的零点与方程根的关系。关键在于画出函数 \(f(x)\) 的图象，利用数形结合思想分析。

1. 写出分段函数并分区间讨论其值域，判断选项 A。
2. 分段解不等式 \(f(x) \ge 1\)，注意区间的开闭与并集，判断选项 B。
3. 由 \(f(a)=f(b)=f(c)=k\) 且 \(a<b<c\)，结合图象分析参数 \(k\) 的范围，并找出三个根与 \(k\) 的关系，进而判断选项 C 和 D。

## 详细解答
函数
\[
f(x) = 
\begin{cases}
|x-1|, & x \le 2, \\
5-x, & x > 2.
\end{cases}
\]

**1. 分析函数的图象与性质**

- 当 \(x \le 2\) 时，\(f(x)=|x-1|\)。
  - 在 \(x \le 1\) 时，\(f(x)=1-x\)，是斜率为 \(-1\) 的射线，值域为 \([0,+\infty)\)。
  - 在 \(1 < x \le 2\) 时，\(f(x)=x-1\)，是斜率为 \(1\) 的线段，值域为 \((0, 1]\)。
  - 该段整体值域为 \([0,+\infty)\)。
- 当 \(x > 2\) 时，\(f(x)=5-x\)，是斜率为 \(-1\) 的射线（不含左端点 \(x=2\)），值域为 \((-\infty, 3)\)。

**A. 判断值域**

合并两段的值域：\([0,+\infty) \cup (-\infty, 3) = \mathbb{R}\)。
所以 \(f(x)\) 的值域为 \(\mathbb{R}\)，选项 A 错误。

**B. 判断不等式 \(f(x) \ge 1\) 的解集**

- 当 \(x \le 2\) 时：\(|x-1| \ge 1 \Rightarrow x-1 \ge 1 \ \text{或} \ x-1 \le -1 \Rightarrow x \ge 2 \ \text{或} \ x \le 0\)。
  结合 \(x \le 2\)，得 \(x \le 0\) 或 \(x=2\)。
- 当 \(x > 2\) 时：\(5-x \ge 1 \Rightarrow x \le 4\)。
  结合 \(x > 2\)，得 \(2 < x \le 4\)。

综上，不等式的解集为 \((-\infty, 0] \cup [2,4]\)，选项 B 错误。

**C 与 D. 结合函数图象分析多解问题**

令 \(f(a)=f(b)=f(c)=k\)，且 \(a<b<c\)。由图象知，存在三个不同自变量对应同一函数值 \(k\) 时，左边两支 V 形和右边斜线必各至少提供一个根。

- 左支（\(x \le 1\)）：\(f(x)=1-x\)，解得 \(x=1-k\)。
- 中段（\(1 < x \le 2\)）：\(f(x)=x-1\)，解得 \(x=1+k\)。该段最大值为 \(f(2)=1\)，故需 \(k \le 1\)。
- 右支（\(x>2\)）：\(f(x)=5-x\)，解得 \(x=5-k\)。该段值域 \((-\infty, 3)\)，需 \(k<3\)。

要使这三个解不同且满足 \(a<b<c\)，由图象特征，只能是：
\(a=1-k\)（左支），\(b=1+k\)（中段），\(c=5-k\)（右支）。
且有 \(1-k < 1+k < 5-k\)。解 \(1+k < 5-k\) 得 \(k<2\)；\(1-k < 1+k\) 等价于 \(k>0\)。
同时，为保证解不重合且恰好三个不同解，需要 \(k \in (0,1]\)（当 \(k=0\) 时左支与中段重合于 \(x=1\)；当 \(k>1\) 时中段无解）。

**判断选项 C**
\(a+b = (1-k)+(1+k)=2\)，与 \(k\) 无关，正确。

**判断选项 D**
\(a+b+c = 2 + c = 2 + (5-k) = 7-k\)。
因为 \(k \in (0,1]\)，所以 \(a+b+c \in [6,7)\)，正确。

## 答案
CD

## 总结
1. 处理分段函数问题，分段讨论是最基本的方法，注意定义域的分界点。
2. 含绝对值的式子通常化为分段形式处理，利用其几何意义（如对称性）可以快速得到关系（如本题 \(a+b=2\)）。
3. 对于“多值对应”问题，数形结合、分析公共取值 \(k\) 的范围是关键，切勿忽略区间开闭对解的个数的影响。
