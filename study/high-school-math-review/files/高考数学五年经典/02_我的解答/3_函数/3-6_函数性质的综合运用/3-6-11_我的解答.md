# 3-6-11 我的解答

对应题目：[[01_题目/3_函数/3-6_函数性质的综合运用/3-6-11.md|3-6-11]]

## 完整题目

2.(2020 新高考卷 I ，8)若定义在 \( \mathbf{R} \) 上的奇函数 \( f\left( x\right) \) 在 \( \left( {-\infty ,0}\right) \) 上单调递减，且 \( f\left( 2\right)  = 0 \) ，则满足 \( {xf}\left( {x - 1}\right)  \geq  0 \) 的 \( x \) 的取值范围是

A. \( \left\lbrack  {-1,1}\right\rbrack   \cup  \lbrack 3, + \infty ) \)

B. \( \left\lbrack  {-3, - 1}\right\rbrack   \cup  \left\lbrack  {0,1}\right\rbrack \)

C. \( \left\lbrack  {-1,0}\right\rbrack   \cup  \lbrack 1, + \infty ) \)

D. \( \left\lbrack  {-1,0}\right\rbrack   \cup  \left\lbrack  {1,3}\right\rbrack \)

## 题目勘误
无明显勘误。

## 思路分析
由奇函数的对称性和已知单调区间，可推出 \( f(x) \) 在 \((0,+\infty)\) 上的单调性及零点分布。进而将不等式 \( x f(x-1) \geq 0 \) 转化为 \( x \) 与 \( f(x-1) \) 同号（或至少一个为零）的问题，通过分类讨论并结合 \( f(x-1) \) 的符号区间求解。

## 详细解答
因为 \( f(x) \) 是定义在 \( \mathbf{R} \) 上的奇函数，所以 \( f(0)=0 \)，且 \( f(-x) = -f(x) \)。  
又 \( f(2)=0 \)，由奇函数性质得 \( f(-2) = -f(2) = 0 \)。

已知 \( f(x) \) 在 \( (-\infty,0) \) 上单调递减，由奇函数在对称区间上的单调性相同，可知 \( f(x) \) 在 \( (0,+\infty) \) 上也单调递减。

结合单调性与零点可得 \( f(x) \) 的符号分布：
- 当 \( x < -2 \) 时，\( f(x) > f(-2) = 0 \)；
- 当 \( -2 < x < 0 \) 时，\( f(x) < 0 \)；
- 当 \( 0 < x < 2 \) 时，\( f(x) > f(2) = 0 \)；
- 当 \( x > 2 \) 时，\( f(x) < 0 \)。

因此，
\[
f(t) \geq 0 \iff t \in (-\infty, -2] \cup [0, 2],
\]
\[
f(t) \leq 0 \iff t \in [-2, 0] \cup [2, +\infty).
\]

令 \( t = x-1 \)，则
\[
f(x-1) \geq 0 \iff x-1 \in (-\infty, -2] \cup [0, 2] \iff x \in (-\infty, -1] \cup [1, 3],
\]
\[
f(x-1) \leq 0 \iff x-1 \in [-2, 0] \cup [2, +\infty) \iff x \in [-1, 1] \cup [3, +\infty).
\]

现在解不等式 \( x f(x-1) \geq 0 \)：

**情形 1：** \( x > 0 \) 且 \( f(x-1) \geq 0 \)。  
此时 \( x \in (0, +\infty) \cap \big( (-\infty, -1] \cup [1, 3] \big) = [1, 3] \)。

**情形 2：** \( x < 0 \) 且 \( f(x-1) \leq 0 \)。  
此时 \( x \in (-\infty, 0) \cap \big( [-1, 1] \cup [3, +\infty) \big) = [-1, 0) \)。

**情形 3：** \( x = 0 \) 时，\( 0 \cdot f(-1) = 0 \)，满足不等式。

取并集得原不等式的解集为 \( [-1,0] \cup [1,3] \)。

## 答案
D. \([-1,0] \cup [1,3]\)

## 总结
本题主要考查奇函数的单调性对称特征以及利用函数性质解不等式。关键步骤为：根据奇函数与单调性确定零点及符号区间；通过“同号得正”将原不等式拆分为两个情形讨论；注意区间端点处乘积为零的情况。
