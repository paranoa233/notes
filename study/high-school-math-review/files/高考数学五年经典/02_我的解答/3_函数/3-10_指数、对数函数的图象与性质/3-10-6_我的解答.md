# 3-10-6 我的解答

对应题目：[[01_题目/3_函数/3-10_指数、对数函数的图象与性质/3-10-6.md|3-10-6]]

## 完整题目

6.(2023 广东新高考开学调研)若正实数 \( a, b \) 满足 \( a > b \) ，且 \( \ln a \cdot  \ln b > 0 \) ，则下列不等式一定成立的是 ( )

A. \( {\log }_{a}\frac{1}{b} > 0 \) B. \( a - b > \frac{1}{b} - \frac{1}{a} \)

C. \( {3}^{{ab} + 1} < {3}^{a + b} \) D. \( {a}^{b - 1} < {b}^{a - 1} \)

## 题目勘误
无明显勘误。

## 思路分析
由条件 \(\ln a \cdot \ln b > 0\) 且 \(a > b > 0\) 可得 \(\ln a\) 与 \(\ln b\) 同号。结合 \(a > b\) 知，必有 \(a > b > 1\) 或 \(0 < b < a < 1\)。选项涉及对数、指数不等式，可分类讨论，并借助作差、函数单调性等方法判断是否一定成立。

## 详细解答
**第一步：翻译条件。**
正实数 \(a, b\) 满足 \(a > b\) 且 \(\ln a \cdot \ln b > 0\)。
因为乘积大于零，所以 \(\ln a\) 与 \(\ln b\) 同号。
结合 \(a > b > 0\)，存在且只存在两种情形：
1. \(a > b > 1\)，此时 \(\ln a > \ln b > 0\)；
2. \(0 < b < a < 1\)，此时 \(\ln b < \ln a < 0\)（注意绝对值大小，因为是负数）。

**第二步：逐项分析。**

**A.** \(\log_a \frac{1}{b} > 0\)。
化简得 \(-\log_a b > 0\)，即 \(\log_a b < 0\)。
- 若 \(a > b > 1\)，则 \(y=\log_a x\) 是增函数，由 \(b>1\) 得 \(\log_a b > \log_a 1 = 0\)，与 \(\log_a b < 0\) 矛盾。
- 若 \(0 < b < a < 1\)，则底数 \(a \in (0,1)\)，\(y=\log_a x\) 是减函数，由 \(b < a\) 得 \(\log_a b > \log_a a = 1 > 0\)，也与 \(\log_a b < 0\) 矛盾。
故 A 不成立。

**B.** \(a - b > \frac{1}{b} - \frac{1}{a}\)。
移项得 \(a + \frac{1}{a} > b + \frac{1}{b}\)。构造函数 \(f(x) = x + \frac{1}{x}\)（\(x>0\)）。
\(f'(x) = 1 - \frac{1}{x^2}\)，在 \((0,1)\) 上 \(f'(x)<0\)，\(f(x)\) 单调递减；在 \((1,+\infty)\) 上 \(f'(x)>0\)，\(f(x)\) 单调递增。
- 若 \(a > b > 1\)，因为 \(f(x)\) 在 \((1,+\infty)\) 递增，所以 \(f(a) > f(b)\)，不等式成立。
- 若 \(0 < b < a < 1\)，因为 \(f(x)\) 在 \((0,1)\) 递减，所以 \(f(a) < f(b)\)，不等式不成立。
因此 B 不一定成立。

**C.** \(3^{ab+1} < 3^{a+b}\)。
底数 \(3>1\)，指数函数递增，原式等价于 \(ab+1 < a+b\)，即 \(ab - a - b + 1 < 0\)，因式分解为 \((a-1)(b-1) < 0\)。
- 若 \(a > b > 1\)，则 \(a-1>0\)，\(b-1>0\)，乘积 \((a-1)(b-1) > 0\)，不等式不成立。
- 若 \(0 < b < a < 1\)，则 \(a-1<0\)，\(b-1<0\)，乘积仍为正，不等式也不成立。
故 C 一定不成立。

**D.** \(a^{b-1} < b^{a-1}\)。
两边取自然对数，等价于 \((b-1)\ln a < (a-1)\ln b\)。
构造函数 \(g(x) = \frac{\ln x}{x-1}\)，\(x>0\) 且 \(x \neq 1\)。
求导得 \(g'(x) = \frac{ \frac{1}{x}(x-1) - \ln x }{(x-1)^2} = \frac{1 - \frac{1}{x} - \ln x}{(x-1)^2}\)。
令 \(\varphi(x) = 1 - \frac{1}{x} - \ln x\)，则 \(\varphi'(x) = \frac{1}{x^2} - \frac{1}{x} = \frac{1-x}{x^2}\)。
易知当 \(0<x<1\) 时 \(\varphi'(x)>0\)，\(\varphi(x)\) 递增；当 \(x>1\) 时 \(\varphi'(x)<0\)，\(\varphi(x)\) 递减。
而 \(\varphi(1)=0\)，故对所有 \(x>0\) 且 \(x \neq 1\)，恒有 \(\varphi(x) < 0\)。
从而 \(g'(x) < 0\)，\(g(x)\) 在 \((0,1)\) 和 \((1,+\infty)\) 上均严格单调递减。
由条件知 \(a > b\)，且两者同在 \((0,1)\) 或同在 \((1,+\infty)\)，因此 \(g(a) < g(b)\)，即
\[
\frac{\ln a}{a-1} < \frac{\ln b}{b-1}.
\]
此时需注意分母的符号：
- 若 \(a > b > 1\)，则 \(a-1>0\)，\(b-1>0\)，两边同乘正数 \((a-1)(b-1)\) 得 \((b-1)\ln a < (a-1)\ln b\)。
- 若 \(0 < b < a < 1\)，则 \(a-1<0\)，\(b-1<0\)，乘积 \((a-1)(b-1)>0\)，同样同乘正数不等号方向不变，仍得 \((b-1)\ln a < (a-1)\ln b\)。
该式恰好等价于 \(a^{b-1} < b^{a-1}\)，故 D 一定成立。

## 答案
D

## 总结
本题综合考查了对数的符号法则、指数函数与对数函数的单调性，以及利用导数构造函数比较大小的方法。关键是通过 \(\ln a \cdot \ln b > 0\) 得出 \(a,b\) 同侧分布，再结合单调性进行论证。构造辅助函数 \(g(x)=\frac{\ln x}{x-1}\) 并判断其单调性是解决 D 选项的常用技巧。
