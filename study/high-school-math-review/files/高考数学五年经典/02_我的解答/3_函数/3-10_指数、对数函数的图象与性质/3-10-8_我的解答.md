# 3-10-8 我的解答

对应题目：[[01_题目/3_函数/3-10_指数、对数函数的图象与性质/3-10-8.md|3-10-8]]

## 完整题目

8. (多选题，2021 山东日照一模)已知 \( {x}_{1} \downarrow \; {\log }_{3}{x}_{1} = 0,{x}_{2} + {\log }_{2}{x}_{2} = 0 \) ,则 ( )

A. \( 0 < {x}_{2} < {x}_{1} < 1 \)

B. \( 0 < {x}_{1} < {x}_{2} < 1 \)

C. \( {x}_{2}\lg {x}_{1} - {x}_{1}\lg {x}_{2} < 0 \)

D. \( {x}_{2}\lg {x}_{1} - {x}_{1}\lg {x}_{2} > 0 \)

## 题目勘误
题干中“\( {x}_{1} \downarrow \; {\log }_{3}{x}_{1} = 0 \)”存在 OCR 识别错误，符号“\(\downarrow\)”无意义。根据选项逻辑及第二个方程的结构，此处应为加号。
- 原文：\( {x}_{1} \downarrow \; {\log }_{3}{x}_{1} = 0 \)
- 建议更正：\( x_1 + \log_3 x_1 = 0 \)
- 理由：参照后一个方程 \( x_2 + \log_2 x_2 = 0 \) 的形式，以及选项均是比较 \(x_1\) 与 \(x_2\) 的大小和相应代数式的符号，可推断“\(\downarrow\)”为“\(+\)”的误识别。

## 思路分析
将两个方程分别变形为 \(\log_3 x_1 = -x_1\) 和 \(\log_2 x_2 = -x_2\)，则 \(x_1, x_2\) 分别为相应对数函数与直线 \(y=-x\) 交点的横坐标。利用指数式转化或构造辅助函数，结合函数单调性即可比较 \(x_1\) 与 \(x_2\) 的大小，进而判断各选项。

## 详细解答
**第一步：确定 \(x_1, x_2\) 的范围。**
由 \(x_1 + \log_3 x_1 = 0\) 得 \(\log_3 x_1 = -x_1\)。
因为对数真数大于零，故 \(x_1 > 0\)。
若 \(x_1 \ge 1\)，则 \(\log_3 x_1 \ge 0\)，但 \(-x_1 \le -1 < 0\)，等式不可能成立，故 \(0 < x_1 < 1\)。
同理，由 \(x_2 + \log_2 x_2 = 0\) 得 \(0 < x_2 < 1\)。

**第二步：比较 \(x_1\) 与 \(x_2\) 的大小（判断 A、B 选项）。**
将原方程化为指数形式：
\[
x_1 = 3^{-x_1},\qquad x_2 = 2^{-x_2}.
\]
构造函数 \(p(x) = 2^{-x} - x\) 与 \(q(x) = 3^{-x} - x\)。由上面两式知：
\[
p(x_2) = 0,\quad q(x_1) = 0.
\]
当 \(x > 0\) 时，由于指数函数 \(a^x\) 随底数增大而增大（底数 \(>1\)），有 \(3^{-x} < 2^{-x}\)，从而
\[
q(x) < p(x) \quad (x > 0).
\]
因此 \(q(x_1) = 0\)，代入得 \(p(x_1) > q(x_1) = 0 = p(x_2)\)，即
\[
p(x_1) > p(x_2).
\]
又因为 \(p(x) = 2^{-x} - x\) 在 \(\mathbb{R}\) 上单调递减（\(2^{-x}\) 递减，\(-x\) 递减），故由 \(p(x_1) > p(x_2)\) 可得
\[
x_1 < x_2.
\]
结合第一步的范围有 \(0 < x_1 < x_2 < 1\)，因此 **B 正确，A 错误**。

**第三步：判断 \(x_2\lg x_1 - x_1\lg x_2\) 的符号（判断 C、D 选项）。**
由于 \(x_1, x_2 \in (0,1)\)，有 \(x_1 x_2 > 0\)，可将目标式等价变形：
\[
x_2\lg x_1 - x_1\lg x_2 = x_1x_2\left( \frac{\lg x_1}{x_1} - \frac{\lg x_2}{x_2} \right).
\]
令 \(f(x) = \frac{\lg x}{x} = \frac{1}{\ln 10} \cdot \frac{\ln x}{x}\)。在 \((0,1)\) 上，\(\frac{\ln x}{x}\) 的导数为 \(\frac{1 - \ln x}{x^2} > 0\)，故 \(f(x)\) 在 \((0,1)\) 上单调递增。
由第二步知 \(x_1 < x_2\)，所以
\[
f(x_1) < f(x_2) \quad \Rightarrow \quad \frac{\lg x_1}{x_1} < \frac{\lg x_2}{x_2}.
\]
于是
\[
\frac{\lg x_1}{x_1} - \frac{\lg x_2}{x_2} < 0,
\]
进而
\[
x_2\lg x_1 - x_1\lg x_2 < 0.
\]
因此 **C 正确，D 错误**。

## 答案
BC

## 总结
本题主要考查指数、对数函数的图象与性质，及利用函数单调性比较大小、判断不等式。解题关键：
1. 将方程 \(x + \log_a x = 0\) 转化为 \(\log_a x = -x\) 或 \(x = a^{-x}\)，从而将根视为图象交点。
2. 不同底数幂的大小比较，需结合单调性构造辅助函数。
3. 处理形如 \(x_2\lg x_1 - x_1\lg x_2\) 的对称式时，常用“同除以 \(x_1x_2\)”的技巧转化为单一变量函数 \(\frac{\lg x}{x}\)，利用单调性判断符号。
