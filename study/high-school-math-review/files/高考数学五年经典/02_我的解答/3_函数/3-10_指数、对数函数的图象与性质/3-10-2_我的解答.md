# 3-10-2 我的解答

对应题目：[[01_题目/3_函数/3-10_指数、对数函数的图象与性质/3-10-2.md|3-10-2]]

## 完整题目

2.(2021 福建名校联盟联考)若 \( {\left( \frac{1}{3}\right) }^{a} < {\left( \frac{1}{3}\right) }^{b} < \) 1 ,则下列各式一定成立的是 ( )

A. \( \ln \left( {a - b}\right)  > 0 \)

B. \( {2}^{b - a} > 1 \)

C. \( - \frac{1}{a} >  - \frac{1}{b} \)

D. \( {\log }_{c}a > {\log }_{c}b\left( {c > 0\text{ 且 }c \neq  1}\right) \)

## 题目勘误
无明显勘误。
## 思路分析
本题主要考查指数函数与对数函数的单调性。首先根据指数不等式 \(\left(\frac{1}{3}\right)^a < \left(\frac{1}{3}\right)^b < 1\) 明确 \(a\) 与 \(b\) 的大小关系及范围，然后利用对数函数、幂函数及指数函数的性质逐一分析各选项是否一定成立。
## 详细解答
由已知不等式 \(\left(\frac{1}{3}\right)^a < \left(\frac{1}{3}\right)^b < 1\)，
因为底数 \(\frac{1}{3} \in (0,1)\)，指数函数 \(y = \left(\frac{1}{3}\right)^x\) 在 \(\mathbb{R}\) 上单调递减，
所以由 \(\left(\frac{1}{3}\right)^a < \left(\frac{1}{3}\right)^b\) 可得：\(a > b\)。
由 \(\left(\frac{1}{3}\right)^b < 1 = \left(\frac{1}{3}\right)^0\) 可得：\(b > 0\)。
综上可知：\(a > b > 0\)。

下面逐项判断：
**A选项**：\(\ln(a - b) > 0\) 等价于 \(a - b > 1\)。
已知仅有 \(a > b > 0\)，并不能保证 \(a - b > 1\)。 
反例：取 \(a = 0.6, \; b = 0.5\)，满足 \(a > b > 0\)，但 \(a - b = 0.1\)，\(\ln 0.1 < 0\)，故A不一定成立。

**B选项**：\(2^{b-a} > 1 = 2^0\)。
因为底数 \(2 > 1\)，指数函数 \(y = 2^x\) 单调递增，所以该不等式等价于 \(b - a > 0\)，即 \(b > a\)。
这与已知的 \(a > b\) 矛盾，故B一定不成立。

**C选项**：\(-\frac{1}{a} > -\frac{1}{b}\)。
将两边同乘 \(-1\)（不等号方向改变），等价于 \(\frac{1}{a} < \frac{1}{b}\)。
因为 \(a > b > 0\)，由反比例函数 \(y = \frac{1}{x}\) 在 \((0, +\infty)\) 上单调递减，可知 \(\frac{1}{a} < \frac{1}{b}\) 一定成立。
也可用差法：\(\frac{1}{a} - \frac{1}{b} = \frac{b-a}{ab}\)，由 \(a > b > 0\) 知 \(b-a < 0\)，\(ab > 0\)，故差为负，同样得证。因此C一定成立。

**D选项**：\(\log_c a > \log_c b \; (c > 0 \text{ 且 } c \neq 1)\)。
对数函数 \(y = \log_c x\) 的单调性由底数 \(c\) 决定：
- 若 \(c > 1\)，函数单调递增，由 \(a > b\) 可推出 \(\log_c a > \log_c b\)；
- 若 \(0 < c < 1\)，函数单调递减，由 \(a > b\) 推出的是 \(\log_c a < \log_c b\)。
由于底数 \(c\) 不确定，故D不一定成立。

## 答案
C
## 总结
1. 处理底数在 \((0,1)\) 内的指数不等式时，注意单调递减会导致不等号反向。
2. 对于“一定成立”类问题，只要存在反例即可排除选项；正例则需严格证明。
3. 涉及对数比较大小时，务必区分底数大于1和介于0到1的两种情况，当底数不确定时结论通常也不确定。
