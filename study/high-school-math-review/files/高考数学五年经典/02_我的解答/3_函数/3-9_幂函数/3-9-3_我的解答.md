# 3-9-3 我的解答

对应题目：[[01_题目/3_函数/3-9_幂函数/3-9-3.md|3-9-3]]

## 完整题目

3.(2020 山东肥城模拟)已知函数 \( f\left( x\right)  = \; \left\{  \begin{array}{l} {3}^{-x} + 1, x \leq  0, \\  {x}^{a} + 2, x > 0. \end{array}\right. \) 若 \( f\left\lbrack  {f\left( {-1}\right) }\right\rbrack   = {18} \) ,则实数 \( a \) 的值是 ( )

A. 4 B. 1 C. 2 D. 3

## 题目勘误
无明显勘误。

## 思路分析
先由分段函数计算内层 \( f(-1) \)，注意自变量 \(-1 \leq 0\) 应代入第一段。再根据所得函数值是否大于 0 选择外层的对应解析式，代入 \( f(f(-1)) = 18 \) 建立关于 \( a \) 的方程，解之即可。

## 详细解答
计算内层函数值：
\[
f(-1) = 3^{-(-1)} + 1 = 3^1 + 1 = 4.
\]
因此
\[
f\bigl(f(-1)\bigr) = f(4).
\]
因为 \( 4 > 0 \)，代入第二段解析式：
\[
f(4) = 4^a + 2.
\]
由已知条件 \( f\bigl(f(-1)\bigr) = 18 \) 得
\[
4^a + 2 = 18,
\]
\[
4^a = 16 = 4^2,
\]
所以
\[
a = 2.
\]

## 答案
C

## 总结
处理分段函数的复合求值时，必须逐层判断自变量所在的区间，选择正确的对应法则。本题先由 \( x = -1 \) 得内层值 4，再由 \( x = 4 \) 代入幂函数部分求得参数 \( a \)，核心是避免混淆不同区间的解析式。
