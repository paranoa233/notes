# 3-9-2 我的解答

对应题目：[[01_题目/3_函数/3-9_幂函数/3-9-2.md|3-9-2]]

## 完整题目

2.(2021 湖北武昌调研)已知点 \( \left( {m,8}\right) \) 在幂函数 \( f\left( x\right)  = \left( {m - 1}\right) {x}^{n} \) 的图象上. 设 \( a = f\left( \frac{m}{n}\right) , b = \)

 \( f\left( {\ln \pi }\right) , c = f\left( n\right) \) ,则 ( )

A. \( b < a < c \) B. \( a < b < c \)

C. \( b < c < a \) D. \( a < c < b \)

## 题目勘误
无明显勘误。
## 思路分析
由幂函数定义可知系数为 \(1\)，结合点 \((m,8)\) 在图象上可求出 \(m,n\)，进而得到 \(f(x)\) 的解析式。分别算出 \(a,b,c\) 的值，借助中间量比较立方数的大小，或直接估算数值即可排序。
## 详细解答
因为 \(f(x)=(m-1)x^n\) 是幂函数，所以 \(m-1=1\)，得 \(m=2\)。  
此时 \(f(x)=x^n\)。  
又点 \((m,8)\) 即 \((2,8)\) 在图象上，代入得  
\[
2^n = 8 \implies n=3.
\]
故 \(f(x)=x^3\)。

计算：
\[
a = f\!\left(\frac{m}{n}\right) = f\!\left(\frac{2}{3}\right) = \left(\frac{2}{3}\right)^3 = \frac{8}{27},
\]
\[
b = f(\ln\pi) = (\ln\pi)^3,
\]
\[
c = f(n) = 3^3 = 27.
\]

比较大小：
\[
a = \frac{8}{27} \approx 0.296,
\]
因为 \(e \approx 2.718 < \pi < e^2\)，所以 \(1 < \ln\pi < 2\)，具体 \(\ln\pi \approx 1.1447\)，于是
\[
b \approx 1.1447^3 \approx 1.5,
\]
\[
c = 27.
\]
显然 \(a < b < c\)。
## 答案
B
## 总结
幂函数 \(f(x)=kx^\alpha\) 中要求系数 \(k=1\)，这是解题的突破口；求出解析式后，利用函数的单调性（\(x^3\) 在 \(\mathbb{R}\) 上递增）亦可直接比较 \(a,b,c\) 对应的自变量大小：\(\frac{2}{3} < \ln\pi < 3\)，所以 \(a<b<c\)，避免具体计算仍然可靠。
