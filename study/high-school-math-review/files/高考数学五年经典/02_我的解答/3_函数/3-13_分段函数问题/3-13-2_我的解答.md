# 3-13-2 我的解答

对应题目：[[01_题目/3_函数/3-13_分段函数问题/3-13-2.md|3-13-2]]

## 完整题目

2.(2021 江苏苏锡常镇情况调研(一)若函数 \( f\left( x\right)  = \left\{  \begin{array}{ll} {x}^{3} - \frac{16}{x}, & x \neq  0, \\  0, & x = 0, \end{array}\right. \) 则满足 \( {xf}\left( {x - 1}\right)  \geq  0 \) 的 \( x \) 的取值范围是 ( )

A. \( \left\lbrack  {-1,1}\right\rbrack   \cup  \lbrack 3, + \infty ) \)

B. \( ( - \infty , - 1\rbrack  \cup  \left\lbrack  {0,1}\right\rbrack   \cup  \lbrack 3, + \infty ) \)

C. \( \left\lbrack  {-1,0}\right\rbrack   \cup  \lbrack 1, + \infty ) \)

D. \( ( - \infty , - 3\rbrack  \cup  \left\lbrack  {-1,0}\right\rbrack   \cup  \lbrack 1, + \infty ) \)

## 题目勘误
无明显勘误。

## 思路分析
首先分析函数 \( f(x) \) 的解析式，将其变形为易于判断符号的形式，求出 \( f(x) > 0 \)、\( f(x) = 0 \) 和 \( f(x) < 0 \) 的解集。  
然后将不等式 \( x f(x-1) \ge 0 \) 转化为关于 \( x \) 的分类讨论：\( x \ge 0 \) 且 \( f(x-1) \ge 0 \)，或 \( x \le 0 \) 且 \( f(x-1) \le 0 \)。  
利用换元或直接代入 \( x-1 \) 的范围，分别解出每个情形下的 \( x \) 的取值范围，最后取并集即可。

## 详细解答

1. **分析 \( f(x) \) 的符号**  
   当 \( x \neq 0 \) 时，
   \[
   f(x) = x^3 - \frac{16}{x} = \frac{x^4 - 16}{x} = \frac{(x^2 - 4)(x^2 + 4)}{x}.
   \]
   由于 \( x^2 + 4 > 0 \) 恒成立，符号仅由 \( (x^2 - 4) \) 和分母 \( x \) 决定。  
   令 \( f(x) = 0 \)：
   \[
   x^2 - 4 = 0 \Rightarrow x = \pm 2 \quad (x \neq 0)，
   \]
   且已知 \( f(0)=0 \)。  
   所以 \( f(x) \) 的零点为 \( x = -2, 0, 2 \)。

   利用数轴划分区间并判断符号：
   - \( f(x) \ge 0 \)：解不等式 \( \frac{x^4 - 16}{x} \ge 0 \)（考虑 \( x=0 \) 处有定义且函数值为 \( 0 \)）。  
     当 \( x > 0 \) 时，分子 \( \ge 0 \) 得 \( x^4 \ge 16 \Rightarrow x \ge 2 \)；  
     当 \( x < 0 \) 时，分子 \( \le 0 \)（分母为负）得 \( x^4 \le 16 \Rightarrow -2 \le x < 0 \)；  
     且 \( x=0 \) 满足 \( f(0)=0 \)。  
     故 \( f(x) \ge 0 \) 的解集为 \( [-2, 0] \cup [2, +\infty) \)。

   - \( f(x) \le 0 \)：由于已得 \( f(x) \ge 0 \) 的解集，且 \( f(x) \) 只在 \( x \neq 0 \) 时分式有定义，补集并考虑零点得：
     \[
     f(x) \le 0 \text{ 的解集为 } (-\infty, -2] \cup [0, 2].
     \]
     （检验：区间 \( (-\infty, -2) \) 和 \( (0, 2) \) 上 \( f(x) < 0 \)，在 \( x=-2, 0, 2 \) 处 \( f(x)=0 \)）。

2. **解不等式 \( x f(x-1) \ge 0 \)**
   不等式等价于两种情况：
   \[
   \begin{cases}
   x \ge 0, \\
   f(x-1) \ge 0,
   \end{cases}
   \quad \text{或} \quad
   \begin{cases}
   x \le 0, \\
   f(x-1) \le 0.
   \end{cases}
   \]

   **情况一：\( x \ge 0 \) 且 \( f(x-1) \ge 0 \)**
   由 \( f(x-1) \ge 0 \) 得
   \[
   x-1 \in [-2, 0] \cup [2, +\infty).
   \]
   即
   \[
   -2 \le x-1 \le 0 \quad \text{或} \quad x-1 \ge 2,
   \]
   \[
   -1 \le x \le 1 \quad \text{或} \quad x \ge 3.
   \]
   与 \( x \ge 0 \) 取交集：
   \[
   x \in [0, 1] \cup [3, +\infty).
   \]

   **情况二：\( x \le 0 \) 且 \( f(x-1) \le 0 \)**
   由 \( f(x-1) \le 0 \) 得
   \[
   x-1 \in (-\infty, -2] \cup [0, 2].
   \]
   即
   \[
   x-1 \le -2 \quad \text{或} \quad 0 \le x-1 \le 2,
   \]
   \[
   x \le -1 \quad \text{或} \quad 1 \le x \le 3.
   \]
   与 \( x \le 0 \) 取交集：
   \[
   x \in (-\infty, -1] \quad (\text{后一段 } [1,3] \text{ 与 } x \le 0 \text{ 无交集}).
   \]

3. **求并集**
   将两种情况的结果合并：
   \[
   x \in (-\infty, -1] \cup [0, 1] \cup [3, +\infty).
   \]

## 答案
B

## 总结
分段函数与不等式结合时，关键是先理清函数本身的零点与符号变化规律，再将原不等式等价转化为若干个联立的不等式组，分别求解后取并集。注意边界值的归属，必要时可代入检验。本题还涉及分式函数的变形，属基础题型，分类讨论的思想贯穿始终。
