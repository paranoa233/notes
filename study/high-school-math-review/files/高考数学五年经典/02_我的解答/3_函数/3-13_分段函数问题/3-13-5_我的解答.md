# 3-13-5 我的解答

对应题目：[[01_题目/3_函数/3-13_分段函数问题/3-13-5.md|3-13-5]]

## 完整题目

5.(2024 湖南新高考十八校联盟 3 月联考)设方

程 \( {2}^{x} \cdot  \left| {{\log }_{2}x}\right|  = 1 \) 的两根为 \( {x}_{1},{x}_{2}\left( {{x}_{1} < {x}_{2}}\right) \) ,则 ( )

A. \( 0 < {x}_{1} < 1,{x}_{2} > 2 \) B. \( {x}_{1} > \frac{1}{{x}_{2}} \)

C. \( 0 < {x}_{1}{x}_{2} < 1 \) D. \( {x}_{1} + {x}_{2} > 3 \)

## 题目勘误
无明显勘误。

## 思路分析
将方程 \(2^x \cdot |\log_2 x| = 1\) 转化为 \(|\log_2 x| = 2^{-x}\)，则原方程的根即为函数 \(y = |\log_2 x|\) 与 \(y = (\frac{1}{2})^x\) 图像交点的横坐标。通过分析图像可得 \(0 < x_1 < 1 < x_2 < 2\)，排除 A 与 D。对于选项 B、C，需比较 \(x_1\) 与 \(\frac{1}{x_2}\) 的大小。构造函数 \(h(x) = 2^x (-\log_2 x)\) (\(0 < x < 1\))，利用其单调性及 \(x_2\) 满足的方程比较 \(h(x_1)\) 与 \(h(\frac{1}{x_2})\)，从而判定 \(x_1 x_2\) 与 \(1\) 的关系。

## 详细解答

由方程 \(2^x \cdot |\log_2 x| = 1\) 可得
\[
|\log_2 x| = 2^{-x}.
\]
设 \(f(x) = 2^x |\log_2 x|\)，其定义域为 \((0,+\infty)\)。

**第一步：确定根的大致范围**

当 \(0 < x < 1\) 时，\(|\log_2 x| = -\log_2 x\)，\(f(x) = -2^x \log_2 x\)。
- \(\lim\limits_{x \to 0^+} f(x) = +\infty\)，\(f(1) = 0\)，且 \(f\) 连续，故在 \((0,1)\) 内存在唯一的 \(x_1\) 使 \(f(x_1)=1\)，且满足 \(0 < x_1 < 1\)。

当 \(x > 1\) 时，\(|\log_2 x| = \log_2 x\)，\(f(x) = 2^x \log_2 x\)。
- 当 \(x = 1\) 时，\(f(1) = 0\)；当 \(x = 2\) 时，\(f(2) = 4 \cdot 1 = 4 > 1\)；\(f(x)\) 在 \([1,+\infty)\) 上严格单调递增，故存在唯一的 \(x_2\) 使 \(f(x_2)=1\)，且 \(1 < x_2 < 2\)。

因此 \(0 < x_1 < 1 < x_2 < 2\)。于是选项 A 中 \(x_2 > 2\) 错误。

**第二步：分析选项 B、C**

选项 B 等价于 \(x_1 > \frac{1}{x_2}\)，即 \(x_1 x_2 > 1\)；选项 C 为 \(0 < x_1 x_2 < 1\)。需判断 \(x_1\) 与 \(\frac{1}{x_2}\) 的大小。

由两根满足的方程：
\[
2^{x_1}(-\log_2 x_1) = 1, \quad 2^{x_2}\log_2 x_2 = 1.
\]
构造辅助函数 \(h(x) = 2^x (-\log_2 x) = -2^x \cdot \frac{\ln x}{\ln 2}\)，\(x \in (0,1)\)。
对 \(h(x)\) 求导：
\[
h'(x) = 2^x \ln 2 \cdot (-\log_2 x) + 2^x \cdot \left(-\frac{1}{x \ln 2}\right)
= 2^x \left( -\ln x - \frac{1}{x \ln 2} \right).
\]
当 \(0 < x < 1\) 时，\(-\ln x > 0\)，令 \(q(x) = -x \ln x - \frac{1}{\ln 2}\)。易知 \(-x \ln x\) 在 \((0,1)\) 上的最大值为 \(\frac{1}{e} \approx 0.367\)，而 \(\frac{1}{\ln 2} \approx 1.44\)，故 \(q(x) \le 0.367 - 1.44 < 0\)，从而 \(h'(x) < 0\)。所以 \(h(x)\) 在 \((0,1)\) 上严格单调递减。

计算 \(h\!\left(\frac{1}{x_2}\right)\)：
\[
h\!\left(\frac{1}{x_2}\right) = 2^{1/x_2} \left( -\log_2 \frac{1}{x_2} \right) = 2^{1/x_2} \log_2 x_2.
\]
利用 \(x_2\) 满足的方程 \(\log_2 x_2 = 2^{-x_2}\)，代入得
\[
h\!\left(\frac{1}{x_2}\right) = 2^{1/x_2} \cdot 2^{-x_2} = 2^{\frac{1}{x_2} - x_2}.
\]
因为 \(x_2 > 1\)，所以 \(\frac{1}{x_2} - x_2 < 0\)，故 \(2^{\frac{1}{x_2} - x_2} < 2^0 = 1\)，即有
\[
h\!\left(\frac{1}{x_2}\right) < 1 = h(x_1).
\]
由 \(h(x)\) 在 \((0,1)\) 上单调递减，可知 \(\frac{1}{x_2} > x_1\)，即 \(x_1 x_2 < 1\)。又因两根均为正，故 \(0 < x_1 x_2 < 1\)，C 正确，B 错误。

**第三步：验证选项 D**

由上述已知 \(x_1 < \frac{1}{x_2}\)，且 \(1 < x_2 < 2\)，则
\[
x_1 + x_2 < \frac{1}{x_2} + x_2.
\]
考虑函数 \(g(t) = t + \frac{1}{t}\) 在 \((1,2)\) 上单调递增（因为 \(g'(t) = 1 - \frac{1}{t^2} > 0\)），其最大值在 \(t \to 2\) 时趋近于 \(2.5\)，因此 \(x_2 + \frac{1}{x_2} < 2.5 < 3\)。故 \(x_1 + x_2 < 3\)，D 错误。

## 答案
C

## 总结
本题主要考查指数、对数函数的图像与性质，以及利用构造函数和单调性比较数值大小的方法。解题关键在于将方程根的问题转化为函数图像交点问题以确定根的范围，再通过单调性建立不等式比较两根乘积与 1 的关系，进而鉴别选项。
