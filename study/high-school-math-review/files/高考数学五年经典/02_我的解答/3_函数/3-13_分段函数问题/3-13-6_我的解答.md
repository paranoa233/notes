# 3-13-6 我的解答

对应题目：[[01_题目/3_函数/3-13_分段函数问题/3-13-6.md|3-13-6]]

## 完整题目

6. (多选题, 2023 河北石家庄期末) 已知函数 \( f\left( x\right)  = \left\{  \begin{array}{l} {2}^{x - 1} + {2}^{1 - x} - 2, x \geq  0, \\  {x}^{2} + {2x} + \frac{1}{2}, x < 0. \end{array}\right. \) 若 \( f\left( {x}_{1}\right)  = f\left( {x}_{2}\right) \; = f\left( {x}_{3}\right)  = f\left( {x}_{4}\right) \) ，且 \( {x}_{1} < {x}_{2} < {x}_{3} < {x}_{4} \) ，则()

A. \( {x}_{1} + {x}_{2} =  - 2 \)

B. \( 0 < {x}_{3} < 1 < {x}_{4} < 2 \)

C. \( {x}_{3}{x}_{4} \geq  1 \)

D. \( {x}_{1} + {x}_{2} + {x}_{3} + {x}_{4} = 0 \)

## 题目勘误
无明显勘误。

## 思路分析
函数为分段函数，要求满足 \(f(x_1)=f(x_2)=f(x_3)=f(x_4)\) 且 \(x_1<x_2<x_3<x_4\) 的四个不同实数解。  
本质是研究水平直线 \(y=k\) 与 \(f(x)\) 图象有四个不同交点的条件，并利用二次函数对称性、指数型函数的对称性得出各根之间的关系，进而判断选项。

## 详细解答
首先分析函数 \(f(x)\) 在各段的单调性与值域。

当 \(x<0\) 时，\(f(x)=x^2+2x+\frac12=(x+1)^2-\frac12\)。  
图象为开口向上的抛物线，对称轴为 \(x=-1\)，在 \((-\infty,-1)\) 上单调递减，在 \((-1,0)\) 上单调递增，最小值 \(f(-1)=-\frac12\)。

当 \(x\ge0\) 时，\(f(x)=2^{x-1}+2^{1-x}-2\)。令 \(t=x-1\ge-1\)，则 \(f(x)=2^t+2^{-t}-2\)，该函数关于 \(t=0\)（即 \(x=1\)）对称，且 \(2^t+2^{-t}\ge2\)，故 \(f(x)\ge0\)，最小值 \(f(1)=0\)。在 \([0,1]\) 上单调递减，值域为 \([0,\frac12]\)；在 \([1,+\infty)\) 上单调递增，值域为 \([0,+\infty)\)。

要使得方程 \(f(x)=k\) 有四个不同的实数根 \(x_1<x_2<x_3<x_4\)，直线 \(y=k\) 必须与左支（\(x<0\)）有两个交点，与右支（\(x\ge0\)）有两个交点。

对左支：\(x^2+2x+\frac12=k\) 有两个不等实根，且均小于 \(0\)。  
判别式 \(\Delta=4-4\left(\frac12-k\right)=2+4k>0\Rightarrow k>-\frac12\)。  
两根为 \(x=-1\pm\sqrt{\frac12+k}\)，大根 \(-1+\sqrt{\frac12+k}<0\Rightarrow k<\frac12\)。  
故左支有两根的条件为 \(-\frac12<k<\frac12\)。

对右支：当 \(k=0\) 时只有一根 \(x=1\)；当 \(k=\frac12\) 时只有一根 \(x=0\)。要使右支有两个不同正根，必须 \(0<k<\frac12\)。

综上，当且仅当 \(0<k<\frac12\) 时，\(f(x)=k\) 恰有四个根，不妨设  
\(x_1<x_2<0<x_3<1<x_4\)，其中 \(x_1,x_2\) 来自左支抛物线，\(x_3,x_4\) 来自右支指数型曲线。

由韦达定理，\(x_1+x_2=-2\)，选项 A 正确。

由 \(t=x-1\) 的对称性，方程 \(2^{x-1}+2^{1-x}-2=k\) 的根满足 \((x_3-1)=-(x_4-1)\)，即 \(x_3+x_4=2\)，进而 \(x_1+x_2+x_3+x_4=0\)，选项 D 正确。

因为 \(x_3>0\)，故 \(x_4=2-x_3<2\)，结合 \(0<x_3<1<x_4\) 知 \(x_4<2\)，选项 B 正确。

对 \(x_3x_4\)，由基本不等式得 \(x_3x_4\le\left(\frac{x_3+x_4}{2}\right)^2=1\)，等号当且仅当 \(x_3=x_4=1\) 时成立，但此时 \(x_3\neq x_4\)，故 \(x_3x_4<1\)，选项 C 错误。

## 答案
正确选项为 A、B、D。

## 总结
处理分段函数的等高线问题时，要分段研究单调性与值域，找到能产生四个不同交点的公共范围。利用二次函数与偶函数型指数函数的对称性，可直接得出根的和与根的积的范围，快速判断选项。
