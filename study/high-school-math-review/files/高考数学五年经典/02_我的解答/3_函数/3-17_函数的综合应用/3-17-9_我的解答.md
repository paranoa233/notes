# 3-17-9 我的解答

对应题目：[[01_题目/3_函数/3-17_函数的综合应用/3-17-9.md|3-17-9]]

## 完整题目

9.(2024 九省联考)以 max \( M \) 表示数集 \( M \) 中最大的数. 设 \( 0 < a < b < c < 1 \) ,已知 \( b \geq  {2a} \) 或 \( a + \; b \leq  1 \) ,则 \( \max \{ b - a, c - b,1 - c\} \) 的最小值为___。

## 题目勘误
无明显勘误。
## 思路分析
将 \(b-a\)、\(c-b\)、\(1-c\) 设为三个正数 \(x,y,z\)，用它们表示 \(a\)，并将已知条件转化为关于 \(x,y,z\) 的不等式。问题变为在 \(x,y,z>0\) 且 \(x+y+z<1\) 的条件下，求 \(\max\{x,y,z\}\) 的最小值。通过分析不等式对 \(x,y,z\) 的约束，找到最小的可能上界，并构造实例验证等号可以成立。
## 详细解答
设  
\[
x = b-a,\quad y = c-b,\quad z = 1-c,
\]  
则由 \(0<a<b<c<1\) 知 \(x>0,\;y>0,\;z>0\)，且  
\[
a + x + y + z = 1 \quad\Longrightarrow\quad a = 1 - x - y - z > 0,
\]  
从而有  
\[
x + y + z < 1. \tag{1}
\]

将条件用 \(x,y,z\) 改写：  
- \(b \ge 2a \iff a+x \ge 2a \iff x \ge a = 1-x-y-z\)  
  \[
  \iff 2x + y + z \ge 1. \tag{2}
  \]
- \(a + b \le 1 \iff a + (a+x) \le 1 \iff 2a + x \le 1\)  
  代入 \(a = 1-x-y-z\)，得  
  \[
  2(1-x-y-z) + x \le 1 \iff x + 2y + 2z \ge 1. \tag{3}
  \]

因此，原问题等价于：在条件 (1) 以及 “(2) 或 (3)” 成立的前提下，求 \(\max\{x,y,z\}\) 的最小值。

令 \(t = \max\{x,y,z\}\)，则 \(x \le t,\;y \le t,\;z \le t\)。  
若 (2) 成立，则  
\[
1 \le 2x + y + z \le 2t + t + t = 4t \;\Longrightarrow\; t \ge \frac{1}{4}.
\]  
若 (3) 成立，则  
\[
1 \le x + 2y + 2z \le t + 2t + 2t = 5t \;\Longrightarrow\; t \ge \frac{1}{5}.
\]  
因为条件是“或”，所以只要 \(t \ge \dfrac{1}{5}\)，就有可能通过满足 (3) 来实现。

当 \(t = \dfrac{1}{5}\) 时，取 \(x = y = z = \dfrac{1}{5}\)，检验：
- 满足 \(x,y,z > 0\)，且 \(x+y+z = \dfrac{3}{5} < 1\)，即 (1) 成立；
- \(x + 2y + 2z = \dfrac{1}{5} + \dfrac{2}{5} + \dfrac{2}{5} = 1\)，满足 (3)；
- 对应的 \(a = 1 - \dfrac{3}{5} = \dfrac{2}{5},\; b = a+x = \dfrac{3}{5},\; c = b+y = \dfrac{4}{5}\)，满足 \(0<a<b<c<1\) 以及 \(a+b=1\)（即原条件 \(a+b\le 1\)）。

因此，\(\max\{b-a,\,c-b,\,1-c\} = \max\left\{\dfrac{1}{5},\,\dfrac{1}{5},\,\dfrac{1}{5}\right\} = \dfrac{1}{5}\) 是可以取到的。  
若 \(t < \dfrac{1}{5}\)，则 \(5t < 1\) 且 \(4t < \dfrac{4}{5} < 1\)，此时 (2)(3) 均不成立，无解。

综上，所求最小值为 \(\dfrac{1}{5}\)。
## 答案
\(\dfrac{1}{5}\)
## 总结
本题通过引入“间隔”变量将原条件转化为线性不等式组，从而将最大值的最小值问题转化为求不等式组的相容下界。处理此类问题时，常通过设出相等的变量来达到下界，并验证边界条件的可满足性。
