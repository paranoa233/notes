# 3-17-7 我的解答

对应题目：[[01_题目/3_函数/3-17_函数的综合应用/3-17-7.md|3-17-7]]

## 完整题目

7.(2024 山东济南一模)已知集合 \( A = \{ u\left( x\right) \; u\left( x\right)  = a{x}^{2} - \left( {a + b}\right) x + b, a, b \in  \mathbf{R}\} \) ,函数 \( f\left( x\right) \; = {x}^{2} - 1 \) . 若函数 \( g\left( x\right) \) 满足:对任意 \( u\left( x\right)  \in  A \) ,存在 \( \lambda ,\mu  \in  \mathbf{R} \) ,使得 \( u\left( x\right)  = {\lambda f}\left( x\right)  + {\mu g}\left( x\right) \) ,则 \( g\left( x\right) \) 的解析式可以是___. (写出一个满足条件的函数解析式即可)

## 题目勘误
无明显勘误。

## 思路分析
题目给出了一个二次函数集合 \(A\) 和一个确定的函数 \(f(x)=x^2-1\)。要求写出一个函数 \(g(x)\)，使得 \(A\) 中的任意一个函数都能被 \(f(x)\) 和 \(g(x)\) 线性表示。这相当于在函数构成的线性空间中，\(A\) 是二维子空间，且 \(f(x)\) 已经位于 \(A\) 内。要使 \(\{f, g\}\) 能线性生成整个 \(A\)，只需 \(g(x)\) 也属于 \(A\) 且与 \(f(x)\) 线性无关即可。因此，先把集合 \(A\) 整理成标准形式，找出 \(f(x)\) 对应的参数，再任取一个与之线性无关的参数对，即可得到一个满足条件的 \(g(x)\)。

## 详细解答
集合 \(A\) 中的函数形如  
\[
u(x) = a x^{2} - (a+b)x + b,\quad a,b\in\mathbf{R}.
\]  
重新按 \(a,b\) 整理：
\[
u(x) = a(x^{2} - x) + b(1 - x).
\]  
由此可知 \(A\) 是由基函数 \(x^{2}-x\) 与 \(1-x\) 张成的二维线性空间。

已知 \(f(x)=x^{2}-1\)。检查它是否属于 \(A\)：令  
\[
a(x^{2} - x) + b(1 - x) = x^{2} - 1,
\]  
即 \(a x^{2} - a x - b x + b = x^{2} - 1\)。比较系数得  
\[
\begin{cases}
a = 1,\\
-a - b = 0,\\
b = -1,
\end{cases}
\]  
解得 \(a=1,\,b=-1\)。故 \(f(x)\in A\)，且对应的系数对为 \((1,-1)\)。

要使命题“对任意 \(u\in A\)，存在 \(\lambda,\mu\in\mathbf{R}\) 使得 \(u = \lambda f + \mu g\)”成立，等价于 \(\{f, g\}\) 能够线性生成 \(A\)。因为 \(A\) 是二维空间且 \(f\in A\)，只需选取 \(g\in A\) 且 \(g\) 与 \(f\) 线性无关（即对应的参数对与 \((1,-1)\) 不成比例）。

例如，取 \(a=0,\,b=1\)，得  
\[
g(x) = 0\cdot x^{2} - (0+1)x + 1 = 1 - x.
\]  
此时 \(g\in A\)，且参数 \((0,1)\) 与 \((1,-1)\) 线性无关，故 \(f\) 与 \(g\) 线性无关，它们的张成空间就是整个 \(A\)。因此 \(g(x)=1-x\) 满足题意。

（注：答案不唯一，也可取 \(g(x)=x^{2}-x\) 等。）

## 答案
\(g(x)=1-x\)（写出一个满足条件的即可）

## 总结
本题表面是函数解析式问题，实质是函数集合的线性结构问题。将函数视为向量，将集合视为线性空间，利用“线性生成”与“线性无关”的概念，可以迅速把条件转化为对基的选择。解题时先将已知集合写成基的形式，判断已知函数是否在其中，再补充一个线性无关的元素即可。
