# 3-17-2 我的解答

对应题目：[[01_题目/3_函数/3-17_函数的综合应用/3-17-2.md|3-17-2]]

## 完整题目

2.(2023 安徽合肥二检)设 \( A, B, C, D \) 是曲线 \( y = \; {x}^{3} - {mx} \) 上的四个动点. 若以这四个动点为顶点的正方形有且只有一个,则实数 \( m \) 的值为 ( )

A. 4 B. \( 2\sqrt{3} \) C. 3 D. \( 2\sqrt{2} \)

## 题目勘误
无明显勘误。

## 思路分析
函数 \(y = x^3 - mx\) 为奇函数，图像关于原点中心对称。若其上的四点能构成正方形，利用对称性可设正方形的中心为原点，四个顶点可设为 \(A(a,b), B(-b,a), C(-a,-b), D(b,-a)\)。  
由点在曲线上可得关于 \(a, b, m\) 的方程组，消去 \(m\) 后得到 \(a,b\) 满足的齐次关系。引入参数 \(t = \frac{a}{b}\)，并进一步代换 \(u = t - \frac{1}{t}\)，可将问题转化为关于 \(u\) 的二次方程。正方形的存在性等价于该二次方程有正根，而“有且只有一个”则要求正根唯一，即方程有重根，由此求出 \(m\)。

## 详细解答
因为 \(f(x)=x^3-mx\) 是奇函数，若其上的四个动点能构成正方形，则正方形的中心必为坐标原点（否则与奇函数的中心对称性矛盾，此处不作严格证明，由选择题特性直接使用该结论）。  
设正方形的四个顶点为  
\[
A(a,b),\quad B(-b,a),\quad C(-a,-b),\quad D(b,-a),
\]  
且它们均在曲线 \(y = x^3 - mx\) 上，故有  
\[
\begin{cases}
b = a^3 - ma, \quad (1)\\
a = (-b)^3 - m(-b) = -b^3 + mb. \quad (2)
\end{cases}
\]
由 (1) 得 \(m = \dfrac{a^3 - b}{a}\ (a \neq 0)\)，由 (2) 得 \(m = \dfrac{a + b^3}{b}\ (b \neq 0)\)。  
联立消去 \(m\)：
\[
\frac{a^3 - b}{a} = \frac{a + b^3}{b} \implies b(a^3 - b) = a(a + b^3),
\]
整理得
\[
a^3b - b^2 = a^2 + ab^3 \implies ab(a^2 - b^2) - (a^2 + b^2) = 0.
\]
令 \(t = \dfrac{a}{b}\ (b \neq 0)\)，代入上式得
\[
b^4 t(t^2 - 1) - b^2(t^2 + 1) = 0.
\]
因为 \(b \neq 0\)，两边同除以 \(b^2\)：
\[
b^2 t(t^2 - 1) = t^2 + 1 \implies b^2 = \frac{t^2 + 1}{t(t^2 - 1)}.
\]
由 \(b^2 > 0\) 以及 \(t^2 + 1 > 0\)，可知必须有
\[
t(t^2 - 1) > 0 \implies t \in (-1, 0) \cup (1, +\infty).
\]
从 (1) 又可得
\[
m = a^2 - \frac{b}{a} = t^2 b^2 - \frac{1}{t} = t^2 \cdot \frac{t^2 + 1}{t(t^2 - 1)} - \frac{1}{t} = \frac{t^4 + 1}{t(t^2 - 1)}.
\]
作代换 \(u = t - \dfrac{1}{t}\)，则 \(u \neq 0\)，且
\[
t^2 + \frac{1}{t^2} = u^2 + 2,\quad t(t^2 - 1) = t^2 u,
\]
于是
\[
m = \frac{t^4 + 1}{t(t^2 - 1)} = \frac{t^2 + \frac{1}{t^2}}{t - \frac{1}{t}} = \frac{u^2 + 2}{u} = u + \frac{2}{u}.
\]
整理得到关于 \(u\) 的二次方程：
\[
u^2 - mu + 2 = 0.
\]
正方形存在的条件是上述方程有实数根，且由于 \(b^2 > 0 \iff t(t^2 - 1) > 0 \iff u > 0\)，故需要正根。  
方程 \(u^2 - mu + 2 = 0\) 有两正根的充要条件是
\[
\Delta = m^2 - 8 \ge 0,\quad \text{且根的和 } m > 0,\ \text{积 } 2 > 0.
\]
当 \(m > 2\sqrt{2}\) 时，\(\Delta > 0\)，方程有两个不同的正根 \(u_1, u_2\)，每个 \(u\) 通过 \(t - \frac{1}{t} = u\) 解出互为倒数的两个 \(t\)，它们对应同一个正方形（只因选择起点不同），因此此时曲线上存在两个不同的正方形。  
当 \(m = 2\sqrt{2}\) 时，\(\Delta = 0\)，方程有唯一正根 \(u = \frac{m}{2} = \sqrt{2}\)，对应唯一的正方形。  
当 \(m < 2\sqrt{2}\) 时，\(\Delta < 0\)，无正根，不存在这样的正方形。  
题目要求“以这四个动点为顶点的正方形有且只有一个”，故必须且只需 \(m = 2\sqrt{2}\)。

## 答案
选 D. \(2\sqrt{2}\)

## 总结
本题利用奇函数的对称性将正方形的中心锁定在原点，通过设出顶点坐标、引入参数比例 \(t\) 及代换 \(u = t - \frac{1}{t}\)，把存在性转化为关于 \(u\) 的二次方程的根的问题。由正方形“有且只有一个”得到方程应有唯一正根，即判别式为零，从而解出 \(m\)。注意推理中要确保 \(u>0\) 以保证参数 \(b^2>0\)，排除无效解。
