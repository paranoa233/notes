# 3-8-7 我的解答

对应题目：[[01_题目/3_函数/3-8_二次函数在给定区间上的最值问题/3-8-7.md|3-8-7]]

## 完整题目

7.(2024 重庆渝北阶段练习)已知三个实数 \( a, b \) ， \( c \) ,其中 \( c > 0, b \leq  {2a} + {3c} \) 且 \( {bc} = {a}^{2} \) ,则 \( \frac{a - {2c}}{b} \) 的最大值为___.

## 题目勘误
无明显勘误。
## 思路分析
题目给出关于三个实数 \(a,b,c\) 的两个条件：\(c>0,\ b\leq 2a+3c\) 以及 \(bc=a^2\)，求 \(\dfrac{a-2c}{b}\) 的最大值。可以从等式 \(bc=a^2\) 出发消去 \(b\)，将所求表达式用 \(a\) 和 \(c\) 表示。再将不等式约束转化为 \(a\) 与 \(c\) 的齐次不等式，通过换元（设 \(t=a/c\)）将问题转化为关于 \(t\) 的一元函数在给定区间上的最值问题，利用二次函数性质求解最大值。
## 详细解答
由 \(bc = a^{2}\) 且 \(c > 0\)，若 \(a = 0\)，则 \(b = 0\)，原式分母为零无意义，故 \(a \neq 0\)，从而 \(b = \dfrac{a^{2}}{c} > 0\)。  
所求式为  
\[
\frac{a-2c}{b} = \frac{a-2c}{a^{2}/c} = \frac{c(a-2c)}{a^{2}}.
\]

将不等式 \(b \leq 2a + 3c\) 两边同乘正数 \(c\)，得  
\[
a^{2} \leq 2ac + 3c^{2} \quad\Longrightarrow\quad a^{2} - 2ac - 3c^{2} \leq 0.
\]  
因式分解得  
\[
(a - 3c)(a + c) \leq 0.
\]  
因为 \(c > 0\)，故 \(a + c > 0\)，从而 \(a - 3c \leq 0\)，即  
\[
a \leq 3c .
\]

设 \(t = \dfrac{a}{c}\)，由于 \(c > 0\) 且 \(a \neq 0\)，有 \(t \in (-\infty,0) \cup (0,3]\)。  
原式化为  
\[
E = \frac{t-2}{t^{2}}.
\]

再令 \(u = \dfrac{1}{t}\ (t \neq 0)\)，则当 \(t \in (0,3]\) 时，\(u \in [\frac{1}{3}, +\infty)\)；当 \(t < 0\) 时，\(u < 0\)。于是  
\[
E = u - 2u^{2} = -2u^{2} + u.
\]

函数 \(f(u) = -2u^{2} + u\) 是开口向下的二次函数，对称轴为 \(u = \dfrac{1}{4}\)。  
- 当 \(u \in [\frac{1}{3}, +\infty)\) 时，区间在对称轴右侧，函数单调递减，最大值在 \(u = \frac{1}{3}\) 处取得：  
\[
f\!\left(\frac{1}{3}\right) = -2 \times \frac{1}{9} + \frac{1}{3} = \frac{1}{9}.
\]
- 当 \(u < 0\) 时，\(f(u) = u(1-2u)\)，由于 \(u < 0\) 且 \(1-2u > 0\)，\(f(u) < 0\)，最大值趋近于 \(0\)（无法取到），小于 \(\frac{1}{9}\)。

综上，所求表达式的最大值为 \(\dfrac{1}{9}\)，此时取 \(t = 3\)，即 \(a = 3c\)，\(b = 9c\)，且满足 \(b = 2a + 3c\)。
## 答案
\(\dfrac{1}{9}\)
## 总结
本题通过等式消元，将分式化为齐次式，再利用不等式得到变量的范围，最后通过换元转化为二次函数在区间上的最值问题。关键步骤是发现 \(b = a^2/c\) 并正确得出 \(a \leq 3c\) 的约束。
