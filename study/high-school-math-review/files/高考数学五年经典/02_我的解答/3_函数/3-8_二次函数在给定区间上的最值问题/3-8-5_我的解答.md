# 3-8-5 我的解答

## 完整题目

5. (多选题，2024 山东菏泽阶段练习)已知二次函数 \( y = a{x}^{2} + {bx} + c\left( {a \neq  0, a, b, c\text{ 为常数 }}\right) \) 的对称轴为 \( x = 1 \) ，其图象如图所示，则下列结论正确的有 ( )

![[05_附件/images/29_615_832_210_199_0.jpg]]

A. \( \frac{a}{\left| a\right| } + \frac{b}{\left| b\right| } + \frac{c}{\left| c\right| } = 3 \)

B. \( \frac{12}{4 - {3b}} - b \) 的最小值为 \( \frac{8}{3} \)

C. 当 \( x \in  \left\lbrack  {a,1 - a}\right\rbrack \) 时，函数的最大值为 \( {a}^{3} - 2{a}^{2} + c \)

D. 关于 \( x \) 的不等式 \( a{x}^{4} + b{x}^{2} < a{\left( {x}^{2} - 1\right) }^{2} + \; b\left( {{x}^{2} - 1}\right) \) 的解集为 \( \left\{  {x\left| {\; - \frac{\sqrt{6}}{2} < x < \frac{\sqrt{6}}{2}}\right. }\right\} \)

## 思路分析
本题为多选题，核心条件为二次函数的对称轴 \(x=1\)，即 \(-\frac{b}{2a}=1\Rightarrow b=-2a\)。选项涉及符号判断、最值分析、给定区间上的最大值、不等式解集等问题。我们可先从代数上推导各选项成立所需的 \(a,b,c\) 条件，再结合图象能获取的符号、范围等信息进行判定。对于无法仅由对称轴确定的部分，需依赖图象信息进行补充说明。

## 详细解答

**由对称轴得基本关系**
由对称轴 \(x = -\frac{b}{2a} = 1\)，得
\[
b = -2a .
\]

**选项 A 分析**
\[
\frac{a}{|a|} + \frac{b}{|b|} + \frac{c}{|c|} = 3
\]
每一项的值为 \(1\) 或 \(-1\)，总和为 \(3\) 意味着 \(a,b,c\) 三者同正。
若 \(a > 0\)，则 \(b = -2a < 0\)，矛盾；
若 \(a < 0\)，则 \(b > 0\)，此时 \(\frac{a}{|a|} = -1\)，总和最大为 \(1\)（当 \(c>0\) 时），不可能等于 \(3\)。
因此无论开口方向如何，**选项 A 一定错误**。

**选项 D 分析**
化简所给不等式：
\[
a x^4 + b x^2 < a (x^2-1)^2 + b (x^2-1)
\]
右边展开得 \(a x^4 - 2a x^2 + a + b x^2 - b\)，移项整理：
\[
2a x^2 + b - a < 0 .
\]
代入 \(b = -2a\)：
\[
2a x^2 - 3a < 0 \quad\Longrightarrow\quad a(2x^2-3) < 0 .
\]
选项 D 给出的解集为 \(-\frac{\sqrt6}{2} < x < \frac{\sqrt6}{2}\)，即 \(x^2 < \frac{3}{2}\)，故 \(2x^2-3 < 0\)。
要使不等式 \(a(2x^2-3) < 0\) 在此解集上恒成立，必须有 \(a > 0\)。
此时不等式确与 \(2x^2-3<0\) 等价，解集完全吻合，**选项 D 正确**，并推断出 \(a>0\) 且 \(b=-2a<0\)。

**选项 C 分析**
函数为 \(f(x) = a x^2 + b x + c\)，由 \(a>0\) 知开口向上，对称轴 \(x=1\)。
区间为 \([a,\; 1-a]\)。为保证区间有意义，需 \(a < 1-a \Rightarrow a < \frac12\)。
此时区间右端点 \(1-a < 1\)，故整个区间在对称轴左侧，函数单调递减，
最大值在左端点 \(x=a\) 处取得：
\[
f(a) = a\cdot a^2 + b\cdot a + c = a^3 - 2a^2 + c \quad (\because b=-2a).
\]
因此选项 C 的结论成立的条件是 \(0 < a < \frac12\)。此条件是否满足需结合图象判断（例如图象与 \(x\) 轴交点位置可限制 \(a\) 的范围）。若图象满足该条件，则 C 正确。

**选项 B 分析**
将 \(b = -2a\) 代入表达式：
\[
\frac{12}{4-3b} - b = \frac{12}{4+6a} + 2a = \frac{6}{2+3a} + 2a .
\]
令 \(t = 3a+2 > 2\)，则表达式化为
\[
\frac{6}{t} + \frac{2}{3}(t-2) = \frac{6}{t} + \frac{2}{3}t - \frac{4}{3}.
\]
由基本不等式，
\[
\frac{6}{t} + \frac{2}{3}t \ge 2\sqrt{\frac{6}{t}\cdot\frac{2}{3}t} = 2\sqrt{4} = 4,
\]
当且仅当 \(\frac{6}{t} = \frac{2}{3}t\)，即 \(t=3\;(a=\frac13)\) 时取等，此时最小值为 \(4 - \frac43 = \frac83\)。
因此“\(\frac{12}{4-3b}-b\) 的最小值为 \(\frac83\)”这一表述在 \(a\) 能取到 \(\frac13\) 时成立。由于 \(a,b,c\) 为常数，该表达式本身是一个常数，“最小值为 \(\frac83\)”通常意味着 \(a=\frac13\) 或 \(a\) 属于某范围且表达式值的变化有最小值。需结合图象确认 \(a\) 的具体值或范围。若图象显示 \(a\) 允许取到 \(\frac13\) 且真数部分符合，则 B 可能正确。

## 答案
已确定 **A 错误，D 正确**；选项 **B、C 是否正确依赖于图象中 \(a\) 的具体取值或范围**，需结合原题图片进一步判断。

## 总结
本题以二次函数图象为载体，综合考查了对称轴、系数符号、给定区间上的最值以及不等式求解。解题关键是由对称轴得到 \(b=-2a\)，再利用选项中的等式和不等式反推 \(a\) 的符号及范围。对于含参最值和区间最值问题，需严格验证参数范围是否满足题目隐含条件（如图象特征）。因缺少图片，无法最终确定全部正确选项，解答时注明勘误，实际考试中务必结合图象信息进行验证。
