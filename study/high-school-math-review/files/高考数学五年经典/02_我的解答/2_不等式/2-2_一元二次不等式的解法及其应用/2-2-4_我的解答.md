# 2-2-4 我的解答

对应题目：[[01_题目/2_不等式/2-2_一元二次不等式的解法及其应用/2-2-4.md|2-2-4]]

## 完整题目

4.(2022 江苏苏北四市期末)不等式 \( x - \frac{1}{x} > 0 \) 成立的一个充分条件是 ( )

A. \( x <  - 1 \) B. \( x >  - 1 \)

C. \( - 1 < x < 0 \) D. \( 0 < x < 1 \)

## 题目勘误
无明显勘误。

## 思路分析
本题要求找出不等式 \(x - \frac{1}{x} > 0\) 成立的一个充分条件。首先需要求出该不等式的解集，然后根据“充分条件”的定义，判断哪个选项对应的集合是解集的子集（即由选项可以推出不等式成立）。

## 详细解答
**第一步：求不等式的解集。**
不等式为 \(x - \frac{1}{x} > 0\)，显然 \(x \neq 0\)。
通分得：
\[
\frac{x^2 - 1}{x} > 0 \quad \Rightarrow \quad \frac{(x-1)(x+1)}{x} > 0.
\]
该分式不等式的零点为 \(x = -1\)，\(x = 0\)，\(x = 1\)。用“穿根法”确定符号：
当 \(x > 1\) 时，各因式均为正，分式值为正；
在 \((0, 1)\) 区间，\(x-1 < 0\)，其余为正，分式值为负；
在 \((-1, 0)\) 区间，\(x\) 为负，\(x-1\) 为负，\(x+1\) 为正，负负得正，分式值为正；
当 \(x < -1\) 时，三因式均为负，分式值为负。
所以不等式 \(\frac{(x-1)(x+1)}{x} > 0\) 的解集为：
\[
(-1,\,0) \cup (1,\,+\infty).
\]

**第二步：逐项判断充分性。**
- A. \(x < -1\)：该区间为 \((-\infty, -1)\)，不在解集中，不能推出原不等式成立。不是充分条件。
- B. \(x > -1\)：该区间为 \((-1, +\infty)\)，包含 \((0, 1]\) 等使原不等式不成立的取值范围。不是充分条件。
- C. \(-1 < x < 0\)：该区间为 \((-1, 0)\)，是解集的子集。在此条件下，原不等式必然成立，因此是该不等式成立的一个充分条件。
- D. \(0 < x < 1\)：该区间为 \((0, 1)\)，不在解集中，不能推出原不等式成立。不是充分条件。

综上所述，选项 C 符合题意。

## 答案
C

## 总结
处理此类充分条件问题，一般先将不等式（或方程、条件）的解集求出，再验证选项所表示的集合是否为解集的子集。分式不等式求解时要注意分母不为零，并熟练使用“穿根法”确定符号。
