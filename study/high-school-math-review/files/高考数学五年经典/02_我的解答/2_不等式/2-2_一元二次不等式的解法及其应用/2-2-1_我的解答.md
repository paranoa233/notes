# 2-2-1 我的解答

对应题目：[[01_题目/2_不等式/2-2_一元二次不等式的解法及其应用/2-2-1.md|2-2-1]]

## 完整题目

1.(2023 河北石家庄正定中学模拟预测)已知集合 \( A = \left\{  {x \mid  2{x}^{2} - x - 3 < 0}\right\}  , B = \left\{  {y \mid  y = 2 - {3x} + }\right. \; \left. {x}^{2}\right\} \) ,则 \( A \cap  B = \) ( )

A. \( \left\{  {x\left| {\; - \frac{1}{4} \leq  x < \frac{3}{2}}\right. }\right\} \)

B. \( \left\{  {x\left| {\; - 1 < x <  - \frac{1}{4}}\right. }\right\} \)

C. \( \{ x \mid   - 1 \leq  x < 3\} \)

D. \( \left\{  {x\left| {\; - 3 \leq  x <  - \frac{1}{4}}\right. }\right\} \)

## 题目勘误
无明显勘误。集合 \( B \) 的定义以“\( y = 2 - {3x} + \)”换行接“\( {x}^{2} \)”出现，系排版换行所致，实际为 \( B = \{ y \mid y = 2 - 3x + x^2 \} \)。

## 思路分析
先解一元二次不等式求出集合 \( A \)；再由二次函数求值域得出集合 \( B \)；最后求两实数集合的交集，结合选项做出判断。

## 详细解答
**1. 求集合 \( A \)：**
\[
A = \{ x \mid 2x^2 - x - 3 < 0 \}
\]
因式分解：
\[
2x^2 - x - 3 = (2x-3)(x+1)
\]
不等式 \( (2x-3)(x+1) < 0 \) 的解为两根之间：
\[
-1 < x < \frac{3}{2}
\]
所以 \( A = \left( -1, \frac{3}{2} \right) \)。

**2. 求集合 \( B \)：**
\[
B = \{ y \mid y = 2 - 3x + x^2 \}
\]
其中 \( x \in \mathbb{R} \)，即求二次函数 \( y = x^2 - 3x + 2 \) 的值域。配方：
\[
y = x^2 - 3x + 2 = \left( x - \frac{3}{2} \right)^2 - \frac{9}{4} + 2 = \left( x - \frac{3}{2} \right)^2 - \frac{1}{4}
\]
因为 \( \left( x - \frac{3}{2} \right)^2 \ge 0 \)，所以 \( y \ge -\frac{1}{4} \)。故  
\[
B = \left[ -\frac{1}{4}, +\infty \right)
\]

**3. 求 \( A \cap B \)：**
尽管 \( A \) 用 \( x \) 表示，\( B \) 用 \( y \) 表示，但两者都是实数集，交集为公共实数部分。
\[
A \cap B = \left( -1, \frac{3}{2} \right) \cap \left[ -\frac{1}{4}, +\infty \right) = \left[ -\frac{1}{4}, \frac{3}{2} \right)
\]
即 \( -\dfrac{1}{4} \le x < \dfrac{3}{2} \)，对应选项 A。

## 答案
A

## 总结
处理两个集合的交集时，若集合元素均为实数，即使变量不同也可直接对范围取交。本题关键是将二次不等式与二次函数的值域分别求出，再利用数轴求公共部分。
