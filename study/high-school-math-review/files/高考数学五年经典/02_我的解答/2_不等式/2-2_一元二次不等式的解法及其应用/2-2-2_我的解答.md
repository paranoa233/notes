# 2-2-2 我的解答

对应题目：[[01_题目/2_不等式/2-2_一元二次不等式的解法及其应用/2-2-2.md|2-2-2]]

## 完整题目

2.(2020 山东枣庄部分中学联考)若不等式 \( a{x}^{2} + {bx} + 2 > 0 \) 的解集为 \( \{ x \mid   - 1 < x < 2\} \) ,则不等式 \( 2{x}^{2} + {bx} + a > 0 \) 的解集为 ( )

A. \( \left\{  {x\left| {\;x <  - 1}\right. }\right. \) 或 \( \left. {x > \frac{1}{2}}\right\} \)

B. \( \left\{  {x\left| {\; - 1 < x < \frac{1}{2}}\right. }\right\} \)

C. \( \{ x \mid   - 2 < x < 1\} \)

D. \( \{ x \mid  x <  - 2 \) 或 \( x > 1\} \)

## 题目勘误
无明显勘误。
## 思路分析
由已知不等式的解集可反推出二次项系数 \(a\) 和一次项系数 \(b\)，再利用这两个系数写出第二个不等式并求解。关键在于：
1. 根据“大于0”的解集形式判断抛物线开口方向，确定 \(a\) 的符号。
2. 利用二次方程根与系数的关系（韦达定理）求出 \(a\) 与 \(b\) 的值。
3. 解出新的一元二次不等式，注意二次项系数为正时的求解方法。
## 详细解答
已知不等式 \( a x^{2} + b x + 2 > 0 \) 的解集为 \( \{x \mid -1 < x < 2 \} \)。  
由于解集是开区间且不等式为“\(>0\)”，对应的二次函数 \(y = a x^{2} + b x + 2\) 的图象必须开口向下，即 \(a < 0\)，且与 \(x\) 轴交于 \(x = -1\) 和 \(x = 2\)。  
因此，方程 \(a x^{2} + b x + 2 = 0\) 的两根为 \(x_1 = -1\)，\(x_2 = 2\)。  
由韦达定理可得：
\[
x_1 + x_2 = -\frac{b}{a} = 1, \quad x_1 x_2 = \frac{2}{a} = -2.
\]
由 \(\frac{2}{a} = -2\) 解得：
\[
a = -1.
\]
将 \(a = -1\) 代入 \(-\frac{b}{a} = 1\)：
\[
-\frac{b}{-1} = 1 \implies b = 1.
\]
所以 \(a = -1\)，\(b = 1\)。  
第二个不等式为 \(2x^{2} + b x + a > 0\)，代入得：
\[
2x^{2} + x - 1 > 0.
\]
对应方程 \(2x^{2} + x - 1 = 0\) 的根可由因式分解求得：
\[
(2x - 1)(x + 1) = 0 \implies x_1 = -1,\; x_2 = \frac{1}{2}.
\]
因为二次项系数 \(2 > 0\)，抛物线开口向上，不等式大于 \(0\) 的解集在两根之外：
\[
x < -1 \quad \text{或} \quad x > \frac{1}{2}.
\]
所以解集为 \(\{x \mid x < -1 \text{ 或 } x > \frac{1}{2} \}\)。
## 答案
A
## 总结
解题的关键在于利用已知不等式的解集反求系数。注意：
1. 解集 \((x_1, x_2)\) 与不等式 \(ax^2+bx+c>0\) 对应时，必有 \(a<0\)。
2. 用韦达定理建立等式，快速求出系数。
3. 解新不等式时，先求根，再根据开口方向确定解集的范围（两根之外或两根之间）。
