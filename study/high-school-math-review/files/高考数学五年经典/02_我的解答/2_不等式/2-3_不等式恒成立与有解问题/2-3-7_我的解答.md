# 2-3-7 我的解答

对应题目：[[01_题目/2_不等式/2-3_不等式恒成立与有解问题/2-3-7.md|2-3-7]]

## 完整题目

7.(2021 广东茂名五校联考)若函数 \( f\left( x\right)  = \; \frac{1}{2}\cos {2x} + {3a}\left( {\sin x - \cos x}\right)  + \left( {{4a} - 1}\right) x \) 在 \( \left\lbrack  {-\frac{\pi }{2}}\right. \) , 0]上单调递增,则实数 \( a \) 的取值范围是 ( )

A. \( \left\lbrack  {\frac{1}{7},1}\right\rbrack \)

B. \( \left\lbrack  {-1,\frac{1}{7}}\right\rbrack \)

C. \( \left( {-\infty , - \frac{1}{7}}\right\rbrack   \cup  \lbrack 1, + \infty ) \)

D. \( \lbrack 1, + \infty ) \)

## 题目勘误
无明显勘误。

## 思路分析
函数在闭区间上单调递增可转化为其导数在区间内非负。由于导数表达式中含有三角函数，可尝试利用三角恒等变换将导数转化为关于 \(\sin x + \cos x\) 的二次函数，进而利用二次函数的性质求参数的取值范围。

## 详细解答
函数为：
\[
f(x) = \frac{1}{2}\cos 2x + 3a(\sin x - \cos x) + (4a - 1)x.
\]
求导得：
\[
f'(x) = -\sin 2x + 3a(\cos x + \sin x) + (4a - 1).
\]
令 \(t = \sin x + \cos x\)，则 \(\sin 2x = 2\sin x \cos x = (\sin x + \cos x)^2 - 1 = t^2 - 1\)。
代入 \(f'(x)\) 得：
\[
f'(x) = -(t^2 - 1) + 3a t + (4a - 1) = -t^2 + 3a t + 4a.
\]
由题意，\(f(x)\) 在 \(\left[-\frac{\pi}{2}, 0\right]\) 上单调递增，故在该区间上 \(f'(x) \geq 0\) 恒成立。

当 \(x \in \left[-\frac{\pi}{2}, 0\right]\) 时，有
\[
x + \frac{\pi}{4} \in \left[-\frac{\pi}{4}, \frac{\pi}{4}\right],
\]
所以
\[
t = \sqrt{2}\sin\left(x + \frac{\pi}{4}\right) \in [-1, 1].
\]
于是原条件等价于：当 \(t \in [-1, 1]\) 时，关于 \(t\) 的二次函数
\[
g(t) = -t^2 + 3a t + 4a \geq 0
\]
恒成立。

\(g(t)\) 的二次项系数为 \(-1 < 0\)，其图象是开口向下的抛物线，因此 \(g(t)\) 在闭区间 \([-1, 1]\) 上的最小值必在端点处取得。故 \(g(t) \geq 0\) 恒成立当且仅当
\[
\begin{cases}
g(-1) \geq 0, \\
g(1) \geq 0.
\end{cases}
\]
计算端点值：
\[
g(-1) = -1 - 3a + 4a = a - 1 \geq 0 \quad\Rightarrow\quad a \geq 1;
\]
\[
g(1) = -1 + 3a + 4a = 7a - 1 \geq 0 \quad\Rightarrow\quad a \geq \frac{1}{7}.
\]
取交集得实数 \(a\) 的取值范围为 \([1, +\infty)\)。

## 答案
D

## 总结
处理含参三角函数的单调性问题时，优先求导并尝试将导数表达式中的三角函数通过换元（如 \(t = \sin x + \cos x\)）统一变为代数函数。转化为二次型函数在闭区间上恒非负的问题后，利用二次函数的开口方向与对称轴位置可直接由端点值约束求解参数范围。
