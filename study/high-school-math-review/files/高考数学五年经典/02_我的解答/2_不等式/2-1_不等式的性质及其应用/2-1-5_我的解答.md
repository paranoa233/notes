# 2-1-5 我的解答

对应题目：[[01_题目/2_不等式/2-1_不等式的性质及其应用/2-1-5.md|2-1-5]]

## 完整题目

5. (多选题，2023 辽宁期末联考)已知 \( \ln \left( {-a}\right)  > \; \ln b \) ,则 0

A. \( \frac{1}{a} < \frac{1}{b} \) B. \( {ab} + {b}^{2} < 0 \)

C. \( {a}^{2} < {b}^{2} \) D. \( {2}^{a} < {\left( \frac{1}{2}\right) }^{b} \)

## 题目勘误
原文“已知 \( \ln \left( {-a}\right)  > \; \ln b \) ,则 0”疑似 OCR 错误，可能原题为“已知 \(\ln(-a) > \ln b\)，则（ ）”。“0”字多余，不影响题意，解答按正常选项处理。
## 思路分析
从对数不等式 \(\ln(-a) > \ln b\) 出发，先由真数大于零得到 \(a, b\) 的符号范围，再利用对数函数的单调性去掉对数符号，得出 \(a, b\) 的大小关系。然后逐项将选项转化为等价形式，结合已知条件判断是否成立。
## 详细解答
由 \(\ln(-a) > \ln b\) 可知：
真数条件：\(-a > 0\) 且 \(b > 0\)，即 \(a < 0\)，\(b > 0\)。
又因为 \(y = \ln x\) 是增函数，所以 \(-a > b > 0\)。
因此得到：
\[ a < 0,\quad b > 0,\quad a + b < 0,\quad |a| > |b|. \]

**逐一判断选项：**

- **A.** \(\frac{1}{a} < \frac{1}{b}\)
因为 \(a < 0\)，\(b > 0\)，所以 \(\frac{1}{a} < 0\)，\(\frac{1}{b} > 0\)，显然 \(\frac{1}{a} < \frac{1}{b}\) 成立。**正确**。

- **B.** \(ab + b^2 < 0\)
\(ab + b^2 = b(a + b)\)，由条件 \(b > 0\) 且 \(a + b < 0\)，得 \(b(a + b) < 0\)。**正确**。

- **C.** \(a^2 < b^2\)
\(a^2 < b^2 \Leftrightarrow |a| < |b|\)。由 \(-a > b > 0\) 得 \(|a| > |b|\)，矛盾。也可举反例：取 \(b = 1, -a = 2\) 即 \(a = -2\)，满足条件，但 \(a^2 = 4 > 1 = b^2\)。**错误**。

- **D.** \(2^a < \left( \frac{1}{2} \right)^b\)
\(\left( \frac{1}{2} \right)^b = 2^{-b}\)，函数 \(y = 2^x\) 单调递增，不等式等价于 \(a < -b\)，即 \(a + b < 0\)，这恰为已推得的条件。**正确**。

正确答案为 A、B、D。
## 答案
ABD
## 总结
解对数不等式时，首先要确保真数大于零，用好单调性等价转化。得到 \(a < 0, b > 0, a+b<0\) 后，各选项分别通过符号分析、因式分解、平方比较、或同底指数函数化为已知条件判断，注意举反例排除错误选项。
