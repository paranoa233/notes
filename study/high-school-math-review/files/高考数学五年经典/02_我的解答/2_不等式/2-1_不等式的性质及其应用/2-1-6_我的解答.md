# 2-1-6 我的解答

对应题目：[[01_题目/2_不等式/2-1_不等式的性质及其应用/2-1-6.md|2-1-6]]

## 完整题目

6. (多选题,2021 江苏南京、盐城二模) 已知 \( a > \; b > 0 \) ,下列选项正确的为 ( )

A. 若 \( \sqrt{a} - \sqrt{b} = 1 \) ,则 \( a - b < 1 \)

B. 若 \( {a}^{2} - {b}^{2} = 1 \) ,则 \( a - b < 1 \)

C. 若 \( {2}^{a} - {2}^{b} = 1 \) ,则 \( a - b < 1 \)

D. 若 \( {\log }_{2}a - {\log }_{2}b = 1 \) ,则 \( a - o < 1 \)

## 题目勘误
D 选项中“则 \( a - o < 1 \)”有 OCR 错误，`o` 应更正为 `b`，即“则 \( a - b < 1 \)”。其余无明显勘误。
## 思路分析
本题已知 \(a > b > 0\)，需要逐个判断四个条件是否能推出 \(a - b < 1\)。核心是利用函数单调性、代数变形或反例来验证。A选项利用平方差公式或函数凸性；B选项可设特值或分解因式；C选项利用指数函数增长性质或指数不等式；D选项利用对数运算转化为线性关系后直接判断。
## 详细解答
已知 \(a > b > 0\)。

**A 选项**：若 \(\sqrt{a} - \sqrt{b} = 1\)，则 \(a - b < 1\)。
因为 \(a - b = (\sqrt{a} - \sqrt{b})(\sqrt{a} + \sqrt{b}) = 1 \cdot (\sqrt{a} + \sqrt{b})\)。
由 \(a > b > 0\) 知 \(\sqrt{a} > \sqrt{b} > 0\)，且 \(\sqrt{a} - \sqrt{b} = 1\)，则 \(\sqrt{a} + \sqrt{b} > 1\)，因此 \(a - b = \sqrt{a} + \sqrt{b} > 1\)。
故 \(a - b < 1\) 不成立，A 错误。

**B 选项**：若 \(a^2 - b^2 = 1\)，则 \(a - b < 1\)。
由 \(a^2 - b^2 = (a - b)(a + b) = 1\) 得 \(a - b = \frac{1}{a + b}\)。
因为 \(a > b > 0\)，所以 \(a + b > 1\)（若 \(b \ge 0.5\) 等不一定，但 \(a > b > 0\) 且 \(a^2 = b^2+1\)，最小情况可取 \(b \to 0^+\)，\(a \to 1^+\)，则 \(a+b > 1\)），因此 \(a - b = \frac{1}{a + b} < 1\)。
故 \(a - b < 1\) 成立，B 正确。

**C 选项**：若 \(2^a - 2^b = 1\)，则 \(a - b < 1\)。
考察函数 \(f(x) = 2^x\)，由拉格朗日中值定理或指数函数的严格凸性可知：
存在 \(\xi \in (b, a)\)，使得 \(2^a - 2^b = 2^\xi \ln 2 \cdot (a - b) = 1\)。
由于 \(2^\xi > 2^b \ge 1\)（因为 \(b>0\)，\(2^b > 1\)），故 \(2^\xi \ln 2 > 1 \cdot \ln 2 \approx 0.693\)，但还需证 \(a-b<1\)。
更严谨地，由 \(2^a - 2^b = 1 \Rightarrow 2^a = 2^b + 1\)，因为 \(a > b > 0\)，有 \(2^{a-b} = \frac{2^a}{2^b} = 1 + \frac{1}{2^b}\)。
因为 \(b>0\)，所以 \(2^b > 1\)，则 \(1 + \frac{1}{2^b} < 2\)，从而 \(2^{a-b} < 2\)，因此 \(a - b < 1\)。
故 \(a - b < 1\) 成立，C 正确。

**D 选项**：若 \(\log_2 a - \log_2 b = 1\)，则 \(a - b < 1\)。
由条件得 \(\log_2 \frac{a}{b} = 1 \Rightarrow \frac{a}{b} = 2 \Rightarrow a = 2b\)。
则 \(a - b = 2b - b = b\)，而题目仅知 \(a > b > 0\)，没有限制 \(b < 1\)。
例如取 \(b = 2\)，则 \(a = 4\)，满足 \(\log_2 a - \log_2 b = 1\)，但 \(a - b = 2 > 1\)。
故 \(a - b < 1\) 不一定成立，D 错误。

综上，正确的选项为 B 和 C。
## 答案
BC
## 总结
处理此类由条件推不等式的问题，关键在于将条件等价转化为容易比较的形式：A、B利用平方差公式转化为和或积；C利用指数运算转化为商并结合指数函数单调性；D直接得到线性关系，通过反例排除。记住严格的不等式放缩方向以及特值验证是常用方法。
