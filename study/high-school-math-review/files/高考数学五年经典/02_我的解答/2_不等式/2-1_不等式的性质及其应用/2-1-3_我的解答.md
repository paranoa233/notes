# 2-1-3 我的解答

对应题目：[[01_题目/2_不等式/2-1_不等式的性质及其应用/2-1-3.md|2-1-3]]

## 完整题目

3. (多选题,2023 山东济南二模) 已知实数 \( a, b, c \) 满足 \( a > b > c \) ,且 \( a + b + c = 0 \) ,则下列说法正确的是 ( )

A. \( \frac{1}{a - c} > \frac{1}{b - c} \) B. \( a - c > {2b} \)

C. \( {a}^{2} > {b}^{2} \) D. \( {ab} + {bc} > 0 \)

## 题目勘误
无明显勘误。

## 思路分析
由已知条件 \( a > b > c \) 且 \( a + b + c = 0 \)，可以推导出 \( a > 0 \)，\( c < 0 \)，\( b \) 的符号不确定，但必然有 \( a + c = -b \)。接着逐项分析四个选项：
- **A 选项**：比较两个正分数的分母大小，利用倒数性质。
- **B 选项**：消去 \( c \) 转化为关于 \( a, b \) 的不等式，结合 \( a > b \) 和 \( a + b + c = 0 \) 判断。
- **C 选项**：对 \( a, b \) 的符号分类讨论，或利用条件转化为绝对值比较。
- **D 选项**：因式分解后恰好能用已知等式化简，直接判断符号。

## 详细解答

由 \( a > b > c \) 且 \( a + b + c = 0 \) 可得：
1. 若 \( a \leq 0 \)，则 \( b < a \leq 0 \)，\( c < b < 0 \)，三者之和必定小于 \( 0 \)，与 \( a+b+c=0 \) 矛盾，故必有 \( a > 0 \)。
2. 同理，若 \( c \geq 0 \)，三者之和必大于 \( 0 \)，矛盾，故必有 \( c < 0 \)。
3. \( b \) 可能为正、负或零。

---

**A 选项**：判断 \( \frac{1}{a - c} > \frac{1}{b - c} \) 是否成立。

由 \( a > b \) 得 \( a - c > b - c \)。
又因为 \( c \) 是三个数中最小的，所以 \( a - c > 0 \)，\( b - c > 0 \)。
对正数而言，分母越大，分数值越小，因此
\[
\frac{1}{a-c} < \frac{1}{b-c}.
\]
A 选项错误。

---

**B 选项**：判断 \( a - c > 2b \) 是否成立。

由 \( a + b + c = 0 \) 得 \( c = -a - b \)，代入不等式左边：
\[
a - c = a - (-a - b) = 2a + b.
\]
原不等式等价于
\[
2a + b > 2b \quad\Longrightarrow\quad 2a > b.
\]
因为 \( a > b \)，易见：
- 若 \( b \leq 0 \)，左边 \( 2a > 0 \)，右边 \( b \leq 0 \)，不等式显然成立；
- 若 \( b > 0 \)，则 \( a > b > 0 \)，有 \( 2a > 2b > b \)，不等式也成立。
综上，\( 2a > b \) 恒成立，故 B 选项正确。

---

**C 选项**：判断 \( a^2 > b^2 \) 是否成立。

等价于判断 \( |a| > |b| \)。
- 当 \( b \geq 0 \) 时，有 \( a > b \geq 0 \)，直接得 \( a^2 > b^2 \)。
- 当 \( b < 0 \) 时，由条件 \( b > c \) 及 \( a+b+c=0 \) 得 \( c = -a-b \)，所以
  \[
  b > -a - b \quad\Longrightarrow\quad a + 2b > 0 \quad\Longrightarrow\quad a > -2b = 2|b|.
  \]
  因此 \( a > 2|b| > |b| \)，平方得 \( a^2 > b^2 \)。
综上，恒有 \( a^2 > b^2 \)，C 选项正确。

---

**D 选项**：判断 \( ab + bc > 0 \) 是否成立。

将 \( ab + bc \) 提取公因式：
\[
ab + bc = b(a + c).
\]
由已知 \( a + b + c = 0 \) 得 \( a + c = -b \)，代入得
\[
ab + bc = b(-b) = -b^2.
\]
因为 \( b \) 为实数，\( b^2 \geq 0 \)，所以 \( -b^2 \leq 0 \)。
当 \( b = 0 \) 时（例如 \( a=1, b=0, c=-1 \)，满足所有条件），\( ab+bc = 0 \)，不满足严格大于 \( 0 \)。
D 选项错误。

## 答案
正确选项为 **B、C**。

## 总结
解决此类“已知多个实数满足不等关系与和条件，判断相关表达式大小”的问题，关键在于：
1. 利用条件和不等关系推出部分变量的符号（如 \( a>0, c<0 \)）；
2. 将多变量表达式通过等式 \( a+b+c=0 \) 进行消元代换；
3. 对变量的不同符号情形分类讨论，确保判断的严密性；
4. 举反例是快速排除错误选项的有效方式（如 D 选项取 \( b=0 \)）。
