# 2-1-1 我的解答

对应题目：[[01_题目/2_不等式/2-1_不等式的性质及其应用/2-1-1.md|2-1-1]]

## 完整题目

1. (多选题,2020 山东泰安期末) 已知 \( a, b, c, d \) 均为实数，则下列命题正确的是 ( )

A. 若 \( a > b, c > d \) ,则 \( {ac} > {bd} \)

B. 若 \( {ab} > 0,{bc} - {ad} > 0 \) ,则 \( \frac{c}{a} - \frac{d}{b} > 0 \)

C. 若 \( a > b, c > d \) ,则 \( a - d > b - c \)

D. 若 \( a > b, c > d > 0 \) ,则 \( \frac{a}{d} > \frac{b}{c} \)

## 题目勘误
无明显勘误。
## 思路分析
本题是一道多选题,考查不等式的基本性质及其应用。需要逐项分析各选项,对于正确选项要给出严格的代数推导,对于错误选项要通过构造反例来排除。尤其注意实数乘法与除法的符号限制,以及同向不等式的可加性。
## 详细解答
对于选项 **A**:
若 \( a > b,\ c > d \), 取反例: \( a = 1,\ b = 0,\ c = -1,\ d = -2 \)。
满足 \( 1 > 0 \) 且 \( -1 > -2 \), 但 \( ac = -1 \), \( bd = 0 \), 此时 \( ac < bd \), 与原结论矛盾。
故 A 错误。

对于选项 **B**:
已知 \( ab > 0 \) 且 \( bc - ad > 0 \)。将目标式通分:
\[
\frac{c}{a} - \frac{d}{b} = \frac{bc - ad}{ab}.
\]
因为 \( bc - ad > 0 \) 且分母 \( ab > 0 \), 所以整个分式大于 0。命题成立。
故 B 正确。

对于选项 **C**:
已知 \( a > b,\ c > d \)。将两式相加得:
\[
a + c > b + d.
\]
移项变形:
\[
a + c > b + d \iff a - d > b - c.
\]
该变形恒成立,因此 \( a - d > b - c \) 正确。
故 C 正确。

对于选项 **D**:
已知 \( a > b,\ c > d > 0 \)。取反例: \( a = -1,\ b = -2,\ c = 3,\ d = 1 \)。
满足 \( -1 > -2 \) 且 \( 3 > 1 > 0 \)。计算得:
\[
\frac{a}{d} = -1,\quad \frac{b}{c} = -\frac{2}{3}.
\]
此时 \( -1 < -\frac{2}{3} \), 与 \( \frac{a}{d} > \frac{b}{c} \) 矛盾。
故 D 错误。

综上,正确选项为 B 和 C。
## 答案
BC
## 总结
处理不等式问题时,必须严格注意各实数的符号条件。同向不等式相加是恒成立的,但乘法与除法必须保证参与运算的数为正数才能保持不等号方向。对于选择题中的错误选项,只需举出一个反例即可排除。
