# 2-1-7 我的解答

对应题目：[[01_题目/2_不等式/2-1_不等式的性质及其应用/2-1-7.md|2-1-7]]

## 完整题目

7. (多选题,2021 湖北荆州期末) 设 \( a > 0, b > 0 \) , 则下列不等式中,恒成立的有 ( )

A. \( \sqrt{\left| a - b\right| } \geq  \left| {\sqrt{a} - \sqrt{b}}\right| \)

B. \( \frac{2ab}{a + b} \geq  \sqrt{ab} \)

C. \( \frac{b}{\sqrt{a}} + \frac{a}{\sqrt{b}} \leq  \sqrt{a} + \sqrt{b} \)

D. \( \frac{a}{1 + a} + \frac{b}{1 + b} > \frac{a + b}{1 + a + b} \)

## 题目勘误
无明显勘误。
## 思路分析
本题是多选题，需对四个不等式逐一判断是否对所有正数 \(a,b\) 恒成立。可利用平方去根号、基本不等式链、作差比较等方法进行推理。A 选项平方后比较；B 选项联想调和平均与几何平均的大小关系；C 选项可通分或乘 \(\sqrt{ab}\) 转化为比较 \((a-b)(\sqrt{a}-\sqrt{b})\) 的符号；D 选项可通分作差或利用函数单调性判断。
## 详细解答
已知 \(a>0,\;b>0\)。

**选项 A**：\(\sqrt{|a-b|} \geq |\sqrt{a} - \sqrt{b}|\)

两边非负，平方得：\(|a-b| \geq (\sqrt{a} - \sqrt{b})^2 = a + b - 2\sqrt{ab}\)。  
当 \(a \ge b\) 时，\(|a-b|=a-b\)，则
\[
a-b - (a+b-2\sqrt{ab}) = -2b + 2\sqrt{ab} = 2\sqrt{b}(\sqrt{a} - \sqrt{b}) \ge 0;
\]
当 \(a < b\) 时，\(|a-b|=b-a\)，则
\[
b-a - (a+b-2\sqrt{ab}) = -2a + 2\sqrt{ab} = 2\sqrt{a}(\sqrt{b} - \sqrt{a}) \ge 0.
\]
故 \(|a-b| \ge (\sqrt{a} - \sqrt{b})^2\) 恒成立，开方即得 \(\sqrt{|a-b|} \geq |\sqrt{a} - \sqrt{b}|\)，A 正确。

**选项 B**：\(\dfrac{2ab}{a+b} \geq \sqrt{ab}\)

由基本不等式链知，对正数 \(a,b\) 有调和平均 \(\le\) 几何平均，即
\[
\frac{2}{\frac{1}{a}+\frac{1}{b}} = \frac{2ab}{a+b} \le \sqrt{ab}.
\]
等号仅在 \(a=b\) 时取得，因此不等号方向反了，B 错误。

**选项 C**：\(\dfrac{b}{\sqrt{a}} + \dfrac{a}{\sqrt{b}} \leq \sqrt{a} + \sqrt{b}\)

两边同乘 \(\sqrt{ab}>0\)，得
\[
b\sqrt{b} + a\sqrt{a} \le a\sqrt{b} + b\sqrt{a}.
\]
移项：
\[
(a\sqrt{a} - a\sqrt{b}) + (b\sqrt{b} - b\sqrt{a}) \le 0
\]
\[
a(\sqrt{a} - \sqrt{b}) - b(\sqrt{a} - \sqrt{b}) = (a-b)(\sqrt{a}-\sqrt{b}) \le 0.
\]
由于 \(a-b\) 与 \(\sqrt{a}-\sqrt{b}\) 同号，\((a-b)(\sqrt{a}-\sqrt{b}) \ge 0\)，仅当 \(a=b\) 时取等。故原不等式应为反向的 \(\ge\)，C 错误。

**选项 D**：\(\dfrac{a}{1+a} + \dfrac{b}{1+b} > \dfrac{a+b}{1+a+b}\)

作差比较：
\[
\frac{a}{1+a} + \frac{b}{1+b} - \frac{a+b}{1+a+b}
= \frac{a(1+b)(1+a+b) + b(1+a)(1+a+b) - (a+b)(1+a)(1+b)}{(1+a)(1+b)(1+a+b)}.
\]
分母恒正。分子展开、合并同类项：
\[
\begin{aligned}
& a(1+b)(1+a+b) = a(1+a+b + b+ab+b^2) = a + a^2 + 2ab + a^2b + ab^2, \\
& b(1+a)(1+a+b) = b(1+a+b + a+a^2+ab) = b + 2ab + b^2 + a^2b + ab^2, \\
& (a+b)(1+a)(1+b) = (a+b)(1+a+b+ab) = a+b + a^2 + b^2 + 2ab + a^2b + ab^2.
\end{aligned}
\]
前两式之和减去第三式得：
\[
2ab + a^2b + ab^2 = ab(2 + a + b) > 0.
\]
因此左减右恒正，不等式恒成立，D 正确。
## 答案
AD
## 总结
判断条件不等式是否恒成立，常用方法有平方去根号、利用已知不等式链、作差比较以及同乘正因子等。本题 A、D 为恒成立的不等式，B 调和平均 \(\le\) 几何平均，C 实际上反向成立。注意多选题需逐项严格证明或举反例。
