# 2-1-2 我的解答

对应题目：[[01_题目/2_不等式/2-1_不等式的性质及其应用/2-1-2.md|2-1-2]]

## 完整题目

2. (多选题,2024 江苏南京学情调研) 若 \( a < 0 < \; b \) ,且 \( a + b > 0 \) ,则 ( )

A. \( \frac{a}{b} >  - 1 \) B. \( \left| a\right|  < \left| b\right| \)

C. \( \frac{1}{a} + \frac{1}{b} > 0 \) D. \( \left( {a - 1}\right) \left( {b - 1}\right)  < 1 \)

## 题目勘误
无明显勘误。

## 思路分析
本题为多选题，考查不等式的基本性质及其应用。已知条件为 \(a<0<b\) 且 \(a+b>0\)，由此可推导出 \(|a| < |b|\)。选项涉及分式、绝对值、倒数之和以及多项式乘积的不等式。可通过不等式性质直接推理，也可选取符合题意的特殊值（如 \(a=-2,\;b=3\)）进行验证。

## 详细解答
已知 \(a < 0 < b\) 且 \(a + b > 0\)。

**选项 A：** \(\frac{a}{b} > -1\)

因为 \(b > 0\)，不等式两边同乘以正数 \(b\)，不等号方向不变：
\[
\frac{a}{b} > -1 \iff a > -b
\]
由已知 \(a + b > 0\) 移项得 \(a > -b\)，故原不等式成立。**A 正确。**

**选项 B：** \(|a| < |b|\)

因为 \(a < 0\)，所以 \(|a| = -a\)；因为 \(b > 0\)，所以 \(|b| = b\)。
由 \(a + b > 0\) 得 \(b > -a\)，即 \(|b| > |a|\)，亦即 \(|a| < |b|\)。**B 正确。**

**选项 C：** \(\frac{1}{a} + \frac{1}{b} > 0\)

将左边通分：
\[
\frac{1}{a} + \frac{1}{b} = \frac{a + b}{ab}
\]
已知 \(a < 0 < b\)，故 \(ab < 0\)；又 \(a + b > 0\)。
因此 \(\frac{a + b}{ab} < 0\)，即 \(\frac{1}{a} + \frac{1}{b} < 0\)，原不等式不成立。**C 错误。**

**选项 D：** \((a - 1)(b - 1) < 1\)

展开左边：
\[
(a - 1)(b - 1) = ab - a - b + 1
\]
原不等式等价于：
\[
ab - a - b + 1 < 1 \iff ab - a - b < 0 \iff ab < a + b
\]
因为 \(a < 0,\;b > 0\)，所以 \(ab < 0\)；又已知 \(a + b > 0\)。
因此 \(ab < 0 < a + b\)，即 \(ab < a + b\) 恒成立，从而原不等式成立。**D 正确。**

（也可取特殊值验证：令 \(a = -2,\;b = 3\)，则 \(a+b=1>0\)。A：\(-\frac{2}{3} > -1\) 成立；B：\(2 < 3\) 成立；C：\(-\frac{1}{2} + \frac{1}{3} = -\frac{1}{6} < 0\) 不成立；D：\((-3) \times 2 = -6 < 1\) 成立。结论一致。）

综上，正确选项为 A、B、D。

## 答案
ABD

## 总结
处理此类由基本条件出发的不等式多选问题时，要注意：
1. 利用不等式的可乘性时，务必明确所乘因子的正负，以确定不等号方向。
2. 善用等价变形，将复杂表达式转化为易于用已知条件判断的形式。
3. 对于抽象条件，灵活使用特值法可快速排除错误选项或验证猜想。本题核心是由 \(a<0<b\) 和 \(a+b>0\) 推出 \(b > -a\)，进而得到 \(|a|<|b|\) 等一系列结论。
