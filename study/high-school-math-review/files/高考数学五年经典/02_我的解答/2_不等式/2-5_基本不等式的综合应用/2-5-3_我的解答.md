# 2-5-3 我的解答

对应题目：[[01_题目/2_不等式/2-5_基本不等式的综合应用/2-5-3.md|2-5-3]]

## 完整题目

3.(2021 湖南邵阳重点中学模拟)已知在锐角三角形 \( {ABC} \) 中,角 \( A, B, C \) 的对边分别为 \( a, b, c \) . 若 \( {2b}\cos C = c\cos B \) ,则 \( \frac{1}{\tan A} + \frac{1}{\tan B} + \frac{1}{\tan C} \) 的最小值为 ( )

A. \( \frac{2\sqrt{7}}{3} \) B. \( \sqrt{5} \) C. \( \frac{\sqrt{7}}{3} \) D. \( 2\sqrt{5} \)

## 题目勘误
无明显勘误。

## 思路分析
由边角关系 \(2b\cos C = c\cos B\) 结合正弦定理转化为角的正弦余弦关系，进而得到角 \(B\) 与角 \(C\) 的正切关系。将目标式 \(\frac{1}{\tan A} + \frac{1}{\tan B} + \frac{1}{\tan C}\) 全部转化为 \(\tan B\) 的函数，利用基本不等式即可求得最小值。

## 详细解答
由正弦定理，\(b = 2R\sin B\)，\(c = 2R\sin C\)，代入已知条件 \(2b\cos C = c\cos B\) 得：
\[
2 \cdot 2R\sin B \cos C = 2R\sin C \cos B
\]
化简得：
\[
2\sin B \cos C = \sin C \cos B
\]
两边同除以 \(\cos B\cos C\)（因为锐角三角形，\(\cos B,\cos C>0\)），得：
\[
2\tan B = \tan C
\]
即 \(\tan C = 2\tan B\)。

在 \(\triangle ABC\) 中，\(A+B+C=\pi\)，所以 \(\tan A = -\tan(B+C) = \frac{\tan B + \tan C}{\tan B \tan C - 1}\)。
由三角形为锐角三角形，知 \(\tan A>0\)，且 \(\tan B>0,\tan C>0\)，故分母 \(\tan B\tan C - 1 > 0\)，即 \(2\tan^2 B > 1\)，得 \(\tan B > \frac{1}{\sqrt{2}}\)。

目标式：
\[
y = \frac{1}{\tan A} + \frac{1}{\tan B} + \frac{1}{\tan C}
\]
代入 \(\tan C = 2\tan B\) 及 \(\tan A\)：
\[
\frac{1}{\tan A} = \frac{\tan B \tan C - 1}{\tan B + \tan C} = \frac{2\tan^2 B - 1}{3\tan B}
\]
\[
\frac{1}{\tan B} = \frac{1}{\tan B},\quad \frac{1}{\tan C} = \frac{1}{2\tan B}
\]
因此：
\[
y = \frac{2\tan^2 B - 1}{3\tan B} + \frac{1}{\tan B} + \frac{1}{2\tan B}
= \frac{2\tan^2 B - 1}{3\tan B} + \frac{3}{2\tan B}
\]
通分合并：
\[
y = \frac{2(2\tan^2 B - 1) + 9}{6\tan B} = \frac{4\tan^2 B + 7}{6\tan B}
\]
拆项：
\[
y = \frac{2}{3}\tan B + \frac{7}{6\tan B}
\]
因为 \(\tan B > 0\)，由基本不等式得：
\[
y \ge 2\sqrt{\frac{2}{3}\tan B \cdot \frac{7}{6\tan B}} = 2\sqrt{\frac{14}{18}} = 2\sqrt{\frac{7}{9}} = \frac{2\sqrt{7}}{3}
\]
等号成立当且仅当 \(\frac{2}{3}\tan B = \frac{7}{6\tan B}\)，即 \(\tan^2 B = \frac{7}{4}\)，又 \(\tan B>0\)，解得 \(\tan B = \frac{\sqrt{7}}{2}\)。
此时 \(\tan C = \sqrt{7}\)，\(\tan B\tan C = \frac{7}{2}>1\)，满足锐角三角形条件。故最小值可取到 \(\frac{2\sqrt{7}}{3}\)。

## 答案
A

## 总结
本题属于三角形中的最值问题，关键步骤为：
1. 利用正弦定理将边角关系转化为正切关系；
2. 用三角形内角和公式将目标式统一为单变量 \(\tan B\)；
3. 采用基本不等式求最值，注意验证等号成立条件是否满足锐角三角形的限制。
