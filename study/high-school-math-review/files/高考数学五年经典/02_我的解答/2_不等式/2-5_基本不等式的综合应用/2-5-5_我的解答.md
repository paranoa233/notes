# 2-5-5 我的解答

对应题目：[[01_题目/2_不等式/2-5_基本不等式的综合应用/2-5-5.md|2-5-5]]

## 完整题目

5.(2021 甘肃平凉一中模拟)若曲线 \( y = \ln (x + \; a) \) 的一条切线为 \( y = \mathrm{e}x - b \) ( \( \mathrm{e} \) 为自然对数的底数),其中 \( a, b \) 为正实数,则 \( \frac{1}{\mathrm{e}a} + \frac{1}{b} \) 的取值范围是 ( )

A. \( \left\lbrack  {2,\mathrm{e}}\right\rbrack \) B. \( (\mathrm{e},4\rbrack \)

C. \( \lbrack 2, + \infty ) \) D. \( \lbrack \mathrm{e}, + \infty ) \)

## 题目勘误
无明显勘误。
## 思路分析
由切线的斜率等于曲线在该点的导数，可建立切点横坐标与参数 \(a\) 的关系。利用切点在曲线与切线上，消去切点坐标，得到 \(a\) 与 \(b\) 的等量关系 \(b=2-\mathrm{e}a\)。结合 \(a,b\) 为正实数，确定 \(a\) 的取值范围。将目标式 \(\frac{1}{\mathrm{e}a}+\frac{1}{b}\) 化为一元函数，利用函数单调性或基本不等式求取值范围。
## 详细解答
设切点为 \((x_0,y_0)\)。曲线 \(y = \ln(x+a)\) 的导数为
\[ y' = \frac{1}{x+a}. \]
因为切线为 \(y = \mathrm{e}x - b\)，其斜率为 \(\mathrm{e}\)，所以
\[ \frac{1}{x_0 + a} = \mathrm{e} \quad\Rightarrow\quad x_0 + a = \frac{1}{\mathrm{e}}. \tag{1} \]
切点在曲线上，故
\[ y_0 = \ln(x_0+a) = \ln\frac{1}{\mathrm{e}} = -1. \tag{2} \]
切点也在切线上，得
\[ y_0 = \mathrm{e}x_0 - b. \tag{3} \]
将 \((1)(2)\) 代入 \((3)\)：
\[ -1 = \mathrm{e}x_0 - b. \]
由 \((1)\) 知 \(x_0 = \frac{1}{\mathrm{e}} - a\)，代入上式：
\[ -1 = \mathrm{e}\left(\frac{1}{\mathrm{e}} - a\right) - b = 1 - \mathrm{e}a - b, \]
解得
\[ b = 2 - \mathrm{e}a. \tag{4} \]
因为 \(a,b\) 均为正实数，所以
\[ a > 0,\quad b = 2 - \mathrm{e}a > 0 \;\Rightarrow\; a < \frac{2}{\mathrm{e}}. \]
因此 \(a \in \left(0, \frac{2}{\mathrm{e}}\right)\)，令 \(t = \mathrm{e}a\)，则 \(t \in (0, 2)\)。

所求表达式为
\[
\frac{1}{\mathrm{e}a} + \frac{1}{b} = \frac{1}{t} + \frac{1}{2-t}.
\]
构造函数 \(f(t) = \frac{1}{t} + \frac{1}{2-t}\)，\(t \in (0,2)\)。  
求导得
\[
f'(t) = -\frac{1}{t^2} + \frac{1}{(2-t)^2}
= \frac{t^2 - (2-t)^2}{t^2(2-t)^2}
= \frac{4(t-1)}{t^2(2-t)^2}.
\]
当 \(0 < t < 1\) 时，\(f'(t) < 0\)，\(f(t)\) 单调递减；  
当 \(1 < t < 2\) 时，\(f'(t) > 0\)，\(f(t)\) 单调递增。  
故 \(f(t)\) 在 \(t = 1\) 处取得最小值：
\[
f(1) = \frac{1}{1} + \frac{1}{2-1} = 2.
\]
当 \(t \to 0^+\) 或 \(t \to 2^-\) 时，\(f(t) \to +\infty\)。  
所以 \(f(t)\) 的值域为 \([2, +\infty)\)。
由 \(t = \mathrm{e}a\)，对应的 \(a = \frac{1}{\mathrm{e}}\)，\(b = 2 - 1 = 1 > 0\)，可以取到最小值 \(2\)。

因此 \(\frac{1}{\mathrm{e}a} + \frac{1}{b}\) 的取值范围是 \([2, +\infty)\)，对应选项 C。
## 答案
C
## 总结
本题以曲线的切线为载体，考查导数的几何意义与函数的值域（或基本不等式）。关键是利用切点“既在曲线上又在切线上”的双重身份，将多元问题转化为一元函数问题，再利用单调性或基本不等式求取值范围。需注意检验定义域端点与等号成立的条件，确保 \(a,b\) 为正实数。
