# 2-4-9 我的解答

对应题目：[[01_题目/2_不等式/2-4_利用基本不等式求最值/2-4-9.md|2-4-9]]

## 完整题目

9.(2021 广东揭阳质量检测)已知 \( \bigtriangleup  {ABC} \) 的内角 \( A, B, C \) 所对的边分别为 \( a, b, c \) ，且满足 \( a = 2,{a}^{2} = \; 2{b}^{2} + {c}^{2} \) ，则 \( \bigtriangleup  {ABC} \) 的面积的最大值为___.

## 题目勘误
无明显勘误。

## 思路分析
由已知边的关系 \(a^2=2b^2+c^2\) 及 \(a=2\)，结合余弦定理可求出 \(\cos A\) 进而得到 \(\sin A\)，将三角形面积表示为边 \(b\) 的函数，再利用基本不等式或二次函数求最值。注意根据边的限制条件确定自变量的取值范围，并验证等号成立条件。

## 详细解答
已知 \(a=2\)，且 \(a^2 = 2b^2 + c^2\)，即  
\[ 4 = 2b^2 + c^2 \quad \Rightarrow \quad c^2 = 4 - 2b^2. \]

由余弦定理：  
\[ \cos A = \frac{b^2 + c^2 - a^2}{2bc} = \frac{b^2 + c^2 - (2b^2 + c^2)}{2bc} = -\frac{b}{2c}. \]
因此  
\[ \sin A = \sqrt{1 - \cos^2 A} = \sqrt{1 - \frac{b^2}{4c^2}} = \frac{\sqrt{4c^2 - b^2}}{2c}. \]

\(\triangle ABC\) 的面积  
\[ S = \frac{1}{2}bc\sin A = \frac{1}{2}bc \cdot \frac{\sqrt{4c^2 - b^2}}{2c} = \frac{b}{4}\sqrt{4c^2 - b^2}. \]

将 \(c^2 = 4 - 2b^2\) 代入根号内：  
\[ 4c^2 - b^2 = 4(4 - 2b^2) - b^2 = 16 - 9b^2, \]
所以  
\[ S = \frac{b}{4}\sqrt{16 - 9b^2}. \]

由 \(c^2 = 4 - 2b^2 > 0\) 及 \(16 - 9b^2 \ge 0\)，得  
\[ 0 < b^2 \le \frac{16}{9}. \]

将面积表达式改写为  
\[ S = \frac{1}{4}\sqrt{b^2(16 - 9b^2)} = \frac{1}{4}\sqrt{9b^2\left(\frac{16}{9} - b^2\right)}. \]

对正数 \(b^2\) 与 \(\frac{16}{9} - b^2\) 应用基本不等式：  
\[ b^2\left(\frac{16}{9} - b^2\right) \le \left( \frac{b^2 + \frac{16}{9} - b^2}{2} \right)^2 = \left( \frac{8}{9} \right)^2 = \frac{64}{81}, \]
当且仅当 \(b^2 = \frac{16}{9} - b^2\)，即 \(b^2 = \frac{8}{9}\) 时等号成立。

此时 \(b^2 = \frac{8}{9}\) 在范围内，因此  
\[ S_{\max} = \frac{1}{4} \sqrt{9 \times \frac{64}{81}} = \frac{1}{4} \times \frac{8}{3} = \frac{2}{3}. \]

## 答案
\(\dfrac{2}{3}\)

## 总结
本题将三角形面积转化为关于边 \(b\) 的函数，利用已知关系消去 \(c\)，构造出适合用基本不等式的积的形式。注意需先由边长限制确定自变量的取值范围，并验证等号成立条件。基本不等式在求最值问题中常与函数、几何相结合，构造定值、验证取等是得分关键。
