# 2-4-1 我的解答

对应题目：[[01_题目/2_不等式/2-4_利用基本不等式求最值/2-4-1.md|2-4-1]]

## 完整题目

1. (2021 江苏苏州期初调研)设 \( a > 0 \) ， \( b > 0 \) ，且 \( {2a} + b = 1 \) ，则 \( \frac{1}{a} + \frac{2a}{a + b} \) ( )

A. 有最小值为 4 B. 有最小值为 \( 2\sqrt{2} + 1 \)

C. 有最小值为 \( \frac{14}{3} \) D. 无最小值

## 题目勘误
无明显勘误。
## 思路分析
已知正数 \(a, b\) 满足 \(2a + b = 1\)，求 \(\frac{1}{a} + \frac{2a}{a + b}\) 的最小值。可通过消元转化为关于 \(a\) 的函数求导，或利用“1”的代换结合基本不等式求最值。考虑到分母中有 \(a+b\)，可先用条件消去 \(b\)，转化为一元函数；也可将 \(1\) 代换为 \(2a+b\)，再通过换元凑出可运用基本不等式的形式。
## 详细解答

**法一（消元求导）**  
由 \(2a + b = 1\)，且 \(a>0,\;b>0\)，得 \(b = 1 - 2a > 0\)，故 \(0 < a < \frac{1}{2}\)。  
代入所求式：
\[
\frac{1}{a} + \frac{2a}{a + b} = \frac{1}{a} + \frac{2a}{a + (1 - 2a)} = \frac{1}{a} + \frac{2a}{1 - a}.
\]
设 \(f(a) = \dfrac{1}{a} + \dfrac{2a}{1 - a}\)，求导：
\[
f'(a) = -\frac{1}{a^2} + \frac{2(1 - a) - 2a(-1)}{(1 - a)^2} = -\frac{1}{a^2} + \frac{2}{(1 - a)^2}.
\]
令 \(f'(a) = 0\) 得：
\[
\frac{2}{(1 - a)^2} = \frac{1}{a^2} \implies \sqrt{2}a = 1 - a \quad (a>0),
\]
解得 \(a = \dfrac{1}{\sqrt{2} + 1} = \sqrt{2} - 1 \in \left(0, \frac{1}{2}\right)\)。
当 \(0 < a < \sqrt{2} - 1\) 时，\(f'(a) < 0\)；当 \(\sqrt{2} - 1 < a < \frac{1}{2}\) 时，\(f'(a) > 0\)。  
故 \(f(a)\) 在 \(a = \sqrt{2} - 1\) 处取得最小值：
\[
\begin{aligned}
f_{\min} &= \frac{1}{\sqrt{2} - 1} + \frac{2(\sqrt{2} - 1)}{1 - (\sqrt{2} - 1)} \\
&= (\sqrt{2} + 1) + \frac{2(\sqrt{2} - 1)}{2 - \sqrt{2}} \\
&= \sqrt{2} + 1 + \frac{2(\sqrt{2} - 1)}{\sqrt{2}(\sqrt{2} - 1)} \\
&= \sqrt{2} + 1 + \sqrt{2} = 2\sqrt{2} + 1.
\end{aligned}
\]

**法二（“1”的代换结合基本不等式）**  
由 \(2a + b = 1\) 得 \(\dfrac{1}{a} = \dfrac{2a + b}{a} = 2 + \dfrac{b}{a}\)，  
则原式
\[
\frac{1}{a} + \frac{2a}{a + b} = 2 + \frac{b}{a} + \frac{2a}{a + b}.
\]
令 \(t = \dfrac{b}{a} > 0\)，由 \(2a + b = 1\) 得 \(a = \dfrac{1}{2 + t}\)，且 \(a + b = a(1 + t) = \dfrac{1 + t}{2 + t}\)。
于是
\[
\frac{b}{a} + \frac{2a}{a + b} = t + \frac{2 \cdot \frac{1}{2 + t}}{\frac{1 + t}{2 + t}} = t + \frac{2}{1 + t}.
\]
故原式
\[
= 2 + t + \frac{2}{1 + t} = 1 + \left[(1 + t) + \frac{2}{1 + t}\right].
\]
由基本不等式，对 \(x = 1 + t > 1\)，
\[
x + \frac{2}{x} \geq 2\sqrt{x \cdot \frac{2}{x}} = 2\sqrt{2},
\]
当且仅当 \(x = \frac{2}{x}\) 即 \(x = \sqrt{2}\) 时取等号。此时 \(1 + t = \sqrt{2}\)，\(t = \sqrt{2} - 1 > 0\)，满足条件。  
因此原式的最小值为 \(1 + 2\sqrt{2}\)。

## 答案
B

## 总结
本题给出二元正数的一次约束条件，求分式表达式的最值。可通过消元转化为一元函数求解，亦可利用“1”的代换与换元，将目标式化为可运用基本不等式的形式。务必验证等号成立条件是否在定义域内。
