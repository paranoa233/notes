# 9-2-5 我的解答

对应题目：[[01_题目/9_直线与圆/9-2_两条直线的位置关系与距离公式/9-2-5.md|9-2-5]]

## 完整题目

5.(2024 重庆第三次联合诊断检测)当点 \( P\left( {-1,0}\right) \) 到直线 \( l : \left( {{3\lambda } + 1}\right) x + \left( {\lambda  + 1}\right) y - \left( {{4\lambda } + }\right. \) 2) \( = 0 \) 的距离最大时,实数 \( \lambda \) 的值为 ( )

A. -1 B. 1 C. -2 D. 2

## 题目勘误
原题中直线 \( l \) 的方程显示为：
\[
\left( {{3\lambda } + 1}\right) x + \left( {\lambda  + 1}\right) y - \left( {{4\lambda } + }\right. 2) = 0
\]
其中后半部分 “\(-\left( {{4\lambda } + }\right. 2)\)” 存在括号不匹配和排版断裂。建议更正为：
\[
(3\lambda + 1)x + (\lambda + 1)y - (4\lambda + 2) = 0
\]
理由：原式明显为排版错误，参照同类型“含参数的直线过定点”问题的常见命题习惯，该部分应为 \(-(4\lambda+2)\)。

## 思路分析
直线方程中含参数 \(\lambda\)，可将其整理为关于 \(\lambda\) 的方程，求出直线所过的定点 \(Q\)。点 \(P\) 到直线的距离的最大值即为 \(P\) 到定点 \(Q\) 的距离，此时直线 \(l\) 与 \(PQ\) 垂直，利用法向量与 \(\overrightarrow{PQ}\) 共线可解得 \(\lambda\)。

## 详细解答
将直线 \(l\) 的方程按 \(\lambda\) 整理：
\[
(3\lambda + 1)x + (\lambda + 1)y - (4\lambda + 2) = 0
\]
\[
\Rightarrow \lambda(3x + y - 4) + (x + y - 2) = 0
\]
该式对任意实数 \(\lambda\) 恒成立，故直线 \(l\) 过定点 \(Q\)，坐标满足：
\[
\begin{cases}
3x + y - 4 = 0 \\
x + y - 2 = 0
\end{cases}
\]
解方程组：两式相减得 \(2x - 2 = 0\)，解得 \(x = 1\)；代入 \(x + y = 2\) 得 \(y = 1\)。
所以定点为 \(Q(1, 1)\)。

点 \(P(-1, 0)\) 到定点 \(Q(1, 1)\) 的距离为：
\[
|PQ| = \sqrt{(1 - (-1))^2 + (1 - 0)^2} = \sqrt{2^2 + 1^2} = \sqrt{5}
\]
当且仅当直线 \(l\) 与 \(PQ\) 垂直时，点 \(P\) 到直线 \(l\) 的距离达到最大，最大值为 \(\sqrt{5}\)。
此时直线 \(l\) 的法向量 \(\vec{n} = (3\lambda + 1,\ \lambda + 1)\) 与向量 \(\overrightarrow{PQ} = (2,\ 1)\) 平行，有：
\[
\frac{3\lambda + 1}{2} = \frac{\lambda + 1}{1}
\]
解得：
\[
3\lambda + 1 = 2\lambda + 2 \quad\Rightarrow\quad \lambda = 1
\]

## 答案
B

## 总结
对于含参数直线族问题，分离参数求定点是关键。点 \(P\) 到动直线的最大距离即为 \(P\) 到定点的距离，达到最大时直线垂直于两点连线，从而由法向量共线求解参数。
