# 9-2-2 我的解答

对应题目：[[01_题目/9_直线与圆/9-2_两条直线的位置关系与距离公式/9-2-2.md|9-2-2]]

## 完整题目

2.(2024 江苏无锡四校期初调研)若直线 \( {l}_{1} : {ax} + \; \left( {1 - a}\right) y - 3 = 0 \) 与直线 \( {l}_{2} : \left( {a - 1}\right) x + \left( {{2a} + 3}\right) y \; - 2 = 0 \) 互相垂直，则 \( a \) 的值为 ( )

A. -3

B. \( - \frac{1}{2} \)

C. 0 或 \( - \frac{3}{2} \) D. 1 或 -3

## 题目勘误
无明显勘误。
## 思路分析
两直线互相垂直的充要条件是它们的一般式方程中 \(x\) 和 \(y\) 的系数满足 \(A_1A_2 + B_1B_2 = 0\)。将直线 \(l_1\) 和 \(l_2\) 的系数代入，得到关于 \(a\) 的方程，解方程并检验是否出现直线退化（即 \(x, y\) 系数同时为 \(0\)）的情况，最终确定 \(a\) 的值。
## 详细解答
直线 \(l_1\) 的一般式为：
\[
a x + (1-a) y - 3 = 0
\]
其中 \(A_1 = a,\; B_1 = 1-a\)。

直线 \(l_2\) 的一般式为：
\[
(a-1) x + (2a+3) y - 2 = 0
\]
其中 \(A_2 = a-1,\; B_2 = 2a+3\)。

由两直线垂直的充要条件 \(A_1A_2 + B_1B_2 = 0\)，得：
\[
a(a-1) + (1-a)(2a+3) = 0
\]

展开并合并同类项：
\[
\begin{aligned}
a^2 - a + (1-a)(2a+3) &= a^2 - a + \big[(2a+3) - a(2a+3)\big] \\
&= a^2 - a + 2a + 3 - 2a^2 - 3a \\
&= (a^2 - 2a^2) + (-a + 2a - 3a) + 3 \\
&= -a^2 - 2a + 3 = 0
\end{aligned}
\]

两边同乘 \(-1\)：
\[
a^2 + 2a - 3 = 0
\]

因式分解得：
\[
(a+3)(a-1) = 0
\]
解得 \(a = -3\) 或 \(a = 1\)。

当 \(a = 1\) 时，\(l_1: x - 3 = 0\) （垂直于 \(x\) 轴的直线），\(l_2: 5y - 2 = 0\) （垂直于 \(y\) 轴的直线），它们互相垂直。
当 \(a = -3\) 时，\(l_1: -3x + 4y - 3 = 0\)，\(l_2: -4x - 3y - 2 = 0\)，满足垂直条件。
两个值均符合题意。
## 答案
D
## 总结
求解两条含参数直线互相垂直的问题，可直接利用一般式中系数关系 \(A_1A_2 + B_1B_2 = 0\) 建立方程，解出参数后需简单检验直线是否有效（\(x, y\) 系数不全为零）。本题中求得的两个值均使两条直线正常表示且垂直。
