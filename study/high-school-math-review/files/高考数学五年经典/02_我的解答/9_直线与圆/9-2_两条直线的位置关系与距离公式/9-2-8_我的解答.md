# 9-2-8 我的解答

对应题目：[[01_题目/9_直线与圆/9-2_两条直线的位置关系与距离公式/9-2-8.md|9-2-8]]

## 完整题目

8.(2024四川成都七中二诊)已知直线 \( l \) 经过点 \( P\left( {0,1}\right) \) ,且被两条平行直线 \( {l}_{1} : \sqrt{3}x + y + 1 = 0 \) 和 \( {l}_{2} : \sqrt{3}x + y + 5 = 0 \) 截得的线段长为 \( 2\sqrt{2} \) ,则直线 \( l \) 的方程为___.

## 题目勘误
无明显勘误。

## 思路分析
直线 \(l\) 过定点 \(P(0,1)\)，被两条平行直线 \(l_1\) 与 \(l_2\) 截得的线段长已知。两平行线间的距离可以直接由距离公式求出。该距离与截得的线段长构成直角三角形的直角边和斜边，由此可得直线 \(l\) 与平行线的夹角。再利用两直线夹角公式，结合平行线的斜率，即可求出直线 \(l\) 的斜率，从而得到直线方程。本题考查平行线间距离公式、两直线夹角公式以及斜率的几何意义。

## 详细解答
**第一步：求两条平行直线间的距离**
直线 \(l_1: \sqrt{3}x + y + 1 = 0\)，\(l_2: \sqrt{3}x + y + 5 = 0\)。
由两平行线间距离公式，得
\[
d = \frac{|5 - 1|}{\sqrt{(\sqrt{3})^2 + 1^2}} = \frac{4}{\sqrt{3 + 1}} = \frac{4}{2} = 2.
\]

**第二步：利用截得线段长求直线 \(l\) 与平行线的夹角**
已知直线 \(l\) 被 \(l_1\) 和 \(l_2\) 截得的线段长为 \(L = 2\sqrt{2}\)。
设直线 \(l\) 与平行线所成的锐角为 \(\theta\)，则有
\[
\sin\theta = \frac{d}{L} = \frac{2}{2\sqrt{2}} = \frac{\sqrt{2}}{2},
\]
所以 \(\theta = 45^\circ\)。
即直线 \(l\) 与已知平行线的夹角为 \(45^\circ\)。

**第三步：求已知平行线的斜率**
将 \(l_1\) 的方程化为斜截式：
\[
y = -\sqrt{3}x - 1,
\]
可知平行线的斜率 \(k_0 = -\sqrt{3}\)。

**第四步：利用夹角公式求直线 \(l\) 的斜率**
设直线 \(l\) 的斜率为 \(k\)，由两直线夹角公式得：
\[
\tan 45^\circ = \left| \frac{k - k_0}{1 + k \cdot k_0} \right|.
\]
代入 \(k_0 = -\sqrt{3}\)，有
\[
1 = \left| \frac{k + \sqrt{3}}{1 - \sqrt{3}k} \right|.
\]
分两种情况讨论：
① \(\dfrac{k + \sqrt{3}}{1 - \sqrt{3}k} = 1\)
\[
k + \sqrt{3} = 1 - \sqrt{3}k \quad \Rightarrow \quad k + \sqrt{3}k = 1 - \sqrt{3} \quad \Rightarrow \quad k(1 + \sqrt{3}) = 1 - \sqrt{3}
\]
\[
k = \frac{1 - \sqrt{3}}{1 + \sqrt{3}} = \frac{(1 - \sqrt{3})^2}{(1 + \sqrt{3})(1 - \sqrt{3})} = \frac{1 - 2\sqrt{3} + 3}{1 - 3} = \frac{4 - 2\sqrt{3}}{-2} = \sqrt{3} - 2.
\]
② \(\dfrac{k + \sqrt{3}}{1 - \sqrt{3}k} = -1\)
\[
k + \sqrt{3} = -1 + \sqrt{3}k \quad \Rightarrow \quad k - \sqrt{3}k = -1 - \sqrt{3} \quad \Rightarrow \quad k(1 - \sqrt{3}) = -(1 + \sqrt{3})
\]
\[
k = \frac{-(1 + \sqrt{3})}{1 - \sqrt{3}} = \frac{1 + \sqrt{3}}{\sqrt{3} - 1} = \frac{(1 + \sqrt{3})^2}{(\sqrt{3} - 1)(\sqrt{3} + 1)} = \frac{1 + 2\sqrt{3} + 3}{3 - 1} = \frac{4 + 2\sqrt{3}}{2} = 2 + \sqrt{3}.
\]
因此直线 \(l\) 的斜率 \(k = \sqrt{3} - 2\) 或 \(k = 2 + \sqrt{3}\)。

**第五步：写出直线 \(l\) 的方程**
直线 \(l\) 过点 \(P(0,1)\)，由点斜式可得：
当 \(k = \sqrt{3} - 2\) 时，方程为
\[
y - 1 = (\sqrt{3} - 2)(x - 0) \quad \Rightarrow \quad y = (\sqrt{3} - 2)x + 1,
\]
化为一般式：\((\sqrt{3} - 2)x - y + 1 = 0\)。
当 \(k = 2 + \sqrt{3}\) 时，方程为
\[
y - 1 = (2 + \sqrt{3})(x - 0) \quad \Rightarrow \quad y = (2 + \sqrt{3})x + 1,
\]
化为一般式：\((2 + \sqrt{3})x - y + 1 = 0\)。

## 答案
\(y = (\sqrt{3} - 2)x + 1\) 或 \(y = (2 + \sqrt{3})x + 1\)（一般式：\((\sqrt{3} - 2)x - y + 1 = 0\) 或 \((2 + \sqrt{3})x - y + 1 = 0\)）

## 总结
本题以平行线截线段长为条件，实质是考查平行线间距离与直线夹角的关系。关键步骤在于：
1. 求出平行线间的距离，并利用截得线段长得到直线与平行线的夹角；
2. 通过夹角公式建立关于斜率的方程；
3. 解绝对值方程时注意分情况讨论，避免漏解。
此类问题若涉及定点，常可从交点坐标或几何图形（直角三角形）角度切入，夹角关系是沟通已知平行线与所求直线斜率的桥梁。
