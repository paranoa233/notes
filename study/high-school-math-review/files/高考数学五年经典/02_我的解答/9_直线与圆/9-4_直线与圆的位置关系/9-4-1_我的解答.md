# 9-4-1 我的解答

对应题目：[[01_题目/9_直线与圆/9-4_直线与圆的位置关系/9-4-1.md|9-4-1]]

## 完整题目

1.(2024 河北 3 月省级联测)已知圆 \( O : {x}^{2} + \; {y}^{2} = 2 \) ，过点 \( P\left( {1,3}\right) \) 的直线 \( l \) 交圆 \( O \) 于 \( A \) ， \( B \) 两点,且 \( \frac{\left| PB\right| }{\left| PA\right| } = 2 \) ,则满足上述条件的一条直线 \( l \) 的方程为___.

## 题目勘误
无明显勘误。

## 思路分析
本题给出圆 \(O: x^{2}+y^{2}=2\) 及圆外一点 \(P(1,3)\)，过 \(P\) 的直线 \(l\) 与圆交于 \(A,B\)，且满足 \(\frac{|PB|}{|PA|}=2\)。可利用直线的参数方程，利用参数 \(t\) 的几何意义（表示从 \(P\) 到直线上点的有向距离），将线段长度之比转化为关于 \(t\) 的一元二次方程的根之比。由 \(P\) 在圆外，直线与圆相交，两交点分居 \(P\) 两侧，两根异号，由此结合韦达定理可解出直线的斜率，进而得到直线方程。

## 详细解答
设过点 \(P(1,3)\) 的直线 \(l\) 的倾斜角为 \(\alpha\)，则其参数方程为：
\[
\begin{cases}
x = 1 + t\cos\alpha, \\
y = 3 + t\sin\alpha,
\end{cases}
\]
其中参数 \(t\) 的几何意义是直线上点到 \(P\) 的有向距离，\(|t|\) 即为该点到 \(P\) 的距离。

将参数方程代入圆 \(O: x^2 + y^2 = 2\)，得：
\[
(1 + t\cos\alpha)^2 + (3 + t\sin\alpha)^2 = 2.
\]
展开并整理：
\[
1 + 2t\cos\alpha + t^2\cos^2\alpha + 9 + 6t\sin\alpha + t^2\sin^2\alpha = 2,
\]
\[
t^2(\cos^2\alpha + \sin^2\alpha) + 2t(\cos\alpha + 3\sin\alpha) + 10 - 2 = 0,
\]
\[
t^2 + 2t(\cos\alpha + 3\sin\alpha) + 8 = 0.
\]
这是一个关于 \(t\) 的一元二次方程，设其两根为 \(t_1, t_2\)，则它们分别对应交点 \(A, B\) 到 \(P\) 的有向距离。
依题意，\(|PA| = |t_1|\)，\(|PB| = |t_2|\)，且 \(\frac{|PB|}{|PA|} = 2\)，不妨设 \(|t_2| = 2|t_1|\)。

因为点 \(P(1,3)\) 满足 \(1^2+3^2=10>2\)，故 \(P\) 在圆外。过 \(P\) 的直线与圆相交于两点，这两点必在 \(P\) 的两侧，因此 \(t_1\) 与 \(t_2\) 异号。结合长度关系，可令：
\[
t_2 = -2t_1 \quad \text{或} \quad t_1 = -2t_2,
\]
两者本质对称，只需求出其中一个即可。

由韦达定理，对于方程 \(t^2 + 2t(\cos\alpha + 3\sin\alpha) + 8 = 0\)，有：
\[
t_1 + t_2 = -2(\cos\alpha + 3\sin\alpha), \quad t_1 t_2 = 8.
\]
若令 \(t_2 = -2t_1\)，代入两根之积：
\[
t_1 \cdot (-2t_1) = -2t_1^2 = 8 \quad \Rightarrow \quad t_1^2 = -4,
\]
这不可能（因为两根为实数需满足 \(\Delta \geq 0\) 且实际求值应得实数），说明此比例关系不能直接令为异号且差两倍？等等，我们异号且 \(|t_2|=2|t_1|\) 且异号，只能是 \(t_2 = -2t_1\) 或 \(t_1 = -2t_2\)。但代入两根之积为正 8，则 \(t_1 t_2 = 8\)，异号两根之积应为负，这出现了矛盾。 

检查计算：代入圆方程：
\((1+t\cos\alpha)^2+(3+t\sin\alpha)^2 = 1 + 2t\cos\alpha + t^2\cos^2\alpha + 9 + 6t\sin\alpha + t^2\sin^2\alpha = t^2 + 2t(\cos\alpha+3\sin\alpha) + 10 = 2\)。得到 \(t^2 + 2t(\cos\alpha+3\sin\alpha) + 8 = 0\)，两根之积为常数 8，恒正！这意味着无论 \(\alpha\) 为何值，只要直线与圆有两交点，对应的 \(t_1\) 与 \(t_2\) 必定同号。但我们在几何上分析“P在圆外，过P的直线与圆相交，两交点在P的两侧”这个结论是否正确？我们重新思考：若直线穿过圆，P 在圆外，是否一定两交点在 P 的两侧？假设 P 在圆外，过 P 作直线，如果直线沿着不穿过圆的方向，则无交点。要使直线与圆有两个交点，直线必须穿过圆，但 P 在圆外，那么靠近 P 的一侧有一个交点，直线继续延伸穿过圆，从圆的另一侧出来，那个交点在更远处，这两个交点相对于 P 的方向：从 P 出发沿直线方向，先碰到近的交点，继续走再碰到远的交点？如果是这样，那么两个交点都在 P 出发的同一射线上！即 P 到这两个交点的有向线段方向相同，所以 t 值同号！比如从 P 画一条割线，两个交点都在直线同一侧。这是正确的：圆外一点引割线，该点与两个交点的有向距离同号。我们通常设参数 t 时，以 P 为起点，直线方向向量为 (cosα, sinα)，那么直线上的点对应 t，t>0 表示在方向向量指向的一侧，t<0 在另一侧。如果割线的两个交点都在 P 的同一侧，则 t1, t2 同号。确实，当 P 在圆外时，过 P 的直线与圆相交，两个交点必在 P 的同一侧（因为在圆的同一区域），所以 t1 与 t2 同号！我们之前的“异号”判断是错误的。所以两根之积为正符合 t1, t2 同号。

因此，\(t_1\) 与 \(t_2\) 同号，且 \(|t_2| = 2|t_1|\)，可设 \(t_2 = 2t_1\)（若 \(t_1>0\)）或 \(t_2 = 2t_1\)（若 \(t_1<0\)，但比值 \(|t_2|/|t_1| = 2\)，同号则直接有 \(t_2 = 2t_1\) 或 \(t_1 = 2t_2\)，由于对称，只需设 \(t_2 = 2t_1\)）。

由韦达定理：
\[
t_1 + t_2 = 3t_1 = -2(\cos\alpha + 3\sin\alpha),
\]
\[
t_1 t_2 = 2t_1^2 = 8 \quad \Rightarrow \quad t_1^2 = 4, \quad t_1 = \pm 2.
\]
于是 \(t_1 = 2\) 或 \(t_1 = -2\)。
分别代入和的关系：
若 \(t_1 = 2\)，则 \(3 \times 2 = 6 = -2(\cos\alpha + 3\sin\alpha)\)，即 \(\cos\alpha + 3\sin\alpha = -3\)。
若 \(t_1 = -2\)，则 \(3 \times (-2) = -6 = -2(\cos\alpha + 3\sin\alpha)\)，即 \(\cos\alpha + 3\sin\alpha = 3\).

现在求解 \(\alpha\) （或直接求直线斜率 \(k = \tan\alpha\)）。设 \(k = \tan\alpha\)，则 \(\cos\alpha = \frac{1}{\sqrt{1+k^2}}\)，\(\sin\alpha = \frac{k}{\sqrt{1+k^2}}\)（需考虑符号，但代入方程后可平方处理）。

对于第一种情况：
\[
\cos\alpha + 3\sin\alpha = -3.
\]
两边平方并利用 \(\cos^2\alpha + \sin^2\alpha = 1\)？注意这样会引入增根，也可构造辅助角：
\[
\sqrt{1^2+3^2}\sin(\alpha+\varphi) = \sqrt{10}\sin(\alpha+\varphi) = -3,
\]
其中 \(\tan\varphi = \frac{1}{3}\)。此时 \(|\sin(\alpha+\varphi)| = \frac{3}{\sqrt{10}} < 1\)，有解。我们也可以用斜率来求：
\(\cos\alpha + 3\sin\alpha = -3\)，即 \(\frac{1+3k}{\sqrt{1+k^2}} = -3\)。两边平方：
\[
(1+3k)^2 = 9(1+k^2) \quad \Rightarrow \quad 1 + 6k + 9k^2 = 9 + 9k^2 \quad \Rightarrow \quad 6k = 8 \quad \Rightarrow \quad k = \frac{4}{3}.
\]
代回检验：若 \(k = \frac{4}{3}\)，则 \(\cos\alpha\) 与 \(\sin\alpha\) 同号？我们需要看 \(\cos\alpha + 3\sin\alpha\) 的符号。若取 \(\cos\alpha = \frac{3}{5}, \sin\alpha = \frac{4}{5}\)，则 \(\cos\alpha+3\sin\alpha = \frac{3}{5} + \frac{12}{5} = 3\)，不等于 -3。若取相反符号 \(\cos\alpha = -\frac{3}{5}, \sin\alpha = -\frac{4}{5}\)，则和为 \(-\frac{3}{5} - \frac{12}{5} = -3\)，符合！所以此时方向向量取负向，即斜率仍为 \(k = \frac{4}{3}\)。因此直线方程为 \(y - 3 = \frac{4}{3}(x - 1)\)，即 \(4x - 3y + 5 = 0\)。

对于第二种情况：
\[
\cos\alpha + 3\sin\alpha = 3.
\]
同理，\(\frac{1+3k}{\sqrt{1+k^2}} = 3\)。平方得 \((1+3k)^2 = 9(1+k^2)\)，同样化简得 \(6k=8\)，\(k=\frac{4}{3}\)。此时取正号：\(\cos\alpha = \frac{3}{5}, \sin\alpha = \frac{4}{5}\) 满足和为 3。此时直线方程仍为 \(y-3 = \frac{4}{3}(x-1)\)，即 \(4x - 3y + 5 = 0\)。两情况得出同一条直线！

咦？那难道只有一条？我们再检查：是否还有其他可能？我们设了 \(t_2 = 2t_1\)，但其实也可以设 \(t_1 = 2t_2\)，所得方程只是交换顺序，本质相同。另外，前面处理平方时可能有丢解吗？当 \(k=4/3\)，我们得到 \(\cos\alpha+3\sin\alpha = \pm 3\) 都可以，说明该直线有两个方向，对应参数 t 的正负不同，但直线是同一条。那么是否还有别的斜率？如果从比例 \(|PB|/|PA| = 2\) 出发，除了同号情况，是否可能 P 在 A, B 之间？即 P 在圆内？但 P 显然在圆外。所以只有这一种情况吗？可是题目说“一条直线”，往往有多解，写出任意一条即可。也许我们要检查还有没有其他解。再审视：我们设 \(t_2 = 2t_1\)，得到 \(t_1 = \pm 2\) 并解出唯一斜率 \(4/3\)。这没问题。

但验证：直线 \(4x-3y+5=0\) 过 P(1,3)（因为 4-9+5=0）。与圆 \(x^2+y^2=2\) 联立，求交点距离。计算：\(x = \frac{3y-5}{4}\)，代入圆：\(\left(\frac{3y-5}{4}\right)^2 + y^2 = 2\)。\(9y^2 -30y +25 + 16y^2 = 32\)。\(25y^2 -30y -7 = 0\)。解得 \(y = \frac{30 \pm \sqrt{900 + 700}}{50} = \frac{30 \pm \sqrt{1600}}{50} = \frac{30 \pm 40}{50}\)。所以 \(y_1 = \frac{70}{50} = \frac{7}{5}\)，\(y_2 = \frac{-10}{50} = -\frac{1}{5}\)。对应 \(x_1 = \frac{21/5 -5}{4} = \frac{-4/5}{4} = -\frac{1}{5}\)；\(x_2 = \frac{-3/5 -5}{4} = \frac{-28/5}{4} = -\frac{7}{5}\)。两点 \(A(-\frac{1}{5}, \frac{7}{5})\)，\(B(-\frac{7}{5}, -\frac{1}{5})\)。计算 \(PA = \sqrt{(1+1/5)^2 + (3-7/5)^2} = \sqrt{(36/25) + (64/25)} = \sqrt{100/25}=2\)；\(PB = \sqrt{(1+7/5)^2 + (3+1/5)^2} = \sqrt{(144/25) + (256/25)} = \sqrt{400/25}=4\)。确实 \(|PB|/|PA| = 2\)。因此满足。

是否还有其他直线？比如斜率不存在（垂直 x 轴）？x=1，代入圆 \(1+y^2=2\) 得 \(y=\pm 1\)，交点 (1,1) 和 (1,-1)。\(PA = |3-1|=2\)，\(PB = |3-(-1)|=4\)，比值也是 2！验证：x=1 过 P，P(1,3)，与圆 \(x^2+y^2=2\) 交于 \(y=\pm 1\)。\(|PA|=2\)，\(|PB|=4\)，比值 2。所以直线 \(x=1\) 也满足！之前我们的参数方程假设了直线有斜率，用 \(k=\tan\alpha\) 排除了斜率不存在的情况。斜率不存在时，倾斜角 \(\alpha = 90^\circ\)，参数方程为 \(x=1, y=3+t\)，代入圆得 \(1+(3+t)^2=2\)，\(t^2+6t+8=0\)，两根 \(t=-2, -4\)。长度 2 和 4，比值 2，确实也符合。所以直线 \(x=1\) 是另一条。

因此至少存在两条直线：\(x=1\) 和 \(4x-3y+5=0\)。题目说“满足上述条件的一条直线 l 的方程为___”，所以写出任意一条均可。

## 答案
\(x = 1\) 或 \(4x - 3y + 5 = 0\)（写出其中一个即可）

## 总结
本题主要考查直线与圆的位置关系，灵活运用直线的参数方程可将距离比转化为参数方程的根之比，并通过韦达定理求解。需注意点与圆的位置关系决定了参数 t 两根的符号，同时不能遗漏斜率不存在的情形。另外，比例关系可能对应多条直线，写出其中一条即可满足填空要求。
