# 9-3-11 我的解答

对应题目：[[01_题目/9_直线与圆/9-3_圆的方程/9-3-11.md|9-3-11]]

## 完整题目

3.(2022全国乙卷理，14)过四点(0，0)，(4，0)， \( \left( {-1,1}\right) ,\left( {4,2}\right) \) 中的三点的一个圆的方程为___.

## 题目勘误
无明显勘误。

## 思路分析
从给定的四个点中任选不共线的三点，即可确定一个圆。题目为填空题，答案不唯一，只需写出其中一个圆的方程。为简化计算，可选择包含直角或对称性较好的三点，如 \((0,0), (4,0), (4,2)\)。利用几何性质（直角所对弦为直径）或待定系数法均可求出圆的标准方程或一般方程。

## 详细解答

**方法一：几何法（选三点 \((0,0), (4,0), (4,2)\)）**

设 \(A(0,0), B(4,0), C(4,2)\)。

- 计算斜率：\(k_{AB} = \frac{0-0}{4-0}=0\)，\(k_{BC} = \frac{2-0}{4-4}\) 不存在，
  所以 \(AB \perp BC\)，\(\angle ABC = 90^\circ\)。
- 在圆中，\(90^\circ\) 的圆周角所对的弦为直径，故 \(AC\) 为圆的直径。
- 圆心为 \(AC\) 的中点 \(M\left(\frac{0+4}{2}, \frac{0+2}{2}\right) = (2,1)\)。
- 半径 \(r = \frac{1}{2}|AC| = \frac{1}{2}\sqrt{(4-0)^2+(2-0)^2} = \frac{1}{2}\sqrt{20} = \sqrt{5}\)。
- 圆的标准方程为：
  \[
  (x-2)^2 + (y-1)^2 = 5
  \]
  化为一般方程：
  \[
  x^2 + y^2 - 4x - 2y = 0
  \]

**方法二：待定系数法（选三点 \((0,0), (4,0), (-1,1)\)）**

设圆的一般方程为 \(x^2 + y^2 + Dx + Ey + F = 0\)。

- 代入 \((0,0)\)：\(F = 0\)。
- 代入 \((4,0)\)：\(16 + 0 + 4D + 0 + 0 = 0 \Rightarrow 4D = -16 \Rightarrow D = -4\)。
- 代入 \((-1,1)\)：\(1 + 1 - D + E = 0 \Rightarrow 2 - (-4) + E = 0 \Rightarrow E = -6\)。

圆的一般方程为：
\[
x^2 + y^2 - 4x - 6y = 0
\]

（注：选择不同三点可得不同方程，其他合理答案同样正确。）

## 答案
\(x^2 + y^2 - 4x - 2y = 0\)（或 \((x-2)^2 + (y-1)^2 = 5\)；或 \(x^2 + y^2 - 4x - 6y = 0\) 等，任写一个即可）

## 总结
处理“过若干点中的三点求圆方程”问题时，关键是找出不共线的三点。优先观察是否存在垂直关系，可利用“直径所对圆周角为直角”快速确定圆心和半径；若无特殊关系，则直接设一般方程，代入三点坐标解方程组。本题答案不唯一，写出任意一个满足条件的方程即可。
