# 9-3-3 我的解答

对应题目：[[01_题目/9_直线与圆/9-3_圆的方程/9-3-3.md|9-3-3]]

## 完整题目

3. (2024 山东聊城三模)已知圆 \( C \) 与两坐标轴及直线 \( x + y - 2 = 0 \) 都相切，且圆心在第二象限，则圆 \( C \) 的方程为 ( )

A. \( {\left( x + \sqrt{2}\right) }^{2} + {\left( y - \sqrt{2}\right) }^{2} = \sqrt{2} \)

B. \( {\left( x - \sqrt{2}\right) }^{2} + {\left( y + \sqrt{2}\right) }^{2} = 2 \)

C. \( {\left( x - \sqrt{2}\right) }^{2} + {\left( y + \sqrt{2}\right) }^{2} = \sqrt{2} \)

D. \( {\left( x + \sqrt{2}\right) }^{2} + {\left( y - \sqrt{2}\right) }^{2} = 2 \)

## 题目勘误
无明显勘误。
## 思路分析
圆与两坐标轴及一条已知直线都相切，且圆心在第二象限。由圆心在第二象限以及与两坐标轴相切，可设圆心坐标为 \((-r,r)\)，半径为 \(r\)。再利用直线与圆相切的条件（圆心到直线的距离等于半径）求出 \(r\)，即可得到圆的方程，最后比对选项。
## 详细解答
设圆 \(C\) 的圆心为 \((a,b)\)，半径为 \(r\)。  
因为圆心在第二象限，所以 \(a<0,\;b>0\)。  
圆与 \(x\) 轴（\(y=0\)）相切，则 \(r = |b| = b\)；与 \(y\) 轴（\(x=0\)）相切，则 \(r = |a| = -a\)。  
因此 \(a = -r,\; b = r\)，圆心坐标为 \((-r, r)\)。  

圆与直线 \(x + y - 2 = 0\) 相切，圆心到直线的距离等于半径 \(r\)：
\[
\frac{|-r + r - 2|}{\sqrt{1^2+1^2}} = r
\]
\[
\frac{|-2|}{\sqrt{2}} = r \quad \Rightarrow \quad \frac{2}{\sqrt{2}} = r \quad \Rightarrow \quad r = \sqrt{2}.
\]

于是圆心为 \((-\sqrt{2},\; \sqrt{2})\)，半径 \(r = \sqrt{2}\)。  
圆的标准方程为：
\[
(x + \sqrt{2})^2 + (y - \sqrt{2})^2 = (\sqrt{2})^2 = 2.
\]

比对选项，与选项 D 一致。
## 答案
D
## 总结
处理圆与坐标轴相切的问题，关键是由圆心所在象限确定坐标符号，然后利用“圆心到坐标轴的距离等于半径”建立 \(a,b,r\) 的关系。对于与其他直线相切的条件，用点到直线距离公式列方程即可顺利求解。
