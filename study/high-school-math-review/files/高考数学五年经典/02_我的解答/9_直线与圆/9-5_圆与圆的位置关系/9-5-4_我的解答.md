# 9-5-4 我的解答

对应题目：[[01_题目/9_直线与圆/9-5_圆与圆的位置关系/9-5-4.md|9-5-4]]

## 完整题目

4. (多选题,2024 安徽合肥二模) 已知圆 \( O : {x}^{2} + {y}^{2} \; = 1 \) ,圆 \( C : {\left( x - a\right) }^{2} + {\left( y - 1\right) }^{2} = 4, a \in  \mathbf{R} \) ,则(   )

A. 两圆的圆心距 \( \left| {OC}\right| \) 的最小值为 1

B. 若圆 \( O \) 与圆 \( C \) 相切,则 \( a =  \pm  2\sqrt{2} \)

C. 若圆 \( O \) 与圆 \( C \) 恰有两条公切线,则 \( - 2\sqrt{2} < a \; < 2\sqrt{2} \)

D. 若圆 \( O \) 与圆 \( C \) 相交,则公共弦长的最大值为 2

## 题目勘误
无明显勘误。

## 思路分析
本题综合考查圆与圆的位置关系及其几何性质。先写出两圆的圆心与半径，计算圆心距 \(d = |OC| = \sqrt{a^2+1}\)。然后根据选项逐一利用两圆位置关系的判定条件（相切、相交、公切线条数）以及公共弦长的计算公式进行判断，特别注意内切、外切以及 \(a=0\) 等临界情况。

## 详细解答

由已知：
圆 \(O: x^2 + y^2 = 1\)，圆心 \(O(0,0)\)，半径 \(r_1 = 1\)；
圆 \(C: (x-a)^2 + (y-1)^2 = 4\)，圆心 \(C(a,1)\)，半径 \(r_2 = 2\)；
圆心距 \(d = |OC| = \sqrt{a^2 + 1}\)。

**A 选项**：求 \(|OC|\) 的最小值。
由于 \(a^2 \ge 0\)，有 \(d = \sqrt{a^2 + 1} \ge \sqrt{0+1} = 1\)，
当 \(a = 0\) 时取等号。
故 A 正确。

**B 选项**：若圆 \(O\) 与圆 \(C\) 相切，求 \(a\) 的值。
两圆相切分为外切与内切：
外切时 \(d = r_1 + r_2 = 3\)，即 \(\sqrt{a^2 + 1} = 3\)，解得 \(a^2 = 8\)，\(a = \pm 2\sqrt{2}\)；
内切时 \(d = |r_2 - r_1| = 1\)，即 \(\sqrt{a^2 + 1} = 1\)，解得 \(a = 0\)。
因此两圆相切时，\(a = \pm 2\sqrt{2}\) 或 \(a = 0\)。
选项 B 只列出了外切情况，遗漏了 \(a=0\)，故 B 错误。

**C 选项**：若圆 \(O\) 与圆 \(C\) 恰有两条公切线，求 \(a\) 的范围。
两圆恰有两条公切线，等价于两圆相交。
相交的条件为 \(|r_2 - r_1| < d < r_1 + r_2\)，即：
\(1 < \sqrt{a^2 + 1} < 3\)。
平方得 \(1 < a^2 + 1 < 9\)，即 \(0 < a^2 < 8\)，
解得 \(-2\sqrt{2} < a < 2\sqrt{2}\) 且 \(a \neq 0\)。
而选项 C 给出的范围是 \(-2\sqrt{2} < a < 2\sqrt{2}\)，它包含了 \(a = 0\)。当 \(a = 0\) 时，\(d = 1\)，两圆内切，公切线只有 1 条，与题意不符。
故 C 错误。

**D 选项**：若圆 \(O\) 与圆 \(C\) 相交，求公共弦长的最大值。
两圆方程作差得公共弦所在直线 \(l\)：
\((x^2 + y^2 - 1) - \big[(x-a)^2 + (y-1)^2 - 4\big] = 0\)
化简得 \(2ax + 2y - a^2 + 2 = 0\)，即 \(ax + y - \big(\frac{a^2}{2} - 1\big) = 0\)。
圆 \(O\) 的圆心到直线 \(l\) 的距离为
\(d_1 = \frac{\big| a\cdot 0 + 0 - \big(\frac{a^2}{2}-1\big) \big|}{\sqrt{a^2+1}} = \frac{\big|1 - \frac{a^2}{2}\big|}{\sqrt{a^2+1}}\)。
则公共弦长
\(L = 2\sqrt{r_1^2 - d_1^2} = 2\sqrt{1 - d_1^2} \le 2 \times 1 = 2\)，
当 \(d_1 = 0\) 时取最大值 \(2\)。
令 \(d_1 = 0\)，得 \(1 - \frac{a^2}{2} = 0\)，即 \(a = \pm \sqrt{2}\)。
此时 \(d = \sqrt{2 + 1} = \sqrt{3}\)，满足相交条件 \(1 < \sqrt{3} < 3\)，故最大弦长可以达到 2。
故 D 正确。

综上，正确选项为 A、D。

## 答案
AD

## 总结
处理圆与圆的位置关系问题，要抓住圆心距 \(d\) 与两圆半径 \(R, r\) 的数量关系，并注意区分内切、外切以及“相交”是否需要排除内切。在求公共弦长时，可化为圆心到弦所在直线的距离，利用半径、半弦长与弦心距构成的直角三角形求解。临界情况（如 \(a=0\)）往往是易错点，需单独验证。
