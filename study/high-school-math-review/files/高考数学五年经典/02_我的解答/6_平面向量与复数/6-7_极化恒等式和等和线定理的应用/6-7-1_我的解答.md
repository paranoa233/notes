# 6-7-1 我的解答

对应题目：[[01_题目/6_平面向量与复数/6-7_极化恒等式和等和线定理的应用/6-7-1.md|6-7-1]]

## 完整题目

1.(2024 辽宁高考扣题卷(二)已知平行四边形 \( {ABCD} \) ,点 \( P \) 在 \( \bigtriangleup {BCD} \) 的内部 (不含边界),则 \( \overrightarrow{AP} \) 可能满足的关系式为( )

A. \( \overrightarrow{AP} = \frac{1}{5}\overrightarrow{AB} + \frac{3}{5}\overrightarrow{AD} \)

B. \( \overrightarrow{AP} = \frac{1}{4}\overrightarrow{AB} + \frac{3}{4}\overrightarrow{AD} \)

C. \( \overrightarrow{AP} = \frac{2}{3}\overrightarrow{AB} + \frac{3}{4}\overrightarrow{AD} \)

D. \( \overrightarrow{AP} = \frac{2}{3}\overrightarrow{AB} + \frac{4}{3}\overrightarrow{AD} \)

## 题目勘误
无明显勘误。

## 思路分析
在平行四边形 \(ABCD\) 中，以 \(A\) 为原点，\(\overrightarrow{AB}\) 和 \(\overrightarrow{AD}\) 为一组基底建立仿射坐标系。则平面内任意一点 \(P\) 对应的向量 \(\overrightarrow{AP} = x\overrightarrow{AB} + y\overrightarrow{AD}\) 可由有序实数对 \((x,y)\) 唯一确定。  
点 \(P\) 在 \(\triangle BCD\) 的内部（不含边界），等价于坐标 \((x,y)\) 满足：
\[
0 < x < 1,\quad 0 < y < 1,\quad x + y > 1.
\]
（理由：\(B(1,0), C(1,1), D(0,1)\)，\(\triangle BCD\) 是平行四边形去掉 \(\triangle ABD\) 的区域，其内部不含边界即为上述不等式组。）  

因此只需逐一检验四个选项所对应的坐标 \((x,y)\) 是否满足上述条件，即可得出正确选项。这一方法也体现了“等和线定理”的应用：\(x+y\) 的值决定了点 \(P\) 所在直线与 \(BD\) 的平行关系，\(\triangle BCD\) 内部恰好对应 \(x+y>1\) 且 \(x,y\in(0,1)\)。

## 详细解答
以 \(A\) 为原点，设基底 \(\overrightarrow{AB}, \overrightarrow{AD}\)，则点 \(P\) 的坐标为 \((x,y)\)，其中 \(\overrightarrow{AP} = x\overrightarrow{AB} + y\overrightarrow{AD}\)。顶点坐标为：
\[
A(0,0),\quad B(1,0),\quad C(1,1),\quad D(0,1).
\]
\(\triangle BCD\) 的边分别为：
- \(BD\)：\(x+y=1\)（\(0 \le x \le 1\)）；
- \(BC\)：\(x=1\)（\(0 \le y \le 1\)）；
- \(CD\)：\(y=1\)（\(0 \le x \le 1\)）。

点 \(P\) 在 \(\triangle BCD\) 内部（不含边界）时，需同时满足：
\[
0 < x < 1,\quad 0 < y < 1,\quad x + y > 1.
\]

分别代入各选项：

**选项 A**：\(\overrightarrow{AP} = \frac{1}{5}\overrightarrow{AB} + \frac{3}{5}\overrightarrow{AD}\)  
此时 \(x = \frac{1}{5},\ y = \frac{3}{5}\)，\(x+y = \frac{4}{5} < 1\)，不满足 \(x+y>1\)。  
故选项 A 错误。

**选项 B**：\(\overrightarrow{AP} = \frac{1}{4}\overrightarrow{AB} + \frac{3}{4}\overrightarrow{AD}\)  
此时 \(x = \frac{1}{4},\ y = \frac{3}{4}\)，\(x+y = 1\)。点 \(P\) 恰好落在 \(BD\) 边界上，不满足“不含边界”。  
故选项 B 错误。

**选项 C**：\(\overrightarrow{AP} = \frac{2}{3}\overrightarrow{AB} + \frac{3}{4}\overrightarrow{AD}\)  
此时 \(x = \frac{2}{3},\ y = \frac{3}{4}\)。计算：
\[
0 < \frac{2}{3} < 1,\quad 0 < \frac{3}{4} < 1,\quad x+y = \frac{2}{3} + \frac{3}{4} = \frac{17}{12} > 1.
\]
全部满足条件，点 \(P\) 在 \(\triangle BCD\) 内部。  
故选项 C 正确。

**选项 D**：\(\overrightarrow{AP} = \frac{2}{3}\overrightarrow{AB} + \frac{4}{3}\overrightarrow{AD}\)  
此时 \(y = \frac{4}{3} > 1\)，点 \(P\) 的纵坐标已超出平行四边形范围（落在 \(CD\) 上方），不在 \(\triangle BCD\) 内部。  
故选项 D 错误。

## 答案
C

## 总结
本题以平行四边形为背景，考查平面向量基本定理及等和线定理的应用。解题的关键在于建立坐标系，将向量系数转化为点的坐标，并结合 \(\triangle BCD\) 的内部区域对应的不等式组进行验证。  
对于选择题，可采用“先定性排除，再定量计算”的策略。等和线 \(x+y\) 的值为常数，\(\triangle BCD\) 内部对应 \(x+y>1\)（不含边界），抓住这一核心可快速判断。注意边界情形（如 \(x+y=1\) 或 \(x=1\) 或 \(y=1\)）需要排除。
