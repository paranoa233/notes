# 6-5-26 我的解答

对应题目：[[01_题目/6_平面向量与复数/6-5_平面向量数量积的运算/6-5-26.md|6-5-26]]

## 完整题目

(2023 全国甲卷理, 4)向量 \( \left| \mathbf{a}\right|  = \left| \mathbf{b}\right|  = 1,\left| \mathbf{c}\right| \) = 3,且 \( a + b + c = 0 \) ,则 \( \cos \langle a - c, b - c\rangle  = \)

( )

A. \( - \frac{1}{5} \) B. \( - \frac{2}{5} \) C. \( \frac{2}{5} \) D. \( \frac{4}{5} \)

## 题目勘误
原文“\(\left| \mathbf{c}\right| = 3\)”存在条件矛盾：已知 \(\mathbf{a}+\mathbf{b}+\mathbf{c}=\mathbf{0}\)，则 \(\mathbf{c} = -(\mathbf{a}+\mathbf{b})\)，从而 \(|\mathbf{c}| = |\mathbf{a}+\mathbf{b}| \le |\mathbf{a}|+|\mathbf{b}| = 2\)，不可能等于 3。根据 2023 年全国甲卷理科真题，此处应为 \(|\mathbf{c}| = \sqrt{2}\)，推测为 OCR 识别错误。以下解答基于更正后的数据 \(|\mathbf{c}| = \sqrt{2}\)，建议核实原题。

## 思路分析
由条件 \(\mathbf{a}+\mathbf{b}+\mathbf{c}=\mathbf{0}\) 可将 \(\mathbf{c}\) 用 \(\mathbf{a},\mathbf{b}\) 表示，进而通过模长条件求出 \(\mathbf{a}\cdot\mathbf{b}\)。再将 \(\mathbf{a}-\mathbf{c}\) 和 \(\mathbf{b}-\mathbf{c}\) 转化为 \(\mathbf{a},\mathbf{b}\) 的线性组合，利用数量积计算夹角余弦。由于 \(|\mathbf{c}|\) 的数据直接决定 \(\mathbf{a}\cdot\mathbf{b}\) 的值，须先修正矛盾数据后才能完成计算。

## 详细解答
**第一步：由已知模长求 \(\mathbf{a}\cdot\mathbf{b}\)（使用更正数据）**  
由 \(\mathbf{a}+\mathbf{b}+\mathbf{c}=\mathbf{0}\) 得 \(\mathbf{c} = -(\mathbf{a}+\mathbf{b})\)。  
两边取模长的平方：  
\[
|\mathbf{c}|^2 = |\mathbf{a}+\mathbf{b}|^2 = |\mathbf{a}|^2 + |\mathbf{b}|^2 + 2\mathbf{a}\cdot\mathbf{b}.
\]  
代入 \(|\mathbf{a}|=|\mathbf{b}|=1\) 及更正后的 \(|\mathbf{c}|=\sqrt{2}\)：  
\[
(\sqrt{2})^2 = 1^2 + 1^2 + 2\mathbf{a}\cdot\mathbf{b} \;\Longrightarrow\; 2 = 2 + 2\mathbf{a}\cdot\mathbf{b} \;\Longrightarrow\; \mathbf{a}\cdot\mathbf{b} = 0.
\]

**第二步：表示目标向量 \(\mathbf{a}-\mathbf{c}\) 与 \(\mathbf{b}-\mathbf{c}\)**  
将 \(\mathbf{c}=-(\mathbf{a}+\mathbf{b})\) 代入：  
\[
\mathbf{a}-\mathbf{c} = \mathbf{a} - (-(\mathbf{a}+\mathbf{b})) = 2\mathbf{a}+\mathbf{b},
\]  
\[
\mathbf{b}-\mathbf{c} = \mathbf{b} - (-(\mathbf{a}+\mathbf{b})) = \mathbf{a}+2\mathbf{b}.
\]

**第三步：计算两向量的数量积**  
\[
(2\mathbf{a}+\mathbf{b})\cdot(\mathbf{a}+2\mathbf{b}) = 2\mathbf{a}\cdot\mathbf{a} + 4\mathbf{a}\cdot\mathbf{b} + \mathbf{b}\cdot\mathbf{a} + 2\mathbf{b}\cdot\mathbf{b}
\]  
\[
= 2|\mathbf{a}|^2 + 5\mathbf{a}\cdot\mathbf{b} + 2|\mathbf{b}|^2.
\]  
代入 \(|\mathbf{a}|=|\mathbf{b}|=1\) 和 \(\mathbf{a}\cdot\mathbf{b}=0\)：  
\[
(2\mathbf{a}+\mathbf{b})\cdot(\mathbf{a}+2\mathbf{b}) = 2\times 1 + 5\times 0 + 2\times 1 = 4.
\]

**第四步：计算两向量的模长**  
\[
|2\mathbf{a}+\mathbf{b}|^2 = (2\mathbf{a}+\mathbf{b})\cdot(2\mathbf{a}+\mathbf{b}) = 4|\mathbf{a}|^2 + 4\mathbf{a}\cdot\mathbf{b} + |\mathbf{b}|^2 = 4+0+1 = 5,
\]  
\[
|\mathbf{a}+2\mathbf{b}|^2 = |\mathbf{a}|^2 + 4\mathbf{a}\cdot\mathbf{b} + 4|\mathbf{b}|^2 = 1+0+4 = 5.
\]  
所以 \(|2\mathbf{a}+\mathbf{b}| = \sqrt{5}\)，\(|\mathbf{a}+2\mathbf{b}| = \sqrt{5}\)。

**第五步：求夹角余弦**  
\[
\cos\langle \mathbf{a}-\mathbf{c}, \mathbf{b}-\mathbf{c}\rangle = \frac{(2\mathbf{a}+\mathbf{b})\cdot(\mathbf{a}+2\mathbf{b})}{|2\mathbf{a}+\mathbf{b}|\,|\mathbf{a}+2\mathbf{b}|} = \frac{4}{\sqrt{5}\times\sqrt{5}} = \frac{4}{5}.
\]

## 答案
D

## 总结
本题核心是利用“和为零向量”将未知向量用已知模长的向量线性表示，通过模长平方建立数量积方程。需注意题目数据必须自洽（满足三角不等式）。修正数据后，计算目标向量的数量积与模长即可得到夹角余弦。常见错误：忽略数据矛盾强行计算，或转化目标向量时符号出错。
