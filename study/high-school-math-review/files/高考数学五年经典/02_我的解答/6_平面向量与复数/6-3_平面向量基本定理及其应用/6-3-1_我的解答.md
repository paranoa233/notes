# 6-3-1 我的解答

对应题目：[[01_题目/6_平面向量与复数/6-3_平面向量基本定理及其应用/6-3-1.md|6-3-1]]

## 完整题目

1.(2024 广西普通高等学校招生押题卷(一))在 \( \bigtriangleup {ABC} \) 中, \( \overrightarrow{AB} = 4\overrightarrow{AD},\overrightarrow{CE} = 2\overrightarrow{ED} \) . 若 \( \overrightarrow{BC} = \lambda \overrightarrow{AE} \; + \mu \overrightarrow{CD} \) ,则 ( )

A. \( \lambda  + \mu  = 5 \) B. \( \lambda  - \mu  = 1 \)

C. \( {\lambda \mu } = 6 \) D. \( \frac{\lambda }{\mu } = 3 \)

## 题目勘误
无明显勘误。

## 思路分析
本题条件给出两个向量共线关系，目标是求 \(\overrightarrow{BC}\) 用 \(\overrightarrow{AE}\) 与 \(\overrightarrow{CD}\) 表示的系数 \(\lambda,\mu\)。
由于涉及三角形中的向量线性表示，可选取一组平面向量基底（如 \(\overrightarrow{AB},\overrightarrow{AC}\)），将已知关系转化为基底下坐标的线性方程组，解出 \(\lambda,\mu\) 后逐个验证选项。

## 详细解答
选取基底：设 \(\overrightarrow{AB} = \mathbf{b},\ \overrightarrow{AC} = \mathbf{c}\)。

由条件 \(\overrightarrow{AB} = 4\overrightarrow{AD}\) 得
\[
\overrightarrow{AD} = \frac{1}{4}\overrightarrow{AB} = \frac{1}{4}\mathbf{b}.
\]
于是
\[
\overrightarrow{CD} = \overrightarrow{AD} - \overrightarrow{AC} = \frac{1}{4}\mathbf{b} - \mathbf{c}.
\]

对条件 \(\overrightarrow{CE} = 2\overrightarrow{ED}\)，由 \(\overrightarrow{ED} = \overrightarrow{CD} - \overrightarrow{CE}\) 代入得
\[
\overrightarrow{CE} = 2(\overrightarrow{CD} - \overrightarrow{CE}) \quad\Longrightarrow\quad 3\overrightarrow{CE} = 2\overrightarrow{CD} \quad\Longrightarrow\quad \overrightarrow{CE} = \frac{2}{3}\overrightarrow{CD}.
\]
从而
\[
\overrightarrow{AE} = \overrightarrow{AC} + \overrightarrow{CE}
= \mathbf{c} + \frac{2}{3}\left(\frac{1}{4}\mathbf{b} - \mathbf{c}\right)
= \frac{1}{6}\mathbf{b} + \frac{1}{3}\mathbf{c}.
\]

又有
\[
\overrightarrow{BC} = \overrightarrow{AC} - \overrightarrow{AB} = \mathbf{c} - \mathbf{b}.
\]

由题意 \(\overrightarrow{BC} = \lambda \overrightarrow{AE} + \mu \overrightarrow{CD}\)，将上述向量代入：
\[
\mathbf{c} - \mathbf{b}
= \lambda\left(\frac{1}{6}\mathbf{b} + \frac{1}{3}\mathbf{c}\right)
+ \mu\left(\frac{1}{4}\mathbf{b} - \mathbf{c}\right)
= \left(\frac{\lambda}{6} + \frac{\mu}{4}\right)\mathbf{b}
+ \left(\frac{\lambda}{3} - \mu\right)\mathbf{c}.
\]

因为 \(\mathbf{b},\mathbf{c}\) 不共线，比较对应系数得方程组：
\[
\begin{cases}
\dfrac{\lambda}{6} + \dfrac{\mu}{4} = -1, \\[6pt]
\dfrac{\lambda}{3} - \mu = 1.
\end{cases}
\]

整理第一个方程为 \(2\lambda + 3\mu = -12\) （乘以12），  
第二个方程为 \(\lambda - 3\mu = 3\) （乘以3）。

两式相加：
\[
3\lambda = -9 \quad\Longrightarrow\quad \lambda = -3.
\]
代回 \(\lambda - 3\mu = 3\)：
\[
-3 - 3\mu = 3 \quad\Longrightarrow\quad \mu = -2.
\]

因此 \(\lambda = -3,\ \mu = -2\)。检验各选项：
- A：\(\lambda + \mu = -5 \neq 5\)，错误；
- B：\(\lambda - \mu = -1 \neq 1\)，错误；
- C：\(\lambda\mu = (-3)\times(-2) = 6\)，正确；
- D：\(\dfrac{\lambda}{\mu} = \dfrac{-3}{-2} = \dfrac{3}{2} \neq 3\)，错误。

正确选项为 C。

## 答案
C

## 总结
解决此类平面向量线性表示问题，通常选定两个不共线向量作为基底，将所有相关向量用基底表示，再比较系数得到方程（组），解出待定系数后验证选项。本题中 \(\lambda,\mu\) 均为负数，数值计算需小心符号。
