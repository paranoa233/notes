# 6-3-7 我的解答

对应题目：[[01_题目/6_平面向量与复数/6-3_平面向量基本定理及其应用/6-3-7.md|6-3-7]]

## 完整题目

7. (多选题, 2021 湖北八市 3 月联考) 已知在 \( \bigtriangleup {ABC} \) 中, \( D \) 为边 \( {AC} \) 上的一点,且满足 \( \overrightarrow{AD} = \; \frac{1}{3}\overrightarrow{AC} \) ,若 \( P \) 为边 \( {BD} \) 上的一点,且满足 \( \overrightarrow{AP} = \)

\( m\overrightarrow{AB} + n\overrightarrow{AC}\left( {m > 0, n > 0}\right) \) ,则下列结论正确的是 ( )

A. \( m + {2n} = 1 \)

B. \( {mn} \) 的最大值为 \( \frac{1}{12} \)

C. \( \frac{4}{m} + \frac{1}{n} \) 的最小值为 \( 6 + 4\sqrt{2} \)

D. \( {m}^{2} + 9{n}^{2} \) 的最小值为 \( \frac{1}{2} \)

## 题目勘误
无明显勘误。

## 思路分析
本题以三角形为载体，考查平面向量基本定理与共线定理，并结合基本不等式求最值。将 \(\overrightarrow{AB}, \overrightarrow{AC}\) 作为平面的一组基底，由点 \(D\) 在 \(AC\) 上及 \(P\) 在 \(BD\) 上，利用向量共线条件建立 \(m\) 与 \(n\) 的等量关系，再代入各选项逐一分析判断。

## 详细解答

依题意，在 \(\triangle ABC\) 中，\(D\) 在边 \(AC\) 上，且 \(\overrightarrow{AD} = \frac{1}{3}\overrightarrow{AC}\)。  
\(P\) 为边 \(BD\) 上一点，故 \(B,P,D\) 三点共线。由平面向量基本定理，存在实数 \(t \in [0,1]\)，使得
\[
\overrightarrow{AP} = (1-t)\overrightarrow{AB} + t\overrightarrow{AD}.
\]
代入 \(\overrightarrow{AD} = \frac{1}{3}\overrightarrow{AC}\)，得
\[
\overrightarrow{AP} = (1-t)\overrightarrow{AB} + \frac{t}{3}\overrightarrow{AC}.
\]
对比已知表达式 \(\overrightarrow{AP} = m\overrightarrow{AB} + n\overrightarrow{AC}\) (\(m>0,n>0\))，可得
\[
m = 1-t,\quad n = \frac{t}{3}.
\]
消去参数 \(t\)，得到 \(m,n\) 满足的等式：
\[
m + 3n = 1 \quad (m>0,\, n>0).
\]

**选项 A：** 由上述推导得 \(m + 3n = 1\)，并非 \(m + 2n = 1\)，故 A 错误。

**选项 B：** 求 \(mn\) 的最大值。  
已知 \(m + 3n = 1\) 且 \(m,n>0\)，由基本不等式：
\[
1 = m + 3n \ge 2\sqrt{3mn} \implies 3mn \le \frac{1}{4} \implies mn \le \frac{1}{12}.
\]
当且仅当 \(m = 3n = \frac{1}{2}\)，即 \(m = \frac{1}{2}, n = \frac{1}{6}\) 时取等号。最大值为 \(\frac{1}{12}\)，故 B 正确。

**选项 C：** 求 \(\frac{4}{m} + \frac{1}{n}\) 的最小值。  
利用 \(m + 3n = 1\) 进行“1”的代换：
\[
\frac{4}{m} + \frac{1}{n} = \left( \frac{4}{m} + \frac{1}{n} \right)(m + 3n) = 4 + \frac{12n}{m} + \frac{m}{n} + 3 = 7 + \frac{12n}{m} + \frac{m}{n}.
\]
由基本不等式，\(\frac{12n}{m} + \frac{m}{n} \ge 2\sqrt{12} = 4\sqrt{3}\)，  
因此 \(\frac{4}{m} + \frac{1}{n} \ge 7 + 4\sqrt{3}\)。  
等号当 \(\frac{12n}{m} = \frac{m}{n}\) 即 \(m = 2\sqrt{3}n\) 时可取，符合条件。选项给出的 \(6+4\sqrt{2}\) 不正确，故 C 错误。

**选项 D：** 求 \(m^2 + 9n^2\) 的最小值。  
由柯西不等式：
\[
(m^2 + (3n)^2)(1^2 + 1^2) \ge (m + 3n)^2 = 1,
\]
即 \(2(m^2 + 9n^2) \ge 1\)，所以 \(m^2 + 9n^2 \ge \frac{1}{2}\)。  
当且仅当 \(m = 3n = \frac{1}{2}\)，即 \(m = \frac{1}{2}, n = \frac{1}{6}\) 时取等号，与 B 选项等号条件一致。最小值为 \(\frac{1}{2}\)，故 D 正确。

综上所述，正确的选项为 B 和 D。

## 答案
BD

## 总结
解决此类问题的关键在于：  
1. 选择合适的基底（此处选 \(\overrightarrow{AB}, \overrightarrow{AC}\)），将动点满足的共线条件转化为向量系数的线性关系，得到 \(m + 3n = 1\)。  
2. 结合约束条件 \(m,n>0\)，利用基本不等式或柯西不等式求各代数式的最值，需注意等号成立的条件是否满足。  
常见易错点为混淆系数关系（如误写成 \(m+2n=1\)），以及误用不等式求最值时忽略取等条件。
