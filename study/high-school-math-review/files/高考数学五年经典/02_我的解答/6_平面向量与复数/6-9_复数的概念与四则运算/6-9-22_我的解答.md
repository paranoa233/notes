# 6-9-22 我的解答

对应题目：[[01_题目/6_平面向量与复数/6-9_复数的概念与四则运算/6-9-22.md|6-9-22]]

## 完整题目

6.(2022全国乙卷理，2)已知 \( z = 1 - 2\mathrm{i} \) ，且 \( z + a\bar{z} \; + b = 0 \) ，其中 \( a \) ， \( b \) 为实数，则 ( )

A. \( a = 1, b =  - 2 \) B. \( a =  - 1, b = 2 \)

C. \( a = 1, b = 2 \) D. \( a =  - 1, b =  - 2 \)

## 题目勘误
无明显勘误。
## 思路分析
已知 \(z\) 的表达式以及含参数 \(a,b\) 的等式 \(z + a\bar{z} + b = 0\)，且 \(a,b\) 为实数。思路是先求出 \(z\) 的共轭复数 \(\bar{z}\)，代入等式，将左侧整理成“实部 + 虚部 \(\mathrm{i}\)”的形式。利用复数等于零的充要条件（实部与虚部分别为零），得到关于 \(a,b\) 的二元一次方程组，解之即可确定选项。
## 详细解答
1. 由 \(z = 1 - 2\mathrm{i}\)，得其共轭复数  
   \[
   \bar{z} = 1 + 2\mathrm{i}.
   \]

2. 代入等式 \(z + a\bar{z} + b = 0\)：  
   \[
   (1 - 2\mathrm{i}) + a(1 + 2\mathrm{i}) + b = 0.
   \]

3. 展开并分离实部和虚部：  
   \[
   1 - 2\mathrm{i} + a + 2a\mathrm{i} + b = 0,
   \]  
   \[
   (1 + a + b) + (-2 + 2a)\mathrm{i} = 0.
   \]

4. 因为 \(a,b\) 为实数，根据复数相等的定义，实部与虚部必须同时为零，得到方程组：
   \[
   \begin{cases}
   1 + a + b = 0, \\
   -2 + 2a = 0.
   \end{cases}
   \]

5. 解方程组：  
   由 \(-2 + 2a = 0\) 得 \(2a = 2\)，即 \(a = 1\)。  
   将 \(a = 1\) 代入 \(1 + a + b = 0\)，得 \(1 + 1 + b = 0\)，所以 \(b = -2\)。

6. 因此 \(a = 1\)，\(b = -2\)，对应选项 A。
## 答案
A
## 总结
解此类含参数的复数等式问题，常用“化虚为实”的思想：首先求出相关共轭复数，代入整理，再利用复数等于零的充要条件将问题转化为实方程组求解。注意运算时准确合并实部与虚部，避免符号错误。
