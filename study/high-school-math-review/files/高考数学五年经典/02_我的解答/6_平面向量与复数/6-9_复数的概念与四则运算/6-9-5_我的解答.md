# 6-9-5 我的解答

对应题目：[[01_题目/6_平面向量与复数/6-9_复数的概念与四则运算/6-9-5.md|6-9-5]]

## 完整题目

5.(2024 山东临沂一模)若虚数单位 i 是关于 \( x \) 的方程 \( a{x}^{3} + b{x}^{2} + {bx} + 1 = 0\left( {a, b \in  \mathbf{R}}\right) \) 的一个根,则 \( \left| {a + b\mathrm{i}}\right|  = \) ( )

A. 0 B. 1 C. \( \sqrt{2} \) D. 2

## 题目勘误
无明显勘误。
## 思路分析
因为 \(i\) 是实系数方程 \(ax^3+bx^2+bx+1=0\) 的根，可将 \(x=i\) 代入方程，利用复数相等的条件（实部与虚部分别为零）得到关于实数 \(a,b\) 的方程组。解出 \(a,b\) 后，计算复数 \(a+bi\) 的模 \(\left|a+bi\right|=\sqrt{a^2+b^2}\)。
## 详细解答
将 \(x=i\) 代入方程 \(ax^3+bx^2+bx+1=0\) 中：
\[
a i^3 + b i^2 + b i + 1 = 0
\]
由 \(i^2=-1\)，\(i^3=-i\)，得
\[
a(-i) + b(-1) + b i + 1 = 0
\]
整理得
\[
-ai - b + bi + 1 = 0
\]
\[
(1 - b) + (b - a)i = 0
\]
因为 \(a,b \in \mathbf{R}\)，根据复数等于零的充要条件，实部与虚部均为零：
\[
\begin{cases}
1 - b = 0 \\
b - a = 0
\end{cases}
\]
解得
\[
b = 1,\quad a = 1
\]
所以
\[
|a + bi| = |1 + i| = \sqrt{1^2 + 1^2} = \sqrt{2}
\]
## 答案
C. \(\sqrt{2}\)
## 总结
处理实系数方程虚根问题的基本方法是将根直接代入，利用复数相等的条件转化为实方程组求解。本题中代入 \(x=i\) 后，通过实部与虚部分别为零可迅速解出 \(a,b\)，进而求模。注意计算 \(i\) 的各次幂时不要出错。
