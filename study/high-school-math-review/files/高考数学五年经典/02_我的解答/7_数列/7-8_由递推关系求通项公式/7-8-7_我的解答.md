# 7-8-7 我的解答

对应题目：[[01_题目/7_数列/7-8_由递推关系求通项公式/7-8-7.md|7-8-7]]

## 完整题目

7.(2022 安徽马鞍山模拟)已知整数数列 \( \left\{  {b}_{n}\right\} \) 满足 \( {4}^{n} - \frac{1}{7} < {b}_{n + 1} - {b}_{n - 1} < {4}^{n} + \frac{1}{7},{b}_{2} = 4 \) ,则 \( {b}_{2022} = \) ( )

A. \( \frac{{2}^{2022} - {64}}{3} \) B. \( \frac{{4}^{2023} - 4}{15} \)

C. \( \frac{{2}^{1018} - {37}}{3} \) D. \( \frac{{4}^{2022} - {56}}{15} \)

## 题目勘误
无明显勘误。

## 思路分析
已知数列 \(\{b_n\}\) 是整数数列，且满足不等式 \(4^n - \frac{1}{7} < b_{n+1} - b_{n-1} < 4^n + \frac{1}{7}\)。区间长度为 \(\frac{2}{7}<1\)，而 \(b_{n+1} - b_{n-1}\) 为整数，\(4^n\) 也为整数，故必有 \(b_{n+1} - b_{n-1} = 4^n\)。由此得到隔项递推关系。注意到目标项 \(b_{2022}\) 下标为偶数，可单独列出偶数项的递推，并通过累加法结合等比数列求和求出 \(b_{2022}\)。

## 详细解答
因为 \(b_n \in \mathbb{Z}\)，所以 \(b_{n+1} - b_{n-1} \in \mathbb{Z}\)。  
又 \(4^n \in \mathbb{Z}\)，且
\[
4^n - \frac{1}{7} < b_{n+1} - b_{n-1} < 4^n + \frac{1}{7},
\]
区间长度 \(\left(4^n + \frac{1}{7}\right) - \left(4^n - \frac{1}{7}\right) = \frac{2}{7} < 1\)，  
因此该区间内至多只有一个整数，这个整数只能是 \(4^n\)。故对 \(n \geqslant 2\)，有
\[
b_{n+1} - b_{n-1} = 4^n.
\]
令 \(n = 2k-1\)（其中 \(k \geqslant 2\)），得
\[
b_{2k} - b_{2k-2} = 4^{2k-1}.
\]
这表明数列的偶数项满足一阶差分递推。已知 \(b_2 = 4\)，则对 \(m \geqslant 2\)，累加可得
\[
b_{2m} = b_2 + \sum_{k=2}^{m} (b_{2k} - b_{2k-2})
= 4 + \sum_{k=2}^{m} 4^{2k-1}.
\]
求和部分是等比数列：首项为 \(4^3 = 64\)，公比为 \(4^2 = 16\)，项数为 \(m-1\)。于是
\[
\sum_{k=2}^{m} 4^{2k-1} = \frac{64(16^{m-1} - 1)}{16 - 1}
= \frac{64(4^{2m-2} - 1)}{15}.
\]
因此
\[
b_{2m} = 4 + \frac{64 \cdot 4^{2m-2} - 64}{15}
= \frac{60 - 64}{15} + \frac{64 \cdot 4^{2m-2}}{15}
= \frac{4^3 \cdot 4^{2m-2} - 4}{15}
= \frac{4^{2m+1} - 4}{15}.
\]
要求 \(b_{2022}\)，令 \(2m = 2022\)，得 \(m = 1011\)，此时 \(2m+1 = 2023\)，代入得
\[
b_{2022} = \frac{4^{2023} - 4}{15}.
\]

## 答案
B

## 总结
本题的突破口在于利用数列的整数性质以及不等式所给区间的长度小于 1，将不等式约束转化为精确的递推等式 \(b_{n+1} - b_{n-1} = 4^n\)。之后按奇偶项拆分为两个独立的递推数列，再对偶数项使用累加法转化为等比数列求和，即可顺利求得指定项。处理此类“不等式定等式”问题时，抓住整数与区间长度的关系是关键。
