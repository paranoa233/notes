# 7-1-7 我的解答

对应题目：[[01_题目/7_数列/7-1_等差数列基本量的计算/7-1-7.md|7-1-7]]

## 完整题目

7. (多选题，2021 湖北武汉 3 月质检)两个等差数列 \( \left\{  {a}_{n}\right\} \) 和 \( \left\{  {b}_{n}\right\} \) ,其公差分别为 \( {d}_{1} \) 和 \( {d}_{2} \) ,其前 \( n \) 项和分别为 \( {S}_{n} \) 和 \( {T}_{n} \) ,则下列命题正确的是

( )

A. 若 \( \left\{  \sqrt{{S}_{n}}\right\} \) 为等差数列,则 \( {d}_{1} = 2{a}_{1} \)

B. 若 \( \left\{  {{S}_{n} + {T}_{n}}\right\} \) 为等差数列,则 \( {d}_{1} + {d}_{2} = 0 \)

C. 若 \( \left\{  {{a}_{n}{b}_{n}}\right\} \) 为等差数列,则 \( {d}_{1} = {d}_{2} = 0 \)

D 若 \( {b}_{n} \in  {\mathbf{N}}^{ * } \) ，则 \( \left\{  {a}_{{b}_{n}}\right\} \) 也为等差数列，且公差为 \( {d}_{1} + {d}_{2} \)

## 题目勘误
无明显勘误。

## 思路分析
本题是多选题，考查等差数列的定义与通项、前 \(n\) 项和的代数结构。依次分析四个选项：
- **选项 A**：将 \(\{\sqrt{S_n}\}\) 为等差数列转化为 \(S_n\) 是关于 \(n\) 的完全平方式，利用 \(S_n\) 的二次函数表达式比较系数；
- **选项 B**：将 \(\{S_n+T_n\}\) 为等差数列转化为 \(S_n+T_n\) 是关于 \(n\) 的一次函数，令二次项系数为零；
- **选项 C**：计算 \(\{a_nb_n\}\) 的通项，其为关于 \(n\) 的二次多项式，若为等差数列则二次项系数为零，但未必 \(d_1\) 与 \(d_2\) 同时为零；
- **选项 D**：直接计算 \(\{a_{b_n}\}\) 的通项，利用复合函数得到新公差，与所给公差比较。

## 详细解答
设 \(\{a_n\}\) 的首项为 \(a_1\)，公差为 \(d_1\)；\(\{b_n\}\) 的首项为 \(b_1\)，公差为 \(d_2\)。则
\[
a_n = a_1 + (n-1)d_1,\quad b_n = b_1 + (n-1)d_2,
\]
\[
S_n = na_1 + \frac{n(n-1)}{2}d_1 = \frac{d_1}{2}n^2 + \left(a_1 - \frac{d_1}{2}\right)n,
\]
\[
T_n = nb_1 + \frac{n(n-1)}{2}d_2.
\]

**选项 A**：若 \(\{\sqrt{S_n}\}\) 为等差数列，设 \(\sqrt{S_n} = pn + q\)（一次式）。  
令 \(S_0 = 0\)，则 \(\sqrt{S_0} = 0\)，从而 \(q=0\)，\(\sqrt{S_n} = pn\)，平方得
\[
S_n = p^2 n^2.
\]
与 \(S_n = \frac{d_1}{2}n^2 + \left(a_1 - \frac{d_1}{2}\right)n\) 对比系数得：
\[
a_1 - \frac{d_1}{2} = 0,\quad \frac{d_1}{2} = p^2,
\]
因此 \(d_1 = 2a_1\)。故 A 正确。

**选项 B**：若 \(\{S_n+T_n\}\) 为等差数列，则 \(S_n+T_n\) 是 \(n\) 的一次函数。计算得
\[
S_n + T_n = \frac{d_1 + d_2}{2}n^2 + \left(a_1 + b_1 - \frac{d_1 + d_2}{2}\right)n.
\]
令二次项系数为零，即 \(\frac{d_1 + d_2}{2} = 0\)，得 \(d_1 + d_2 = 0\)。故 B 正确。

**选项 C**：计算 \(\{a_nb_n\}\) 的通项：
\[
a_nb_n = \bigl[a_1 + (n-1)d_1\bigr]\bigl[b_1 + (n-1)d_2\bigr]
= a_1b_1 + (n-1)(a_1d_2 + b_1d_1) + (n-1)^2 d_1 d_2.
\]
若该数列为等差数列，必须是关于 \(n\) 的一次式，故二次项系数 \(d_1 d_2 = 0\)，即 \(d_1 = 0\) 或 \(d_2 = 0\)，不要求 \(d_1 = d_2 = 0\)。例如取 \(d_1=0, d_2\neq 0\)，则 \(a_n\) 为常数列，\(a_nb_n\) 为等差数列，但 \(d_2 \neq 0\)。故 C 错误。

**选项 D**：已知 \(b_n \in \mathbf{N}^*\)，计算 \(\{a_{b_n}\}\) 的通项：
\[
a_{b_n} = a_1 + (b_n - 1)d_1 = a_1 + \bigl[b_1 + (n-1)d_2 - 1\bigr]d_1 = \bigl[a_1 + (b_1 - 1)d_1\bigr] + (n-1)d_1 d_2.
\]
因此 \(\{a_{b_n}\}\) 仍为等差数列，但其公差为 \(d_1 d_2\)，而非 \(d_1 + d_2\)。故 D 错误。

综上，正确命题为 A、B。

## 答案
AB

## 总结
- 处理与等差数列前 \(n\) 项和相关的命题时，常将 \(S_n\) 写成关于 \(n\) 的二次函数（无常数项），利用等差数列等价于一次函数的性质提取系数关系。
- 两个等差数列的积数列为等差数列的充要条件是其通项中二次项系数为零（即 \(d_1 d_2 = 0\)），并不是公差同时为零。
- 等差数列的下标复合 \(\{a_{b_n}\}\)，新公差等于原两公差之积 \(d_1 d_2\)，易误加。
