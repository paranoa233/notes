# 7-4-10 我的解答

对应题目：[[01_题目/7_数列/7-4_等比数列基本量的计算/7-4-10.md|7-4-10]]

## 完整题目

1. (2020 全国卷Ⅰ文，10)设 \( \left\{  {a}_{n}\right\} \) 是等比数列，且 \( {a}_{1} + \; {a}_{2} + {a}_{3} = 1,{a}_{2} + {a}_{3} + {a}_{4} = 2 \) ,则 \( {a}_{6} + {a}_{7} + {a}_{8} =  \) (   )

A. 12 B. 24 C. 30 D. 32

## 题目勘误
无明显勘误。
## 思路分析
已知等比数列 \(\left\{ a_n \right\}\) 的两个连续三项和的等式，可设公比为 \(q\)，将后一组三项和用前一组表示，从而求出公比 \(q\)。再利用等比数列的性质，将所求的 \(a_6 + a_7 + a_8\) 也用 \(a_1 + a_2 + a_3\) 及公比 \(q\) 表示，直接代入计算即可。
## 详细解答
设等比数列 \(\left\{ a_n \right\}\) 的公比为 \(q\)。
已知条件：
\[
a_1 + a_2 + a_3 = 1 \quad \text{①}
\]
\[
a_2 + a_3 + a_4 = 2 \quad \text{②}
\]
由于 \(a_2 = a_1 q\)，\(a_3 = a_2 q = a_1 q^2\)，\(a_4 = a_3 q = a_1 q^3\)，因此：
\[
a_2 + a_3 + a_4 = a_1 q + a_2 q + a_3 q = q(a_1 + a_2 + a_3)
\]
代入①式与②式：
\[
2 = q \cdot 1 \implies q = 2
\]
所求式子为 \(a_6 + a_7 + a_8\)，同理可得：
\[
a_6 + a_7 + a_8 = a_1 q^5 + a_2 q^5 + a_3 q^5 = q^5 (a_1 + a_2 + a_3)
\]
将 \(q = 2\) 和 \(a_1 + a_2 + a_3 = 1\) 代入：
\[
a_6 + a_7 + a_8 = 2^5 \times 1 = 32
\]
## 答案
D
## 总结
本题主要考查等比数列基本量的计算，核心技巧是不必单独求出首项 \(a_1\)，而是利用相邻三项和的整体关系：\(a_{k+1} + a_{k+2} + a_{k+3} = q (a_k + a_{k+1} + a_{k+2})\)，从而快速求得公比，再整体代换得出所求。
