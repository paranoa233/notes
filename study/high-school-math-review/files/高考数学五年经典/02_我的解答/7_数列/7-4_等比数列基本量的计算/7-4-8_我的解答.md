# 7-4-8 我的解答

对应题目：[[01_题目/7_数列/7-4_等比数列基本量的计算/7-4-8.md|7-4-8]]

## 完整题目

8.(2022 山东模拟)已知等比数列 \( \left\{  {a}_{n}\right\} \) 的公比为 \( q \) ,且 \( {a}_{5} = 1 \) ,则下列选项错误的是 ( )

A. \( {a}_{3} + {a}_{7} \geq  2 \)

B. \( {a}_{4} + {a}_{6} \geq  2 \)

C. \( {a}_{7} - 2{a}_{6} + 1 \geq  0 \)

D. \( \frac{1}{{a}_{1}} + \frac{1}{{a}_{9}} = {a}_{1} + {a}_{9} \)

## 题目勘误
无明显勘误。

## 思路分析
已知等比数列 \(\{a_n\}\) 的公比为 \(q\)，且 \(a_5 = 1\)。可将各项统一用 \(a_5\) 和 \(q\) 表示，即 \(a_n = a_5 \cdot q^{n-5} = q^{n-5}\)。然后将各选项转化为关于 \(q\)（\(q \ne 0\)）的代数不等式或等式，利用基本不等式、完全平方公式等工具进行判断。注意公比 \(q\) 可能为负数，使用基本不等式时需保证涉及的项为正数。

## 详细解答
由 \(a_5 = 1\) 及等比数列通项公式得：
\[
a_n = a_5 \cdot q^{n-5} = q^{n-5}.
\]
分别写出各选项涉及的项：
\[
a_1 = q^{-4},\quad a_3 = q^{-2},\quad a_4 = q^{-1},\quad a_6 = q,\quad a_7 = q^2,\quad a_9 = q^4.
\]

**A 选项：** \(a_3 + a_7 \ge 2\)
\[
a_3 + a_7 = q^{-2} + q^2.
\]
因为 \(q \ne 0\)，所以 \(q^2 > 0\)，\(q^{-2} > 0\)。由基本不等式：
\[
q^2 + q^{-2} \ge 2\sqrt{q^2 \cdot q^{-2}} = 2,
\]
当且仅当 \(q^2 = q^{-2}\)，即 \(q = \pm 1\) 时取等号。故 A 正确。

**B 选项：** \(a_4 + a_6 \ge 2\)
\[
a_4 + a_6 = q^{-1} + q = \frac{1}{q} + q.
\]
若 \(q > 0\)，由基本不等式 \(\frac{1}{q} + q \ge 2\)，成立；  
若 \(q < 0\)，例如取 \(q = -1\)，则 \(a_4 = -1\)，\(a_6 = -1\)，\(a_4 + a_6 = -2 < 2\)，不成立。  
因此该不等式不是恒成立的，故 B 错误。

**C 选项：** \(a_7 - 2a_6 + 1 \ge 0\)
\[
a_7 - 2a_6 + 1 = q^2 - 2q + 1 = (q - 1)^2 \ge 0,
\]
恒成立。故 C 正确。

**D 选项：** \(\frac{1}{a_1} + \frac{1}{a_9} = a_1 + a_9\)
左边：
\[
\frac{1}{a_1} + \frac{1}{a_9} = \frac{1}{q^{-4}} + \frac{1}{q^4} = q^4 + q^{-4}.
\]
右边：
\[
a_1 + a_9 = q^{-4} + q^4.
\]
左边 = 右边，恒成立。故 D 正确。

综上，错误的选项为 B。

## 答案
B

## 总结
处理等比数列给定某一特定项的问题，可将所有项转化为该特定项与公比的幂的乘积，从而减少变量。本题的易错点在于 B 选项直接使用基本不等式而忽略了公比 \(q\) 可能为负的情况，导致误判。对于含有参数的不等式，要注意参数的取值范围，必要时应分类讨论或举反例。
