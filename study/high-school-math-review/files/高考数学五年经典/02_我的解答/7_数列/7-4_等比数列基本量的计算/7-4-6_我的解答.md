# 7-4-6 我的解答

对应题目：[[01_题目/7_数列/7-4_等比数列基本量的计算/7-4-6.md|7-4-6]]

## 完整题目

6.(2023 辽宁期末联考)在等比数列 \( \left\{  {a}_{n}\right\} \) 中， \( 0 < \; {a}_{1} < {a}_{8} = 1 \) ,则能使不等式 \( \left( {{a}_{1} - \frac{1}{{a}_{1}}}\right)  + \left( {{a}_{2} - \frac{1}{{a}_{2}}}\right) \; + \cdots  + \left( {{a}_{n} - \frac{1}{{a}_{n}}}\right)  \leq  0 \) 成立的正整数 \( n \) 的最大值为 ( )

A. 13 B. 14 C. 15 D. 16

## 题目勘误
无明显勘误。

## 思路分析
首先由等比数列条件和 \(a_8=1\)，可将通项写成 \(a_n = q^{n-8}\)，从而把差式 \(a_n - \frac{1}{a_n}\) 化为 \(q^{n-8} - q^{8-n}\)。该表达式关于 \(n=8\) 呈反对称，即前后对应项之和为零。利用这一对称性可以求出部分和 \(S_{15}=0\)，进而判断使不等式成立的最大正整数 \(n\)。

## 详细解答
设等比数列 \(\{a_n\}\) 的公比为 \(q\)。  
由 \(0 < a_1 < a_8 = 1\)，知数列各项均为正数，故 \(q > 0\)。  
又 \(a_8 = a_1 q^7 = 1\)，得 \(a_1 = q^{-7}\)。结合 \(a_1 < a_8\)，有 \(q^{-7} < 1\)，从而 \(q > 1\)。

于是通项为  
\[
a_n = a_1 q^{n-1} = q^{-7} \cdot q^{n-1} = q^{n-8} \quad (n \in \mathbb{N}^*).
\]

记  
\[
b_n = a_n - \frac{1}{a_n} = q^{n-8} - q^{8-n}.
\]
原不等式即 \(S_n = \sum_{k=1}^{n} b_k \leq 0\)。

对任意整数 \(m\)，有  
\[
b_{8+m} = q^{m} - q^{-m}, \quad b_{8-m} = q^{-m} - q^{m} = -b_{8+m}.
\]
可见数列 \(\{b_n\}\) 关于 \(n=8\) 呈反对称，且 \(b_8 = 0\)。

据此可配对所有脚标之和为 16 的项：  
\[
\begin{aligned}
b_1 + b_{15} &= 0, \\
b_2 + b_{14} &= 0, \\
&\ \vdots \\
b_7 + b_9 &= 0,
\end{aligned}
\]
且 \(b_8 = 0\)。

于是前 15 项之和为  
\[
S_{15} = \sum_{k=1}^{15} b_k = 0.
\]
前 16 项之和为  
\[
S_{16} = S_{15} + b_{16} = b_{16} = q^{8} - q^{-8}.
\]
因为 \(q > 1\)，所以 \(q^8 > 1 > q^{-8}\)，得 \(S_{16} > 0\)。

再讨论更小的 \(n\)：  
当 \(k < 8\) 时，\(k-8 < 0\)，\(8-k > 0\)，由于 \(q>1\)，有 \(q^{k-8} < 1 < q^{8-k}\)，故 \(b_k < 0\)；  
当 \(k > 8\) 时，类似可得 \(b_k > 0\)，且 \(|b_k|\) 随 \(|k-8|\) 增大而增大。

结合对称性可知，对于 \(1 \leq n \leq 15\)，未配对的负项总能使总和 \(S_n \leq 0\)（具体地，当 \(n \leq 8\) 时全为负或零，当 \(8 < n < 16\) 时部分和为负或零），而在 \(n = 15\) 时恰好为 0。  
当 \(n \geq 16\) 后，新增的 \(b_k\) 均为正，且其绝对值超过前面剩余的负项之和，导致 \(S_n > 0\)。

因此，使 \(S_n \leq 0\) 成立的正整数 \(n\) 的最大值为 \(15\)。

## 答案
C. \(15\)

## 总结
本题以等比数列为载体，考查通项公式与对称求和。关键步骤有二：一是利用 \(a_8=1\) 将通项转化为 \(q^{n-8}\)，从而将和式写为关于项数对称的形式；二是发现序列 \(\{b_n\}\) 关于 \(n=8\) 的反对称性，迅速得到 \(S_{15}=0\)，再通过单调性确定最大正整数 \(n\)。题目属于中等难度，重在观察代数结构的对称特征。
