# 7-5-9 我的解答

对应题目：[[01_题目/7_数列/7-5_等比数列的性质/7-5-9.md|7-5-9]]

## 完整题目

9. 已知等比数列 \( \left\{  {a}_{n}\right\} \) 共有奇数项,所有奇数项和 \( {S}_{\text{ 奇 }} = {255} \) ,所有偶数项和 \( {S}_{\text{ 偶 }} =  - {126} \) ,末项是 192,则首项 \( {a}_{1} = \) ( )

A. 1 B. 2 C. 3 D. 4

## 题目勘误
无明显勘误。
## 思路分析
等比数列共有奇数项，设项数为 \(2k+1\)（\(k\in \mathbb{N}\)）。奇数项构成以 \(a_1\) 为首项、\(q^2\) 为公比的等比数列，共 \(k+1\) 项；偶数项构成以 \(a_2=a_1q\) 为首项、\(q^2\) 为公比的等比数列，共 \(k\) 项。由题给条件 \(S_{\text{奇}}\) 与 \(S_{\text{偶}}\) 以及末项 \(a_{2k+1}=192\)，可先利用 \(S_{\text{偶}}=q(S_{\text{奇}}-a_{2k+1})\) 求出公比 \(q\)，再结合奇数项求和公式与末项表达式联立方程求出首项 \(a_1\)。
## 详细解答
设等比数列 \(\{a_n\}\) 共有 \(2k+1\) 项（\(k\in\mathbb{N}\)），公比为 \(q\)，首项为 \(a_1\)，末项为 \(a_{2k+1}=192\)。

奇数项为 \(a_1, a_3, \dots, a_{2k+1}\)，偶数项为 \(a_2, a_4, \dots, a_{2k}\)。  
偶数项和等于除末项外的奇数项和乘以 \(q\)：
\[
S_{\text{偶}} = a_2 + a_4 + \cdots + a_{2k} = q(a_1 + a_3 + \cdots + a_{2k-1}) = q(S_{\text{奇}} - a_{2k+1})
\]
代入已知条件 \(S_{\text{奇}}=255\)，\(S_{\text{偶}}=-126\)，\(a_{2k+1}=192\)：
\[
-126 = q(255 - 192) = 63q \quad\Rightarrow\quad q = \frac{-126}{63} = -2
\]

由末项公式：
\[
a_{2k+1} = a_1 q^{2k} = a_1 (q^2)^k = a_1 \cdot 4^k = 192 \tag{1}
\]

奇数项构成以 \(a_1\) 为首项、\(q^2=4\) 为公比的等比数列，共 \(k+1\) 项，其和为：
\[
S_{\text{奇}} = \frac{a_1\bigl(1 - 4^{k+1}\bigr)}{1 - 4} = \frac{a_1(4^{k+1} - 1)}{3} = 255
\]
\[
\Rightarrow\quad a_1(4^{k+1} - 1) = 765 \tag{2}
\]

令 \(x = 4^k\)，则 \(4^{k+1} = 4x\)。  
由 (1) 得 \(a_1 x = 192\)。代入 (2)：
\[
a_1(4x - 1) = 4a_1 x - a_1 = 4 \times 192 - a_1 = 768 - a_1 = 765
\]
\[
\Rightarrow\quad a_1 = 768 - 765 = 3
\]

## 答案
C
## 总结
本题主要考查等比数列奇数项与偶数项的关系以及求和公式的灵活运用。关键步骤是由 \(S_{\text{偶}}=q(S_{\text{奇}}-a_{\text{末}})\) 快速求得公比，避免设出项数再代入求和公式的繁琐运算。注意公比与末项符号对结果的影响，列方程时确保代入无误。
