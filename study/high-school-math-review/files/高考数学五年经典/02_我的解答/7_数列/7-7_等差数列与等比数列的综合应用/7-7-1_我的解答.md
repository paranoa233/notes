# 7-7-1 我的解答

对应题目：[[01_题目/7_数列/7-7_等差数列与等比数列的综合应用/7-7-1.md|7-7-1]]

## 完整题目

1.(2023 湖北 4 月调研)已知数列 \( \left\{  {a}_{n}\right\} \) 是等差数列,数列 \( \left\{  {b}_{n}\right\} \) 是等比数列. 若 \( {a}_{2} + {a}_{4} + {a}_{6} = {5\pi } \) . \( {b}_{2}{b}_{4}{b}_{6} = 3\sqrt{3} \) ,则 \( \tan \frac{{a}_{1} + {a}_{7}}{1 - {b}_{2}{b}_{6}} = \) ( )

A. \( \sqrt{3} \) B. \( - \sqrt{3} \) C. \( \frac{\sqrt{3}}{3} \) D. \( - \frac{\sqrt{3}}{3} \)

## 题目勘误
无明显勘误。
## 思路分析
本题给出等差数列 \(\{a_n\}\) 和等比数列 \(\{b_n\}\) 的特定项关系，求一个含正切函数的复杂分式的值。关键在于利用等差中项和等比中项的性质，将已知条件转化为 \(a_4\) 和 \(b_4\) 的值，再求出 \(a_1+a_7\) 和 \(b_2b_6\) 的表达式，最后代入正切公式并利用诱导公式求得结果。
## 详细解答
由等差数列性质，对于任意正整数 \(m,n\)，若 \(m+n=p+q\)，则 \(a_m + a_n = a_p + a_q\)。  
已知 \(a_2 + a_4 + a_6 = 5\pi\)。因为 \(a_2 + a_6 = 2a_4\)，所以
\[
a_2 + a_4 + a_6 = 3a_4 = 5\pi \quad\Rightarrow\quad a_4 = \frac{5\pi}{3}.
\]
又 \(a_1 + a_7 = 2a_4\)，因此
\[
a_1 + a_7 = 2 \times \frac{5\pi}{3} = \frac{10\pi}{3}.
\]

由等比数列性质，对于任意正整数 \(m,n\)，若 \(m+n=p+q\)，则 \(b_m b_n = b_p b_q\)。  
已知 \(b_2 b_4 b_6 = 3\sqrt{3}\)。因为 \(b_2 b_6 = b_4^2\)，所以
\[
b_2 b_4 b_6 = b_4^3 = 3\sqrt{3} = (\sqrt{3})^3 \quad\Rightarrow\quad b_4 = \sqrt{3}.
\]
于是
\[
b_2 b_6 = b_4^2 = 3.
\]

代入所求表达式：
\[
\tan \frac{a_1 + a_7}{1 - b_2 b_6} = \tan \frac{\frac{10\pi}{3}}{1 - 3} = \tan \left( -\frac{10\pi}{6} \right) = \tan \left( -\frac{5\pi}{3} \right).
\]
由正切函数的周期性和奇偶性，
\[
\tan \left( -\frac{5\pi}{3} \right) = -\tan \frac{5\pi}{3} = -\tan \left( 2\pi - \frac{\pi}{3} \right) = -\left( -\tan \frac{\pi}{3} \right) = \tan \frac{\pi}{3} = \sqrt{3}.
\]

因此，原式的值为 \(\sqrt{3}\)。
## 答案
A
## 总结
等差与等比数列的综合问题，核心是灵活运用中项公式：等差数列中 \(a_m + a_n = 2a_{(m+n)/2}\)（下标和为偶数时），等比数列中 \(b_m b_n = b_{(m+n)/2}^2\)。本题通过将三项关系转化为中间项，简化了计算。结合三角函数的诱导公式，即可快速得出答案。注意符号和角度范围，避免诱导公式符号错误。
