# 7-7-6 我的解答

对应题目：[[01_题目/7_数列/7-7_等差数列与等比数列的综合应用/7-7-6.md|7-7-6]]

## 完整题目

6.(2021 湖北七校联盟期中)已知等差数列 \( \left\{  {a}_{n}\right\} \) 的公差 \( d \) 不为 0,等比数列 \( \left\{  {b}_{n}\right\} \) 的公比 \( q \) 是小于 1 的正有理数. 若 \( {a}_{1} = {b}_{1} = d \) ,且 \( \frac{{a}_{1} + {a}_{2} + {a}_{4}}{{b}_{1} + {b}_{2} + {b}_{3}} \) 是正整数，则 \( q = \) ___.

## 题目勘误
无明显勘误。
## 思路分析
由等差、等比数列的通项公式将分子和分母均用 \(d\) 与 \(q\) 表示，约去非零公因式，得到关于 \(q\) 的分式。设该分式等于正整数 \(k\)，结合 \(0<q<1\) 确定 \(k\) 的可能取值，再解出 \(q\) 并检验其是否为有理数，最终得到公比 \(q\)。
## 详细解答
由等差数列 \(\{a_n\}\) 的公差为 \(d\) (\(d\neq 0\))，且 \(a_1 = d\)，得
\[
a_1 = d,\quad a_2 = a_1 + d = 2d,\quad a_4 = a_1 + 3d = 4d.
\]
所以
\[
a_1 + a_2 + a_4 = d + 2d + 4d = 7d.
\]
由等比数列 \(\{b_n\}\) 的公比为 \(q\)，且 \(b_1 = d\)，得
\[
b_1 = d,\quad b_2 = dq,\quad b_3 = dq^2.
\]
所以
\[
b_1 + b_2 + b_3 = d(1 + q + q^2).
\]
于是
\[
\frac{a_1 + a_2 + a_4}{b_1 + b_2 + b_3} = \frac{7d}{d(1+q+q^2)} = \frac{7}{1+q+q^2}.
\]
设该值为正整数 \(k\)，即
\[
\frac{7}{1+q+q^2} = k \quad (k\in\mathbb N^*).
\]
则
\[
1+q+q^2 = \frac{7}{k}.
\]
因为 \(q\) 是小于 \(1\) 的正数，所以 \(0 < q < 1\)，从而
\[
1 < 1+q+q^2 < 3.
\]
于是
\[
\frac{7}{3} < \frac{7}{1+q+q^2} < 7,
\]
即 \(k\) 的可能值为 \(3,4,5,6\)。

下面分别检验 \(k\) 的取值，并通过解方程求出 \(q\)，同时要求 \(q\) 为小于 \(1\) 的正有理数。

1. 当 \(k=3\) 时：
\[
1+q+q^2 = \frac{7}{3} \implies q^2 + q - \frac{4}{3}=0.
\]
判别式 \(\Delta = 1 + \frac{16}{3} = \frac{19}{3}\)，正根为
\[
q = \frac{-1 + \sqrt{\frac{19}{3}}}{2},
\]
不是有理数，舍去。

2. 当 \(k=4\) 时：
\[
1+q+q^2 = \frac{7}{4} \implies q^2 + q - \frac{3}{4}=0.
\]
因式分解或求根得 \((q+\frac{3}{2})(q-\frac{1}{2})=0\)，正根为 \(q = \dfrac{1}{2}\)，是有理数且满足 \(0<q<1\)，符合题意。

3. 当 \(k=5\) 时：
\[
1+q+q^2 = \frac{7}{5} \implies q^2 + q - \frac{2}{5}=0.
\]
判别式 \(\Delta = 1 + \frac{8}{5} = \frac{13}{5}\)，正根为
\[
q = \frac{-1 + \sqrt{\frac{13}{5}}}{2},
\]
不是有理数，舍去。

4. 当 \(k=6\) 时：
\[
1+q+q^2 = \frac{7}{6} \implies q^2 + q - \frac{1}{6}=0.
\]
判别式 \(\Delta = 1 + \frac{2}{3} = \frac{5}{3}\)，正根为
\[
q = \frac{-1 + \sqrt{\frac{5}{3}}}{2},
\]
不是有理数，舍去。

此外，若 \(k=1\) 或 \(k=2\)，则分母 \(1+q+q^2 = 7\) 或 \(\frac{7}{2}\)，解得 \(q\) 均大于或等于 \(1\)（例如 \(k=1\) 时 \(q=2\)），不满足 \(0<q<1\)。  
若 \(k=7\)，则 \(1+q+q^2=1 \implies q=0\) 或 \(q=-1\)，均不合题意。

综上，只有 \(q = \dfrac{1}{2}\) 满足所有条件。
## 答案
\(q = \dfrac{1}{2}\)
## 总结
本题综合了等差、等比数列的基本量运算与有理数的判别。关键步骤是通过设整数值 \(k\)，利用 \(q\) 的范围缩小 \(k\) 的取值，再解二次方程验证 \(q\) 是否为有理数。解方程时要注意有理根的条件，不能仅看是否在区间内。
