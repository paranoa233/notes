# 7-14-7 我的解答

对应题目：[[01_题目/7_数列/7-14_数列中的不等式证明与放缩问题/7-14-7.md|7-14-7]]

## 完整题目

7. (2023 浙江新高考名校联盟 5 月模拟)已知正项数列 \( \left\{  {a}_{n}\right\} \) 满足 \( {a}_{0} = 0,{a}_{n + 1}^{2} - {a}_{n}^{2} = 2\left( {n + 1}\right) , n \in \) N. 求证:

(1) \( \frac{{a}_{n + 2}}{{a}_{n + 1}} < \frac{{a}_{n + 1}}{{a}_{n}} \) ； (2) \( \frac{1}{{a}_{2}} + \frac{1}{{a}_{3}} + \cdots  + \frac{1}{{a}_{n}} < \ln n. \)

## 题目勘误
无明显勘误。

## 思路分析
1. 由递推关系 \(a_{n+1}^2 - a_n^2 = 2(n+1)\) 及 \(a_0=0\) 易求得 \(a_n\) 的通项公式，进而可化简(1)中比值并直接比较。
2. 第(2)问为和式不等式，项 \(1/a_k\) 可用通项表示。考虑数学归纳法，将归纳步转化为证明单项不等式 \( \frac{1}{a_{k+1}} < \ln\left(1+\frac1k\right) \)，并通过放缩与熟知不等式 \( \ln(1+x) > \frac{x}{1+x} \) 完成证明。

## 详细解答
**第一步：求通项公式**
由 \(a_{n+1}^2 - a_n^2 = 2(n+1)\)，对 \(n\) 依次赋值并累加：
当 \(n=0\) 时，\(a_1^2 - a_0^2 = 2\times1\)；
当 \(n=1\) 时，\(a_2^2 - a_1^2 = 2\times2\)；
……
当 \(n=k-1\) 时，\(a_k^2 - a_{k-1}^2 = 2k\)。
以上各式相加得：
\[
a_n^2 - a_0^2 = 2(1+2+\cdots+n) = 2\cdot\frac{n(n+1)}{2} = n(n+1).
\]
由 \(a_0=0\) 及正项数列，得 \(a_n = \sqrt{n(n+1)}\)，\(n\in\mathbf{N}\)。

**第二步：证明(1) \(\displaystyle \frac{a_{n+2}}{a_{n+1}} < \frac{a_{n+1}}{a_n}\)**
代入通项：
\[
\frac{a_{n+2}}{a_{n+1}} = \frac{\sqrt{(n+2)(n+3)}}{\sqrt{(n+1)(n+2)}} = \sqrt{\frac{n+3}{n+1}},\quad
\frac{a_{n+1}}{a_n} = \frac{\sqrt{(n+1)(n+2)}}{\sqrt{n(n+1)}} = \sqrt{\frac{n+2}{n}}.
\]
要证 \(\displaystyle \sqrt{\frac{n+3}{n+1}} < \sqrt{\frac{n+2}{n}}\)，只需平方后比较：
\[
\frac{n+3}{n+1} < \frac{n+2}{n}
\;\Longleftrightarrow\; n(n+3) < (n+1)(n+2)
\;\Longleftrightarrow\; n^2+3n < n^2+3n+2.
\]
即 \(0<2\)，恒成立。故(1)得证。

**第三步：证明(2) \(\displaystyle \frac{1}{a_2} + \frac{1}{a_3} + \cdots + \frac{1}{a_n} < \ln n\)**
用数学归纳法。
当 \(n=2\) 时，左端 \(= \frac1{a_2} = \frac1{\sqrt{6}} \approx 0.408\)，右端 \(\ln 2 \approx 0.693\)，不等式成立。
假设当 \(n=k\)（\(k\ge 2\)）时不等式成立，即
\[
\frac{1}{a_2} + \frac{1}{a_3} + \cdots + \frac{1}{a_k} < \ln k.
\]
当 \(n=k+1\) 时，
\[
\frac{1}{a_2} + \cdots + \frac{1}{a_k} + \frac{1}{a_{k+1}} < \ln k + \frac{1}{a_{k+1}}.
\]
欲证右端 \(< \ln(k+1) = \ln k + \ln\left(1+\frac1k\right)\)，只需证
\[
\frac{1}{a_{k+1}} < \ln\left(1+\frac1k\right).
\]
代入 \(a_{k+1} = \sqrt{(k+1)(k+2)}\)，即证
\[
\frac{1}{\sqrt{(k+1)(k+2)}} < \ln\left(1+\frac1k\right).
\]
令 \(x = \dfrac1k > 0\)，则左端化为
\[
\frac{1}{\sqrt{(k+1)(k+2)}} = \frac{x}{\sqrt{(1+x)(1+2x)}}.
\]
因为
\[
(1+x)(1+2x) = 1+3x+2x^2 > 1+2x+x^2 = (1+x)^2,
\]
所以
\[
\sqrt{(1+x)(1+2x)} > 1+x,
\]
从而
\[
\frac{x}{\sqrt{(1+x)(1+2x)}} < \frac{x}{1+x}.
\]
又由熟知的不等式 \(\ln(1+x) > \dfrac{x}{1+x}\)（\(x>0\)），得
\[
\frac{x}{\sqrt{(1+x)(1+2x)}} < \frac{x}{1+x} < \ln(1+x),
\]
即 \(\displaystyle \frac{1}{\sqrt{(k+1)(k+2)}} < \ln\left(1+\frac1k\right)\) 成立。
因此 \(\displaystyle \frac{1}{a_2} + \cdots + \frac{1}{a_{k+1}} < \ln(k+1)\)，归纳步完成。

综上，对任意 \(n\ge 2\)，原不等式成立。

## 答案
证明见详细解答。

## 总结
1. 通过累加法求出通项 \(a_n = \sqrt{n(n+1)}\) 是后续证明的基础。
2. 第(1)问直接代入通项，平方比较即得。
3. 第(2)问利用数学归纳法，将和式不等式归结为单项不等式，再通过放缩 \(\sqrt{(1+x)(1+2x)} > 1+x\) 与熟悉的不等式 \(\ln(1+x) > \frac{x}{1+x}\) 完成关键步骤。
