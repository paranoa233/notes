# 7-14-1 我的解答

对应题目：[[01_题目/7_数列/7-14_数列中的不等式证明与放缩问题/7-14-1.md|7-14-1]]

## 完整题目

1.(2023 山东济南一模)各项均为正数的数列 \( \left\{  {a}_{n}\right\} \) ,其前 \( n \) 项和记为 \( {S}_{n} \) ,且满足 \( \forall n \in  {\mathbf{N}}^{ * } \) ,都有 \( 2{S}_{n} = {a}_{n}^{2} + {a}_{n}. \)

(1)求数列 \( \left\{  {a}_{n}\right\} \) 的通项公式；

(2)设 \( {T}_{n} = \frac{1}{{a}_{1}^{2}} + \frac{1}{{a}_{2}^{2}} + \frac{1}{{a}_{3}^{2}} + \cdots  + \frac{1}{{a}_{n}^{2}} \) ，求证: \( {T}_{n} < \frac{7}{4} \)

## 题目勘误
无明显勘误。

## 思路分析
第(1)问根据前 \(n\) 项和 \(S_n\) 与通项 \(a_n\) 的关系式 \(2S_n = a_n^2 + a_n\)，利用 \(a_n = S_n - S_{n-1}\,(n\ge 2)\) 消去 \(S_n\)，结合各项为正可推出 \(\{a_n\}\) 为等差数列，进而求得通项。

第(2)问要求证明 \(T_n = \sum_{k=1}^n \frac{1}{a_k^2} < \frac{7}{4}\)。由(1)知即证 \(\sum_{k=1}^n \frac{1}{k^2} < \frac{7}{4}\)。直接利用熟知的不等式放缩 \(\frac{1}{k^2} < \frac{1}{k(k-1)} = \frac{1}{k-1} - \frac{1}{k}\,(k\ge 2)\) 可得上界 \(2\)，不够紧；若保留前两项从 \(k=3\) 开始放缩，则可得到更紧的上界 \(\frac{7}{4}\)。

## 详细解答
**(1) 求 \(\{a_n\}\) 的通项公式**

由 \(2S_n = a_n^2 + a_n\)，
当 \(n=1\) 时，\(2a_1 = a_1^2 + a_1\)，
移项得 \(a_1^2 - a_1 = 0\)，即 \(a_1(a_1-1)=0\)。
因为各项均为正数，所以 \(a_1 = 1\)。

当 \(n \ge 2\) 时，\(2S_{n-1} = a_{n-1}^2 + a_{n-1}\)。
将两式相减：
\[
2(S_n - S_{n-1}) = (a_n^2 + a_n) - (a_{n-1}^2 + a_{n-1})
\]
又 \(S_n - S_{n-1} = a_n\)，所以
\[
2a_n = a_n^2 - a_{n-1}^2 + a_n - a_{n-1}
\]
整理得
\[
a_n^2 - a_{n-1}^2 - a_n - a_{n-1} = 0
\]
\[
(a_n - a_{n-1})(a_n + a_{n-1}) - (a_n + a_{n-1}) = 0
\]
\[
(a_n + a_{n-1})(a_n - a_{n-1} - 1) = 0
\]
由各项为正知 \(a_n + a_{n-1} > 0\)，故必有
\[
a_n - a_{n-1} = 1 \quad (n\ge 2)
\]
因此 \(\{a_n\}\) 是以 \(1\) 为首项、\(1\) 为公差的等差数列，通项公式为
\[
a_n = 1 + (n-1)\times 1 = n.
\]

**(2) 证明 \(T_n < \dfrac{7}{4}\)**

由(1)知 \(a_n = n\)，故
\[
T_n = \frac{1}{1^2} + \frac{1}{2^2} + \frac{1}{3^2} + \cdots + \frac{1}{n^2}
\]
要证 \(T_n < \dfrac{7}{4}\)。

当 \(n=1\) 时，\(T_1 = 1 < \dfrac{7}{4}\)，成立；
当 \(n=2\) 时，\(T_2 = 1 + \dfrac{1}{4} = \dfrac{5}{4} < \dfrac{7}{4}\)，成立。

当 \(n \ge 3\) 时，对 \(k\ge 3\) 利用放缩：
\[
\frac{1}{k^2} < \frac{1}{k(k-1)} = \frac{1}{k-1} - \frac{1}{k}
\]
于是
\[
\begin{aligned}
T_n &= 1 + \frac{1}{4} + \sum_{k=3}^{n} \frac{1}{k^2} \\
    &< 1 + \frac{1}{4} + \sum_{k=3}^{n} \left(\frac{1}{k-1} - \frac{1}{k}\right) \\
    &= \frac{5}{4} + \left( \frac{1}{2} - \frac{1}{3} \right) + \left( \frac{1}{3} - \frac{1}{4} \right) + \cdots + \left( \frac{1}{n-1} - \frac{1}{n} \right) \\
    &= \frac{5}{4} + \frac{1}{2} - \frac{1}{n} \\
    &= \frac{7}{4} - \frac{1}{n}
\end{aligned}
\]
因为 \(\frac{1}{n} > 0\)，所以 \(T_n < \frac{7}{4}\)。

综上所述，对任意正整数 \(n\)，均有 \(T_n < \dfrac{7}{4}\)。

## 答案
(1) \(a_n = n\)；
(2) 证明见详细解答。

## 总结
已知 \(S_n\) 与 \(a_n\) 的二次关系，通过 \(a_n = S_n - S_{n-1}\) 消元是求通项的基本方法；因式分解后利用各项为正确定单调性，得出等差数列。

证明和式不等式时，当直接放缩得到的界不够精确，可考虑保留前几项、从某项开始放缩，利用“裂项相消”使界收紧。本题保留前两项后改用 \(\frac{1}{k(k-1)}\) 放缩即得目标上界 \(\frac{7}{4}\)。
