# 7-3-1 我的解答

对应题目：[[01_题目/7_数列/7-3_等差数列前-(n-)项和的最值/7-3-1.md|7-3-1]]

## 完整题目

1.(2021 重庆西南大学附属中学月考)等差数列 \( \left\{  {a}_{n}\right\} \) 的前 \( n \) 项和为 \( {S}_{n} \) . 若 \( {a}_{1} > 0,{S}_{25} = 0 \) ,则使 \( {S}_{n} \) 取得最大值时的 \( n \) 的取值为___

## 题目勘误
无明显勘误。
## 思路分析
已知等差数列 \(\{a_n\}\) 的前 \(n\) 项和为 \(S_n\)，\(a_1>0\) 且 \(S_{25}=0\)。由等差数列前 \(n\) 项和公式建立 \(a_1\) 与公差 \(d\) 的关系，判断数列的单调性；再将 \(S_n\) 表示为关于 \(n\) 的二次函数，利用二次函数的对称轴及 \(n\) 为正整数的特点，确定使 \(S_n\) 最大的 \(n\)。
## 详细解答
设等差数列 \(\{a_n\}\) 的公差为 \(d\)。  
由 \(S_{25}=0\)，得  
\[
S_{25} = 25a_1 + \frac{25\times 24}{2}d = 25a_1 + 300d = 0,
\]  
两边同除以 \(25\)，得  
\[
a_1 + 12d = 0 \quad \Rightarrow \quad a_1 = -12d.
\]  
因为 \(a_1>0\)，所以 \(-12d>0\)，即 \(d<0\)，数列单调递减。  

前 \(n\) 项和  
\[
S_n = na_1 + \frac{n(n-1)}{2}d
      = n(-12d) + \frac{n(n-1)}{2}d
      = d\left(-12n + \frac{n^2 - n}{2}\right)
      = d\left(\frac{n^2}{2} - \frac{25}{2}n\right)
      = \frac{d}{2}\,n(n-25).
\]  
因为 \(d<0\)，\(\frac{d}{2}<0\)，所以 \(S_n\) 是开口向下的关于 \(n\) 的二次函数（定义域为正整数）。  
对称轴方程为 \(n = \frac{0+25}{2} = 12.5\)。  
由于 \(n\) 取正整数，且对称轴两侧等距离的点函数值相等，故当 \(n=12\) 或 \(n=13\) 时，\(S_n\) 同时取得最大值。  

验证：  
\(S_{12} = \frac{d}{2}\times 12\times (12-25) = \frac{d}{2}\times 12\times (-13) = -78d\)，  
\(S_{13} = \frac{d}{2}\times 13\times (13-25) = \frac{d}{2}\times 13\times (-12) = -78d\)，  
两者相等，均为最大值。
## 答案
\(12\) 或 \(13\)
## 总结
本题利用等差数列前 \(n\) 项和公式转化为二次函数的最值问题。当二次函数对称轴为半整数时，最值在相邻的两个整数点同时取得。注意先由条件确定公差的符号，才能判断开口方向。
