# 7-3-2 我的解答

对应题目：[[01_题目/7_数列/7-3_等差数列前-(n-)项和的最值/7-3-2.md|7-3-2]]

## 完整题目

2.(2024 广东广州一模)已知数列 \( \left\{  {a}_{n}\right\} \) 的前 \( n \) 项和 \( {S}_{n} = {n}^{2} + n \) ,当 \( \frac{{S}_{n} + 9}{{a}_{n}} \) 取最小值时, \( n = \) ___。

## 题目勘误
无明显勘误。
## 思路分析
已知前 \(n\) 项和 \(S_n\)，先求出通项 \(a_n\)。将 \(\frac{S_n+9}{a_n}\) 表示为关于 \(n\) 的函数（数列），由于 \(n\) 为正整数，可借助基本不等式或对勾函数单调性求最小值，并确定对应的 \(n\)。
## 详细解答
由 \(S_n = n^2 + n\)，
当 \(n = 1\) 时，\(a_1 = S_1 = 1^2 + 1 = 2\)。
当 \(n \ge 2\) 时，
\[
a_n = S_n - S_{n-1} = (n^2 + n) - \left[(n-1)^2 + (n-1)\right] = n^2 + n - (n^2 - n) = 2n.
\]
检验 \(n=1\) 时 \(2 \times 1 = 2 = a_1\)，故 \(a_n = 2n\ (n \in \mathbb{N}^*)\)。

代入目标式：
\[
\frac{S_n + 9}{a_n} = \frac{n^2 + n + 9}{2n} = \frac{1}{2}\left(n + 1 + \frac{9}{n}\right).
\]
令 \(f(n) = n + \frac{9}{n} + 1\)，其中 \(n \in \mathbb{N}^*\)。
由基本不等式，\(n + \frac{9}{n} \ge 2\sqrt{n \cdot \frac{9}{n}} = 6\)，
当且仅当 \(n = \frac{9}{n}\)，即 \(n = 3\) 时取等号。
此时 \(f(3) = 3 + 3 + 1 = 7\)，\(\frac{S_3 + 9}{a_3} = \frac{7}{2}\) 为最小值。

因此，当 \(\frac{S_n + 9}{a_n}\) 取最小值时，\(n = 3\)。
## 答案
\(3\)
## 总结
由 \(S_n\) 求 \(a_n\) 时需注意分段验证；分子分母同含 \(n\) 的分式最值常通过分离常数或基本不等式求解，并检验 \(n\) 能否取到等号对应的正整数值。
