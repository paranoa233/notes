# 7-3-5 我的解答

对应题目：[[01_题目/7_数列/7-3_等差数列前-(n-)项和的最值/7-3-5.md|7-3-5]]

## 完整题目

5. (多选题, 2023 辽宁教研联盟一调) 设等差数列 \( \left\{  {a}_{n}\right\} \) 的前 \( n \) 项和是 \( {S}_{n} \) . 若 \( {a}_{2} <  - {a}_{11} < {a}_{1} \) ,则 ( )

A. \( {a}_{6} < a \) . B. \( {S}_{10} > 0 \)

C. \( {S}_{15} < 0 \) D. \( {S}_{n} \leq  {S}_{5} \)

6 (多选题, 2024 安徽六校教育研究会二测) 已知等差数列 \( \left\{  {a}_{n}\right\} \) 的前 \( n \) 项和为 \( {S}_{n},{a}_{1} > 0 \) ,且 \( \left( {S}_{11}\right. \; \left. {-{S}_{7}}\right) \left( {{S}_{11} - {S}_{8}}\right)  < 0 \) ,则 ( )

A. \( {a}_{9} + {a}_{10} > 0 \)

B. \( {S}_{7} < {S}_{11} < {S}_{8} \)

C. 当 \( n = {10} \) 时, \( {S}_{n} \) 取最大值

D. 当 \( {S}_{n} < 0 \) 时, \( n \) 的最小值为 19

## 题目勘误
选项 A 中出现了单独的 “\(a\)”，在题目语境下并无定义，高度疑似为数字 “\(0\)” 的 OCR 识别错误（手写体或印刷体 0 常被误认为字母 a）。建议将选项 A 更正为 “\(a_6 < 0\)”。其余题干和选项暂无明显勘误，但题目原始排版依赖图片，建议人工核对原卷确认 A 选项的真实内容。

## 思路分析
题设给出等差数列 \(\{a_n\}\) 前 \(n\) 项和 \(S_n\) 的不等式条件 \(a_2 < -a_{11} < a_1\)。  
可将其拆解为 \(a_2 + a_{11} < 0\) 和 \(a_1 + a_{11} > 0\)，利用等差数列下标和性质，将条件转化为关于 \(a_6\) 与 \(a_6+a_7\) 的符号约束。  
结合公差的正负，分析数列各项的符号变化，进而判断前 \(n\) 项和的正负与最值。

## 详细解答
设等差数列 \(\{a_n\}\) 的公差为 \(d\)。

由 \(a_2 < -a_{11} < a_1\) 可得两个不等式：
1. \(a_2 < -a_{11} \;\Longrightarrow\; a_2 + a_{11} < 0\)；
2. \(-a_{11} < a_1 \;\Longrightarrow\; a_1 + a_{11} > 0\)。

利用等差数列性质 \(a_m + a_n = a_p + a_q\) 当 \(m+n=p+q\)：
- 由 \(a_1 + a_{11} > 0\)，且 \(1+11=6+6\)，得 \(a_1 + a_{11} = 2a_6 > 0\)，故 \(a_6 > 0\)。
- 由 \(a_2 + a_{11} < 0\)，且 \(2+11=6+7\)，得 \(a_2 + a_{11} = a_6 + a_7 < 0\)。

由 \(a_6 > 0\) 且 \(a_6 + a_7 < 0\) 可得 \(a_7 < -a_6 < 0\)，因此 \(a_6 > 0 > a_7\)，从而公差 \(d = a_7 - a_6 < 0\)。  
数列是递减的：\(a_1 > 0\)，从某一项开始变负。

**逐项判断选项（将 A 理解为 \(a_6 < 0\)）：**

- **A**：由上已得 \(a_6 > 0\)，故 \(a_6 < 0\) 不成立。A 错误。

- **B**：  
\(S_{10} = \dfrac{10}{2}(a_1+a_{10}) = 5(a_1+a_{10})\)。  
又 \(S_{11} = \dfrac{11}{2}(a_1+a_{11}) = 11a_6 > 0\)，且 \(S_{10} = S_{11} - a_{11}\)。  
由于 \(a_{11} = a_6 + 5d\)，而由 \(a_7 = a_6 + d < 0\) 得 \(d < -a_6\)，于是  
\(a_{11} = a_6 + 5d < a_6 + 5(-a_6) = -4a_6 < 0\)，即 \(a_{11} < 0\)。  
因此 \(S_{10} = S_{11} - a_{11} = S_{11} + |a_{11}| > S_{11} > 0\)，B 正确。

- **C**：  
\(S_{15} = \dfrac{15}{2}(a_1 + a_{15}) = 15a_8\)。  
\(a_8 = a_7 + d\)，由 \(a_7 < 0\) 且 \(d < 0\) 知 \(a_8 < 0\)，所以 \(S_{15} = 15a_8 < 0\)，C 正确。

- **D**：  
因为 \(a_6 > 0\)，所以 \(S_6 = S_5 + a_6 > S_5\)，即 \(S_6 > S_5\)。  
又 \(a_7 < 0\)，得 \(S_7 = S_6 + a_7 < S_6\)。  
因此 \(S_n\) 在 \(n=6\) 时取得最大值，\(S_5\) 并非最大值，故 \(S_n \leq S_5\) 不恒成立。D 错误。

## 答案
BC

## 总结
本题核心是将已知不等式转化为等差中项关系，借助“下标和相等则项和相等”的性质，快速定出分界项 \(a_6 > 0, a_7 < 0\)，由此确定数列单调递减、正负分界及前 \(n\) 项和的最值与符号。注意识别因 OCR 造成的选项文本错误（如“0”误作“a”），避免误判。
