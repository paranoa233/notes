# 7-3-3 我的解答

对应题目：[[01_题目/7_数列/7-3_等差数列前-(n-)项和的最值/7-3-3.md|7-3-3]]

## 完整题目

3.(2021 上海市上海中学期中)已知数列 \( \left\{  {a}_{n}\right\} \) 为等差数列, \( {a}_{1} + {a}_{3} + {a}_{5} = 1,{S}_{n} \) 表示数列 \( \left\{  {a}_{n}\right\} \) 的前 \( n \) 项和,若当且仅当 \( n = {20} \) 时, \( {S}_{n} \) 取到最大值,则 \( {a}_{2} + {a}_{4} + {a}_{6} \) 的取值范围是___.

## 题目勘误
无明显勘误。

## 思路分析
本题主要考查等差数列的通项公式、前 \(n\) 项和的性质以及取最值的条件。关键突破口在于：
1. 由 \(a_1 + a_3 + a_5 = 1\) 求出 \(a_3\)。
2. 利用“当且仅当 \(n=20\) 时 \(S_n\) 取最大值”得出公差 \(d<0\) 且 \(a_{20}>0,\;a_{21}<0\)。由此解出 \(d\) 的取值范围。
3. 将目标式 \(a_2 + a_4 + a_6\) 用 \(a_1\) 和 \(d\) 表示，再结合已知条件转化为关于 \(d\) 的一次函数，利用 \(d\) 的范围求值域。

## 详细解答
设等差数列 \(\{a_n\}\) 的首项为 \(a_1\)，公差为 \(d\)。
由已知 \(a_1 + a_3 + a_5 = 1\)，得
\[
a_1 + (a_1 + 2d) + (a_1 + 4d) = 3a_1 + 6d = 1,
\]
即
\[
a_1 + 2d = \frac{1}{3} \quad \text{或} \quad a_3 = \frac{1}{3}.
\]

因为等差数列的前 \(n\) 项和 \(S_n\) 当且仅当 \(n=20\) 时取最大值，所以 \(\{a_n\}\) 为递减数列，即 \(d < 0\)。
\(S_n\) 在 \(n=20\) 处取最大值（且唯一）的充要条件是：
\[
a_{20} > 0 \quad \text{且} \quad a_{21} < 0.
\]
由通项公式 \(a_n = a_3 + (n-3)d = \frac{1}{3} + (n-3)d\)，代入 \(n=20,21\)：
\[
a_{20} = \frac{1}{3} + 17d > 0 \implies d > -\frac{1}{51},
\]
\[
a_{21} = \frac{1}{3} + 18d < 0 \implies d < -\frac{1}{54}.
\]
因此 \(d\) 的取值范围是
\[
d \in \left(-\frac{1}{51},\; -\frac{1}{54}\right).
\]

接下来求 \(a_2 + a_4 + a_6\)：
\[
a_2 + a_4 + a_6 = (a_1 + d) + (a_1 + 3d) + (a_1 + 5d) = 3a_1 + 9d.
\]
由 \(3a_1 + 6d = 1\) 得 \(3a_1 = 1 - 6d\)，代入上式：
\[
a_2 + a_4 + a_6 = (1 - 6d) + 9d = 1 + 3d.
\]
将 \(d\) 的范围代入，得
\[
1 + 3 \cdot \left(-\frac{1}{51}\right) < 1 + 3d < 1 + 3 \cdot \left(-\frac{1}{54}\right),
\]
\[
1 - \frac{1}{17} < 1 + 3d < 1 - \frac{1}{18},
\]
\[
\frac{16}{17} < a_2 + a_4 + a_6 < \frac{17}{18}.
\]
因为 \(d\) 的区间为开区间，所求取值范围也是开区间。

## 答案
\(\displaystyle\left(\frac{16}{17},\; \frac{17}{18}\right)\)

## 总结
处理等差数列前 \(n\) 项和的最值问题时，往往转化为寻找正负项的分界点。当且仅当 \(S_n\) 在 \(n=k\) 处取最大值（最小值）时，必有 \(a_k >0,\;a_{k+1}<0\)（或 \(a_k <0,\;a_{k+1}>0\)），等号是否取到需根据唯一性判断。本题将目标式转化为公差 \(d\) 的线性函数，结合不等式解出范围。
