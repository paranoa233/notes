# 7-2-5 我的解答

对应题目：[[01_题目/7_数列/7-2_等差数列的性质/7-2-5.md|7-2-5]]

## 完整题目

5. (多选题,2022 江苏苏锡常镇调研(一)) 记 \( {S}_{n} \) 为等差数列 \( \left\{  {a}_{n}\right\} \) 的前 \( n \) 项和,则 ( )

A. \( {S}_{6} = 2{S}_{4} - {S}_{2} \)

B. \( {S}_{6} = 3\left( {{S}_{4} - {S}_{2}}\right) \)

C. \( {S}_{2n},{S}_{4n} - {S}_{2n},{S}_{6n} - {S}_{4n} \) 成等差数列

D. \( \frac{{S}_{2}}{2},\frac{{S}_{4}}{4},\frac{{S}_{6}}{6} \) 成等差数列

## 题目勘误
无明显勘误。
## 思路分析
本题考查等差数列前 \(n\) 项和 \(S_n\) 的常见性质。对各个选项，可以运用公式 \(S_n = na_1 + \frac{n(n-1)}{2}d\) 直接计算验证，也可以利用等差数列的“片段和”性质：将原数列按等长分段，各段之和仍成等差数列；以及数列 \(\left\{\frac{S_n}{n}\right\}\) 是等差数列的性质。
## 详细解答
设等差数列 \(\{a_n\}\) 的首项为 \(a_1\)，公差为 \(d\)，前 \(n\) 项和为 \(S_n\)。

**A 选项**  
计算 \(S_6\) 与 \(2S_4 - S_2\)：  
\[
\begin{aligned}
S_6 &= 6a_1 + \frac{6 \times 5}{2}d = 6a_1 + 15d,\\
S_4 &= 4a_1 + 6d,\quad S_2 = 2a_1 + d,\\
2S_4 - S_2 &= 2(4a_1 + 6d) - (2a_1 + d) = 6a_1 + 11d.
\end{aligned}
\]  
仅当 \(d=0\) 时两者相等，一般等差数列中不恒成立，故 A 错误。

**B 选项**  
利用 \(S_4 - S_2 = a_3 + a_4 = (a_1 + 2d) + (a_1 + 3d) = 2a_1 + 5d\)，  
则 \(3(S_4 - S_2) = 3(2a_1 + 5d) = 6a_1 + 15d = S_6\)，恒成立，故 B 正确。

**C 选项**  
将数列等分为长度为 \(2n\) 的段：第一段和为 \(S_{2n}\)，第二段和为 \(S_{4n} - S_{2n}\)，第三段和为 \(S_{6n} - S_{4n}\)。  
由等差数列的片段和性质知，等长连续片段之和仍成等差数列，故这三项成等差数列，C 正确。

**D 选项**  
由 \(S_n = na_1 + \frac{n(n-1)}{2}d\)，得  
\[
\frac{S_n}{n} = a_1 + \frac{n-1}{2}d,
\]  
这是关于 \(n\) 的一次函数，因此数列 \(\left\{\frac{S_n}{n}\right\}\) 是公差为 \(\frac{d}{2}\) 的等差数列。  
其中下标 \(2,4,6\) 成等差数列，故 \(\frac{S_2}{2}, \frac{S_4}{4}, \frac{S_6}{6}\) 也成等差数列，D 正确。

综上，正确选项为 B、C、D。
## 答案
BCD
## 总结
处理等差数列前 \(n\) 项和的等式判断题，常用方法有：
1. 基本量法：将 \(S_n\) 写成 \(na_1 + \frac{n(n-1)}{2}d\)，直接作差比较。
2. 性质法：如等长片段和成等差数列，\(\frac{S_n}{n}\) 成等差数列等，可快速判断。
本题 A 选项易误选，需注意 \(2S_4 - S_2 = S_4 + (S_4 - S_2)\) 不等于 \(S_6\)，一般 \(S_6 = S_4 + (S_6 - S_4) = S_4 + a_5 + a_6\)。
