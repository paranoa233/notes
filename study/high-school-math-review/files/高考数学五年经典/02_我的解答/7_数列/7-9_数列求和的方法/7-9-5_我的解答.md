# 7-9-5 我的解答

对应题目：[[01_题目/7_数列/7-9_数列求和的方法/7-9-5.md|7-9-5]]

## 完整题目

5.(2024 湖北宜荆荆三模)设 \( \left\{  {a}_{n}\right\} \) 是正数组成的数列,其前 \( n \) 项和为 \( {S}_{n} \) ,已知 \( {a}_{n} \) 与 2 的等差中项等于 \( {S}_{n} \) 与 2 的等比中项.

(1)求数列 \( \left\{  {a}_{n}\right\} \) 的通项公式；

(2)令 \( {b}_{n} = \frac{1}{2}\left( {\frac{{a}_{n + 1}}{{a}_{n}} + \frac{{a}_{n}}{{a}_{n + 1}}}\right) \left( {n \in  {\mathbf{N}}^{ * }}\right) \) ，求 \( \left\{  {b}_{n}\right\} \) 的前 \( n \) 项和.

## 题目勘误
无明显勘误。

## 思路分析
第(1)问：根据“\(a_n\) 与 \(2\) 的等差中项等于 \(S_n\) 与 \(2\) 的等比中项”以及各项均为正数，列出等式；平方后得到 \(a_n\) 与 \(S_n\) 的关系式；利用 \(a_n=S_n-S_{n-1}(n\ge2)\) 消去 \(S_n\)，得到相邻项的递推关系，并求出首项，进而得到通项公式。  
第(2)问：将(1)中求出的 \(a_n\) 代入 \(b_n\) 的表达式，化简后通过裂项相消法求出前 \(n\) 项和。

## 详细解答
**(1) 求 \(\{a_n\}\) 的通项公式**  
由题意，\(a_n\) 与 \(2\) 的等差中项为 \(\frac{a_n+2}{2}\)，\(S_n\) 与 \(2\) 的等比中项为 \(\sqrt{2S_n}\)（因 \(a_n>0\)，\(S_n>0\)，故取正根）。  
依题意得  
\[
\frac{a_n+2}{2} = \sqrt{2S_n}
\]  
两边平方，整理得  
\[
(a_n+2)^2 = 8S_n \quad ①
\]  
当 \(n\ge2\) 时，  
\[
(a_{n-1}+2)^2 = 8S_{n-1} \quad ②
\]  
\(①-②\)，并利用 \(S_n-S_{n-1}=a_n\)，得  
\[
(a_n+2)^2 - (a_{n-1}+2)^2 = 8a_n
\]  
展开得  
\[
a_n^2 + 4a_n + 4 - a_{n-1}^2 - 4a_{n-1} - 4 = 8a_n
\]  
整理得  
\[
a_n^2 - a_{n-1}^2 - 4a_n - 4a_{n-1} = 0
\]  
因式分解  
\[
(a_n + a_{n-1})(a_n - a_{n-1}) - 4(a_n + a_{n-1}) = 0
\]  
\[
(a_n + a_{n-1})(a_n - a_{n-1} - 4) = 0
\]  
因 \(a_n>0\)，\(a_{n-1}>0\)，故 \(a_n+a_{n-1}>0\)，于是  
\[
a_n - a_{n-1} = 4 \quad (n\ge2)
\]  
又当 \(n=1\) 时，由①得 \((a_1+2)^2 = 8S_1 = 8a_1\)，即  
\[
a_1^2 + 4a_1 + 4 = 8a_1 \implies a_1^2 - 4a_1 + 4 = 0 \implies (a_1-2)^2=0
\]  
解得 \(a_1=2\)。  
因此，数列 \(\{a_n\}\) 是首项为 \(2\)，公差为 \(4\) 的等差数列，其通项公式为  
\[
a_n = 2 + (n-1)\times 4 = 4n-2
\]

**(2) 求 \(\{b_n\}\) 的前 \(n\) 项和**  
由(1)得 \(a_n = 4n-2 = 2(2n-1)\)，则 \(a_{n+1} = 4n+2 = 2(2n+1)\)。  
代入 \(b_n\) 的表达式：  
\[
b_n = \frac{1}{2}\left( \frac{a_{n+1}}{a_n} + \frac{a_n}{a_{n+1}} \right)
= \frac{1}{2}\left( \frac{2n+1}{2n-1} + \frac{2n-1}{2n+1} \right)
\]  
通分计算：  
\[
\frac{2n+1}{2n-1} + \frac{2n-1}{2n+1} = \frac{(2n+1)^2 + (2n-1)^2}{(2n-1)(2n+1)} = \frac{8n^2+2}{4n^2-1} = \frac{2(4n^2+1)}{4n^2-1}
\]  
于是  
\[
b_n = \frac{1}{2} \cdot \frac{2(4n^2+1)}{4n^2-1} = \frac{4n^2+1}{4n^2-1}
\]  
进一步变形：  
\[
b_n = \frac{4n^2-1+2}{4n^2-1} = 1 + \frac{2}{4n^2-1} = 1 + \frac{2}{(2n-1)(2n+1)}
\]  
裂项可得  
\[
\frac{2}{(2n-1)(2n+1)} = \frac{1}{2n-1} - \frac{1}{2n+1}
\]  
所以  
\[
b_n = 1 + \frac{1}{2n-1} - \frac{1}{2n+1}
\]  
设 \(\{b_n\}\) 的前 \(n\) 项和为 \(T_n\)，则  
\[
\begin{aligned}
T_n &= \sum_{k=1}^n \left(1 + \frac{1}{2k-1} - \frac{1}{2k+1}\right) \\
&= n + \left(1 - \frac{1}{3}\right) + \left(\frac{1}{3} - \frac{1}{5}\right) + \cdots + \left(\frac{1}{2n-1} - \frac{1}{2n+1}\right) \\
&= n + 1 - \frac{1}{2n+1}
\end{aligned}
\]  
也可整理为  
\[
T_n = \frac{2n^2+3n}{2n+1}
\]

## 答案
(1) \(a_n = 4n-2\)  
(2) \(T_n = n + 1 - \dfrac{1}{2n+1}\)（或 \(\dfrac{2n^2+3n}{2n+1}\)）

## 总结
本题以等差中项与等比中项为条件，通过消去前 \(n\) 项和得到等差数列的通项，属于数列求通项的基本题型。第二问在通项基础上构造新数列 \(b_n\)，化简后利用裂项相消法求和，考查了代数变形与求和技巧的综合运用。
