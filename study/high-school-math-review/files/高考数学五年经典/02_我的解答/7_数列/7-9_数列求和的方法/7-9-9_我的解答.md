# 7-9-9 我的解答

对应题目：[[01_题目/7_数列/7-9_数列求和的方法/7-9-9.md|7-9-9]]

## 完整题目

9.(2024 云南一模)已知 \( \left\{  {a}_{n}\right\} \) 为等比数列，记 \( {S}_{n} \) ， \( {T}_{n} \) 分别为数列 \( \left\{  {a}_{n}\right\} \) ， \( \left\{  {b}_{n}\right\} \) 的前 \( n \) 项和， \( {S}_{5} = {62} \) ， \( {S}_{10} = {2046},2{T}_{n} = n{b}_{n} + n,{b}_{2} = 3 \) .

(1)求 \( \left\{  {a}_{n}\right\}  ,\left\{  {b}_{n}\right\} \) 的通项公式.

(2)是否存在整数 \( c \) ，使 \( \frac{{b}_{1}}{{a}_{1}} + \frac{{b}_{2}}{{a}_{2}} + \cdots  + \frac{{b}_{n}}{{a}_{n}} < c \) 对任意的正整数 \( n \) 都成立? 若存在,求 \( c \) 的最小值; 若不存在, 请说明理由.

## 题目勘误
无明显勘误。

## 思路分析
(1) 对于等比数列 \(\{a_n\}\)，由前 \(5\) 项和与前 \(10\) 项和可建立关于首项与公比的方程组，结合 \(S_{10} \neq 2S_5\) 排除公比 \(q=1\) 的情况，解出 \(a_1\) 与 \(q\) 即可得 \(\{a_n\}\) 通项。  
对于数列 \(\{b_n\}\)，已知前 \(n\) 项和 \(T_n\) 满足的关系式，利用 \(b_n = T_n - T_{n-1}\,(n\ge 2)\) 推导递推关系，结合 \(b_2=3\) 求出通项（可猜想为等差数列并验证）。  
(2) 求出 \(\frac{b_n}{a_n}\) 的通项后，利用错位相减法求前 \(n\) 项和 \(S_n\)，将 \(S_n\) 化简为一个常数减去一个正项的形式，判断其单调性与极限，由此确定使不等式恒成立的整数 \(c\) 及其最小值。

## 详细解答
### (1) 求 \(\{a_n\}\)，\(\{b_n\}\) 的通项公式
设等比数列 \(\{a_n\}\) 的首项为 \(a_1\)，公比为 \(q\)（\(q \neq 0\)）。  
由 \(S_5=62,\,S_{10}=2046\) 且 \(S_{10} \neq 2S_5\)，可知 \(q\neq 1\)。利用等比数列求和公式：
\[
S_5 = \frac{a_1(1-q^5)}{1-q}=62,\qquad 
S_{10}= \frac{a_1(1-q^{10})}{1-q}=2046.
\]
两式相除得
\[
\frac{1-q^{10}}{1-q^5} = \frac{2046}{62}=33.
\]
令 \(q^5 = t\)，则 \(\frac{1-t^2}{1-t}=1+t=33\)，解得 \(t=32\)，即 \(q^5=32\)，于是 \(q=2\)。  
代回 \(S_5\)：
\[
\frac{a_1(1-2^5)}{1-2}= \frac{a_1(-31)}{-1}=31a_1=62 \implies a_1=2.
\]
所以 \(\{a_n\}\) 的通项公式为
\[
a_n = 2 \cdot 2^{n-1} = 2^n.
\]

对于 \(\{b_n\}\)，已知 \(2T_n = n b_n + n\)。  
当 \(n=1\) 时，\(2T_1 = 1\cdot b_1+1\)，而 \(T_1=b_1\)，代入得 \(2b_1 = b_1+1 \implies b_1 = 1\)。  
当 \(n\ge 2\) 时，由 \(2T_n = n b_n + n\) 与 \(2T_{n-1} = (n-1)b_{n-1}+(n-1)\) 相减：
\[
2b_n = n b_n - (n-1)b_{n-1} + 1,
\]
整理得
\[
(n-2)b_n = (n-1)b_{n-1} - 1. \tag{*}
\]
已知 \(b_1=1\)，\(b_2=3\)，猜想 \(b_n = 2n-1\)（等差数列）。代入 \((*)\) 检验：左边 \((n-2)(2n-1)\)，右边 \((n-1)(2n-3)-1 = 2n^2-5n+3-1=2n^2-5n+2\)，左边展开也是 \(2n^2-5n+2\)，满足。结合初始条件，由数学归纳法可知该猜想成立。  
因此 \(\{b_n\}\) 的通项公式为
\[
b_n = 2n-1.
\]

### (2) 判断是否存在整数 \(c\) 使不等式恒成立
令 \(c_n = \frac{b_n}{a_n} = \frac{2n-1}{2^n}\)，其前 \(n\) 项和为
\[
S_n = \frac{b_1}{a_1} + \frac{b_2}{a_2} + \cdots + \frac{b_n}{a_n} = \sum_{k=1}^n \frac{2k-1}{2^k}.
\]
用错位相减法求和：
\[
S_n = \frac{1}{2} + \frac{3}{2^2} + \frac{5}{2^3} + \cdots + \frac{2n-1}{2^n},
\]
\[
\frac{1}{2}S_n = \frac{1}{2^2} + \frac{3}{2^3} + \cdots + \frac{2n-3}{2^n} + \frac{2n-1}{2^{n+1}}.
\]
两式相减：
\[
\frac{1}{2}S_n = \frac{1}{2} + \left( \frac{2}{2^2} + \frac{2}{2^3} + \cdots + \frac{2}{2^n} \right) - \frac{2n-1}{2^{n+1}}.
\]
其中括号内为等比数列求和：
\[
\frac{2}{2^2} + \frac{2}{2^3} + \cdots + \frac{2}{2^n} = \frac{1}{2} + \frac{1}{2^2} + \cdots + \frac{1}{2^{n-1}} = \frac{\frac{1}{2}\left(1-\frac{1}{2^{n-1}}\right)}{1-\frac{1}{2}} = 1 - \frac{1}{2^{n-1}}.
\]
所以
\[
\frac{1}{2}S_n = \frac{1}{2} + \left(1 - \frac{1}{2^{n-1}}\right) - \frac{2n-1}{2^{n+1}} = \frac{3}{2} - \frac{2}{2^n} - \frac{2n-1}{2^{n+1}}.
\]
两边乘 \(2\)：
\[
S_n = 3 - \frac{4}{2^n} - \frac{2n-1}{2^n} = 3 - \frac{2n+3}{2^n}.
\]
对于任意正整数 \(n\)，\(\frac{2n+3}{2^n} > 0\)，故 \(S_n < 3\)。  
又当 \(n \to \infty\) 时，\(\frac{2n+3}{2^n} \to 0\)，所以 \(S_n \to 3\) 且 \(S_n\) 单调递增。  
因此要使 \(S_n < c\) 对所有正整数 \(n\) 都成立，必须有 \(c \ge 3\)。  
当 \(c=3\) 时，\(S_n < 3\) 恒成立；当 \(c=2\) 时，\(S_4 = 3 - \frac{11}{16} = \frac{37}{16} = 2.3125 > 2\)，不成立。  
所以存在整数 \(c\)，且 \(c\) 的最小值为 \(3\)。

## 答案
(1) \(a_n = 2^n\)，\(b_n = 2n-1\)（\(n \in \mathbb{N}^*\)）。  
(2) 存在，\(c\) 的最小值为 \(3\)。

## 总结
本题综合考查等比数列求和、由和式关系求通项以及差比数列的错位相减法求和。  
- 求等比数列通项时注意讨论公比是否为 \(1\)。  
- 已知 \(T_n\) 与 \(b_n\) 的关系式，通过作差消去 \(T_n\) 是求 \(b_n\) 通项的常用手段。  
- 差比数列求和后用常数减去正项的形式可直观判断和的界限，进而解决恒成立问题中的参数取值。
