# 11-6-7 我的解答

对应题目：[[01_题目/11_概率与统计/11-6_事件的相互独立性、条件概率与全概率公式/11-6-7.md|11-6-7]]

## 完整题目

7. (多选题, 2024 云南昆明三模) 在一个有限样本空间中,事件 \( A, B, C \) 发生的概率满足 \( P\left( A\right) \; = P\left( B\right)  = P\left( C\right)  = \frac{1}{3}, P\left( {A \cup  B}\right)  = \frac{5}{9}, A \) 与 \( C \) 互斥，则下列说法正确的是 ( )

A. \( P\left( {A\bar{C}}\right)  = \frac{1}{3} \)

B. \( A \) 与 \( B \) 相互独立

C. \( P\left( {ABC}\right)  = \frac{1}{27} \)

D. \( P\left( {A \cup  B \cup  C}\right)  \leq  \frac{8}{9} \)

## 题目勘误
无明显勘误。

## 思路分析
本题已知三个事件概率及其部分关系，需逐项判断四个选项。关键信息：\(A\) 与 \(C\) 互斥可得 \(P(AC)=0\)；由加法公式结合 \(P(A \cup B)\) 求出 \(P(AB)\) 并检验独立性；再借助互斥性得出 \(P(ABC)=0\) 判断 C 项；最后用容斥原理表示 \(P(A \cup B \cup C)\) 并利用概率非负性得到上界。

## 详细解答
**已知条件**  
有限样本空间，事件 \(A,B,C\) 满足  
\[
P(A)=P(B)=P(C)=\frac13,\quad P(A\cup B)=\frac59,
\]  
且 \(A\) 与 \(C\) 互斥，因此  
\[
P(AC)=0 .
\]

---

**选项 A**  
\(A\bar{C}\) 表示 \(A\) 发生且 \(C\) 不发生的事件。因为 \(A\) 与 \(C\) 互斥，故 \(A\) 发生时 \(C\) 必然不发生，即 \(A\subseteq \bar{C}\)，从而 \(A\bar{C}=A\)。  
\[
P(A\bar{C}) = P(A) - P(AC) = \frac13 - 0 = \frac13 .
\]  
所以 A 正确。

---

**选项 B**  
由概率的加法公式：  
\[
P(A\cup B) = P(A) + P(B) - P(AB),
\]  
代入已知：  
\[
\frac59 = \frac13 + \frac13 - P(AB) \;\Longrightarrow\; \frac59 = \frac23 - P(AB),
\]  
解得  
\[
P(AB) = \frac23 - \frac59 = \frac19 .
\]  
又因为  
\[
P(A)P(B) = \frac13 \times \frac13 = \frac19 = P(AB),
\]  
所以事件 \(A\) 与 \(B\) 相互独立。B 正确。

---

**选项 C**  
\(ABC\) 表示三个事件同时发生。由于 \(A\) 与 \(C\) 互斥，必有 \(AC = \varnothing\)，于是  
\[
ABC = (AC) \cap B = \varnothing \cap B = \varnothing,
\]  
故  
\[
P(ABC) = 0 \neq \frac{1}{27}.
\]  
C 错误。

---

**选项 D**  
由三个事件的容斥原理：  
\[
P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC).
\]  
代入已得数据：  
\[
P(A \cup B \cup C) = \frac13 + \frac13 + \frac13 - \frac19 - 0 - P(BC) + 0 = 1 - \frac19 - P(BC) = \frac89 - P(BC).
\]  
因为任意概率非负，\(P(BC) \geq 0\)，所以  
\[
P(A \cup B \cup C) \leq \frac89 .
\]  
D 正确。

---

## 答案
ABD

## 总结
解题的关键在于利用“互斥”得出交事件概率为零，并通过概率的加法公式与容斥原理逐步推导各交、并事件的概率。此外，判断独立性只需验证 \(P(AB)=P(A)P(B)\)；求概率上界时应结合概率的非负性，把表达式中未直接给出的项（如 \(P(BC)\)）消去或放缩。
