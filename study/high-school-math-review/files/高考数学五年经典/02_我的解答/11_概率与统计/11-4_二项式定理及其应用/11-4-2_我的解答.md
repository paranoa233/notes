# 11-4-2 我的解答

对应题目：[[01_题目/11_概率与统计/11-4_二项式定理及其应用/11-4-2.md|11-4-2]]

## 完整题目

2.(2024 江苏苏锡常镇调研(一)设(1 + 2x) \( {}^{5} = \; {a}_{0} + {a}_{1}x + {a}_{2}{x}^{2} + \cdots  + {a}_{5}{x}^{5} \) ,则 \( {a}_{1} + {a}_{2} + \cdots  + {a}_{5} \; = \) ( )

A. -2 B. -1 C. 242 D. 243

## 题目勘误
无明显勘误。
## 思路分析
本题求二项展开式中除常数项外各项系数之和。注意到 \(a_0\) 是常数项，而 \(a_1+a_2+\dots+a_5\) 是所有非零次项系数之和。利用赋值法：先令 \(x=0\) 求出 \(a_0\)，再令 \(x=1\) 求出所有系数之和 \(a_0+a_1+\cdots+a_5\)，两者相减即得所求。
## 详细解答
已知
\[
(1+2x)^5 = a_0 + a_1 x + a_2 x^2 + \cdots + a_5 x^5.
\]

**第一步：求常数项 \(a_0\)。**
令 \(x = 0\)，得
\[
a_0 = (1+2\cdot 0)^5 = 1^5 = 1.
\]

**第二步：求所有系数之和。**
令 \(x = 1\)，得
\[
a_0 + a_1 + a_2 + \cdots + a_5 = (1+2\cdot 1)^5 = 3^5 = 243.
\]

**第三步：求 \(a_1 + a_2 + \cdots + a_5\)。**
将上两式相减：
\[
\begin{aligned}
a_1 + a_2 + \cdots + a_5 &= (a_0 + a_1 + a_2 + \cdots + a_5) - a_0 \\
&= 243 - 1 = 242.
\end{aligned}
\]

故所求值为 \(242\)。
## 答案
C
## 总结
对于多项式展开的系数和问题，赋值法是常用技巧。一般先令所有变量为 \(1\) 求系数总和，再根据题目需要令变量为适当值（如 \(0\)）求出特定项，最后通过加减得到部分系数的和。本题中令 \(x=0\) 分离常数项是关键一步。
