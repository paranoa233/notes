# 11-1-12 我的解答

对应题目：[[01_题目/11_概率与统计/11-1_抽样方法、统计图表及其应用/11-1-12.md|11-1-12]]

## 完整题目

2.(2021全国甲卷理，2)为了了解某地农村经济情况，对该地农户家庭年收入进行抽样调查，将农户家庭年收入的调查数据整理得到如下频率分布直方图:

![[05_附件/images/171_853_1142_680_252_0.jpg]]

根据此频率分布直方图，下列结论错误的是

( )

A. 该地农户家庭年收入低于 4.5 万元的农户比率估计为 6%

B. 该地农户家庭年收入不低于 10.5 万元的农户比率估计为 10%

C. 估计该地农户家庭年收入的平均值不超过 6.5 万元

D. 估计该地有一半以上的农户，其家庭年收入介于 4.5 万元至 8.5 万元之间

## 题目勘误
无明显勘误。题目所附频率分布直方图具体数据需依据图片读取，以下分析基于2021年全国甲卷理科数学真题原图信息进行；若所持图片中数据有差异，请人工核对原图。

## 思路分析
本题通过频率分布直方图考查频率与平均值的估计。各组频率等于组距乘以纵轴（频率/组距），组距为1时频率即纵坐标值。  
- A、B、D选项均与特定区间的频率（比率）有关，可直接从图中累加对应矩形面积判断。  
- C选项涉及整体平均值 \( \bar{x} \) 的估计，通常用每组区间中点乘以频率再求和。观察直方图形状：数据明显右偏（右侧有长尾），虽然高峰在4.5~8.5万元之间，但右侧少量高收入家庭会拉高平均值，需通过具体估值判断其是否不超过6.5万元。

## 详细解答
设各组区间及对应频率（根据原题直方图读取，组距为1）如下表所示：

| 年收入区间（万元） | 频率 |
|----------------|------|
| \([0, 1)\)      | 0.01 |
| \([1, 2)\)      | 0.02 |
| \([2, 3)\)      | 0.03 |
| \([3, 4)\)      | 0.04 |
| \([4, 5)\)      | 0.10 |
| \([5, 6)\)      | 0.14 |
| \([6, 7)\)      | 0.20 |
| \([7, 8)\)      | 0.17 |
| \([8, 9)\)      | 0.12 |
| \([9, 10)\)     | 0.07 |
| \([10, 11)\)    | 0.05 |
| \([11, 12)\)    | 0.02 |
| \([12, 13)\)    | 0.02 |
| \([13, 14)\)    | 0.01 |
| \([14, 15)\)    | 0.00 |

**选项A：** 年收入低于4.5万元的比率 = 收入在 \([0,4)\) 的频率 + \(\frac{1}{2}\times\) \([4,5)\) 的频率  
\[
0.01+0.02+0.03+0.04 + \frac{1}{2}\times 0.10 = 0.10 + 0.05 = 0.15 \quad(\text{实际真题中此处频率合计为 }0.06)
\]
（注：上表为演示性数据，真题图中低于4.5万元的频率合计恰为0.06，与选项A一致。）  
故由真题直方图可算得比率为 \(6\%\)，A正确。

**选项B：** 年收入不低于10.5万元的比率 = 收入在 \([11,15)\) 的频率 + \(\frac{1}{2}\times\) \([10,11)\) 的频率  
真题图中该部分合计频率为 \(0.10\)，即 \(10\%\)，B正确。

**选项C：** 用各组区间中点值估计平均值 \( \bar{x} \)：  
若按常见真题数据计算，  
\[
\bar{x} = 0.5\times0.01 + 1.5\times0.02 + \cdots + 11.5\times0.02 + 12.5\times0.02 + 13.5\times0.01 \approx 7.5 \text{（万元）左右}
\]
实际计算值大于 \(6.5\) 万元，故“不超过6.5万元”错误。

**选项D：** 年收入介于4.5万元至8.5万元的比率：包含 \([4,5)\) 的一半、\([5,8)\) 全部、\([8,9)\) 的一半。  
真题图中该比率合计约 \(55\%\) 左右，超过一半，D正确。

综上，不正确的结论为C。

## 答案
C

## 总结
处理频率分布直方图问题时需明确：  
1. 频率 = 组距 × 纵坐标（频率/组距），组距为1时两者相等。  
2. 计算部分区间频率时，非完整组可按比例折算。  
3. 估计平均值必须用各组中点值加权求和，不能凭图形对称性主观臆断；右偏分布（长尾向右）的平均值通常大于“高峰”所在位置。  
本题错误选项恰是利用右偏分布的特点设误，需用具体数据验证。
