# 11-2-5 我的解答

对应题目：[[01_题目/11_概率与统计/11-2_样本的数字特征/11-2-5.md|11-2-5]]

## 完整题目

5.(多选题，2024 广东汕头一模)某次数学考试后, 为分析学生的学习情况, 某校从某年级中随机抽取了 200 名学生的成绩, 整理得到如图所示的频率分布直方图. 为进一步分析高分学生的成绩分布情况，计算得到这 200 名学生中，位于[80, 90)内的学生成绩的方差为12，位于 \( \left\lbrack  {{90},{100}}\right\rbrack \) 内的学生成绩的方差为 10 , 则 ( )

参考公式:样本划分为 2 层，各层的容量、平均数和方差分别为 \( m,\bar{x},{s}_{1}^{2};n,\bar{y},{s}_{2}^{2} \) . 记样本平均数为 \( \overline{\omega } \) ,样本方差为 \( {s}^{2},{s}^{2} = \frac{m}{m + n}\left\lbrack  {{s}_{1}^{2} + {\left( \bar{x} - \overline{\omega }\right) }^{2}}\right\rbrack   + \; \frac{n}{m + n}\left\lbrack  {{s}_{2}^{2} + {\left( \bar{y} - \overline{\omega }\right) }^{2}}\right\rbrack  . \)

![[05_附件/images/172_1007_1386_355_292_0.jpg]]

A. \( a = {0.004} \)

B. 估计该年级学生成绩的中位数为 77.14

C. 估计该年级在 80 分及以上的学生成绩的平均数为 87.50

D. 估计该年级在 80 分及以上的学生成绩的方差为 30.25

## 题目勘误
图片信息缺失，无法获取频率分布直方图的具体数据（如各组区间、频率/组距、字母 \(a\) 的含义等）；仅凭文字信息无法完整确定 A、B 选项的正误，需人工核对直方图后再判定。

## 思路分析
本题主要考查频率分布直方图的识图、中位数与平均数的估计，以及分层样本方差公式的应用。由于图片缺失，只能基于题中给出的数字特征（样本容量 200、位于 \([80,90)\) 内的成绩方差为 12、位于 \([90,100]\) 内的成绩方差为 10）以及选项 C、D 的数据，结合分层方差公式进行反向验证。对于 A、B 选项，必须依赖直方图的完整数据才能判断。

## 详细解答
已知样本容量 \(n=200\)，将 \(80\) 分及以上的学生分为两层：
第一层：\([80,90)\)，设容量为 \(m\)，平均数为 \(\bar{x}\)，方差 \(s_1^2=12\)；
第二层：\([90,100]\)，设容量为 \(n_1\)，平均数为 \(\bar{y}\)，方差 \(s_2^2=10\)。
在不知道实际直方图的情况下，选项 C 直接给出了这两层合并后的平均数估计值 \(87.50\)；选项 D 给出了合并方差 \(30.25\)。我们可以利用分层抽样方差公式检验两者的协调性。

分层样本平均数为：
\[
\overline{\omega} = \frac{m\bar{x} + n_1\bar{y}}{m+n_1}
\]
分层样本方差公式（题中已给）：
\[
s^2 = \frac{m}{m+n_1}\left[s_1^2 + (\bar{x} - \overline{\omega})^2\right] + \frac{n_1}{m+n_1}\left[s_2^2 + (\bar{y} - \overline{\omega})^2\right]
\]
在频率分布直方图中，通常以组中值估计各区间平均分，即取 \(\bar{x}=85\)，\(\bar{y}=95\)。若选项 C 成立，则平均分 \(\overline{\omega}=87.50\)。此时：
\[
87.50 = \frac{m\cdot 85 + n_1\cdot 95}{m+n_1} \;\Longrightarrow\; 87.5(m+n_1) = 85m + 95n_1
\]
整理得 \(2.5m = 7.5n_1\)，所以 \(m = 3n_1\)，即容量比 \(m:n_1 = 3:1\)。
代入分层方差公式：
\[
\begin{aligned}
s^2 &= \frac{3}{4}\left[12 + (85-87.5)^2\right] + \frac{1}{4}\left[10 + (95-87.5)^2\right] \\
&= \frac{3}{4}\left[12 + 6.25\right] + \frac{1}{4}\left[10 + 56.25\right] \\
&= \frac{3}{4}\times 18.25 + \frac{1}{4}\times 66.25 \\
&= 13.6875 + 16.5625 = 30.25
\end{aligned}
\]
结果恰与选项 D 吻合。这说明 **若选项 C 正确，且假定各组平均数取组中值，则选项 D 必然正确**。但是否 C 真正成立，取决于直方图中 \([80,90)\) 与 \([90,100]\) 的真实人数比例是不是 \(3:1\)，以及总样本中这两层人数之和是否与平均分 \(87.50\) 相容。因此 C 和 D 的正确性必须由直方图原始数据确认。

对于 A 选项（\(a=0.004\)）与 B 选项（中位数约为 \(77.14\)），需要根据直方图的完整形状（各组频率、组距）计算频率总和并确定中位数位置，无图则无法判断。

## 答案
因题目所附频率分布直方图缺失，无法依据完整信息对 A、B、C、D 全部选项做出确切判定。若能通过直方图确认 \(80\) 分及以上两组学生数之比为 \(3:1\)，则 C 和 D 同时正确；A、B 需结合直方图验证。

## 总结
解决此类统计多选题时，应注意频率分布直方图的识读以及分层方差公式的灵活运用。本题的关键在于：即使部分图形信息缺失，仍可借助公式和选项间的数量关系进行推理，但最终结论的可靠性完全建立在完整数据之上。面对信息不完整的题目，应先指出缺失内容，再基于已有数据给出部分推理。
