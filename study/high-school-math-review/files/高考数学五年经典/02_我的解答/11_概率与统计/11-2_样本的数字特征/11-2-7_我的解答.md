# 11-2-7 我的解答

对应题目：[[01_题目/11_概率与统计/11-2_样本的数字特征/11-2-7.md|11-2-7]]

## 完整题目

7. (多选题, 2024 湖北武汉 4 月调研) 如图所示, 下列频率分布直方图显示了三种不同的分布形态. 图(1)形成对称形态，图(2)形成“右拖尾”形态，图(3)形成“左拖尾”形态，根据所给图作出以下判断，正确的是 ( )

![[05_附件/images/173_164_1222_611_411_0.jpg]]

A. 图(1)的平均数 \( = \) 中位数 \( = \) 众数

B. 图(2)的平均数 \( < \) 众数 \( < \) 中位数

C. 图(2)的众数 \( < \) 中位数 \( < \) 众数

B. 图(3)的平均数 \( < \) 中位数

## 题目勘误
无明显勘误。
（注：题干选项编号存在输入笔误：选项列表中第三个应为“C. 图(2)的众数 < 中位数 < 平均数”，原文误作“众数 < 中位数 < 众数”；第四个选项编号应为“D”，原文误作“B”。以下解答按正确编号与内容分析。）

## 思路分析
本题考查频率分布直方图的分布形态与数字特征（平均数、中位数、众数）之间的关系。三个图形分别代表对称分布、右偏分布（右拖尾）和左偏分布（左拖尾）。对于单峰分布：
- 对称分布：平均数 = 中位数 = 众数；
- 右偏（正偏）分布：众数 < 中位数 < 平均数；
- 左偏（负偏）分布：平均数 < 中位数 < 众数。
据此逐项判断选项正误。

## 详细解答
根据频率分布直方图的形态与数字特征关系：
- 图(1)为对称形态，其平均数、中位数和众数三者相等。
  **A选项**：图(1)的平均数 \( = \) 中位数 \( = \) 众数，正确。
- 图(2)为“右拖尾”形态，即右偏分布。数据右侧有长尾，平均数受极端值影响向右移动，故有 **众数 < 中位数 < 平均数**。
  **B选项**：图(2)的平均数 < 众数 < 中位数，关系错误，不正确。
  **C选项**（更正后）：图(2)的众数 < 中位数 < 平均数，符合右偏分布特征，正确。
- 图(3)为“左拖尾”形态，即左偏分布。数据左侧有长尾，平均数向左移动，故有 **平均数 < 中位数 < 众数**。
  由此可知平均数 < 中位数成立。
  **D选项**：图(3)的平均数 < 中位数，正确。

综合所述，正确的选项为 A、C、D。

## 答案
A、C、D

## 总结
解此类题需牢记三种单峰分布的数字特征关系：
1. 对称分布：平均数 = 中位数 = 众数；
2. 右偏（正偏）分布：众数 < 中位数 < 平均数；
3. 左偏（负偏）分布：平均数 < 中位数 < 众数。
由“拖尾”方向可直观判断：拖尾在右则平均数大于中位数，拖尾在左则平均数小于中位数。
