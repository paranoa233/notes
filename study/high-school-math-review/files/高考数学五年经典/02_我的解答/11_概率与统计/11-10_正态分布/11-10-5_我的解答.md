# 11-10-5 我的解答

对应题目：[[01_题目/11_概率与统计/11-10_正态分布/11-10-5.md|11-10-5]]

## 完整题目

5.(2024 福建泉州质监(三))中心极限定理是概率论中的一个重要结论. 根据该定理，若随机变量 \( \xi  \sim  B\left( {n, p}\right) \) ,则当 \( {np} > 5 \) 且 \( n\left( {1 - p}\right)  > 5 \) 时, \( \xi \) 可以由服从正态分布的随机变量 \( \eta \) 近似替代,且 \( \xi \) 的期望与方差分别与 \( \eta \) 的均值与方差近似相等. 现投掷一枚质地均匀分布的骰子 2500 次，利用正态分布估算骰子向上的点数为偶数的次数少于 1300 的概率约为 ( )

附:若 \( \eta  \sim  N\left( {\mu ,{\sigma }^{2}}\right) \) ,则 \( P\left( {\mu  - \sigma  < \eta  < \mu  + \sigma }\right)  \approx \; {0.6827}, P\left( {\mu  - {2\sigma } < \eta  < \mu  + {2\sigma }}\right)  \approx  {0.9545}, P(\mu  - \; {3\sigma } < \eta  < \mu  + {3\sigma }) \approx  {0.9973}. \)

A. 0.0027 B. 0.5

C. 0.8414 D. 0.9773

## 题目勘误
无明显勘误。

## 思路分析
掷一枚质地均匀的骰子 \(2500\) 次，每次出现偶数点的概率 \(p = \frac{3}{6} = \frac{1}{2}\)。设 \(\xi\) 为出现偶数点的总次数，则 \(\xi \sim B(2500, 0.5)\)。根据中心极限定理，当 \(np\) 和 \(n(1-p)\) 均大于 \(5\) 时，可用正态分布近似。先计算正态分布的参数 \(\mu\) 和 \(\sigma\)，然后利用标准的正态分布区间概率，估算 \(\xi < 1300\) 的概率。

## 详细解答
1. **确定二项分布参数**  
   掷一枚均匀骰子，点数为偶数的概率 \(p = \frac{3}{6} = \frac{1}{2}\)。  
   试验次数 \(n = 2500\)。  
   \(\xi \sim B(2500, 0.5)\)。

2. **检验近似条件**  
   \(np = 2500 \times 0.5 = 1250 > 5\)  
   \(n(1-p) = 2500 \times 0.5 = 1250 > 5\)  
   条件满足，可用正态分布近似。

3. **正态分布的均值与方差**  
   均值 \(\mu = np = 1250\)  
   方差 \(\sigma^2 = np(1-p) = 2500 \times 0.5 \times 0.5 = 625\)  
   标准差 \(\sigma = \sqrt{625} = 25\)  
   因此可用 \(\eta \sim N(1250, 25^2)\) 近似 \(\xi\) 的分布。

4. **将所求概率化为正态分布问题**  
   要求“骰子向上的点数为偶数的次数少于 \(1300\) 的概率”，即 \(P(\xi < 1300)\)。  
   用正态分布近似：\(P(\eta < 1300)\)。

5. **计算 \(1300\) 对应的标准差倍数**  
   \[
   1300 = \mu + k\sigma = 1250 + 25k
   \]
   解得 \(k = \frac{1300 - 1250}{25} = 2\)。  
   即 \(1300 = \mu + 2\sigma\)。

6. **利用给出的正态分布区间概率求值**  
   题目给出：  
   \(P(\mu - \sigma < \eta < \mu + \sigma) \approx 0.6827\)  
   \(P(\mu - 2\sigma < \eta < \mu + 2\sigma) \approx 0.9545\)  
   \(P(\mu - 3\sigma < \eta < \mu + 3\sigma) \approx 0.9973\)。  
   标准正态分布的对称性：\(\eta\) 的分布关于 \(\mu\) 对称。  
   \[
   P(\eta < \mu + 2\sigma) = P(\eta < \mu) + P(\mu \le \eta < \mu + 2\sigma) = 0.5 + \frac{P(\mu - 2\sigma < \eta < \mu + 2\sigma)}{2}
   \]
   代入数值：
   \[
   P(\eta < 1300) \approx 0.5 + \frac{0.9545}{2} = 0.5 + 0.47725 = 0.97725
   \]
   四舍五入约等于 \(0.9773\)。

7. **结论**  
   估算该概率约为 \(0.9773\)，对应选项 D。

## 答案
D

## 总结
本题主要考查二项分布的正态近似。关键在于正确求出 \(n, p\) 后得到均值 \(\mu\) 和标准差 \(\sigma\)，并将待求事件“次数少于 1300”转化为标准正态分布中 \(\mu + 2\sigma\) 左侧的概率。利用正态分布的对称性及已知区间概率即可快速得到结果。注意此处无需连续性校正，直接利用给定的区间概率近似即可。
