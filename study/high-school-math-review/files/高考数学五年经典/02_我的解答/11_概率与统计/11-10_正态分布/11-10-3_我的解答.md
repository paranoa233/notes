# 11-10-3 我的解答

对应题目：[[01_题目/11_概率与统计/11-10_正态分布/11-10-3.md|11-10-3]]

## 完整题目

3. (2024 安徽合肥一模)已知随机变量 \( X \) 服从正态分布 \( N\left( {2,{\sigma }^{2}}\right) \) ，且 \( P\left( {2 < X \leq  {2.5}}\right)  = {0.36} \) ，则 \( P\left( {X > {1.5}}\right)  = \) ( )

A. 0.14 B. 0.62 C. 0.72 D. 0.86

## 题目勘误
无明显勘误。

## 思路分析
本题主要考查正态分布的对称性。已知随机变量 \( X \sim N(2, \sigma^2) \)，均值 \( \mu=2 \)。利用正态曲线关于直线 \( x=\mu \) 对称的性质，可以将未知区间 \( (1.5, +\infty) \) 的概率转化为已知区间概率与对称区间概率之和，从而快速求解。

## 详细解答
因为随机变量 \( X \sim N(2,\sigma^2) \)，所以其概率密度曲线关于直线 \( x=2 \) 对称，且 \( P(X \ge 2) = 0.5 \)。

由对称性可知：
\[ P(1.5 \le X < 2) = P(2 < X \le 2.5) = 0.36. \]
于是，
\[ P(X > 1.5) = P(1.5 < X < 2) + P(X \ge 2) = 0.36 + 0.5 = 0.86. \]

（另解：由 \( P(2 < X \le 2.5) = 0.36 \) 及对称性得 \( P(X \le 1.5) = P(X \ge 2.5) = 0.5 - 0.36 = 0.14 \)，故 \( P(X > 1.5) = 1 - P(X \le 1.5) = 1 - 0.14 = 0.86 \)。）

## 答案
D

## 总结
求解正态分布相关概率的关键是画图并利用对称轴 \( x=\mu \)。记住以下常用结论：
1. \( P(X \ge \mu) = P(X \le \mu) = 0.5 \)；
2. 对于任意 \( a > 0 \)，\( P(\mu - a < X < \mu) = P(\mu < X < \mu + a) \)；
3. \( P(X < \mu - a) = P(X > \mu + a) \)。
