# 导入入口

把待整理的错题材料先放在这里。

## 推荐文件名

- 单题图片：`10-3-3_错题.png`
- 单题文本：`10-3-W001.md`
- 多题材料：`2026-05-28_周测错题.md`
- 原始试卷：`2026-05-28_数学周测.pdf`

## 给 agent 的处理要求

1. 先读取 `../00_控制台/Agent导入与分析说明.md`。
2. 再用 `../90_模板/错题模板.md` 生成错题页。
3. 分类不确定时放入 `../01_错题/00_待分类`。
4. 原始材料保留在本文件夹，或复制到 `../05_附件/docs` 后在错题页链接。
