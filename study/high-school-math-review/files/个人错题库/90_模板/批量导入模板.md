# 批量导入模板

把多道错题整理给 agent 时，建议按下面格式输出。每道题之间用 `---` 分隔。

```markdown
## 错题

来源：
原题号：
分类线索：
题型：
难度：
优先级：
考试/练习日期：

### 题目

### 我的错误答案

### 标准答案

### 解析

### 我觉得错在哪里

### 下次重做前提醒我

### 附件
```

## 导入要求

- 一段只放一道题。
- 不确定的字段可以留空。
- 如果题目来自图片，先写图片文件名。
- 如果有原题库题号，必须写在 `原题号`。
- 如果不知道分类，写关键词，例如“导数构造函数”“数列递推”“圆锥曲线定点”。
