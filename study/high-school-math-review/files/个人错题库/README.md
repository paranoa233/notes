这个库用来收集、复盘和分析自己的高中数学错题。分类沿用原来的 `Obsidian题库/01_题目` 体系：大类为专题，小类为讲次或知识模块。

## Obsidian 查看

这个文件夹已经带有 `.obsidian` 配置，可以直接在 Obsidian 中作为 vault 打开。打开后从 `00_控制台/总览.md` 进入即可。

已包含：

- 控制台页面：总览、错题浏览、待复习、错因分析、最近导入。
- 学习工作台：每日入口，显示复习队列、薄弱题和待确认分类。
- 离线 HTML 看板：`00_控制台/错题看板.html`。
- 公式兼容插件：支持 `\( ... \)` 和 `\[ ... \]` 公式显示。
- 模板目录：Obsidian 模板插件指向 `90_模板`。

## 使用方式

1. 新错题先放到 `99_导入入口`，可以是 Markdown、图片、PDF、Word 或纯文本。
2. 让 agent 按 `00_控制台/Agent导入与分析说明.md` 导入。
3. 每道错题整理成一份 Markdown，放入 `01_错题/专题/小节`。
4. 图片和原始材料放入 `05_附件`，错题页里用链接引用。
5. 复习后在错题页更新 `status`、`mastery`、`last_review`、`next_review` 和重练记录。

## 命名建议

- 来自原题库：`原题号_错题.md`，例如 `10-3-3_错题.md`。
- 非原题库来源：`小节编号-W001.md`，例如 `10-3-W001.md`。
- 不确定分类：先放入 `01_错题/00_待分类`，待分析后再移动。

## 状态建议

- `待复习`：已经导入，尚未重做。
- `一刷通过`：第一次复习做对。
- `仍错`：复习时仍然出错。
- `已掌握`：最近两次独立完成正确，且能说清方法。
