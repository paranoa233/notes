# 质量检查

用于发现导入不完整、分类不确定或字段缺失的错题。

## 缺少关键字段

```dataview
TABLE wrong_id AS 编号, source_problem_id AS 原题号, module AS 专题, topic AS 小节, status AS 状态, created AS 创建
FROM "01_错题"
WHERE type = "wrong_problem" AND (
  !wrong_id OR !lesson_id OR !module OR !topic OR !wrong_reason OR !knowledge_points OR !methods OR !anti_mistake_tip OR !created
)
SORT file.mtime DESC
```

## 分类待确认

```dataview
TABLE wrong_id AS 编号, source AS 来源, module AS 专题, topic AS 小节, classification_confidence AS 置信度
FROM "01_错题"
WHERE type = "wrong_problem" AND classification_confidence = "low"
SORT file.mtime DESC
```

## 质量标记

```dataview
TABLE wrong_id AS 编号, module AS 专题, topic AS 小节, quality_flags AS 质量标记
FROM "01_错题"
WHERE type = "wrong_problem" AND quality_flags
SORT file.mtime DESC
```
