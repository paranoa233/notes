# 导入错题

## 手动导入

1. 把图片、PDF、Word 或文字材料放进 `99_导入入口`。
2. 用 [[../90_模板/错题模板|错题模板]] 整理成一题一页。
3. 按 [[分类清单.csv]] 放入对应目录。
4. 不确定分类时先放到 `01_错题/00_待分类`。

## 让 agent 导入

把材料放进 `99_导入入口` 后，对 agent 说：

```text
请按 个人错题库/00_控制台/Agent导入与分析说明.md 导入这些错题，并用错题模板整理。
```

agent 应保留原始材料，并在错题页中写清楚题目、错误答案、标准答案、错因、正确解法和防错提醒。

## 导入后刷新

批量导入后运行：

```powershell
.\tools\update_wrong_bank.ps1
```

这样会更新 [[错题统计]]、[[系统体检]]、[[复习日历]] 和 [[可视化看板]]。
