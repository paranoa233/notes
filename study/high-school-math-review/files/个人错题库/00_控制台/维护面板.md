# 维护面板

## 一键维护

在 `个人错题库` 目录运行：

```powershell
.\tools\update_wrong_bank.ps1
```

它会刷新：

- [[错题统计]]
- [[系统体检]]
- [[复习日历]]
- [[可视化看板]]
- [[内容复习建议]]
- [[相似题推荐]]

## 手动维护

```powershell
python tools\refresh_wrong_bank.py
```

## 维护顺序

1. 批量导入错题。
2. 运行一键维护。
3. 打开 [[系统体检]]。
4. 修复字段缺失、日期格式、分类不确定等问题。
5. 打开 [[学习工作台]] 开始复习。
