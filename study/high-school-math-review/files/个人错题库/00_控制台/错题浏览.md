# 错题浏览

```dataview
TABLE wrong_id AS 编号, source_problem_id AS 原题号, priority AS 优先级, module AS 专题, topic AS 小节, wrong_reason AS 错因, mastery AS 掌握度, status AS 状态, next_review AS 下次复习
FROM "01_错题"
WHERE type = "wrong_problem"
SORT priority ASC, module_id ASC, lesson_id ASC, wrong_id ASC
```

## 不用 Dataview 时

直接在左侧文件树打开 `01_错题`，目录已经按原题库分类排好。
