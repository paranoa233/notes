# 字段字典

本页定义错题页 YAML 字段。agent 导入、统计和体检时以这里为准。

## 身份字段

| 字段 | 必填 | 说明 | 示例 |
|---|---|---|---|
| `type` | 是 | 固定为 `wrong_problem` | `wrong_problem` |
| `wrong_id` | 是 | 错题库内部编号 | `10-3-1-W001` |
| `source_problem_id` | 否 | 原题库题号 | `10-3-1` |
| `source` | 是 | 题目来源 | `原题库 10-3-1` |
| `original_file` | 否 | 原始文件路径 | `Obsidian题库/.../10-3-1.md` |

## 分类字段

| 字段 | 必填 | 说明 | 示例 |
|---|---|---|---|
| `module_id` | 是 | 原题库大类编号 | `10` |
| `module` | 是 | 原题库大类名称 | `数列` |
| `lesson_id` | 是 | 原题库小节编号 | `10-3` |
| `lesson_title` | 否 | 原题库小节标题 | `数列：特征根` |
| `topic` | 是 | 小节主题 | `特征根` |
| `classification_confidence` | 是 | 分类置信度 | `high` / `medium` / `low` |

## 复习字段

| 字段 | 必填 | 说明 | 建议值 |
|---|---|---|---|
| `priority` | 是 | 复习优先级 | `P1` / `P2` / `P3` |
| `status` | 是 | 当前复习状态 | `待复习` / `仍错` / `已掌握` |
| `mastery` | 是 | 掌握度，1 到 5 | `2` |
| `review_stage` | 是 | 复习阶段 | `new` / `reviewing` / `stable` / `archived` |
| `repeat_count` | 是 | 重复出错次数 | `0` |
| `last_review` | 否 | 上次复习日期 | `2026-05-28` |
| `next_review` | 否 | 下次复习日期 | `2026-05-31` |

## 分析字段

| 字段 | 必填 | 说明 |
|---|---|---|
| `wrong_reason` | 是 | 错因标签，可多选 |
| `knowledge_points` | 是 | 知识点，可多选 |
| `methods` | 是 | 解法或方法，可多选 |
| `anti_mistake_tip` | 是 | 一句话防错抓手 |
| `quality_flags` | 否 | 质量问题，如 `OCR待校对`、`答案待确认` |

## 质量规则

- 机器统计只读取 `type: "wrong_problem"` 的文件。
- 缺失必填字段会出现在 [[系统体检]]。
- `classification_confidence = low` 会出现在 [[质量检查]]。
- `next_review` 使用 `YYYY-MM-DD` 格式。
