# 复习队列

优先顺序：P1、仍错题、到期题、掌握度低的题。

```dataviewjs
const today = dv.date("today");
const pages = dv.pages('"01_错题"')
  .where(p => p.type === "wrong_problem")
  .where(p => {
    const next = p.next_review ? dv.date(p.next_review) : null;
    const status = String(p.status ?? "");
    return status === "仍错" || status === "待复习" || (next && next <= today);
  })
  .sort(p => String(p.priority ?? "P2"), "asc")
  .sort(p => String(p.status ?? "") === "仍错" ? 0 : 1, "asc")
  .sort(p => Number(p.mastery ?? 9), "asc")
  .sort(p => p.next_review ?? "9999-12-31", "asc");

dv.table(
  ["错题", "优先级", "专题", "小节", "状态", "掌握度", "重错次数", "下次复习", "防错抓手"],
  pages.map(p => [
    p.file.link,
    p.priority ?? "P2",
    p.module ?? "待补充",
    p.topic ?? "待补充",
    p.status ?? "待复习",
    p.mastery ?? "",
    p.repeat_count ?? 0,
    p.next_review ?? "",
    p.anti_mistake_tip ?? ""
  ])
);
```

## 复习后更新

复习完一题，至少更新这几个字段：

- `status`：`一刷通过`、`仍错`、`二刷通过`、`已掌握`
- `mastery`：1 到 5
- `last_review`：今天日期
- `next_review`：下一次复习日期
- `repeat_count`：仍错时加 1
