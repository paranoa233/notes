# 个人错题库总览

这个页面是 Obsidian 入口。若 Dataview 已安装，下面会自动显示错题统计；若没有安装，也可以打开 [[可视化看板]]。

```dataviewjs
const pages = dv.pages('"01_错题"').where(p => p.type === "wrong_problem");
const total = pages.length;
const pending = pages.where(p => String(p.status ?? "") === "待复习").length;
const wrongAgain = pages.where(p => String(p.status ?? "") === "仍错").length;
const mastered = pages.where(p => String(p.status ?? "") === "已掌握").length;
const avgMastery = total ? (pages.array().reduce((sum, p) => sum + Number(p.mastery ?? 0), 0) / total).toFixed(1) : "0.0";
function card(title, value, note = "") {
  return `<div style="border:1px solid #e5e7eb;border-radius:8px;padding:12px;background:#fff">
    <div style="color:#64748b;font-size:13px">${title}</div>
    <div style="font-size:28px;font-weight:700;margin-top:4px">${value}</div>
    <div style="color:#64748b;font-size:12px;margin-top:4px">${note}</div>
  </div>`;
}
dv.paragraph(`<div style="display:grid;grid-template-columns:repeat(5,minmax(0,1fr));gap:12px;margin:12px 0 18px">
  ${card("总错题", total, "type = wrong_problem")}
  ${card("待复习", pending, "需要安排重做")}
  ${card("仍错", wrongAgain, "优先处理")}
  ${card("已掌握", mastered, "可以低频回看")}
  ${card("平均掌握度", avgMastery, "1 到 5 分")}
</div>`);
const modules = pages.groupBy(p => `${p.module_id ?? ""} ${p.module ?? "未分类"}`)
  .map(g => [g.key.trim(), g.rows.length, g.rows.where(p => String(p.status ?? "") === "仍错").length, g.rows.where(p => String(p.status ?? "") === "已掌握").length]);
dv.table(["专题", "错题数", "仍错", "已掌握"], modules);
```

## 每周复盘建议

每周让 agent 扫描 `01_错题`，输出：

- 新增错题数，按专题和小节统计。
- 高频错因，按 `wrong_reason` 统计。
- 需要优先复习的题，按 `next_review`、`mastery`、`repeat_count` 排序。
- 已掌握题和仍需二刷题。
