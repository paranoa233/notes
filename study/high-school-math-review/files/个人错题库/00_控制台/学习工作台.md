## 内容闭环

- [[今日学习路线]]：按“错题 -> AI 笔记 -> 二轮同类题 -> 五年经典迁移题”生成今日路线。
- [[错题关联笔记]]：查看每道错题命中的标准笔记；没有命中时，把错题整理成方法卡。
- [[内容复习建议]]：从错因、知识点和方法角度看薄弱点。
- [[相似题推荐]]：从二轮题库和五年经典里找迁移题。

```dataviewjs
const pages = dv.pages('"01_错题"').where(p => p.type === "wrong_problem");
const today = dv.date("today");
const total = pages.length;
const due = pages.where(p => {
  const next = p.next_review ? dv.date(p.next_review) : null;
  return String(p.status ?? "") === "待复习" || String(p.status ?? "") === "仍错" || (next && next <= today);
}).length;
const weak = pages.where(p => Number(p.mastery ?? 0) <= 2).length;
const lowClass = pages.where(p => String(p.classification_confidence ?? "") === "low").length;
const mastered = pages.where(p => String(p.status ?? "") === "已掌握").length;
function card(name, value, note) {
  return `<div class="wb-card"><b>${name}</b><strong>${value}</strong><span class="wb-muted">${note}</span></div>`;
}
dv.paragraph(`<div class="wb-grid">
${card("总错题", total, "已结构化")}
${card("今日队列", due, "待复习 / 到期 / 仍错")}
${card("薄弱题", weak, "掌握度 <= 2")}
${card("待确认分类", lowClass, "需要人工确认")}
${card("已掌握", mastered, "可低频回看")}
</div>`);
```

## 今日优先

```dataview
TABLE module AS 专题, topic AS 小节, status AS 状态, mastery AS 掌握度, wrong_reason AS 错因, next_review AS 下次复习
FROM "01_错题"
WHERE type = "wrong_problem" AND (status = "仍错" OR status = "待复习" OR next_review <= date(today))
SORT status ASC, next_review ASC, mastery ASC
LIMIT 12
```
