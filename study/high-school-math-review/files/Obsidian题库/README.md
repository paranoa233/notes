# 高中数学 Obsidian 题库

- [[00_控制台/总览]]
- [[00_控制台/上传解答]]
- [[00_控制台/题目浏览]]
- [[00_控制台/AI笔记整理]]
- [[08_AI笔记管理/AI学习流水线]]
- [[08_AI笔记管理/对话导入清单]]
- [[08_AI笔记管理/AI笔记转化看板]]

后续上传解答时，把文件放进 `99_上传入口`，文件名以题号开头，例如 `10-3-3.png`、`10-3-3.md`、`10-3-3 我的解答.pdf`，再运行 `tools/import_solutions.py`。

AI 笔记整理入口在 `08_AI笔记管理`。把每节导出的 AI 对话放入 `08_AI笔记管理/01_待整理/00_AI对话导入入口`，再按 [[08_AI笔记管理/AI学习流水线]] 生成 Claude Code / DeepSeek v4 Pro 提示词或直接调用 API。
