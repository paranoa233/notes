# 论文阅读工作流示例

下面是一种适合长期复用的读论文模板。

## 基本信息

| 字段 | 内容 |
| --- | --- |
| 标题 | 论文标题 |
| 作者 | 作者姓名 |
| 来源 | 期刊 / 会议 |
| 年份 | 2026 |
| 链接 | 原文地址 |

## 三句话摘要

1. 这篇文章要解决什么问题？
2. 它最核心的方法是什么？
3. 它和已有方法相比，真正的新东西是什么？

## 阅读时固定关注

### 1. 问题定义

- 输入输出是什么
- 评价指标是什么
- 实验设置是否合理

### 2. 方法结构

- 核心假设是什么
- 哪一步最容易失效
- 复杂度是否可接受

### 3. 结果可信度

- 是否有消融实验
- 对比方法是否公平
- 是否存在挑数据集的嫌疑

## 我自己的落地动作

你可以在每篇笔记最后补三项：

- `可复现实验`：最值得先复现的部分
- `可迁移思路`：能否迁移到自己的课题
- `待确认问题`：下次和导师讨论时要问什么

## 推荐的记录风格

不要只抄原文，要把自己的判断也写进去。最有价值的部分通常不是“作者说了什么”，而是：

- 我认同什么
- 我怀疑什么
- 我准备怎么验证
