# 欢迎来到我的笔记站

这个小站适合用来做三类内容：

- 公开分享的学习记录
- 给同学或朋友看的协作说明
- 自己长期查阅的技术与论文笔记

## 以后怎么发新笔记

只需要两步：

1. 在 `notes/` 目录里新增一个 Markdown 文件，比如 `new-note.md`
2. 在 `notes/notes.json` 里登记它的标题、日期、标签和文件名

一个最小条目长这样：

```json
{
  "slug": "new-note",
  "title": "一篇新笔记",
  "summary": "这里写一句摘要",
  "date": "2026-04-08",
  "tags": ["标签1", "标签2"],
  "file": "new-note.md"
}
```

## 这个模板已经支持什么

- Markdown 标题、列表、表格、引用、代码块
- 左侧目录导航
- 关键词搜索
- 标签筛选
- 数学公式渲染

## 后续还可以继续加

如果你想把它做成更像个人站，我们后面还能继续加：

- 评论区，比如接入 Giscus
- 首页时间轴
- 置顶文章
- 深色模式
- 论文阅读卡片模板

> 现在这个版本主打的是简单和稳：不需要构建，不依赖 GitHub Actions，直接放到 GitHub Pages 就能跑。
