#### 9.2.3 无理函数积分的例子

一般来说, 无理函数的不定积分并不总是积得出来的. 例如, 即使看似简单的二项式微分式的积分

\[
\int {x}^{m}{\left( a + b{x}^{n}\right) }^{p}\mathrm{\;d}x,
\]

其中 \( a, b \) 为常数, \( m, n, p \) 为有理数,也仅仅在 \( p,\left( {m + 1}\right) /n \) 或 \( \left( {m + 1}\right) /n + p \) 为整数这三种特殊情况下才能积得出来. 其余情况下的二项式微分式都积不出来, 见 [14] 第二卷 279 小节. 在前面我们已经提到,对于形如 \( \int R\left( {x,\sqrt{a{x}^{2} + {bx} + c}}\right) \mathrm{d}x \; \left( {a \neq  0}\right) \) ,其中 \( R \) 为有理分式的不定积分,可以利用 Euler 变换或三角变换化为有理函数进行积分 (见例题 9.1.1 的注 2). 下面我们再举一些例题, 说明如何对一些特殊的无理函数进行积分.

例题 9.2.7 计算不定积分 \( \int \frac{1}{x}\sqrt{\frac{x + 2}{x - 2}}\mathrm{\;d}x \) .

解 令 \( t = \sqrt{\frac{x + 2}{x - 2}} \) ,则 \( x = \frac{2\left( {{t}^{2} + 1}\right) }{{t}^{2} - 1},\mathrm{\;d}x =  - \frac{8t}{{\left( {t}^{2} - 1\right) }^{2}}\mathrm{\;d}t \) . 因此

\[
\int \frac{1}{x}\sqrt{\frac{x + 2}{x - 2}}\mathrm{\;d}x = \int \frac{4{t}^{2}}{\left( {1 - {t}^{2}}\right) \left( {1 + {t}^{2}}\right) }\mathrm{d}t
\]

\[
= 2\int \left( {\frac{1}{1 - {t}^{2}} - \frac{1}{1 + {t}^{2}}}\right) \mathrm{d}t
\]

\[
= \ln \left| \frac{1 + t}{1 - t}\right|  - 2\arctan t + C
\]

\[
= \ln \left| \frac{1 + \sqrt{\frac{x + 2}{x - 2}}}{1 - \sqrt{\frac{x + 2}{x - 2}}}\right|  - 2\arctan \sqrt{\frac{x + 2}{x - 2}} + C
\]

\[
= \ln \left| {x + \sqrt{{x}^{2} - 4}}\right|  + \arctan \frac{\sqrt{{x}^{2} - 4}}{2} + C.
\]

注 对于形如 \( \int R\left( {x,\sqrt[n]{\frac{{ax} + b}{{cx} + d}}}\right) \mathrm{d}x\left( {n > 1,{ad} - {bc} \neq  0}\right) \) 的不定积分,一般只要令 \( t = \sqrt[n]{\frac{{ax} + b}{{cx} + d}} \) ,就可以将其化为有理函数积分.

例题 9.2.8 计算不定积分 \( I = \int \frac{1 - \sqrt{x + 1}}{\left( {x + 1}\right) \left( {1 + \sqrt[3]{x + 1}}\right) }\mathrm{d}x \) .

解 令 \( t = \sqrt[6]{x + 1} \) ,则 \( x = {t}^{6} - 1,\mathrm{\;d}x = 6{t}^{5}\mathrm{\;d}t \) . 因此

\[
I = \int \frac{6\left( {1 - {t}^{3}}\right) {t}^{5}}{{t}^{6}\left( {1 + {t}^{2}}\right) }\mathrm{d}t
\]

\[
= 6\int \frac{1 - {t}^{3}}{t\left( {1 + {t}^{2}}\right) }\mathrm{d}t
\]

\[
= 6\int \frac{\left( {1 + {t}^{2}}\right)  - t\left( {t - 1}\right)  - t\left( {1 + {t}^{2}}\right) }{t\left( {1 + {t}^{2}}\right) }\mathrm{d}t
\]

\[
= 6\int \left( {\frac{1}{t} - \frac{t - 1}{{t}^{2} + 1} - 1}\right) \mathrm{d}t
\]

\[
= 6\left( {\ln \left| t\right|  - \frac{1}{2}\ln \left( {1 + {t}^{2}}\right)  + \arctan t - t}\right)  + C
\]

\[
= 6\left( {\ln \sqrt[6]{x + 1} - \frac{1}{2}\ln \left( {1 + \sqrt[3]{x + 1}}\right)  + \arctan \sqrt[6]{x + 1} - \sqrt[6]{x + 1}}\right)  + C
\]

\[
= 3\ln \frac{\sqrt[3]{x + 1}}{1 + \sqrt[3]{x + 1}} + 6\arctan \sqrt[6]{x + 1} - 6\sqrt[6]{x + 1} + C.
\]

注 对于形如 \( \int R\left( {x,\sqrt[n]{{ax} + b},\sqrt[m]{{ax} + b}}\right) \mathrm{d}x \) 的不定积分,其中 \( m, n \) 是正整数, 可以用 \( t = \sqrt[p]{{ax} + b} \) ，其中 \( p \) 是 \( m, n \) 的最小公倍数，将其有理化.

最后, 我们举出一些被积函数既含有无理式, 又含有超越函数 (指数函数、对数函数、三角函数与反三角函数) 的不定积分例题. 对这类题, 并没有固定的一般解法, 只有根据经验, 对具体问题进行分析和尝试, 寻找合适的解法.

例题 9.2.9 计算不定积分 \( I = \int \sqrt{1 + \sin x}\mathrm{\;d}x \) .

分析 关键是去掉根号或化掉根号下的三角函数. 我们介绍下列两种方法.

解 1 用适当的三角恒等式, 可计算如下:

\[
I = \int \sqrt{{\sin }^{2}\frac{x}{2} + {\cos }^{2}\frac{x}{2} + 2\sin \frac{x}{2}\cos \frac{x}{2}}\mathrm{\;d}x
\]

\[
= \int \left( {\sin \frac{x}{2} + \cos \frac{x}{2}}\right) \mathrm{d}x
\]

\[
=  - 2\cos \frac{x}{2} + 2\sin \frac{x}{2} + C\text{ . }
\]

解 2 用代换法,令 \( t = \sqrt{1 + \sin x} \) ,则

\[
x = \arcsin \left( {{t}^{2} - 1}\right) ,\;\mathrm{d}x = \frac{{2t}\mathrm{\;d}t}{\sqrt{2{t}^{2} - {t}^{4}}},
\]

因此

\[
I = \int \frac{2{t}^{2}\mathrm{\;d}t}{\sqrt{2{t}^{2} - {t}^{4}}} = \int \frac{\mathrm{d}{t}^{2}}{\sqrt{2 - {t}^{2}}} =  - 2\sqrt{2 - {t}^{2}} + C =  - 2\sqrt{1 - \sin x} + C.
\]

例题 9.2.10 计算不定积分 \( I = \int \arctan \sqrt{\frac{a - x}{a + x}}\mathrm{\;d}x\left( {a > 0}\right) \) .

解 1 用分部积分法:

\[
I = x\arctan \sqrt{\frac{a - x}{a + x}} - \int x \cdot  \frac{1}{1 + \frac{a - x}{a + x}} \cdot  \frac{1}{2\sqrt{\frac{a - x}{a + x}}} \cdot  \frac{-{2a}}{{\left( a + x\right) }^{2}}\mathrm{\;d}x
\]

\[
= x\arctan \sqrt{\frac{a - x}{a + x}} + \frac{1}{2}\int \frac{x}{\sqrt{{a}^{2} - {x}^{2}}}\mathrm{\;d}x
\]

\[
= x\arctan \sqrt{\frac{a - x}{a + x}} - \frac{1}{2}\sqrt{{a}^{2} - {x}^{2}} + C.
\]

解 2 令 \( x = a\cos t \) ,则有

\[
\arctan \sqrt{\frac{a - x}{a + x}} = \arctan \sqrt{\frac{1 - \cos t}{1 + \cos t}} = \arctan \sqrt{\frac{2{\sin }^{2}\frac{t}{2}}{2{\cos }^{2}\frac{t}{2}}} = \arctan \left( {\tan \frac{t}{2}}\right)  = \frac{t}{2},
\]

因此

\[
I = a\int \frac{t}{2}\mathrm{\;d}\cos t = a \cdot  \frac{t}{2}\cos t - \frac{a}{2}\int \cos t\mathrm{\;d}t
\]

\[
= \frac{at}{2}\cos t - \frac{a}{2}\sin t + C
\]

\[
= \frac{x}{2}\arccos \frac{x}{a} - \frac{1}{2}\sqrt{{a}^{2} - {x}^{2}} + C.
\]
