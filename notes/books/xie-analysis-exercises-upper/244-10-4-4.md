#### 10.4.4 用递推方法求定积分

设有一列函数 \( {f}_{n}\left( x\right)  \in  R\left\lbrack  {a, b}\right\rbrack \) ，其中下标 \( n \in  {\mathbf{N}}_{ + } \) 为参数. 为了计算积分 \( {\int }_{a}^{b}{f}_{n}\left( x\right) \mathrm{d}x \) ,我们可用各种方法将下标为 \( n \) 的积分 \( {\int }_{a}^{b}{f}_{n}\left( x\right) \mathrm{d}x \) 化成与下标 \( k < n \) 的积分 \( {\int }_{a}^{b}{f}_{k}\left( x\right) \mathrm{d}x \) 有关的表达式 (即递推公式). 继续如此做下去,就有可能将问题化为求下标最小的一个或几个积分. 计算定积分的这种方法, 称为递推方法.

例题 10.4.9 计算 \( {I}_{n} = {\int }_{0}^{\pi /2}{\sin }^{n}x\mathrm{\;d}x = {\int }_{0}^{\pi /2}{\cos }^{n}x\mathrm{\;d}x \) .

解 两个积分相等可以由换元 \( t = \frac{\pi }{2} - x \) 得到. 由定积分的分部积分法,

\[
{I}_{n} = {\int }_{0}^{\pi /2}{\sin }^{n - 1}x\mathrm{\;d}\left( {-\cos x}\right)  =  - {\left. {\sin }^{n - 1}x\cos x\right| }_{0}^{\pi /2} + {\int }_{0}^{\pi /2}\cos x\mathrm{\;d}\left( {{\sin }^{n - 1}x}\right)
\]

\[
= \left( {n - 1}\right) {\int }_{0}^{\pi /2}{\sin }^{n - 2}x{\cos }^{2}x\mathrm{\;d}x = \left( {n - 1}\right) {I}_{n - 2} - \left( {n - 1}\right) {I}_{n},
\]

移项后得递推公式:

\[
{I}_{n} = \frac{n - 1}{n}{I}_{n - 2}\left( {n \geq  2}\right) .
\]

重复使用上述公式，由于

\[
{I}_{0} = {\int }_{0}^{\pi /2}\mathrm{\;d}x = \frac{\pi }{2},\;{I}_{1} = {\int }_{0}^{\pi /2}\sin x\mathrm{\;d}x = 1,
\]

就得到

\[
{I}_{n} = \left\{  \begin{array}{ll} \frac{\left( {n - 1}\right) !!}{n!!}, & n\text{ 为奇数, } \\  \frac{\left( {n - 1}\right) !!}{n!!} \cdot  \frac{\pi }{2}, & n\text{ 为偶数. } \end{array}\right. \tag{10.16}
\]

注 请初学者注意, 在定积分的计算中, 公式 (10.16) 经常有用, 因此需要记住这个公式并能熟练应用. 此外, \( {I}_{n} \) 在 \( n \) 为奇数和偶数时的表达式不同是今后导出 Wallis 公式 (命题 11.4.1) 的关键.

例题 10.4.10 设 \( m, n \) 为正整数，计算含双参数的积分

\[
B\left( {m, n}\right)  = {\int }_{0}^{1}{x}^{m - 1}{\left( 1 - x\right) }^{n - 1}\mathrm{\;d}x.
\]

解 令 \( {x}^{m - 1}\mathrm{\;d}x = \mathrm{d}v,{\left( 1 - x\right) }^{n - 1} = u \) ,进行分部积分,得到递推公式:

\[
B\left( {m, n}\right)  = {\left. \frac{{x}^{m}{\left( 1 - x\right) }^{n - 1}}{m}\right| }_{0}^{1} + \frac{n - 1}{m}{\int }_{0}^{1}{x}^{m}{\left( 1 - x\right) }^{n - 2}\mathrm{\;d}x
\]

\[
= \frac{n - 1}{m}B\left( {m + 1, n - 1}\right) ,
\]

连续应用上述公式, 得到

\[
B\left( {m, n}\right)  = \frac{\left( {n - 1}\right) \left( {n - 2}\right) \cdots \left( {n - \left( {n - 1}\right) }\right) }{m\left( {m + 1}\right) \cdots \left( {m + n - 2}\right) }B\left( {m + n - 1,1}\right)
\]

\[
= \frac{\left( {n - 1}\right) !}{m\left( {m + 1}\right) \cdots \left( {m + n - 2}\right) }{\int }_{0}^{1}{x}^{m + n - 2}\mathrm{\;d}x
\]

\[
= \frac{\left( {n - 1}\right) !}{m\left( {m + 1}\right) \cdots \left( {m + n - 2}\right) \left( {m + n - 1}\right) }
\]

\[
= \frac{\left( {n - 1}\right) !\left( {m - 1}\right) !}{\left( {m + n - 1}\right) !}.
\]

注 读者可将本题与例题 9.1.10 比较, 以看出定积分的分部积分法与不定积分的分部积分法之间的差别.
