#### 11.4.3 Taylor 公式的积分型余项

在第七章“微分学的基本定理”中已介绍了带有 Peano 余项、Lagrange 余项与 Cauchy 余项的 Taylor 公式. 这里将介绍带积分型余项的 Taylor 公式.

命题 11.4.3 设 \( f\left( x\right) \) 在区间 \( \left( {{x}_{0} - r,{x}_{0} + r}\right) \) 上有 \( n + 1 \) 阶连续导函数,则对每个 \( x \in  \left( {{x}_{0} - r,{x}_{0} + r}\right) \) 成立

\[
f\left( x\right)  = \mathop{\sum }\limits_{{k = 0}}^{n}\frac{{f}^{\left( k\right) }\left( {x}_{0}\right) }{k!}{\left( x - {x}_{0}\right) }^{k} + {R}_{n}\left( x\right) ,
\]

其中余项

\[
{R}_{n}\left( x\right)  = \frac{1}{n!}{\int }_{{x}_{0}}^{x}{f}^{\left( n + 1\right) }\left( t\right) {\left( x - t\right) }^{n}\mathrm{\;d}t. \tag{11.37}
\]

证 从

\[
{R}_{n}\left( x\right)  = f\left( x\right)  - \mathop{\sum }\limits_{{k = 0}}^{n}\frac{{f}^{\left( k\right) }\left( {x}_{0}\right) }{k!}{\left( x - {x}_{0}\right) }^{k},
\]

可以得到

\[
{R}_{n}^{\left( k\right) }\left( {x}_{0}\right)  = 0, k = 1,2,\cdots , n,\;{R}_{n}^{\left( n + 1\right) }\left( x\right)  = {f}^{\left( n + 1\right) }\left( x\right) .
\]

利用逐次分部积分运算就可以有

\[
{R}_{n}\left( x\right)  = {\int }_{{x}_{0}}^{x}{R}_{n}^{\prime }\left( t\right) \mathrm{d}t = {\left. \left( t - x\right) {R}_{n}^{\prime }\left( t\right) \right| }_{{x}_{0}}^{x} + {\int }_{{x}_{0}}^{x}{R}_{n}^{\prime \prime }\left( t\right) \left( {x - t}\right) \mathrm{d}t
\]

\[
= {\int }_{{x}_{0}}^{x}{R}_{n}^{\prime \prime }\left( t\right) \left( {x - t}\right) \mathrm{d}t = \frac{1}{2}{\int }_{{x}_{0}}^{x}{R}_{n}^{\prime \prime \prime }\left( t\right) {\left( x - t\right) }^{2}\mathrm{\;d}t
\]

\[
= \cdots  = \frac{1}{n!}{\int }_{{x}_{0}}^{x}{R}_{n}^{\left( n + 1\right) }\left( t\right) {\left( x - t\right) }^{n}\mathrm{\;d}t
\]

\[
= \frac{1}{n!}{\int }_{{x}_{0}}^{x}{f}^{\left( n + 1\right) }\left( t\right) {\left( x - t\right) }^{n}\mathrm{\;d}t.
\]

注 (1) 对余项 (11.37) 右边用第一中值定理,在 \( {x}_{0} \) 与 \( x \) 之间有 \( \xi \) ,使得

\[
{R}_{n}\left( x\right)  = \frac{1}{n!}{\int }_{{x}_{0}}^{x}{f}^{\left( n + 1\right) }\left( t\right) {\left( x - t\right) }^{n}\mathrm{\;d}t = \frac{1}{n!}{f}^{\left( n + 1\right) }\left( \xi \right) {\int }_{{x}_{0}}^{x}{\left( x - t\right) }^{n}\mathrm{\;d}t
\]

\[
=  - {\left. \frac{1}{\left( {n + 1}\right) !}{f}^{\left( n + 1\right) }\left( \xi \right) {\left( x - t\right) }^{n + 1}\right| }_{{x}_{0}}^{x}
\]

\[
= \frac{1}{\left( {n + 1}\right) !}{f}^{\left( n + 1\right) }\left( \xi \right) {\left( x - {x}_{0}\right) }^{n + 1}.
\]

这就是 Lagrange 余项 (命题 7.2.3).

(2) 在 (11.37) 右边的积分中,把被积函数看作 \( {f}^{\left( n + 1\right) }\left( t\right) {\left( x - t\right) }^{n} \) 与 1 的乘积, 由积分第一中值定理,存在 \( \xi \) 在 \( {x}_{0} \) 与 \( x \) 之间,使

\[
{R}_{n}\left( x\right)  = \frac{1}{n!}{f}^{\left( n + 1\right) }\left( \xi \right) {\left( x - \xi \right) }^{n}{\int }_{{x}_{0}}^{x}\mathrm{\;d}t
\]

\[
= \frac{1}{n!}{f}^{\left( n + 1\right) }\left( \xi \right) {\left( x - \xi \right) }^{n}\left( {x - {x}_{0}}\right) .
\]

将 \( \xi \) 改写成 \( \xi  = {x}_{0} + \eta \left( {x - {x}_{0}}\right) ,0 \leq  \eta  \leq  1 \) ,则上式成为

\[
{R}_{n}\left( x\right)  = \frac{1}{n!}{f}^{\left( n + 1\right) }\left( {{x}_{0} + \eta \left( {x - {x}_{0}}\right) }\right) {\left( 1 - \eta \right) }^{n}{\left( x - {x}_{0}\right) }^{n + 1}.
\]

这就是 Cauchy 余项 (命题 7.2.4).

(3) Lagrange 余项与 Cauchy 余项分别含有不完全确定的中值 \( \xi \) 与 \( \eta \) ,而积分型余项中则不含中值, 这无疑是一个优点. 由于这个原因, 带积分型余项的 Taylor 公式常被用于比较精确的表达式中.
