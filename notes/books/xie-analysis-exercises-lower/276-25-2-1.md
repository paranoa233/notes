#### 25.2.1 第二型曲面积分的定义和计算

设 \( S \) 是逐片光滑的定向曲面， \( P, Q, R \) 是在曲面 \( S \) 上有定义的函数. 在曲面 \( S \) 所指定的一侧作分割 \( T \) ,它把曲面 \( S \) 分为 \( n \) 个小曲面 \( {S}_{1},{S}_{2},\cdots ,{S}_{n} \) ,分割 \( T \) 的细度 \( \parallel T\parallel  = \mathop{\max }\limits_{{1 \leq  i \leq  n}}\left\{  {S}_{i}\right. \) 的直径 \( \} \) ,以 \( \Delta {S}_{i}^{\left( 1\right) },\Delta {S}_{i}^{\left( 2\right) },\Delta {S}_{i}^{\left( 3\right) } \) 分别表示 \( {S}_{i} \) 在 \( {yOz},{zOx} \) , \( {xOy} \) 三个坐标平面上的投影区域的面积. 它们的符号由 \( {S}_{i} \) 的方向来确定,当 \( {S}_{i} \) 的法线正向与 \( z \) 轴正向成锐角时, \( {S}_{i} \) 在 \( {xOy} \) 平面的投影区域的面积 \( \Delta {S}_{i}^{\left( 3\right) } \) 为正,反之面积为负. \( \Delta {S}_{i}^{\left( 1\right) } \) 与 \( \Delta {S}_{i}^{\left( 2\right) } \) 的符号可类似地定义. 在每个 \( {S}_{i} \) 上任取一点 \( \left( {{\xi }_{i},{\eta }_{i},{\zeta }_{i}}\right) \) , 若

\[
\mathop{\lim }\limits_{{\parallel T\parallel  \rightarrow  0}}\mathop{\sum }\limits_{{i = 1}}^{n}P\left( {{\xi }_{i},{\eta }_{i},{\zeta }_{i}}\right) \Delta {S}_{i}^{\left( 1\right) } + \mathop{\lim }\limits_{{\parallel T\parallel  \rightarrow  0}}\mathop{\sum }\limits_{{i = 1}}^{n}Q\left( {{\xi }_{i},{\eta }_{i},{\zeta }_{i}}\right) \Delta {S}_{i}^{\left( 2\right) }
\]

\[
+ \mathop{\lim }\limits_{{\parallel T\parallel  \rightarrow  0}}\mathop{\sum }\limits_{{i = 1}}^{n}R\left( {{\xi }_{i},{\eta }_{i},{\zeta }_{i}}\right) \Delta {S}_{i}^{\left( 3\right) }
\]

存在,且与曲面 \( S \) 的分割 \( T \) 以及 \( \left( {{\xi }_{i},{\eta }_{i},{\zeta }_{i}}\right) \) 在 \( {S}_{i} \) 上的取法无关,则称此极限为函数 \( P, Q, R \) 在曲面 \( S \) 上所指定的一侧上的第二型曲面积分,记为

\[
{\iint }_{S}P\left( {x, y, z}\right) \mathrm{d}y\mathrm{\;d}z + Q\left( {x, y, z}\right) \mathrm{d}z\mathrm{\;d}x + R\left( {x, y, z}\right) \mathrm{d}x\mathrm{\;d}y.
\]

设 \( R\left( {x, y, z}\right) \) 是定义在光滑曲面

\[
S : z = z\left( {x, y}\right) ,\;\left( {x, y}\right)  \in  {D}_{xy}
\]

上的连续函数,以 \( S \) 的上侧为正侧,则有

\[
{\iint }_{S}R\left( {x, y, z}\right) \mathrm{d}x\mathrm{\;d}y =  \pm  {\iint }_{{D}_{xy}}R\left( {x, y, z\left( {x, y}\right) }\right) \mathrm{d}x\mathrm{\;d}y.
\]

如果在正侧积分, 积分号前取正号; 如果在负侧积分, 积分号前取负号.

类似地,当 \( P\left( {x, y, z}\right) \) 在光滑曲面

\[
S : x = x\left( {y, z}\right) ,\;\left( {y, z}\right)  \in  {D}_{yz}
\]

上连续时, 有

\[
{\iint }_{S}P\left( {x, y, z}\right) \mathrm{d}y\mathrm{\;d}z =  \pm  {\iint }_{{D}_{yz}}P\left( {x\left( {y, z}\right) , y, z}\right) \mathrm{d}y\mathrm{\;d}z,
\]

这里 \( S \) 是以 \( S \) 的法线方向与 \( x \) 轴的正向成锐角的那一侧为正侧.

当 \( Q\left( {x, y, z}\right) \) 在光滑曲面

\[
S : y = y\left( {z, x}\right) ,\;\left( {z, x}\right)  \in  {D}_{zx}
\]

上连续时, 有

\[
{\iint }_{S}Q\left( {x, y, z}\right) \mathrm{d}z\mathrm{\;d}x =  \pm  {\iint }_{{D}_{zx}}Q\left( {x, y\left( {z, x}\right) , z}\right) \mathrm{d}z\mathrm{\;d}x,
\]

这里 \( S \) 是以 \( S \) 的法线方向与 \( y \) 轴的正向成锐角的那一侧为正侧.

如果光滑曲面 \( S \) 由参数方程给出:

\[
S : x = x\left( {u, v}\right) , y = y\left( {u, v}\right) , z = z\left( {u, v}\right) ,\left( {u, v}\right)  \in  D,
\]

且行列式

\[
A = \frac{\partial \left( {y, z}\right) }{\partial \left( {u, v}\right) }, B = \frac{\partial \left( {z, x}\right) }{\partial \left( {u, v}\right) }, C = \frac{\partial \left( {x, y}\right) }{\partial \left( {u, v}\right) }
\]

不同时为 0 , 则

\[
{\iint }_{S}P\mathrm{\;d}y\mathrm{\;d}z + Q\mathrm{\;d}z\mathrm{\;d}x + R\mathrm{\;d}x\mathrm{\;d}y =  \pm  {\iint }_{D}\left( {{PA} + {QB} + {RC}}\right) \mathrm{d}u\mathrm{\;d}v, \tag{25.3}
\]

其中积分号前正负号的选取法则如下: 若向量 \( \left( {A, B, C}\right) \) 与曲面 \( S \) 上预先选定的侧的法向量方向所成角不大于直角，则取 “+” 号，否则取 “-” 号.

例题 25.2.1 设 \( \sum \) 为上半单位球面 \( z = \sqrt{1 - \left( {{x}^{2} + {y}^{2}}\right) } \) ,取内侧,求

\[
I = {\iint }_{\sum }\mathrm{d}y\mathrm{\;d}z + \mathrm{d}z\mathrm{\;d}x + \mathrm{d}x\mathrm{\;d}y.
\]

解 1 用直角坐标系计算

\[
I = {\iint }_{\sum }\mathrm{d}y\mathrm{\;d}z + {\iint }_{\sum }\mathrm{d}z\mathrm{\;d}x + {\iint }_{\sum }\mathrm{d}x\mathrm{\;d}y = {I}_{1} + {I}_{2} + {I}_{3}.
\]

计算 \( {I}_{1} : \sum  = {\sum }_{1} + {\sum }_{2} \) ,其中

\[
{\sum }_{1} : x = \sqrt{1 - {y}^{2} - {z}^{2}},\;\left( {y, z}\right)  \in  {D}_{yz} = \left\{  {{y}^{2} + {z}^{2} \leq  1, z \geq  0}\right\}  \text{ ,取后侧, }
\]

\[
{\sum }_{2} : x =  - \sqrt{1 - {y}^{2} - {z}^{2}},\;\left( {y, z}\right)  \in  {D}_{yz}\text{ ,取前侧, }
\]

则

\[
{I}_{1} = {\iint }_{{\sum }_{1}}\mathrm{\;d}y\mathrm{\;d}z + {\iint }_{{\sum }_{2}}\mathrm{\;d}y\mathrm{\;d}z =  - {\iint }_{{D}_{yz}}\mathrm{\;d}y\mathrm{\;d}z + {\iint }_{{D}_{yz}}\mathrm{\;d}y\mathrm{\;d}z = 0.
\]

同理有

\[
{I}_{2} = 0,{I}_{3} =  - {\iint }_{{D}_{xy}}\mathrm{\;d}x\mathrm{\;d}y =  - \pi .
\]

最后得到

\[
I =  - \pi
\]

解 2 用参数方程

\[
\sum  : x = \sin \varphi \cos \theta , y = \sin \varphi \sin \theta , z = \cos \varphi ,
\]

\[
0 \leq  \varphi  \leq  \frac{\pi }{2},0 \leq  \theta  \leq  {2\pi }
\]

计算行列式

\[
A = \frac{\partial \left( {y, z}\right) }{\partial \left( {\varphi ,\theta }\right) } = {\sin }^{2}\varphi \cos \theta ,\;B = \frac{\partial \left( {z, x}\right) }{\partial \left( {\varphi ,\theta }\right) } = {\sin }^{2}\varphi \sin \theta ,
\]

\[
C = \frac{\partial \left( {x, y}\right) }{\partial \left( {\varphi ,\theta }\right) } = \sin \varphi \cos \varphi .
\]

因为 \( \left( {A, B, C}\right) \) 的方向与上半球面 \( S \) 内侧的法线方向相反,故积分号前取 “-” 号, 得到

\[
I =  - {\iint }_{D}\left( {{\sin }^{2}\varphi \cos \theta  + {\sin }^{2}\varphi \sin \theta  + \sin \varphi \cos \varphi }\right) \mathrm{d}\varphi \mathrm{d}\theta
\]

\[
=  - {\int }_{0}^{\pi /2}\mathrm{\;d}\varphi {\int }_{0}^{2\pi }\left\lbrack  {{\sin }^{2}\varphi \left( {\cos \theta  + \sin \theta }\right)  + \sin \varphi \cos \varphi }\right\rbrack  \mathrm{d}\theta
\]

\[
=  - {2\pi }{\int }_{0}^{\pi /2}\sin \varphi \cos \varphi \mathrm{d}\varphi  =  - \pi
\]

例题 25.2.2 求

\[
I = {\oiint }_{\sum }\left( {z + x}\right) \mathrm{d}y\mathrm{\;d}z + \left( {x + y}\right) \mathrm{d}z\mathrm{\;d}x + \left( {y + z}\right) \mathrm{d}x\mathrm{\;d}y
\]

其中 \( \sum \) 是由 \( {x}^{2} + {y}^{2} = 1, z = 1 \) 及三个坐标平面围成的立体在第一卦限的部分的表面, 取外侧.

解 见图 25.2,因为 \( \sum \) 分块较多 (需分 5 块),不便于用参数式,故应在直角坐标系中计算. 记 \( \sum  = {\sum }_{1} + {\sum }_{2} + {\sum }_{3} + {\sum }_{4} + {\sum }_{5} \) ,其中 \( {\sum }_{1} \) 为圆柱面, \( {\sum }_{2} \) 为下底面, \( {\sum }_{3} \) 为上底面， \( {\sum }_{4} \) 为左侧面 \( y = 0 \) ， \( {\sum }_{5} \) 为右侧面 \( x = 0 \) ，则有

\[
I = {\oiint }_{\sum }\left( {z + x}\right) \mathrm{d}y\mathrm{\;d}z + {\oiint }_{\sum }\left( {x + y}\right) \mathrm{d}z\mathrm{\;d}x + {\oiint }_{\sum }\left( {y + z}\right) \mathrm{d}x\mathrm{\;d}y = {I}_{1} + {I}_{2} + {I}_{3}.
\]

计算 \( {I}_{1} \) ，有

\[
{I}_{1} = \mathop{\sum }\limits_{{i = 1}}^{5}{\iint }_{{\sum }_{i}}\left( {x + z}\right) \mathrm{d}y\mathrm{\;d}z.
\]

因为 \( {\sum }_{2},{\sum }_{3},{\sum }_{4} \) 在 \( {yOz} \) 平面上的投影面积为零,于是

\[
{\iint }_{{\sum }_{i}}\left( {x + z}\right) \mathrm{d}y\mathrm{\;d}z = 0,\;i = 2,3,4,
\]

而

\[
{\sum }_{1} : x = \sqrt{1 - {y}^{2}},\left( {y, z}\right)  \in  {D}_{yz} = \{ 0 \leq  y \leq  1,0 \leq  z \leq  1\} \text{ ,取前侧, }
\]

\[
{\sum }_{5} : x = 0,\left( {y, z}\right)  \in  {D}_{yz}\text{ ,取后侧. }
\]

于是

\[
{I}_{1} = {\iint }_{{\sum }_{1}}\left( {x + z}\right) \mathrm{d}y\mathrm{\;d}z + {\iint }_{{\sum }_{5}}\left( {x + z}\right) \mathrm{d}y\mathrm{\;d}z
\]

\[
= {\iint }_{{D}_{yz}}\left( {\sqrt{1 - {y}^{2}} + z}\right) \mathrm{d}y\mathrm{\;d}z - {\iint }_{{D}_{yz}}z\mathrm{\;d}y\mathrm{\;d}z
\]

\[
= {\int }_{0}^{1}{\int }_{0}^{1}\sqrt{1 - {y}^{2}}\mathrm{\;d}y\mathrm{\;d}z = \frac{\pi }{4}.
\]

用类似的方法可求出 \( {I}_{2} = \frac{\pi }{4} \) .

然后求

\[
{I}_{3} = \mathop{\sum }\limits_{{i = 1}}^{5}{\iint }_{{\sum }_{i}}\left( {y + z}\right) \mathrm{d}x\mathrm{\;d}y.
\]

显然有

\[
{\iint }_{{\sum }_{i}}\left( {y + z}\right) \mathrm{d}x\mathrm{\;d}y = 0,\;i = 1,4,5,
\]

而

\[
{\sum }_{2} : z = 0,\left( {x, y}\right)  \in  {D}_{xy} = \left\{  {{x}^{2} + {y}^{2} \leq  1, x \geq  0, y \geq  0}\right\}  \text{ ,取下侧, }
\]

\( {\sum }_{3} : z = 1,\left( {x, y}\right)  \in  {D}_{xy} \) ,取上侧,

于是

\[
{I}_{3} = {\iint }_{{\sum }_{2}}\left( {y + z}\right) \mathrm{d}x\mathrm{\;d}y + {\iint }_{{\sum }_{3}}\left( {y + z}\right) \mathrm{d}x\mathrm{\;d}y
\]

\[
=  - {\iint }_{{D}_{xy}}y\mathrm{\;d}x\mathrm{\;d}y + {\iint }_{{D}_{xy}}\left( {1 + y}\right) \mathrm{d}x\mathrm{\;d}y
\]

\[
= {\iint }_{{D}_{xy}}\mathrm{\;d}x\mathrm{\;d}y = \frac{\pi }{4}.
\]

最后得到

\[
I = {I}_{1} + {I}_{2} + {I}_{3} = \frac{3}{4}\pi .
\]

注 我们已经知道, 利用第二型曲线积分可以计算平面图形的面积. 类似地, 利用第二型曲面积分也可以计算空间立体的体积 (见表达式 (25.7)—(25.11)).
