#### 20.1.4 练习题

1. 设 \( y = y\left( x\right) \) 由下述方程确定,求 \( {y}^{\prime },{y}^{\prime \prime } \) :

(1) \( \ln \left( {{x}^{2} + {y}^{2}}\right)  = \arctan \frac{y}{x} \) ;

(2) \( {x}^{y} = {y}^{x} \) ;

(3) \( {xy} - {2}^{x}\ln 2 + {2}^{y} = 0 \) .

2. 求在指定点的导数:

(1) \( {y}^{3} + y - {x}^{2} = 0 \) ,求 \( {y}^{\prime }\left( 0\right) \) ;

(2) \( {x}^{\frac{2}{3}} + {y}^{\frac{2}{3}} - 4 = 0 \) ,求在点 \( \left( {1,3\sqrt{3}}\right) \) 处的导数 \( {y}^{\prime } \) ;

(3) \( \sin x + 2\cos y - 1 = 0 \) ,求在点 \( \left( {\frac{\pi }{2},\frac{3\pi }{2}}\right) \) 处的 \( {y}^{\prime },{y}^{\prime \prime } \) .

3. 设 \( z = z\left( {x, y}\right) \) 是由下列方程确定的隐函数,求指定的导数或微分:

(1) \( x + y + z = {\mathrm{e}}^{-\left( {x + y + z}\right) } \) ,求 \( \frac{\partial z}{\partial x},\frac{\partial z}{\partial y},\frac{{\partial }^{2}z}{\partial {y}^{2}},\frac{{\partial }^{2}z}{\partial {x}^{2}},\frac{{\partial }^{2}z}{\partial x\partial y} \) ;

(2) \( {x}^{3} + {y}^{3} + {z}^{3} - {3xyz} - 4 = 0 \) ,求在点 \( \left( {1,1,2}\right) \) 处的偏导数 \( \frac{\partial z}{\partial x},\frac{\partial z}{\partial y} \) ;

(3) \( z = \sqrt{{x}^{2} - {y}^{2}}\tan \frac{z}{\sqrt{{x}^{2} - {y}^{2}}} \) ,求 \( \frac{\partial z}{\partial x},\frac{\partial z}{\partial y} \) ;

(4) \( \frac{x}{z} = \ln \frac{z}{y} \) ,求 \( \mathrm{d}z \) ；

(5) \( {xy} + {yz} + {zx} = 1 \) ,求 \( \frac{\partial z}{\partial x},\frac{\partial z}{\partial y},\frac{{\partial }^{2}z}{\partial {y}^{2}},\frac{{\partial }^{2}z}{\partial {x}^{2}},\frac{{\partial }^{2}z}{\partial x\partial y} \) .

4. 设 \( z = z\left( {x, y}\right) \) 由下列方程确定,求指定的导数或微分:

(1) \( f\left( {x + y + z,{x}^{2} + {y}^{2} + {z}^{2}}\right)  = 0 \) ,求 \( \mathrm{d}z \) ;

(2) \( f\left( {x, x + y, x + y + z}\right)  = 0 \) ,求 \( \frac{\partial z}{\partial x},\frac{\partial z}{\partial y} \) ;

(3) \( f\left( {x + y, y + z, z + x}\right)  = 0 \) ,求 \( \frac{{\partial }^{2}z}{\partial {x}^{2}},\frac{{\partial }^{2}z}{\partial {y}^{2}},\frac{{\partial }^{2}z}{\partial x\partial y} \) .

5. 验证下列各题中给出的隐函数满足指定的方程:

(1) \( x{z}_{x} - y{z}_{y} = {2x} \) ,其中 \( z = z\left( {x, y}\right) \) 是由方程 \( F\left( {{xy}, z - {2x}}\right)  = 0 \) 确定的隐函数;

(2) \( \left( {{x}^{2} - {y}^{2} - {z}^{2}}\right) {z}_{x} + {2xy}{z}_{y} = {2xz} \) ,其中 \( z = z\left( {x, y}\right) \) 是由方程 \( {x}^{2} + {y}^{2} + {z}^{2} = \; {yf}\left( \frac{z}{y}\right) \) 确定的隐函数;

(3) \( {z}_{xx}{z}_{yy} - {z}_{xy}^{2} = 0 \) ,其中 \( z = z\left( {x, y}\right) \) 是由方程 \( \frac{x}{z} = \varphi \left( \frac{y}{z}\right) \) 确定的隐函数, \( \varphi \) 二次连续可微,且 \( x - y{\varphi }^{\prime }\left( \frac{y}{z}\right)  \neq  0 \) ;

(4) \( {u}_{x}^{2} + {u}_{y}^{2} + {u}_{z}^{2} = 2\left( {x{u}_{x} + y{u}_{y} + z{u}_{z}}\right) \) ,其中 \( u = u\left( {x, y, z}\right) \) 是由方程

\[
\frac{{x}^{2}}{{a}^{2} + u} + \frac{{y}^{2}}{{b}^{2} + u} + \frac{{z}^{2}}{{c}^{2} + u} = 1
\]

确定的隐函数.
