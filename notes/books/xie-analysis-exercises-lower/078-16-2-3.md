#### 16.2.3 练习题

1. 设已知 \( \mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n - 1}{a}_{n} = A,\mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{{2n} - 1} = B \) ,证明: \( \mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n} \) 收敛并求其和.

2. 设 \( P\left( x\right)  = {a}_{0} + {a}_{1}x + \cdots  + {a}_{m}{x}^{m} \) 为 \( m \) 次多项式,求级数 \( \mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{P\left( n\right) }{n!} \) 的和.

3. 求 \( 1 - \frac{{2}^{3}}{1!} + \frac{{3}^{3}}{2!} - \frac{{4}^{3}}{3!} + \cdots \) 的和.

4. 求下列级数和: \( \left( 1\right) \mathop{\sum }\limits_{{n = 1}}^{\infty }\arctan \frac{1}{2{n}^{2}};\left( 2\right) \mathop{\sum }\limits_{{n = 1}}^{\infty }\arctan \frac{2}{{n}^{2}} \) .

5. 设 \( a > 1 \) ,求 \( \mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{{2}^{n}}{{a}^{{2}^{n}} + 1} \) 的和.

6. 求 \( 1 + \frac{1}{3} - \frac{1}{5} - \frac{1}{7} + \frac{1}{9} + \frac{1}{11} - \cdots \) 的和.

7. 求 \( 1 - \frac{1}{7} + \frac{1}{9} - \frac{1}{15} + \frac{1}{17} - \frac{1}{23} + \cdots \) 的和.

8. 求 \( 1 - \frac{1}{4} + \frac{1}{7} - \frac{1}{10} + \cdots \) 的和.

9. 设 \( {a}_{n} = 1 + \frac{1}{2} + \cdots  + \frac{1}{n}, n = 1,2,\cdots \) ,求 \( \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{a}_{n}}{n\left( {n + 1}\right) } \) 的和.

10. 求 \( \mathop{\sum }\limits_{{n = 0}}^{\infty }\left( {\frac{1}{{4n} + 1} + \frac{1}{{4n} + 3} - \frac{1}{{2n} + 2}}\right) \) 的和.

11. 求 \( 1 - \frac{1}{4} + \frac{1}{6} - \frac{1}{9} + \frac{1}{11} - \frac{1}{14} + \cdots \) 的和.

12. 求 \( \frac{{x}^{3}}{3!} + \frac{{x}^{9}}{9!} + \frac{{x}^{15}}{{15}!} + \cdots \) 的和函数

13. 求 \( \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{\left\lbrack  \left( n - 1\right) !\right\rbrack  }^{2}}{\left( {2n}\right) !}{\left( 2x\right) }^{2n} \) 的和函数.

14. 求 \( \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{x}^{n + 1}}{\left( {1 - {x}^{n}}\right) \left( {1 - {x}^{n + 1}}\right) } \) 的和函数.

15. 设 \( \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{{a}_{n}} \) 为发散的正项级数, \( x > 0 \) ,求 \( \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{a}_{1}{a}_{2}\cdots {a}_{n}}{\left( {{a}_{2} + x}\right) \left( {{a}_{3} + x}\right) \cdots \left( {{a}_{n + 1} + x}\right) } \) 的和函数.

16. 设 \( x > 1 \) ,求 \( \frac{x}{x + 1} + \frac{{x}^{2}}{\left( {x + 1}\right) \left( {{x}^{2} + 1}\right) } + \frac{{x}^{4}}{\left( {x + 1}\right) \left( {{x}^{2} + 1}\right) \left( {{x}^{4} + 1}\right) } + \cdots \) 的和函数.
