## \( \text{ § }{23.3}\mathrm{\;B} \) 函数与 \( \Gamma \) 函数

在计算积分或者解常微分方程时, 常常遇到其解不能表示为初等函数的问题. 解决这个问题的方法之一是引进一些新的函数, 它们可能是函数项级数的和函数, 或者是用含参变量积分表示的函数, 然后研究它们的性质, 甚至做出函数值表. 这类函数一般称为特殊函数. 这一节介绍的 Beta 函数 (记为 B 函数) 与 Gamma 函数 (记为 \( \Gamma \) 函数) 就属于最重要特殊函数之列. 它们在数学的很多分支中都有应用. 有不少重要的定积分值可以用它们表示出来. 此外, Gamma 函数的一些性质的证明也是数学分析中很好的训练，在 23.3.4 小节中的部分内容可作为习题课的补充材料. 这方面还可参考 \( \left\lbrack  {{25},{55}}\right\rbrack \) 的 \( \text{ § }{7.4} \) 中的内容.

#### 23.3.1 \( \mathrm{B} \) 函数

B 函数也称为第一类 Euler 积分. B 函数是一个二元函数, 它的定义是

\[
\mathrm{B}\left( {p, q}\right)  = {\int }_{0}^{1}{x}^{p - 1}{\left( 1 - x\right) }^{q - 1}\mathrm{\;d}x,\;p, q > 0. \tag{23.9}
\]

它还有下列等价的积分表示

\[
\mathrm{B}\left( {p, q}\right)  = {\int }_{0}^{+\infty }\frac{{y}^{p - 1}}{{\left( 1 + y\right) }^{p + q}}\mathrm{\;d}y = {\int }_{0}^{+\infty }\frac{{y}^{q - 1}}{{\left( 1 + y\right) }^{p + q}}\mathrm{\;d}y
\]

\[
= \frac{1}{2}{\int }_{0}^{+\infty }\frac{{y}^{p - 1} + {y}^{q - 1}}{{\left( 1 + y\right) }^{p + q}}\mathrm{\;d}y
\]

\[
\mathrm{B}\left( {p, q}\right)  = 2{\int }_{0}^{\pi /2}{\cos }^{{2p} - 1}\theta {\sin }^{{2q} - 1}\theta \mathrm{d}\theta .
\]

B 函数的主要性质如下:

1. 对称性: \( \mathrm{B}\left( {p, q}\right)  = \mathrm{B}\left( {q, p}\right) \) ;

2. \( \mathrm{B}\left( {p, q}\right) \) 在其定义域上连续,且有任意阶连续偏导数; 3. 递推公式:

\[
\mathrm{B}\left( {p, q + 1}\right)  = \frac{q}{p + q}\mathrm{\;B}\left( {p, q}\right) ,\;\mathrm{B}\left( {p + 1, q}\right)  = \frac{p}{p + q}\mathrm{\;B}\left( {p, q}\right) .
\]

如果 \( m, n \) 都是正整数,则

\[
\mathrm{B}\left( {m, n}\right)  = \frac{\left( {m - 1}\right) !\left( {n - 1}\right) !}{\left( {m + n - 1}\right) !}\;\text{ (参见上册 326 页例题 10.4.10). }
\]

#### 23.3.2 Γ 函数

\( \Gamma \) 函数也称为第二类 Euler 积分. 它是一元函数,其含参积分定义为

\[
\Gamma \left( x\right)  = {\int }_{0}^{+\infty }{t}^{x - 1}{\mathrm{e}}^{-t}\mathrm{\;d}t, x > 0. \tag{23.10}
\]

它有如下的 Gauss 无穷乘积分解 (也称为 Euler-Gauss 公式) \( {}^{0} \)

\[
\Gamma \left( x\right)  = \mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{n!{n}^{x}}{x\left( {x + 1}\right) \cdots \left( {x + n}\right) }. \tag{23.11}
\]

\( \Gamma \) 函数的主要性质如下:

1.Γ 函数与 \( \mathrm{B} \) 函数的关系

\[
\mathrm{B}\left( {p, q}\right)  = \frac{\Gamma \left( p\right)  \cdot  \Gamma \left( q\right) }{\Gamma \left( {p + q}\right) },\;p > 0, q > 0;
\]

2. \( \Gamma \left( x\right) \) 在 \( \left( {0, + \infty }\right) \) 上为严格下凸函数,它及其任意阶导数都连续,且

\[
{\Gamma }^{\left( n\right) }\left( x\right)  = {\int }_{0}^{+\infty }{t}^{x - 1}{\left( \ln t\right) }^{n}{\mathrm{e}}^{-t}\mathrm{\;d}t
\]

3. 递推公式 (由此从 \( \Gamma \left( 1\right)  = 1 \) 出发得到 \( \Gamma \left( {n + 1}\right)  = \)

\[
n!,\forall n \in  {\mathbf{N}}_{ + })
\]

\[
\Gamma \left( {x + 1}\right)  = {x\Gamma }\left( x\right) ,\;x > 0 \tag{23.12}
\]

4. \( \ln \Gamma \left( x\right) \) 在 \( \left( {0, + \infty }\right) \) 上为严格下凸函数;

5. Legendre 加倍公式: 对于 \( x > 0 \) 有

\[
\Gamma \left( {2x}\right)  = \frac{{2}^{{2x} - 1}}{\sqrt{\pi }}\Gamma \left( x\right) \Gamma \left( {x + \frac{1}{2}}\right) ;
\]

6. 余元公式: 对于 \( 0 < x < 1 \) 有

\[
\Gamma \left( x\right) \Gamma \left( {1 - x}\right)  = \frac{\pi }{\sin {\pi x}}.
\]

注 1 从积分定义 \( \left( {23.10}\right) \) 出发，利用递推公式 (23.12) 可以将 \( \Gamma \) 函数的定义域开拓如下. 将 (23.12) 变形为

\[
\Gamma \left( x\right)  = \frac{\Gamma \left( {x + 1}\right) }{x}. \tag{23.13}
\]

注意到右边当 \( - 1 < x < 0 \) 时也有定义,于是我们用 (23.13) 的右边来定义 \( - 1 < x < 0 \) 时的 \( \Gamma \) 函数的值,以此类推, \( \Gamma \) 函数的定义域可以开拓到除去 0 和负整数的一切实数. 当然这样的开拓结果与 (13.37) 的无穷乘积定义完全一致.

#### 23.3.3 例题

例题 23.3.1 设平面 \( x = 0, y = 0, z = 0 \) 与 \( x + y + z = 1 \) 围成四面体 \( V \) ,证明:

\[
{\iiint }_{V}{x}^{a - 1}{y}^{b - 1}{z}^{c - 1}\mathrm{\;d}x\mathrm{\;d}y\mathrm{\;d}z = \frac{\Gamma \left( a\right) \Gamma \left( b\right) \Gamma \left( c\right) }{\left( {a + b + c}\right) \Gamma \left( {a + b + c}\right) }\;\left( {a, b, c > 0}\right) .
\]

证 作以下计算即可:

\[
I = {\iiint }_{V}{x}^{a - 1}{y}^{b - 1}{z}^{c - 1}\mathrm{\;d}x\mathrm{\;d}y\mathrm{\;d}z = {\int }_{0}^{1}\mathrm{\;d}x{\int }_{0}^{1 - x}\mathrm{\;d}y{\int }_{0}^{1 - x - y}{x}^{a - 1}{y}^{b - 1}{z}^{c - 1}\mathrm{\;d}z
\]

\[
= {\int }_{0}^{1}{x}^{a - 1}\mathrm{\;d}x{\int }_{0}^{1 - x}{\left. \left( {y}^{b - 1}\frac{1}{c}{z}^{c}\right) \right| }_{0}^{1 - x - y}\mathrm{\;d}y
\]

\[
= \frac{1}{c}{\int }_{0}^{1}{x}^{a - 1}\mathrm{\;d}x{\int }_{0}^{1 - x}{y}^{b - 1}{\left( 1 - x - y\right) }^{c}\mathrm{\;d}y.
\]

再令 \( y = \left( {1 - x}\right) t \) ,则

\[
I = \frac{1}{c}{\int }_{0}^{1}{x}^{a - 1}\mathrm{\;d}x{\int }_{0}^{1}{\left( 1 - x\right) }^{b - 1}{t}^{b - 1}{\left( 1 - x\right) }^{c}{\left( 1 - t\right) }^{c}\left( {1 - x}\right) \mathrm{d}t
\]

\[
= \frac{1}{c}{\int }_{0}^{1}{x}^{a - 1}{\left( 1 - x\right) }^{b + c}\mathrm{\;d}x{\int }_{0}^{1}{t}^{b - 1}{\left( 1 - t\right) }^{c}\mathrm{\;d}t
\]

\[
= \frac{1}{c}\mathrm{\;B}\left( {a, b + c + 1}\right) \mathrm{B}\left( {b, c + 1}\right)
\]

\[
= \frac{1}{c}\frac{\Gamma \left( a\right) \Gamma \left( {b + c + 1}\right) }{\Gamma \left( {a + b + c + 1}\right) } \cdot  \frac{\Gamma \left( b\right) \Gamma \left( {c + 1}\right) }{\Gamma \left( {b + c + 1}\right) }
\]

\[
= \frac{\Gamma \left( a\right) \Gamma \left( b\right) \Gamma \left( c\right) }{\left( {a + b + c}\right) \Gamma \left( {a + b + c}\right) }.
\]

例题 23.3.2 确定 \( \alpha ,\beta ,\gamma \) ,使

\[
I = {\iiint }_{D}\frac{\mathrm{d}x\mathrm{\;d}y\mathrm{\;d}z}{1 + {x}^{\alpha } + {y}^{\beta } + {z}^{\gamma }} <  + \infty ,
\]

并求 \( I \) 的值,其中 \( D = \{ \left( {x, y, z}\right)  \mid  x \geq  0, y \geq  0, z \geq  0\} \) .

解 首先应该有 \( \alpha  > 0,\beta  > 0,\gamma  > 0 \) . 至于进一步的条件,我们将在计算中得到. 令 \( x = {u}^{2/\alpha }, y = {v}^{2/\beta }, z = {w}^{2/\gamma } \) ,则

\[
I = \frac{8}{\alpha \beta \gamma }{\iiint }_{\Omega }\frac{{u}^{2/\alpha  - 1}{v}^{2/\beta  - 1}{w}^{2/\gamma  - 1}}{1 + {u}^{2} + {v}^{2} + {w}^{2}}\mathrm{\;d}u\mathrm{\;d}v\mathrm{\;d}w,
\]

其中 \( \Omega  = \{ \left( {u, v, w}\right)  \mid  u \geq  0, v \geq  0, w \geq  0\} \) . 再作球坐标变换

\[
u = \rho \sin \varphi \cos \theta ,\;v = \rho \sin \varphi \sin \theta ,\;w = \rho \cos \varphi ,
\]

\[
\rho  \geq  0,0 \leq  \varphi  \leq  \frac{\pi }{2},0 \leq  \theta  \leq  \frac{\pi }{2},
\]

则

\[
I = \frac{8}{\alpha \beta \gamma }{\int }_{0}^{\pi /2}{\cos }^{2/\alpha  - 1}\theta {\sin }^{2/\beta  - 1}\theta \mathrm{d}\theta  \cdot  {\int }_{0}^{\pi /2}{\sin }^{2\left( {\frac{1}{\alpha } + \frac{1}{\beta }}\right)  - 1}\varphi {\cos }^{2/\gamma  - 1}\varphi \mathrm{d}\varphi
\]

\[
\times  {\int }_{0}^{+\infty }\frac{{\rho }^{2\left( {\frac{1}{\alpha } + \frac{1}{\beta } + \frac{1}{\gamma }}\right)  - 1}}{1 + {\rho }^{2}}\mathrm{\;d}\rho
\]

\[
= \frac{1}{\alpha \beta \gamma }\mathrm{B}\left( {\frac{1}{\alpha },\frac{1}{\beta }}\right) \mathrm{B}\left( {\frac{1}{\alpha } + \frac{1}{\beta },\frac{1}{\gamma }}\right) {\int }_{0}^{+\infty }\frac{{t}^{\frac{1}{\alpha } + \frac{1}{\beta } + \frac{1}{\gamma } - 1}}{1 + t}\mathrm{\;d}t.
\]

可见当且仅当 \( \frac{1}{\alpha } + \frac{1}{\beta } + \frac{1}{\gamma } < 1 \) 时后一个积分收敛,并且有

\[
{\int }_{0}^{+\infty }\frac{{t}^{\frac{1}{\alpha } + \frac{1}{\beta } + \frac{1}{\gamma } - 1}}{1 + t}\mathrm{\;d}t = \mathrm{B}\left( {\frac{1}{\alpha } + \frac{1}{\beta } + \frac{1}{\gamma },1 - \left( {\frac{1}{\alpha } + \frac{1}{\beta } + \frac{1}{\gamma }}\right) }\right) ,
\]

这样便得到

\[
I = \frac{1}{\alpha \beta \gamma }\Gamma \left( \frac{1}{\alpha }\right) \Gamma \left( \frac{1}{\beta }\right) \Gamma \left( \frac{1}{\gamma }\right) \Gamma \left( {1 - \left( {\frac{1}{\alpha } + \frac{1}{\beta } + \frac{1}{\gamma }}\right) }\right) .
\]

再令 \( {yt} = z \) ，则

例题 23.3.4 证明 Riemann 的 zeta 函数的积分形式

\[
\zeta \left( s\right)  = \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{{n}^{s}} = \frac{1}{\Gamma \left( s\right) }{\int }_{0}^{+\infty }\frac{{x}^{s - 1}}{{\mathrm{e}}^{x} - 1}\mathrm{\;d}x,\;s > 1.
\]

证 对于 \( x > 0 \) 有展开式

\[
\frac{1}{{\mathrm{e}}^{x} - 1} = \frac{{\mathrm{e}}^{-x}}{1 - {\mathrm{e}}^{-x}}
\]

\[
= {\mathrm{e}}^{-x}\left( {1 + {\mathrm{e}}^{-x} + {\mathrm{e}}^{-{2x}} + \cdots }\right)  = \mathop{\sum }\limits_{{n = 1}}^{\infty }{\mathrm{e}}^{-{nx}}.
\]

于是 \( \forall A > 0 \) 有

\[
{\int }_{0}^{A}\frac{{x}^{s - 1}}{{\mathrm{e}}^{x} - 1}\mathrm{\;d}x = {\int }_{0}^{A}{x}^{s - 1}\left( {\mathop{\sum }\limits_{{n = 1}}^{\infty }{\mathrm{e}}^{-{nx}}}\right) \mathrm{d}x.
\]

对于固定的 \( s > 1 \) ,级数 \( \mathop{\sum }\limits_{{n = 1}}^{\infty }{x}^{s - 1}{\mathrm{e}}^{-{nx}} \) 关于 \( x \in  \lbrack 0, + \infty ) \) 是一致收敛的,所以

\[
{\int }_{0}^{A}\frac{{x}^{s - 1}}{{\mathrm{e}}^{x} - 1}\mathrm{\;d}x = \mathop{\sum }\limits_{{n = 1}}^{\infty }{\int }_{0}^{A}{x}^{s - 1}{\mathrm{e}}^{-{nx}}\mathrm{\;d}x = \mathop{\sum }\limits_{{n = 1}}^{\infty }{\int }_{0}^{nA}{\left( \frac{y}{n}\right) }^{s - 1}\left( \frac{{\mathrm{e}}^{-y}}{n}\right) \mathrm{d}y
\]

\[
= \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{{n}^{s}}{\int }_{0}^{nA}{y}^{s - 1}{\mathrm{e}}^{-y}\mathrm{\;d}y.
\]

这个级数对于 \( A \in  \lbrack 0, + \infty ) \) 是一致收敛的,于是令 \( A \rightarrow   + \infty \) ,得

\[
{\int }_{0}^{+\infty }\frac{{x}^{s - 1}}{{\mathrm{e}}^{x} - 1}\mathrm{\;d}x = \Gamma \left( s\right) \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{{n}^{s}}.
\]

## \( {23.3.4}\Gamma \) 函数的特征刻画和几个重要公式的证明

前面已介绍过 \( \Gamma \) 函数满足如下三条性质:

1. 当 \( x > 0 \) 时, \( \Gamma \left( x\right)  > 0 \) ,且 \( \Gamma \left( 1\right)  = 1 \) ;

2. \( \Gamma \left( {x + 1}\right)  = {x\Gamma }\left( x\right) \) ;

3. \( \ln \Gamma \left( x\right) \) 在 \( \left( {0, + \infty }\right) \) 上为下凸函数.

关于 \( \Gamma \) 函数的一个非常漂亮的结果是 Bohr (玻尔) 与 Mollerup (莫勒鲁普) 定理 (1922 年),即上面的三条性质完全刻画了 \( \Gamma \) 函数.

命题 23.3.1 (Bohr-Mollerup 定理) 如果定义在 \( \left( {0, + \infty }\right) \) 上的函数 \( f \) 满足下列三个条件:

(1) \( f\left( x\right)  > 0 \) ,且 \( f\left( 1\right)  = 1 \) ,

(2) \( f\left( {x + 1}\right)  = {xf}\left( x\right) \) ,

(3) \( \ln f\left( x\right) \) 是 \( \left( {0, + \infty }\right) \) 上的下凸函数,

则 \( f\left( x\right)  \equiv  \Gamma \left( x\right) , x \in  \left( {0, + \infty }\right) \) .

证 已知 \( \Gamma \) 函数满足上述三条,故只要证明 \( f \) 是由 \( \left( 1\right) ,\left( 2\right) ,\left( 3\right) \) 惟一确定的函数就可以了. 而由 (2) 只要对 \( x \in  \left( {0,1}\right) \) 进行证明. 令 \( \varphi \left( x\right)  = \ln f\left( x\right) \) ,则

\[
\varphi \left( {x + 1}\right)  = \varphi \left( x\right)  + \ln x,\;0 < x <  + \infty , \tag{23.14}
\]

\( \varphi \left( 1\right)  = 0 \) ,且 \( \varphi \) 是下凸函数. 设 \( 0 < x < 1 \) ,考虑 \( \varphi \) 在

\[
\left\lbrack  {n, n + 1}\right\rbrack  ,\;\left\lbrack  {n + 1, n + 1 + x}\right\rbrack  ,\;\left\lbrack  {n + 1, n + 2}\right\rbrack
\]

三个闭区间上的差商, 有

\[
\ln n = \varphi \left( {n + 1}\right)  - \varphi \left( n\right)  \leq  \frac{\varphi \left( {n + 1 + x}\right)  - \varphi \left( {n + 1}\right) }{x}
\]

\[
\leq  \varphi \left( {n + 2}\right)  - \varphi \left( {n + 1}\right)  = \ln \left( {n + 1}\right) . \tag{23.15}
\]

另一方面, 重复 (23.14) 可得

\[
\varphi \left( {n + 1 + x}\right)  = \varphi \left( x\right)  + \ln \left\lbrack  {x\left( {x + 1}\right) \cdots \left( {x + n}\right) }\right\rbrack  .
\]

代入(23.15)并整理得

\[
0 \leq  \varphi \left( x\right)  - \ln \left( \frac{n!{n}^{x}}{x\left( {x + 1}\right) \cdots \left( {x + n}\right) }\right)  \leq  x\ln \left( {1 + \frac{1}{n}}\right) .
\]

由对数函数的连续性,令 \( n \rightarrow  \infty \) ,得

\[
\varphi \left( x\right)  = \ln \left( {\mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{n!{n}^{x}}{x\left( {x + 1}\right) \cdots \left( {x + n}\right) }}\right) .
\]

因此

\[
f\left( x\right)  = \mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{n!{n}^{x}}{x\left( {x + 1}\right) \cdots \left( {x + n}\right) }.
\]

利用例题 16.1.4 的结论知 \( f\left( x\right)  = \Gamma \left( x\right) \) .

注 既然这个定理完全刻画了 \( \Gamma \) 函数,因此就可以由它出发导出 \( \Gamma \) 函数的基本性质 (参见 [7] 的第 7 章), 下面就是一个例子.

命题 23.3.2 (Legendre 加倍公式) 对于 \( x > 0 \) 成立

\[
\Gamma \left( {2x}\right)  = \frac{{2}^{{2x} - 1}}{\sqrt{\pi }}\Gamma \left( x\right) \Gamma \left( {x + \frac{1}{2}}\right) .
\]

证 设 \( g\left( x\right)  = \frac{{2}^{x - 1}}{\sqrt{\pi }}\Gamma \left( \frac{x}{2}\right) \Gamma \left( \frac{x + 1}{2}\right) \) ,利用 \( \Gamma \left( \frac{1}{2}\right)  = \sqrt{\pi } \) ,容易检验 \( g\left( x\right) \) 满足 Bohr-Mollerup 定理中的条件(1)一(3)，可见有

\[
\Gamma \left( x\right)  = \frac{{2}^{x - 1}}{\sqrt{\pi }}\Gamma \left( \frac{x}{2}\right) \Gamma \left( \frac{x + 1}{2}\right) ,
\]

将其中 \( x \) 换为 \( {2x} \) 就得到所要的加倍公式.

命题 23.3.3 (余元公式) 对于 \( 0 < x < 1 \) 成立

\[
\Gamma \left( x\right) \Gamma \left( {1 - x}\right)  = \frac{\pi }{\sin {\pi x}}.
\]

根据本书前面的知识, 我们可以对余元公式给出下列三种证法:

证 1 利用 \( \Gamma \) 函数的 Euler-Gauss 无穷乘积表达式 (23.11) (即 (13.38)) 与正弦函数的无穷乘积表达式 (13.30). 具体细节见例题 13.4.4 以及 (13.40).

证 2 首先利用 \( \Gamma \) 函数与 \( \mathrm{B} \) 函数的关系得到

\[
\Gamma \left( x\right) \Gamma \left( {1 - x}\right)  = \mathrm{B}\left( {x,1 - x}\right)  = {\int }_{0}^{+\infty }\frac{1}{{y}^{x}\left( {1 + y}\right) }\mathrm{d}y,
\]

然后利用例题 16.1.3 的 Euler 积分公式.

证 3 利用上册 402 页参考题 8(2) 的结果: 当 \( {2m} + 1 < {2n} \) 时,有

\[
{\int }_{-\infty }^{+\infty }\frac{{x}^{2m}}{1 + {x}^{2n}}\mathrm{\;d}x = \frac{\pi }{n}\csc \frac{\left( {{2m} + 1}\right) \pi }{2n},
\]

于是有

\[
\Gamma \left( \frac{{2m} + 1}{2n}\right) \Gamma \left( {1 - \frac{{2m} + 1}{2n}}\right)  = \mathrm{B}\left( {\frac{{2m} + 1}{2n},1 - \frac{{2m} + 1}{2n}}\right)
\]

\[
= {\int }_{0}^{+\infty }\frac{{y}^{\frac{{2m} + 1}{2n} - 1}}{1 + y}\mathrm{\;d}y
\]

\[
= {2n}{\int }_{0}^{+\infty }\frac{{x}^{2m}}{1 + {x}^{2n}}\mathrm{\;d}x
\]

\[
= \pi \csc \frac{\left( {{2m} + 1}\right) \pi }{2n}
\]

当 \( x \) 为无理数时,通过形如 \( \frac{{2m} + 1}{2n} \) 的有理数取极限.

命题 23.3.4 ( \( \Gamma \) 函数的 Stirling 公式) 关于 \( \Gamma \) 函数有渐近公式 \( {}^{\text{ ① }} \) :

\[
\Gamma \left( {x + 1}\right)  \sim  \sqrt{2\pi x}{\left( \frac{x}{\mathrm{e}}\right) }^{x}\left( {x \rightarrow   + \infty }\right) . \tag{23.16}
\]

证 (此证明取自 [46]) 在 \( \Gamma \left( x\right)  = {\int }_{0}^{+\infty }{t}^{x - 1}{\mathrm{e}}^{-t}\mathrm{\;d}t \) 中令 \( t = x\left( {1 + u}\right) \) ,得到

\[
\Gamma \left( {x + 1}\right)  = {x}^{x + 1}{\mathrm{e}}^{-x}{\int }_{-1}^{+\infty }{\left\lbrack  \left( 1 + u\right) {\mathrm{e}}^{-u}\right\rbrack  }^{x}\mathrm{\;d}u. \tag{23.17}
\]

令

\[
h\left( u\right)  = \left\{  \begin{array}{ll} \frac{2}{{u}^{2}}\left\lbrack  {u - \ln \left( {1 + u}\right) }\right\rbrack  , &  - 1 < u <  + \infty ,\;u \neq  0, \\  1, & u = 0, \end{array}\right.
\]

则 \( h \) 在 \( \left( {-1, + \infty }\right) \) 上为单调减少的连续函数,并满足

\[
\left( {1 + u}\right) {\mathrm{e}}^{-u} = \exp \left( {-\frac{{u}^{2}}{2}h\left( u\right) }\right) .
\]

于是,对 (23.17) 作代换 \( u = s\sqrt{\frac{2}{x}} \) ,得

\[
\Gamma \left( {x + 1}\right)  = {x}^{x}{\mathrm{e}}^{-x}\sqrt{2x}{\int }_{-\infty }^{+\infty }{\psi }_{x}\left( s\right) \mathrm{d}s,
\]

其中

\[
{\psi }_{x}\left( s\right)  = \left\{  \begin{array}{ll} \exp \left\lbrack  {-{s}^{2}h\left( {s\sqrt{\frac{2}{x}}}\right) }\right\rbrack  , &  - \sqrt{\frac{x}{2}} < s <  + \infty , \\  0, & s \leq   - \sqrt{\frac{x}{2}}. \end{array}\right.
\]

---

① 对于 \( x \) 为正整数的情况，在上册 364-365 页已给出了 \( n \) ! 的 Stirling 公式的证明. 在 13.4.3 小节练习题 10 中指出可以用无穷乘积方法证明关于 \( n \) ! 的 Stirling 公式. 又在上册 374 页的参考题 15 中指出如何得到更为精确的 Stirling 公式. 此外, 还在上册 363 页的 (11.31) 不加证明地给出了含有 Bernoulli 数的一般形式的 Stirling 公式.

---

可以验证:

(1) 对每个 \( s \) 而言,当 \( x \rightarrow   + \infty \) 时, \( {\psi }_{x}\left( s\right)  \rightarrow  {\mathrm{e}}^{-{s}^{2}} \) ;

(2) 当 \( x \geq  1 \) 时,对 \( s > 0 \) ,有 \( 0 < {\psi }_{x}\left( s\right)  < {\psi }_{1}\left( s\right) \) ;

(3) 对 \( s < 0 \) ,有 \( 0 < {\psi }_{x}\left( s\right)  < {\mathrm{e}}^{-{s}^{2}} \) ;

(4) 对任意 \( A > 0 \) ,含参变量 \( x \) 的广义积分 \( {\int }_{-\infty }^{+\infty }{\psi }_{x}\left( s\right) \mathrm{d}s \) 在闭区间 \( \left\lbrack  {-A, A}\right\rbrack \) 上一致收敛;

(5) \( {\int }_{0}^{+\infty }{\psi }_{1}\left( s\right) \mathrm{d}s \) 收敛.

因而, 就可以得到

\[
\mathop{\lim }\limits_{{x \rightarrow   + \infty }}{\int }_{-\infty }^{+\infty }{\psi }_{x}\left( s\right) \mathrm{d}s = {\int }_{-\infty }^{+\infty }{\mathrm{e}}^{-{s}^{2}}\mathrm{\;d}s = \sqrt{\pi },
\]

即

\[
\mathop{\lim }\limits_{{x \rightarrow   + \infty }}\frac{\Gamma \left( {x + 1}\right) }{\sqrt{2\pi x}{x}^{x}{\mathrm{e}}^{-x}} = 1,
\]

这就是关于 \( \Gamma \) 函数的 Stirling 公式 (23.16).

注 这只是关于 \( \Gamma \) 函数的 Stirling 公式的最简单形式. 与上册 363 页 (11.31) 类似的关于 \( \Gamma \) 函数的一般 Stirling 公式见 [18] (第二卷的 540 小节) 等参考书.

#### 23.3.5 练习题

1. 计算

(1) \( {\int }_{0}^{1}\frac{\mathrm{d}x}{\sqrt{x\ln \frac{1}{x}}};\; \) (2) \( {\int }_{0}^{+\infty }\frac{\sqrt[4]{x}}{{\left( 1 + x\right) }^{2}}\mathrm{\;d}x \) .

2. 试用 \( \Gamma \) 函数或 \( \mathrm{B} \) 函数表示

(1) \( {\int }_{0}^{\pi /2}{\tan }^{\alpha }x\mathrm{\;d}x\left( {\left| \alpha \right|  < 1}\right) \) ;

(2) \( {\int }_{-1}^{1}{\left( 1 + x\right) }^{a}{\left( 1 - x\right) }^{b}\mathrm{\;d}x\;\left( {a, b > 0}\right) \) .

3. \( n \) 为正整数, \( p > 0 \) ,证明:

\[
\mathrm{B}\left( {p, n}\right)  = \frac{\left( {n - 1}\right) !}{p\left( {p + 1}\right) \cdots \left( {p + n - 1}\right) }.
\]

4. 证明: \( \ln \Gamma \left( x\right) \) 是下凸函数.

5. 按照下列步骤证明公式 \( \mathrm{B}\left( {p, q}\right)  = \frac{\Gamma \left( p\right) \Gamma \left( q\right) }{\Gamma \left( {p + q}\right) } \) .

(1) \( \Gamma \left( p\right)  = 2{\int }_{0}^{+\infty }{u}^{{2p} - 1}{\mathrm{e}}^{-{u}^{2}}\mathrm{\;d}u \)

(2) \( \Gamma \left( p\right) \Gamma \left( q\right)  = \mathop{\lim }\limits_{{A \rightarrow   + \infty }}4{\iint }_{G\left( A\right) }f\left( {u, v}\right) \mathrm{d}u\mathrm{\;d}v \) ,其中

\[
f\left( {u, v}\right)  = {u}^{{2p} - 1}{v}^{{2q} - 1}{\mathrm{e}}^{-\left( {{u}^{2} + {v}^{2}}\right) },
\]

G\left( A\right)  = \{ \left( {u, v}\right)  \mid  0 \leq  u \leq  A,0 \leq  v \leq  A\}

\[
\mathop{\lim }\limits_{{A \rightarrow   + \infty }}{\iint }_{D\left( A\right) }f\left( {u, v}\right) \mathrm{d}u\mathrm{\;d}v = \frac{1}{4}\mathrm{\;B}\left( {p, q}\right) \Gamma \left( {p + q}\right)
\]

\[
\mathop{\lim }\limits_{{A \rightarrow   + \infty }}{\iint }_{D\left( {\sqrt{2}A}\right) }f\left( {u, v}\right) \mathrm{d}u\mathrm{\;d}v = \frac{1}{4}\mathrm{\;B}\left( {p, q}\right) \Gamma \left( {p + q}\right)
\]

(4) \( \mathrm{B}\left( {p, q}\right)  = \frac{\Gamma \left( p\right) \Gamma \left( q\right) }{\Gamma \left( {p + q}\right) } \) .
