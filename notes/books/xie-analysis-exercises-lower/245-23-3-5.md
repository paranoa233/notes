#### 23.3.5 练习题

1. 计算

(1) \( {\int }_{0}^{1}\frac{\mathrm{d}x}{\sqrt{x\ln \frac{1}{x}}};\; \) (2) \( {\int }_{0}^{+\infty }\frac{\sqrt[4]{x}}{{\left( 1 + x\right) }^{2}}\mathrm{\;d}x \) .

2. 试用 \( \Gamma \) 函数或 \( \mathrm{B} \) 函数表示

(1) \( {\int }_{0}^{\pi /2}{\tan }^{\alpha }x\mathrm{\;d}x\left( {\left| \alpha \right|  < 1}\right) \) ;

(2) \( {\int }_{-1}^{1}{\left( 1 + x\right) }^{a}{\left( 1 - x\right) }^{b}\mathrm{\;d}x\;\left( {a, b > 0}\right) \) .

3. \( n \) 为正整数, \( p > 0 \) ,证明:

\[
\mathrm{B}\left( {p, n}\right)  = \frac{\left( {n - 1}\right) !}{p\left( {p + 1}\right) \cdots \left( {p + n - 1}\right) }.
\]

4. 证明: \( \ln \Gamma \left( x\right) \) 是下凸函数.

5. 按照下列步骤证明公式 \( \mathrm{B}\left( {p, q}\right)  = \frac{\Gamma \left( p\right) \Gamma \left( q\right) }{\Gamma \left( {p + q}\right) } \) .

(1) \( \Gamma \left( p\right)  = 2{\int }_{0}^{+\infty }{u}^{{2p} - 1}{\mathrm{e}}^{-{u}^{2}}\mathrm{\;d}u \)

(2) \( \Gamma \left( p\right) \Gamma \left( q\right)  = \mathop{\lim }\limits_{{A \rightarrow   + \infty }}4{\iint }_{G\left( A\right) }f\left( {u, v}\right) \mathrm{d}u\mathrm{\;d}v \) ,其中

\[
f\left( {u, v}\right)  = {u}^{{2p} - 1}{v}^{{2q} - 1}{\mathrm{e}}^{-\left( {{u}^{2} + {v}^{2}}\right) },
\]

\[
G\left( A\right)  = \{ \left( {u, v}\right)  \mid  0 \leq  u \leq  A,0 \leq  v \leq  A\}
\]

(3) 令 \( D\left( R\right)  = \left\{  {\left( {r,\theta }\right) \left| {\;0 \leq  r \leq  R,0 \leq  \theta  \leq  \frac{\pi }{2}}\right. }\right\} \) ,则

\[
\mathop{\lim }\limits_{{A \rightarrow   + \infty }}{\iint }_{D\left( A\right) }f\left( {u, v}\right) \mathrm{d}u\mathrm{\;d}v = \frac{1}{4}\mathrm{\;B}\left( {p, q}\right) \Gamma \left( {p + q}\right)
\]

\[
\mathop{\lim }\limits_{{A \rightarrow   + \infty }}{\iint }_{D\left( {\sqrt{2}A}\right) }f\left( {u, v}\right) \mathrm{d}u\mathrm{\;d}v = \frac{1}{4}\mathrm{\;B}\left( {p, q}\right) \Gamma \left( {p + q}\right)
\]

(4) \( \mathrm{B}\left( {p, q}\right)  = \frac{\Gamma \left( p\right) \Gamma \left( q\right) }{\Gamma \left( {p + q}\right) } \) .
