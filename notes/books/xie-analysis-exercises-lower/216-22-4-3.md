#### 22.4.3 例题

例题 22.4.1 计算 \( {\iint }_{{\mathbf{R}}^{2}}{\mathrm{e}}^{-\left( {{x}^{2} + {y}^{2}}\right) }\mathrm{d}x\mathrm{\;d}y \) ,并求 Poisson 积分 \( {\int }_{-\infty }^{+\infty }{\mathrm{e}}^{-{x}^{2}}\mathrm{\;d}x \) .

解 被积函数为 \( {\mathrm{e}}^{-{r}^{2}} \) . 当 \( r \rightarrow   + \infty \) 时,它比任何 \( \frac{1}{{r}^{p}}\left( {p > 2}\right) \) 都更快地趋于零, 所以广义二重积分是收敛的.

取同心圆族

\[
{\Omega }_{\rho } = \left\{  {{x}^{2} + {y}^{2} \leq  {\rho }^{2}}\right\}
\]

于是

\[
{\iint }_{{\mathbf{R}}^{2}}{\mathrm{e}}^{-\left( {{x}^{2} + {y}^{2}}\right) }\mathrm{d}x\mathrm{\;d}y = \mathop{\lim }\limits_{{\rho  \rightarrow   + \infty }}{\iint }_{{\Omega }_{\rho }}{\mathrm{e}}^{-\left( {{x}^{2} + {y}^{2}}\right) }\mathrm{d}x\mathrm{\;d}y
\]

\[
= \mathop{\lim }\limits_{{\rho  \rightarrow   + \infty }}{\int }_{0}^{2\pi }\mathrm{d}\theta {\int }_{0}^{\rho }{\mathrm{e}}^{-{r}^{2}}r\mathrm{\;d}r = \mathop{\lim }\limits_{{\rho  \rightarrow   + \infty }}\pi \left( {1 - {\mathrm{e}}^{-{\rho }^{2}}}\right)  = \pi .
\]

在上述计算中, 如果取正方形族

\[
{\Omega }_{l} = \{  - l \leq  x \leq  l, - l \leq  y \leq  l\} ,
\]

则

\[
{\iint }_{{\mathbf{R}}^{2}}{\mathrm{e}}^{-\left( {{x}^{2} + {y}^{2}}\right) }\mathrm{d}x\mathrm{\;d}y = \mathop{\lim }\limits_{{l \rightarrow   + \infty }}{\iint }_{{\Omega }_{l}}{\mathrm{e}}^{-\left( {{x}^{2} + {y}^{2}}\right) }\mathrm{d}x\mathrm{\;d}y
\]

\[
= \mathop{\lim }\limits_{{l \rightarrow   + \infty }}\left\{  {{\int }_{-l}^{l}{\mathrm{e}}^{-{x}^{2}}\mathrm{\;d}x{\int }_{-l}^{l}{\mathrm{e}}^{-{y}^{2}}\mathrm{\;d}y}\right\}   = {\left\{  {\int }_{-\infty }^{+\infty }{\mathrm{e}}^{-{x}^{2}}\mathrm{\;d}x\right\}  }^{2}.
\]

因此

\[
{\int }_{-\infty }^{+\infty }{\mathrm{e}}^{-{x}^{2}}\mathrm{\;d}x = \sqrt{\pi }.
\]

注 Poisson 积分中 \( {\mathrm{e}}^{-{x}^{2}} \) 的原函数不是初等函数, Poisson 敏锐地观察到极坐标下二重积分有因子 \( r\mathrm{\;d}r\mathrm{\;d}\theta \) ,由此出发他给出了上述巧妙的算法 \( {}^{\text{ ① }} \) .

例题 22.4.2 讨论广义重积分

\[
I = {\iint }_{D}\frac{\mathrm{d}x\mathrm{\;d}y}{{\left( x + y\right) }^{p}}
\]

的收敛性,其中 \( D = \{ \left( {x, y}\right)  \mid  0 \leq  x \leq  1, x + y \geq  1\} \) . 当积分收敛时,求积分的值.

解 由于被积函数恒正,因此可以取任一列趋于 \( D \) 的有界区域列,使得积分容易计算, 为此取

\[
{D}_{n} = \{ \left( {x, y}\right)  \mid  0 \leq  x \leq  1,1 \leq  x + y \leq  n\} ,
\]

---

① Poisson 积分也称为 Euler-Poisson 积分或概率积分. 该积分的其他计算方法在前面已有介绍, 参见上册 392 页例题 12.3.7 及其注解, 第十六章例题 16.1.2.

---

则

\[
{I}_{n} = {\iint }_{{D}_{n}}\frac{\mathrm{d}x\mathrm{\;d}y}{{\left( x + y\right) }^{p}}.
\]

作变量替换 \( x = u, x + y = v \) ,则

\[
{I}_{n} = {\int }_{0}^{1}\mathrm{\;d}u{\int }_{1}^{n}\frac{\mathrm{d}v}{{v}^{p}} = \frac{1}{1 - p}\left( {{n}^{1 - p} - 1}\right) .
\]

从而当 \( p > 1 \) 时,积分收敛,且

\[
{\iint }_{D}\frac{\mathrm{d}x\mathrm{\;d}y}{{\left( x + y\right) }^{p}} = \frac{1}{p - 1}.
\]

注 如果取

\[
{D}_{n} = \{ \left( {x, y}\right)  \mid  0 \leq  x \leq  1, x + y \geq  1, y \leq  n\} ,
\]

则计算要复杂得多.

例题 22.4.3 证明: 广义二重积分

\[
{\iint }_{\begin{matrix} {x \geq  1} \\  {y \geq  1} \end{matrix}}\frac{{x}^{2} - {y}^{2}}{{\left( {x}^{2} + {y}^{2}\right) }^{2}}\mathrm{\;d}x\mathrm{\;d}y
\]

发散.

证 我们将证明在无限扇形 \( {D}^{\prime } = \{ \left( {x, y}\right)  \mid  {2y} \leq  x \leq  {3y}, x \geq  1, y \geq  1\} \) 上

\[
\left| \frac{{x}^{2} - {y}^{2}}{{\left( {x}^{2} + {y}^{2}\right) }^{2}}\right|  \geq  \frac{C}{{r}^{2}}
\]

其中 \( r = \sqrt{{x}^{2} + {y}^{2}}, C \) 为正常数. 事实上当 \( {2y} \leq  x \leq  {3y} \) 时,

\[
4{y}^{2} \leq  {x}^{2} \leq  9{y}^{2},\;3{y}^{2} \leq  {x}^{2} - {y}^{2} \leq  8{y}^{2},\;5{y}^{2} \leq  {x}^{2} + {y}^{2} \leq  {10}{y}^{2},
\]

从而

\[
{x}^{2} - {y}^{2} \geq  3{y}^{2} = \frac{3}{10} \cdot  {10}{y}^{2} \geq  \frac{3}{10}\left( {{x}^{2} + {y}^{2}}\right) .
\]

于是

\[
\frac{{x}^{2} - {y}^{2}}{{\left( {x}^{2} + {y}^{2}\right) }^{2}} \geq  \frac{3}{{10}{r}^{2}}
\]

所以原广义二重积分发散.
