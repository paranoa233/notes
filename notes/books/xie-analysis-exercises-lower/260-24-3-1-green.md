#### 24.3.1 Green 公式

Green 公式是将平面内的某区域上的二重积分与该区域边界上的一个特定的第二型曲线积分之间建立联系的一个重要公式.

设 \( D \) 是 \( {\mathbf{R}}^{2} \) 内的一个有界闭区域,其边界 \( \partial D \) 由光滑曲线或逐段光滑曲线组成. 又设函数 \( P, Q \) 在 \( D \) 上有关于自变量 \( x \) 和 \( y \) 的连续偏导数,则下列 Green 公式成立:

\[
{\iint }_{D}\left( {-\frac{\partial P}{\partial y} + \frac{\partial Q}{\partial x}}\right) \mathrm{d}x\mathrm{\;d}y = {\oint }_{\partial D}P\mathrm{\;d}x + Q\mathrm{\;d}y, \tag{24.9}
\]

其中 \( \partial D \) 的方向关于 \( D \) 是正向的.

由平面上两类曲线积分之间的关系式 (24.6) 知, Green 公式 (24.9) 又可以写为

\[
{\iint }_{D}\left( {\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y}}\right) \mathrm{d}x\mathrm{\;d}y = {\oint }_{\partial D} - Q\mathrm{\;d}x + P\mathrm{\;d}y
\]

\[
= {\oint }_{\partial D}\left\lbrack  {Q\sin \left( {x,\mathbf{n}}\right)  + P\cos \left( {x,\mathbf{n}}\right) }\right\rbrack  \mathrm{d}s
\]

其中 \( \mathbf{n} \) 是 \( \partial D \) 上的单位外法向量. 利用 \( \angle \left( {x,\mathbf{n}}\right)  = \angle \left( {x, y}\right)  + \angle \left( {y,\mathbf{n}}\right)  = \frac{\mathbf{\pi }}{2} + \angle \left( {y,\mathbf{n}}\right) \) (参见图 24.2), 则

\[
{\iint }_{D}\left( {\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y}}\right) \mathrm{d}x\mathrm{\;d}y = {\oint }_{\partial D}\left\lbrack  {P\cos \left( {x,\mathbf{n}}\right)  + Q\cos \left( {y,\mathbf{n}}\right) }\right\rbrack  \mathrm{d}s. \tag{24.10}
\]

公式 (24.9), (24.10) 给我们提供了间接求平面上曲线积分的方法, 即利用 Green 公式将曲线积分化为二重积分,即使曲线 \( C \) 不封闭,也可以采用添加 “辅助线”的方法去做.

例题 24.3.1 \( C \) 为抛物线 \( {2x} = \pi {y}^{2} \) 自 \( \left( {0,0}\right) \) 到 \( \left( {\frac{\pi }{2},1}\right) \) 的弧段,求

\[
I = {\int }_{C}\left( {{2x}{y}^{3} - {y}^{2}\cos x}\right) \mathrm{d}x + \left( {1 - {2y}\sin x + 3{x}^{2}{y}^{2}}\right) \mathrm{d}y.
\]

解 令 \( P\left( {x, y}\right)  = {2x}{y}^{3} - {y}^{2}\cos x, Q\left( {x, y}\right)  = 1 - {2y}\sin x + 3{x}^{2}{y}^{2} \) ,则

\[
- \frac{\partial P}{\partial y} + \frac{\partial Q}{\partial x} = 0
\]

为了利用 Green 公式 (24.9), 添加辅助线 (如图 24.3), 则

\[
I = {\int }_{C} + {\int }_{\widetilde{BA}} + {\int }_{\widetilde{AO}} + {\int }_{\widetilde{AB}} + {\int }_{\widetilde{OA}}.
\]

由 Green 公式, 前三项的积分为零, 于是

\[
I = {\int }_{\overset{\text{ ⏜ }}{AB}} + {\int }_{\overset{\text{ ⏜ }}{OA}}
\]

\[
= {\int }_{0}^{1}\left\lbrack  {1 - {2y}\sin \frac{\pi }{2} + 3{\left( \frac{\pi }{2}\right) }^{2}{y}^{2}}\right\rbrack  \mathrm{d}y = \frac{{\pi }^{2}}{4}\text{ . }
\]

例题 24.3.2 计算积分

\[
I = {\oint }_{C}\frac{\cos \left( {\mathbf{r},\mathbf{n}}\right) }{r}\mathrm{\;d}s
\]

其中 \( C \) 为逐段光滑的简单闭曲线, \( \mathbf{r} = \left( {x, y}\right) , r = \left| \mathbf{r}\right|  = \sqrt{{x}^{2} + {y}^{2}},\mathbf{n} \) 是 \( C \) 上的单位外法向量.

解 由 \( \cos \left( {\mathbf{r},\mathbf{n}}\right)  = \frac{\mathbf{r} \cdot  \mathbf{n}}{r} = \frac{1}{r}\left( {x\cos \left( {\mathbf{n}, x}\right)  + y\cos \left( {\mathbf{n}, y}\right) }\right) \) 得到

\[
I = {\oint }_{C}\left( {\frac{x}{{r}^{2}}\cos \left( {\mathbf{n}, x}\right)  + \frac{y}{{r}^{2}}\cos \left( {\mathbf{n}, y}\right) }\right) \mathrm{d}s.
\]

当 \( \left( {0,0}\right) \) 在 \( C \) 外时,利用 Green 公式 (24.10),则

\[
I = {\iint }_{D}\left\lbrack  {{\partial }_{x}\left( \frac{x}{{r}^{2}}\right)  + {\partial }_{y}\left( \frac{y}{{r}^{2}}\right) }\right\rbrack  \mathrm{d}x\mathrm{\;d}y = 0,
\]

其中 \( D \) 是以 \( C \) 为边界的区域.

当 \( \left( {0,0}\right) \) 在 \( C \) 内时,以 \( \left( {0,0}\right) \) 为圆心,以充分小的 \( \varepsilon \) 为半径作圆 \( {C}_{\varepsilon } \) ,使得 \( {C}_{\varepsilon } \) 在 \( C \) 内,以 \( C \) 及 \( {C}_{\varepsilon } \) 为边界的区域为 \( {D}_{\varepsilon } \) ,则

\[
I = {\oint }_{C}\left( {\frac{x}{{r}^{2}}\cos \left( {\mathbf{n}, x}\right)  + \frac{y}{{r}^{2}}\cos \left( {\mathbf{n}, y}\right) }\right) \mathrm{d}s + {\oint }_{{C}_{\varepsilon }}\left( {\frac{x}{{r}^{2}}\cos \left( {\mathbf{n}, x}\right)  + \frac{y}{{r}^{2}}\cos \left( {\mathbf{n}, y}\right) }\right) \mathrm{d}s
\]

\[
- {\oint }_{{C}_{\varepsilon }}\left( {\frac{x}{{r}^{2}}\cos \left( {\mathbf{n}, x}\right)  + \frac{y}{{r}^{2}}\cos \left( {\mathbf{n}, y}\right) }\right) \mathrm{d}s.
\]

其中 \( {C}_{\varepsilon } \) 上的单位法向量 \( \mathbf{n} \) 的方向指向坐标原点. 对前两项用 Green 公式,则

\[
I =  - {\oint }_{{C}_{\varepsilon }}\left( {\frac{x}{{r}^{2}}\cos \left( {\mathbf{n}, x}\right)  + \frac{y}{{r}^{2}}\cos \left( {\mathbf{n}, y}\right) }\right) \mathrm{d}s.
\]

在圆周 \( {C}_{\varepsilon } \) 上,

\[
\cos \left( {\mathbf{n}, x}\right)  =  - \frac{x}{\varepsilon },\;\cos \left( {\mathbf{n}, y}\right)  =  - \frac{y}{\varepsilon },\;r = \varepsilon ,
\]

从而

\[
I = {\oint }_{{C}_{\varepsilon }}\frac{1}{\varepsilon }\mathrm{d}s = {2\pi }.
\]

当 \( \left( {0,0}\right)  \in  C \) 时,过原点作曲线 \( C \) 的切线 \( {OA},{OB} \) ,设 \( {OA},{OB} \) 的夹角为 \( \theta \) (见图 24.4), 如果曲线 \( C \) 在原点光滑,则 \( \theta  = \pi \) .

作一个以 \( \left( {0,0}\right) \) 点为心, \( \varepsilon \) 为半径的圆 \( {B}_{\varepsilon } \) , 记 \( {B}_{\varepsilon } \) 的圆周在 \( C \) 内的部分为 \( {C}_{\varepsilon } \) ,由上面的计算知

\[
I = \mathop{\lim }\limits_{{\varepsilon  \rightarrow  0}}{\int }_{{C}_{\varepsilon }}\frac{1}{\varepsilon }\mathrm{d}s = \mathop{\lim }\limits_{{\varepsilon  \rightarrow  0}}{\theta }_{\varepsilon } = \theta ,
\]

其中 \( {\theta }_{\varepsilon } \) 是 \( {C}_{\varepsilon } \) 所对应的圆心角.

注 注意使用 Green 公式的条件: \( P, Q \) 在 \( D \) 上连续可微. 这正是在本题中要区分 \( \left( {0,0}\right) \) 在 \( C \) 外, \( C \) 内和 \( C \) 上三种情况讨论的缘故. 许多学生没有能够注意到这里的区别,尤其是当点 \( \left( {0,0}\right) \) 在 \( C \) 内时,他们会错误地认为被积函数在 \( C \) 上是连续可微的, 因而 Green 公式就可以用了.

例题 24.3.3 计算

\[
I = {\oint }_{C}\frac{{\mathrm{e}}^{y}}{{x}^{2} + {y}^{2}}\left\lbrack  {\left( {x\sin x + y\cos x}\right) \mathrm{d}x + \left( {y\sin x - x\cos x}\right) \mathrm{d}y}\right\rbrack  ,
\]

其中 \( C : {x}^{2} + {y}^{2} = 1 \) ,取逆时针方向.

解 令

\[
P\left( {x, y}\right)  = \frac{{\mathrm{e}}^{y}}{{x}^{2} + {y}^{2}}\left( {x\sin x + y\cos x}\right) ,
\]

\[
Q\left( {x, y}\right)  = \frac{{\mathrm{e}}^{y}}{{x}^{2} + {y}^{2}}\left( {y\sin x - x\cos x}\right) ,
\]

由计算知 \( \frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x} \) ,从而可在不包含原点的区域上用 Green 公式. 为此取 \( {C}_{\varepsilon } : {x}^{2} + {y}^{2} = {\varepsilon }^{2} \) ,取逆时针方向,则

\[
I = {\oint }_{C} - {\oint }_{{C}_{\varepsilon }} + {\oint }_{{C}_{\varepsilon }}.
\]

对等式右边前两项用 Green 公式，于是

\[
I = {\oint }_{{C}_{\varepsilon }}\frac{{\mathrm{e}}^{y}}{{x}^{2} + {y}^{2}}\left\lbrack  {\left( {x\sin x + y\cos x}\right) \mathrm{d}x + \left( {y\sin x - x\cos x}\right) \mathrm{d}y}\right\rbrack
\]

\[
= \frac{1}{{\varepsilon }^{2}}{\oint }_{{C}_{\varepsilon }}{\mathrm{e}}^{y}\left\lbrack  {\left( {x\sin x + y\cos x}\right) \mathrm{d}x + \left( {y\sin x - x\cos x}\right) \mathrm{d}y}\right\rbrack  .
\]

记 \( {C}_{\varepsilon } \) 围成的区域为 \( {D}_{\varepsilon } \) ,再用一次 Green 公式,则

\[
I = \frac{1}{{\varepsilon }^{2}}{\iint }_{{D}_{\varepsilon }} - 2{\mathrm{e}}^{y}\cos x\mathrm{\;d}x\mathrm{\;d}y.
\]

应用积分中值定理得到

\[
I = \frac{1}{{\varepsilon }^{2}}\left( {-{2\pi }{\varepsilon }^{2}{\mathrm{e}}^{\eta }\cos \xi }\right)  =  - {2\pi }{\mathrm{e}}^{\eta }\cos \xi ,
\]

其中 \( \left( {\xi ,\eta }\right)  \in  {D}_{\varepsilon } \) . 上述等式 \( \forall \varepsilon  > 0 \) 都对,令 \( \varepsilon  \rightarrow  0 \) ,得

\[
I = \mathop{\lim }\limits_{{\varepsilon  \rightarrow  {0}^{ + }}}\left( {-{2\pi }{\mathrm{e}}^{\eta }\cos \xi }\right)  =  - {2\pi }.
\]

平面图形的面积 作为 Green 公式的直接推论, 可以得到如下的面积公式.

由逐段光滑的简单曲线 \( C \) 所界的面积 \( S \) 可用曲线积分表示为

\[
S = {\oint }_{C}x\mathrm{\;d}y =  - {\oint }_{C}y\mathrm{\;d}x = \frac{1}{2}{\oint }_{C}x\mathrm{\;d}y - y\mathrm{\;d}x,
\]

其中曲线的正向为逆时针方向 (参见上册 336-340 页的公式 (11.1) 及例题).

例题 24.3.4 计算双纽线

\[
{\left( {x}^{2} + {y}^{2}\right) }^{2} = {a}^{2}\left( {{x}^{2} - {y}^{2}}\right) \tag{24.11}
\]

所围区域的面积.

解 1 由对称性只需计算第一、四象限的面积. 令

\[
x = r\cos \theta ,\;y = r\sin \theta ,\; - \frac{\pi }{4} \leq  \theta  \leq  \frac{\pi }{4},
\]

代入方程 (24.11) 得 \( r = a\sqrt{\cos {2\theta }} \) ,则参数方程为

\[
x\left( \theta \right)  = a\cos \theta \sqrt{\cos {2\theta }},\;y\left( \theta \right)  = a\sin \theta \sqrt{\cos {2\theta }}, - \frac{\pi }{4} \leq  \theta  \leq  \frac{\pi }{4}.
\]

并且

\[
{x}^{\prime }\left( \theta \right)  = a\left( {-\sin \theta }\right) \sqrt{\cos {2\theta }} + a\cos \theta \frac{-\sin {2\theta }}{\sqrt{\cos {2\theta }}},
\]

\[
{y}^{\prime }\left( \theta \right)  = a\cos \theta \sqrt{\cos {2\theta }} + a\sin \theta \frac{-\sin {2\theta }}{\sqrt{\cos {2\theta }}},
\]

所以

\[
x{y}^{\prime } - y{x}^{\prime } = {a}^{2}\cos {2\theta }.
\]

由面积公式得到

\[
S = 2 \cdot  \frac{1}{2}\oint x\mathrm{\;d}y - y\mathrm{\;d}x = {\int }_{-\pi /4}^{\pi /4}{a}^{2}\cos {2\theta }\mathrm{d}\theta  = {a}^{2}.
\]

解 2 (用定积分求面积) 在极坐标系中, 双纽线 (24.11) 的方程为

\[
{r}^{2} = {a}^{2}\cos {2\theta },\;\theta  \in  \left\lbrack  {-\frac{\pi }{4},\frac{\pi }{4}}\right\rbrack   \cup  \left\lbrack  {\frac{3\pi }{4},\frac{5\pi }{4}}\right\rbrack  .
\]

于是

\[
S = 4 \cdot  \frac{1}{2}{\int }_{0}^{\pi /4}{r}^{2}\mathrm{\;d}\theta  = 2{\int }_{0}^{\pi /4}{a}^{2}\cos {2\theta }\mathrm{d}\theta  = {a}^{2}.
\]
