1. 应用 Green 公式求下列第二型曲线积分.

(1) \( {\oint }_{C}\left( {{x}^{2} + {xy}}\right) \mathrm{d}x + \left( {{x}^{2} + {y}^{2}}\right) \mathrm{d}y, C \) 为由 \( x =  \pm  1, y =  \pm  1 \) 围成的正方形,取

(2) 求 \( {\oint }_{C}\ln \frac{2 + y}{1 + {x}^{2}}\mathrm{\;d}x + \frac{x\left( {y + 1}\right) }{2 + y}\mathrm{\;d}y, C \) 的定义同 (1);

(3) 求 \( {\oint }_{C}\left( {{x}^{2} - {y}^{2}}\right) \mathrm{d}x - {2xy}\mathrm{\;d}y, C \) 是由 \( {x}^{2} + {y}^{2} = 1, x = y \) 及 \( y \) 轴围成的曲边三角形, 取正向;

(4) 求 \( {\int }_{C}\frac{1}{{x}^{2} + {y}^{2}}\left( {x\mathrm{\;d}y - y\mathrm{\;d}x}\right) \) ,其中 \( C \) 是: (a) 旋轮线 \( x = a\left( {t - \sin t}\right)  - {a\pi } \) , \( y = a\left( {1 - \cos t}\right) \) 对应于 \( t = 0 \) 到 \( t = {2\pi } \) 的一拱; (b) \( {\left( x - 1\right) }^{2} + {\left( y - 1\right) }^{2} = 1 \) 从 \( \left( {2,1}\right) \) 经上半圆到 \( \left( {0,1}\right) \) 的一段弧.

2. 设 \( f\left( x\right) \) 连续可微, \( L \) 为逐段光滑闭曲线,证明:

(1) \( {\oint }_{L}f\left( {xy}\right) \left( {y\mathrm{\;d}x + x\mathrm{\;d}y}\right)  = 0 \)

(2) \( {\oint }_{L}f\left( {{x}^{2} + {y}^{2}}\right) \left( {x\mathrm{\;d}x + y\mathrm{\;d}y}\right)  = 0 \) .

3. 求 \( {\oint }_{C}\frac{\partial u}{\partial \mathbf{n}}\mathrm{d}s \) . 其中 \( u = {x}^{2} + {y}^{2}, C \) 为 \( {x}^{2} + {y}^{2} = {6x},\mathbf{n} \) 为 \( C \) 上的单位外法向量.

4. 设 \( C \) 为逐段光滑的简单闭曲线, \( \mathbf{l} \) 为给定方向,证明:

\[
{\oint }_{C}\cos \left( {\mathbf{l},\mathbf{n}}\right) \mathrm{d}s = 0
\]

其中 \( \mathbf{n} \) 为 \( C \) 上的单位外法向量.

5. 设 \( C \) 为包围原点的逐段光滑的简单闭曲线, \( {a}_{ij}\left( {i, j = 1,2}\right) \) 均为常数, \( X = \; {a}_{11}x + {a}_{12}y, Y = {a}_{21}x + {a}_{22}y,{a}_{11}{a}_{22} - {a}_{12}{a}_{21} \neq  0 \) ,证明:

\[
{\oint }_{C}\frac{X\mathrm{\;d}Y - Y\mathrm{\;d}X}{{X}^{2} + {Y}^{2}} = {2\pi }\operatorname{sgn}\left( {{a}_{11}{a}_{22} - {a}_{12}{a}_{21}}\right) .
\]

6. 设 \( L \) 是单位圆周 \( {x}^{2} + {y}^{2} = 1 \) ,方向为逆时针,求积分

\[
{\oint }_{L}\frac{\left( {x - y}\right) \mathrm{d}x + \left( {x + {4y}}\right) \mathrm{d}y}{{x}^{2} + 4{y}^{2}}.
\]

7. 利用曲线积分求下述曲线所围区域的面积:

(1) \( x = a{\cos }^{3}t, y = b{\sin }^{3}t,0 \leq  t \leq  {2\pi } \)

(2) \( {x}^{3} + {y}^{3} = {3axy} \)

(3) \( {\left( \frac{x}{a}\right) }^{{2n} + 1} + {\left( \frac{y}{b}\right) }^{{2n} + 1} = C{\left( \frac{x}{a}\right) }^{n}{\left( \frac{y}{b}\right) }^{n}, a, b, C > 0, n \) 为正整数.

8. 先证明曲线积分与路径无关，然后计算积分值:

(1) \( {\int }_{\left( 1,2\right) }^{\left( 3,4\right) }\varphi \left( x\right) \mathrm{d}x + \psi \left( y\right) \mathrm{d}y \) ,其中 \( \varphi \left( x\right) ,\psi \left( y\right) \) 是连续函数;

(2) \( {\int }_{\left( 1,0\right) }^{\left( 6,8\right) }\frac{x\mathrm{\;d}x + y\mathrm{\;d}y}{{x}^{2} + {y}^{2}} \) ,沿不通过原点的路径.

9. 对于以下一阶微分形式 \( w \) ,求函数 \( M\left( {x, y}\right)  \neq  0 \) ,使得在适当的区域内 \( {Mw} \) 为全微分, 并求其原函数:

(1) \( w = \left\lbrack  {-y\sqrt{{x}^{2} + {y}^{2} + 1} - x\left( {{x}^{2} + {y}^{2}}\right) }\right\rbrack  \mathrm{d}x + \left\lbrack  {x\sqrt{{x}^{2} + {y}^{2} + 1} - y\left( {{x}^{2} + {y}^{2}}\right) }\right\rbrack  \mathrm{d}y \) ;

(2) \( w = x\left\lbrack  {{\left( ay + bx\right) }^{3} + a{y}^{3}}\right\rbrack  \mathrm{d}x + y\left\lbrack  {{\left( ay + bx\right) }^{3} + b{x}^{3}}\right\rbrack  \mathrm{d}y \) .
